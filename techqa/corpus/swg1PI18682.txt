APAR STATUS
 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  While processing EJB 3.0 business methods, ITCAM assumes that
   the local and remote interfaces will have the same set of
   methods and processes only one of them.  Both the interfaces
   need to be interrogated to find out the universal set of
   business methods.
