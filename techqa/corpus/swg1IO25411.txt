APAR STATUS
 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  Content Engine preprocessor errors do not get routed to the IBM
   Content Navigator GUI on document creation.
   
   
    
   
   

LOCAL FIX

PROBLEM SUMMARY
 *  Content Engine preprocessor error was not getting passed to the
   error dialog on a document add.
   
   
    
   
   

PROBLEM CONCLUSION
 *  This has been fixed in IBM Content Navigator V2.0.3.8 LA010,
   V3.0 LA007, and V3.0.1. With this fix, any CE preprocessor
   error will show up on the document add dialog.
