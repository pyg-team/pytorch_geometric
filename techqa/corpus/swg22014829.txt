Title: IBM Security Bulletin: IBM SPSS Statistics Server is affected by CVE-2018-2579, CVE-2018-2602, CVE-2018-2603, CVE-2018-2633, and CVE-2018-2634 vulnerabilities - United States

Text:
 SECURITY BULLETIN

SUMMARY
 IBM SPSS Statistics has addressed the following vulnerabilities. 

VULNERABILITY DETAILS

DESCRIPTION:An unspecified vulnerability in Oracle Java SE related to the Java SE, Java SE Embedded, JRockit Libraries component could allow an unauthenticated attacker to obtain sensitive information resulting in a low confidentiality impact using unknown attack vectors.

SPSS Statistics Server V21 and V22 users can receive a free complementary upgrade to SPSS Statistics Server 25 edition. This will include upgrades of both the V21/V22 Server and all connected version V21/V22 clients. The upgrade will be a 1:1 upgrade, meaning equal server PVUs, client users, and module/bundle functionality. The offer is available now until 30 Sep 2018. SPSS Statistics 25 is a major upgrade from V21/V22, with major features, better reliability, and improved security from V21/V22. 

Contact SPSS Statistics Product Support [https://www.ibm.com/mysupport/s/topic/0TO500000001xpQGAQ/1742fe293dcd33daef1b0d51618ce39dspss-statistics?language=en_US&productId=01t50000004XAyOAAW], login with your IBMid, and open a new Support ticket citing this Secruity Bulletin to take advantage of this offer.
