Title: IBM Misc ELAN-related error during Modeler Desktop start up - United States

Text:
misc elan subcode four sentinel slm license licence TECHNOTE (TROUBLESHOOTING)

PROBLEM(ABSTRACT)
 When starting the IBM SPSS Modeler Desktop application, an error message is displayed:

==============================

License Error: Miscellaneous ELAN-related error.

Error sub-code: 4

==============================


SYMPTOM
Starting Modeler Desktop when using a Concurrent User License will bring up the Misc ELAN-related error message.
When clicking 'OK', error message is acknowledged and Modeler Desktop does not start.


CAUSE
This was caused by an incorrect value for the Concurrent License Manager during the install of SPSS Modeler Desktop.

ENVIRONMENT
Windows OS

DIAGNOSING THE PROBLEM
Starting SPSS Modeler Desktop using trial license did not produce this error message.
Confirm value of the DaemonHost parameter within the spssprod.inf file of <Modeler_Install>/bin directory.

RESOLVING THE PROBLEM
Insert the correct IP Address or Hostname of the server housing the Concurrent License Manager for the DaemonHost parameter of spssprod.inf.
