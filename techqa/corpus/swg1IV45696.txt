PROBLEM SUMMARY
 *  Provide system property to allow setting of enabled protocols
   when default SSLServerSocketFactory or SSLSocketFactory is used.
   Use of this system property is not recommended.  Use explicit
   setting of SSLProtocol instead.  This will avoid the
   complications of system properties.
    * com.ibm.jsse2.overrideDefaultProtocol = SSLv3 <HAT>00 SSL_TLS
   <HAT>00 SSL_TLSv2 <HAT>00 TLS <HAT>00 TLSv11 <HAT>00 TLSv12
    *       default - not set
    *       SSLv3   - set default enabled protocol to SSLv3
    *       SSL_TLS - set default enabled protocols to SSLv3 and
   TLS 1.0
    *       SSL_TLSv2 - set default enabled protocols to SSLv3, TLS
   1.0, TLS 1.1 and TLS 1.2
    *       TLSv1   - set default enabled protocol to TLS 1.0
    *       TLSv1.1 - set default enabled protocol to TLS 1.1
    *       TLSv1.2 - set default enabled protocol to TLS 1.2
