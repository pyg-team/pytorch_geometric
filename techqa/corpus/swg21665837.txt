Title: IBM FileNet Workplace XT  - Annotations fail to save - United States

Text:
 TECHNOTE (TROUBLESHOOTING)

PROBLEM(ABSTRACT)
 Using FileNet Workplace XT 1.1.5.x with IBM Production Imaging Edition Viewer Pro on IBM WebSphere Application Server 8.5.x. Saving annotations to TIFF or PDF format files via the Java Viewer Pro fails 

SYMPTOM
 

The following error message is displayed in the Java Viewer Pro : 

Annotations save failed

(Your login session has expired. Any unsaved changes have been lost. Close all viewer windows and log in again.)


IBM WebSphere Application Server SystemOut.log contains the following messages: 


com.ibm.ws.webcontainer.filter.FilterInstanceWrapper doFilter SRVE8109W: Uncaught exception thrown by filter SecurityPluginFilter: java.io.FileNotFoundException: SRVE0190E: File not found: /iviewpro/FnJavaV1Files/ji/server/logging/FastLoggingDateFormatImpl.class



CAUSE
This behavior may be encountered if WebSphere Session cookies are restricted to HTTP Only.



ENVIRONMENT
CPE 5.2.x
XT 1.1.5.x 
Java Viewer Pro 5.2 
IBM WebSphere Application Server 8.5.x 



RESOLVING THE PROBLEM
Disable the HTTP Only option for Session cookies in IBM WebSphere Application Server as indicated in Technote 7023814 : 


http://www.ibm.com/support/docview.wss?uid=swg27023814 [http://www.ibm.com/support/docview.wss?uid=swg27023814]

RELATED INFORMATION
 Disable HTTP Only option for Session cookies [http://www.ibm.com/support/docview.wss?uid=swg27023814]
