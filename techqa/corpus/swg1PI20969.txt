ERROR DESCRIPTION
 *  Microsoft Windows 7 IBM Rational Team Concert (RTC) Shell client
   task tray icon is unresponsive and disappears after a few
   minutes.
   
   What steps will reproduce the problem?
   1. Install RTC Shell client 4.0.6
   2. Dismiss the startup dialog box.
   3. Right click on the RTC Shell client icon in the task tray.
   
   What is the expected output?
   The startup dialog box closes. A context menu appears above the
   RTC Shell icon.The icon is responsive and does not disappear.
   
   What do you see instead?
   The startup dialog does not close. No context menu appears. The
   icon is unresponsive and disappears after few minutes.
   
   Operations like setting a sandbox, switching from basic to
   advanced mode or vice-versa from the RTC Shell Control Panel
   page do not complete.
