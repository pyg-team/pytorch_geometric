Title: IBM PM66495: JOB RUNNING MODELER STREAM WITH STATISTICS OUTPUT NODE PRODUCES 0-BYTE HTML OUTPUT WHEN WRITTEN TO REPOSITORY - United States

APAR STATUS
 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  When running a job with a Modeler stream containing a statistics
   output node, while setting the Result location of the jobstep to
   "Repository", the node produces a 0-byte HTML. However if the
   location is set to "application server file system", it creates
   a correct HTML file.
