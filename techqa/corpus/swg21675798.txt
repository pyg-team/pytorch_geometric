Title: IBM Security Bulletin: Vulnerabilities in IBM Tivoli Monitoring agent and shared library with potential for privilege escalation (CVE-2013-5467) - United States

Text:
privilege CVE-2013-5467 SECURITY BULLETIN

SUMMARY
 Privilege escalation vulnerability in an IBM Tivoli Monitoring (ITM) agent could be exploited by a local user to gain elevated privilege of another user. This vulnerability also affects a shared library shipped with all ITM components and agents. 

VULNERABILITY DETAILS
CVEID: CVE-2013-5467 [http://cve.mitre.org/cgi-bin/cvename.cgi?name=CVE-2013-5467]
Description: Privilege escalation vulnerability could be exploited by a local user to gain elevated privilege of another user. 

CVSS Base Score: 6.9
CVSS Temporal Score: See https://exchange.xforce.ibmcloud.com/vulnerabilities/88370 [https://exchange.xforce.ibmcloud.com/vulnerabilities/88370] for the current score
CVSS Environmental Score*: Undefined
CVSS Vector: (AV:L/AC:M/Au:N/C:C/I:C/A:C)
