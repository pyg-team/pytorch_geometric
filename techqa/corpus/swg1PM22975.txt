Title: IBM PM22975: XMLLIB FUNCTION XPATH DOES NOT WORK ON AIX - United States

APAR STATUS
 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  The xmllib function XPATH does not work on AIX.  The map works
   fine under Windows but when run on AIX, the xmllib->XPATH
   function evaluates incorrectly and produces NULL results.
   
   
    
   
   

LOCAL FIX
 *  WTXCQ - WTX00035521
   BD  / BD
   Circumvention: None
   
   
    
   
   

PROBLEM SUMMARY
 *  Users Affected:
   WebSphere Transformation Extender customers using AIX
   xmllib->XPATH functions
   
   Problem Description:
   
   xmllib->XPATH functions will not evaluate properly on AIX. You
   will get message that exit core failed and not output from the
   function.
   
   Platforms Affected:
   AIX32
   AIX64
   
   
    
   
   

PROBLEM CONCLUSION
 *  Problem Conclusion:
   XALAN library was corrupted. Libraries were extracted from
   tar.gz file and module was relined and now works properly. In
   addition data returned in Aix was UTF-16BE had to set that in
   character type.
   
   Delivered in:
   8.3.0.3
   
   
    
   
   

TEMPORARY FIX
