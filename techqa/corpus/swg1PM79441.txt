   The following incidents have been fixed:
   
   1 - Text '&CMOWN' has been removed from the prolog of
   SADBDENU(ADBBIND) to address the following TCz incident:
   
   Template Error - CCQC024S ISPF file tailoring failed for
   'ADBBIND' template in the 'SADBDENU' metadata library.
   
   2 - Specifying a STOGROUP for 'Change Management database
   task' did not get used by job template ADBMCMA 'Upgrade the
   Change Management tables'.
   
   3 - Specifying an alternate input value for step 'Change
   Management objects owner' did not get used by job template
   ADBMCMA 'Upgrade the Change Management tables' and job
   template ADBCDCH 'Allow change deletion'.
   
   4 - Submission of job template ADBCHANG 'Create Change
   Management database' receives the following SQLCODE=-204 /
   SQLCODEN204 error on the DROP statement:
   
   DSNT408I SQLCODE = -204, ERROR: ADBDCHG  IS AN
   UNDEFINED NAME
   DSNT418I SQLSTATE   = 42704 SQLSTATE RETURN CODE
