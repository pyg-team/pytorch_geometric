Title: IBM PM77875: SUPPORT MULTI-VALUED ATTRIBUTES FOR DISPLAY NAME - United States

Text:
  A FIX IS AVAILABLE
Fixes integrated in WebSphere Portal 7.0.0.1 & 7.0.0.2 Combined Cumulative Fixes [http://www-01.ibm.com/support/docview.wss?uid=swg24029452]

APAR STATUS
 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  The wimdomain.xsd shows that the cn attribute is single-valued,
   whereas the givenName attribute can carry multiple values. These
   settings match the defaults of a Portal installation.
   
   However, portal does not support multi-valued user attributes
   for the user display name.
   
   If a customer changes user.displaynameattribute to givenname in
   place of cn, interacting with the shelf throws the exception
   mentioned:

   as user.displaynameattribute.
