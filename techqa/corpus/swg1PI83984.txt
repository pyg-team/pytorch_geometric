PROBLEM SUMMARY
 *  ****************************************************************
   * USERS AFFECTED: All users of CICS TS Feature Pack for        *
   *                 Dynamic Scripting V2.0                       *
   ****************************************************************
   * PROBLEM DESCRIPTION: CICS Dynamic Scripting V2.0             *
   *                      applications cannot be installed.       *
   ****************************************************************
   CICS Dynamic Scripting V2.0 applications cannot be installed in
   CICS TS v5.4.  A CWWKZ0402E message is written to messages.log
   for the Liberty JVM server and an FFDC incident is raised.
   The FFDC log contains a message Caused by:
   java.lang.NoSuchMethodError:
   com/ibm/cics/php/CICSLogger.getSJTraceLevel()I
   
   
    
   
   

PROBLEM CONCLUSION
 *  CICS internal Java APIs are corrected to provide the required
   method to allow CICS Dynamic Scripting applications to install.
