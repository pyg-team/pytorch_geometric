Title: IBM Entering data in forms - intermittently crashes with error 'The RPC server is unavailable' - United States

 TECHNOTE (TROUBLESHOOTING)

PROBLEM(ABSTRACT)

SYMPTOM

CAUSE
Networking problems between remote site and headquarters (where application server(s) located). User selected to end an Excel process while Controller is working. 



ENVIRONMENT
Remote user located in different continent, and accesses Controller via WAN network.



RESOLVING THE PROBLEM
Ask I.T. department's network administrators to improve the quality of WAN network link to/from the remote user. 

 
Do not use CTRL-ALT-DEL, End Process on Excel while Controller is attempting to run a request.
