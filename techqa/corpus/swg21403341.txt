QUESTION
 Where can I download Product and Documentation for Transformation Server and/or InfoSphere Change Data Capture 

ANSWER
There are 2 links from which to download legacy DataMirror Software - one for Downlevel Code, and one for Current Versions. 
Down Level Code is available to existing customers via the IBM site.

https://www14.software.ibm.com/webapp/iwm/web/preLogin.do?lang=en_US&source=swg-datamirror [https://www14.software.ibm.com/webapp/iwm/web/preLogin.do?lang=en_US&source=swg-datamirror]


 * Customers must register on the site. 
 * Once registered the customer returns to the sign in page and proceeds to select the offerings they wish to download. 
 * The next step is to enter the access key. The access key is 7SQGWQDRZQ 
 * The customer must also view the licence and confirm that they accept the terms. 
 * Once terms have been accepted, the customer is directed to the Download Director and selects the files they want to download. 
 * After confirming selection, the download commences. 
 * Down Level Code is available for the following products 
 * Transformation Server 
 * iReflect 
 * iCluster 
 * PointBase (Version 5.7 ONLY)


Current Versions / Latest Release of software is available for download to existing customers with active maintenance as well as partners that provide support to their customers via the IBM Software and Services website. 

https://www-112.ibm.com/software/howtobuy/softwareandservices [https://www-112.ibm.com/software/howtobuy/softwareandservices] 
 * Customers must first register on this site. 
 * Once registered customers will need their SAP id to download software. 
 * This was sent to all customers at Transfer of Business
