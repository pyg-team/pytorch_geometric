Title: IBM IT11639: "DETAILS" BUTTON IN THE "CONFIGURE CRYPTO CERTIFICATE" GUI MIGHT BE DISPLAYED INACTIVE WHEN IT SHOULD BE ACTIVE - United States

APAR STATUS
 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  In the WebGui on  "Configure
   Crypto Certificate" panel  the "Details ..." button   is not
   displayed active  in situations where it should be active.
   This might happen when there is no passwort  on the certificate.
   The issue is  only  occuring on some browsers. The  button  is
   inactive  and consequently  certificate details cannot be
   displayed in the GUI.
   
   
    
   
   

LOCAL FIX
 *  Try the same  operation using a different Internet Browser.
   
   
    
   
   

PROBLEM SUMMARY
 *  Affected is viewing of certificate details.
   
   DataPower Web GUI users may encounter errors when viewing
   certificate details, or not be able to see certificate details
   at all.
   
   
    
   
   

PROBLEM CONCLUSION
 *  Fix is available in 7.2.0.2
   
   For a list of the latest fixpacks available, please see:
   http://www-01.ibm.com/support/docview.wss?uid=swg21237631 [http://www-01.ibm.com/support/docview.wss?uid=swg21237631]
