APAR STATUS
 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  Although the maximum size of the Collector Server trace file was
   set by TRACEMAX keyword in the server configuration profile, the
   trace file grows until Collector Server crashes or hangs.
   
   
    
   
   

LOCAL FIX
 *  Clean the log files on a regular basis.
