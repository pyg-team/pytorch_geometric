ERROR DESCRIPTION
 *  Users Affected:
   All users of ITCAM AD 7.1 Data Collector
   
   WebSphere shutdown is held by non-daemon TimerTak.
   Diagnostics reveal it to be a Timer created for MDD
   schedule in ITCAM AD Data Collector. Timer tasks are
   created as non-daemon by default.  If the MDD schedule
   happens to be running at shutdown, the application
   server shutdown may hang.
