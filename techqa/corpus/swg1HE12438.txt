APAR STATUS
 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  Using the Select Fields button in the Data Source Maintenance
   configuration within the Admin console, the Status field can be
   disabled or enabled.  This field is a required field for Data
   Sources, but it is not marked as such in any way, nor does our
   documentation indicate this requirement. Admin > Data Source
   Maintenance > "Approver View/Create/Edit"
   and "Mapper View/Create/Edit".  Once this field is deselected,
   User is not able to create Data Source.
   
   
    
   
   

LOCAL FIX
 *  N/A.  Field must be enabled.
   
   
    
   
   

PROBLEM SUMMARY
 *  When Status fields is NOT selected Under Admin - > Data Source
   Maintenance - > For Approver View/Create/Edit OR Mapper
   View/Create/Edit, creation or updation of Data Source gives
   system error.
   
   
    
   
   

PROBLEM CONCLUSION
 *  Fix is available in Fix Central in 6033 iFix1.
