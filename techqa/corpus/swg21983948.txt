Title: IBM Hit Error 500 - Internal Server Error when accessing IBM Cognos Workspace - United States

Text:
 TECHNOTE (TROUBLESHOOTING)

PROBLEM(ABSTRACT)
 IBM Cognos Connection can be accessed, and all other Studios are able to be launched with no problem. Only hit error 500 when accessing IBM Cognos Workspace. 

ENVIRONMENT
Cognos BI 10.2.2 is deployed on Weblogic 12c in Redhat Linux.
Weblogic is installed with Oracle JDK 1.7.
Cognos is installed with IBM JDK 1.7.
Weblogic domain is configured with IBM JDK 1.7.

RESOLVING THE PROBLEM
Configure Weblogic domain to use Oracle JDK 1.7 instead of IBM JDK 1.7
