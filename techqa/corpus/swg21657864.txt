Title: IBM CFG1058E Server list is empty or serious error when running migrate.bat - United States

Text:

 US English 

PROBLEM(ABSTRACT)
 Error received when trying to migrate ITCAM 7.1.0.0 DC/Agent to ITCAM 7.2


SYMPTOM
CFG1058E:Server list is empty or serious error occurred. 
Explanation: 
The server list is empty or a serious error occurred. 

Operator response: 
Gather the provider trace log and, if possible, the terminal log for
this session, and contact IBMÂ® support. 


CAUSE
The ITCAM for AD 7.1 level is below Fixpack 3 (7.1.0.3)

RESOLVING THE PROBLEM
The ITCAM for AD 7.1 level has to be at minimum 7.1 Fixpack 3(7.1.0.3) or higher along with the JVM being configured at that level as well, before running the migrate.bat. If the level is below 7.1.0.3, the error in the title is displayed
