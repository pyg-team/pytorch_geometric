Title: IBM Importing Atria license key results in "No Rational License keys found" error - United States

Text:
No; Rational; keys; found; remove; from; machine; according; terms; agreement; Atria; Classic; Clearcase; import TECHNOTE (TROUBLESHOOTING)

PROBLEM(ABSTRACT)
 Attempts to import Atria (IBM Rational ClearCase classic license key) results in the error "No Rational License Key Found". 

SYMPTOM
The full error message is as follows:

No Rational License Keys were found in this file. This will remove Rational licenses from your machine according to the terms of your license agreement.


image
[/support/docview.wss?uid=swg21633548&amp;aid=1]

CAUSE
Atria license keys should not be imported through IBM Rational License Key Administrator. The Rational License Key Administrator is used to import if the license is a Flex based Floating license.



DIAGNOSING THE PROBLEM
Check for the type of license key being used during import: 

 * Is the license a Flex based Floating license? 
 * Is the license a classic ClearCase ATRIA license?

RESOLVING THE PROBLEM
 [http://www-01.ibm.com/support/docview.wss?uid=swg21128958]Use the ClearCase Control Panel to import a Classic ClearCase floating license (Atria). 

Review technote 1128958: About Rational ClearCase Atria Licensing [http://www.ibm.com/support/docview.wss?uid=swg21128958] for instructions to import an Atria license.
