DOWNLOAD DESCRIPTION
IBM WebSphere eXtreme Scale is an elastic, scalable in-memory data grid that provides a key-addressable store of values or graphs. This product supports a processing unit state store for applications written in Java. You can replicate the data stored in the grid depending on availability requirements, and you can achieve out-of-the-box database integration. 

WebSphere eXtreme Scale can be integrated with existing WebSphere Application Server and WebSphere Application Server, Network Deployment installations or you can install the product separately, supporting both Java Platform, Standard Edition and Java Platform, Enterprise Edition application environments.

INSTALLATION INSTRUCTIONS
Downloading and assembling the product
1. Navigate your browser to the Find Downloads and Media [http://www-112.ibm.com/software/howtobuy/passportadvantage/paocustomer/sdma/SDMA?PO=DOWNLOAD_FIND_BY_PART_NUMBER] Web page on the IBM Passport Advantage Online Web site.
2. Select the Find by part number search option under Download finder options.
3. Search for each downloadable image by part number.
4. Download all of the required parts.
5. Install IBM Installation Manager [http://www-01.ibm.com/support/docview.wss?uid=swg24031300].
6. Use Installation Manager to install the product from the downloadable product repositories.
