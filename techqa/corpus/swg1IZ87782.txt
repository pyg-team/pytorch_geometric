ERROR DESCRIPTION
 *  Error Message: When processing a core file using the jextract
   tool, the following message was seen:
   Error. Internal failure detected during jextract
   .
   Stack Trace: N/A
   .
   
   
    
   
   

LOCAL FIX

PROBLEM SUMMARY
 *  The error was found to be caused by running jextract against a
   corrupted core file.
   The error message was misleading.  Printing "internal error"
   implied that the problem was within jextract, whereas in fact,
   the core file itself was invalid.
   
   
    
   
   

PROBLEM CONCLUSION
 *  This defect will be fixed in:
   5.0.0 SR13
   .
   The error message has been changed to be:
   Error. No known JVM found in dump
   .
   To obtain the fix:
   Install build 20101014 or later
