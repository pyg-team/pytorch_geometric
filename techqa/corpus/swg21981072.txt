Title: IBM Issue with connecting to LDAP server in Atlas - United States

Text:
LDAP ; filter ; atlas TECHNOTE (FAQ)

QUESTION
 Why does the connection to LDAP server fails in Atlas with the following error ?



Found 1000 matches found for the user [sAMAccountName= , Search Filter = (objectClass=user)] in LDAP server 

CAUSE
Using Search Filter in the LDAP Authentication component will cause this issue

ANSWER
In the LDAP Authentication component make sure the Search Filter is blank with no value.
