Title: IBM pxbridge: An error occurred while creating the table. Statement = create table IA_PXVALIDATE_97609440 (TEST char(1)) in iadb.IADBSPACE in InfoSphere Information Analyzer. - United States

Text:
 TECHNOTE (TROUBLESHOOTING)

PROBLEM(ABSTRACT)
 pxbridge: An error occurred while creating the table. Statement = "create table IA_PXVALIDATE_97609440 (TEST char(1)) in iadb.IADBSPACE in InfoSphere Information Analyzer. 
Once you are unable to create the pxvalidate table, it tries to insert a record, which fails with a database error message that the table does not exist.
Example DB2 error message: pxbridge,0: ODBC function "SQLExecute" reported: SQLSTATE = 42S02: Native Error Code = -204: Msg = [IBM(DataDirect OEM)][ODBC DB2 Wire Protocol driver][UDB DB2 for Windows, UNIX, and Linux]XMETAGRD.IA_PXVALIDATE 

SYMPTOM
Unable to run column analysis, run data rules, etc.


CAUSE
There is insufficient space available in the database, tablespace or on the mount point where database resides to complete the transaction.

DIAGNOSING THE PROBLEM
Work with your DBA to confirm space available and use df -k to check that there is sufficient disk space.

RESOLVING THE PROBLEM
Expand or clear space for database, tablespace or disk space
