APAR STATUS
 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  When trying to verify a Greenplum Inspection Engine using v9.1
   or v9.5 IE verification process, both basic and advanced
   verification will fail.
   
   
    
   
   

LOCAL FIX
 *  Patch v9p513 can be used to correct the advanced verification
   process. Full fix will be made available on FixCentral in the
   Next GPU.
   
   
    
   
   

PROBLEM SUMMARY
 *  Issue with advanced verification procedure which was failing for
   GreenplumDB database type. Fix was implemented in advance
   verification procedure to ensure both standard and advanced
   verification works and LOGIN_FAILED and SQL_ERROR are being
   logged in GDM_EXCEPTION as expected.
   
   
    
   
   

PROBLEM CONCLUSION
 *  Fix in advanced Verification procedure implemented and available
   in v9.0p600 (v9 GPU 600) on FixCentral
