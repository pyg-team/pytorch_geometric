ERROR DESCRIPTION
 *  Issue with Drag-n-drop from an entry document template
   
   Steps to Reproduce:
   1. Log in to Workplace XT.
   2. Create a folder
   3. Create an entry template and
   map the folder to the entry template (right click in XT >
   Settings/Preferences > Map entry templates...
   5. Log into IBM Conetent Navigator
   5. Open the Browse view in a Object Store.
   6. Drag a document from the DESKTOP into the folder and it
   should fail on the second attempt where the document title is
   missing and document security cannot be set.
   
   
    
   
   

LOCAL FIX

PROBLEM SUMMARY
 *  P8) When a folder has one associated entry template, after you
   add the first document, the following problems occur on
   subsequent adds:
   
       The security settings are not displayed.
       The document file name is not used as the default value for
   the Document Title property.
   
   
    
   
   

PROBLEM CONCLUSION
 *  IBM Content Navigator, Version 2.0.1 Fix Pack 001 resolves this
   issue.
