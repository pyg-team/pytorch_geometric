 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  TBSM 6.1
   
   
   Enter an expression that TBSM will use to determine the output
   value for this rule. The expression can be either the name of a
   field or a statement in the TBSM expression syntax. You can use
   the field list to select fields defined in the data feed.*
   
   Ex. Int(Value), Number*100
   
   This part from Configuring 'Incoming Status Rule ->Based on a
   numerical value' fails with following pop-up error message.
   
   CTGBB0105E The out put value expression named Int(Class) is not
   valid
   
   
    
   
   

LOCAL FIX
 *  NA
   
   
    
   
   

PROBLEM SUMMARY
 *  TBSM Status rule is unable to use IPL expressions in metric
   output rules. The rule was unable to be saved due to the
   expression trying to be executed before real data was
   substituted into the expression.
   
   
    
   
   

PROBLEM CONCLUSION
 *  Corrected issue so TBSM rules can continue to use IPL
   expressions in metric output rules.
   The fix for this APAR is contained in the following maintenance
   package:
   | Fixpack | 6.1.0-TIV-BSM-FP0001
