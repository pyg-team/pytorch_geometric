APAR STATUS
 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  Under data manager bulk assign search results, if you hover over
   the tags to show the task counts by tag popup, you cannot scroll
   if you are using Internet Explorer.  If you click on the scroll
   bar the popup will disappear.  This will only happen with
   Internet Explorer 8.0, it is working fine with Mozilla 3.6.
   
   The user is using Internet Explorer 8.0.7601.17514, also i
   tested with IE 8.0.7600.16385 and I am having the same issue.
   
   
    
   
   

LOCAL FIX
 *  Use Firefox or Mozilla browser.
   
   
    
   
   

PROBLEM SUMMARY
 *  ****************************************************************
   The pop up box under the Tags will not allow for scrolling when
   there are more than the screen allowed items to display.
   When trying to scroll the box will disappear, this only happen
   with IE.   The widget 'Tooltip' of Javascript framework Dojo
   does not support scrolling.
   ****************************************************************
   
   
    
   
   

PROBLEM CONCLUSION
 *  ****************************************************************
   Using widget 'DropDownButton' for the Tags pop up box to support
   scrolling. To fix this issue install the December 2012 Fix Pack
   for 9.7 Inspector
   ****************************************************************
