Title: IBM 'System.OutOfMemoryException' error when ingesting large files in IBM Content Collector - United States

Text:
 TECHNOTE (TROUBLESHOOTING)

PROBLEM(ABSTRACT)
 'System.OutOfMemoryException' error when ingesting large files in IBM Content Collector (ICC) 

SYMPTOM
The following error can be observed in the P8 connector log:

E_UNEXPECTED_EXCEPTION : An unexpected exception occurred.
Reason: E_UNEXPECTED_EXCEPTION : An unexpected exception occurred..
Reason: An unexpected exception occurred..
Reason: Exception of type 'System.OutOfMemoryException' was thrown.
Source: address:06B1526B




ENVIRONMENT
IBM Content Collector 

IBM FileNet P8 Repository


DIAGNOSING THE PROBLEM
Even though Web Services Enhancements (WSE) is not required when ICC is using SSL to communicate with P8, it may have already been installed on the ICC server. 

 * Verify that the following patch is applied for WSE:
   https://support.microsoft.com/en-us/kb/943508 [https://support.microsoft.com/en-us/kb/943508]


If WSE is not already installed on the ICC server, you do not need to apply the patch. 

RESOLVING THE PROBLEM
Install the following patch: 

 * https://support.microsoft.com/en-us/kb/943508 [https://support.microsoft.com/en-us/kb/943508]
