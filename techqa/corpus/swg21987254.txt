Title: IBM What I need to download from PPA to install or upgrade TRIRIGA - United States

Text:
TRIINSTALL; TRIUPGRADE ; #iotsupport ; #supiwms; #jalynch TECHNOTE (FAQ)

QUESTION
 Is it documented somewhere what the different IBM TRIRIGA part numbers in Passport Advantage (PPA) are? 

CAUSE
I am trying to figure out what I need to download.

ANSWER
Whenever a new release of TRIRIGA is made available, a 'Download Instructions' technote with a name like the following is typically published: 


IBM TRIRIGA <Application Version> and 
IBM TRIRIGA Application Platform <Platform Version> 
Download Instructions 


The 'Download Instructions' technote for a given TRIRIGA release can be found by doing a search on 'TRIRIGA download <platform version>' and contains the following information:  

 * A list of what TRIRIGA images (software bundles, installers, files, etc.) are part of the release
   
 * A list of what each TRIRIGA image contains
   
 * A list of what part number to specify in Passport Advantage in order to download each of these images
