 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  Opening DataPower WebGUI from a browser and selecting Blueprint
   or Pattern Console from the left panel might result in a blank
   page or "Session Expired" message.
   
   
    
   
   

LOCAL FIX
 *  -Use another supported browser.
   -Log in as requested.
   
   
    
   
   

PROBLEM SUMMARY
 *  Datapower users clicking the Blueprint/Pattern Console link from
   the WebGUI will see a session expired message or blank page in
   some situations.
   
   
    
   
   

PROBLEM CONCLUSION
 *  Fix is available in 6.0.1.7, 7.0.0.4 and 7.1.0.0.
