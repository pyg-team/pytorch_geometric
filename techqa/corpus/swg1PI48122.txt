Title: IBM PI48122: JCR XPATH QUERY RESULT SET INDEX PAGINATION INVALID - United States

Text:
  A FIX IS AVAILABLE
Fixes integrated in WebSphere Portal 8.0.0.1 Combined Cumulative Fixes [http://www-01.ibm.com/support/docview.wss?uid=swg24034497]



APAR STATUS
 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  Under certain conditions the JCR will incorrectly duplicate
   nodes in consecutive calls to the XPATH result set when index
   pagination is used.
   
   This can also result in the incorrect reporting of last index
   information. This happens when PAC filtering correctly removes
   nodes but the index pagination does not correctly account for
   the removal.
