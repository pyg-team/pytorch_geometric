APAR STATUS
 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  If the client operating system and browser that connect to the
   Dashboard server are set up in Japanese, after you enter the
   credentials in the login screen of the Dashboard, the main page
   does not appear.
   
   
    
   
   

LOCAL FIX

PROBLEM SUMMARY
 *   The main page should appear regardless of the
   system or browser language setting.
   
   
    
   
   

PROBLEM CONCLUSION
 *   The problem is fixed, and you can now use
   the Dashboard with a Japanese operating
   system and browser.
