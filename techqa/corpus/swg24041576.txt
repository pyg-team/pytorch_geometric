Recommended Interim Fixes
This document provides you with links to the interim fixes that are recommended for V8.5 Refresh Pack 7 for the following products: 

 * IBM Business Process Manager Express 
 * IBM Business Process Manager Standard 
 * IBM Business Process Manager Advanced 
 * IBM Business Process Manager Advanced – Process Server 

Note: You need to clear the Show recommended only option in Installation Manager to see all the WebSphere Application Server fixes needed by IBM Business Process Manager. 

Recommended : WebSphere Application Server V8.5.5 Fix Pack 8 interim fixes for the IBM Business Process Manager V8.5 Refresh Pack 7 products as outlined in the Remediation/Fixes section of document Security Bulletin: Multiple vulnerabilities in IBM® Java SDK affect WebSphere Application Server January 2016 CPU (CVE-2016-0475, CVE-2016-0466, CVE-2015-7575, CVE-2016-0448) [http://www.ibm.com/support/docview.wss?uid=swg21975424], specifically the three interim fixes listed under For V8.5.0.0 through 8.5.5.8 WebSphere Application Server Full Profile and WebSphere Application Server Hypervisor Edition. 


The following recommended WebSphere Application Server V8.5.5 Fix Pack 8 interim fixes are already included for new IBM Business Process Manager installation media. However, they all need to be manually installed for upgrade installations.
