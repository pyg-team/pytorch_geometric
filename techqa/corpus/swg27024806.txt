Title: IBM Detailed System Requirements for v9.1 of IBM InfoSphere Optim Test Data Management Solution for Oracle E-Business Suite with Data Masking Option - United States

Text:
 PRODUCT DOCUMENTATION

ABSTRACT
 This document summarizes the hardware, platforms, and software supported by v9.1 of the IBM® InfoSphere® Optim™ Test Data Management Solution for Oracle E-Business Suite with Data Masking Option. 

CONTENT


Version 9.1 of the InfoSphere Optim Test Data Management Solution for Oracle E-Business Suite with Data Masking Option is tested and distributed with the IBM InfoSphere Optim Test Data Management Solution, Version 9.1. Detailed hardware and software requirements for the InfoSphere Optim Test Data Management Solution are at: http://www.ibm.com/support/docview.wss?uid=swg27036209 [http://www.ibm.com/support/docview.wss?uid=swg27024294]. 

Note: Hardware and software requirements for the Optim solution may list database management systems and character sets that the Oracle E-Business Suite application does not support. Therefore, only a subset of the Optim solution information applies to the InfoSphere Optim Test Data Management Solution for Oracle E-Business Suite with Data Masking Option.

SUPPORTED ORACLE E-BUSINESS SUITE VERSIONS
The InfoSpere Optim Test Data Management Solution for Oracle E-Business Suite with Data Masking Option supports test data management for versions 11.5.10 and 12 of Oracle E-Business Suite. It supports data masking for the Core HRMS module of Oracle E-Business Suite versions 11.5.10 and 12.
