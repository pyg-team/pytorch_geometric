Title: IBM SystemP agents on AIX 7.1 TL3 - United States

Text:
SystemP PX PK VA AIX Premium Vios agent HMC support Aix Technology Level TL ITM TECHNOTE (FAQ)

QUESTION
 Are System P agents 6.2.2 FP2 certified on AIX 7.1 Technology Level 3 (TL3)?
This is the maintenance level of the agents of interest:
PX 06.22.02.04
VA 06.22.02.00
PK 06.22.02.03 

ANSWER
Yes, AIX 7.1 TL3 is supported for all the System p agents.
There are some newer interim fixes available as listed in the System P agent wiki:

https://www.ibm.com/developerworks/community/wikis/home?lang=en#!/wiki/Tivoli%20Monitoring/page/System%20P%20Agents [https://www.ibm.com/developerworks/community/wikis/home?lang=en#!/wiki/Tivoli%20Monitoring/page/System%20P%20Agents]

Also, in this wiki you can find the following best practices that the customers should know about:

--- 
Install and run the following agents to monitor your System P environment (see General Considerations above for details):

1) HMC Base agent v6.2.2.3 on any AIX LPAR to monitoring the HMC and the Servers, including CPU metrics on the servers and LPARs.
2) UNIX OS agent v6.3 FP2 or above on each AIX LPAR to monitor all LPAR metrics, including specific AIX metrics that have been ported from the AIX Premium agent.
3) VIOS Premium agent v6.2.2.2 on every VIOS (pre-installed), and install Interim Fix 6.2.2.2-TIV-ITM_VIOS_PREM-IF0005.
4) Remove the CEC Base agent if running the HMC Base agent v6.2.2.3.
5) Remove the AIX Premium agent if running the UNIX OS agent on each AIX LPAR.
