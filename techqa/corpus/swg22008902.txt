Title: IBM Security Bulletin: API Connect Portal is affected by multiple Drupal vulnerabilities - United States

Text:
 SECURITY BULLETIN

SUMMARY
 IBM API Connect has addressed the following vulnerabilities. 

API Connect Portal is affected by multiple Drupal vulnerabilities. 

VULNERABILITY DETAILS
CVEID:CVE-2017-6925 [http://cve.mitre.org/cgi-bin/cvename.cgi?name=CVE-2017-6925]
DESCRIPTION:Drupal could allow a remote attacker to bypass security restrictions, caused by a flaw in the entity access system. An attacker could exploit this vulnerability to gain access to view, create, update, or delete entities.
CVSS Base Score: 5.3
CVSS Temporal Score: Seehttps://exchange.xforce.ibmcloud.com/vulnerabilities/130574 [https://exchange.xforce.ibmcloud.com/vulnerabilities/130574]for the current score
CVSS Environmental Score*: Undefined
CVSS Vector: (CVSS:3.0/AV:N/AC:L/PR:N/UI:N/S:U/C:N/I:L/A:N)

CVEID:CVE-2017-6924 [http://cve.mitre.org/cgi-bin/cvename.cgi?name=CVE-2017-6924]
DESCRIPTION:Drupal could allow a remote attacker to bypass security restrictions, caused by a flaw in the REST API. An attacker could exploit this vulnerability to post comments without permission.
CVSS Base Score: 5.3
CVSS Temporal Score: Seehttps://exchange.xforce.ibmcloud.com/vulnerabilities/130573 [https://exchange.xforce.ibmcloud.com/vulnerabilities/130573]for the current score
CVSS Environmental Score*: Undefined
CVSS Vector: (CVSS:3.0/AV:N/AC:L/PR:N/UI:N/S:U/C:N/I:L/A:N)

CVEID:CVE-2017-6923 [http://cve.mitre.org/cgi-bin/cvename.cgi?name=CVE-2017-6923]
DESCRIPTION:Drupal could allow a remote attacker to bypass security restrictions, caused by improperly restricting access control to the Ajax endpoint. By sending a specially-crafted request, an attacker could exploit this vulnerability to access the Ajax endpoint.
CVSS Base Score: 5.3
CVSS Temporal Score: Seehttps://exchange.xforce.ibmcloud.com/vulnerabilities/130559 [https://exchange.xforce.ibmcloud.com/vulnerabilities/130559]for the current score
CVSS Environmental Score*: Undefined
CVSS Vector: (CVSS:3.0/AV:N/AC:L/PR:N/UI:N/S:U/C:N/I:L/A:N)

CVEID:CVE-2015-7875 [http://cve.mitre.org/cgi-bin/cvename.cgi?name=CVE-2015-7875]
DESCRIPTION:Ctools module for Drupal could allow a remote authenticated attacker to bypass security restrictions, caused by improper validation of the "edit" permission for the "content type" plugins. By sending a specially-crafted request, an attacker could exploit this vulnerability to bypass access restrictions.
CVSS Base Score: 4.3
CVSS Temporal Score: Seehttps://exchange.xforce.ibmcloud.com/vulnerabilities/130512 [https://exchange.xforce.ibmcloud.com/vulnerabilities/130512]for the current score
CVSS Environmental Score*: Undefined
CVSS Vector: (CVSS:3.0/AV:N/AC:L/PR:L/UI:N/S:U/C:N/I:L/A:N)
