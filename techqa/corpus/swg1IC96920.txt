APAR STATUS
 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  When exporting a large domain using the WebGUI, the WebGUI might
   hang and cause a restart.
   
   
    
   
   

LOCAL FIX
 *  Export domain using the CLI.
   
   
    
   
   

PROBLEM SUMMARY
 *  Customers might observe this issue when exporting a domain with
   an extremely large number of objects. During the export, the
   appliance might restart due to a watchdog of inactivity.
   
   
    
   
   

PROBLEM CONCLUSION
 *  Fix is available in 5.0.0.12, 6.0.0.4 and 6.0.1.1.
   
   For a list of the latest fix packs available, please see:
   http://www-01.ibm.com/support/docview.wss?uid=swg21237631 [http://www-01.ibm.com/support/docview.wss?uid=swg21237631]
