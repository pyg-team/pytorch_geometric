ERROR DESCRIPTION
 *  Here are the steps to reproduce the error of type TypeError:
   _be.channels is undefined with RQM 4.0.3
   - Create a test suite (if not already available)
   - Add one or two test cases to the test suite. Each of the test
   cases contains a manual test script.
   - Save your test suite
   - Run your test suite
   - Start the first manual test as part of the execution
   - Acknowledge few steps
   - Before the end of the test, do Stop Run
   - Do Close and Show results
   At this point you will get the error : TypeError: _be.channels
   is undefined
   Then, anytime you enter freshly to RQM from your web browser
   and attempt to view the above stopped results you will get the
   same error.

PROBLEM SUMMARY

 *  Fixed in 4.0.4.
