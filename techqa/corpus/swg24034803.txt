Title: IBM Operational Decision Manager V8.0.1.1 Fix Pack - United States

Text:
 DOWNLOADABLE FILES

ABSTRACT
 This cumulative fix pack updates WebSphere Operational Decision Management V8.0.1 to V8.0.1.1.



DOWNLOAD DESCRIPTION
This cumulative fix pack updates WebSphere Operational Decision Management V8.0.1 to V8.0.1.1. 

This fix pack is entitled. Only customers who have purchased WebSphere Operational Decision Management V8.0.1 can download this fix pack.

How to install the fix pack
If you are not installing both components, skip the instructions regarding Decision Center or Decision Server.


 1.  Download 8.0.1-WS-ODM_DC-<OS>-FP001.zip, and extract its files. (Change <OS> to match your platform: WIN32, AIX, HPUX, LINUX, OS400, SOLARIS, or ZOS.) 
 2.  Extract the archive to a new folder, named WDC. 
 3.  Download 8.0.1-WS-ODM_DS-<OS>-FP001.zip, and extract its files. (Change <OS> to match your platform: WIN32, AIX, HPUX, LINUX, OS400, SOLARIS, or ZOS.) 
 4.  Extract the archive to a new folder, named WDS. 
 5.  Launch IBM Installation Manager. 
 6.  In the toolbar menu, open File > Preferences and do the following: 
 7.  Select repositories. 
 8.  Click Add Repository. 
 9.  Browse to the extracted folder WDC\updates. 
 10. Select the file repository.config. 
 11. Click Add Repository. 
 12. Browse to the extracted folder WDS\updates. 
 13. Select the file repository.config. 
 14. Make sure both repositories are checked in the repository list Apply, and click OK to close. 
 15. Click Update to update your installation. 
 16. Follow the instructions in the installation manager to complete the installation.


Warning: 
If you have installed interim fixes for Operational Decision Manager V8.0.1, you must uninstall the fixes before you can install this fix pack. You must also stop the Operational Decision Manager sample server for this installation. If you do not uninstall the fixes and stop the server, the installation of the fix pack will fail. If you attempt to install the fix pack without uninstalling the interim fixes, the installer will uninstall the interim fixes before it fails. In this case, stop the sample server and restart the installation. PREREQUISITES
You must have Operational Decision Manager V8.0.1 installed to use this update.
