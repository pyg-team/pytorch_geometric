APAR STATUS
 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  Export Report dialog displays unsupported report formats like
   PowerPoint and PostScript in selection list for Who Is On Hold,
   Virtual Interview and other BIRT reports. When user chooses one
   of these, nothing is exported.
   
   
    
   
   

LOCAL FIX
 *  Do not use powerpoint and postscript as export options
   
   
    
   
   

PROBLEM SUMMARY
 *  Export Report dialog displays unsupported report formats like
   PowerPoint and PostScript in selection list for Who Is On Hold,
   Virtual Interview and other BIRT reports. When user chooses one
   of these, nothing is exported.
   Atlas only supports a subset (PDF , Excel) of report export
   formats, whereas the dialog displays the default list of BIRT
   supported export formats.
