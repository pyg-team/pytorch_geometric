Title: IBM User Credentials error and user being re-prompted to login in IBM Filenet Application Engine - United States

PROBLEM(ABSTRACT)
 Why are users being re-prompted to login when viewing content? 

CAUSE
There are known security issues with IE browsers and custom applications opened in an IFrame reported and security related configurations in IE browsers. Issue cannot be reproduced using other browers and specific to IE browsers only.

RESOLVING THE PROBLEM
To resolve the issue, configure the following:




 Click on the Tools button in IE browser and select Internet Options and enable as follows: 

1. Override automatic cookie handling - checked 

2. First-party Cookies - Set to Accept 

3. Third-party Cookies - Set to Accept 

4. Always allow session cookies - Checked
