ERROR DESCRIPTION
 *  BPC Explorer requires a user name in order to retrieve data from
   Business Flow Manager and Task Flow Manager.
   
   If Application Security is disabled the user is not set
   correctly in BPC Explorer so that the application cannot be
   properly initialized and BPC Explorer does not show up.
