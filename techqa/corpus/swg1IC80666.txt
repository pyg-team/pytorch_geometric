ERROR DESCRIPTION
 *  Steps to reproduce:
   1. Create a BPD.
   2. Create a Human Service with a simple coach.
   3. Add the Human Service to the created above BPD as an
   Activity.
   4. Start a new Instance of the created above BPD.
   5. From the Inspector perspective, open a task from the started
   BPD Instance. Do not submit the coach.
   6. Open the Process Portal in another window of the same browser
   and start the same task from there; sumbit the coach.
   7. Try to submit the coach started from the Inspector
   perspective.
   
   ===> The error message appears that the session is expired. The
   timeout error is incorrect in this case.
   
   Expected Result: The error message should appear saying that the
   task has been already closed.
   
   Note 1: The issue is reproduced on 7.1 and 7.5 as well.
   Note 2: The issue is not reproduced when opening the coaches
   from different instances.
