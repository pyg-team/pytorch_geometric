This fix upgrades the embedded WebSphere Application Server (eWAS) and IHS which is shipped as part of the IBM Tivoli Monitoring portal server, to version 8.0.00.14 plus additional Interim Fixes referred to as Interim Fix Block 1.

PI87445: Multiple vulnerabilities in IBM HTTP Server
