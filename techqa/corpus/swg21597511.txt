Title: IBM DataPower WebGUI supports Internet Explorer 9  in Compatibility View only - United States

Text:
 TECHNOTE (TROUBLESHOOTING)

PROBLEM(ABSTRACT)
 When using Internet Explorer 9 to access the DataPower WebGUI, if the Browser Mode setting is not configured to use Internet Explorer 9 Compatibility View, the log on might fail. 

RESOLVING THE PROBLEM
When using Internet Explorer 9 to access the DataPower WebGUI, make sure that the Browser Mode setting is set to 'Internet Explorer 9 Compatibility View'. 

To verify the current Browser Mode: 

 1. Go to the DataPower WebGUI log on page 
 2. Select the browser Tools menu. If there is a check mark beside Compatibility View, the WebGUI is using the correct Browser Mode.


To ensure that the proper mode is always used when accessing the WebGUI:  1. Select Tools > Compatibility View Settings 
 2. In the Add this website section, enter the WebGUI URL. 
 3. Click Add. 
 4. Close the dialog. 
 5. Restart Internet Explorer, and access the WebGUI.
