Title: IBM How do I modify the Cam_Passport cookie path. - United States

Text:
 TECHNOTE (FAQ)

QUESTION
 Is it possible to modify the cookie path for the CAM_passport cookie so that it can be available to all apps running on the same server? 

ANSWER


By default the cookie path is the root path to ReportNet or Cognos 8. 

This value can be me modified in Cognos Configuration using the steps below.

Steps:1. Open Cognos Configuration. 2. Click Actions--> Edit Global Configuration

3. Modify the cookie settings:

 * in IBM Cognos ReportNet, click the Cookie Settings tab.
 * in Cognos 8, click the Server tab.



4. Modify the path and other information as required.

5. Click OK.

Stop and restart the Service.
