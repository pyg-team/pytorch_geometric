Title: IBM Security Bulletin: A security vulnerabilities has been identified in Websphere Application Server shipped with Jazz for Service Management (CVE-2016-0306) - United States

Text:
 SECURITY BULLETIN

SUMMARY
 Websphere Application Server is shipped as a component of Jazz for Service Management. Information about a security vulnerability affecting Websphere Application Server has been published in a security bulletin. 

VULNERABILITY DETAILS
Please refer the WAS security bulletin Security Bulletin: Potential security vulnerability in IBM WebSphere Application Server if FIPS 140-2 is enabled (CVE-2016-0306) [http://www-01.ibm.com/support/docview.wss?uid=swg21979231]

AFFECTED PRODUCTS AND VERSIONS
 

Principal Product and Version(s) Affected Supporting Product and Version Jazz for Service Management version 1.1, 1.1.1, 1.1.2 and fixpacks Websphere Application Server 8.5.5 Full Profile REMEDIATION/FIXES
Please refer to WAS ifix for fixing the vulnerability 

Note: WAS needs 8.5.5.7 as the minimal level for fixing the vulnerability. JazzSM 1.1.2.1 is certified with WAS 8.5.5.8

WORKAROUNDS AND MITIGATIONS
None
