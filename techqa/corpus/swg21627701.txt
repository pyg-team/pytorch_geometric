Title: IBM Failure in unconfiguring Registry Services Web Application or Database error - United States

Text:
jazzsm11relnotes TECHNOTE (TROUBLESHOOTING)

PROBLEM(ABSTRACT)
 When you try to install Jazz for Service Management, the installation fails with the following errors:
CTGFI3004E: Failed in unconfiguring Registry Services Web Application.
CTGFI3006E: Failed in unconfiguring Registry Services Database. 

CAUSE
The error message is displayed when there is not enough space on the disk. When you perform Simple Installation, the launchpad runs the convenience script for a simple installation and displays the overall result in the Prerequisite Scan panel. However, when you perform Custom installation, you must manually run the convenience script for custom installation and ensure that your environment meets the hardware and software prerequisites for installing Jazz for Service Management.

RESOLVING THE PROBLEM
You must have sufficient disk space to perform Jazz for Service Management Installation. Free up the disk space and then perform the installation.
