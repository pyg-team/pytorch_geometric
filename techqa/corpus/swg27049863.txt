 DETAILED SYSTEM REQUIREMENTS

ABSTRACT
 This document describes the system requirements for IBM InfoSphere Data Replication for DB2 for z/OS V11.4. 

CONTENT


PRODUCT DESCRIPTION
IBM InfoSphere Data Replication for DB2 for z/OS consist of three technologies:

 * CDC Replication
 * Q Replication
 * SQL Replication

For more information about IBM InfoSphere Data Replication for DB2 for z/OS, see the product overview [http://www-03.ibm.com/software/products/en/infosphere-data-replication-db2-z].
