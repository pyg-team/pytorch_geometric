ERROR DESCRIPTION
 *  Drafting (GenDraft) / Font size not taken from
   Standards
   .
   Scenario:
   1 - Open CATIA in admin mode
   2 - Navigate to
   Tools / Standards / Drafting / ISO / Styles /
   Length/Distance
   Default / Font
   3 - Change BOLD = Yes, Underline = Yes, Color =
   Green and Size = 14
   4 - File / New / Part and create a simple pad
   5 - File / New / Drawing and select ISO format.
   6 - Take a Front View of the pad
   7 - Select Dimensions and take a measurement
   from the front-view.
   Result is OK ,e.g. Green, underlined, bold text at
   14mm size.
   8 - Tools / Options / General / Parameters and
   Measure / Units
   and change Length to measure in Inches (inch)
   9 -Redo steps 4-7
   .
   Problem:Result is KO for Font Size
   .
   Expected result:Result should be OK for Font Size.
   .
   .
