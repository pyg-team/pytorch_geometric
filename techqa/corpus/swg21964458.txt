Title: IBM Unable to Add Document Using Entry Template in IBM Content Navigator - United States

Text:
ICN Content Navigator Entry Templates AddOn Add-on FNRCR0081E CIWEB1016 TECHNOTE (TROUBLESHOOTING)

PROBLEM(ABSTRACT)
 Users receive a CIWEB1016 error when trying to add a document using an Entry Template in IBM Content Navigator (ICN). 

SYMPTOM
 

 * Users receive a CIWEB1016 error when trying to add a document using an Entry Template in IBM Content Navigator. 
 * In the SystemOut.log an exception will be recorded:
   
   com.filenet.api.exception.EngineRuntimeException: FNRCR0081E:
   RETRIEVE_PROPERTY_NOT_DEFINED: The property
   EntryTemplateObjectStoreName is not defined. 
   
   See this link [http://www-01.ibm.com/support/knowledgecenter/SSEUEX_2.0.2/com.ibm.installingeuc.doc/eucts003.dita?lang=en] for more information on SystemOut.log for ICN.


CAUSE
Workplace Template Extensions add-on needs to be installed.

DIAGNOSING THE PROBLEM
 

Parse the SystemOut.log for ICN for the following exception: 


com.filenet.api.exception.EngineRuntimeException: FNRCR0081E:
RETRIEVE_PROPERTY_NOT_DEFINED: The property
EntryTemplateObjectStoreName is not defined.


RESOLVING THE PROBLEM
Install Workplace Templates Extensions add-on and the Workplace XT add-on in FileNet. For more details on how to install the add-ons refer to Connecting and configuring IBM FileNet Content Manager repositories [http://www-01.ibm.com/support/knowledgecenter/SSEUEX_2.0.3/com.ibm.installingeuc.doc/eucco015.htm?lang=en].
