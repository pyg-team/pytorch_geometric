ERROR DESCRIPTION
 *  SCENARIO:
   1.In Tools/Options/General/General, be sure to check "Enable
     drag and drop for cut/copy and paste"
   2.Open  the product containing a part with external parameters
   3.Be sure your product is activated. If not, double-click on it.
   4.Try to delete external parameter, it works (in given data,
     delete String.1 in Part2/External parameters) -> No problem
   5.Do a selection box (click on right mouse button, hold it,
     and drag it to create a square, then release it). No need to
     select anything
   6.Try to delete external parameter (in given data, delete
     String.1 in Part2/External parameters)
   
   RESULT: Impossible to delete external parameter. Your session
   is now corrupted.
   It is now impossible to delete any external parameter. You can
   open another product OR part with external parameter(s),
   impossible to delete them.
   
   Work around: Need to restart your session
   EXPECTED RESULT: The external parameter should be deleted
