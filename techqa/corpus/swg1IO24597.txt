ERROR DESCRIPTION
 *  *
   Open PMR in ICSW (assumes ICSW is open and PMR is not archived)
   
   Problem Description/Reason for Collaboration:
   Datacap Navigator can't handle the multiple periods in the
   filename.
   The code is probably searching for the first period and then
   assuming what is after is the extension.
   The file is BRS US CDZ EDM Interface with NFE DL v1.04.doc and
   after Navigator Scan, it becomes
   tm000001.04
   
   Environment:
   Datacap 9.0.1 fix pack 1
   
   Actions Taken/Things Tried:
   L2 reproduced the issue inhouse using client sample image
   
   Provided information/documents:
   Sample batch folder
   Client sample image
   
   Provided Logs/Time and Errors observed in Logs:
   No error
