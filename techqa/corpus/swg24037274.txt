Problems fixed APAR Summary IV58161
PMR 78419,017,649

Scan returns FAIL results for prerequisite properties incorrectly added to a set of configuration files in Fix Pack 10


Download iFix You can download the interim fix from Fix Central.

 1.  In the target environment that has Prerequisite Scanner, create a temporary directory for interim fixes, referred to as the PRS_iFixes_Home directory. 
     
     
 2.  Download the interim fix compressed file, 1.2.0.10-Tivoli-PRS-Windows-IF0001, from IBM Fix Central as follows: 
 3.  Open your web browser. Ensure that you sign into IBM.com or IBM Support Portal; enter the following URL:
     Fix Central landing page [http://www.ibm.com/support/fixcentral/] (http://www.ibm.com/support/fixcentral/) 
 4.  In the Product selector field, start typing Prerequisite Scanner; then select Prerequisite Scanner from the returned result list.
 5.  Verify that 1.2.0.10 is selected in the Installed Version list.
 6.  From the Platform list, select the platform on which you want to install the interim fix.
 7.  Click Continue. The Identify Fixes page opens.
 8.  Use the default option, Browse for fixes; click Continue.
 9.  On the Select fixes page, select the 1.2.0.10-Tivoli-PRS-Windows-IF0001 compressed file; click Continue.Note: If you have not yet signed into IBM.com, you are now prompted to sign in or register.
     
     
 10. On the Download option page, select one of the following download options; click Continue.  * Download using Download Director
      * Download using your browser
     
     
 11. Review the terms and conditions, and click I agree.
 12. Follow the online instructions for your chosen download option, and download the interim fix.
 13. Extract the compressed file contents to the PRS_iFixes_Home/1.2.0.10-Tivoli-PRS-Windows-IF0001 directory. Note: On Windows systems, the extraction process can create the 1.2.0.10-Tivoli-PRS-Windows-IF0001 directory.

Installation instructions  * 
 *  - [http://www.ibm.com/i/c.gif] BEFORE INSTALLATION
   Make a back up of the following configuration file in the ips_root\Windows directory, where ips_root refers to the path to the root directory of Prerequisite Scanner:
   
   BSM_06010100.cfg
   
   Move the backed up file to another directory that is not in the ips_root directory.
   
   
   
 * 
 *  - [http://www.ibm.com/i/c.gif] INSTALLING THE IFIX
   Copy the following configuration file from the PRS_iFixes_Home/1.2.0.10-Tivoli-PRS-Windows-IF0001/Windows to the ips_root\Windows directory:
   
   BSM_06010100.cfg
   
   
   You can then run Prerequisite Scanner to check your target environment for the hardware and software prerequisites of the offering being installed.
   
   
 * 
 *  - [http://www.ibm.com/i/c.gif] UNINSTALLING THE IFIX
   You can uninstall the interim fix at any time. Overwrite the configuration file in the ips_root\Windows directory with the file that you backed up, as outlined in the Before installation section.
