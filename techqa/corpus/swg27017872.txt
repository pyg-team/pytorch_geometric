Title: IBM ECM Widgets 4.5.2 Release Notes - United States

Text:
 RELEASE NOTES

ABSTRACT
The IBM Enterprise Content Management Wigets (IBM ECM Widgets) Version 4.5.2 Release Notes include links to the IBM ECM Widgets documentation in the IBM FileNet P8 Information Center, information about new features, and a list of known issues.

CONTENT
 

IBM ECM Widgets is a collection of portable, reusable components, also called widgets, that you can use to create user interfaces for Enterprise Content Management (ECM) applications and business process management (BPM) applications. 


WHAT'S NEW
The features and changes that were introduced in the this release of IBM ECM Widgets include the following capabilities and features:

 * New widgets:  * Process History widget
    * IBM WEBi widget
   
   
 * Support for IBM Content Manager through the Attachment widget, the Viewer widget, and the IBM WEBi widget
 * Enhancements to existing widgets to enable partner/customer integration
 * User interface improvements
 * Support for AIX and Oracle
 * Arabic Language Support
 * Improved Configuration
 * Improved Logging and Tracing
 * Support for Business Space Powered by WebSphere 7.0
