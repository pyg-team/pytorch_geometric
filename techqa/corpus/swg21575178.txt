Title: IBM Rational 소프트웨어 전달 플랫폼 - 데스크탑 제품에서 임시키를 사용중 영구키를 활성화 하는 방법 - United States

Text:
RSA; RSM; RSD; rda; rpt; rft; rmt; rdbe; rad; A. 영구; 임시; 실시간; 라이센스; 키; 임시; 라이센스; 평가; GA; 버전; 1254371; SDP; RMC; 제품 활성화 킷 가져오기; 라이센스 그렇지만 부동 구성 TECHNOTE (FAQ)

질문
 [Korean] Rational Software Architect 와 같은 소프트웨어 전달 플랫폼의 데스크탑 제품의 평가판을 설치후 시험판을 영구 라이센스가 적용된 제품으로 변경하려면 어떻게 합니까? 

응답
다음 지시사항을 따라서 활성화 킷을 설치하거나 Floating 라이센스 키를 사용하여 주십시오. 제품 활성화 킷(Activation kit) 에는 평가판 Rational 제품을 전환할수있는 영구 라이센스 키가 포함되어 있습니다. 제품 활성화 킷을 다운로드한 다음 IBM Installation Manager를 사용하여 다운로드 위치에서 해당 킷을 적용하여야 합니다. 

참고: Rational Method Composer 제품은 활성화 킷을 , 라이센스 키 그리고 Installation Manager 를 버젼 7.2 와 7.5부터 지원하고있습니다 하지만 플로팅 라이센스키는 지원하지 않습니다. 

활성화 킷을 적용하여 새 라이센스 키 를 사용하는 방법 

1. IBM Installation Manager를 시작하십시오. 

2. 기본 페이지에서 라이센스 관리(Manage Licenses) 를 클릭하십시오. 

3. 패키지 설치 페이지에서 패키지를 선택하고 제품 사용 킷 가져오기를 클릭하십시오. 

4. 다음(Next) 을 누르십시오. 

선택한 패키지의 세부사항인 라이센스 유형 및 라이센스가 적용되는 버전 범위가 표기됩니다. 

5. 활성화 킷의 다운로드 위치 경로를 찾은 후 해당 Java 아카이브 (JAR) 파일을 선택하고 열기(Open)를 클릭하십시오. 

6. 다음(Next) 을 누르십시오. 요약 페이지에서 활성화 킷의 대상 설치 디렉토리, 새 라이센스가 적용된 제품 및 버전 정보를 표시합니다. 

7. 완료(Finish) 를 누르십시오. 






Floating 라이센스 키 를 적용하는 방법 

1. IBM Installation Manager를 시작하십시오. 

2. 기본 페이지에서 라이센스 관리(Manage licenses) 를 클릭하십시오. 

3. 패키지 설치 페이지에서, 패키지 번호를 선택한 다음 라이센스 서버 구성(Configure License servers) 을 클릭하고 다음(Next) 을 클릭하십시오. 

4. 라이센스 서버 페이지에서 Rational Common Licensing 사용 선택란(Enable Rational Common Licensing) 을 선택하십시오. 

5. 하나 이상의 라이센스 서버 연결을 구성하십시오. 

 a. 추가 단추(Add) 를 클릭합니다. 

 b. 서버 추가 창에서, 이름 필드( Name) 에 라이센스 서버의 호스트 이름을 입력하십시오. 

 c. (선택사항) 포트 필드(Port) 에서, 방화벽이 사용되는 환경의 값을 입력하십시오. 관리자의 지시가 없으면 이 포트에 값을 지정하지 마십시오. 

 d. 확인 (OK) 을 누르십시오. 

 e. 시스템 관리자가 추가 라이센스 서버의 호스트 이름을 제공한 경우, 이 단계를 반복하여 각 서 버를 추가하십시오. 

백업 라이센스 서버(Redundant License server) 구성 (세 개의 서버가 사용되는 경우), 각 호스트 이름 형식: hostname1, hostname2, hostname3 을 입력하여주십시오 또한 각각의 포트 번호를 입력하여주십시오. : host1port#, host2port#, host3port#. 

6. 다음(Next) 을 누르십시오. 

7. (선택사항) 사용자 정의 패키지의 라이센스 사용 순서를 구성하십시오. 목록의 라이센스 순서가 제품이 특정 라이센스의 라이센스 키를 가져오는 순서를 결정합니다. 

8. 완료(Finish) 를 누르십시오. 

참고: 라이센스 관리 마법사가 Floating 라이센스 구성의 성공 여부를 표시합니다. 다음 번에 제품을 열면, 라이센스 서버에 대한 연결이 작성되어 사용 가능한 Floating 라이센스 키의 풀에서 라이센스 키를 얻습니다. 

최신 버전의 

Floating 라이센스 키 사용 

1. Installation Manager에서 라이센스 관리(Manage Licenses) 를 선택하신후 , 라이센스를 적용 하시고자 하는 제품을 선택하여 주십시오. 

2. Floating 라이센스 지원 구성( Configure Floating license support) 을 선택하고 다음을 클릭하십시오. 

3. Floating 라이센스 적용 (Enable Floating license enforcement) 을 사용 가능하게 하고+ 단추를 클릭하십시오 

4. 서버 유형(Server Type) > 추가 이름(Add name) > 포트 추가 (Add port) 를 선택하여 주십시오. 

5. 연결 테스트> 확인을 클릭하십시오. 

6. 다음> 완료를 클릭하십시오. 

활성화 킷을 적용하여 새로운 라이센스를 활성화 하는 방법 

1. IBM Installation Manager를 실행하십시오. 

2. 라이센스 관리 버튼을 클릭하여 주십시오 ( Manage Licenses) 

3. 활성화 키트를 적용하시려는 제품을 하이라이트 하여 주십시오. 

4. 활성화 키트 불러오기 ( Import Product activation kit) 버튼을 눌러주십시오. 

5. 다음을 클릭하십시오. 

6. .jar 파일의 경로를 검색한후 다음(Next) 을 클릭하여주십시오. 

7. 라이센스 계약을 읽고 승인한 후 완료(Finish) 를 클릭하십시오. 

8. 본 라이센스 적용이 성공했다는 메시지가 표시됩니다. 완료를 누르십시오. 

9. 이제 다시 설치 관리자의 기본 페이지가 표시됩니다. . 

10. 라이센스 관리 버튼을 클릭하여 주십시오 ( Manage Licenses) 

11. 활성화 키트를 적용시킨 제품을 선택하여 주십시오. 

12. 플로팅 라이센스 지원 구성 ( Configure Floating license support) 버튼을 클릭하여 주십시오 

13. 다음(Next) 을 누르십시오. 

14. 플로팅 라이센스 비활성화 ( Disable Floating License) 를 클릭하여 주십시오. 

15. 완료를 클릭하십시오.
