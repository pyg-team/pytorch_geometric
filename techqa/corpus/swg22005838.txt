Title: IBM Can I use more than one network interface card (NIC) in the same computer with IBM FileNet Application Engine, IBM FileNet Workplace XT or IBM Content Navigator. - United States

Text:
 TECHNOTE (FAQ)

QUESTION
 Can I use more than one network interface card (NIC) in the same computer with IBM FileNet Application Engine, IBM FileNet Workplace XT or IBM Content Navigator. 

ANSWER
Configuring multiple ethernet adapters (multi-NIC) is not supported for IBM FileNet Application Engine, Workplace, Workplace XT and IBM Content Navigator because it may cause a network connectivity problem.
