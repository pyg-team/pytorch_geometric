Title: IBM How do I obtain the Content Manager OnDemand for i (5770-RD1) product software for Version 7.3? - United States

Text:
 FLASH (ALERT)

ABSTRACT
 IBMÂ® Content Manager OnDemand for i, product number 5770-RD1, is not sent on the stacked media (CDs) that you receive with operating system upgrades but rather must be downloaded from the Passport Advantage web site. You must be Passport Advantage-entitled (with your software subscription) to the 5770-RD1 product before you can download it. 

CONTENT
You should verify with your IBM marketing representative or IBM business partner that you are Passport Advantage-entitled. If you are, you can go to the Passport Advantage Web site and download the Content Manager OnDemand for i software. 

To download the software: 

1. Go to http://www.ibm.com/software/howtobuy/passportadvantage [http://www.ibm.com/software/howtobuy/passportadvantage] and click the tab labeled Passport Advantage Online. That will bring you to http://www.ibm.com/software/howtobuy/passportadvantage/pao_customers.htm [http://www.ibm.com/software/howtobuy/passportadvantage/pao_customers.htm], where you can click Customer sign-in.

2. Enter your registration ID or site number. If you do not know what this is or you do not have a registration ID, contact the Passport Advantage group at 1-800-978-2246. Their hours of operation are Monday through Friday, 8 a.m. to 8 p.m. eastern time (EDT (summer) or EST (winter)), excluding U.S. legal holidays. For customers outside the U.S. or Canada, see https://www.ibm.com/software/howtobuy/passportadvantage/paocustomer/docs/en_US/ecare.html [https://www-112.ibm.com/software/howtobuy/passportadvantage/paocustomer/docs/en_US/ecare.html] . If, on the other hand, you need your password reset or are having issues logging on to Passport Advantage Online, dial 888-426-4409.

3. Click on Software download & media access, then Continue to proceed to the "Download Software" tab. All entitled downloads should be listed. Select IBM Content Manager OnDemand for i. On the next panel, select "All operating systems" (not "IBM i" as you might think) and specify your language, then GO, Confirm that the Version says 7.3.0 and that there is a checkmark next to "Required: 23 files". Click on the plus sign next to "Select individual files" and look for IBM Content Manager OnDemand for i V7.3 IBM i Multilingual eAssembly (CRZ69ML). Expand the Twistie and place a checkmark next to IBM Content Manager OnDemand for i V7.3 IBM i Multilingual (CN9K9ML). This contains the base Server code (*BASE of 5770-RD1), known as Common Server, and also contains OnDemand Web Enablement Kit (ODWEK). Then click on the Download button. On the screen that follows, you will select a Download location and Download method. Once you agree to the license agreement's terms and conditions, the files will begin downloading. 

4. The CN9K9ML option will download a CN9K9ML.zip file, which, depending on the download method selected, might get placed in the C:\DownloadDirector directory. The zip file extracts into two files. These are a) the RD1 code in a file named BASE and b) a CN9K9ML.pdf file that has the instructions for uploading this code to the IBM i server and loading the Licensed Program Product, 5770-RD1. This PDF instruction file is also provided below. 

CN9K9ML.pdf [/support/docview.wss?uid=swg21976946&aid=2]CN9K9ML.pdf [/support/docview.wss?uid=swg21976946&aid=1]

5. Follow the instructions inside the CN9K9ML document to upload and install the product on the IBM i.

Note: You might see a choice for "Optional files" listed on step 3, underneath the "Required files" checkbox. This box should be checked if you are interested in downloading IBM Content Navigator or WebSphere Application Server supplements

6. If you are downloading the PDF Indexer, then click on the radio button "IBM Content Manager OnDemand for i PDF Indexer" while on the "Find downloads & media" screen as in step 3 above. If you click on the plus sign to expand the the eAssembly named IBM Content Manager OnDemand for i PDF Indexer V7.3 IBM i Multilingual eAssembly (CRZ6EML) you will see IBM Content Manager OnDemand for i PDF Indexer V7.3 IBM i Multilingual (CN9KAML), which contains the PDF Indexer code (Option 12 of 5770-RD1). Select the check boxes next to the items you want and then click I agree and Download now at the bottom of the page. 

7. The code might get placed in c:\DownloadDirector, unless you are able to override this. The CN9KAMLzip extracts into two files. These are a) the code itself in a file named PDF and b) a CN9KAML.pdf file that has the instructions for uploading the code to the IBM i server and loading this option of the Licensed Program Product.
