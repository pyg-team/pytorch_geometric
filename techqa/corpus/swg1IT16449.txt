APAR STATUS
 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  Dynamic multistep action leak may occur in domains having SSH
   Server Front side handler (such as SFTP Server FSH, SFTP Poller
   FSH) with AAA policy configured.
   
   
    
   
   

LOCAL FIX

PROBLEM SUMMARY
 *  Affected is the use of AAA with SSH server Front side handler.
   
   Memory might be leaked.
   
   
    
   
   

PROBLEM CONCLUSION
 *  The fix is available in 7.0.0.16, 7.1.0.13, 7.2.0.10, 7.5.0.4,
   7.5.1.3 and 7.5.2.1
