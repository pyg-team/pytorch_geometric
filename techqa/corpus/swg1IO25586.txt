APAR STATUS
 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  Unable to view MS Office documents ingested as
   'application/octate-stream' after upgrading ICN to V2.0.3.8.
   When downloading, ICN associates these documents with .bin.  The
   same documents can be opened using ICN V2.0.3.6
   
   
    
   
   

LOCAL FIX

PROBLEM SUMMARY
 *  When a user downloads a document as original, the file is
   downloaded with a .bin or .dat file extension or without the
   expected extension.
   
   
    
   
   

PROBLEM CONCLUSION
 *  This has been addressed in IBM Content Navigator V2.0.3.8
   LA013, V3.0 LA010, V3.0.1 LA003, and V3.0.2.
   
   
    
   
   

TEMPORARY FIX
