Title: IBM (参考) IBM WebSphere Portal に同梱されている IBM WebSphere Application Server で確認された複数の脆弱性について - United States

Text:
 SECURITY BULLETIN

SUMMARY
 IBM WebSphere Portal のコンポーネントとして同梱されている IBM WebSphere Application Server (WAS) に対して、複数の脆弱性に関する問題が報告されました。 

VULNERABILITY DETAILS

脆弱性の詳細に関しては、IBM WebSphere Application Server のセキュリティー情報をご参照ください。

(英文)「Security Bulletin: Multiple vulnerabilities in IBMR Java SDK affect WebSphere Application Server January 2016 CPU (CVE-2016-0475, CVE-2016-0466, CVE-2015-7575, CVE-2016-0448)」(Technote #1975424) [http://www-01.ibm.com/support/docview.wss?uid=swg21975424]
(英文)「Security Bulletin: Cross-site scripting vulnerability in IBM WebSphere Application Server (CVE-2015-7417)」(Technote #1974520) [http://www-01.ibm.com/support/docview.wss?uid=swg21974520]
(英文)「Security Bulletin: Multiple vulnerabilities in IBMR Java SDK affect WebSphere Application Server October 2015 CPU (CVE-2015-4872, CVE-2015-4734, CVE-2015-5006)」(Technote #1969620) [http://www-01.ibm.com/support/docview.wss?uid=swg21969620]
(英文)「Security Bulletin: Multiple Security Vulerabilities fixed in IBM WebSphere Application Server 8.5.5.8」(Technote #1972261) [http://www-01.ibm.com/support/docview.wss?uid=swg21972261]

(英文)「Security Bulletin: Apache HTTPComponents vulnerabilities in WebSphere Application Server (CVE-2012-6153, CVE-2014-3577)」(Technote #1969251) [http://www-01.ibm.com/support/docview.wss?uid=swg21969251]

AFFECTED PRODUCTS AND VERSIONS
 

対象となる IBM WebSphere Portal のバージョン 脆弱性の影響を受ける WAS のバージョン IBM WebSphere Portal version 6.1, 6.1.5, 7.0, 8.0 および 8.5 IBM WebSphere Application Server version 6.1, 7.0, 8.0 および 8.5 GET NOTIFIED ABOUT FUTURE SECURITY BULLETINS
