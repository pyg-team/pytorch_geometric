ERROR DESCRIPTION
 *  SHAPEDSGN - MEASURE BETWEEN TAKES 0MM VALUE IS OFFSET VALUE
   IS CHANGED EVEN IF AUTOMATIC UPDATE IN PART IS SELECTED
   .
   SCENARIO:
   1. Open CATPart (Any workbench)
   2. Activate "Automatic update in part" of "Update" section
    (Tools/Options --> General/Parameters and Measure/"Measure
   Tools" tab)
   3. Change offset value
    --> geometry is changed
   ===> Value of "measure between" changes into "0mm" and moves
   to origin  point (THIS IS PROBLEM)
   .
