APAR STATUS
 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  Because IBM Process Federation Server does not index some tasks
   from BPEL processes running on federated IBM Business Process
   Manager (BPM) systems, you cannot see all BPEL tasks if IBM
   Process Portal is configured for a federated IBM Business
   Process Manager (BPM) environment.
   
   PRODUCTS AFFECTED
   IBM BPM Advanced
   IBM BPM Standard
   
   
    
   
   

LOCAL FIX

PROBLEM SUMMARY
 *  No additional information is available.
   
   
    
   
   

PROBLEM CONCLUSION
 *  A fix is available for IBM Business Process Manager - Process
   Federation Server V8.5.7.0 cumulative fix 2016.07 (8.5.7.0
   CF201607) that indexes the previously skipped BPEL process
   tasks. For download and installation instructions, see "IBM
   Business Process Manager - Process Federation Server 8.5.7.0
   Cumulative Fix 2016.07 (8.5.7.0 CF2016.07)" at
   http://www.ibm.com/support/docview.wss?uid=swg24042498. [http://www.ibm.com/support/docview.wss?uid=swg24042498.]
