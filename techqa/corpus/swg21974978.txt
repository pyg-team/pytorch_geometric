QUESTION
 When will the IBM Security License Key and Download Center website begin using a SHA-2 certificate? 

CAUSE
This change is part of an industry-wide migration from SHA-1 to SHA-2 to increase authentication security. 
Please see the following resources for further information: 

Chromium Blog - Gradually Sunsetting SHA-1 

http://blog.chromium.org/2014/09/gradually-sunsetting-sha-1.html [http://blog.chromium.org/2014/09/gradually-sunsetting-sha-1.html] 

Microsoft Security Advisory 2880823 - Deprecation of SHA-1 Hashing Algorithm for Microsoft Root Certificate Program 

https://technet.microsoft.com/en-us/library/security/2880823.aspx [https://technet.microsoft.com/en-us/library/security/2880823.aspx] 

Mozilla Security Blog - Phasing Out Certificates with SHA-1 based Signature Algorithms 

https://blog.mozilla.org/security/2014/09/23/phasing-out-certificates-with-sha-1-based-signature-algorithms/ [https://blog.mozilla.org/security/2014/09/23/phasing-out-certificates-with-sha-1-based-signature-algorithms/]



ANSWER
The IBM Security License Key and Download Center website, which is located at https://ibmss.flexnetoperations.com [https://ibmss.flexnetoperations.com], will begin using a SHA-2 certificate on April 7, 2016.
