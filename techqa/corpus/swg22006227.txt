Title: IBM How to locate the installation directory of an IBM SPSS desktop product on Windows operating systems. - United States

Text:
 TECHNOTE (FAQ)

QUESTION
 How do I locate the installation directory of an IBM SPSS desktop product installed on a Windows operating system? 

ANSWER
The following instructions refer to the default installation directory for IBM SPSS products.
IF the product is not installed to the default directory THEN adjust the instructions to account for the non-default installation location.

In the following instructions:


 1. Replace <product name> with the actual name of the product (example: Statistics). 
 2. Replace <product version> with the actual version number of the product (example: 24).


IBM SPSS Modeler 

The default installation directory of IBM SPSS Modeler desktop products depends upon the bit-version of the operating system (32-bit/64-bit) and the bit-version of the product (32-bit/64-bit). The default installation directory is either 

\Program Files\IBM\SPSS\Modeler\ <product version>\bin\ 

OR 

\Program Files (x86)\IBM\SPSS\Modeler\ <product version>\bin\ 

All other IBM SPSS products including IBM SPSS Statistics 

The default installation directory of IBM SPSS desktop products depends upon the bit-version of the operating system (32-bit/64-bit) and the bit-version of the product (32-bit/64-bit). The default installation directory is either: 

\Program Files\IBM\SPSS\ <product name>\ <product version>\ 

OR 

\Program Files (x86)\IBM\SPSS\ <product name>\ <product version>\
