APAR STATUS
 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  For Example following are the  few  error messages does not show
   the host or channel information in the error.
   
   AMQ9259: Connection timed out from host ''.
   AMQ9642: No SSL or TLS certificate for channel ''.
   AMQ9716: Remote SSL certificate revocation status check failed
    for channel ''.
   
   
    
   
   

LOCAL FIX

PROBLEM SUMMARY
 *  ****************************************************************
   USERS AFFECTED:
   This issue affects users of WebSphere MQ who are using the ping
   channel command to evaluate the status of an MQ channel.
   
   
   Platforms affected:
   MultiPlatform
   
   ****************************************************************
   PROBLEM DESCRIPTION:
    The appropriate values were not inserted into the message prior
   to it being displayed on the runmqsc console output.
   
   
    
   
   

PROBLEM CONCLUSION
 *  Code has been modified so that the channel information will
   populate in the error messages.
