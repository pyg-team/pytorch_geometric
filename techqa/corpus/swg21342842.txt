Title: IBM Test connection fails: RSV-SRV-0056 The Report Server process has not been properly initialized. - United States

Text:
 TECHNOTE (TROUBLESHOOTING)

PROBLEM(ABSTRACT)
 All data sources fails when testing connection. 

SYMPTOM
RSV-SRV-0056 The Report Server process has not been properly initialized. Please contact your Administrator 

Additional errors in cogserver.log: 

CCL-SRV-0501 The server process caught an unexpected exception.
UT-DEF-0032 A file system error has occurred File exists.

CAUSE
Cognos 8 service was started with an domain account which didn't have local administrator rights.

RESOLVING THE PROBLEM
Add the domain account to local Administrators group.
