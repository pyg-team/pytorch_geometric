Title: IBM Security Bulletin: Multiple vulnerabilities in IBM Java Runtime affect IBM Tivoli Access Manager for e-business (CVE-2014-4263, CVE-2014-4244) - United States

Text:
PSIRT SECURITY BULLETIN

SUMMARY
 There are multiple vulnerabilities in IBM® Runtime Java™ Technology Edition, various versions of which are used by affect IBM Tivoli Access Manager for e-business. These issues were disclosed as part of the IBM Java SDK updates in July 2014. 

VULNERABILITY DETAILS

CVEID: CVE-2014-4263 [http://cve.mitre.org/cgi-bin/cvename.cgi?name=CVE-2014-4263]

DESCRIPTION: An unspecified vulnerability related to the Security component has partial confidentiality impact, partial integrity impact, and no availability impact.

CVSS:
CVSS Base Score: 4
CVSS Temporal Score: See https://exchange.xforce.ibmcloud.com/vulnerabilities/94606 [https://exchange.xforce.ibmcloud.com/vulnerabilities/94606] for the current score
CVSS Environmental Score*: Undefined
CVSS Vector: (AV:N/AC:H/Au:N/C:P/I:P/A:N)



CVEID: CVE-2014-4244 [http://cve.mitre.org/cgi-bin/cvename.cgi?name=CVE-2014-4244]

DESCRIPTION: An unspecified vulnerability related to the Security component has partial confidentiality impact, partial integrity impact, and no availability impact.

CVSS:
CVSS Base Score: 4
CVSS Temporal Score: See https://exchange.xforce.ibmcloud.com/vulnerabilities/94605 [https://exchange.xforce.ibmcloud.com/vulnerabilities/94605] for the current score
CVSS Environmental Score*: Undefined
CVSS Vector: (AV:N/AC:H/Au:N/C:P/I:P/A:N)

AFFECTED PRODUCTS AND VERSIONS
IBM Tivoli Access Manager for e-business versions 5.1, 6.0, 6.1, and 6.1.1

REMEDIATION/FIXES
IBM Tivoli Access Manager for e-business 5.1, 6.0, 6.1, 6.1.1 

IBM Tivoli Access Manager for e-business (TAMeb) software is not directly affected. However, IBM WebSphere is shipped as a component of IBM Tivoli Access Manager for eBusiness software. Information about a security vulnerability affecting IBM WebSphere has been published in a separate security bulletin. Please consult the security bulletin "Security Bulletin: Multiple vulnerabilities in IBM Java SDK affect WebSphere Application Server July 2014 CPU" [http://www-01.ibm.com/support/docview.wss?uid=swg21680418] for vulnerability and patch details.
