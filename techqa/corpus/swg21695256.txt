Title: IBM Error 'Map does not exist' encountered when map caching enabled. - United States

Text:
WTX WMB Cache exist TECHNOTE (TROUBLESHOOTING)

PROBLEM(ABSTRACT)
 Error 'Map does not exist' encountered when map caching enabled. 

SYMPTOM
Error 'Map does not exist' encountered when map caching enabled.


CAUSE
The IBM WebSphere Transformation Extender (WTX) Map was deleted.

ENVIRONMENT
Using WTX with IBM WebSphere Integration Bus (IIB) to run a WTX map using the WTX Map node in the IIB Toolkit with Map caching enabled.

DIAGNOSING THE PROBLEM
The following error is encountered.

[8]DTXMB1003E WTX for WMB error: Map status for /PATH/MAP.MMC, Map does not exist (3). Confirm map was deployed correctly. {1}: BR_ESB02D. 04c86902-4b01-0000-0080-e6b2751517b6: TXInterface.cpp: 806: _LoadMapTX: 


RESOLVING THE PROBLEM
The WTX Map needs to be present even if map caching enabled. Make sure the map is present in the location specified.
