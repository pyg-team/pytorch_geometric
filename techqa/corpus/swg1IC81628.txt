Title: IBM IC81628: ABRUPTLY TERMINATING A LARGE NUMBER OF MQ CONNECTIONS BETWEEN DATAPOWER AND THE MQ SERVER MAY CAUSE A RESTART - United States

Text:
 SUBSCRIBE
You can track all active APARs for this component.



APAR STATUS
 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  If DataPower does not have a stable network connection to its MQ
   Server, DataPower may restart if a large number of MQ
   connections are abruptly dropped.
   
   
    
   
   

LOCAL FIX

PROBLEM SUMMARY
 *  Affected are customers who use an MQ QM object under stress test
   if the network is not stable.
   
   The appliance might restart when it encounters a network error
   after destroying MQ connections.
   
   
    
   
   

PROBLEM CONCLUSION
 *  The fix is available on 3.8.2.12, 4.0.1.10 and 4.0.2.6.
