Title: IBM Cannot connect to Business Space WebDAV - United States

Text:
business space; webdav; connect; windows TECHNOTE (TROUBLESHOOTING)

PROBLEM(ABSTRACT)
 Attempts to connect to the WebDAV using Microsoft Windows 7 or Windows 2008 result in an error message. 

SYMPTOM
Windows might ask you to enter the credentials several times. After that, the following error message is shown: 

The folder you entered does not appear to be valid. Please choose another. [/support/docview.wss?uid=swg21590077&amp;aid=1]


ENVIRONMENT
Using Windows 7 or Windows 2008 to map the network location for Business Space WebDAV.


RESOLVING THE PROBLEM
Two steps are necessary to resolve this issue: 

 1. Make sure you are using the HTTPS URL to access the WebDAV. This URL would look like this:
    https://<bspace_host>:<bspace_secure_port>/mum/mycontenthandler/mm/dav/filestore
    e.g. https://bpm751:9443/mum/mycontenthandler/mm/dav/filestore 
 2. Install the required Microsoft Hotfix for KB2555258: http://support.microsoft.com/kb/2555258 [http://support.microsoft.com/kb/2555258]
