Title: IBM IY45552: CCSID 923 NOT LISTED AS SUPPORTED CODE PAGE FOR US ENGLISH ON AIX - United States

Text:
AIX SUBSCRIBE
You can track all active APARs for this component.



APAR STATUS
 * CLOSED AS DOCUMENTATION ERROR.
    
   
   

ERROR DESCRIPTION
 *  The list of supported native CCSIDs on AIX for US English in the
   WebSphere MQ Application Programming Reference manual, Appendix
   H, Code Page Conversion, does not contain CCSID 923 (US English
   with euro support).
   
   
    
   
   

LOCAL FIX

PROBLEM SUMMARY
 *  The WebSphere MQ Application Programming Reference Version 5.3
   book (SC34-6062-00) will be changed to add the missing native
   CCSID 923.
   
   
    
   
   

PROBLEM CONCLUSION
 *  This information needs to be added to the following section of
   the manual WebSphere MQ Application Programming Reference
   (SC34-6062-00):
   .
      Appendix H : Code page conversion
      Section    : National languages
      Subsection : US English
   .
      Add CCSID 923 to the AIX entry in the table, so that the
      native CCSIDs are listed as "819, 850, 923, 5348".
   .
   
   
    
   
   

TEMPORARY FIX
