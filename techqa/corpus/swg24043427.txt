UPDATE IBM WEBSPHERE MQ JCA RESOURCE ADAPTER TO VERSION 7.1.0.7 + APARs IT14385 and IT15559 

LOCAL FIX: 

PROBLEM SUMMARY

USERS AFFECTED:
All users of IBM WebSphere Application Server

PROBLEM DESCRIPTION:
The IBM WebSphere MQ JCA Resource Adapter shipped with IBM WebSphere Application Server needs updating.

RECOMMENDATION:
None

The WebSphere MQ JCA Resource Adapter shipped within WebSphere Application Server is used by all applications connecting to WebSphere MQ from within the application server. WebSphere Application Server now ships a new version of WebSphere MQ fix pack 7.1.0.7 that also includes WMQ RA APARs IT14385 and IT15559.

PROBLEM CONCLUSION: 
Update IBM WebSphere MQ JCA Resource Adapter to latest version of 7.1.0.7 which includes WMQ RA APARs IT14385 and IT15559. 

The fix for this APAR is currently targeted for inclusion in fix pack 8.5.5.12. Please refer to the Recommended Updates page for delivery information:
