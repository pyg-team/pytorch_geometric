ERROR DESCRIPTION
 *  AFTER ROUTING WIRE, BEND RADIUS RATIO
   VALUE CHANGES
   .
   Settings
   Select Tools/ Options/ Equipment
   Systems/Electrical HarnessDiscipline/Electrical
   Process Interface Tab
   1. Check on  External Data Interfacing
   2. Define the directory of xml on
   ExternalDataRepository
   .
   1. Open the  Product.CATProduct.
   2. Change Workbench to Electrical Harness
   Assembly.
   3. To confirm the Bend Radius Ratio, Activate Multi-
   branchable1
   4. Select Bundle segment.3 and Select Branch
   Definition icon
   5. Check the Bend Radius Ratio Value  1
   6. Double click Product1
   7. Change Workbench to Electrical Wire Routing.
   8. Select External Systems icon. System Selection
   window is displayed. test.xml sent to current
   systems. Select one xml as Current Systems.
   9. Select Automatic Wire routing icon. Wire and
   Wire group Routing window is displayed.
   10. All wire which is displayed on Available Wires /
   Wire groups is sent to Wire / Wire groups To Route
   11. Click Route button. Automatic Wire Routing
   Report window is displayed. Confirm that Status is
   successfully routed
   12. Result is OK.
   13. To confirm the Bend Radius Ratio, Activate
   Multi-branchable1
   14. Select Bundle segment.3 and Select Branch
   Definition icon
   15. Bend Radius Ratio value is changed to 4.67.
   .
   Problem: Bend Ratio is changed.
   .
   Expected result: Bend ratio should not be changed.
   .
