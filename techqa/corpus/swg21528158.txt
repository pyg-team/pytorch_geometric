Title: IBM DMRTSUB does not resolve symbolic variables (SCI22712) - United States

Text:
STERLINGPRI TECHNOTE (TROUBLESHOOTING)

PROBLEM(ABSTRACT)
 DMRTSUB does not resolve symbolic variables (SCI22712) 

SYMPTOM
DMRTSUB does not resolve symbolic variables


CAUSE
Symbolic variables are enclosed in single quotes

RESOLVING THE PROBLEM
Double quotes must be used to delimit the parameter statements for DMRTSUB, when the parameters contain symbolic variables that must be passed to the JCL.

Example: DMRTS PROC SNODE=snode-id -
&X = %NUM1
SYMBOL &Y = A.B.C || &X
RUN TASK (PGM=DMRTSUB - 
PARM=("DSN=pds(member),DISP=SHR", - 
"DSN &Y" )) SNODE
