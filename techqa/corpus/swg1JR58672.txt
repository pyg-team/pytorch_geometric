APAR STATUS
 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  The formatDate(), formatTime and formatDateTime() Java Script
   functions always return the time zone settings from an IBM
   Process Server even though the time zone settings might be
   defined in the user preferences.
   
   PRODUCTS AFFECTED
   IBM BPM Advanced V8.5.7 and earlier versions
   IBM BPM Standard V8.5.7 and earlier versions
   IBM BPM Express V8.5.7, V8.6, and earlier versions
   IBM BPM V8.6
   
   
    
   
   

LOCAL FIX

PROBLEM SUMMARY
 *  No additional information is available.
   
   
    
   
   

PROBLEM CONCLUSION
 *  A fix will be included in a future release of IBM BPM that
   ensures the time zone settings defined in the user preferences
   take precedence over the time zone settings on a process server.
    An interim fix is also available for IBM BPM V8.5.6.0 CF02.
