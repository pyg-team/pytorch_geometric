Title: IBM Export Filters and Views from one DASH environment to another - United States

Text:
DASHL2; DASHL2IMPEXP; export; filters ; WebGUI; event viewer TECHNOTE (FAQ)

QUESTION
 Why do the filters I configured in WebGUI not get exported when I use the Export Wizard in the Dashboard Application Services Hub (DASH)? 

CAUSE
The Export Wizard exports only DASH related items. 

ANSWER
As the Export Wizard in DASH only exports DASH related items, you will need to use the consolecli script instead. 

Please see this link for the corresponding information:

Copying data between Web GUI servers
https://www.ibm.com/support/knowledgecenter/SSSHTQ_8.1.0/com.ibm.netcool_OMNIbus.doc_8.1.0/webtop/wip/task/web_adm_exportimport.html?lang=en [https://www.ibm.com/support/knowledgecenter/SSSHTQ_8.1.0/com.ibm.netcool_OMNIbus.doc_8.1.0/webtop/wip/task/web_adm_exportimport.html?lang=en] 

If both servers are on the same version of DASH and WebGUI, you can use the information provided in the section "Cloning data between servers"
