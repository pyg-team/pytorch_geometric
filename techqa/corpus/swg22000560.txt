apache struts vulnerability TECHNOTE (FAQ)

QUESTION
 Is IBM Sterling B2B Integrator or IBM Sterling File Gateway affected by the Apache Struts vulnerability CVE-2017-5638? 

ANSWER
IBM Sterling B2B Integrator and IBM Sterling File Gateway are not affected by the Apache Struts vulnerability CVE-2017-5638. 

CVE-2017-5638 only affects Apache Struts 2.x. Sterling B2B Integrator and Sterling File Gateway use Apache Struts 1.1. There are currently no plans to move to a newer version of Apache Struts.
