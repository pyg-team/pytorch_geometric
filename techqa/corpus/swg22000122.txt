Title: IBM WebSphere Application Server is not vulnerable to CVE-2017-5638 - United States

Text:
 FLASH (ALERT)

ABSTRACT
 IBM WebSphere Application Server is not vulnerable to the Apache Struts 2 vulnerability CVE-2017-5638 

CONTENT
IBM WebSphere Application Server in all editions and all platforms is NOT vulnerable to the Apache Struts 2 vulnerability (CVE-2017-5638). The IBM HTTP Server is also not affected. 
NOTE: You should check your applications to determine if they are using the vulnerable Apache Struts APIs and update your Apache Struts 2 accordingly. Refer to https://cwiki.apache.org/confluence/display/WW/S2-045 [https://cwiki.apache.org/confluence/display/WW/S2-045]
