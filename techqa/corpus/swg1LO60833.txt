Title: IBM LO60833: WIPE ON NOKIA DEVICE WILL WIPE STORAGE CARD EVEN IF NOT SELECTED. - United States

Text:
 SUBSCRIBE
You can track all active APARs for this component.



APAR STATUS
 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  Customer reported that on Nokia devices the wipe command will
   hard reset the Nokia device as expected, but also wipe the
   storage card.  It should not wipe the storage card by default
   since that is an option for the wipe.
   
   
    
   
   

LOCAL FIX
 *  NA
   
   
    
   
   

PROBLEM SUMMARY
 *  Security wipe of Nokia client would wipe the storage card when
   not selected as one of the wipe options.
   
   
    
   
   

PROBLEM CONCLUSION
 *  This problem has been resolved.  Wipe options will function as
   expected.
   
   
    
   
   

TEMPORARY FIX
 *  This fix is included in Lotus Traveler 8.5.2.3 and all later
   releases.
