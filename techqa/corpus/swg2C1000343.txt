Title: IBM Security Bulletin: A security vulnerability has been identified in IBM WebSphere Application Server, Business Process Manager, IBM Tivoli Monitoring shipped with IBM Cloud Orchestrator (CVE-2017-1194) - United States

Text:
 SECURITY BULLETIN

SUMMARY
 A security vulnerability affects WebSphere Application Server that is shipped with IBM Cloud Orchestrator. IBM Cloud Orchestrator has addressed the vulnerability. 

VULNERABILITY DETAILS
CVEID: CVE-2017-1194 [http://cve.mitre.org/cgi-bin/cvename.cgi?name=CVE-2017-1194]
DESCRIPTION: IBM WebSphere Application Server is vulnerable to cross-site request forgery which could allow an attacker to execute malicious and unauthorized actions transmitted from a user that the website trusts.
CVSS Base Score: 4.3
CVSS Temporal Score: See https://exchange.xforce.ibmcloud.com/vulnerabilities/123669 [https://exchange.xforce.ibmcloud.com/vulnerabilities/123669] for the current score
CVSS Environmental Score*: Undefined
CVSS Vector: (CVSS:3.0/AV:N/AC:L/PR:N/UI:R/S:U/C:N/I:L/A:N)
