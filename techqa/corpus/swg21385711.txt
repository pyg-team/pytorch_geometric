QUESTION
 What releases of z/OS and CICS Transaction Server for z/OS (CICS TS) are compatible with CICS Performance Analyzer for z/OS (CICS PA) V3.1? 

ANSWER
CICS PA V3.1 supports z/OS 1.6 or later, and the following releases of CICS: 

 * CICS TS for z/OS V4.1 (5655-S97) 
 * CICS TS for z/OS V3.2 and V3.1 (5655-M15) 
 * CICS TS for z/OS V2.2 and V2.3 (5697-E93)


CICS PA V3.1 drops support for SMF records created by CICS TS V1.3 and V2.1. 

CICS PA V3.1 also supports CICS Transaction Gateway V7.1 statistics (SMF type 111 records). 

Note: CICS PA V3.1 does not support CICS TS for z/OS V4.2 (5655-S97). CICS PA V3.2 is required to support CICS TS V4.2 with APAR PM30692 [http://www.ibm.com/support/docview.wss?uid=swg1PM30692]. 
These and other operating environment specifications can be found under Technical information in the CICS PA V3.1 Announcement Letter 209-086 [http://www.ibm.com/common/ssi/cgi-bin/ssialias?infotype=an&subtype=ca&supplier=897&letternum=ENUS209-086]:

Hardware requirements 

CICS PA V3.1 runs on any S/390 or IBM eServer zSeries machine on which the applicable operating and software will run. 

Software requirements 

CICS PA V3.1 requires System Modification Program/Extended (SMP/E) of the supported z/OS or OS/390 system (5655-G44) for installation and maintenance. It operates with the following CICS configurations: 

 * CICS TS for z/OS V4.1 (5655-S97), V3.1, and V3.2 (5655-M15) 
 * CICS TS for z/OS V2.2 and V2.3 (5697-E93)


Note : CICS TS for z/OS V4.1 requires APAR PK83926 [http://www.ibm.com/support/docview.wss?uid=swg1PK83926] to be applied. 

The following products are also required:  * z/OS V1.6 or later (contains SMP/E) (5694-A01)
