ERROR DESCRIPTION
 *  Within a Worklight application a user can subscribe to
   notifications or unsubscribe.
   This events are handled by a Worklight adapter that register or
   delete users subscriptions in the database.
   But in case the application is uninstalled without
   unsubscribing, the user subscription  will remain the the
   database  .
   additional Keywords :
   subscribe ,unsubscribe ,Push,GCM,onDeviceUnsubscribe
   
   
   Worklight Versions Affected:
   6.x
   Initial Impact:
   LOW
