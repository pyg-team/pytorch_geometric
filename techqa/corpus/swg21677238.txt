Title: IBM TM1 Web HTTP Error 500.0 - Internal Server Error - United States

Text:
 TECHNOTE (TROUBLESHOOTING)

PROBLEM(ABSTRACT)
 Customer is unable to navigate to tm1web. They receive
TM1 Web HTTP Error 500.0 - Internal Server Error 

SYMPTOM
HTTP Error 500.0 - Internal Server Error 

The page cannot be displayed because the internall server error has
occurred
IsapiFilterModule
AuthenticateRequest
StaticFile
0x8007001
Requested URL 
http://servername:portnumbe/tm1web [http://servername:portnumbe/tm1web]
physical path D:\ibm\cognos\tm1_64\tm1web
They are using an IIS redirect.


CAUSE
They had an apache ISAPI redirect setup 


ENVIRONMENT
TM1 10.1.1



RESOLVING THE PROBLEM
Customer had another environment that was functioning. 

They had to copy the required files from that server to their server that was not working. 

The following IBM documentation discusses what files are needed:
http://pic.dhe.ibm.com/infocenter/cicm/v8r0m3/index.jsp?topic=%2Fcom.ibm.swg.ba.cognos.ig_icm.8.0.3.doc%2Ft_configure-iis-to-forward-requests-to-ibm-cognos-icm.html [http://pic.dhe.ibm.com/infocenter/cicm/v8r0m3/index.jsp?topic=%2Fcom.ibm.swg.ba.cognos.ig_icm.8.0.3.doc%2Ft_configure-iis-to-forward-requests-to-ibm-cognos-icm.html]
