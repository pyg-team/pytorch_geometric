OVERVIEW
IBM Watson Explorer Version 9.0.0.8 is a fix pack that addresses security issues. See the following Security Bulletins for more information: 

 * Security Bulletin: Vulnerability in OpenSSL affects Watson Explorer (CVE-2016-7055) [http://www.ibm.com/support/docview.wss?uid=swg21999447] 
 * Security Bulletin: Multiple Vulnerabilities Affect Watson Explorer (CVE-2016-8610, CVE-2016-2177, CVE-2016-2183, CVE-2016-5597, CVE-2016-5542) [http://www.ibm.com/support/docview.wss?uid=swg21995749]

For more information about IBM Watson Explorer Version 9.0.0.8, see Release Notes, Version 9.0.0.5 [http://www.ibm.com/support/knowledgecenter/SS8NLW_9.0.0/com.ibm.swg.im.infosphere.dataexpl.relnotes.doc/c_rn-wrapper.html].
