Title: IBM Jazz for Service Management (JazzSM) installation using IBM Installation Manager (IIM) reports wrong disk space requirement - United States

Text:
 TECHNOTE (TROUBLESHOOTING)

PROBLEM(ABSTRACT)
 Installation of Tivoli Common Reporting (TCR) 3.1.2 or 3.1.2.1 using IIM, disk space requirement is more than what is reported in IIM


RESOLVING THE PROBLEM
Recommendation is to run Pre-reqchcker for diskspace verification prior to the actual installation using IIM
