Title: IBM LO50125: HOLLIDAY NOT SAVED ON NOKIA DEVICE. - United States

Text:

APAR STATUS
 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  Customer reported that some calendar events are not saved on
   their Nokia device.  They get this error message in the logs:
   
   "ERROR Calendar event ... failed to be added to the device."
   
   
    
   
   

LOCAL FIX
 *  NA
   
   
    
   
   

PROBLEM SUMMARY
 *  Customer was seeing "ERROR Calendar event ... failed to be added
   to the device." for some Holidays on the Nokia device.
   
   
    
   
   

PROBLEM CONCLUSION
 *  The missing Holidays were recurring events with exceptions.  The
   handling of these types of events has been improved.
   
   
    
   
   

TEMPORARY FIX
 *  Interim Fix is available on request.  It will be included in
   8513 and later releases.  See this technote for latest
   maintenance info:
   
   http://www.ibm.com/support/docview.wss?uid=swg24019529 [http://www.ibm.com/support/docview.wss?uid=swg24019529]
