Title: IBM In Studio, while validating an SOQL with a relationship table, receiving the error 'INVALID_TYPE'. - United States

Text:
INVALID_TYPE salesforce.com sdfc validate query TECHNOTE (TROUBLESHOOTING)

PROBLEM(ABSTRACT)
 In WebSphere Cast Iron Studio, while validating an SOQL with a relationship table, you receive the error 'INVALID_TYPE'. 

CAUSE
The salesforce.com API being used in Cast Iron is not the latest API version.

RESOLVING THE PROBLEM
Use the latest API in the salesforce.com endpoint and validate the SOQL. 

If it still fails to validate, contact IBM Support through the SR tool [https://www-946.ibm.com/support/servicerequest/help/srHelp.action] or by phone (1-800-IBM-Serv).
