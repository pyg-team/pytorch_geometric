APAR STATUS
 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  java.lang.ExceptionInInitializerError
   can occur if the classloader in use while using the SOAP Extract
   node is not the WebServices parser.
   .
   One or more of the following errors may be seen.
   Error BIP2121E: The thread bootstrap code caught an unhandled
   exception on thread '125'.
   FatalException BIP3702E: A Fatal Java exception was thrown
   whilst calling the Java JNI method
   ''method_com_ibm_broker_axis2_SoapParserUtil_attachmentCount''.
   The Java exception was
   ''java.lang.ExceptionInInitializerError''.
   UserTrace BIP3484E: ''Please edit axis2.xml and replace with the
   same class in org.apache.axis2.dispatchers package''
   An adapter component has written the diagnostic message included
   here.
   
   
    
   
   

LOCAL FIX

PROBLEM SUMMARY
 *  ****************************************************************
   * USERS AFFECTED: All users of WebSphere Message Broker using  *
   *                 the SOAP Extract node to process SOAP        *
   *                 attachments                                  *
   ****************************************************************
   * PROBLEM DESCRIPTION: Processing of SOAP attachments may      *
   *                      cause                                   *
   *                      java.lang.ExceptionInInitializerError.  *
   *                      The following errors may be seen:       *
   *                      BIP2121E, BIP3702E and BIP3484E.        *
   ****************************************************************
   * RECOMMENDATION:                                              *
   ****************************************************************
   Processing of SOAP attachments may cause
   java.lang.ExceptionInInitializerError.
   The following errors may be seen:
   BIP2121E, BIP3702E and BIP3484E.
   
   This is caused by the SOAP Extract node being able to use an
   incorrect ClassLoader.
   
   
    
   
   

PROBLEM CONCLUSION
 *  WebSphere Message Broker has been modified to ensure the SOAP
   Extract node uses the correct ClassLoader.
   
   
    
   
   

TEMPORARY FIX
