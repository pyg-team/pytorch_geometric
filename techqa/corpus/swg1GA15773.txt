APAR STATUS
 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  After making a change and customizing a layout and assigning to
   a role users cannot login to the GUI.
   After entering the password and hitting enter an
   error occurs: unexpected error has occurred. Please contact
   your system administrator
   
   There is a 2nd details tab that shows:
   java.lang.NullPointerException
   
   
    
   
   

LOCAL FIX
 *  Remove the role assigned to the custom layout from users
   
   
    
   
   

PROBLEM SUMMARY
 *  Description: After making layout changes and assigning to the
   admin user any user with admin role cannot login to the GUI.
   After entering the password and hitting enter an error occurs:
   unexpected error as occurred. Please contact your system
   administrator.
   
   Additionally, error will occur when:
   1) assign multiple roles to the user
   2) create custom layouts for assigned roles
   3) reset custom layout for one or more roles of that user
   4) try to login as that user
   
   Solution: This is caused by the incorrect reset process of
   custom role layouts during user login. Code has been revised to
   fix this issue.
   
   
    
   
   

PROBLEM CONCLUSION
 *  Fix is available in v10.1 or higher patch level.
