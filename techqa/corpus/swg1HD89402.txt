ERROR DESCRIPTION
 *  ABSTRACT:
   Incorrect filter in XML Clash reports
   SCENARIO:
   1)
   Set Options to create non-federated XML output.
   2) Open
   DMUNav workbench
   3) Open attached product
   4) Open
   interference Results.1
   5) Choose "Export As" and export to
   xml. Browser will be started
   6) In the browser, make sure to
   allow active Content (JavaScript)
   7) Open filter option in
   browser window and uncheck all but "Relevant" for "Filter by
   Status"
   8) Apply the filter
   RESULT:
   Incorrect filter in XML
   Clash Reports. Detail Information for irrelevant results is
   still visible
   EXPECTED RESULT:
   Filter should give correct &
   relevant results only.

PROBLEM SUMMARY
 *  ABSTRACT:
   Incorrect filter in XML Clash reports
   SCENARIO:
   1) Set Options to create non-federated XML output.
   2) Open DMUNav workbench
   3) Open attached product
   4) Open interference Results.1
   5) Choose "Export As" and export to xml. Browser will be
   started
   6) In the browser, make sure to allow active Content
   (JavaScript)
   7) Open filter option in browser window and uncheck all but
   "Relevant" for "Filter by Status"
   8) Apply the filter
   RESULT:
   Incorrect filter in XML Clash Reports. Detail Information for
   irrelevant results is still visible
   EXPECTED RESULT:
   Filter should give correct & relevant results only.
