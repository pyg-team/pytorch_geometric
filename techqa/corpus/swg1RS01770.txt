ERROR DESCRIPTION
 *  When trying to copy a rule with ODM 8.5.1. Decision Center
   Enterprise Console , the target folder looks like PROJECT_NAME:
   /%current_key:/TARGET_BASELINE/TARGET_FOLDER with the following
   error message "The value 'PROJECT_NAME:/%current_key:/TARGET_
   BASELINE/TARGET_FOLDER' is not valid for property Folder". It
   affects projects migrated from JRules 7.1.1 to ODM 8.5.1.
