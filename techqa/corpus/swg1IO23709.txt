APAR STATUS
 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  Using http (non-SSL) might be a security risk and it is
   recommended all users to use https (SSL) connection.
   
   
    
   
   

LOCAL FIX
 *  Use https (SSL) request URL specifically.
   e.g. https://<hostname>:12100/IDSWebApp [https://<hostname>:12100/IDSWebApp]
   
   
    
   
   

PROBLEM SUMMARY
 *  It is done by modifying application and deployment script.
   Application level security is implemented. This type of
   security provides application isolation and requirements for
   authenticating application users.
   
   
    
   
   

PROBLEM CONCLUSION
 *  The fix for this APAR is contained in the following maintenance
