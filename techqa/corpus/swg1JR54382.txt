APAR STATUS
 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  When you log in to the Business Process Choreographer Explorer
   in the BPESystemMonitor Java EE role, you cannot see activity
   input/output messages.
   
   
    
   
   

LOCAL FIX

PROBLEM SUMMARY
 *  The BPESystemMonitor Java EE-role is not sufficiently regarded
   during the authorization check and, therefore, the
   getInputMessage and getOutputMessage actions are not granted for
   users in this role.
   
   
    
   
   

PROBLEM CONCLUSION
 *  A fix is/will be available for IBM BPM V8.5.5.0 that checks for
   the BPESystemMonitor role.
   
   On Fix Central (http://www.ibm.com/support/fixcentral), search
   for JR54382:
   
   1. Select IBM Business Process Manager with your edition from
     the product selector, the installed version to the fix pack
     level, and your platform, and then click Continue.
   
   2. Select APAR or SPR, enter JR54382 , and click Continue.
   
   When you download fix packages, ensure that you also download
   the readme file for each fix. Review each readme file for
   additional installation instructions and information about the
   fix.
   
   
    
   
   

TEMPORARY FIX
 *  Not applicable
