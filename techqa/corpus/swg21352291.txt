Title: IBM Inserting carraige return in concatinated fields - United States

Text:
 TECHNOTE (TROUBLESHOOTING)

PROBLEM(ABSTRACT)
 Is it possible to insert a carriage return or line break in a concatenated field? 

RESOLVING THE PROBLEM
Try the following:

1. Add the two (or more) columns into the report
2. Select the text of one column and 'Cut' from the Edit menu.
3. Edit the text of the other column and 'Paste' from the Edit menu.
4. In the field properties, select multi-line with word wrap.
5. Go to the location in the text that you would like to have a carriage return and hit [ENTER].
