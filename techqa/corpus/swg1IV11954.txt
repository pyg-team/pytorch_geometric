Title: IBM IV11954: FP13 FOR MYSAP AGENT BREAKS THE LANGUAGE PACK - United States

Text:
 SUBSCRIBE
You can track all active APARs for this component.



APAR STATUS
 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  A problem with ITM for MySAP agent. After applying IF013 it
   breaks language pack.
   Only shows MySAP workspaces in English. Previous  to that it was
   working fine showing in all languages.
   Steps to reproduce this issue.
   1) Install a TEPS and TEMS with Language support(in customers
   case  it is Brasil- Portuguese)
   2) Then apply app support for SAP agent at any version prior to
   IF13, the TEPS will display the SAP workpsaces in the correct
   language.
   3) If you apply 6.2.0-TIV-ITM_SAP-IF0013 app support and
   restart TEPS the SAP agent display goes back to english
   
   Operating System:                         Linux
   Operating System Version/Release:         redhat linux 5
   Product with Version/Release:             tivoli monitoring
   agent 6.2
   Stand-alone or Bundle:                   Stand-alone

   Agent IFix13.
