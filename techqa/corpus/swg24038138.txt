III. INSTALLATION INSTRUCTIONS:

To deploy/install this iFix it must be done for each eWAS/TIP server where
TNPM 1.3.3 is installed.

1) Save the downloaded iFix file to a temporary work location.
( i.e. /tmp/if10/1.3.3.0-TIV-TNPM-IF0010.tar.gz )

2) Change directory to where the TNPM 1.3.3 GA release is currently installed:
cd <tip>/products/tnpm

( i.e. cd /opt/IBM/tivoli/tip/products/tnpm )

3) Uncompress and Untar the iFix:
gunzip -c /tmp/if10/1.3.3.0-TIV-TNPM-IF0010.tar.gz | tar -xvf -

4) Run the patch script to deploy the iFix:
./dataview/bin/patch.sh -tipuser <username> -tippassword <password>

(i.e. ./dataview/bin/patch.sh -tipuser tipadmin -tippassword tipadmin )

5) Stop and restart the TIP server as described in the documentation

To stop:
<tipComponents>/TCRComponent/bin/stopTCRserver.sh <username> <password>

(i.e. /opt/IBM/tivoli/tip21Components/TCRComponent/bin/stopTCRserver.sh tipadmin tipadmin )

To start:
<tipComponents>/TCRComponent/bin/startTCRserver.sh <username> <password>

(i.e. /opt/IBM/tivoli/tip21Components/TCRComponent/bin/startTCRserver.sh tipadmin tipadmin )

6) <Optional> To verify the deployment:
Change directory to where the TNPM 1.3.3 GA release is currently installed:
cd <tip>/products/tnpm
( i.e. cd /opt/IBM/tivoli/tip/products/tnpm )

./dataview/bin/information.sh <tip> <user> <password>

(./dataview/bin/information.sh /opt/IBM/tivoli/tip tipadmin tipadmin )
