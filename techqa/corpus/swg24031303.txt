PREREQUISITES
Prior to applying this Interim Fix, if the Interim Fix will be installed directly from the IBM update server, then you must have IBM Integration Designer 7.5 or higher installed. If the Interim Fix will be installed locally, then you must have IBM Integration Designer V7.5.1 installed. Also, ensure that you have the IBM Installation Manager [https://www-304.ibm.com/support/docview.wss?uid=swg24030152] 1.5 or higher installed.

INSTALLATION INSTRUCTIONS
It is recommended that you install the Interim Fix directly from the IBM update server; however, it is also available for download. You might want to download and install the interim fix in the following situations: 


 * You have a slow or unstable Internet connection and want to use a download manager that can resume the download. 
 * 
 * You have difficulty accessing the live IBM update servers from behind a firewall. 
 * 
 * You prefer to download the Interim Fix and install it later.
