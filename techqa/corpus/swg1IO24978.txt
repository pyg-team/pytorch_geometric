APAR STATUS
 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  IBM Content Navigator V2.0.3, when creating new search and
   share, a user with insufficient CM8 privileges will receive ACL
   error when adding another user with AllPrivs privilege set to
   share the new search.
   
   
    
   
   

LOCAL FIX

PROBLEM SUMMARY
 *  Users that do not have the UserACLOwner privilege receive an
   error when sharing a new search with other users.
   
   
    
   
   

PROBLEM CONCLUSION
 *  This has been addressed in IBM Content Navigator V3.0.1. With
   this fix, users that do not have the UserACLOwner privilege can
   share a new search with other users.
