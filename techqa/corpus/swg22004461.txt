Title: IBM Security Bulletin: Cross-Site Scripting vulnerability affects IBM Security Guardium (CVE-2017-1256) - United States

Text:
 SECURITY BULLETIN

SUMMARY
 Cross-site scripting vulnerability allows users to embed arbitrary JavaScript code in the Web UI thus altering the intended functionality potentially leading to credentials disclosure within a trusted session. IBM Security Guardium has fixed this vulnerability 

VULNERABILITY DETAILS
CVEID: CVE-2017-1256 [http://cve.mitre.org/cgi-bin/cvename.cgi?name=CVE-2017-1256]
DESCRIPTION: IBM Security Guardium is vulnerable to cross-site scripting. This vulnerability allows users to embed arbitrary JavaScript code in the Web UI thus altering the intended functionality potentially leading to credentials disclosure within a trusted session.
CVSS Base Score: 6.1
CVSS Temporal Score: See https://exchange.xforce.ibmcloud.com/vulnerabilities/124678 [https://exchange.xforce.ibmcloud.com/vulnerabilities/124678] for the current score
CVSS Environmental Score*: Undefined
CVSS Vector: (CVSS:3.0/AV:N/AC:L/PR:N/UI:R/S:C/C:L/I:L/A:N) 

AFFECTED PRODUCTS AND VERSIONS
IBM Security Guardium V10.0, 10.0.1, 10.1, 10.1.2.
