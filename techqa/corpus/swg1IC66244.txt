Title: IBM IC66244: UPDATE HTTPREQUEST AND SOAPREQUEST NODE INFORMATION CENTER TOPICS TO INCLUDE A NOTE ABOUT THE TIMEOUT VALUE. - United States

Text:

APAR STATUS
 * CLOSED AS DOCUMENTATION ERROR.
    
   
   

ERROR DESCRIPTION
 *  The Information center topics for the HTTPRequest node
   (ac04595_) and SOAPRequest node (ac56190_) need to include a
   note under the timeout properties to warn that the timeout may
   take up to a second longer than the value specified.
   
   
    
   
   

LOCAL FIX
 *  Not Applicable
   
   
    
   
   

PROBLEM SUMMARY
 *  ****************************************************************
   USERS AFFECTED:
   All users of WebSphere Message Broker on all platforms who use
   the HTTPRequest and SOAPRequest nodes.
   
   Platforms Affected:
   All Platforms
   ****************************************************************
   PROBLEM SUMMARY:
   The Information center topics for the HTTPRequest node
   (ac04595_) and SOAPRequest node (ac56190_) need to include a
   note under the timeout properties to warn that the timeout may
   take up to a second longer than the value specified.
   
   
    
   
   

PROBLEM CONCLUSION
 *  WebSphere Message Broker Information center topics for the
   HTTPRequest node (ac04595_) and SOAPRequest node (ac56190_)
   have been modified to include a note in the "Request timeout"
   properties "Description" column to warn that the timeout may
   take up to a second longer than the value specified.
