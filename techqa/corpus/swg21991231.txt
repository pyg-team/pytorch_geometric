Title: IBM Unable to start Cognos - CFG-ERR-0106 IBM Cognos Configuration did not receive a response from the IBM Cognos service in the time allotted. - United States

Text:
 TECHNOTE (FAQ)

QUESTION
 Cognos does not start !

No new cogserver.log file just this error in the cbs_start_WebSphereLiberty.log file under <cognos_install_location>/logs
Fri Sep 23 16:03:33 2016 ERROR t[e14] CBSSocketCommand failed to connect, CAM error: <errorDetail><errorCode>-12</errorCode><errorMessage>CAM-CRP-0026 The underlying socket: '10.80.12.43:9300' returned an error.</errorMessage><errorStack><errorCode>10061</errorCode><errorMessage>Could not connect the socket, errno: 0x274d(10061)</errorMessage></errorStack></errorDetail>

but in the message.log file under <cognos_install_location>/wlp/usr/servers/cognosserver/logs, this error can be found.
[9/23/16 17:21:47:331 CEST] 00000001 com.ibm.ws.kernel.launch.internal.FrameworkManager A CWWKE0001I: The server cognosserver has been launched.
[9/23/16 17:21:48:003 CEST] 00000001 com.ibm.ws.logging.internal.impl.IncidentImpl I FFDC1015I: An FFDC Incident has been created: "java.lang.RuntimeException: Error initializing storage. com.ibm.ws.kernel.launch.internal.FrameworkManager 506" at ffdc_16.09.23_17.21.47.0.log 

CAUSE
The root cause, a corruption in the <cognos_install_location>\wlp\usr\servers\cognosserver\workarea for WebSphere Liberty Profile (WLP)

ANSWER
First, backup the "workarea" folder then delete the "workarea" folder and restart Cognos.
