ABSTRACT
 This fix upgrades the embedded WebSphere Application Server (eWAS) that is provided with IBM Tivoli Monitoring portal server version..


DOWNLOAD DESCRIPTION
This fix upgrades the embedded WebSphere Application Server (eWAS) which is shipped as part of the IBM Tivoli Monitoring portal server to version 8.0.0.10. It addresses security vulnerabilities as documented in the Security bulletin: http://www.ibm.com/support/docview.wss?uid=swg21688073 [http://www.ibm.com/support/docview.wss?uid=swg21688073].




Note: The link to the patch below has been updated to address an install issue a few customers encountered. The package has been renamed from 6.X.X-TIV-ITM_EWAS_xxxxxxx_8.00.10.00 to 6.X.X-TIV-ITM_EWAS_xxxxxxx_8.00.10.01, however it applies the _SAME_ update to embedded WebSphere Application Server. If you already applied 6.X.X-TIV-ITM_EWAS_xxxxxxx_8.00.10.00 successfully, there is no need or benefit to also apply 6.X.X-TIV-ITM_EWAS_xxxxxxx_8.00.10.01.
