ERROR DESCRIPTION
 *  Environment:
   All windows versions.
   Microsoft Active Directory Agent 6.3.1.2
   
   Problem Description:
   Due to incorrect value of PATH environment Variable, Active
   Directory 64-bit Agent does not connect to Tivoli Enterprise
   Monitoring Server (TEMS) using IPSPIPE.
   
   
   Detailed Recreation Procedure:
   1. Install the Active Directory Agent 6.3.1.2 (64 bit)
   2. Reconfigure the Active Directory agent and use the protocol
   IP.SPIPE.
   3. Same way reconfigure the TEMS, TEPS and use protocol
   IP.SPIPE.
   4. Start monitoring AD Agent.
   5. No data will be display on portal.
