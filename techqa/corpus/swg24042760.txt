 This document contains links to the download page and the readme file for IBM Datacap Version 9.0.1 Feature Pack 2 

DOWNLOAD DESCRIPTION
You can download IBM Datacap Version 9.0.1 Feature Pack 2 from Passport Advantage or Fix Central. 

Follow these instructions to obtain Feature Pack 2 from Fix Central. If you have forgotten your IBM ID or have never registered, follow the instructions on the sign-in page. 

1. Click this link: http://www.ibm.com/support/fixcentral/ [http://www.ibm.com/support/fixcentral/]
2. Select Product Group: Enterprise Content Management.
3. Select Product: Datacap.
4. Select Installed Version: 9.0.1.1 (or 9.0.1.0)
5. Select Platform: All and click Continue.
6. Select Browse for fixes and click Continue.
7. Select the feature pack, click Continue. 
8. Sign in with your IBM ID and Password, then click Submit.
9. Download the package.

For a list of resolved problems, see the "Issues fixed in IBM Datacap 9.0.1 Feature Pack 2" topic in the feature pack readme:
