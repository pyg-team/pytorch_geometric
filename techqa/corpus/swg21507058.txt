Title: IBM Upgrading a DataPower appliance to add a field upgrade feature - United States

Text:
 TECHNOTE (FAQ)

QUESTION
 How do I add a field upgrade feature to an IBM WebSphere DataPower appliance? 

ANSWER
See the technote that is the most appropriate for your feature and machine type. 

 * For installing the Application Optimization feature on a DataPower Type 9235 appliance or DataPower Type 4195 blade: [http://www.ibm.com/support/docview.wss?uid=swg21402883]  * 
    * Upgrading a DataPower Type 9235 or Type 4195 appliance to add the Application Optimization feature
   
   
 * For installing the Application Optimization feature on a DataPower Type 7199 appliance:  * 
    * Upgrading an IBM WebSphere DataPower Integration XI52 appliance to add the Application Optimization feature
   
   
 * For installing the Application Optimization feature on a DataPower Type 7198 appliance:  * 
    * Upgrading an IBM WebSphere DataPower Security Gateway XG45 appliance to add the Application Optimization feature
   
   
 * For installing the Data Integration Module on a DataPower Type 7198 appliance:  * 
    * Upgrading an IBM WebSphere DataPower Security Gateway XG45 appliance to add the Data Integration Module feature
