QUESTION
 How do you determine an alert from the LED indicators on the XGS and GX appliances? 

ANSWER
The LED indicators on the front panel and on the network adapter ports of the GX and XGS appliances provide information that is related to the activity of the device, and can also report a failure or an alert.

The following tables include the information that is related to the meaning of each LED and color indicator:

Management Port (Network Adapter Indicators): 

Management Port 1 & 2 Left Green Steady Link status UP Left Green Flashing Link status UP and data that is transmitted or received Right Amber Steady Link Speed 1Gbps Right Green Steady Link Speed 100Mbps Right Off Off Link Speed 10Mbps 

Protected Ports (Security Interfaces): 

Protected Ports Left Green Steady Link status UP [/support/docview.wss?uid=swg21959769&amp;aid=1] Left Green Flashing Link status UP and data that is transmitted or receive [/support/docview.wss?uid=swg21959769&amp;aid=1] Right Green Steady Link Speed 1Gbps [/support/docview.wss?uid=swg21959769&amp;aid=1] Right Abber Steady Link Speed 100Mbps [/support/docview.wss?uid=swg21959769&amp;aid=1] Right Off Off Link Speed 10Mbps 

Power LED: 

Green Steady Powered up Amber Flashing There is a power supply fault (redundant power supply) Off Off [/support/docview.wss?uid=swg21959769&amp;aid=1] 

Hard Disk Activity LED: 

LED Color State Description Blue Steady There is activity on the disk Amber On There is a HDD or RAID fault Off Off No activity nor fault
