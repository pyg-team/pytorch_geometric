Title: IBM Security Bulletin: Multiple Mozilla Firefox vulnerability issues in IBM Storwize V7000 Unified. - United States

Text:
 SECURITY BULLETIN

SUMMARY
 There are security vulnerabilities in versions of Mozilla Firefox that are shipped with versions 1.5.1.0 to 1.5.2.5 of IBM Storwize V7000 Unified 

VULNERABILITY DETAILS
IBM Storwize V7000 Unified is shipped with Mozilla Firefox. There are vulnerabilities in certain versions of Mozilla Firefox shipped in certain versions of IBM Storwize V7000 Unified. These vulnerabilities concern the potential ability of a remote attacker to execute arbitrary code on a vulnerable system or cause a denial of service.
CVEID: CVE-2017-5398 [https://cve.mitre.org/cgi-bin/cvename.cgi?name=CVE-2017-5398]
DESCRIPTION: Mozilla Firefox could allow a remote attacker to execute arbitrary code on the system, caused by memory safety bugs within the browser engine. By persuading a victim to visit a specially-crafted Web site, a remote attacker could exploit this vulnerability using unknown attack vectors to execute arbitrary code on the vulnerable system or cause a denial of service.
CVSS Base Score: 8.8
CVSS Temporal Score: See https://exchange.xforce.ibmcloud.com/vulnerabilities/122818 [https://exchange.xforce.ibmcloud.com/vulnerabilities/122818] for the current score
CVSS Environmental Score*: Undefined
CVSS Vector: (CVSS:3.0/AV:N/AC:L/PR:N/UI:R/S:U/C:H/I:H/A:H) 

CVEID: CVE-2017-5400 [https://cve.mitre.org/cgi-bin/cvename.cgi?name=CVE-2017-5400]
DESCRIPTION: Mozilla Firefox could allow a remote attacker to bypass security restrictions, caused by an error related to the combination of JIT-spray targeting asm.js and a heap spray. By persuading a victim to visit a specially-crafted Web site, a remote attacker could exploit this vulnerability to bypass ASLR and DEP protections and launch further attacks on the system.
CVSS Base Score: 4.3

AFFECTED PRODUCTS AND VERSIONS
IBM Storwize V7000 Unified 

The product is affected when running code releases 1.5.1.0 to 1.5.2.5.

REMEDIATION/FIXES
A fix for these issues is in version 1.5.2.6 of IBM Storwize V7000 Unified. Customers running an affected version of IBM Storwize V7000 Unified should upgrade to 1.5.2.6 or a later version, so that the fix gets applied.

Latest Storwize V7000 Unified Software [http://www-01.ibm.com/support/docview.wss?uid=ssg1S1003918&myns=s028&mynp=OCST5Q4U&mync=E] 

WORKAROUNDS AND MITIGATIONS
Workaround(s) :
Normal operation of IBM Storwize V7000 Unified does not require or call for customers to use Firefox to access the Internet. Although IBM recommends that you install a level of IBM Storwize V7000 Unified code with a fix, you can avoid these vulnerabilities by not using Mozilla Firefox within your IBM Storwize V7000 Unified system to access the Internet.
