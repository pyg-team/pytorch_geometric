   Scenario:
   1. Start 32 bit CATIA.
   2. Select menu bar / Tools / Utility.>> The Batch Monitor
   appears.
   3. Double click the PrintBatch
   >> The PrintBatch panel appears.
   4. Click Add a document button and select the attached
   CATDrawing
   5. Click on the OK button.
   Problem: The printing is filled in black unexpectedly.
   .
   Expected Result:
   An unnecessarily filled area should not appear when using
   Print Batch. Print result should be the same as in case of
   interactive Printing.
