QUESTION
 How can I create a copy of a Dashboard page in JazzSM Dashboard Application Services Hub (DASH), and store it in a different location? 

ANSWER
In order to copy a page to a new location, you need to create first a clone copy of the page in question, and then move it to the folder you want to. 

Follow these steps: 

1) Edit the page you want a copy of. Click on "Page Actions" and select Edit. 

2) Then Click "Page Actions" again and select "Save As" 

[/support/docview.wss?uid=swg21699576&amp;aid=1]

3) Give this page a new name in the Page Settings

[/support/docview.wss?uid=swg21699576&aid=2] [/support/docview.wss?uid=swg21699576&aid=2]

4) Click on "Location" and select the new directory you want to place this page into:
