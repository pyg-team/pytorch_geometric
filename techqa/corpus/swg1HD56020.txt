ERROR DESCRIPTION
 *  Invalid effectivity can be created with Min date
   greater than Max date
   .
   Scenario :
   - Create a part PartA and set configurable
   - Create a second part PartB
   - Copy PartB and paste to PartA
   - Select Commit Modifications and choose Commit
   with Parameters
   - Click on Add Rule
   - Click on Enable Date
   - In the Min area, select from the calendar, any
   date in the future
   - In the Max area, leave today's date
   - Result in this example: August 17, 2006 / August
   11, 2006
   - Select Add Rule
   - Commit
   Problem : Effectivity on the link is invalid because
   Min Date is greater than Max date
   .
