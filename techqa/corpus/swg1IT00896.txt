LOCAL FIX
 *  STRRTC - 418004
   RJ/RJ
   Circumvention:
   Update to latest SEAS Build
   
   
    
   
   

PROBLEM SUMMARY
 *  Unable to launch the SEAS WebStart GUI when client browser is
   running under Java 7 U51 update or above.  Getting
   java.lang.SecurityException: Missing required Permissions
   
   
    
   
   

PROBLEM CONCLUSION
 *  Corrected the jar signing methods so that the SEAS WebStart
   will run with the correct permissions.
