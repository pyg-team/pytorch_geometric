Title: IBM Security Bulletin: IBM Client Application Access is affected by Open Source libpng Vulnerabilities - United States

Text:
 SECURITY BULLETIN

SUMMARY
 Open Source libpng library is vulnerable to a denial of service caused by a NULL pointer dereference in the png_set_text_2 function. ICAA uses Open Source libpng library. ICAA has addressed these vulnerabilities 

VULNERABILITY DETAILS
CVEID:CVE-2016-10087 [http://cve.mitre.org/cgi-bin/cvename.cgi?name=CVE-2016-10087]
DESCRIPTION:libpng is vulnerable to a denial of service, caused by a NULL pointer dereference in the png_set_text_2 function. By loading a text chunk into a png structure, removing the text, and then adding another text chunk to the structure, a remote attacker could exploit this vulnerability to cause a denial of service.
CVSS Base Score: 7.5
CVSS Temporal Score: Seehttps://exchange.xforce.ibmcloud.com/vulnerabilities/124207 [https://exchange.xforce.ibmcloud.com/vulnerabilities/124207]for the current score
CVSS Environmental Score*: Undefined
CVSS Vector: (CVSS:3.0/AV:N/AC:L/PR:N/UI:N/S:U/C:N/I:N/A:H)
