ERROR DESCRIPTION
 *  The customer has the Microsoft Office Excel Viewer
   installed on his machine and is able to open and

   However, the instantiation of catalog parts fails with
   the following message:
   .

   possible to instantiate the catalog parts.

LOCAL FIX

PROBLEM SUMMARY
 *  CATALOG PART INSTANTIATION NOT POSSIBLE WITH THE MICROSOFT OFFI
   CE EXCEL VIEWER
   The customer has the Microsoft Office Excel Viewer
   installed on his machine and is able to open and
   correctly view .xls design design tables.
   However, the instantiation of catalog parts fails with
   the following message:
   .
   Error
   New From Action failed.
   No Design Table with 'Part Number' column found
   .
   Note:
   If Microsoft Office is installed on the machine, it is
   possible to instantiate the catalog parts.
   .
   
   
    
   
   

PROBLEM CONCLUSION
 *  It is not possible to instantiate from catalogs
   CATParts that reference .xls design tables if only
   the Microsoft Excel Viewer or OpenOffice Viewer is
   installed.
   It is necessary to install Microsoft Excel to use such
   catalogs.
   Note:
   It is also possible to avoid this restriction by
   using .txt design tables instead of .xls design tables
   under windows.
   .
   THIS PROBLEM IS PERMANENT RESTRICTION
   .
