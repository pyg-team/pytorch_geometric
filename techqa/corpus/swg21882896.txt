Title: IBM TADDM Admin Password reset in LDAP - United States

Text:
TADDM; maintenance; discovery TECHNOTE (TROUBLESHOOTING)

PROBLEM(ABSTRACT)
 Need to understand the impact of changing the administrator password. Will it affect any settings or currently scheduled tasks in TADDM. 

RESOLVING THE PROBLEM
Password change for the admin will only affect customized scripts. 
Default TADDM functions will not be affected when changing the Administrators password
