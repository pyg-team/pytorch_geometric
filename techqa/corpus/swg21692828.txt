Title: IBM Security Bulletin: Security Vulnerabilities have been identified in multiple products that ship with IBM Predictive Maintenance (CVE-2014-3566, CVE-2014-6145, CVE-2014-1568, CVE-2014-4263, CVE-2012-5784, CVE-2014-3513, CVE-2014-3567, CVE-2014-3568) - United States

Text:
 SECURITY BULLETIN

SUMMARY
 IBM WebSphere Message Broker, IBM Cognos Business Intelligence Server, IBM InfoSphere Master Data Management and IBM Installation Manager are shipped as components of IBM Predictive Maintenance and Quality. Information about security vulnerabilities affecting IBM WebSphere Message Broker, IBM Cognos Business Intelligence, IBM InfoSphere Master Data Management, and IBM Installation Manager has been published in the following security bulletins. 

VULNERABILITY DETAILS
Please consult the security bulletin IBM Websphere Message Broker and IBM Integration Bus are affected by SSLv3 Vulnerability (CVE-2014-3566 and CVE-ID: CVE-2014-3568) [http://www-01.ibm.com/support/docview.wss?uid=swg21687678] for vulnerability details and information about fixes. 

Please consult the security bulletin Vulnerabilities in OpenSSL affect IBM InfoSphere Master Data Management (CVE-2014-3513, CVE-2014-3567) [https://www-304.ibm.com/support/docview.wss?uid=swg21690523] for vulnerability details and information about fixes. 

Please consult the security bulletin Vulnerability in SSLv3 affects IBM Installation Manager (CVE-2014-3566) [http://www-01.ibm.com/support/docview.wss?uid=swg21688724] for vulnerability details and information about fixes.

Please consult the security bulletin IBM Cognos Business Intelligence Server is affected by multiple vulnerabilities (CVE-2014-3566, CVE-2014-6145, CVE-2014-1568, CVE-2014-4263, CVE-2012-5784, CVE-2014-3513, CVE-2014-3567 and CVE-2014-3568). [http://www-01.ibm.com/support/docview.wss?uid=swg21692267] for vulnerability details and information about fixes.

AFFECTED PRODUCTS AND VERSIONS
 

Principal Product and Version(s) Affected Supporting Product and Version IBM Predictive Maintenance and Quality 1.0 WebSphere Message Broker 8.0.0.1 IBM Infosphere Master Data Management Collaboration Server V10.1.0 IBM Installation Manager 1.6.0 IBM Cognos Business Intelligence Server 10.2 [/support/docview.wss?uid=swg21692828&amp;aid=1] [/support/docview.wss?uid=swg21692828&amp;aid=1] IBM Predictive Maintenance and Quality 2.0 IBM Integration Bus 9.0.0.1 IBM InfoSphere Master Data Management Collaboration Server V11.0 IBM Installation Manager 1.6.2 IBM Cognos Business Intelligence Server 10.2.1.1 REMEDIATION/FIXES
Users of the affected principal product should apply the fixes for the affected supporting products and versions described in the security bulletins referenced above.
