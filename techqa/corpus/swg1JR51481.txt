APAR STATUS
 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  The WC database supports an SEO meta description up to 1024
   characters. However, Management Center supports only 254
   characters. Dataload works fine for meta descriptions greater
   than 254 characters, however when  a business user accesses a
   product in Management Center, an error message is displayed
   which causes the product to not load and therefore not be
   published.
   
   
    
   
   

LOCAL FIX
 *  Use SEO meta description less than 254 characters
   
   
    
   
   

PROBLEM SUMMARY
 *  USERS AFFECTED:
   Commerce on Cloud users who load products with SEO meta
   description greater than 1024 characters.
   
   PROBLEM ABSTRACT:
   Modieis the SEO metadesc field to accept more than 254
   characters in Management Center.
   
   BUSINESS IMPACT:
   Unable to load products with SEO meta description greater than
   1024 characters.
   
   RECOMMENDATION:
   
   
    
   
   

PROBLEM CONCLUSION
 *  Modified catalog common object definitions and content page
   object definition file to allow seometadesc property of maximum
   1024 characters.
