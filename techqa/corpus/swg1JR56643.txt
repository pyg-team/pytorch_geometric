APAR STATUS
 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  When you view the results of a saved search as a table, the
   content of the table cells can be too long to fit. In this case,
    the content is truncated and "..." are added.
   
   PRODUCTS AFFECTED
   IBM Business Process Manager (BPM) Advanced
   IBM BPM Standard
   IBM BPM Express
   
   
    
   
   

LOCAL FIX
 *  To see the full content, manually expand the column width.
   
   
    
   
   

PROBLEM SUMMARY
 *  No additional information is available.
   
   
    
   
   

PROBLEM CONCLUSION
 *  A fix is available in IBM BPM V8.5.7 cumulative fix 2016.06
   (http://www.ibm.com/support/docview.wss?uid=swg24042266) that
   adds a tool tip that, while you hover over the content of a
   cell, shows the full content.
   
   With IBM BPM V8.5.7 cumulative fix 2016.06 installed, search for
   JR56643 on Fix Central (http://www.ibm.com/support/fixcentral):
   
   1. Select IBM Business Process Manager with your edition from
   the product selector, the installed version to the fix pack
   level, and your platform, and then click Continue.
   2. Select APAR or SPR, enter JR56643, and click Continue.
   
   When you download fix packages, ensure that you also download
   the readme file for each fix. Review each readme file for
   additional installation instructions and information about the
   fix.
   
   
    
   
   

TEMPORARY FIX
 *  Not applicable
