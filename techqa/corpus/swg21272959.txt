ClearCase; Licensing; AIX; HP-UX; IRIX; Linux; Solaris; Windows; 7.0; 7.0.0.1; 7.0.1; 1272959 TECHNOTE (TROUBLESHOOTING)

PROBLEM(ABSTRACT)
 This technote explains why the error clearlicense: Error: Unable to find albd
clearlicense: Warning: Can't get credentials for use on host XXXX; using "nobody" might occur after upgrading to IBM® Rational® ClearCase® version 7.x. 

SYMPTOM
 

After upgrading to ClearCase version 7.x the following error is reported while attempting to run ClearCase on the upgraded host:

clearlicense: Error: Unable to find albd
clearlicense: Warning: Can't get credentials for use on host XXXX; using "nobody"



CAUSE
 

The ClearCase license server may be running ClearCase version 4.1 or earlier, which is not compatible with ClearCase version 7.x.


RESOLVING THE PROBLEM
Upgrade the license server to ClearCase version 7.x or migrate the license server to a ClearCase version 7.x machine.
