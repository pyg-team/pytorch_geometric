ERROR DESCRIPTION
 *  The TFIM CLI wsadmin commands don't work on WAS v7.x.
   
   The TFIM commands are not registered properly as command
   framework extensions under WebSphere version 7.0

   commands the following error will be displayed.
   
   wsadmin>$AdminTask help TFIMCommands

   nds"; exception information:

    help
   arguments: TFIMCommands
