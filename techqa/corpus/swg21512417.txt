Title: IBM SPSS Client error and Visual C++ Java Runtime error - United States

Text:
 TECHNOTE (TROUBLESHOOTING)

PROBLEM(ABSTRACT)
 I am installing Statistics 19.0 as network on Win7 64-bits OS. I am getting error "SPSS Client" and when I closed that error, I am getting "Visual C++ Java Runtime error and terminate the application" This is ONLY happen on Window 7 64-bits installing as network. I can't use the application. 
Is there a fix for this? Is this a defect? 

ENVIRONMENT
Win7 64-bits installing Network type

RESOLVING THE PROBLEM
Please do the following to correct your problem.


1. Navigate to the Statistics 19 installation folder which is C:\Program Files\IBM\SPSS\Statistics\19 

2. Find and modify spssprod.inf with Notepad 

3. Check and remove all the blank lines and saved the file 

 

Removing the blank lines should allow the application to launch.
