PREREQUISITES
To run IBM Rational Publishing Engine 1.1 you need: 

 * One of the following operating systems installed on your computer:
 *  Windows XP Professional with SP3 Windows Vista Enterprise Edition with SP1 Windows 2003 Server R2 32 bit RedHat Linux Enterprise Edition 5.0 32 bit and 5.3 32 bit 
 * The minimum system requirements for Microsoft Windows operating systems are:  * Processor: Pentium 500 Mhz
    * RAM: 512Mb
    * Virtual Memory: 1024Mb
    * Disk space (installation): 400Mb 
    * Java 6 or newer Runtime Environment in order to run. Java 6 update 10 or later is recommended. 
   
   
 * The minimum system requirements for RedHat Linux Enterprise Edition are: 
 *  Processor: Pentium 500 Mhz RAM: 512Mb Virtual Memory: 1024Mb Disk space (installation): 400Mb Java 6 or newer Runtime Environment in order to run. Java 6 update 10 or later is recommended. A screen resolution of 1600x1200 or higher is recommended for working with Document Studio
