Title: IBM Prerequisite Scanner product codes - United States

Text:
 PRODUCT DOCUMENTATION

ABSTRACT
 IBM® Prerequisite Scanner uses a multicharacter code, <tt>product_code</tt>, to identify the product, individual supported platform, and version of operating system. 

CONTENT


 The ips_root/codename.cfg file contains the name value pairs to represent the product code for the product, its supported platform, and the version of the operating system. 

The following table outlines the current set of predefined product codes. 

Restriction: IBM Tivoli® Monitoring and Tivoli Composite Application Manager have predefined product codes that Prerequisite Scanner considers as reserved. These codes must not be used as Prerequisite Scanner product codes unless they refer to their associated IBM Tivoli Monitoring and Tivoli Composite Application Manager agents. For more information about the product codes, see the ITM 6.X Product Codes Technote [http://www.ibm.com/support/docview.wss?rs=2366&context=SSZ8F3&dc=DB520&dc=DB560&uid=swg21265222&loc=en_US&cs=UTF-8&;lang=en&rss=ct2366tivoli ].
