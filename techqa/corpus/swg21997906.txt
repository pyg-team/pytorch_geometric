Title: IBM Security Bulletin: WebSphere Message Broker / IBM Integration Bus WEBUI does not set X-FRAME-OPTIONS (CVE-2016-9010) - United States

Text:
WMB IIB SECURITY BULLETIN

SUMMARY
 WebSphere Message Broker / IBM Integration Bus WEBUI does not set X-FRAME-OPTIONS 

VULNERABILITY DETAILS
CVEID: CVE-2016-9010 [http://cve.mitre.org/cgi-bin/cvename.cgi?name=CVE-2016-9010]
DESCRIPTION: IBM WebSphere Message Broker could allow a remote attacker to hijack the clicking action of the victim. By persuading a victim to visit a malicious web site, a remote attacker could exploit this vulnerability to hijack the victim's click actions and possibly launch further attacks against the victim.
CVSS Base Score: 6.1
CVSS Temporal Score: See https://exchange.xforce.ibmcloud.com/vulnerabilities/119398 [https://exchange.xforce.ibmcloud.com/vulnerabilities/119398] for the current score
CVSS Environmental Score*: Undefined
CVSS Vector: (CVSS:3.0/AV:N/AC:L/PR:N/UI:R/S:C/C:L/I:L/A:N)
