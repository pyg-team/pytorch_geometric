PROBLEM SUMMARY: 
Ship Java 8 SR5 FP15 for WebSphere Application Server traditional bundled Java 8 

USERS AFFECTED:
This combination of WebSphere Application Server and Java is available with fix pack 855.11 and later.

This i-fix includes two possible upgrade scenarios:
a) If applied to WebSphere 8.5.5.11 or later where bundled Java 8 is installed, it will upgrade your bundled Java 8 to SR5 FP15.
b) If applied to WebSphere 8.5.5.11 or later where the original Java 6 is installed, it will remove Java 6 and replace it with bundled Java 8 SR5 FP15. All profiles and WebSphere infrastructure will be upgraded to use bundled Java 8.

Linux PPC64 LE does not support bundled Java 8 at this time, since the existing bundled Java on that platform is Java 7.1, and it is not going out of service yet. Please continue to use the Java 8 extension offering for your Java 8 needs on this platform.

PROBLEM DESCRIPTION: 
Ship Java 8 SR5 FP15 for WebSphere Application Server traditional bundled Java 8 

RECOMMENDATION:
Apply this i-fix to your SDK to upgrade or install bundled Java 8 to Java 8 SR5 FP15.

NOTE:
This i-fix includes two possible upgrade scenarios:
a) If applied to WebSphere 8.5.5.11 or later where bundled Java 8 is installed, it will upgrade your bundled Java 8 to SR5 FP15.
b) If applied to WebSphere 8.5.5.11 or later where the original Java 6 is installed, it will remove Java 6, and replace it with Java 8 SR5 FP15. All profiles and WebSphere infrastructure will be updgraded to use bundled Java 8. 

 

NOTE:
This i-fix will back up, and then replace the WAS_HOME/java1.8_xx/jre/lib/security/java.security file. This file contains a security fix that is critical to keeping your environment secure. The back up file will be placed at WAS_HOME/java1.8_xx/jre/lib/security/java.security.bak. If you have previously updated this file, you will need to merge your changes into the new copy.

NOTE:
A corrected z/OS i-fix has been added on May 22, 2018.


PROBLEM CONCLUSION: 
Applying this apar will upgrade or install bundled Java 8 to Java 8 SR5 FP15 (Oracle 1.8.0_u171 ).

The fixes in this Java SDK are found here:
https://www.ibm.com/developerworks/java/jdk/fixes/8/#SR5FP515 [https://www.ibm.com/developerworks/java/jdk/fixes/8/#SR5FP515]

When downloading the fix from fix central, please also download the readme.txt with instructions for installation.

This Java SDK level was minimally tested with WebSphere 8.5.5.12.
