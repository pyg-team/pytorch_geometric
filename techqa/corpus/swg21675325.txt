Title: IBM Unexpected WebSphere Application Server warnings about expired LPTA tokens and asynchronous processing can exist in the SystemOut.log file for the Process Center in IBM Business Process Manager (BPM) Advanced - United States

Text:
ibpma855relnotes; LTPA; asynchronous servlet; Process Designer; SECJ0371W; SRVE8092W TECHNOTE (TROUBLESHOOTING)

PROBLEM(ABSTRACT)
 In IBM Business Process Manager (BPM) Advanced, if the Process Designer web editor is open for a long period of time, you might notice unexpected messages in the SystemOut.log file for the Process Center. For eample, you might see 0000035e, SRVE8092W, 000001b0, or SECJ0371W messages. 

SYMPTOM
The following text displays with the previously mentioned error codes: 

0000035e async W com.ibm.ws.webcontainer.async.ListenerHelper _invokeAsyncErrorHandling 
SRVE8092W: An exception occurred invoking the async error mechanism.
000001b0 LTPAServerObj W
SECJ0371W: Validation of the LTPA token failed because the token expired

RESOLVING THE PROBLEM
These WebSphere Application Server messages are written to the SystemOut.log file for the Process Center in situations where the Process Designer web editor has been running for a long period of time. They do not affect any functionality and can be safely ignored.
