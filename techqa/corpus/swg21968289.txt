Title: IBM Security Bulletin: Security vulnerabilities have been identified in IBM WebSphere Application Server shipped with IBM WebSphere Portal - United States

Text:
 SECURITY BULLETIN

SUMMARY
 IBM WebSphere Application Server is shipped as a component of IBM WebSphere Portal. Information about security vulnerabilities affecting IBM WebSphere Application Server has been published in security bulletins. Vulnerablities affecting IBM HTTP Server have also been puslished in security bulletins. 

VULNERABILITY DETAILS
Please consult the security bulletins for IBM WebSphere Application Server published at: 

 * Security Bulletin: Multiple Security Vulnerabilities fixed in IBM WebSphere Application Server 8.5.5.7 [http://www-01.ibm.com/support/docview.wss?uid=swg21965444] 
 * Security Bulletin: Vulnerability with Java Portlet Specification JSR 286 may affect WebSphere Application Server (CVE-2015-1926) [http://www-01.ibm.com/support/docview.wss?uid=swg21962107]


Please consult the security bulletins for IBM HTTP Server published at: 
 * Security Bulletin: Denial of service may affect IBM HTTP Server (CVE-2015-1283) [http://www-01.ibm.com/support/docview.wss?uid=swg21964428] [http://www-01.ibm.com/support/docview.wss?uid=swg21963362] 
 * Security Bulletin: Denial of service may affect IBM HTTP Server (CVE-2015-1788) [http://www-01.ibm.com/support/docview.wss?uid=swg21963362] [http://www-01.ibm.com/support/docview.wss?uid=swg21959081] 
 * Security Bulletin: Potential denial of service may affect IBM HTTP Server on Windows (CVE-2015-1829) [http://www-01.ibm.com/support/docview.wss?uid=swg21959081] 
 * Security Bulletin: HTTP Request smuggling vulnerability may affect IBM HTTP Server (CVE-2015-3183) [http://www-01.ibm.com/support/docview.wss?uid=swg21963361] [http://www-01.ibm.com/support/docview.wss?uid=swg21965419] 
 * Security Bulletin: Stack Buffer overflow may affect IBM HTTP Server (CVE-2015-4947) [http://www-01.ibm.com/support/docview.wss?uid=swg21965419] [/support/docview.wss?uid=swg21968289&amp;aid=1] [/support/docview.wss?uid=swg21968289&amp;aid=1]

 IBM strongly suggests that all System z customers be subscribed to the System z Security Portal to receive the latest critical System z security and integrity service.
