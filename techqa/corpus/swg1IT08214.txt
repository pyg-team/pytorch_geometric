PROBLEM SUMMARY
 *  ****************************************************************
   USERS AFFECTED:
   All users of IBM Integration Bus V9.0 serializing a model in
   DFDL that includes groups or choices.
   
   
   Platforms affected:
   z/OS, MultiPlatform
   
   ****************************************************************
   PROBLEM DESCRIPTION:
   Intermittent abends can occur in the Integration Node during
   DFDL serialization. The abends have the following traceback
   stack:
   
   ProcessingError_setValues
   @77@SequenceGroupItem_processValue
   DFDLUnparserImpl_elementStringValue
   DFDLSerializer_elementStringValue
   ImbDFDLWriter::writeElement(ImbSyntaxElement*)
   ImbDFDLWriter::writeElement(ImbSyntaxElement*)
   ImbDFDLWriter::writeElement(ImbSyntaxElement*)
   ImbDFDLWriter::writeRootElement(ImbSyntaxElement*)
   ImbDFDLWriter::writeAll(bool)
   ImbDFDLParser::refreshBitStreamFromElementsInner(const ImbMe
   
   There are a number of resource name changes between WebSphere
   Message Broker and IBM Integration Bus Version 9.0. For details
   visit
   http://www-01.ibm.com/support/knowledgecenter/SSMKHH_9.0.0/com.i
   bm.etools.mft.doc/bb23814_.htm [http://www-01.ibm.com/support/knowledgecenter/SSMKHH_9.0.0/com.ibm.etools.mft.doc/bb23814_.htm]
