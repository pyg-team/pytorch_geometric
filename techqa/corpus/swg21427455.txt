RESOLVING THE PROBLEM
This is caused due to the missing hidden page: 

com.ibm.wps.hiddenpage.wcm.control.Authoring_Portlet in the Virtual Portal. To resolve this issue, please follow these steps:
1) Login to the base portal
2) Go into Administration
3) Click Manage Pages
4) Select Search by: Unique name contains
5) Enter Search: com.ibm.wps.hiddenpage.wcm.Authoring_Portlet
6) Click the export button to the right and save the xml export
7) Login to the virtual portal
8) Go into Administration
9) Click Import XML under Portal Settings
10) Import the XML from the base portal
