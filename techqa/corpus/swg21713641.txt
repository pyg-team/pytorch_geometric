Title: IBM Security Bulletin: IBM License Metric Tool is vulnerable to sensitive information exposure (CVE-2014-4776) - United States

Text:
PSIRT SECURITY BULLETIN

SUMMARY
 The IBM License Metric Tool contains a vulnerability that can expose sensitive information. 

VULNERABILITY DETAILS

CVE: CVE-2014-4776 [http://cve.mitre.org/cgi-bin/cvename.cgi?name=CVE-2014-4776]

Description: IBM License Metric Tool could allow a remote attacker to obtain sensitive information, caused by the lack of an off autocomplete attribute for authentication fields within the tool. An attacker could exploit this vulnerability to obtain sensitive information.

CVSS Base Score: 2.1
CVSS Temporal Score: See https://exchange.xforce.ibmcloud.com/vulnerabilities/94918 [https://exchange.xforce.ibmcloud.com/vulnerabilities/94918] for the current score
CVSS Environmental Score*: Undefined
CVSS Vector: (AV:L/AC:L/Au:N/C:P/I:N/A:N)

AFFECTED PRODUCTS AND VERSIONS
IBM License Metric Tool v9

REMEDIATION/FIXES
Upgrade to version 9.1.0.2 or later: 

 * In IBM Endpoint Manager console, expand IBM License Reporting node under Sites node in the tree panel. 
 * Click Fixlets and Tasks node. Fixlets and Tasks panel will be displayed on the right. 
 * In the Fixlets and Tasks panel locate Upgrade to the newest version of IBM License Metric Tool 9.x fixlet and run it against the computer that hosts your IBM License Metric Tool server.


Note: In an airgapped environment, you have to run BESAirgapTool and BESDownloadCacher first in order to update your site. WORKAROUNDS AND MITIGATIONS
None
