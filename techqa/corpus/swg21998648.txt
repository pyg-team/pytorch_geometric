Title: IBM Security Bulletin: IBM MQ and IBM WebSphere MQ invalid requests could cause denial of service to MQXR listener (CVE-2016-8986) - United States

Text:
 SECURITY BULLETIN

SUMMARY
 Invalid HTTP requests could cause a denial of service to the IBM MQ MQXR listener. 

VULNERABILITY DETAILS
CVEID: CVE-2016-8986 [http://cve.mitre.org/cgi-bin/cvename.cgi?name=CVE-2016-8986]
DESCRIPTION: IBM WebSphere MQ could allow an authenticated user with access to the queue manager to bring down MQ channels using specially crafted HTTP requests.
CVSS Base Score: 4.3
CVSS Temporal Score: See https://exchange.xforce.ibmcloud.com/vulnerabilities/118981 [https://exchange.xforce.ibmcloud.com/vulnerabilities/118981] for the current score
CVSS Environmental Score*: Undefined
CVSS Vector: (CVSS:3.0/AV:N/AC:L/PR:L/UI:N/S:U/C:N/I:N/A:L) 

AFFECTED PRODUCTS AND VERSIONS
IBM WebSphere MQ V7.5 

IBM WebSphere MQ 7.5.0.0 - 7.5.0.7 maintenance levels 

IBM MQ V8.0 

IBM MQ 8.0.0.0 - 8.0.0.5 maintenance levels 

IBM MQ V9 CD 

IBM MQ 9.0.1 only 

IBM MQ V9 LTS 

IBM MQ 9.0.0.0 only

REMEDIATION/FIXES
IBM WebSphere MQ V7.5 

Apply 7.5.0.8 [http://www-01.ibm.com/support/docview.wss?uid=swg22005413] maintenance level 

IBM MQ V8.0 

Apply 8.0.0.6 [http://www-01.ibm.com/support/docview.wss?uid=swg21995100] maintenance level

IBM MQ V9 LTS 

Apply Fix Pack 9.0.0.1 [http://www-01.ibm.com/support/docview.wss?uid=swg27048059]
