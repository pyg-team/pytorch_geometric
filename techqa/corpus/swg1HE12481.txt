ERROR DESCRIPTION
 *  Emails appear in blacklist due to the following error:
   AFUL0009E: An unrecoverable error has occurred: This message
   could not be stubbed due to an unsupported item type, only
   RichText items are supported for stubbing: Field length stored
   in document is incorrect
   
   
    
   
   

LOCAL FIX
 *  N/A
   
   
    
   
   

PROBLEM SUMMARY
 *  If Lotus Domino email documents contains large portion in rich
   text body, IBM Content Collector fails to stub them and added
   these documents to the IBM Content Collector blacklist. The
   following
     error message is written to the Email Connector log file:
     [AFUL0009E: An unrecoverable error has occurred: This message
   could not be stubbed due to an unsupported item type, only
   RichText items are supported for stubbing: Field length stored
   in document is incorrect ("<DominoServer>\<Mailfile>|<Document
   UNID>") [Origin: cas_domino_if\lnapi_base_stub.cpp:329]]
   
   
    
   
   

PROBLEM CONCLUSION
 *  Starting with IBM Content Collector 4.0.1.0 Fix Pack 2, the
   memory issue that caused the Email Connector to fail to stub
   these documents and add them to the blacklist has been fixed.
   
   
    
   
   

TEMPORARY FIX
 *  Fixed in 4.0.1.2-IBM-ICC-FP002
