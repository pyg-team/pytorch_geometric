Title: IBM FileNet P8 Platform Version 5.2 documentation updates for installable information center - United States

Text:
 DOWNLOADABLE FILES

ABSTRACT
 This document provides updates for your installed IBM® FileNet® P8 Platform V5.2 information center. These updates have already been applied to the ibm.com information center. 

DOWNLOAD DESCRIPTION
The documentation update files available from this document include updates from several documentation refreshes as well as several security patches.


29 November 2013 update:

The update includes: 

 * Documentation for IBM Enterprise Records Version 5.1.2, available as a separate download (p8docs520_ier512_11_29_2013.zip).


25 October 2013 update: 

The update includes:  * Updated IBM System Dashboard for Enterprise Content Management documentation. 
 * Updates for new and changed functionality included in Content Engine Platform 5.2.0.1 and 5.2.0.2 fix packs. 
 * Fixes for known documentation issues and other errors.


3 April 2013 update: 

The update provides translated documentation for IBM FileNet Workplace XT, IBM FileNet Integration for Microsoft Office, and IBM FileNet Connector for Microsoft SharePoint. 

20 January 2014 update:  * Fix for cross-site scripting vulnerability in the IBM FileNet P8 Platform Documentation Installable Info Center (see the security bulletin at http://www.ibm.com/support/docview.wss?uid=swg21662360 [http://www.ibm.com/support/docview.wss?uid=swg21662360] for more information) 

INSTALLATION INSTRUCTIONS
To install the documentation updates or the cross-site scripting vulnerability fix, extract the files from your information center WAR file and add the new plug-ins to the information center structure. Re-create the WAR file, clear the information center cache, and redeploy your information center. 

Security patch installation: 

 1. Expand the existing p8docs.war file to a working directory. 
 2. Download the security patch using the link below. 
 3. Unzip the security patch and copy the resulting files to the directory where you expanded p8docs.war. 
 4. Re-create the p8docs.war file. 
 5. Replace the original p8docs.war file with the one you created in the previous step.
