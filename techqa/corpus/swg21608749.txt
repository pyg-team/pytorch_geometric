Title: IBM A multi-object store search might return unexpected results - United States

Text:
security; search; object store TECHNOTE (TROUBLESHOOTING)

PROBLEM(ABSTRACT)
 Under certain circumstances, a multi-object store search might yield unexpected results by returning a result set that includes the properties of documents for which the user does not have access rights.

This issue is corrected in 5.0.0.1-P8CE-FP001 (and higher) and 5.1.0.1-P8CE-FP001 (and higher). It is mandatory that these fix pack updates be installed.

This issue does not occur in any release before Content Engine (CE) 5.0 and does not apply to single object store searches. 

RESOLVING THE PROBLEM
Under certain circumstances, a multi-object store search might yield unexpected results by returning a result set that includes the properties of documents for which the user does not have access rights.
Note that the user cannot access the document content and cannot change the returned document properties.

This issue affects the following IBM FileNet P8 Content Engine (CE) product components:

5.0.0.0-P8CE at the GA base level
5.1.0.0-P8CE at the GA base level

5.0.0.1-P8CE-FP001 (or higher) and 5.1.0.1-P8CE-FP001 (or higher) address this issue and are mandatory updates for all IBM FileNet Content Engine 5.0.0 and 5.1.0 environments.

These fix packs are already available on Fix Central. Please follow the standard procedure to download the mandatory Fix Pack required for your environment.

If you have any questions, please contact the IBM Response Center at 1-800-IBM-SERV.
