Title: IBM Security Bulletin: TLS padding vulnerability affects IBM HTTP Server shipped with IBM WebSphere Portal (CVE-2014-8730) - United States

Text:
 SECURITY BULLETIN

SUMMARY
 IBM HTTP Server is shipped as a component of IBM WebSphere Portal. Information about a security vulnerability affecting IBM HTTP Server has been published in a security bulletin. 

VULNERABILITY DETAILS
Please consult the security bulletin TLS padding vulnerability affects IBM HTTP Server (CVE-2014-8730) [http://www-01.ibm.com/support/docview.wss?uid=swg21692502] for vulnerability details. 

AFFECTED PRODUCTS AND VERSIONS
 

Principal Product and Version(s) Affected Supporting Product and Version IBM WebSphere Portal 6.1, 7.0, 8.0, 8.5 IBM HTTP Server 6.0.2, 6.1, 7.0, 8.0, 8.5, 8.5.5 If you are using an unsupported version, IBM strongly recommends to upgrade to a supported version.
