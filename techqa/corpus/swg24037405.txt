ABSTRACT
 This document describes how to download the Oracle E-Business portion of the IBM® InfoSphere® Optim Solution for Oracle Applications v11.3. 

DOWNLOAD DESCRIPTION
You can download the Oracle E-Business Suite with Data Masking portion of the IBM InfoSphere Optim Solution for Oracle Applications using the Passport Advantage® website. 

The Optim Test Data Management Solution for Oracle E-Business Suite helps you to build realistic and right sized non production databases, such as testing, development, and training databases. Using it, you can extract a selected subset of data from your source database and insert the data into your target database. The Test Data Management Solution for Oracle E-Business Suite can be used with the data masking option for greater security. The data masking option substitutes sensitive employee data with fictionalized data that is contextually accurate and usable. 

The Optim Test Data Management Solution for Oracle E-Business Suite with Data Masking Option consists of the Optim Test Data Management Solution, Data Privacy tables, and definitions needed to perform test data management and data masking processing. The data masking option also includes an SQL script used to create additional objects needed to mask data.

PREREQUISITES
You must install IBM InfoSphere Optim Solution v11.3 before you install the Optim Test Data Management Solution for Oracle E-Business Suite with Data Masking Option. 

For information about hardware and software requirements for the InfoSphere Optim Test Data Management Solution for Oracle E-Business Suite with Data Masking Option, see the detailed system requirements.
