Title: IBM Support for HFM 11.1.2.2 and HFM 11.1.2.3 32-bit client in IIS 7 and later - United States

Text:
 PRODUCT DOCUMENTATION

ABSTRACT
 In Cognos Disclosure Management 10.2.2 Fix Pack 1, support has been added for the HFM 11.1.2.2 and HFM 11.1.2.3 32-bit clients. Through these clients, Cognos Disclosure Management can connect to both the HFM 11.1.2.3 and HFM 11.1.2.2 servers. 

CONTENT
Installing the Cognos Disclosure Management 10.2.2 Fix Pack 1 after HFM 11.1.2.2 or HFM 11.1.2.3 client installation 

Ensure the following services and features are installed and enabled before applying Fix Pack 1.

Procedure
1. Click Start > All Programs > Administrative Tools > Server Manager.
2. In the Server Manager window, select Features. The Feature Summary View is displayed.
3. Click Add Features. The Add Features Wizard opens.
4. Click Next to select features to install.
5. Ensure .NET Framework Features is selected.
6. Expand .NET Framework Features, then expand WCF Activation and ensure that HTTP Activation and Non-HTTP Activation are selected.
7. Ensure that the Windows Process Activation Service feature is selected.
8. In the Server Manager window, select Services.
a) Ensure the Windows Process Activation Service is enabled.
b) Ensure the Net.Pipe Listener Adapter service is enabled.
9. In the Control Panel, click User Accounts > Change User Account Control settings and drag the slider to Never notify.

Installing the Cognos Disclosure Management 10.2.2 Fix Pack 1 before HFM 11.1.2.2 or HFM 11.1.2.3 client installation

Before you begin
Your User Account Control (UAC) must be set to Never notify on your server.

Ensure the following services and features are installed and enabled.

Procedure
1. Click Start > All Programs > Administrative Tools > Server Manager.
2. In the Server Manager window, select Features. The Feature Summary View is displayed.
3. Click Add Features. The Add Features Wizard opens.
4. Click Next to select features to install.
5. Ensure .NET Framework Features is selected.
6. Expand .NET Framework Features, then expand WCF Activation and ensure that HTTP Activation and Non-HTTP Activation are selected.
7. Ensure that the Windows Process Activation Service feature is selected.
8. In the Server Manager window, select Services.
a) Ensure the Windows Process Activation Service is enabled.
b) Ensure the Net.Pipe Listener Adapter service is enabled.
9. In the Control Panel, click User Accounts > Change User Account Control settings and drag the slider to Never notify.

After installing the HFM 11.1.2.2 or HFM 11.1.2.3 client, the Ibm.CognosDM.HfmServices.Deploy.exe tool from the Tools\HfmDeploy folder in the Cognos Disclosure Management installation folder must be run with the following parameters: 

 * /basePath: path to Cognos Disclosure Management installation folder 
 * /baseSiteName: name of the main Cognos Disclosure Management site in IIS 
 * /baseApplicationName: name of the Cognos Disclosure Management application in IIS 
 * /baseAppPoolName: name of the Cognos Disclosure Management application pool in IIS
