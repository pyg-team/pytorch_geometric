Title: IBM InfoSphere InformationServer 11.5 now supports Oracle Automatic Storage Management (ASM) and Oracle Advanced Security Option (ASO) - United States

Text:
Oracle ASM; Oracle ASO; InformationServer 11.5 TECHNOTE (FAQ)

QUESTION
 Oracle Automatic Storage Management support for InformationServer 

CAUSE
 Oracle Automatic Storage Management Support requires Modified Metadata Repository and Information Analyzer analysis Database Creation Scripts

ANSWER
InfoSphere InformationServer 11.5 now supports Oracle Automatic Storage Management and Oracle Advanced Security Option 

 * InformationServer Metadata Repository , Information Analyzer analysis, DSODB, Standardization Rules Designer DB are now supported to use Oracle Automatic Storage Management as one of the storage options 
 * Customer Installing InformationServer 11.5 with Oracle Automatic Storage Management Support should use the new Metadata Repository & Information Analyzer analysis database database creation scripts provided below


Windows: 
1)Metadata repository
