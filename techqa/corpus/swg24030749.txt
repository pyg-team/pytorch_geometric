ABSTRACT
 This document describes how to download IBM WebSphere MQ Hypervisor for AIX V7.0.1 eAssembly images from the Passport Advantage website. 

DOWNLOAD DESCRIPTION
This document was created during March 2012 for V7.0.1

Terminology

WebSphere MQ Hypervisor for AIX V7.0.1 is supplied in three eAssemblies. These consist of an archive file containing the Quick Start Guide, an archive file containing the User Guide PDF, and an Open Virtualization Format virtual machine image for use with an IBM Workload Deployer. 
Related product documentation

WebSphere MQ Hypervisor for AIX contains a guest operating system and WebSphere MQ. The most up to date information about WebSphere MQ can be found in the US English WebSphere MQ V7.0.1 readme file at: http://www.ibm.com/support/docview.wss?rs=171&uid=swg27006097 [http://www.ibm.com/support/docview.wss?rs=171&uid=swg27006097]

Downloading the relevant eAssembly images from the Passport Advantage Xtreme Leverage website  1. From the eAssemblies table below, decide which of the eAssemblies you want to download.
 2. Go to the Download package table below and follow the link to the Passport Advantage Xtreme Leverage website.
 3. Sign in to download all of the images that you need.
 4. Unpack each of the images onto your system.
