Title: IBM GSKit V7 - Install, Uninstall and Upgrade instructions for AIX systems - United States

Text:
gskit; install; uninstall; upgrade; aix; gsk7 TECHNOTE (FAQ)

QUESTION
 Procedures to install, uninstall or upgrade GSKit V7 for Tivoli Directory Server on AIX systems. 

CAUSE
GSKit V7 provided with different versions of Tivoli Directory server needs to be upgraded as part of periodic preventive maintenance.

ANSWER




This document provides install, uninstall and upgrade instructions for GSKit V7 on Tivoli Directory Server supported AIX systems. 

Refer Recommended Fixes [http://www.ibm.com/support/docview.wss?uid=swg27009778] to find the latest recommended level of GSKit for a given version of Tivoli Directory Server.

Directory Server provided GSKit V7 latest fix levels are now available on Fix Central for entitled customers. Download GSKit V7 (7.0.4.35 or later fix levels) from Fix Central [http://www-933.ibm.com/support/fixcentral/swg/selectFixes?product=ibm/Tivoli/IBM+Global+Security+Kit&release=All&platform=All&function=fixId&fixids=7.0.4.*]. 

Also refer to GSKit V7- Instructions to find current installed version details [http://www.ibm.com/support/docview.wss?uid=swg21443435].
