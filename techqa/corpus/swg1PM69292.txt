APAR STATUS
 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  You would see an IllegalStateException in an FFDC during
   server shutdown when running a server with Enterprise Bundle
   Archive (EBA) applications installed.
