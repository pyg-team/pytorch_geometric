APAR STATUS
 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  For search filters in the Add Employee panel and People in
   Scope page, Hold Notice Recipients page or Release Notice
   Recipients page, user is not able to use the Enter key to
   submit a search. User can only click the search button to
   submit the search
   
   
    
   
   

LOCAL FIX
 *  Click search button
   
   
    
   
   

PROBLEM SUMMARY
 *  For search filters in the Add Employee panel and People in Scope
    page, Hold Notice Recipients page or Release Notice Recipients
   page, user was not able to use the Enter key to submit a search.
   User could only click the search button to submit the search.
   
   
    
   
   

PROBLEM CONCLUSION
 *  This issue is fixed by enabling the Enter key to submit the
   search in search filters of Atlas pages.
   
   The fix is available from fix central in Atlas 6.0.3.2 iFix 2
