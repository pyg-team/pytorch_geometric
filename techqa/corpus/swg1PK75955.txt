PROBLEM SUMMARY
 *  JVM Crashes with SIGSEGV with NX feature enabled
   
   
    
   
   

PROBLEM CONCLUSION
 *  This defect will be fixed in:
   1.4.2 SR13
   .
   NX feature inhibites code lying in data segment from execution.
   JIT library had some such code, in conjunction with an issue
   pertaining to execution of static initialization of classes from
   JITted methods caused the above crash.
   The issue is fixed in SR13.
   .
   To obtain the fix:
   Install build 20081225 or later
