APAR STATUS
 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  In January 2014 (or shortly thereafter), Oracle and IBM will be
   releasing new JREs.
   The warnings concerning manifest attributes that currently
   appear in the java console will turn into hard errors and
   applets will cease to run.
   
   In addition, the applet signing should now be done with a newer
   JRE that uses SHA-2 digests rather than the older SHA-1 digests.
   
   
    
   
   

LOCAL FIX

PROBLEM SUMMARY
 *  ****************************************************************
   * USERS AFFECTED:                                              *
   * 4.5.1.7-P8CE-FP007 and earlier systems                       *
   ****************************************************************
   * PROBLEM DESCRIPTION:                                         *
   * RE 1.7.0_051 requires new certificates.  Without these new   *
   * certificates, any clients will get a security failure error. *
   ****************************************************************
   * RECOMMENDATION:                                              *
   * Upgrade to 4.5.1.7-P8CE-IF002 or 4.5.1.8-P8CE-FP008 and      *
   * higher.                                                      *
   ****************************************************************
   
   
    
   
   

PROBLEM CONCLUSION
 *  JRE 1.7.0_051 requires new certificates so upgrade to
   4.5.1.7-P8CE-IF002 or 4.5.1.8-P8CE-FP008 and higher.
