 TECHNOTE (TROUBLESHOOTING)

PROBLEM(ABSTRACT)
 If get "ComputerRollup: Failed" error in IEM import log, we need try to reset IEM SCA (IBM Endpoint Manager Security Compliance Analytics) datasource sequence first. How to reset IEM SCA datasource sequence?

Error in SCA import log:

INFO: ETL warehouse task: Warehouse: SCM::ComputerRollup: Start
INFO: ETL warehouse task: Warehouse: SCM::ComputerRollup: Failed
ERROR: Sequel::CheckConstraintViolation: Java::ComMicrosoftSqlserverJdbc::SQLServerException: The INSERT statement conflicted with the CHECK constraint "CK__computer_rollups__5A1A5A11". The conflict occurred in database "BFAnalytics", table "scm.computer_rollups".
