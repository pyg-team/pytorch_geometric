ERROR DESCRIPTION
 *  Environment:
   ITM Microsoft Exchange Server Agent V6.2.0.
   All versions of MS Exchange Server
   All Windows versions.
   .
   Problem Description:
   1. IF MS Exchange agent has been configured for the mailbox
   reachability attribute group, in some environment
   kexsendemail.exe fails with following error message in its log
   file.  <CreateOneOff> failed. 0x80070057.
   2. The agent does not display data for the same attribute groups
   when it encounters timeout while collecting the reachability
   data.
   3. Kexsendemial.exe crashes while freeing some memory variables
   and
   the agent stops after this crash.
   .
   Detailed Recreation Procedure:
   1. Configure the agent to Enable the mailbox reachability.
   2. Start the Agent.
   .
   Note:
   This problem was never recreated at L3 setup and issue was
   resolved directly using customer's environment. The agent does
   not show the data for this attribute in case of request gets
   timeout.
   .
   Related Files and Output:
   We can also Observe the following output in the log file at the
   location <%CANDLE_HOME%>\IBM\ITM\TMAITM6\logs.
   .
   1.Log file created for kexsendemail.exe named *kexsendemail*.log
   this log file will contain following error message:
   <CreateOneOff> failed. 0x80070057
   2. Log file created for kexcma.exe named *kexcma*.log this log
   file would print the timeout message for reachability data
   collection.
   Approver: Steven Barnett
