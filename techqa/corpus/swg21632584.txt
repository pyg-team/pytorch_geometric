QUESTION
 Does Extended agent for Oracle database display SQL statements in Tivoli Enterprise Portal (TEP)? 

ANSWER
The attribute group RDB SQL TEXT contains the SQL Text attribute to deliver the SQL statement text, up to 4000 characters in length. 

SQL statement text is displayed in TEP in the linked workspaces from the Session and Top SQL workspaces:

1) Select the Session workspace from the TEP navigator. Use workspace link SQL Text from a row in the Session Detail table view to display SQL Text workspace.

2) Select Top SQL workspace in TEP navigator. Follow workspace link "SQL Text and Execution Plan" from Top SQL By Elapsed Time table view to display SQL Text.
