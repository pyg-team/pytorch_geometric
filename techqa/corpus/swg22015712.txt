Title: IBM Security Bulletin: Multiple security vulnerabilities affect the Report Builder that is shipped with Jazz Reporting Service (CVE-2017-1750, CVE-2018-1363) - United States

Text:
CVE-2017-1750; CVE-2018-1363; JRS SECURITY BULLETIN

SUMMARY
 There are multiple security vulnerabilities in the Report Builder shipped with Jazz Reporting Service. 

VULNERABILITY DETAILS
CVEID:CVE-2017-1750 [http://cve.mitre.org/cgi-bin/cvename.cgi?name=CVE-2017-1750]
DESCRIPTION:IBM Jazz Reporting Service (JRS) is vulnerable to cross-site scripting. This vulnerability allows users to embed arbitrary JavaScript code in the Web UI thus altering the intended functionality potentially leading to credentials disclosure within a trusted session.
CVSS Base Score: 5.4
CVSS Temporal Score: Seehttps://exchange.xforce.ibmcloud.com/vulnerabilities/135523 [https://exchange.xforce.ibmcloud.com/vulnerabilities/135523]for the current score
CVSS Environmental Score*: Undefined
CVSS Vector: (CVSS:3.0/AV:N/AC:L/PR:L/UI:R/S:C/C:L/I:L/A:N)
CVEID:CVE-2018-1363 [http://cve.mitre.org/cgi-bin/cvename.cgi?name=CVE-2018-1363]
DESCRIPTION:IBM Jazz Reporting Service (JRS) is vulnerable to cross-site scripting. This vulnerability allows users to embed arbitrary JavaScript code in the Web UI thus altering the intended functionality potentially leading to credentials disclosure within a trusted session.
CVSS Base Score: 5.4
CVSS Temporal Score: Seehttps://exchange.xforce.ibmcloud.com/vulnerabilities/137448 [https://exchange.xforce.ibmcloud.com/vulnerabilities/137448]for the current score
CVSS Environmental Score*: Undefined
CVSS Vector: (CVSS:3.0/AV:N/AC:L/PR:L/UI:R/S:C/C:L/I:L/A:N) 

AFFECTED PRODUCTS AND VERSIONS
Jazz Reporting Service 5.0, 5.0.1, 5.0.2, 6.0, 6.0.1, 6.0.2, 6.0.3, 6.0.4 and 6.0.5.

REMEDIATION/FIXES

Apply the recommended fixes to all affected versions of Jazz Reporting Service.



Product VRMF Remediation Jazz Reporting Service 5.0, 5.0.1, 5.0.2 Download the interim fix 5.0.2-Rational-CLM-ifix025 (or later) package from the 5.0.2 iFix release site [http://www-933.ibm.com/support/fixcentral/swg/selectFixes?parent=ibm~Rational&product=ibm/Rational/Rational+Collaborative+Lifecycle+Management+Solution&release=5.0.2&platform=All&function=all] and follow the instructions inside the package for patch application. Jazz Reporting Service 6.0, 6.0.1, 6.0.2 Download the interim fix 6.0.2-Rational-CLM-ifix016 (or later) package from the 6.0.2 iFix release site [http://www-933.ibm.com/support/fixcentral/swg/selectFixes?parent=ibm~Rational&product=ibm/Rational/Rational+Collaborative+Lifecycle+Management+Solution&release=6.0.2&platform=All&function=all] and follow the instructions inside the package for patch application. Jazz Reporting Service 6.0.3, 6.0.4, 6.0.5 Download the interim fix 6.0.5-Rational-CLM-ifix004 (or later) package from the 6.0.4 iFix release site [https://www-945.ibm.com/support/fixcentral/swg/selectFixes?parent=ibm~Rational&product=ibm%2FRational%2FRational+Collaborative+Lifecycle+Management+Solution&release=6.0.4&platform=All&function=all] and follow the instructions inside the package for patch application.
