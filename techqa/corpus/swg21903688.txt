Title: IBM JazzSM Dash ATKRST208E widget error - United States

Text:
DASHL2; DASHL2CONF; Dashboard Application Services Hub; DASH; JazzSM; ATKRST208E TECHNOTE (TROUBLESHOOTING)

PROBLEM(ABSTRACT)
 A newly configured DASH widget returns the following error when the widget is loaded:

ATKRST208E The remote provider is unable to communicate with the server... Reason: java.net.ConnectException:

Connection timed out: no further information'. 

CAUSE
The widget in question was configured with the "Update context based on selections made in other portlets" option set, but there were no other widgets on the page.

DIAGNOSING THE PROBLEM
If there are no other widgets on the page, check the widget properties to see if the "Update context based on selections made in other portlets" is set.

RESOLVING THE PROBLEM
Uncheck the "Update context based on selections made in other portlets" option in the widget properties.
