QUESTION
 How to set Trace for ITCAM Agent for Data Power Appliance ? 

CAUSE
Mostly needed for debugging purpose.

ANSWER
 

[/support/docview.wss?uid=swg21681635&amp;aid=1] Setting and Collecting Trace log Windows 1. edit <ITM_HOME>\TMAITM6\KBNENV_<instanceName> 
(ex:c:\IBM\ITM\TMAITM6\KBNENV_dpl2virt2a)

2. On "KBNENV_<instanceName>" file, locate "JAVA_TRACE_LEVEL"
3. Change the value to ERROR or DEBUG_MIN
depends on your need.
(ex: JAVA_TRACE_LEVEL=ERROR, or
JAVA_TRACE_LEVEL=DEBUG_MIN)
4. Restart the Agent and reproduce the issue.

5. Collect the trace log like <ITM_HOME>\TMAITM6\logs\kbn_*_trace.log
(Note: If you prefer send the output from "<ITM_HOME>\bin\pdcollect" run) Unix/Linux 1. edit <ITM_HOME>/config/bn_<instanceName>.cfg 

2. On "bn_<instanceName>.cfg" file, locate "JAVA_TRACE_LEVEL"
3. Change the value to ERROR or DEBUG_MIN
depends on your need.
(ex: JAVA_TRACE_LEVEL=ERROR, or
JAVA_TRACE_LEVEL=DEBUG_MIN)
4. Restart the Agent and reproduce the issue.

5. Collect the trace log like <ITM_HOME>../logs/kbn_*_trace.log
(Note: If you prefer send the output from "<ITM_HOME>/bin/pdcollect" run)
