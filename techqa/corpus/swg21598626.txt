Title: IBM Unable to find supported browser message for Rational Developer for Power v8.5 launchpad on Linux systems with Firefox browser with versions 10 or higher - United States

Text:
 TECHNOTE (TROUBLESHOOTING)

PROBLEM(ABSTRACT)
 When launchpad.sh is executed on a Linux system with Firefox browser version 10.0 or higher, it will give an error message and launchpad cannot be launched for installation. 

SYMPTOM
An error message is shown indicating that the browser is not supported for starting the product from launchpad. 

For example, this message is shown when launchpad.sh is executed on any Linux system with Firefox version 10.0 or higher "Unable to find supported browser. An error occurred while starting the launchpad. this error typically occurs when the launchpad is unable to find a supported browser. 
Check your product's documentation for a list of supported browser".

CAUSE
Launchpad for Rational Developer for Power Systems 8.5 does not support Firefox browser versions 10.0 or higher.



ENVIRONMENT
Linux



RESOLVING THE PROBLEM
Install IBM Installation Manager 1.5.2 and Rational Developer for Power Systems Software V8.5 by executing ./install from the InstallerImage_linux folder on the first installation disk. Follow the instructions on screen and complete the installation. 


For detailed information on installation steps using Installation Manager, please refer to the Installation Guide [https://www-304.ibm.com/support/docview.wss?uid=swg27024935&wv=1] .
