Title: IBM Documentation unclear regarding UNIX Application support for ITM - United States

Text:
 TECHNOTE (FAQ)

QUESTION
 Currently there is no AIX application support package available for download . Although in the IBM Tivoli Monitoring for Tivoli Network Manager IP User's Guide is referencing UNIX , instead of just Linux 

CAUSE

ANSWER
