 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  An extra white blank space can be seen on the right side of the
   custom jsp when hide left navigation pane.
   
   Reproduce steps:
   
   1) Create a Custom Tools script, the type is ASP/JSP like.
   
   p1:
   
   <div style='background-color:lightgray;width:100%;height:100%'>
   
   style='background-color:lightgray;width:100%;height:100%'
   
   <%
   
   var i=0;
   
   for(i=0;i<=33;i++) {
   
    out.println("<table><tr><td>" + i + "</td></tr></table>");
   
   }
   
   %>
   
   </div>
   
   p2:
   
   <div style='background-color:lightgray;width:100%;height:100%'>
   
   style='background-color:lightgray;width:100%;height:100%'
   
   <%
   
   var i=0;
   
   for(i=0;i<=150;i++) {
   
    out.println("<table><tr><td>" + i + "</td></tr></table>");
   
   }
   
   %>
   
   </div>
   
   The unique difference between p1 and p2 is: loop count ( 33 for
   p1 vs 150 for p2 )
   
   2)In "Home" --> "My Settings", check "Hide left navigation
   pane", and choose p1 or p2 for "Configure the home page".
   
   3)Then logout UI and login again, you will see the white blank
   space on right of screen.
   
   4)This issue just can be seen when user login to UI. After
   that, when click "Restore Down/Maximize" at upper-right , or
   click "Custom Tools" to choose p1 or p2, you can't see the
   white blank space any more.
