 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  After restart the Sybase db server, cursor SYSMON fails to get
   data. The agent returns 100% Engine CPU_Busy_Percent and other
   wrong attributes.
   
   
   
   Detailed Recreation Procedure:
   1. Start the Sybase agent.
   2. Stop and then stop the Sybase db server.
   3. Check "Servers" navigator node "Engine Summary" workspace
   "Engine Summary" view, the value of attribute "CPU Busy Percent"
   is 100%.
   
   
   
   Related Files and Output: N/A
