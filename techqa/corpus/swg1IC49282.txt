ERROR DESCRIPTION
 *  1) INSTMQM needs better diagnostics during install OSS socket.
   The $ZPMON server and the $ZPMON.#ZPLS server must be started
   prior to the install.
   2) MQDEFAULTPREFIX is no longer supported for v5.3. In v5.3,
   if using OSS you should add to .profile command to execute
   wmqprofile.
   Guardian users must OBEY WMQCSTM found in v5.3 sample
   installation subvolume. This needs to be added to the product
   README.
