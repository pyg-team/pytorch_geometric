Title: IBM Netcool/OMNIbus Transport Module Common Integration Library is not affected by the vulnerability in Netty (CVE-2016-4970) - United States

Text:
 FLASH (ALERT)

ABSTRACT
 The Netcool/OMNIbus Transport Module Common Integration Library is not affected by the vulnerability in Netty (CVE-2016-4970). 

CONTENT
Netty is shipped as a dependent component of the Netcool/OMNIbus Transport Module Common Integration Library. 

The Netcool/OMNIbus Transport Module Common Integration Library is a dependent component of the Netcool/OMNIbus Integrations Probe for Message Bus and Gateway for Message Bus. 

The Netcool/OMNIbus Integrations are not affected by the vulnerability in Netty as they were not implemented to interface with an installed openSSL library. 

The following Netcool/OMNIbus Integrations ships the vulnerable dependant component: 

 * IBM Tivoli Netcool/OMINbus Integration - Transport Module Common Integration Library (common-transportmodule-12_0 up to and including common-transportmodule-17_0) 
 * IBM Tivoli Netcool/OMINbus Integration - Gateway for Message Bus (nco-g-xml-8_0) 
 * IBM Tivoli Netcool/OMINbus Integration - Probe for Message Bus (nco-p-message-bus-3_0 up to and including nco-p-message-bus-6_0)


The following Netcool/OMNIbus Integrations is updated with the resolved dependant component: 

 * IBM Tivoli Netcool/OMINbus Integration - Transport Module Common Integration Library (common-transportmodule-18_0) [http://www-01.ibm.com/support/docview.wss?uid=swg21698166], or later 
 * IBM Tivoli Netcool/OMINbus Integration - Gateway for Message Bus (nco-g-xml-9_0) [https://www-304.ibm.com/support/docview.wss?uid=swg21665219], or later 
 * IBM Tivoli Netcool/OMINbus Integration - Probe for Message Bus (nco-p-message-bus-7_0) [http://www-01.ibm.com/support/docview.wss?uid=swg21970413], or later
