This Limited Availability Interim Fix resolves following APAR: 

PM27125: Support DC install on WIN 2008 OS with ITCAM for WAS 6.1 FP4 refresh CD

ITCAM 6.1 FP4 Refresh CD for Data Collector fails to install on Windows 2008 platform.

unbelievably, Got IOException while reading prereq xml file: +

INSTALLATION INSTRUCTIONS
Contact Level 2 support to get the fix binary: 6.1.0.4-TIV-ITCAMfWAS_MP-LA0002.zip.

Use the following steps to install the update:

1. Extract 6.1.0.4-TIV-ITCAMfWAS_MP-LA0002.zip to a temporary directory.

2. From the ITCAM Data Collector for WebSphere (Fix Pack 4) CD image directory, back up following files:

setup_DC.jar
DCPrereqs.xml

3. Copy the files extracted from the fix zip file to the installer directory.

4. Run the installer for Data Collector.
