Title: IBM IT08117: A STANDBY CONTROL VLAN MIGRATION MIGHT FAIL WHEN THERE IS NO AUX VIP - United States

Text:
  A FIX IS AVAILABLE
Fix packs for DataPower Gateway version 7.1 [http://www-01.ibm.com/support/docview.wss?uid=swg24038524]

APAR STATUS
 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  A pre-7.0.0.0 configuration with standby control on a VLAN might
   fail after upgrade to 7.0.0.0 or later.
   
   
    
   
   

LOCAL FIX
 *  Re-enable the Standby Group configuration on the VLAN
   interfaces.
   
   
    
   
   

PROBLEM SUMMARY
 *  Affected are all users of the Standby Control feature who
   upgrade to a firmware version above (and including) 7.0.0 from
   an older firmware version.
   
   During the configuration migration, the Standby Control settings
   will not be applied when the configuration does not include a
   Secondary virtual IP address.
   
   
    
   
   

PROBLEM CONCLUSION
 *  Fix is available in 7.0.0.8 and 7.1.0.5.
   
   For a list of the latest fix packs available, please see:
   http://www-01.ibm.com/support/docview.wss?uid=swg21237631 [http://www-01.ibm.com/support/docview.wss?uid=swg21237631]
