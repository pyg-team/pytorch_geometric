This fix pack includes a cumulative collection of fixes and updated support for several underlying components. This fix pack updates the WebSphere Portal 6.1 (6.1.0.0), 6.1.0.1, 6.1.0.2, 6.1.0.3, 6.1.0.4 or 6.1.0.5 level to the 6.1.0.6 service release level. The fix pack can be used to upgrade the Web Content Manager 6.1 (6.1.0.0), 6.1.0.1, 6.1.0.2, 6.1.0.3, 6.1.0.4 or 6.1.0.5 level to the 6.1.0.6 service release level. This fix pack can also be used to upgrade WebSphere Portal Express 6.1 (6.1.0.0), 6.1.0.1, 6.1.0.2, 6.1.0.3, 6.1.0.4 or 6.1.0.5 level to the 6.1.0.6 service release level. Also included in the fix pack is the update to install the 6.1.5.3 fix pack to the 6.1.5.0, 6.1.5.1 or 6.1.5.2 feature pack (optional). 

The complete list of fixes integrated into this refresh pack is found in "Fix list for WebSphere Portal Version 6.1 [http://www.ibm.com/support/docview.wss?uid=swg27014411#6106]," linked above in the "Fix List" tab. 


Be sure to also download the file WP_PTF_6106.ptflist and place it in the same directory as the WP_PTF_6106.jar file before installing this fix pack with the Portal Update Installer. This file contains the list of integrated fixes and will inform you of the status of any fixes you currently have installed for WebSphere Portal or Web Content Manager. 


Additional Notes:

Refer to the WebSphere Portal V6.1.0.6 hardware and software requirements document for the minimum levels and fixes required for WebSphere Application Server or Process Server or Extended Deployment. 

Installation and support for WebSphere Application Server, Process Server or Extended Deployment fixes and fix packs is provided by the appropriate WebSphere product support teams, not WebSphere Portal support.

PREREQUISITES
6.1.0.6: WebSphere Portal version 6.1, 6.1.0.1, 6.1.0.2, 6.1.0.3, 6.1.0.4 or 6.1.0.5
6.1.5.3: WebSphere Portal version 6.1.5.0, 6.1.5.1 or 6.1.5.2

INSTALLATION INSTRUCTIONS
This update must be installed using the latest WebSphere Portal Update Installer [http://www.ibm.com/support/docview.wss?uid=swg24028760] for version 6.1.
