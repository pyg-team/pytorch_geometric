Title: IBM When importing a projects.csv file, the COORDINATOR column must be in the loginId:Role format. - United States

Text:
 PRODUCT DOCUMENTATION

ABSTRACT
 When you import a projects.csv file, if the COORDINATOR column is not in the loginId:Role format, the import will fail. 

CONTENT
The import fails with following error in the import logs:


 ERROR - FAILED: Create failed for Projects row 1: StringIndexOutOfBoundsException = null

Ensure that the COORDINATOR column is in the loginId:Role format to prevent errors.
