 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  Once more than 150 access roles have been created, Workplace XT
   administrators cannot see newly created Access Roles displayed
   in Site Preferences.
   
   Steps to reproduce:
   1. log in as an Admin to a system that has 150 access roles
   2. navigate to Tools/Administration/Site Preferences/Access
   Roles
   3. add a new Access Role, click apply
   4.  Click OK and log out of Workplace XT
   5.  Logs back in as an Admin and look at the existing access
   roles in Site preferece.
   Result: The new Access Role does not show up in Site Preferences
