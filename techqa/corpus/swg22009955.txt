Title: IBM Issues installing InfoSphere Master Data Management on Oracle 12cR2 with ojdbc8.jar on MDMv11.6.0.3 and below - United States

Text:
 TECHNOTE (TROUBLESHOOTING)

PROBLEM(ABSTRACT)
 During installation of InfoSphere Master Data Management or InfoSphere MDM Reference Data Managment on Oracle 12cR2 databases, the database connection test step in the IBM Installation Manager database configuration panel fails. This is due to the ODBC JAR file version. 

SYMPTOM
During the database connection test on the Database panel in IBM Installation Manager, the following error message is shown: 

com.ibm.im.mdm.install.validation.util.OracleService, The database cannot be connected. The database home is not valid. 


CAUSE
The Infosphere MDM or InfoSphere RDM installation application for versions 11.6.0.x do not support ojdbc8.jar while using Oracle 12cR2 as the database. 


ENVIRONMENT
Oracle 12c R2 



RESOLVING THE PROBLEM
To resolve this issue: 

 *  Download ojdbc6.jar from the Oracle web site: http://www.oracle.com/technetwork/apps-tech/jdbc-112010-090769.html [http://www.oracle.com/technetwork/apps-tech/jdbc-112010-090769.html] Copy the downloaded ojdbc6.jar file to the JDBC location, <ORACLE_HOME>/jdbc/lib/ Run the installation again from IBM Installation Manager. On the database configuration panel, click Test Connection to ensure that the connection test is now successful. Complete the remaining installation steps.
