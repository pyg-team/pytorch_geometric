Title: IBM ACUIR00003E error when executing listIU command - United States

Text:
TIPL2MUSTG; TIPL2; DEL2; lock; ACUIR00003E; listIU; error; query failed with return code TECHNOTE (TROUBLESHOOTING)

PROBLEM(ABSTRACT)
 If a previous DE command was executed and somehow was interrupted, this can leave a .lock file in the $DE_HOME/logs directory, causing a ACUIR00003E error code when executing other DE commands. 

SYMPTOM
Output from executing the listIU.(cmd || sh ) file: 

$DE_HOME/bin\listIU.( cmd || sh )
ACUIR00003E The query failed with return code, 2.


CAUSE
.lock files in the DE home logs directory.


ENVIRONMENT
Windows, AIX, Solaris, Linux 

DE_HOME location:

Windows: 
Installed as Admin: C:\Program Files\IBM\Common\acsi
Installed as a non-Admin user: C:\Documents and Settings\<username>\acsi_<username>

Unix: 
Installed as root: /usr/ibm/common/acsi 
Installed as a non-root user: /home/<username>/.acsi_<username>


DIAGNOSING THE PROBLEM
Check the $DE_HOME logs directory for any .lock* files.



RESOLVING THE PROBLEM
Remove the .lock* files in the $DE_HOME/logs directory and retry the listIU.(cmd || sh) command.
