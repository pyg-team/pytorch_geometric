PROBLEM SUMMARY
 *  ****************************************************************
   * USERS AFFECTED: Websphere Application Server users with a    *
   *                 version 6.0.2 client to a version 6.1        *
   *                 cluster.                                     *
   ****************************************************************
   * PROBLEM DESCRIPTION: The version 6.0.2 client incorrectly    *
   *                      handles version 6.1 cluster             *
   *                      information.                            *
   ****************************************************************
   * RECOMMENDATION:                                              *
   ****************************************************************
   The version 6.0.2 code was unable to handle the information
   being propagated from the version 6.1 code.
   
   
    
   
   

PROBLEM CONCLUSION
 *  The version 6.0.2 and 6.1 level of the code was changed to
   handle the differences between each other to allow for
   interopability.
   
   The fix for this APAR is currently targeted for inclusion
   in fixpack 6.0.2.11.
   Please refer to the recommended updates page for delivery
   information:
   http://www.ibm.com/support/docview.wss?rs=180&uid=swg27004980 [http://www.ibm.com/support/docview.wss?rs=180&uid=swg27004980]
