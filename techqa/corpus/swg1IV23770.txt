ERROR DESCRIPTION
 *  Approver: FS
   
   Environment: Monitoring Agent for Linux OS 6.22 fp4
   
   Problem Description:
   The Linux os agent does not support nfs file systems with a
   huge size.
   In this case, the linux os agent for nfs filesystems shows value
   for the attribute Size (MB) less than Disk Used (MB).
   Detailed Recreation Procedure: mount a filesystems greater than
   2 TeraByte on a linux machine and open the TEPS the "Disk usage"
   view.
   
   Reference: defect 141709
