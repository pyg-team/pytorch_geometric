Title: IBM Security Bulletin: Unauthorized disclosure of system information in IBM Business Process Manager (BPM) 8.5.x (CVE-2014-3076) - United States

Text:
 SECURITY BULLETIN

SUMMARY
 System information is provided on an unprotected diagnostic page. 

VULNERABILITY DETAILS
CVEID: CVE-2014-3076 [http://cve.mitre.org/cgi-bin/cvename.cgi?name=CVE-2014-3076]
DESCRIPTION: 
IBM Business Process Manager 8.5 contains an unprotected JavaServerâ„¢ Pages (JSP) file that returns system information to unauthenticated users. An attacker might use this information to aid in further attacks against the system.
