Title: IBM Security Bulletin: API Connect minimatch CVE-2017-1556 - United States

Text:
 SECURITY BULLETIN

SUMMARY
 API Connect has addressed the following vulnerability. 

API Connect is vulnerable to a regular expression attack that could allow an authenticated attacker to use a regex and cause the system to slow or hang. 

VULNERABILITY DETAILS
CVEID: CVE-2017-1556 [http://cve.mitre.org/cgi-bin/cvename.cgi?name=CVE-2017-1556]
DESCRIPTION: IBM API Connect is vulnerable to a regular expression attack that could allow an authenticated attacker to use a regex and cause the system to slow or hang.
CVSS Base Score: 4.3
CVSS Temporal Score: See https://exchange.xforce.ibmcloud.com/vulnerabilities/131546 [https://exchange.xforce.ibmcloud.com/vulnerabilities/131546] for the current score
CVSS Environmental Score*: Undefined
CVSS Vector: (CVSS:3.0/AV:N/AC:L/PR:L/UI:N/S:U/C:N/I:N/A:L)
