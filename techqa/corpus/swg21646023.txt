Title: IBM Read .sav file in IBM SPSS Amos - United States

Text:
 TECHNOTE (FAQ)

QUESTION
 When I tried to open a .sav file in IBM SPSS Amos, I received the following error message:
To display IBM SPSS Statistics files, the IBM SPSS Statistics module must be installed on this system. Amos could not find the IBM SPSS Statistics module. 

CAUSE
If you want to open .sav file from IBM SPSS Amos, you need to install and license IBM SPSS Statistics on the same computer.

ANSWER
Install and license IBM SPSS Statistics on the same computer and then you can open .sav files with IBM SPSS Amos.
