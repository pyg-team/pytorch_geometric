ERROR DESCRIPTION
 *  Problems with pub/sub data queries are as follows:
   - Query to QMSUBDEF (Subscription Definitions) using SUBID match
   does not work for z/OS MQ agent.
   - Query to QMATTR (Object Attribute Details) using SUBID for the
   ATTRUID match does not work for z/OS MQ agent.
   - Query to QMTPCDEF (Topic Definitions) for generic match on
   TPCNAME (Topic Name) does not work for z/OS MQ agent.
   - Extraneous messages occur in RKLVLOG.
