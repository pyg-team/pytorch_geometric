   search when refining search
   
   Extended Problem Description:
   
   User is not able to perform the Refine operation in Inventory
   Attribute search and getting the error BMXAA4214E Unknown error
   has occured on item attribute search
   
   ENVIRONMENT:
   
   Maximo 7118
   Oil&Gas 7120
   Calibration 7120

   1)      Login to Maximo with a user that does not just have
   READONLY
   privileges
   2)      Go to the  Inventory Module
   3)      Select  Attribute Search  from the  Item  field popup
   4)      Add new row and enter an attribute name
   5)      Enter a search value
   6)      Click refine- at this point the unknown error dialog
   appears
