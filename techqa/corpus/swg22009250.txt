Title: IBM Security Bulletin: IBM Integration Bus is affected by an Apache Tomcat related vulnerability (CVE-2017-7674) - United States

Text:
IIB SECURITY BULLETIN

SUMMARY
 IBM Integration Bus is affected by an Apache Tomcat related vulnerability (CVE-2017-7674). 

VULNERABILITY DETAILS
CVEID: CVE-2017-7674 [http://cve.mitre.org/cgi-bin/cvename.cgi?name=CVE-2017-7674]
DESCRIPTION: Apache Tomcat could provide weaker than expected security, caused by the failure to add an HTTP Vary header to indicate that the response varies depending on Origin by the CORS Filter. A remote attacker could exploit this vulnerability to conduct client and server side cache poisoning.
CVSS Base Score: 7.3
CVSS Temporal Score: See https://exchange.xforce.ibmcloud.com/vulnerabilities/130248 [https://exchange.xforce.ibmcloud.com/vulnerabilities/130248] for the current score
CVSS Environmental Score*: Undefined
CVSS Vector: (CVSS:3.0/AV:N/AC:L/PR:N/UI:N/S:U/C:L/I:L/A:L) 

AFFECTED PRODUCTS AND VERSIONS
IBM Integration Bus V9.0.0.0 - V9.0.0.8 and V10.0.0.0 - V10.0.0.9
