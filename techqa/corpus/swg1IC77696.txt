ERROR DESCRIPTION
 *  The issue is when trying to add a WSDL from an NFS server, i.e.
   dpnfs://. DataPower does not support adding a WSDL from an NFS
   server, the user will need to choose another source. The
   dpnfs:// must be removed from the Add WSDL drop-down on the
   WebGUI.
   
   
    
   
   

LOCAL FIX
 *  No Work-Around
   
   
    
   
   

PROBLEM SUMMARY
 *  The webGUI dropdown for selecting a WSDL location will be
   corrected to remove invalid options. The fix will be in the next
   major release.
