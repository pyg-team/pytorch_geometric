ERROR DESCRIPTION
 *  Steps to reproduce :
   Step 1: instal & configure FP4 DC, TEMA
    Step 2: start the server and TEMA
     Step 3: generate the workload, check the request analysis work
   space,
     Step 4: in the Requests-Current internal table, click anyone,
   request will enter the Request -Datasource workspace. but, for
   those requests that do not use datasouce, the Request
   -Datasource workspace also contains their data. It doesn't make
   sense for those requests which do not use any datasource.
