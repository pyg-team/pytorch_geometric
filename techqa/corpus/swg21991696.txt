Title: IBM Error issued while attempting to view documents using IBM Daeja ViewONE in IBM Content Navigator - United States

Text:
 TECHNOTE (TROUBLESHOOTING)

PROBLEM(ABSTRACT)
 Unable to open the documents using ViewONE Pro and ViewONE Virtual in IBM Content Navigator (ICN). 

SYMPTOM
The message below is popped up. 

IBM Daeja ViewONE 4.0. has been installed
Please refresh the current browser window in order to complete the process and load the updated viewer.


CAUSE
It's caused by cached data.


RESOLVING THE PROBLEM
On end user workstation, clear browser cache, daeja cache and java cache. 

 * 
 *   * Close all the browser sessions. 
    * Clear browser cache. 
    * Clear daeja viewer cache and delete all the files and folders under daeja viewer cache directory. By default, it's at C:\Users\UserName\AppData\Local\Temp\filenet viewer. 
    * Clear java cache and delete all the files and folders under java cache directory. For Sun Java, by default, it's at C:\Users\UserName\AppData\LocalLow\Sun\Java\Deployment\cache\6.0.
