Title: IBM Can IBM Content Collector (ICC) unzip files prior to storing them in the repository - United States

Text:
 TECHNOTE (FAQ)

QUESTION
 We have a requirement to unzip files prior to ICC collecting them to store in our IBM content repository 

CAUSE
Customer has zip files that need to be unzipped prior to be ingested into the repository

ANSWER
This feature is not supported out of the box. You can explore customizing using the ICC Script connector
