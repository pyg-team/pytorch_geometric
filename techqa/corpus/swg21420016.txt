Title: IBM BIP3453E BIP3450E in WMB SAP connectivity sample failures - United States

Text:
mqiinfo; BIP3453E; BIP3450E; SAP sample TECHNOTE (TROUBLESHOOTING)

PROBLEM(ABSTRACT)
 The WebSphere Message Broker (WMB) SAP connectivity sample fails during the runtime operation. 

SYMPTOM
The following errors are seen in the broker logs - 

BIP3453E - 
Event ID: 3453
( BDHLUW03461.RANL ) Adapter input node ''MF_MATMAS05Flow'' failed to process an event called ''SapMatmas05'' from adapter component ''ADC_MATMAS05.inadapter''.
Inbound adapter component ''ADC_MATMAS05.inadapter'' has delivered an event called ''SapMatmas05'' to the input node ''MF_MATMAS05Flow''. An error occurred during the conversion of this event to a Message Broker message. See the following message to determine the cause of the error.
None
BIP3450E -
Event ID: 3450
( BDHLUW03461.RANL ) An adapter error occurred during the processing of a message. The adapter error message is ''com.sap.mw.jco.JCO$Exception: (1000) com.ibm.despi.exception.DESPIException: java.lang.NullPointerException: com.ibm.despi.exception.DESPIException: com.ibm.despi.exception.DESPIException: java.lang.NullPointerException
com.ibm.despi.exception.DESPIException: com.ibm.despi.exception.DESPIException: java.lang.NullPointerException
com.ibm.despi.exception.DESPIException: java.lang.NullPointerException
java.lang.NullPointerException
Looking at the logs on the adapter side, the following errors are noticed -

The following message describes the diagnostic information that is provided by the adapter 

''com.sap.mw.jco.JCO$Exception: (1000) com.ibm.despi.exception.DESPIException: java.lang.NullPointerException: com.ibm.despi.exception.DESPIException: com.ibm.despi.exception.DESPIException: java.lang.NullPointerException



CAUSE
There is a mismatch between the version of the SAP system on which the sample was created and the version on which it is being run.


RESOLVING THE PROBLEM
The solution is to rebuild the SAP Adapter against the actual version of SAP in use.
