RESOLVING THE PROBLEM
Workaround: Run the Optimisation with the Analyse Schema option un-ticked. 

It will still perform all needed actions for everyday purposes, all that will be missing is a deeper re-indexing action that can help performance. 

Fix: Reboot the Oracle database server.
