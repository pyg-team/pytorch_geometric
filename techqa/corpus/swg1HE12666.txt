ERROR DESCRIPTION
 *  When installing ICC OE 4.0.1.4 on a Windows 10 Build 1703,
   following error message is observed on the welcome screen:
   
   Warning
   .Net 4.5.1 framework cannot be found.
   The Outlook Extension for IBM Content Collector cannot be
   installed, as .NET Framework 4.5.1 is not installed.
   
   Environment:
   ICC 4.0.1 FP4
   Outlook 2016 (16.0.7329.1051) 32 Bit
   ICC Outlook Extension Version: 4.0.1.4
   
   As per ICC prerequisites .NET Framework 4.5 or later is
   supported. With latest Windows 10 build, .NET Framework 4.7 is
   shipped.
   Outlook Extension installer is currently unable to detect the
   .NET framework version 4.7.
   
   
    
   
   

LOCAL FIX
 *  Contact IBM Support for fix
   
   
    
   
   

PROBLEM SUMMARY
 *  When installing ICC OE 4.0.1.4 on a Windows 10 Build 1703,
   following error message is observed on the welcome screen:
   
   Warning
   .Net 4.5.1 framework cannot be found.
   The Outlook Extension for IBM Content Collector cannot be
   installed, as .NET Framework 4.5.1 is not installed.
   
   Environment:
   ICC 4.0.1 FP4
   Outlook 2016 (16.0.7329.1051) 32 Bit
   ICC Outlook Extension Version: 4.0.1.4
   
   As per ICC prerequisites .NET Framework 4.5 or later is
   supported. With latest Windows 10 build, .NET Framework 4.7 is
   shipped.
   Outlook Extension installer is currently unable to detect the
   .NET framework version 4.7.
   
   
    
   
   

PROBLEM CONCLUSION
 *  Starting with IBM Content Collector 4.0.1 FP5 release, the
   Outlook Extension installer has been fixed for GUI installation
   and it will detect the .Net Framework 4.7 pre-requisite.
   Outlook Extension silent and console installation has been fixed
    starting with IBM Content Collector 4.0.1 FP6 release (APAR
   HE12698 [http://www-01.ibm.com/support/docview.wss?uid=swg1HE12698]).
