ERROR DESCRIPTION
 *  The FTP proxy Call Home connection type was removed from
   PureApplication version 2.0.0.1.  A coding error resulted
   in any change to the Call Home configuration defaulting
   to the FTP proxy connection type.
   
   
    
   
   

LOCAL FIX

PROBLEM SUMMARY
 *  The FTP proxy Call Home connection type was removed from
   PureApplication version 2.0.0.1.  A coding error resulted
   in any change to the Call Home configuration setting the
   connection type to FTP proxy.
   
   
    
   
   

PROBLEM CONCLUSION
 *  A code fix is being provided in PureApplication System
   version 2.1.0.0 and now the correct connection type
   value is used.
