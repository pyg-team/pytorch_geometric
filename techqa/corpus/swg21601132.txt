PROBLEM(ABSTRACT)
 Attempts to run "appscanserverdbmgr.bat" to link IBM Security AppScan Enterprise server with IBM Security AppScan Source database results in: 'Exception in thread "main" java.lang.NoClassDefFoundError: org.eclipse.swt.widgets.Dialog'. 

SYMPTOM

 * "plugins" sub-folders are missing in the AppScan Source installation

CAUSE
This problem occurs if you run appscanserverdbmgr.bat on the machine where AppScan Source client components have not been installed.



RESOLVING THE PROBLEM
 

 * Run appscanserverdbmgr.bat on system where the AppScan Source client components have been installed or 
 * Add at least one client component to the system where appscanserverdbmgr.bat is failing
