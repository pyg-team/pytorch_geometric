Title: IBM Installing API Connect Essentials 5.0 for Operational Decision Manager 8.8.1 - United States

Text:
 NEWS

ABSTRACT
 This document explains how to download and install API Connect Essentials V5.0 from IBM Passport Advantage Online. 

CONTENT
Operational Decision Manager V8.8.1 customers are entitled to use API Connect Essentials V5.0 as a supporting program, as indicated in the Decision Server licenses.

For instance, the IBM Decision Server Standard V8.8.1 license states:

The following are Supporting Programs licensed with the Program: 

 * IBM DB2 Workgroup Server Edition 10.5 
 * IBM WebSphere Application Server Network Deployment 8.5 
 * IBM Installation Manager and Packaging Utility for the Rational Software DevelopmenPlatform 1.8 
 * IBM API Connect Essentials 5.0


Downloading 
Follow these instructions to download the eAssembly repository files:  1. Go to IBM Passport Advantage Online, [https://www.ibm.com/software/howtobuy/softwareandservices/registration/Registration?caller=PACSDMA] [http://www.ibm.com/software/howtobuy/passportadvantage/paocustomer/sdma/SDMA?PO=DOWNLOAD_FIND_BY_PART_NUMBER] and use your IBM ID to sign in. 
 2. Select Find by part number in Download finder options. 
 3. Search for your files by using the part numbers that are listed in the table below. 
 4. Download the files. 


Using Installation Manager 
Follow these instructions to install the product by using Installation Manager:  1. Extract each repository from the downloaded file. 
 2. Start IBM Installation Manager. 
 3. Open File > Preferences. 
 4. Under Repositories, click Add repository and browse to an extracted folder. Repeat this step until you finish adding all the folders for repositories that you want to install. 
 5. Click OK. 
 6. Click Install and select the repositories you want to install. 
 7. Click Install to complete the installation.
