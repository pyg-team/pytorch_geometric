ERROR DESCRIPTION
 *  In WebSphere Integration Developer v6.0.1, when used in
   conjunction with WebSphere Enterprise Service Bus or
   WebSphere Process Server v6.0.1.2, a NameNotFoundException
   is seen when using the Component Test functionality.
   Exception is:
   
   com.ibm.wsspi.sibx.mediation.flow.MFCNotFoundException:
   CWSXM1010E: Could not locate the runtime mediation flow for
   mediation component in X module com.ibm.wbit.comptest.runtime
   
   Caused by: javax.naming.NameNotFoundException: Name
   ejb/scax/mediation/com.ibm.wbit.comptest.runtime not found in
   context "local:".
