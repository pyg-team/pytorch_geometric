Profile Management Tool  1.  FAQ: How do I run the profile management tool (PMT) on a 64bit platform?
    Answer: You cannot use the Profile Management Tool to create or augment profiles on WebSphere Process Server installations on 64-bit architectures except on the Linux on zSeries platform. To create profiles on other 64-bit architectures, you can use the manageprofiles command. For information about using the manageprofiles command, see Creating profiles using the manageprofiles command [http://publib.boulder.ibm.com/infocenter/dmndhelp/v7r0mx/index.jsp?topic=/com.ibm.websphere.wps.doc/doc/tins_manageprofiles_create.html]. You can also use the Profile Management Tool on these architectures if you use a WebSphere Process Server 32-bit installation.
