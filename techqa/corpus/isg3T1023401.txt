Title: IBM Security Bulletin: OpenStack vulnerabilities affect IBM Cloud Manager with Openstack (CVE-2015-7713, CVE-2015-5286) - United States

Text:
 SECURITY BULLETIN

SUMMARY
 IBM Cloud Manager with Openstack is vulnerable to several OpenStack vulnerablities. An attacker can exploit these velnerabilities to launch further attacks on the system or to exhaust all available resources. 

VULNERABILITY DETAILS
CVEID: CVE-2015-7713 [http://cve.mitre.org/cgi-bin/cvename.cgi?name=CVE-2015-7713]
DESCRIPTION: OpenStack Nova could provide weaker than expected security, caused by the failure to apply network security changes to already running instances. This could result in instances not beingprotected by the security group. An attacker could exploit this vulnerability and launch further attacks on the system.
CVSS Base Score: 5.3
CVSS Temporal Score: See https://exchange.xforce.ibmcloud.com/vulnerabilities/106973 [https://exchange.xforce.ibmcloud.com/vulnerabilities/106973] for the current score
CVSS Environmental Score*: Undefined
CVSS Vector: (CVSS:3.0/AV:N/AC:L/PR:N/UI:N/S:U/C:N/I:L/A:N) 

CVEID: CVE-2015-5286 [http://cve.mitre.org/cgi-bin/cvename.cgi?name=CVE-2015-5286]
DESCRIPTION: OpenStack Glance is vulnerable to a denial of service. By deleting images uploaded using a token that is about to expire, an attacker could exploit this vulnerability to bypass storage quotas and accumulate untracked image data in the backend to exhaust all available resources.
CVSS Base Score: 5.3
CVSS Temporal Score: See https://exchange.xforce.ibmcloud.com/vulnerabilities/106875 [https://exchange.xforce.ibmcloud.com/vulnerabilities/106875] for the current score
CVSS Environmental Score*: Undefined
CVSS Vector: (CVSS:3.0/AV:N/AC:L/PR:N/UI:N/S:U/C:N/I:N/A:L)

AFFECTED PRODUCTS AND VERSIONS
IBM Cloud Manager with OpenStack 4.1.0 through 4.1.0.5
IBM Cloud Manager with OpenStack 4.2.0 through 4.2.0.3 interim fix 4
IBM Cloud Manager with OpenStack 4.3.0 through 4.3.0.4 interim fix 1
