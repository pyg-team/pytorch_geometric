Title: IBM How to reactive a deactivated request in Atlas ? - United States

Text:
atlas active TECHNOTE (FAQ)

QUESTION
 How to reactive a deactivated request in Atlas ? 

CAUSE
There are situations where the request has been deactivated by mistake.

ANSWER
There is no way to reactivate a request from Atlas UI and the only way is update database table LegalRequest for this requestID status to "In Progress".
