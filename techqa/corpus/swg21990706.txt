Title: IBM Identify Installed or Upgraded Atlas Version - United States

Text:
Atlas Version Schema Upgrade Install TECHNOTE (FAQ)

QUESTION
 How do I identify Installed or Upgraded Atlas version? 

CAUSE
The Atlas version was upgraded and the version needs to be confirmed

ANSWER
 

Application level: 

 1. Login to Atlas as System Administrator 
 2. Navigate to Admin tab 
 3. Review Atlas version on the page and build number in the copyright footer 


Database level:  1. Login to Database as PSSAPL 
 2. Run the select statement - select * from pa_schema_installed_version; 
 3. Review the data under columns INSTALL_VERSION and INSTALL_DATE
