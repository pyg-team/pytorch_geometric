APAR STATUS
 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  Case Analyzer store throws the below exception and the
   processing of CE Events stopped
   
   Error:
   2014/04/12 20:19:04.348+0200 filenet.034B 12571257 [Error]
   Unexpected exception occurred while processing task
   filenet.eventexporter.common.EventPublisherTask for store
   CAStore.; Exception:
   com.microsoft.sqlserver.jdbc.SQLServerException: Cannot insert
   duplicate key row in object 'dbo.D_DMDataField_XXX' with unique
   index 'D_DMDataField_XXX_IX'. The duplicate key value is (Apr 12
   1994  7:53PM).
   
   This occurs when a User defined dimension field of type DATETIME
   is configured to fetch the date information from CE Events.
   
   
    
   
   

LOCAL FIX

PROBLEM SUMMARY
 *  ****************************************************************
   * USERS AFFECTED:                                              *
   * Customers using Case Analyzer store in CPE version           *
   * 5.2.0.3-P8CPE-FP003 or prior versions and having User        *
   * defined field (UDF) of date type.                            *
   ****************************************************************
   * PROBLEM DESCRIPTION:                                         *
   * Customers using Case Analyzer store in CPE versions prior to *
   * 5.2.0.3-P8CPE-FP003 and who has User Defined Field of type   *
   * DateTime may see this issue                                  *
   ****************************************************************
   * RECOMMENDATION:                                              *
   * Upgrade to 5.2.0.3-P8CPE-IF002 or higher versions to resolve *
   * the issue                                                    *
   ****************************************************************
   
   
    
   
   

PROBLEM CONCLUSION
 *  Upgrade to 5.2.0.3-P8CPE-IF002 or higher versions to resolve the
   issue
