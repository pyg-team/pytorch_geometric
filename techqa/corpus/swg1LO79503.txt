Title: IBM LO79503: TRAVELER SHUTDOWN MAY CRASH DOMINO SERVER. - United States

Text:
 SUBSCRIBE
You can track all active APARs for this component.



APAR STATUS
 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  We found internally that on shutdown of the Traveler service one
   of the threads may become hung.  Subsequent the Domino server
   sometimes will crash as a result.
   
   This isn't very common but seen enough times to warrent fixing
   the problem with shutting down the particular thread in
   question.
   
   
    
   
   

LOCAL FIX
 *  Simple restart of Domino server clears things up.
   
   
    
   
   

PROBLEM SUMMARY
 *  Occasionaly during shutdown of Traveler task one thread will
   become hung and cause issues with the Domino server or restart
   of Notes Traveler.
   
   
    
   
   

PROBLEM CONCLUSION
 *  The Notes Traveler server was updated to ensure timely shutdown
   of all threads.
   
   
    
   
   

TEMPORARY FIX
 *  his fix is included in Notes Traveler 9.0.1 IF4 and 9.0.0.1 IF5
   and all future releases.  For the latest Notes Traveler
   maintenance release information please see this technote:
   
   http://www.ibm.com/support/docview.wss?uid=swg24019529 [http://www.ibm.com/support/docview.wss?uid=swg24019529]
