QUESTION
 Is it possible to load Data Formats directly from a COBOL record definition? 

ANSWER
Using WebSphere Data Interchange Client, you can create a Data Format directly from a COBOL copy book. The Help text associated with the Data Dictionary window explains the step-by-step process.
