PROBLEM SUMMARY
 *  Description of issue:  The Control Center GUI will not start
   from the SCC webstart launch page after the the user's local
   machine has been upgraded to Oracle JRE 7 update 51 or later.
   The following exception is thrown: java.lang.SecurityException:
   Missing required Permissions manifest attribute in main jar.
   
   
    
   
   

PROBLEM CONCLUSION
 *  Description of fix:  Modified the SCC build process to set the
   required permissions attribute in the main jar.
