Title: IBM 【セキュリティ情報】WebSphere Commerce 同梱の IBM Java SDKに複数の脆弱性(CVE-2015-4872, CVE-2015-4734, CVE-2015-5006)  (CVE-2015-4000) - United States

Text:
 SECURITY BULLETIN

SUMMARY
 IBM® Java™ SDKは、WebSphere Application ServerがWebSphere Commerceのコンポーネントとして、WebSphere Application Serverに含まれています。
IBM Java SDKのセキュリティ脆弱性に関するセキュリティ情報が公開されました。



VULNERABILITY DETAILS
IBM WebSphere Application ServerのSecurity Bulletinが、公開されていますので、下記を参照し、対応お願いします。
Security Bulletin: Multiple vulnerabilities in IBM® Java SDK affect WebSphere Application Server October 2015 CPU (CVE-2015-4872, CVE-2015-4734, CVE-2015-5006)
http://www.ibm.com/support/docview.wss?uid=swg21969620 [http://www.ibm.com/support/docview.wss?uid=swg21969620]
