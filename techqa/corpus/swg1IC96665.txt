ERROR DESCRIPTION
 *  Certificate details is not displayed in WebGUI if the
   certificate file name has white space.
   For example: "cert:///Mycompany Root CA.cer"
   
   The following error may be returned:
   
   Unable to obtain certificate details for file
   cert:///MycompanyRootCA.cer , it may be password-protected or
   it may not contain a certificate.
   
   
    
   
   

LOCAL FIX
 *  Rename the certificate file without white spaces before
   uploading to the appliance.
   
   
    
   
   

PROBLEM SUMMARY
 *  Certificate details is not displayed in the WebGUI if the
   certificate file name contains white space.
