APAR STATUS
 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  In IBM Content Navigator, when a user downloads a document as
   original, the file is downloaded with a .bin or .dat file
   extension instead of the extension associated with the original
   file.
   
   
    
   
   

LOCAL FIX
 *  Change the backend repository content elements MIME type to
   reflect the desired file extension.
   
   
    
   
   

PROBLEM SUMMARY
 *  When a user downloads a document as original, the file is
   downloaded with a .bin or .dat file extension instead of the
   extension associated with the original file.
   
   
    
   
   

PROBLEM CONCLUSION
 *  With this fix, the document is downloaded with the original
   file name and file extension.
