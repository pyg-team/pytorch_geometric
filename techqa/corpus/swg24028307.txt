[http://www.ibm.com/i/c.gif] INSTALLATION INSTRUCTIONS
Important: 
Back up your systems and business-critical information before starting the installation process, especially if you are upgrading any software components.

1. Use the Planning and preparing for IBM FileNet P8 section in the IBM FileNet P8 information center to prepare for the FileNet P8 Platform and FileNet Content Manager installation.

2. Use the Installing or upgrading IBM FileNet P8 Platform section in the IBM FileNet P8 information center to install and configure the FileNet Content Manager and P8 Platform components, including Content Engine and Process Engine.

3. Use the Installing additional IBM FileNet P8 products section in the IBM FileNet P8 information center to install and configure IBM FileNet Workplace XT.

4. Use the Production Imaging Edition installation guide to prepare for installation and to install and configure IBM Production Imaging Edition Viewer Pro.

5. Use the IBM Datacap Taskmaster Capture installation guide to prepare for installation and to install and configure Datacap Taskmaster Capture.

6. Use the Production Imaging Edition installation guide to perform any final configuration steps to complete the installation and configuration of Production Imaging Edition.
