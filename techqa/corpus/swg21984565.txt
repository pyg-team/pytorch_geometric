Title: IBM Security Bulletin: IBM WebSphere MQ Invalid client protocol flows could cause denial of service (CVE-2016-0379) - United States

Text:
 SECURITY BULLETIN

SUMMARY
 An invalid MQ client protocol flow could cause a memory access violation on the server which could impact other channels running in the same process. 

VULNERABILITY DETAILS
CVEID: CVE-2016-0379 [http://cve.mitre.org/cgi-bin/cvename.cgi?name=CVE-2016-0379]
DESCRIPTION: IBM WebSphere MQ could allow an authenticated user with queue manager rights to cause a denial of service to channels due to incorrect handling of MQ protocol flows.
CVSS Base Score: 3.1
CVSS Temporal Score: See https://exchange.xforce.ibmcloud.com/vulnerabilities/112245 [https://exchange.xforce.ibmcloud.com/vulnerabilities/112245] for the current score
CVSS Environmental Score*: Undefined
CVSS Vector: (CVSS:3.0/AV:N/AC:H/PR:L/UI:N/S:U/C:N/I:N/A:L)

AFFECTED PRODUCTS AND VERSIONS
IBM WebSphere MQ 7.5 

IBM WebSphere MQ 7.5.0.0 through 7.5.0.6 maintenance levels 

IBM WebSphere MQ 8.0 

IBM WebSphere MQ 8.0.0.0 through 8.0.0.4 maintenance levels

REMEDIATION/FIXES
IBM WebSphere MQ 7.5 

Apply IBM WebSphere MQ 7.5.0.7 [http://www-01.ibm.com/support/docview.wss?uid=swg21983301] or later maintenance 

IBM WebSphere 8.0 

Apply IBM WebSphere MQ 8.0.0.5 [http://www-01.ibm.com/support/docview.wss?uid=swg21982686] or later maintenance

WORKAROUNDS AND MITIGATIONS
None
