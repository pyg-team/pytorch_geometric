Title: IBM (参考) IBM Java Runtime の複数の脆弱性の IBM Mobile Connect への影響 (CVE-2016-5597, CVE-2016-5542) - United States

Text:
 SECURITY BULLETIN

SUMMARY
 IBM Mobile Connect で使用されている IBM Runtime Environment Java バージョン 7 で複数の脆弱性が報告されました。 

VULNERABILITY DETAILS

この製品とともに配布されている IBM Java Runtime を使って独自の Java コードを実行している場合、脆弱性の問題がコードに該当するかを確定するために、コードの評価をする必要があります。脆弱性の完全なリストは、IBM Java SDK Security Bulletin を参照してください。

CVEID: CVE-2016-5597 [http://cve.mitre.org/cgi-bin/cvename.cgi?name=CVE-2016-5597]
説明: Networking コンポーネントの不特定の脆弱性により、リモートの攻撃者が未知の攻撃ベクトルを使用して機密情報を取得し、機密性に高い影響を及ぼす可能性があります。

CVSS Base Score: 5.9
CVSS Temporal Score: 現時点のスコアについては ISS X-Force Database [https://exchange.xforce.ibmcloud.com/vulnerabilities/118071] を参照してください。
CVSS Environmental Score*: Undefined
CVSS Vector: (CVSS:3.0/AV:N/AC:H/PR:N/UI:N/S:U/C:H/I:N/A:N) 


CVEID: CVE-2016-5542 [http://cve.mitre.org/cgi-bin/cvename.cgi?name=CVE-2016-5542]
説明: Libraries コンポーネントの不特定の脆弱性により、完全性に低い影響を及ぼす可能性のある脆弱性が存在します。機密性・可用性には影響しません。

CVSS Base Score: 3.1
CVSS Temporal Score: 現時点のスコアについては ISS X-Force Database [https://exchange.xforce.ibmcloud.com/vulnerabilities/118073] を参照してください。
CVSS Environmental Score*: Undefined
CVSS Vector: (CVSS:3.0/AV:N/AC:H/PR:N/UI:R/S:U/C:N/I:L/A:N)

AFFECTED PRODUCTS AND VERSIONS
IBM Mobile Connect (Gatekeeper コンポーネントのみ) 6.1.5, 6.1.5.1, 6.1.5.2

REMEDIATION/FIXES
これらの問題は、IBM Mobile Connect および IBM Mobile Connect とともに出荷された IBM Java Runtime で修正されました。
