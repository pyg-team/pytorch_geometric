Title: IBM Linux APM OS Agents not appearing in APM Dash - United States

Text:
APM UI LINUX OS MISSING TECHNOTE (TROUBLESHOOTING)

PROBLEM(ABSTRACT)
 After installing LINUX OS Agents successfully, the agents are not appearing in the APM 813 dash 

CAUSE
hal daemon not running

DIAGNOSING THE PROBLEM
The LINUX OS APM agent invokes the hal-get-proprty command.Its is a required daemon.

RESOLVING THE PROBLEM
After starting the hal daemon and restarting the LINUX OS agent, the agent appeared in the APM 813 dash.
