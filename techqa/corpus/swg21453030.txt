Title: IBM Can Cast Iron use Windows authentication to connect to a database server? - United States

Text:
 TECHNOTE (FAQ)

QUESTION
 Can Cast Iron use Windows authentication to connect to a database server? 

ANSWER
Currently, as of CIOS version 6.1.0.x, we do not support Windows authentication. You will need to change the database to accept mixed authentication.
