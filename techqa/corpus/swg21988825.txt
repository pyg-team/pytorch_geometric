Title: IBM 【セキュリティ情報】 WebSphere Message Broker, IBM Integration Bus, IBM Integration Bus Healthcare Pack, Manufacturing Pack and Retail Pack　Windowsインストーラ　DLLの脆弱性の影響  (CVE-2016-4560) - United States

Text:
 SECURITY BULLETIN

SUMMARY
 WebSphere Message Broker, IBM Integration Bus, IBM Integration Bus Healthcare Pack, IBM Integration Bus Manufacturing Pack, and IBM Integration Bus Retail Packで提供されているグラフィカル・ユーザ・インタフェース・インストーラー (setup.exe)に脆弱性があり、Windows検索パスに存在する悪意のあるDLLが、本物のファイルの代わりにオペレーティングシステムによってロードされる可能性があります。


VULNERABILITY DETAILS
脆弱性の詳細：
CVEID: CVE-2016-4560 [http://cve.mitre.org/cgi-bin/cvename.cgi?name=CVE-2016-4560]
DESCRIPTION: Flexera InstallAnywhere could allow a local attacker to gain elevated privileges on the system, caused by an untrusted search path. An attacker could exploit this vulnerability, by using a Trojan horse DLL in the current working directory of a setup-launcher executable file, to gain elevated privileges on the system.
CVSS Base Score: 7.8
CVSS Temporal Score: See https://exchange.xforce.ibmcloud.com/vulnerabilities/113016 [https://exchange.xforce.ibmcloud.com/vulnerabilities/113016] for the current score
CVSS Environmental Score*: Undefined
CVSS Vector: (CVSS:3.0/AV:L/AC:L/PR:L/UI:N/S:U/C:H/I:H/A:H) 

AFFECTED PRODUCTS AND VERSIONS
影響のある製品・バージョン： 

次の製品とFix Packのレベルで影響があります。

 * IBM Integration Bus Healthcare Pack V3 for Windows (V3.0.0.0 -> V3.0.0.1) 
 * WebSphere Message Broker Connectivity Pack for Healthcare V8 for Windows (V8.0.0.0) 
 * WebSphere Message Broker Connectivity Pack for Healthcare V7 for Windows (V7.0.0.0 -> V7.0.0.2) 
 * IBM Integration Bus Manufacturing Pack V1 for Windows (V1.0.0.0 -> V1.0.0.1) 
 * IBM Integration Bus Retail Pack V1 for Windows (V1.0.0.0)

REMEDIATION/FIXES
修正策：

WORKAROUNDS AND MITIGATIONS
回避策/軽減策： InstallAnywhere の脆弱性を回避するために、次のステップを完了してください。
信頼できない検索パスの脆弱性を避けるために、次のステップを実行してください。

1) 安全なディレクトリーに新規に一時的に空のディレクトリーを作成してください。
ディレクトリーは以前から存在したものの再利用ではなく、必ず新規に作成し、管理者のみ書き込み権限を与えてください。
2) 実行可能なインストーラーをステップ1)で作成したディレクトリーにコピー、移動もしくは導入zipファイルを解凍してください。
3) DLLファイルがこのディレクトリにないことを確認してください。
4) その新しい場所から実行可能インストーラーを開始してください。
