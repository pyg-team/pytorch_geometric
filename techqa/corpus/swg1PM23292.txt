APAR STATUS
 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  In cases where a portlet supports an event X:Y as a publishing
   as well as a processing event, the new wiring widget available
   in Client Side Rendering Mode potentially tries to create
   invalid wires.
   
   This APARs prevents the wire model controller from accepting
   this invalid data.
