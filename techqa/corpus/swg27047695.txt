Title: IBM Watson Explorer Version 11.0.1 Quick Start Guides - United States

Text:
 PRODUCT DOCUMENTATION

ABSTRACT
 Quick Start Guides for the IBM Watson Explorer Version 11.0.1 Foundational Components and Analytical Components are available in many languages. 

CONTENT
The Quick Start Guides are available in PDF format in the following languages. Use the language code appended to the file name to identify the language version that you want to read. 

ar - Arabic
cs - Czech
de - German
en - English
es - Spanish
fr - French
hu - Hungarian
ja - Japanese
ko - Korean
nl - Dutch
pl - Polish
pt - Portuguese
pt-BR - Brazilian Portuguese
ru - Russian
sk - Slovakian
sl - Slovenian
tr - Turkish
zh-Hans - Simplified Chinese
zh-Hant - Traditional Chinese


 

To access the IBM Watson Explorer Version 11.0.1 Foundational Components Quick Start Guide, download the WatsonExplorerFoundationalV1101_QSG ZIP file that contains your language version (Arabic through Dutch, or Polish through Chinese). 

 

To access the IBM Watson Explorer Version 11.0.1 Analytical Components Quick Start Guide, download the WatsonExplorerAnalyticalV1101_QSG ZIP file that contains your language version (Arabic through Dutch, or Polish through Chinese).
