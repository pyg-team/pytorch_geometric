ERROR DESCRIPTION
 *  You cannot see the 'Cancel', 'Close', 'Save', and 'Save and
   Close' buttons at the bottom of the screen. Only the very tops
   of the buttons might be visible.
   
   
    
   
   

LOCAL FIX
 *  Firefox 3.0 is not supported for InfoSphere Information Server
   Version 8.1.
   
   The following Web browsers are supported:
   
       * Microsoft Internet Explorer 6 Service Pack 2
       * Microsoft Internet Explorer version 7
       * Mozilla Firefox version 2.0
   
   
    
   
   

PROBLEM SUMMARY
 *  ****************************************************************
   USERS AFFECTED:
   Users accessing the IS web Administration console using FireFox
   v 3.*.
   ****************************************************************
   PROBLEM DESCRIPTION:
   Users using FireFox 3 cannot see the 'Cancel', 'Close',
   'Save', and 'Save and Close' buttons at the bottom of the IS
   web Administration console screens. Only the very tops of the
   buttons might be visible.  The licensing upload button does not
   function as well.  This is due to incompatibility issues between
   versions 2 and 3 of the FireFox browser.
   ****************************************************************
   RECOMMENDATION:
   Options to resolve the issue after having upgraded to Firefox 3
   are:
   1. Use the IE browser instead.
   2. Downgrade to FireFox 2
   3. Upgrade to 8.1 Fixpack 1.
   ****************************************************************
   
   
    
   
   

PROBLEM CONCLUSION

TEMPORARY FIX
