Title: IBM IZ42001: AVAILABILITY COUNTERS AND PIDS ARE ALWAYS 0 ON WINDOWS - United States

Text:

APAR STATUS
 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  Problem Description: On Windows, with Agent Builder, V6.2.1,
   the availability attributes will always show zero (0) for
   the PID and counters, instead of valid data.
   
   
   Detailed Recreation Procedure: Create and install a custom agent
   to monitor availability.
   
   Related Files and Output: None
   
   
    
   
   

LOCAL FIX
 *  There is currently no  known work-around for this problem.
   
   
    
   
   

PROBLEM SUMMARY
 *  Availability counters, including PID, always display as zero
   because the 64-bit function was not being used to retrieve the
   data.
   
   
    
   
   

PROBLEM CONCLUSION
 *  The fix for this APAR is included in the following maintenance
   vehicle:
   
      | interim fix | 6.2.1-TIV-ITM_ABLDR-IF0001
   
   Note: Search the IBM technical support web site for maintenance
   package availability.
