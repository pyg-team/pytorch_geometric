Title: IBM WebSphere DataPower SOA Appliances version 6.0.2.0 for Common Criteria use is now available! - United States

Text:
 NEWS

ABSTRACT
 WebSphere DataPower SOA Appliances firmware version 6.0.2.0 for Common Criteria use is now available for download on the following appliances: 

WebSphere DataPower B2B Appliance XB62
WebSphere DataPower Integration Appliance XI52
WebSphere DataPower Service Gateway XG45


CONTENT

This firmware contains changes in the following areas:


 * Audit Logging 
 * Password Policies 
 * SSH Cipher support restrictions


The following firmware images are available: 

xi6020.CommonCriteria.scrypt3 
xg6020.CommonCriteria.scrypt3 
xb6020.CommonCriteria.scrypt3 

Clients who want to adhere to the boundaries of the evaluated configurations used for the Common Criteria Certification can obtain the relevant Secure Deployment Guide by contacting IBM DataPower support. [http://www-01.ibm.com/support/docview.wss?uid=swg21236322]
