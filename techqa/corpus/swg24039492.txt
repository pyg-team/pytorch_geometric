Recommended : WebSphere Application Server V8.5.5 Fix Pack 5 interim fixes for the IBM Business Process Manager V8.5 Refresh Pack 6 products. 


The following recommended WebSphere Application Server V8.5.5 Fix Pack 5 interim fixes other than 8.5.5.5-WS-WASProd-IFPI36346 are already included for new IBM Business Process Manager installation media. However, they all need to be manually installed for upgrade installations. 8.5.5.5-WS-WASProd-IFPI36346 will need to be manually installed for new installations. Also, when silently installing from IBM Business Process Manager installation media the interim fixes on the media need to be added to the silent install response file or else they will not get installed. Note that IFPI35615 is platform specific and only the interim fix that matches your platform should be installed.
