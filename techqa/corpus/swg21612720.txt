Title: IBM CSV output opening in browser instead of prompting to open in Microsoft Excel - United States

Text:
 TECHNOTE (TROUBLESHOOTING)

PROBLEM(ABSTRACT)
 When running a Report Studio report to CSV output format using Internet Explorer, the user is not prompted to Open or Save the report. The CSV output opens directly in a new Internet Explorer tab. 

SYMPTOM
When using Internet Explorer, all CSV outputs are displayed in a new browser tab, instead of the user getting prompted to Open or Save the report in Microsoft Excel.


CAUSE
The Advanced Parameter RSVP.CSV.MIMETYPE must be set for the CSV output to associate with Microsoft Excel.

RESOLVING THE PROBLEM
To set the Advanced Parameter:


1. In Cognos Administration > Status tab
2. Select System
3. Under the System dropdown > Set Properties
4. Select the Settings tab
5. Select Edit... beside Environment - Advanced settings
6. Add the following entries:
Parameter = RSVP.CSV.MIMETYPE
Value = application/vnd.ms-excel
7. Select OK, and OK again to save the parameter.
