Title: IBM Product installation using DE fails with error ACUOSI0050E - United States

Text:
wscanhw; de; install; failure; prod TECHNOTE (TROUBLESHOOTING)

PROBLEM(ABSTRACT)
 Attempt to install an IBM product that uses IBM Deployment Engine (DE) fails with error ACUOSI0050E External command action failed with return code 127 

ENVIRONMENT
IBM Deployment Engine (DE) version 1.3.x.x or 1.4.0.x

RESOLVING THE PROBLEM
To resolve this issue, kindly follow the steps noted below based on your DE version. 

These location are for root installation. 

For non-root, you can find the files under /home/<user name>/.acsi(server_name)/bin 

 

For DE version 1.3.x.x:

1. Open /usr/ibm/common/acsi/bin/wscanhw file
2. Run wscanhw command again (for example: $DE_HOME/bin/wscanhw, -o, /tmp/collector6774799404675114180.tmp, -c, $DE_HOME/dat/config/scanconfig.xml )
3. After 'wscanhw' command execution completion, please re-try product installation 


For DE version 1.4.0.x:

1. Open /usr/ibm/common/acsi/bin/wscanhw file
2. Run wscanhw command again (for example: $DE_HOME/bin/wscanhw, -o, /tmp/collector6774799404675114180.tmp, -c, $DE_HOME/dat/config/scanconfig.xml )
3. After 'wscanhw' command execution completion, please retry product installation
