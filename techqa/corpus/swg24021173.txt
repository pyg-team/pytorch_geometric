ABSTRACT
 This document describes how to download IBM InfoSphere Information Server Pack for JD Edwards EnterpriseOne, Version 1.6.1.1 for IBM AIX from IBM Passport Advantage. 

DOWNLOAD DESCRIPTION


IBM Information Server Pack for JD Edwards EnterpriseOne complements JD Edwards EnterpriseOne standard software and consulting services to enable your enterprise to maximize existing investments in large ERP systems. The pack automatically generates the SQL program to extract data from a JD Edwards EnterpriseOne system. With the Pack, users can browse metadata tables to build an extraction object, then generate an extraction program written in SQL language.

Downloading the product

Your proof of entitlement directs you to your license file. This license file controls which of the product modules you can install and run.

To download the product: 

 1. Go to the Find Downloads and Media [http://www.ibm.com/software/howtobuy/passportadvantage/paocustomer/sdma/SDMA?PO=DOWNLOAD_FIND_BY_PART_NUMBER] web page and log in.
 2. Under the Download finder options, select the Find by part number search option.
 3. Search for each downloadable image by the part number.
 4. Download all the required images.
 5. Extract each of the images into a single, temporary directory on your system. Use a file extracting utility that supports large file extractions.
 6. Download the Quick Start Guide. Follow the instructions to perform the basic installation of the product.

After receiving a proof of entitlement, you can download any or all of the installation images listed below. 

Client images

Only the Microsoft Windows installation image contains both the server and the client installation software. To install the product on platforms other than Microsoft Windows, you must download both the Microsoft Windows installation image and the installation image of the platform you want to install the product.

Go to the Microsoft Windows download document [http://www.ibm.com/support/docview.wss?uid=swg24020760] for information on how to obtain the client images.
