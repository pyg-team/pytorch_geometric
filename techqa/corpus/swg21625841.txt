Title: IBM Not able to Logon to Forms Experience Builder - United States

Text:
forms experience builder not authorised CWWIM2008E TECHNOTE (TROUBLESHOOTING)

PROBLEM
Logon to Forms Experience Builder fails. When the user logged into to FEB server, the following error message is displayed. 

An error has occurred in processing your request. The error message is displayed below.
We are unable to process your request. if the error persists, report the problem to your administrator and provide error reference 

This error happened randomly. This error happened when accessing both the main FEB interface or a single application. 

After a refresh of the browser window (F5 key), access is granted correctly to the FEB.


SYMPTOM
When the user logged into to FEB server, the following error message is displayed. 

An error has occurred in processing your request. The error message is displayed below.
We are unable to process your request. if the error persists, report the problem to your administrator and provide error reference 


CAUSE
"The user isn't authorised to perform the action. User needs to have 'Administrator' and 'Admin security manager' roles."



ENVIRONMENT
Forms Server 8.0.0
