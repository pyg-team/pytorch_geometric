Title: IBM 【セキュリティ情報】IBM WebSphere MQ Internet Pass-Thru V2.0 へのTLSパディング脆弱性の影響　(CVE-2014-8730) - United States

Text:
 SECURITY BULLETIN

SUMMARY
 Transport Layer Security (TLS) には、POODLE (Padding Oracle On Downgraded Legacy Encryption)のような攻撃が行われた場合、暗号化データを解読される脆弱性があります。
IBM WebSphere MQ Internet Pass-Thru (SupportPac MS81)は、この脆弱性の影響があります。



VULNERABILITY DETAILS
脆弱性の詳細：
CVE-ID: CVE-2014-8730 [http://cve.mitre.org/cgi-bin/cvename.cgi?name=CVE-2014-8730]


DESCRIPTION: 

IBM WebSphere MQ Internet Pass-Thru V2.0 could allow a remote attacker to obtain sensitive information, caused by the failure to check the contents of the padding bytes when using CBC cipher suites of some TLS implementations. A remote user with the ability to conduct a man-in-the-middle attack could exploit this vulnerability via a POODLE (Padding Oracle On Downgraded Legacy Encryption) like attack to decrypt sensitive information and calculate the plaintext of secure connections.


CVSS Base Score: 4.3
CVSS Temporal Score: See https://exchange.xforce.ibmcloud.com/vulnerabilities/99216 [https://exchange.xforce.ibmcloud.com/vulnerabilities/99216] for the current score
CVSS Environmental Score*: Undefined
CVSS Vector: (AV:N/AC:M/Au:N/C:P/I:N/A:N)

AFFECTED PRODUCTS AND VERSIONS
影響のある製品・バージョン： 

次の製品に影響があります。 

Version 2.0 and earlier releases of IBM WebSphere MQ Internet Pass-Thru は、影響を受けます。 

Version 2.1 and later releases of IBM WebSphere MQ Internet Pass-Thru は、影響を受けません。

REMEDIATION/FIXES
修正策：


IBM WebSphere MQ Internet Pass-Thru V2.0 and earlier 

IBM WebSphere MQ Internet Pass-Thru SupportPac MS81 [http://www-01.ibm.com/support/docview.wss?uid=swg24006386]を使用されている場合、Version 2.1.0.0 以降にアップグレードしてください。
