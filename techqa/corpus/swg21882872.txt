Title: IBM Unable to move a custom page in DASH - United States

Text:
DASHL2; cannot ; move; custom; page TECHNOTE (TROUBLESHOOTING)

PROBLEM(ABSTRACT)
 In Dashboard Application Service Hub (DASH), a custom page cannot be moved inside the same custom folder (Up/Down action) or into another folder (Previous Folder/Next Folder action) because the system hangs, and no folder list is available. 

SYMPTOM
The pages pane on the right-hand side hangs and a custom location cannot be selected to move the page into: 

[/support/docview.wss?uid=swg21882872&aid=1] [/support/docview.wss?uid=swg21882872&aid=1]






CAUSE
Root cause is not known. 



RESOLVING THE PROBLEM
Upgrade to the latest version of DASH: 

http://www-01.ibm.com/support/docview.wss?uid=swg24038326 [http://www-01.ibm.com/support/docview.wss?uid=swg24038326] 

Download and install the latest version of DASH 3.1.1 (3.1.1.0-201411062200)
