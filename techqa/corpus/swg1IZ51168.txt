   *********************
   Customer cannot dicover and configure SQL Server instances to be
   monitored if the Box has two different version of SQL Server
   instaleld: MSSQL 2000 and MSSQL serevr 2005
   
   Current Limitation of the SQL Server agent is that it can
   monitor only

   different
   version of MS SQL Server has installed on the same machine.
   
   So from MS SQL Server 2005 and MS SQL Server 2000 it will
   monitor  instances of only one Version of SQL Server
