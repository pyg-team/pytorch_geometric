PROBLEM CONCLUSION
 *  APAR  PO02471 is addressed in 6.0.5.3 iFix 7.
   See Release Note :
   http://www-01.ibm.com/support/docview.wss?uid=swg27041595 [http://www-01.ibm.com/support/docview.wss?uid=swg27041595]
   Previously, an issue existed where the threads could
   potentially hang while loading user tab sessions.
    This was due to the document builder instances created in the
   XmlHelper class which were not cached and reused per thread.
   This issue has been resolved.
