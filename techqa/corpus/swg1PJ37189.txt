Title: IBM PJ37189: "IS EMPTY" or "IS NOT EMPTY" search criteria for string properti es returns all documents, regardless of the property value. - United States

Text:

APAR STATUS
 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  When using "is empty" or "is not empty" search criteria for a
   string property in Advanced Search, all documents are returned
   and the search criteria is disregarded.
   
   How long has the problem been occurring (any recent changes)?
   Since Workplace XT 1.1.4.2
   
   Is there a workaround?
   Search Template works
   
   What is the impact to the customer/system?
   Cannot use "is empty" or "is not empty" in Advanced Search
   
   Steps to Reproduce:
   1 Go to Advanced Search.
   2 Select "is empty" or "is not empty" operator for a string
   property criterion.
   3 Search.
   
   Actual Results: All documents are returned when no other
   property criterion is specified.
   Expected Results: Only documents with proper values are
   returned.
   
   
    
   
   

LOCAL FIX

PROBLEM SUMMARY
 *  ****************************************************************
   * USERS AFFECTED:                                              *
   * Users searching for documents that have empty or non-empty   *
   * string properties.                                           *
   ****************************************************************
   * PROBLEM DESCRIPTION:                                         *
   * All documents are returned when searching for a string       *
   * property that is empty or is not empty.                      *
   ****************************************************************
   * RECOMMENDATION:                                              *
   * Upgrade to 1.1.4.4-WPXT-FP004 or higher.                     *
   ****************************************************************
   
   
    
   
   

PROBLEM CONCLUSION
 *  In 1.1.4.4-WPXT-FP004, search results will contain the
   
   documents that have an empty or non-empty string property as
   specified by the search criteria.
