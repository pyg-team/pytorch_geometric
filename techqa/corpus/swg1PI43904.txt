APAR STATUS
 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  You will not be able to import or export a component with a tag.
   When exporting a component that has a tag, or an application
   that contains a component with a tag, that component tag is not
   exported and therefore will not be able to be imported.
   Currently,
   component template tags and application tags are exported and
   imported, but component tags are not.
