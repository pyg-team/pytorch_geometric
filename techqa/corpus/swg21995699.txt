Title: IBM The Case Manager Platform Installation Tool (CMPIT) 5.2.1 failed on step (32/34) configICM.bat with java.lang.IllegalThreadStateException: process has not exited - United States

Text:
ICM 5.2.1; CMPIT; step 32; Configmgr_cl TECHNOTE (TROUBLESHOOTING)

PROBLEM(ABSTRACT)
 Using information from http://www-01.ibm.com/support/docview.wss?uid=swg24037966 [http://www-01.ibm.com/support/docview.wss?uid=swg24037966] technote to install CMPIT 5.2.1 failed on step (32/34) configICM.bat with java.lang.IllegalThreadStateException: process has not exited 

SYMPTOM
Here is the screen shot when it failed at 32/34 step : 

[/support/docview.wss?uid=swg21995699&amp;aid=1]

Here is the excerpt from the CMPIT 5.2.1 log :

Install-InvokeDescription: (32/34) configICM.bat
invokeScript: C:\Program Files (x86)\IBM\cmpit\install-scripts\configICM.bat

Cmd: "C:\Program Files (x86)\IBM\cmpit\install-scripts\configICM.bat" "C:\Program Files (x86)\IBM\cmpit\install-scripts" "C:\Program Files (x86)\IBM\cmpit\decompressed" "D:\CMPIT" 64 "C:\Program Files (x86)\IBM\cmpit"

***IllegalThreadStateException - Timeout Error in monitorProcess: (32/34) configICM.bat. java.lang.IllegalThreadStateException: process has not exited
ERROR in InvokeScripts-function - Terminate the remaining scripts.
***Aborting InvokeScripts-Install function.


CAUSE
Configmgr_cl is looking for JRE in C:\Program Files\IBM\CaseManagement\java\sdk\bin directory instead of C:\Program Files (x86)\IBM\CaseManagement\java\sdk\bin directory.


ENVIRONMENT
Windows, CMPT 5.2.1



RESOLVING THE PROBLEM
 

 1. Modify the C:\Program Files (x86)\IBM\CaseManagement\configure\configmgr.ini and configmgr_cl.ini files to point to C:/Program Files (x86)/IBM/CaseManagement/java/sdk/bin and save the files. 
 2. Remove the cpit.properties file from the CMPIT folder where you run 5.2.1-CMPIT-WIN.EXE.exe and replace it with the cpit.properties from C:\Program Files (x86)\IBM\cpit\install-scripts. This will cause the installer to start at step 32 again. 
 3. Re-run 5.2.1-CMPIT-WIN.EXE.exe.
