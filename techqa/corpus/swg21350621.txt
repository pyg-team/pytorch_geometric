Title: IBM Common Logon (Hex 0x80070005): Access is denied - United States

Text:
 TECHNOTE (TROUBLESHOOTING)

PROBLEM(ABSTRACT)
 When accessing Access Manager Administration to create users, clicking on the login button produces an error. 

SYMPTOM
Common Logon (Hex 0x80070005): Access is denied


ENVIRONMENT
The user used to install the application and who has the active session is a local administrator.

Software was installed from a terminal Server client session.

RESOLVING THE PROBLEM
Set Data Execution Prevention option to 'Turn on DEP for essential windows programs and services only'
