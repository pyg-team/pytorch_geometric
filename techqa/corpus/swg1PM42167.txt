ERROR DESCRIPTION
 *  The following exception is thrown by the command
   deployEventService in a particular configuration described
   below:
   com.ibm.events.admintask.CeiApplicationAdminTasks
   processApplicationTargets error saving updated application
   information for app EventService
   
   com.ibm.websphere.management.exception.AdminException:
   ADMA5026E: No valid target is specified in ObjectName
   WebSphere:cell=<cell>,node=<node>,server=<server> for module
   EventServerMdb.jar+META-INF/ejb-jar.xml.
   at
   com.ibm.ws.management.application.task.ConfigRepoHelper.getServe
   rNames(ConfigRepoHelper.java:2195)
   ...
   
   The configuration in which the error appears is a profile that
   already has the EventService application installed but there
   is no server on that profile on which the application is
   installed.
   
   
    
   
   

LOCAL FIX

PROBLEM SUMMARY
 *  ****************************************************************
   * USERS AFFECTED:  All users of IBM WebSphere Application      *
   *                  Server V8.0 reinstalling Common Event       *
   *                  Infrastructure (CEI) after deletion         *
   *                  of the server                               *
   ****************************************************************
   * PROBLEM DESCRIPTION: The deployEventService command          *
   *                      returns the exception "No valid         *
   *                      target is specified in ObjectName"      *
   ****************************************************************
   * RECOMMENDATION:                                              *
   ****************************************************************
   The following exception is thrown by the command
   deployEventService in a particular configuration described
   below:
   com.ibm.events.admintask.CeiApplicationAdminTasks
   processApplicationTargets error saving updated application
   information for app EventService
   com.ibm.websphere.management.exception.AdminException:
   ADMA5026E: No valid target is specified in ObjectName
   WebSphere:cell=<cell>,node=<node>,server=<server> for module
   EventServerMdb.jar+META-INF/ejb-jar.xml.
   at
   com.ibm.ws.management.application.task.ConfigRepoHelper.getServe
   rNames(ConfigRepoHelper.java:2195)
   ...
   The configuration in which the error appears is a profile that
   already has the EventService application installed but there
   is no server on that profile having the application installed
   on.
   
   
    
   
   

PROBLEM CONCLUSION
 *  The problem was solved by providing default targets to be used
   if the application is installed on the profile but not on the
   current server.
   
   The fix for this APAR is currently targeted for inclusion in
   fix pack 8.0.0.1. Please refer to the Recommended Updates
   page for delivery information:
   http://www.ibm.com/support/docview.wss?rs=180&uid=swg27004980 [http://www.ibm.com/support/docview.wss?rs=180&uid=swg27004980]
