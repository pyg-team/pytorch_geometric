Title: IBM Tivoli Integrated Portal command to list users in repositories - United States

Text:
TIPL2FAQ; TIPL2SEC; list; users ; repositories TECHNOTE (FAQ)

QUESTION
 How can I list all the users in Federated repositories configured in TIP? 

ANSWER
Go to the following dir and run below command. It should give you a list of all the users.


cd <TIPHOME>/profiles/TIPProfile/bin .

./wsadmin.sh -user tipadmin -password tipadmin -lang jython -c "print AdminTask.searchUsers('[-principalName *]')". 

Example; 

Below example will return all the users which has user Id beginning with n.
./wsadmin.sh -user tipadmin -password tipadmin -lang jython -c "print AdminTask.searchUsers('[-principalName n*]')"
