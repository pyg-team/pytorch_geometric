Title: IBM Manually remove a failed DC configuration from Netweaver. - United States

Text:
CAMJ2ALL CAMJ2INST Manually remove DC configuration Netweaver TECHNOTE (TROUBLESHOOTING)

PROBLEM(ABSTRACT)
 I have configured a data collector on my Netweaver application server. The installation completes successfully but fails to correctly configure my server. The server cannot be started so its not possible to use the 'Unconfigure' option to remove the DC. 

CAUSE
There are no backup files the customer can use for a restore as all the start
parameters are stored in the database of Netweaver Application server. 

RESOLVING THE PROBLEM
Use the following steps to manually remove the DC entries. These will be found
in the GUI tool launched by running ./configtool.sh. In the left panel select
the server name and on the right side select the General tab


(Example view - you can find the "check mark" with circle below, un-check them to disregard it)
