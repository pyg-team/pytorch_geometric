ABSTRACT
 This Interim Fix provides product corrections and security updates for IBM Cognos Disclosure Management 10.2.5. 

DOWNLOAD DESCRIPTION
Cognos Disclosure Management 10.2.5 Interim Fix 7 is an update of IBM Cognos Disclosure Management 10.2.5 for existing IBM Cognos Disclosure Management customers. This Interim Fix addresses security vulnerabilities and includes important product corrections detailed in the Problem Solved section below.


Please refer to the following Security Bulletin for more details: 
IBM Cognos Disclosure Management could allow a malicious attacker to execute commands as a lower privileged user that opens a malicious document (CVE-2016-6077) [http://www.ibm.com/support/docview.wss?uid=swg21991584]


This Interim Fix contains also third party updates, most of them around the report generation engine.

PREREQUISITES
This Interim Fix requires that Cognos Disclosure Management 10.2.5 is already installed. An existing installation may optionally include earlier interim fixes for Cognos Disclosure Management 10.2.5.

Our interim fixes are cumulative by design. Hence Cognos Disclosure Management 10.2.5 Interim Fix 7 includes all updates from earlier 10.2.5 interim fixes.
