Title: IBM How do I change the prolog to show standalone="yes" in my XML output file? (SCI14677) - United States

Text:
Mapping; Translation; STERLINGPRI TECHNOTE (FAQ)

QUESTION
 How do I change the prolog to show standalone="yes" in my XML output file? (SCI14677) 

ANSWER
To get the value 'standalone = "yes"' in the prolog of the XML document

1. Right click on the XML File Format Object (Output side of map)
2. Click the 'Output' tab
3. In the section 'XML Prolog and Document Type Declaration', select the 'No prolog or document type declaration' radio button.
