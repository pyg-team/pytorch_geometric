Title: IBM Using WebSphere Application Server version 6.1 with  Rational Application Developer version 8.5 - United States

Text:
WebSphere Application Server 6.1; installation issue of WebSphere Application Server version 6.; 1 in RAD 8.5 TECHNOTE (FAQ)

QUESTION
 Why can I not find WebSphere Application Server v6.1 in the Rational Application Developer bundle version 8.5?
Can I install a WebSphere Application server 6.1 Test environment that was shipped with previous version of Rational Application Developer?
Can I still test my application in Rational Application Developer version 8.5 using WebSphere Application Server version 6.1?


CAUSE
Rational Application Developer version 8.5 no longer ships with the WebSphere Application Server version 6.1. Test Environment 

 Rational Application Developer version 8.5 no longer supports the installation of any
WebSphere Application Server version 6.1 Test environment shipped with previous versions of Rational Application Developer.






ANSWER
Installation 

 

 If you try to install a WebSphere Application Server version 6.1 Test environment that was shipped in previous version of Rational Application Developer, you will get into the following error message:

 *  You can connect to a previously installed WebSphere Application server version 6.1 Test environment under a previous version of Rational Application Developer. 
 *  You can publish and debug your application using the WebSphere Application Server version 6.1. instances
