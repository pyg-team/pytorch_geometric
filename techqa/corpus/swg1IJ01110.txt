ERROR DESCRIPTION
 *  When attempting to use right click tools to run a report there
   is a pop up error and the report does not run.
   
   The attempt results in a popup message with follow contents:
   "DPR-ERR-2079 Firewall
   Security Rejection. Your request was rejected by the security
   firewall.
   "
   
   ITNM Version:  4.2
   Build Level:     Tested on FP2, FP3
   Operating System:  Linux
   Database and Version:   DB2 10.5
   
   Component (if known):    ITNM Reports
   
   APAR being requested:  Yes
   Discussed with SME:      n/a
   
   List of Test Fixes:
   
   
   
   Replicated locally?:     yes
   
   
   Replication steps:
   1)Login to GUI
   2)Open reporting, itnm reports, performance report,
   snmpinbandwidth
   3) Set options, choose a device (remember this device)
   4)Run report and it is successful
   5)Go to HopView or Network Views, find same device as in step 3
   6)Right click the device and choose run report - snmpinbandwidth
   7)Observe error "DPR-ERR-2079 Firewall
   Security Rejection. Your request was rejected by the security
   firewall"
   
   
    
   
   

LOCAL FIX
 *  Run report from within common reporting.
