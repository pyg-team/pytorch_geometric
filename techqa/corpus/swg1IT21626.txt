APAR STATUS
 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  When using the Web Services Management Agent in DataPower under
   very heavyload with APIC, memory might temporarily increase
   without bound if the subscriber is slow to consume the produced
   records.
   
   
    
   
   

LOCAL FIX
 *  add an activity-log policy with content set as none in APIC
   
   
    
   
   

PROBLEM SUMMARY
 *  Affected is use of Web Services Management Agent in DataPower
   under very heavyload with APIC.
   Memory might temporarily increase without bound if the
   subscriber is slow to consume the produced records.
   
   
    
   
   

PROBLEM CONCLUSION
 *  Fix is available in7.1.0.20, 7.2.0.17, 7.5.0.11, 7.5.1.10,
   7.5.2.10 and 7.6.0.3
