Title: IBM System Requirements for InfoSphere Information Server Hypervisor Enterprise Edition Version 9.1 - United States

Text:
 DETAILED SYSTEM REQUIREMENTS

ABSTRACT
 This document contains system requirements for IBM InfoSphere Information Server Hypervisor Edition Version 9.1, which is a virtual image that contains InfoSphere Information Server and Red Hat Enterprise Linux Server 6.2 x86-64 or AIX Version 7.1. The image also includes the necessary GCC C++ compiler 4.4.6 on REDHAT Linux or IBM XL C/C++ compiler 11.1.0.7 for AIX. 

CONTENT


This document describes the minimum product levels that need to be installed before opening a problem report with the InfoSphere Information Server support team. Because other products frequently ship fixes, updates, and new releases, testing every configuration is not possible. In general, you can install and run with updates to supported products if those updates are forward compatible.
