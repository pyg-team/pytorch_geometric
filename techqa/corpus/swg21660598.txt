Title: IBM Sametime 8.5.2 client fix is available for Mac OS X 10.9 and Windows 8.1 - United States

Text:
 NEWS

ABSTRACT
 A Sametime 8.5.2 fix is available for clients running on Mac OS X 10.9 and Windows 8.1. Users with these operating systems should update Sametime using this fix. Users with earlier versions of these operating systems, such as Windows 8 or Mac 10.8, can use their existing Sametime build but applying the fix will also work. 

Note that Windows Metro is not supported by this fix. 

CONTENT

Prerequisite

Before installing this fix on Mac OS X 10.9 or Windows 8.1, install Notes 8.5.3 + FP6.

Download Links

Download the fix for your operating system from IBM Fix Central using the following links:

Installation instructions

Windows

1. Close the Sametime client.

2. Navigate to the folder where you downloaded the fix installation file.

3. Double-click the fix installation file to start it:

Fix for stand-alone client
sametime.hotfix.win32_20131203-0730.exe

Fix for embedded client
sametime.embedded.addon.win32_20131203-0730.exe

4. Choose whether to remove temporary files when the installation is complete, and click Next.

5. Select a language and click Next.

6. Click Accept to accept the license agreement, and then click Next.

7. Click Install to begin the installation.

8. When the install completes, click Finish.

Mac OS X

The fixes for both the stand-alone client and the embedded client are provided as compressed TAR files. When you extract the appropriate TAR file, you will see standard PKG files.

Fix for stand-alone client
sametime.hotfix.macosx_20131203-0730.tar

Fix for embedded client
sametime.10.8.embedded.addon.macosx_20131203-0730.tar

Refer to the Apple installer Manual page for options and parameters that can be used:
http://developer.apple.com/mac/library/documentation/Darwin/Reference/ManPages/man8/installer.8.html [http://developer.apple.com/mac/library/documentation/Darwin/Reference/ManPages/man8/installer.8.html]
