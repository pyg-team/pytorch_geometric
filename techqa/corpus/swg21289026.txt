QUESTION
 How can floating licenses be configured with a non-Admin user account for IBM速 Rational速 Software Delivery Platform (SDP) products? 

ANSWER
IBM Installation Manager (IM) only allows Administrator-level users to install, uninstall, and manage licenses for SDP products. Therefore, to configure floating license for an installed product, non-Admin users must use the following workaround. 

First, create a license key file.
Use the following template to create a license.opt file 



<?xml version="1.0" encoding="UTF-8"?>

<license>

   <flex enabled="true" readonly="false">

     <server name="server #1" port="port #"/> 

 ...
 

     <server name="server #5" port="port #"/>

   </flex>

   <offeringorder readonly="false">

     <offering id="{Product Package Name} #1" version="{Product Version}" name="{Product  ID} #1"/>

 ...

     <offering id="{Product Package Name} #5" version="{Product Version}" name="{Product ID} #5"/>

   </offeringorder>

   <componentorder readonly="false"/> 

</license>
 


Example using IBM Rational Application Developer for WebSphere速 Software: 




<?xml version="1.0" encoding="UTF-8"?>

<license>

   <flex enabled="true" readonly="false">

     <server name=" 10.0.1.1" port="27000"/>

   </flex>

   <offeringorder readonly="false">

     <offering id="com.ibm.rational.application.developer" version=" 7.0.0" 

name="IBM Rational Application Developer"/>

   </offeringorder>

   <componentorder readonly="false"/>

</license>
 

 * 

Second, copy the license.opt file.  * 
 * 
 * C:\Documents and Settings\All Users\Application Data\IBM\Installation Manager\license 
 * For Linux速, copy to: /var/ibm/InstallationManager/license


and restart the installed product.
