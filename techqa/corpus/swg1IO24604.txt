APAR STATUS
 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  In IBM Content Navigator, when building searches or adding
   documents, if the user is not authorized to access the Document
   class, then the list of classes in the drop-down includes
   classes that are not accessible for the user.
   
   
    
   
   

LOCAL FIX
 *  Provide access to the user/group on the Document class.
   
   
    
   
   

PROBLEM SUMMARY
 *  When building searches or adding documents, the list of classes
   in the drop-down includes classes that are not accessible for
   the user.
   
   
    
   
   

PROBLEM CONCLUSION
 *  This has been fixed in IBM Content Navigator V2.0.3 Fix Pack 8,
   V3.0, and V3.0.1 or later.
