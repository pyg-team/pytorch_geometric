Title: IBM Pure Data System for Analytics Maintenance tasks for Clients - United States

Text:
 TECHNOTE (FAQ)

QUESTION
 IBM Pure Data System for Analytics Maintenance tasks for Clients 

ANSWER
 
There are several maintenance tasks that are required to keep your IBM PureData System for Analytics appliance running smoothly. Until now, IBM support has occasionally been engaged to assist in the completion of some of these simpler tasks. IBM has prepared documentation to assist clients in completing these simple tasks independently, without the need to engage IBM support. Effective March 15, 2017 should any of the following tasks be requested, IBM Support will refer clients to appropriate documentation to assist in completion of the task. Please note that IBM Support will continue to be available via PMR to assist, should a client encounter any issues the the execution of these tasks. Below you will find a summary of the tasks and reference to documentation on how each task can be completed.
