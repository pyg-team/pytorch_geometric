WebSphere Application Server V8.0 is an application foundation tailored for single server to moderately sized configurations of departmental or large-scale, dynamic web applications requiring web tier clustering and failover across up to five application server instances, and a variety of pricing alternatives, including socket-based pricing. Centralized administration of WebSphere Application Server is available through the Flexible Management feature of WebSphere Application Server Network Deployment.

WebSphere Application Server, Version 8.0, is installed on IBM i as option 2 of licensed program 5733W80, WebSphere Application Server V8.0 for IBM i.
