Title: IBM Security Bulletin: Multiple vulnerabilities in IBM® Java™ Runtime may affect IBM Mobile Connect as a product bundler. - United States

Text:
 SECURITY BULLETIN

SUMMARY
 There are multiple vulnerabilities in IBM® Runtime Environment Java™ Version 7 bundled with IBM Mobile Connect for use by the Gatekeeper component. 

VULNERABILITY DETAILS
If you run your own Java™ code using the IBM® Java™ Runtime delivered with this product, you should evaluate your code to determine whether the complete list of vulnerabilities are applicable to your code. For a complete list of vulnerabilities please refer to IBM Java SDK Security Bulletin [http://www.ibm.com/support/docview.wss?uid=swg21986642]

AFFECTED PRODUCTS AND VERSIONS
IBM Mobile Connect (Gatekeeper component only) 6.1.5, 6.1.5.1, 6.1.5.2

REMEDIATION/FIXES
An update is available to IBM Mobile Connect and the IBM® Java™ Runtime Environment which is shipped with IBM Mobile Connect.
