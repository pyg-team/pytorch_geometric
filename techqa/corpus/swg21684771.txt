Title: IBM Security Bulletin: Incorrect authorization in IBM Business Process Manager (BPM) Saved Search Admin (CVE-2014-4802) - United States

Text:
 SECURITY BULLETIN

SUMMARY
 When you create and run a saved search from the Saved Search Admin tab of the Process Admin Console, the result set might contain tasks or instances that the current user is not authorized to see. 

VULNERABILITY DETAILS
CVE ID: CVE-2014-4802 [https://cve.mitre.org/cgi-bin/cvename.cgi?name=cve-2014-4802]
DESCRIPTION: 
IBM Business Process Manager Saved Search Admin allows authenticated users to create named queries that might expose an unfiltered result set that the user is not authorized to see.

Note: The unauthorized user is not able to perform any actions on tasks or processes in the result list.

CVSS:
CVSS Base Score: 4.0
CVSS Temporal Score: See https://exchange.xforce.ibmcloud.com/vulnerabilities/95304 [https://exchange.xforce.ibmcloud.com/vulnerabilities/95304] for the current score
CVSS Environmental Score*: Undefined
CVSS Vector: (AV:N/AC:L/Au:S/C:P/I:N/A:N)

AFFECTED PRODUCTS AND VERSIONS
