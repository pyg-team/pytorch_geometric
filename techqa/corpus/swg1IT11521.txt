ERROR DESCRIPTION
 *  Product documentation is incomplete for Data Protection for
   VMware -
   File Restore functionality.
   Although it is possible to use the File Restore functionality
   (browsing the File Restore GUI at
   https://TDPforVE_hostname:9081/FileRestoreUI) on Linux and
   Windows platforms, Data Protection for VMware needs to be
   installed on a supported Windows platform to enable the File
   Restore capabilities.
   Tivoli Storage Manager Versions Affected: Data Protection for
   VMware 71L
   Initial Impact: Low
   Additional Keywords: TSM, Spectrum, VMware, FR, Linux, GUI
   
   
    
   
   

LOCAL FIX
 *  Verify DCF 1902367 at
   http://www-01.ibm.com/support/docview.wss?uid=swg21902367 for
   correct prerequisites.

   Clarification of supported environment for file restore
   operations
   The following Tivoli Storage Manager for Virtual Environments:
   Data Protection for VMware version 7.1.3 Knowledge Center topics
   require clarification regarding the supported environment for
   file restore operations:
   
   "Tivoli Storage Manager file restore interface"
   The following URL is incorrect:
   Accessing the Tivoli Storage Manager file restore interface
   To access the file restore interface, open a web browser and
   enter the URL provided by your administrator. For example:
   https:\\hostname:9081\FileRestoreUI [https:\\hostname:9081\FileRestoreUI]
   The correct URL is https://hostname:9081/FileRestoreUI [https://hostname:9081/FileRestoreUI]
   
   "Installable components"
   File restore interface: The checkmark in the Linux column is
   incorrect. The checkmark applies to Windows only and the
   following note is missing:
   Although the file restore interface component must be installed
   and enabled on a Windows system, you can use this interface to
   restore files on both Windows and Linux guest virtual machines.
