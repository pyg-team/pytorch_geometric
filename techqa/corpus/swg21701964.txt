Title: IBM Security Bulletin:  A security vulnerability has been identified in IBM Cognos BI 10.2, IBM Cognos Disclosure Management 10.2.2.1 and IBM InfoSphere Information Server 9.1 shipped with QR for Solvency II V 1.0.0.4 and QR for Solvency II 1.0.1 (CVE-2014-3 - United States

Text:
 SECURITY BULLETIN

SUMMARY
 IBM Cognos BI 10.2, IBM Cognos Disclosure Management 10.2.2.1 and IBM InfoSphere Information Server 9.1 are shipped as components of QR for Solvency II V 1.0.0.4 and QR for Solvency II 1.0.1. Information about a security vulnerability affecting IBM Cognos BI 10.2, IBM Cognos Disclosure Management 10.2.2.1 and IBM InfoSphere Server 9.1 has been published in security bulletins. 

VULNERABILITY DETAILS
Please consult the following security bulletins for vulnerability details and information about fixes: 

IBM Cognos Business Intelligence Server is affected by multiple vulnerabilities (CVE-2014-3566, CVE-2014-6145, CVE-2014-1568, CVE-2014-4263, CVE-2012-5784, CVE-2014-3513, CVE-2014-3567 and CVE-2014-3568) [http://www-01.ibm.com/support/docview.wss?uid=swg21692267]

Vulnerability in SSLv3 affects IBM Cognos Disclosure Management (CVE-2014-3566) [http://www-01.ibm.com/support/docview.wss?uid=swg21688271]

Multiple vulnerabilities in IBM Java SDK affect IBM InfoSphere Information Server (CVE-2014-6457, CVE-2014-6558, CVE-2014-3566, CVE-2014-3065, CVE-2014-6468) [http://www-01.ibm.com/support/docview.wss?uid=swg21693199]
