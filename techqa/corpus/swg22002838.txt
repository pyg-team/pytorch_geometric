Title: IBM WebSphere Application Server における クロスサイトリクエストフォージェリ (CSRF)の脆弱性の影響について(CVE-2017-1194) - United States

Text:
 SECURITY BULLETIN

SUMMARY
 WebSphere Application Server でOAuthサービス・プロバイダーを構成している場合、クロスサイトリクエストフォージェリ (CSRF) の脆弱性の影響を受ける可能性があります。 

VULNERABILITY DETAILS
最新の情報については、下記URLの文書（英語）をご参照ください。
IBM Security Bulletin: Cross-site request forgery in WebSphere Application Server (CVE-2017-1194)http://www-01.ibm.com/support/docview.wss?uid=swg22001226 [http://www-01.ibm.com/support/docview.wss?uid=swg22001226]



CVEID: CVE-2017-1194 [http://cve.mitre.org/cgi-bin/cvename.cgi?name=CVE-2017-1194]
DESCRIPTION: IBM WebSphere Application Serverは、 クロスサイトリクエストフォージェリ (CSRF)に対して脆弱性があり、悪意のある攻撃者が、権限のないアクションを実行できる可能性があります。

CVSS Base Score: 4.3
CVSS Temporal Score: See https://exchange.xforce.ibmcloud.com/vulnerabilities/123669 [https://exchange.xforce.ibmcloud.com/vulnerabilities/123669] for the current score
CVSS Environmental Score*: Undefined
CVSS Vector: (CVSS:3.0/AV:N/AC:L/PR:N/UI:R/S:U/C:N/I:L/A:N) AFFECTED PRODUCTS AND VERSIONS
この脆弱性は、WebSphere Application Server の下記のバージョンとリリースを使用している場合に影響を受けます。 

 * Liberty 
 * Version 9.0 
 * Version 8.5 
 * Version 8.0 
 * Version 7.0

REMEDIATION/FIXES
推奨される解決方法は、個別修正モジュールAPAR PI77770、または、個別修正モジュールを含むフィックスパックを適用することです。
