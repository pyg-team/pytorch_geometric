 DOWNLOADABLE FILES

ABSTRACT
 This mod pack updates Operational Decision Manager V8.8.0 to V8.8.1. 

DOWNLOAD DESCRIPTION
This mod pack is entitled. Only customers who have purchased Operational Decision Manager V8.8.0 can use this mod pack.


How to install the mod pack
For each Operational Decision Manager component that you use: 

 1. Download its file in the following table. Select the file for your operating system by changing <OS> to match your platform: AIX, HPUX, LIN_PPC, LIN_S390, LIN_X86, MAC, OS400, SOL_SPARC, SOL_X86, or WIN. 
 2. Extract the contents of the downloaded file to a new folder. Use the corresponding folder name in the table. 
 3. If you use WebSphere Application Server, download the profile templates for your component.

Component Download file Folder name (InstallDir) Note Decision Server Insights 8.8.0-WS-ODM_DSI-<OS>-RP001.zip DSI Only available for Linux (LIN) and Windows (WIN). Decision Center 8.8.0-WS-ODM_DC-<OS>-RP001.zip WDC For Windows and Linux, use <install path of ODM 8.8.0>/jdk/bin/jar xf to extract files in the WDC directory. Decision Server Rules 8.8.0-WS-ODM_DSR-<OS>-RP001.zip DSR [/support/docview.wss?uid=swg24042247&amp;aid=1] Decision Server Events 8.8.0-WS-ODM_DSE-<OS>-RP001.zip DSE [/support/docview.wss?uid=swg24042247&amp;aid=1] Decision Center profile templates for WebSphere Application Server 8.8.0-WS-ODM_PTDC-RP001.zip WDC_PT [/support/docview.wss?uid=swg24042247&amp;aid=1] Decision Server Rules profile templates for WebSphere Application Server 8.8.0-WS-ODM_PTR-RP001.zip DSR_PT [/support/docview.wss?uid=swg24042247&amp;aid=1] Decision Server Events profile templates for WebSphere Application Server 8.8.0-WS-ODM_PTE-RP001.zip DSE_PT [/support/docview.wss?uid=swg24042247&amp;aid=1] Installing with IBM Installation Manager:  1. Launch IBM Installation Manager. 
 2. Open File > Preferences > Repositories. 
 3. For each component that you use: 
    a. Click Add Repository.
    b. Browse to the extracted folder <InstallDir>/updates.
    c. Select the repository.config file. 
 4. Make sure the repositories are selected in Repositories. 
 5. Click Apply to save your changes, and click OK to close Repositories. 
 6. Click Update to update your installation. 
 7. Follow the instructions in the installation manager to complete the installation.


Installing with the Installation Manager command line:  1. Locate the imcl command under <IM installation>/eclipse/tools. 
 2. Run the commandimcl updateAll -repositories WDC/updates, DSR/updates, DSE/updates -acceptLicense.

 * Use the option -installDirectory <install path of ODM 8.8.0> to update a specific installation. 
 * If you update only one component, specify the repository accordingly.
