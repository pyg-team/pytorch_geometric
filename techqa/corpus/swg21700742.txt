Title: IBM How to add the description as a comment in the Implementation file (.cpp) for a class, instead of adding it to the .h in IBM Rational Rhapsody - United States

PROBLEM(ABSTRACT)
 In Rational Rhapsody, you can add descriptions for classes and you want to have the descriptions added to the .cpp file as comments. 

CAUSE
By default Rhapsody generates descriptions comments only into the specification file (.h). However, you can add them to the implementation file also. 

RESOLVING THE PROBLEM



1. Open the Features dialog at the project level

2. Locate the “CPP_CG::WriterTemplate::ClassImp” property 

[/support/docview.wss?uid=swg21700742&aid=1] [/support/docview.wss?uid=swg21700742&aid=1]

3. Click the ellipses “…” at the end of the entry.
In the Text Editor, add “[[$description]]” at the beginning of the line. NOTE: the format and ensure that you have the same as below i.e. add a carriage return after the “n” in description 

[/support/docview.wss?uid=swg21700742&amp;aid=2] 


4. Click OK
