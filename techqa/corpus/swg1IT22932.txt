APAR STATUS
 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  Log Targets might incorrectly attempt connection using
   previously
   configured credential on an old log target, if named with the
   same name as the new log target.
   
   
    
   
   

LOCAL FIX
 *  Create distinct Log Target names if they are configured with
   remote logins.
   
   
    
   
   

PROBLEM SUMMARY
 *  Log Targets internal users might be reused on domain/object
   restart or reset.
   
   
    
   
   

PROBLEM CONCLUSION
 *  Fix is available in 7.1.0.22, 7.2.0.19, 7.5.0.14, 7.5.1.13,
   7.5.2.13 and
   7.6.0.6
   For a list of the latest fix packs available, please see:
   http://www-01.ibm.com/support/docview.wss?uid=swg21237631 [http://www-01.ibm.com/support/docview.wss?uid=swg21237631]
