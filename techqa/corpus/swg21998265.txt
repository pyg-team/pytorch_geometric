Title: IBM The export of the configuration ini file fails when the default IBM Content Navigator desktop has the IBM for SAP plug-in disabled. - United States

Text:
 TECHNOTE (TROUBLESHOOTING)

PROBLEM(ABSTRACT)
 In this scenario, users will not be able to export an ICC for

SAP configuration.



1. The default Navigator plugin has ICC for SAP's plugin

disabled



2. Customer is using ICN 2.03 FP08 or later.



After the OK button is selected, the screen will go blank with

a plugin.do url. No 'Save As' dialog box will appear. No errors

will appear. 

RESOLVING THE PROBLEM
Do not disable the ICC for SAP plug-in in the default IBM Content Navigator
desktop.
