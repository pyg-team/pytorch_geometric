ERROR DESCRIPTION
 *  The following MIME types are exported with .DAT extensions:
   .au
   .flv
   .mp3
   .swf
   .cptx
   
   
    
   
   

LOCAL FIX

PROBLEM SUMMARY
 *  Documents stored in ECM repositories with some MIME types may
   be downloaded from Content Navigator with an extension of
   ".dat".
   
   
    
   
   

PROBLEM CONCLUSION
 *  This issue has been addressed in IBM Content Navigator V3.0
   LA004.
