APAR STATUS
 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  Using content assist for files from PDS data set takes
   too long.
   
   
    
   
   

LOCAL FIX

PROBLEM SUMMARY
 *  USERS AFFECTED:
   RD/z users who use RD/z to edit PL/I programs with include files
   or COBOL programs with copybooks.
   ****************************************************************
   *
   PROBLEM DESCRIPTION:
   When a PL/I program or COBOL program was edited in RD/z, either
   with the System z LPEX editor or with the PL/I editor or COBOL
   editor, copybooks and include files without ISPF statistics were
   downloaded twice.
   
   
    
   
   

PROBLEM CONCLUSION
 *  When editing PL/I programs or COBOL programs, copybooks and
   include files without ISPF statistics are downloaded once.
   
   The problem listed in this APAR has been resolved with
   IBM Rational Developer for System z V8.0.3.2 Fix Pack which
   is available from the Recommended Fixes support download page:
   http://www-1.ibm.com/support/docview.wss?rs=2294&uid=swg27006335 [http://www-1.ibm.com/support/docview.wss?rs=2294&uid=swg27006335]
