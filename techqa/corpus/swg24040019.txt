Title: IBM Operational Decision Manager V8.7.1 Refresh Pack - United States

 DOWNLOADABLE FILES

ABSTRACT
 This cumulative refresh pack updates Operational Decision Manager V8.7 to V8.7.1. 

DOWNLOAD DESCRIPTION
This refresh pack is entitled. Only customers who have purchased Operational Decision Manager V8.7.0 can use this refresh pack.


How to install the refresh pack
For each of your Operational Decision Manager components: 

 1. Download its file in the following table. 
    Change <OS> to match your platform: AIX, HPUX, LIN_PPC, LIN_S390, LIN_X86, OS400, SOL_SPARC, SOL_X86, and WIN. 
 2. Extract the contents of the download file to a new folder. Use the corresponding folder name in the table. 

Component Download file Folder name (InstallDir) Note Decision Server Insights 8.7.0-WS-ODM_DSI-<OS>-RP001.zip DSI Only available for Linux (LIN) and Windows (WIN). Decision Center 8.7.0-WS-ODM_DC-<OS>-RP001.zip WDC For Windows and Linux, use <install path of ODM 8.7.0>/jdk/bin/jar xf to extract files in the WDC directory. Decision Server Rules 8.7.0-WS-ODM_DSR-<OS>-RP001.zip DSR [/support/docview.wss?uid=swg24040019&amp;aid=1] Decision Server Events 8.7.0-WS-ODM_DSE-<OS>-RP001.zip DSE [/support/docview.wss?uid=swg24040019&amp;aid=1] Decision Center profile templates for WebSphere Application Server 8.7.0-WS-ODM_PTDC-Multi-RP001.zip WDC_PT [/support/docview.wss?uid=swg24040019&amp;aid=1] Decision Server Rules profile templates for WebSphere Application Server 8.7.0-WS-ODM_PTR-Multi-RP001.zip DSR_PT [/support/docview.wss?uid=swg24040019&amp;aid=1] Decision Server Events profile templates for WebSphere Application Server 8.7.0-WS-ODM_PTE-Multi-RP001.zip DSE_PT [/support/docview.wss?uid=swg24040019&amp;aid=1] Installing with IBM Installation Manager You must update simultaneously all the components that are installed in the same package in Installation Manager. For example, if you installed Decision Server Rules and Decision Server Events together, both repositories must be added to Installation Manager. 

 1. Launch IBM Installation Manager. 
 2. Open File > Preferences > Repositories. 
 3. For each component that you use: 
    a. Click Add Repository.
    b. Browse to the extracted folder <InstallDir>/updates.
    c. Select the repository.config file. 
 4. Make sure the repositories are selected in Repositories. 
 5. Click Apply to save your changes, and click OK to close Repositories. 
 6. Click Update to update your installation. 
 7. Follow the instructions in the installation manager to complete the installation.


Installing with the Installation Manager command line  1. Locate the imcl command under <IM installation>/eclipse/tools. 
 2. Run the commandimcl updateAll -repositories WDC/updates, DSR/updates, DSE/updates -acceptLicense.

 * Use the option -installDirectory <install path of ODM 8.7.0> to update a specific installation. 
 * If you update only one component, specify the repository accordingly.


Applying the refresh pack 
After you install the refresh pack, you might have to do additional steps to apply the refresh in your environment. To complete the update, see When and how to apply a modification or fix pack [http://www-01.ibm.com/support/docview.wss?uid=swg21665766]. DOWNLOAD PACKAGE
