Title: IBM 【セキュリティ情報】 IBM WebSphere MQのIBM Java Runtime の複数の脆弱性の影響  (CVE-2016-0475, CVE-2015-7575, CVE-2016-0448) - United States

Text:
 SECURITY BULLETIN

SUMMARY
 IBM WebSphere MQ で提供されているIBM® Runtime Environment Java™ Technology Editions バージョン6 & 7にセキュリティ脆弱性が存在します。
これらの問題は、2016年1月にIBM Java SDK更新の一部として公開されました。この修正には、"SLOTH"と呼ばれる脆弱性への対応を含みます。

CVSS Base Score: 5.8

WORKAROUNDS AND MITIGATIONS
回避策/軽減策： 


CVE-2015-7575の回避策/軽減策

Java 7 およびそれ以降を使用されている場合、/jre/lib/security/java.security ファイルのMD5アルゴリズムを無効化することで問題に対処することができます。(両方の手順が必要です。)


 * jdk.certpath.disabledAlgorithms プロパティに MD5 を追加します。

例: jdk.certpath.disabledAlgorithms=MD2, RSA keySize < 1024, MD5 

 * jdk.tls.disabledAlgorithms プロパティに MD5withRSA を追加します。

例:  jdk.tls.disabledAlgorithms=SSLv3, RC4, DH keySize < 768, MD5withRSA 



*Java 6はjava.securityファイル修正に加えてJSSE構成要素のコードの変更を必要とするため、JREをアップグレードする必要があります。
