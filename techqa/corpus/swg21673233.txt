Title: IBM Security Bulletin: IBM WebSphere MQ is affected by a vulnerability in IBM SDK, Java Technology Edition, Version 6 (CVE-2014-0411) - United States

Text:
 SECURITY BULLETIN

SUMMARY
 A security vulnerability exists in the IBM Java Runtime Environment component of WebSphere MQ, a patch for this is available in IBM SDK, Java™ Technology Edition, Version 6 Service Refresh 15 Fix Pack 1 

VULNERABILITY DETAILS
CVEID: CVE-2014-0411 [http://cve.mitre.org/cgi-bin/cvename.cgi?name=CVE-2014-0411]
DESCRIPTION: An unspecified vulnerability in IBM Java could allow remote attackers to affect confidentiality and integrity via vectors related to JSSE. 

 

CVSS Base Score: 4
CVSS Temporal Score: See http://xforce.iss.net/xforce/xfdb/90357 [http://xforce.iss.net/xforce/xfdb/90357] for the current score
CVSS Environmental Score*: Undefined
CVSS Vector: (AV:N/AC:H/Au:N/C:P/I:P/A:N)

AFFECTED PRODUCTS AND VERSIONS
IBM SDK, Java™ Technology Edition, Version 6 (maintenance levels older than service refresh 15) provided by; 

 * WebSphere MQ 7.1 on all platforms (except IBM i and z/OS) 
 * WebSphere MQ 7.5 on all platforms (except IBM i and z/OS)

REMEDIATION/FIXES
WebSphere MQ 7.1 
Apply fix pack 7.1.0.5 [http://www-01.ibm.com/support/docview.wss?uid=swg27024302] or later [http://www.ibm.com/support/fixcentral/swg/selectFixes?parent=ibm~WebSphere&product=ibm/WebSphere/WebSphere+MQ&release=7.1.*&platform=All&function=aparId&apars=IC99261] 


WebSphere MQ 7.5
Apply fix pack 7.5.0.4 [http://www-01.ibm.com/support/docview.wss?uid=swg27038184] when available. In the interim apply APAR IC99261 [http://www-947.ibm.com/support/entry/portal/search_results?sn=spe&filter=keywords:ibmsupportfixcentralsearch&q=IC99261]

WORKAROUNDS AND MITIGATIONS
None known
