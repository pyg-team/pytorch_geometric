ERROR DESCRIPTION
 *  Search Templates created using the Search Designer via
   WorkplaceXT that
   include a property search criteria and a Content Search critera
   when
   opened in WorkplaceXT does not show the Content Search field
   just the
   property search field(s).
   
   
   Steps to Reproduce:
   1. Log into XT.
   2. Open the Search Designer.
   3. Create a basic search that includes  -
       - One property search
       - One CBR search
   Pick your own criteria ...BUT a Content Search is required in
   the search template.
   4. Save your search to a folder in your object store.
   5. Browse XT to your new search template.
   6. Open the search template.
   
   The Content Search criteria is missing.
   
   7. Log into AE/Workplace.
   8. Locate the search template.
   9. Open the search template.
   10. The CBR criteria is back.
   
   Actual Results: CBR criteria is missing.
   
   Expected Results: CBR criteria to be available.
   
   Server OS : W2k3 sp2
   Database : MSSQL 2005 sp1 and or sp2
   Network: TCP/IP
   App Server : JBoss 4.0.5 and also WAS 6.1-009
   Browser :IE 7
   CE 4.0.1-008
   WorkplaceXT 1.1.3.3 fp3 and fp4
