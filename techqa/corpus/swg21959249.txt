QUESTION
 What is the recommended License Manager version for the use with IBM SPSS Statistics 23, Modeler 17, & Amos 23? 

ANSWER
Recommended License Manager version:
For the usage with IBM SPSS Statistics 23, Modeler 17, & Amos 23, it is recommended to install the License Manager version 9.5.0.2. With this version you can manage network licenses for older IBM SPSS software releases as well. 

The following list of IBM Software Products have the option of "Concurrent User" licensing and can be activated using the License Manager: 

 * IBM SPSS Modeler Professional 
   IBM SPSS Modeler Premium 
   IBM SPSS Statistics Desktop 
   IBM SPSS Statistics Developer 
   IBM SPSS Amos 
   IBM SPSS Text Analytics for Surveys 
   IBM SPSS SamplePower 
   IBM SPSS Visualization Designer 
   
   

License Manager download: 
On the IBM Software & Download Portal, there are several versions for the License Manager available depending on what operating system (OS) you would like to install the service. 

If you would like to install the License Manager service and the Administration tool (WlmAdmin tool) to administer the network licenses on a Microsoft Windows OS the only download package you need is:
