LOCAL FIX

PROBLEM SUMMARY
 *  The calculation of the projected path is not shown as undefined
   on the Set Path tab of the Process Performance dashboard when
   the calculation fails because of a timeout.
   
   
    
   
   

PROBLEM CONCLUSION
 *  A fix will be included in a future release of IBM BPM that
   ensures the calculation of the projected path is shown as
   undefined on the Set Path tab of the Process Performance
   dashboard when the calculation fails because of a timeout.
