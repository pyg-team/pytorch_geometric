APAR STATUS
 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  The WebSphere MQ V7.1 and V7.5 installation contains a 1.6
   Java Runtime Environments (JREs), supplied within the JRE
   install component.
   The WebSphere MQ V8.0 installation contains a 1.7
   Java Runtime Environments (JREs), supplied within the JRE
   install component.
   New revisions of this JRE are released periodically to address
   program defects, and products which embed the JRE should
   be updated accordingly.
   
   
    
   
   

LOCAL FIX

PROBLEM SUMMARY
 *  ****************************************************************
   USERS AFFECTED:
   The JRE which is embedded into WebSphere MQ is used
   to run the following Java components of the product:
   
     WebSphere MQ Managed File Transfer
     GSKit graphical interface
     WebSphere MQ Explorer
     WebSphere MQ Extended Reach (MQXR) service
   
   
   Platforms affected:
   AIX, Linux on Power, Linux on S390, Linux on x86, Linux on
   x86-64, Windows, HP-UX PA-RISC, HP-UX Itanium, Solaris SPARC,
   Solaris x86-64
   
   ****************************************************************
   PROBLEM DESCRIPTION:
   This APAR updates the Java Runtime Environment (JRE) supplied
   with WebSphere MQ 7.1 & 7.5 to the level:
   
       Java 6.0 Service Refresh 16 Fix Pack 3 (SR16-FP3) with
   interm fix
   
   And the Java Runtime Environment (JRE) supplied
   with WebSphere MQ 8.0 to the level:
   
       Java 7.1 Service Refresh 2 Fix Pack 10 (SR2-FP10) with
   interm fix
   
   
    
   
   

PROBLEM CONCLUSION
 *  This APAR updates the JRE updates for WebSphere MQ V7.1 and V7.5
   on the following platforms:
   
     AIX

     Linux (x86-32)
     Linux (x86-64)
     Linux (PPC64)
     zLinux
