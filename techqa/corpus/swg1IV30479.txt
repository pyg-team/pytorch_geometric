 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  EventProcessor event block could not be checkpointed because of
   NullPointerExceptions, then any following event blocks will hold
   in memory with no more processing by the EventProcessor.
   
   The EventReader keeps polling for events and therefore there is
   a build-up of events in the EventReader EventQueue and resulting
   OutOfMemoryError.
   
   Fix is about setting initial value and checking Null before we
   checkpoint the event block.
