IRMA License feature; FEA License key TECHNOTE (FAQ)

QUESTION
 What license key should you download to activate the IBM Rational System Architect 11.4.2 Integrated Reference Model Architecture (iRMA) features? 

CAUSE
An IBM System Architect license has been downloaded and imported, but the iRMA feature is not active.

ANSWER
In order to activate the iRMA feature associated with IBM Rational System Architect 11.4.2, you must download and import an IBM Rational System Architect Federal Enterprise Architecture (FEA) license key.
