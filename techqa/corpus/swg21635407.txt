Title: IBM Problem retrieving or downloading document from FileNet Workplace XT - United States

Text:
download document; Workplace XT; FNRCR0031E; RETRIEVE_EXCEEDED_BLOB_BUFFER TECHNOTE (TROUBLESHOOTING)

PROBLEM(ABSTRACT)
 Problem retrieving or downloading document from Workplace XT. Getting error, Exceeded maximum length of blob buffer component_types. 

SYMPTOM
 

 * Seeing error, Exceeded maximum length of blob buffer component_types, while downloading the document from Workplace XT. 
 * The same error occur while accessing the Properties of the document in FileNet Enterprise Manager. 
 * Viewing properties of the document in Workplace XT is fine. 
 * Attempt to view the document from Workplace XT returns the properties information instead. 
 * Error logged in the P8_server_error.log:

CAUSE
This issue can occur when all the CE servers are not on the same version. The working CE server is on a higher CE version compared to the problem CE server. 

RESOLVING THE PROBLEM
Ensure all the CE servers in the environment are at the same version.
