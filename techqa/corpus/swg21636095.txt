Title: IBM Installation Manager throws an error when trying to install an iFix on Business Monitor - United States

Text:
Install Manager; package not in synch TECHNOTE (TROUBLESHOOTING)

PROBLEM(ABSTRACT)
 Installation Manager throws the error message Error: 'installation manager package that is not synchronized' when you try to install an iFix on Business Monitor. 

SYMPTOM
Error: 'installation manager package that is not synchronized' is received when using the Installation Manager.


CAUSE
You may have incorrectly used the Update Installer for a WebSphere Application Server package instead of the Installation Manager.

ENVIRONMENT
All

DIAGNOSING THE PROBLEM
The Installation Manager logs may indicate which package is causing the error.

1. Contents of the IBM\WebSphere\AppServer\logs directory.
2. The NIFStack.xml and NIFHistory.xml files from the AppServer\properties\version\nif\backup directory.

RESOLVING THE PROBLEM
1. Try to re-import the WebSphere Application Server installation using the latest import repository below:
http://public.dhe.ibm.com/software/websphere/downloads/was/import/was.7.0.import.zip [http://public.dhe.ibm.com/software/websphere/downloads/was/import/was.7.0.import.zip]

2. Check and see if JDK is out of sync.
