Title: IBM Turkish inputChars() call causes "MethodNotFoundException"  "CRFCN0019E" - United States

Text:
RFT; Rational.Test.Ft.Domain.Net.Wpf.WindowProxy.InputChars; Tr; Turkish; Türkçe; Komut Dosyasi; RATLC01552174; RATLC02847324 TECHNOTE (TROUBLESHOOTING)

PROBLEM(ABSTRACT)
 Attempts to run an IBM Rational Functional Tester (RFT) script against a .NET application result in RationalTestScriptException. 

SYMPTOM
The error message in the console is as follows.

RESOLVING THE PROBLEM
WORKAROUND 

 

The following steps result in an error free playback.




 1. Switch the format to English.
    
    
 2. Start the AUT.
    
    
 3. Switch back to Turkish.
    
    
 4. Run the RFT script.
    
    


Now the script runs error free.
