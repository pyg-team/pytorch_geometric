PROBLEM(ABSTRACT)
 What are the exact DB2 privileges needed by the ITCAM MS "service account" when connecting to and using the 'octigate' database? 

RESOLVING THE PROBLEM
The user must be either of the following types: 

• A DB2 administrator user (for example, db2admin on Windows operating systems, or db2inst1 on Linux or AIX operating systems) 

• A database user specific to the ITCAM Management Server database that has at least the following authorizations: 

o Connect to the database 

o Create tables 

o Create index 

o Perform select, insert, update, and delete operations on the tables in the database.
