Title: IBM Installing and updating a fix pack for WebSphere Message Broker Explorer Version 7.0 - United States

Text:
 PRODUCT DOCUMENTATION

ABSTRACT
 This document contains the installation instructions for upgrading the IBM WebSphere Message Broker Explorer Version 7.0. 

CONTENT
 

Applying service to the WebSphere Message Broker Explorer

You can apply maintenance or fixes to the WebSphere Message Broker Explorer.

Download the latest fix pack from the WebSphere Message Broker Explorer Recommended Fixes [http://www-01.ibm.com/support/docview.wss?uid=swg27006041] Web page:

1. Click the fix that you want to download. Details about the fix and instructions for download and installation, are displayed.

2. Click Problems fixed [http://www-01.ibm.com/support/docview.wss?uid=swg27019226] to find out what APARs have been fixed in the fix pack.

3. Click FC in the Download package section to start the download process. Accept the download terms and conditions and sign in to the download site using your IBM ID. If you do not have an ID, you can register on this page.

4. Decide whether to use the graphical installation, a console installation, or a silent installation. For detailed instructions on the installation options please refer to WebSphere Message Broker documentation [http://www.ibm.com/software/integration/wbimessagebroker/library/].

5. Launch the installation wizard, using the executable file or binary file. On Windows, this file is 7.0.0-WS-MBX-WINIA32-FP0007.exe and on Linux platforms, this file is 7.0.0-WS-MBX-LINUXIA32-FP0007.bin.

6. When you have launched the wizard, continue with the installation process by working through each of the panels. The installation wizard itself contains help information on selected panels. You must install service to the same location as the existing installation on your system.
