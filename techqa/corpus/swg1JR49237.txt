Title: IBM JR49237: INFOCENTER INCLUDES THE WRONG REFERENCE OF BOOTSTRAPPROCESSSERVERDATA FOR BPM ADVANCED-ONLY - United States

Text:
 SUBSCRIBE
You can track all active APARs for this component.



APAR STATUS
 * CLOSED AS DOCUMENTATION ERROR.
    
   
   

ERROR DESCRIPTION
 *  Information Center includes the unnecessary and misleading step
   of bootstrapProcessServerData in the section of
   "Installing IBM Business Process Manager Advanced: Process
   Server using a custom installation and configuration path"
   It includes the step of
     "Loading the database with system information in a network
     deployment environment"
   The bootstrapProcessServerData command will fail if customer
   follows this step for BPM Advanced: Process Server setup.
   
   PRODUCTS AFFECTED
   IBM Business Process Manager (BPM) Advanced
   IBM BPM Standard
   IBM BPM Express
   
   
    
   
   

LOCAL FIX

PROBLEM SUMMARY
 *  The documentation should clearly indicate when it is necessary
   to load the database with system information in a network
   deployment environment for the bootstrapProcessServerData
   command.
   
   
    
   
   

PROBLEM CONCLUSION
 *  The content will be updated the next time the documentation is
   refreshed.
