Title: IBM Attempting to start the Analysis Web Client results in a "System Unsupported" error message or a blank screen in Internet Explorer - United States

Text:
Browser error; AWC; webclient; i2A TECHNOTE (TROUBLESHOOTING)

PROBLEM(ABSTRACT)
 After deploying the Analysis Web Client (AWC) as part of the IBM i2 Analyze product, starting it from Internet Explorer (IE) with the URL, http://<HostName>/awc, results in an error message: 
"System Unsupported".

IE may also display a blank screen.





[/support/docview.wss?uid=swg21991467&aid=1] [/support/docview.wss?uid=swg21991467&aid=1] 


CAUSE
 

The version of IE causing the issue is Internet Explorer 11.0.9600.16.4.28 . 

Also, tests were done on Google Chrome version 52.0.2743.116 and a similar message occurs.

[/support/docview.wss?uid=swg21991467&amp;aid=2]


Testing on FireFox version 48.0.1 worked successfully. 


RESOLVING THE PROBLEM
 

The issue can be resolved by using IE version 11.0.9600.17.9.14 or higher, such as 11.0.9600.18.4.27. These two IE versions work.
Testing on FireFox version 48.0.1 also worked successfully.
