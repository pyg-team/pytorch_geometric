ERROR DESCRIPTION
 *  DataPower sets the value of HTTP Client IP Label for a Web
   Service Proxy to the default value of X-Client-IP, each time the
   Web Service Proxy is modified from the WebGUI, whether accessed
   from the Control Panel in the WebGUI or from Services > Web
   Service Proxy > Edit Web Service Proxy.
   
   
    
   
   

LOCAL FIX
 *  Use the CLI, SOMA, or WebGUI Objects menu to configure the HTTP
   Client IP Label if it is reset to the default value.
   
   
    
   
   

PROBLEM SUMMARY
 *  Affected are DataPower customers using a non-default value for
   HTTP Client IP Label who modify the Web Service Proxy via the
   WebGUI Control Panel icon or Services> Web Service Proxy> Edit
   Web Service Proxy. The HTTP Client IP Label configuration
   reverts to the default value upon configuring the Web Service
   Proxy from the Control Panel link.
   
   
    
   
   

PROBLEM CONCLUSION
 *  Fix is available in 5.0.0.12, 6.0.0.4 and 6.0.1.1.
