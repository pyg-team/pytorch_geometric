Title: IBM API Connect V5.0.2.0 Public Interim Fix 001 is available - United States

Text:
 NEWS

ABSTRACT
 A public interim fix is available for IBM API Connect V5.0.2.0 which addresses several APARs. 

CONTENT


IBM API Connect V5.0.2.0 Public Interim Fix 001 is available. This interim fix provides important APAR fixes on top of IBM API Connect V5.0.2.0. We advise all users of IBM API Connect V5.0.2.0 to install this update to take advantage of the fixes. 
APAR FIXES

THIS IS A CUMULATIVE INTERIM FIX AND INCLUDES THE FOLLOWING APAR FIXES:
APAR Summary LI79132 V5010 THE HEADER-TO-PARAMETER CROSS MAPPING DOES NOT WORK. LI79131 API CONNECT V5020 BECOMES UNRESPONSIVE LI79129 UPGRADE TO V5020 FROM V5010 FAILING ON RSS RE-ADD TO CLUSTER UPGRADE PATHS FOR API CONNECT:
For more information of details on IBM API Connect upgrade paths, see Supported Upgrade Paths. [http://www-01.ibm.com/support/docview.wss?uid=swg21984408]
