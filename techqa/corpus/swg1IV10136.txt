ERROR DESCRIPTION
 *  Under certain circumstances Workmanager-based navigation can
   cause deadlocks on BPEDB tables.
   .
   When using Workmanager-based navigation, there is a chance of
   getting deadlocks on BPEDB tables. In particular there were
   deadlocks observed that involved the process instance table and
   the process context table of the BPEDB.
