ERROR DESCRIPTION
 *  The GNU C Library (glibc) has vulnerabilities as specified in
   CVE-2015-0235 and CVE-2014-6040.
   
   
    
   
   

LOCAL FIX

PROBLEM SUMMARY
 *  CVE-2015-0235 - The gethostbyname functions of the GNU C Library
    (glibc) are vulnerable to a buffer overflow.
   CVE-2014-6040 - The GNU C Library (glibc) is vulnerable to a
   denial of service, caused by the improper validation of input by
   the iconv() function.
   
   
    
   
   

PROBLEM CONCLUSION
 *  Fix is available in 6.0.0.12, 6.0.1.8, 7.0.0.5 and 7.1.0.3.
