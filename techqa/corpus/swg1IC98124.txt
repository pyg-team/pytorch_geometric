APAR STATUS
 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  After launching the WMB V8003 TOOLKIT launcher script, the
   launcher throws the following exception:
   
   Exception in thread "main" java.lang.NoClassDefFoundError:
   com.ibm.broker.config.proxy.LatestBrokerInstallLocation
   Caused by: java.lang.ClassNotFoundException:
   com.ibm.broker.config.proxy.LatestBrokerInstallLocation
    at java.net.URLClassLoader.findClass(URLClassLoader.java:434)
    at java.lang.ClassLoader.loadClass(ClassLoader.java:653)
    at
   sun.misc.Launcher$AppClassLoader.loadClass(Launcher.java:358)
    at java.lang.ClassLoader.loadClass(ClassLoader.java:619)
   
   Could not find the main class:
   com.ibm.broker.config.proxy.LatestBrokerInstallLocation.
   Program will exit.
   
   The system cannot find the path specified.
   
   This error occurs because the ConfigManagerProxy.jar file is
   missing from the /wmbt directory.
   
   
    
   
   

LOCAL FIX
 *  Manually copy the ConfigManagerProxy.jar from
   /plugins/com.ibm.etools.mft.config_8.0.300.v20131024-0926
   to /wmbt/ directory. Then restart the toolkit with the
   launcher script.
   
   
    
   
   

PROBLEM SUMMARY
 *  ****************************************************************
   USERS AFFECTED:
   All users of WebSphere Message Broker Toolkit V8.0.0.3 who
   launch the toolkit with the launcher script.
   
   
   Platforms affected:
   MultiPlatform
   
   ****************************************************************
   PROBLEM SUMMARY:
   The WebSphere Message Broker Toolkit V8.0.0.3 fixpack install
   script refers to an incorrect version of the
   com.ibm.etools.mft.config plugin when copying the
   ConfigManagerProxy.jar file. This results in the
   ConfigManagerProxy.jar file missing from the product install
   location. The ConfigManagerProxy.jar file is required by the
   launcher script.
   
   
    
   
   

PROBLEM CONCLUSION
 *  The product is fixed and no longer throws the error when started
   using the toolkit launcher script.
