APAR STATUS
 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  Through the WCM API and JCR Explorer tools it is possible to
   accidentally delete the "WebContent Syndication" library. If a
   user does this there is no way to get the library back.
   
   
    
   
   

LOCAL FIX

PROBLEM SUMMARY
 *  If the Syndication library (a system library) is erroneously
   deleted by a user (through the support tools portlet) there is
   no way to recreate the library.
   
   
    
   
   

PROBLEM CONCLUSION
 *  If the syndication library is missing at startup, it will be
   automatically recreated.
   
   
    
   
   

TEMPORARY FIX
