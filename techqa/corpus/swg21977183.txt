Title: IBM 【セキュリティ情報】IBM MQ Telemetry (遠隔測定) (MQXR) 安全でないパスワード  (CVE-2015-2012) - United States

Text:
 SECURITY BULLETIN

SUMMARY
 IBM MQ Telemetry (遠隔測定) (MQXR)サービスは、SSL/ TLS接続を使用するように設定することができます。 キーストアにアクセスするために使用するパスフレーズは、プロパティファイルにテキストとして書き込まれているため、読み込み可能です。 

VULNERABILITY DETAILS
脆弱性の詳細：
CVEID: CVE-2015-2012 [http://cve.mitre.org/cgi-bin/cvename.cgi?name=CVE-2015-2012]
DESCRIPTION: IBM WebSphere MQ stores the MQXR keystore password in clear text in a world readable file.
CVSS Base Score: 2.1
CVSS Temporal Score: See https://exchange.xforce.ibmcloud.com/vulnerabilities/103923 [https://exchange.xforce.ibmcloud.com/vulnerabilities/103923] for the current score
CVSS Environmental Score*: Undefined
CVSS Vector: (AV:L/AC:L/Au:N/C:P/I:N/A:N)
