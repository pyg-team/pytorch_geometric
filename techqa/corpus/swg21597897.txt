GLOSSARY: 

CCRC WAN Server: Provides CCRC and CTE with access to ClearCase repositories through web views. 

ClearTeam Explorer Extension: A version of CTE for installation into Rational desktop products. The extension is installed using IBM Installation Manager specifying the CCRC.EXT offering. 

ClearTeam Explorer for Eclipse IDE: A version of CTE for installation into Eclipse IDEs. It is installed using the Eclipse update manager from an update site on either the CCRC WAN Server or a ClearCase client. 

ClearCase zOS Extension: A set of ClearCase features that mainframe users can employ to perform ClearCase operations. 

Global Monitor: A MultiSite option based on IBM Tivoli Monitoring (ITM) that provides Web-based monitoring of a MultiSite deployment. 

MultiSite core server: MultiSite servers including the shipping server. 

MultiSite shipping server only: Includes only the MultiSite shipping server store-and-forward software. 

Native client: A ClearCase client running on the Windows, UNIX system, or Linux operating systems. These clients support dynamic and snapshot views, a command line interface, and GUIs. 

RFE: Request for Enhancement. A request for a product feature or support for a platform or operating system. 

VRMF: VRMF is IBM's product release identification format, which identifies the version, release, modification level, and fixpack level of a software offering.
