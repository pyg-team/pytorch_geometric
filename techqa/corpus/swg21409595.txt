ABSTRACT
 A new version of the Netcool/OMNIbus Gateway for Tivoli EIF will be available to download from September 11, 2014. 

CONTENT
 

The Netcool/OMNIbus V8.1 version of this gateway has been repackaged to support additional Operating Systems. The gateway now supports the following Operating Systems:

AIX, Linux, Solaris, Windows, and zLinux.

You can download the installation package from the IBM Passport Advantage website: 
www.ibm.com/software/howtobuy/passportadvantage/pao_customers.htm [http://www.ibm.com/software/howtobuy/passportadvantage/pao_customers.htm]

You can access the latest version of the guide written for this gateway on the IBM Knowledge Center:

www-01.ibm.com/support/knowledgecenter/#!/SSSHTQ/omnibus/gateways/tivoli_eif/wip/concept/tveifgw_intro.html [http://www-01.ibm.com/support/knowledgecenter/#!/SSSHTQ/omnibus/gateways/tivoli_eif/wip/concept/tveifgw_intro.html]

Netcool/OMNIbus V8.1

If you are running Netcool/OMNIbus V8.1, a single package contains versions of the gateway for all supported Operating Systems. To obtain this new package, use the Find by part number field to search for part number: CN1FNEN

Netcool/OMNIbus V7.3.0, 7.3.1 or 7.4.0

If you are running Netcool/OMNIbus V7.3.0, 7.3.1 or 7.4.0, a separate package is supplied for each Operating System. To obtain this new package, use the Find by part number field to search for the part number that corresponds to your Operating System as shown in the table below.

Part number Operating System CZ9JHEN AIX CZ9JKEN HP-UX PA-RISC CZ9JTEN HP-UX Itanium CZ9JIEN Linux CZ9JGEN Solaris CZ9JHEN Windows CZ9JJEN zLinux Note: Some versions of some web browsers can introduce random characters into text copied from web pages to the clipboard. In such cases, if you copy and paste part numbers from this release notice into Passport Advantage search fields, the search engine might not return the correct result. If you are encountering problems locating packages in Passport Advantage, remove any characters generated by the copy-paste operation before searching, or manually enter the part numbers into the search field.
