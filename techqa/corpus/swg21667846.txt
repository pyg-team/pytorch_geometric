Title: IBM Jazz for Service Management  Version1.1.2.1 / Tivoli Common Reporting Version 3.1.2.1 installation fails on Red Hat Enterprise Linux Version 7 as the Open Motif library is missing - United States

Text:
jazzsm1121relnotes TECHNOTE (TROUBLESHOOTING)

PROBLEM(ABSTRACT)
 Jazz for Service Management Version1.1.2.1 / Tivoli Common Reporting Version 3.1.2.1 installation fails on Red Hat Enterprise Linux Version 7 as the 'os.package.openmotif ' library is missing 

CAUSE
Jazz for Service Management Version 1.1.2.1 installation using Launchpad fails on Red Hat Enterprise Linux Version 7 complaining about 'os.package.openmotif' pre-requesite failures.

DIAGNOSING THE PROBLEM
In Red Hat Enterprise Linux version 7, Open Motif library has been replaced with Motif library and hence Open Motif does not get installed.

RESOLVING THE PROBLEM
To resolve this, install Motif Version 2.3 and install Jazz for Service Management Version 1.1.2.1 using Installation Manager or use Silent installation.
