Product Releases and News 

This section includes our prominent new services, XPU, firmware and product releases. This includes minor version updates as well as 0-day XPU releases. 

 1.  February Super Tuesday XPU 37.020 Release [https://exchange.xforce.ibmcloud.com/search/37.020] 
    PAM update 37.020 contains 28 new events, 30 new moderate event responses, and 30 new aggressive event responses 
 2.  SP4001 Appliance Version 2.9 Firmware Release [http://www-01.ibm.com/support/docview.wss?uid=swg21999141&acss=danl_4205_web] 
    This update contains Microsoft security updates for the underlying operating system on the SP4001 appliance. 
 3.  FLASH Security Bulletin: A vulnerability in Expat XML parser affects IBM Security Network Protection (CVE-2016-0718) [https://www-01.ibm.com/support/docview.wss?uid=swg21995440&acss=danl_4205_web] 
    A security vulnerability has been discovered in Expat XML parser, which is used by IBM Security Network Protection. 
 4.  FLASH Security Bulletin: A vulnerability in GnuPG libgcrypt affects IBM Security Network Protection (CVE-2016-6313) [https://www-01.ibm.com/support/docview.wss?uid=swg21994071&acss=danl_4205_web] 
    A security vulnerability has been discovered in GnuPG libgcrypt, which is used by IBM Security Network Protection. 
 5.  FLASH Security Bulletin: Multiple vulnerabilities in IBM Java Runtime affect IBM Security Network Protection [https://www-01.ibm.com/support/docview.wss?uid=swg21993670&acss=danl_4205_web] 
    There are multiple vulnerabilities in IBM Runtime Environment Java Version 7 that is used by IBM Security Network Protection. These issues were disclosed as part of the IBM Java SDK updates in October 2016. These vulnerabilities include CVE-2016-5569, CVE-2016-5556, CVE-2016-5573, CVE-2016-5597, CVE-2016-5554, and CVE-2016-5542.

PSIRT 

The IBM Product Security Incident Response Team (PSIRT) is a global team that manages the receipt, investigation and internal coordination of security vulnerability information related to IBM offerings. Each month, Security Flashes are published as vulnerabilities are announced by IBM. 

 1.  IBM Product Security Incident Response Blog [https://www.ibm.com/blogs/psirt/?lang=en_us&acss=danl_4205_web] 
    This page contains important information regarding security vulnerabilities that may affect IBM products and solutions. IBM PSIRT follows the NIST guidelines for determining the severity rating of the reported vulnerability. 



Getting Help 

There are several methods to contact support for your technical issues. Additionally, there are many support-related resources as well including the IBM My Notifications and Fix Central. The following are some of the more common and useful resources available as well as details on what is included with each resource. 

 1.  IBM Customer Forum for Infrastructure Security Products [http://www.ibm.com/support/docview.wss?uid=swg21902372&acss=danl_4205_web] 
    Our forum offers a place to ask questions and view answers to questions already posed. You may filter the forum using the tags SiteProtector, XGS, and GX. When asking a question please ensure you apply the appropriate tag to the question. 
 2.  My Notifications [http://www.ibm.com/software/support/einfo.html?acss=danl_4205_web] 
    My Notifications is an IBM Support email subscription notification service. You will be notified immediately by email when the PSIRT team announces new product vulnerabilities and solutions. You can be notified in a daily or weekly digest. 
 3.  Fix Central [http://www.ibm.com/support/fixcentral?acss=danl_4205_web] 
    Fix Central provides fixes and updates for your system's software, hardware, and operating system. Many interim patches and updates can be found here. 
 4.  IBM Support Portal [http://www.ibm.com/support/entry/portal/support?acss=danl_4205_web] 
    Provides a wealth of content for IBM products. If you elect not to subscribe to My Notifications, you can find new product information and technical resources here. If your inbox is busy enough already, this is your alternative to staying in the loop!
