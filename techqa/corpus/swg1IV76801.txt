ERROR DESCRIPTION
 *  L2 engineer: None
   Compid:  5724C04MS Tivoli OMEGAMON ITMS/ENGINE BASIC SERVICES
   
   Environment:
   
     This defect is specific to i5 platforms.
   
   Problem Description:
   
     Tivoli Monitoring i5 agents fail to start after Sept 6, 2015.
   The defect occurs at agent registration, during which a UUID
   create suffers a divide overflow. The Tivoli Monitoring agent
   terminates.
   Detailed Recreation Procedure:
   
    This can be recreated by starting a Tivoli Monitoring agent
   with the system clock at or after Sept 6, 2015 on any i5 system.
   
   Related Files and Output:
   
   The signature of this failure is termination of any and all i5
   Tivoli Monitoring agents started on or after Sept 15, 2015 .
   
   
    
   
   

LOCAL FIX

PROBLEM SUMMARY
 *  IBM Tivoli Monitoring for IBM i OS agents fail to start after
   Sept 6, 2015. The problem occurs at agent registration, during
   which a UUID create suffers a divide overflow.  The IBM i OS
   agent terminates.
   
   
    
   
   

PROBLEM CONCLUSION
 *  The algorithm used to generate the unique identifier was changed
   to prevent the integer divide overflow.
   
   Please see this DCF technote for further information:
   http://www-01.ibm.com/support/docview.wss?uid=swg21966093 [http://www-01.ibm.com/support/docview.wss?uid=swg21966093]
