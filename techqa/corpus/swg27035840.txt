Title: IBM To run Parascript FieldScript V8.0.1 with Datacap Taskmaster Capture V8.1.0 or Datacap V9.0.0, you must update the ICR_P.rrx file - United States

Text:
 RELEASE NOTES

ABSTRACT
 You must modify the header information in the ICR_P.rrx file to run Parascript V8.0.1 with Datacap Taskmaster Capture V8.1.0 or Datacap V9.0.0. 

CONTENT
After you install Parascript FieldScript V8.0.1, you must edit the ICR_P.rrx file to run the product with Datacap Taskmaster Capture V8.1.0 or Datacap V9.0.0: 

 

 1. Navigate to the <Datacap_Installation>\RRS directory and open the ICR_P.rrx file with a text editor. 
 2. In the header information of the RRX file, replace instances of rrunner.rrx with rrunner. 
 3. Replace instances of recog_shared.rrx with recog_shared. 
 4. Save and close the file.
