ABSTRACT
 A new version of the Netcool/OMNIbus Probe for Cisco CTM v9 CORBA will be available to download from September 11, 2014. 

CONTENT
 

The Netcool/OMNIbus V8.1 version of this probe has been repackaged to support additional Operating Systems. The probe now supports the following Operating Systems:

AIX, Linux, Solaris and zLinux.

You can download the installation package from the IBM Passport Advantage website: 
www.ibm.com/software/howtobuy/passportadvantage/pao_customers.htm [http://www.ibm.com/software/howtobuy/passportadvantage/pao_customers.htm]

You can access the latest version of the guide written for this probe on the IBM Knowledge Center:

www-01.ibm.com/support/knowledgecenter/#!/SSSHTQ/omnibus/probes/cisco_ctm_corba_v9/wip/concept/ctmcov9_intro.html [http://www-01.ibm.com/support/knowledgecenter/#!/SSSHTQ/omnibus/probes/cisco_ctm_corba_v9/wip/concept/ctmcov9_intro.html]

Netcool/OMNIbus V8.1

If you are running Netcool/OMNIbus V8.1, a single package contains versions of the probe for all supported Operating Systems. To obtain this new package, use the Find by part number field to search for part number: CN1IJEN

Netcool/OMNIbus V7.3.0, 7.3.1 or 7.4.0

If you are running Netcool/OMNIbus V7.3.0, 7.3.1 or 7.4.0, a separate package is supplied for each Operating System. To obtain this new package, use the Find by part number field to search for the part number that corresponds to your Operating System as shown in the table below.
