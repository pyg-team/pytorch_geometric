 DETAILED SYSTEM REQUIREMENTS

ABSTRACT
 The IBM® InfoSphere® Optim™ Archive Enterprise Edition, Optim Archive Work Group Edition, Optim Data Growth Solution, Optim Solution for Application Retirement, Optim Solution for Application Retirement - Open Data Manager, and Information Lifecycle Governance for Structured Data consist of a number of components. The system requirements for the components are compiled in this document.
