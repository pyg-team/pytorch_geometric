PROBLEM(ABSTRACT)
 There is an option to connect to 'Passport Advantage' from the IBM Packaging Utility. If you select this option, nothing happens. 

CAUSE
The IBM Rational Team Products don't offer this functionality.

RESOLVING THE PROBLEM
For ClearQuest, and other Team Products, you must go to Passport Advantage via a Web Browser, login and download the packages locall first. You can then import them using the Package Utility
