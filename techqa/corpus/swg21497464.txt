Title: IBM Instructions on how to map AFP fonts for the AFP Web Viewer are in the Windows Client Customization Guide - United States

Text:
 TECHNOTE (FAQ)

QUESTION
 Where is the AFP Workbench Technical Reference? How do I map AFP fonts or define which fonts the AFP Web Viewer should use? 

CAUSE
The ODWEK Implementation Guides for Multiplatforms, z/OS, and i incorrectly refer to the AFP Workbench Technical Reference as the source for more instructions on how to define fonts to the AFP Web Viewer.

ANSWER
In the following topics, change all instances of AFP Workbench Technical Reference to Windows Client Customization Guide: 

 * ODWEK Implementation Guide for Multiplatforms  * In the 8.4.1 information center, Mapping AFP fonts [http://publib.boulder.ibm.com/infocenter/cmod/v8r4m1/topic/com.ibm.ondemand.odwekmp.doc/ars1y37189.htm#afpfonts] 
    * In the 8.5 information center, Mapping AFP fonts [http://publib.boulder.ibm.com/infocenter/cmod/v8r5m0/topic/com.ibm.ondemand.odwekmp.doc/ars1y37189.htm#afpfonts]
   
   
 * ODWEK Implementation Guide for i  * In the 8.4.1 information center, Mapping AFP fonts [http://publib.boulder.ibm.com/infocenter/cmod/v8r4m1/topic/com.ibm.ondemand.odweki5os.doc/ars4y05371.htm#afpfonts] 
    * In the 8.5 information center, Mapping AFP fonts [http://publib.boulder.ibm.com/infocenter/cmod/v8r5m0/topic/com.ibm.ondemand.odweki5os.doc/ars4y05371.htm#afpfonts]
   
   
 * ODWEK Implementation Guide for z/OS  * In the 8.4.1 information center, Mapping AFP fonts [http://publib.boulder.ibm.com/infocenter/cmod/v8r4m1/topic/com.ibm.ondemand.odwekzos.doc/ars3y27183.htm#afpfonts] 
    * In the 8.5 information center, Mapping AFP fonts [http://publib.boulder.ibm.com/infocenter/cmod/v8r5m0/topic/com.ibm.ondemand.odwekzos.doc/ars3y27183.htm#afpfonts]
   
   

In the 8.4.1 information center, you can find the Windows Client Customization Guide here [http://publib.boulder.ibm.com/infocenter/cmod/v8r4m1/topic/com.ibm.ondemand.winclient.doc/ars5u271.htm]. In the 8.5 information center, you can find the Windows Client Customization Guide here [http://publib.boulder.ibm.com/infocenter/cmod/v8r5m0/topic/com.ibm.ondemand.winclient.doc/ars5u271.htm].
