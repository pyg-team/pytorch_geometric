Title: IBM API Connect V5.0.7.1 is available - United States

Text:
 NEWS

ABSTRACT
 IBM API Connect V5.0.7.1 is available, which addresses several APARs, and includes product performance enhancements. 

CONTENT


IBM API Connect V5.0.7.1 is now available, which provides important development and APAR fixes. 

We advise all users of IBM API Connect V5.0 to install this update to take advantage of the fixes.
