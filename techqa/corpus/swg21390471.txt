Title: IBM Publishing Designer availability in FileNet Workplace XT - United States

Text:
publishing designer; XT TECHNOTE (FAQ)

QUESTION
 Is the Publishing Designer Java applet that is available FileNet Workplace also available in Workplace XT? 

CAUSE
Workplace uses the Publishing Designer to create and modify publishing templates which specify the properties and security of documents published by Workplace users.

ANSWER
Publishing Designer is not currently supported on Workplace XT but is presently being reviewed for consideration in a future release of Workplace XT. 

As a workaround, Workplace can be installed in order to create publishing templates.
