Title: IBM Action required for IBM MQ on AWS Quick Start for security vulnerabilities in Ubuntu. - United States

Text:
 FLASH (ALERT)

ABSTRACT
 Ubuntu is shipped as a component of IBM MQ in AWS Quick Start. Information about a security vulnerability affecting Ubuntu has been published in a security bulletin. 

CONTENT
Please consult Ubuntu's security bulletin Information Leak via speculative execution side channel attacks (CVE-2017-5715, CVE-2017-5753, CVE-2017-5754 aka Spectre and Meltdown) [https://wiki.ubuntu.com/SecurityTeam/KnowledgeBase/SpectreAndMeltdown?_ga=2.104894214.406230060.1515581170-1977736704.1505735692] for vulnerability details and information about fixes for vulnerabilities:
CVE-2017-5715
CVE-2017-5753
CVE-2017-5754

Affected Products and Versions: 

Principal Product and Version(s) Affected Supporting Product and Versions IBM MQ in AWS Quick Start version 9.0.3 Ubuntu version 16.04 
Remediation: 
IBM strongly recommends Customers using the IBM MQ in AWS Quick Start contact Ubuntu to obtain and install fixes for Ubuntu 16.04, as applicable.
