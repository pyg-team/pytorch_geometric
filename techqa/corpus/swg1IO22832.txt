ERROR DESCRIPTION
 *  1. Desktop / FixUp task - Right-Click and select "DELETE
   document" in batch view and notice it does not delete
   document/image - Incongruous feature
   In DotEdit, (or Verify mode in Desktop) you mark the page for
   deletion and the page's status is changed to reflect that.
   The right click menu does not belong in the batchview when in
   scan/fixup panels are up in Desktop..
   2. Datacap Desktop does not show page sequence.- Unavailable
   feature
   In 8.1 you can specify PageType or Sequence for DotScan, but not
   in DotEdit.
   3.  Not able to use CTRL to highlight multiple documents and
   pages  in DC Desktop 9.0.- Unavailable feature
   In Dotscan you can use CTRL button to select multiple  documents
   and pages
