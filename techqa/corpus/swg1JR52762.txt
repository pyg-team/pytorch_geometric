APAR STATUS
 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  You do not receive an error, but the Java certification does
   not pass. According to a Java certification requirement, for
   embedded JDK the license file should not exist, yet it does.
   
   
    
   
   

LOCAL FIX
 *  The 32bit JRE package under BPM_ROOT/util/migration/jre contain
   license.txt file incorrectly. This ifix will delete the
   license.txt file from this 32bit JRE package.
