   Error Description:
   After installing  Subversive SVN Team Provider
   1.1.0.I20130619-1700 plugin into and RTC client user will
   receive the folowing error when selecting 'jazz work items' from
   properties of an SVN project in RTC the client.
   'The currently displayed page contains invalid values'
   
   
   Reproduction
   Install RTC 4.0.3 client  downloaded  from Jazz.net (this will
   include the 3.6 eclipse client)
   Install the Subversive SVN Team Provider 1.1.0.I20130619-1700
   plugin
   From
   http://download.eclipse.org/technology/subversive/1.1/update-sit [http://download.eclipse.org/technology/subversive/1.1/update-sit]
   e/
   Finish setting up and installing the SVN Kit (1.3.5 or 1.7.9)
   Load a project from the SVN repository.
   Right click choose properties.
   Note the error pops up.
