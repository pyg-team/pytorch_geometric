ERROR DESCRIPTION
 *  Some FileNetP8 actions such as SetDocTitle does not parse smart
   parameters values that do not follow smart parameter navigation
   syntax
   FNP8_SetDocTitle is set to an empty value when using the
   following smart parameter syntax.
   
   action FNP8_SetDocTitle (str="@Barcode")
   
   It looks like '@Barcode' is returning an empty value. The set
   doc title property cannot be empty. Which results in the error:
