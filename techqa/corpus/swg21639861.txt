Title: IBM Installing ITCAM Agent for WebSphere Applications Reports V7.2 on Tivoli Common Report 3.1 - United States

Text:
 TECHNOTE (FAQ)

QUESTION
 How do I install ITCAM Agent for WebSphere Applications Reports V7.2 on Tivoli Common Report 3.1? 

ANSWER
You can install ITCAM Agent for WebSphere Applications Reports 7.2, provided with ITCAM for Applications V7.2.0.0.1, on Tivoli Common Reporting V2.1.1 or V3.1 by performing a manual installation or by using the Reports installer. 
Prerequisites and pre-installation tasks

Before installing ITCAM Agent for WebSphere Applications Reports, verify that you have met the following prerequisites: 

 * Tivoli Monitoring 6.2.2 fix pack 3 or higher 
 * Tivoli Common Reporting 2.1.1 or version 3.1 
 * ITCAM Agent for WebSphere Applications 7.1 Fix Pack 2, interim fix 3, or higher 
 * Tivoli Data Warehouse (including Summarization and Pruning agent [SnP]) 
 * You must have Java Runtime Environment (JRE) 1.5 or higher installed on the systems where you are plan to run the reports installer


The databases that are supported are the same as the databases supported by Tivoli Data Warehouse.
