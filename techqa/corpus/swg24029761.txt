ABSTRACT
 This document describes how to download IBM WebSphere Message Broker Connectivity Pack for Healthcare V7.0 eAssembly images from the Passport Advantage website. 

DOWNLOAD DESCRIPTION


Terminology

IBM WebSphere Message Broker Connectivity Pack for Healthcare V7.0 is supplied in an eAssembly. The eAssembly contains the Quick Start CD and the product code. 
Related product documentation 


For information about tasks you must complete before you install the product, and about the installation process itself, see the IBM WebSphere Message Broker Connectivity Pack for Healthcare V7.0 Installation Guide that is supplied with the product, or see the Information Center [http://publib.boulder.ibm.com/infocenter/wmbhelp/v7r0m0/index.jsp].

The most up-to-date information can be found in the IBM WebSphere Message Broker Connectivity Pack for Healthcare V7.0 readme file at http://www.ibm.com/support/docview.wss?&uid=swg27006917 [http://www.ibm.com/support/docview.wss?rs=171&uid=swg27006917]. 



Downloading the relevant eAssembly images from the Passport Advantage website

 1. Find the IBM WebSphere Message Broker Connectivity Pack for Healthcare V7.0 images for your platform in the following eAssemblies table.
 2. Sign in to download all of the images that you need.
 3. Unpack each of the parts onto your system by using the instructions in the Installation Guide.
