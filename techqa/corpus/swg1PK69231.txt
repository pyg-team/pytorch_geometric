Title: IBM PK69231: ERROR RENDERING RICH TEXT COMPONENT WITH MIXED CASE HTML TAGS - United States

Text:
 
APAR STATUS
 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  Using servlet or portlet rendering on a content with a rich text
   component with the contents of the component being set by using
   the API to set html tags with different case for the starting
   and end tag will result in incorrect html being rendered
   
   In the SXML parser, once an ending tag was found, looking back
   for the starting tag was case insensitive.
   
   
    
   
   

LOCAL FIX
 *  Make the lookup to match the ending and starting html tag be
   case insensitive.
   
   
    
   
   

PROBLEM SUMMARY
 *  Error rendering rich text field with mixed case tags
   
   Using servlet or portlet rendering on a content with a rich text
   component with the contents of the component being set by using
   the API to set html tags with different case for the starting
   and end tag will result in incorrect html being rendered
   
   In the SXML parser, once an ending tag was found, looking back
   for the starting tag was case insensitive.
   
   
    
   
   

PROBLEM CONCLUSION
 *  Modified the lookup to match the ending and starting html tag be
   case insensitive.
