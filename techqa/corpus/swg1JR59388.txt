ERROR DESCRIPTION
 *  According to the Microsoft SQLServer database statement,
   sqljdbc42.jar supports Java 8 and later versions. Therefore,
   support for IBM Business Process Manager (BPM) is needed so you
   can use sqljdbc42.jar with Java 8 to connect to the Microsoft
   SQL Server database.
