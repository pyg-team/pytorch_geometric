ERROR DESCRIPTION
 *  Abstract:
   
   Attempts to install the Maven Integration for WTP on Rational
   Application Developer v8.0.4 results in error: Cannot complete
   the install because one or more required items could not be
   found.
   
   Problem:
   
   There are two scenarios, with different error messages:
   
   Scenario A:
   
   1. You install Rational Application Developer v8.0.4
   2. You attempt to install m2e - Maven Integration for Eclipse
   and Maven Integration for WTP
   3. You see the error:
   Cannot complete the install because one or more required items
   could not be found.
   
   Scenario B:
   
   1. You install Rational Application Developer v8.0.3
   2. You install m2e - Maven Integration for Eclipse and Maven
   Integration for WTP
   3. You attempt to upgrade to Rational Application Developer
   v8.0.4 using IBM Installation Manager
   4. You see the error:
   ERROR CRIMA44820FA3E Eclipse-P2-Reconciler: Eclipse:
   Java was started but returned exit code=13
   
   Local fix:
   
   Use Rational Application Developer v8.0.3 if the Maven
   Integration is required.
