Title: IBM Detailed System Requirements for IBM InfoSphere Optim Test Data Management Solution for Oracle E-Business Suite with Data Masking Option v11.3 - United States

Text:
 PRODUCT DOCUMENTATION

ABSTRACT
 This document summarizes the hardware, platforms, and software supported by the IBM® InfoSphere® Optim™ Test Data Management Solution for Oracle E-Business Suite with Data Masking Option v11.3. 

CONTENT


Version 11.3 of the InfoSphere Optim Test Data Management Solution for Oracle E-Business Suite with Data Masking Option is tested and distributed with the IBM InfoSphere Optim Test Data Management Solution v11.3. Detailed hardware and software requirements for the InfoSphere Optim Test Data Management Solution are at: http://www.ibm.com/support/docview.wss?uid=swg27041700 [http://www.ibm.com/support/docview.wss?uid=swg27041700] 

SUPPORTED ORACLE E-BUSINESS SUITE VERSIONS
The InfoSpere Optim Test Data Management Solution for Oracle E-Business Suite with Data Masking Option supports 

 * Test data management for Oracle E-Business Suite, versions 11.5.10 and 12
 * Data masking for the Core HRMS module of Oracle E-Business Suite, versions 11.5.10 and 12
