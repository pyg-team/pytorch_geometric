ERROR DESCRIPTION
 *  CVE-2013-7423 - The getaddrinfo function of the GNU C Library
   (glibc) might allow DNS queries resulting in information
   disclosure or data loss due to the application encountering
   corrupted data.
   
   CVE-2015-1781 - The GNU C Library (glibc) gethostbyname
   functions are vulnerable to a buffer overflow.
   
   
    
   
   

LOCAL FIX

PROBLEM SUMMARY
 *  DataPower may be effected by GNU C Library (glibc) function
   vulnerabilities as described in CVE-2013-7423 and CVE-2015-1781.
   
   
    
   
   

PROBLEM CONCLUSION
 *  Fix will be available in 6.0.0.16, 6.0.1.12, 7.0.0.9, 7.1.0.6
   and 7.2.0.1
