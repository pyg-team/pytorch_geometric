ERROR DESCRIPTION
 *  The following results were seen when copying a table from Excel
   to Notes 9.0.1 FP3
   
   Testing with EditorPastePreferTextToImage=0
   
   All copied correctly as bitmap.
   
   Tested with EditorPastePreferTextToImage=1:
   
   1) pasting from MS Office Excel as simple paste
   No borders
   
   2) pasting from MS Office Excel as HTML
   No borders
   
   3) pasting from MS Office Excel as Rich Text
   Borders pasted correctly
   
   Tested with EditorPastePreferTextToImage=2:
   
   1) pasting from MS Office Excel as simple paste
   No borders
   
   2) pasting from MS Office Excel as HTML
   No borders
   
   3) pasting from MS Office Excel as Rich Text
   Borders pasted correctly
   
   
   Excel tables should be copied correctly in all cases as was
   previously the case in Notes 9.0
