Title: IBM Netcool/OMNIbus Integrations Release Notice - Probe for CA Spectrum (CORBA) (nco-p-ca-spectrum-corba-1_0) - United States

Text:
release notice; downloads; nco-p-ca-spectrum-corba; Probe for CA Spectrum (CORBA) NEWS

ABSTRACT
 The Netcool/OMNIbus Probe for CA Spectrum (CORBA) will be available to download from December 10, 2015. 

CONTENT
 

This is a new probe written to receive events from CA Spectrum version 9.4. The probe has also been certified against CA Spectrum V10.0.

If you want to received events from CA Spectrum versions 9.0 to 9.3, you must use the Probe for CA Spectrum V9 (CORBA) (nco-p-spectrum-corba-v9-6_0). For details see 
www-01.ibm.com/support/docview.wss?uid=swg21641820 [http://www-01.ibm.com/support/docview.wss?uid=swg21641820]

You can download the installation package from the IBM Passport Advantage website: 
www.ibm.com/software/howtobuy/passportadvantage/pao_customers.htm [http://www.ibm.com/software/howtobuy/passportadvantage/pao_customers.htm]

You can access the latest version of the guide written for this probe on the IBM Knowledge Center:

www-01.ibm.com/support/knowledgecenter/#!/SSSHTQ/omnibus/probes/ca_spectrum_corba/wip/concept/caspec_intro.html [http://www-01.ibm.com/support/knowledgecenter/#!/SSSHTQ/omnibus/probes/ca_spectrum_corba/wip/concept/caspec_intro.html]

Netcool/OMNIbus V8.1

This probe is only supported on Netcool/OMNIbus V8.1. It is provided as a single package which contains versions of the probe for all supported Operating Systems. To obtain this new package, use the Find by part number field to search for part number: CN8B9EN

Netcool/OMNIbus V7.3.0, 7.3.1, or 7.4.0

If you are running Netcool/OMNIbus V7.3.0, 7.3.1, or 7.4.0, you must use the Probe for CA Spectrum V9 (CORBA) (nco-p-spectrum-corba-v9-6_0). For details see 
www-01.ibm.com/support/docview.wss?uid=swg21641820 [http://www-01.ibm.com/support/docview.wss?uid=swg21641820]
