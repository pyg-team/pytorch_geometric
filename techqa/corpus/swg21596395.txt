CAUSE
Your Linux operation system uses XULRunner V10 or higher, which is not compatible with all releases of Eclipse V3.6.x. IBM Integration Designer is based on Eclipse V3.6.x.


RESOLVING THE PROBLEM
Download and install 64-bit XULRunner V1.9.2 at: 

http://ftp.mozilla.org/pub/mozilla.org/xulrunner/nightly/2012/03/2012-03-02-03-32-11-mozilla-1.9.2/xulrunner-1.9.2.28pre.en-US.linux-x86_64.tar.bz2 [http://ftp.mozilla.org/pub/mozilla.org/xulrunner/nightly/2012/03/2012-03-02-03-32-11-mozilla-1.9.2/xulrunner-1.9.2.28pre.en-US.linux-x86_64.tar.bz2]. 

If IBM Integration Designer 32-bit is installed, download and install 32-bit XULRunner V1.9.2 at: 

http://ftp.mozilla.org/pub/mozilla.org/xulrunner/releases/3.6.26/runtimes/xulrunner-3.6.26.en-US.linux-i686.tar.bz2 [http://ftp.mozilla.org/pub/mozilla.org/xulrunner/releases/3.6.26/runtimes/xulrunner-3.6.26.en-US.linux-i686.tar.bz2] 

For installation instructions on XULRunner, refer to the Mozilla Developer Network [https://developer.mozilla.org/en-US/docs/Mozilla/Projects/XULRunner].
