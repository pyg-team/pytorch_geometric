APAR STATUS
 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  The PureApplication Health Check contains a Virtual Applications
   and Shared Services section that contains the number of virtual
   applications and then lists each one and its status in the
   Virtual Applications Details section.
   
   In one example the total number of virtual applications value
   was 27, but the number of virtual applications listed in the
   details section was 1.
   
   
    
   
   

LOCAL FIX

PROBLEM SUMMARY
 *  The virtual applications detail section is only displaying
   virtual applications and not the new virtual system pattern
   instances.
   
   
    
   
   

PROBLEM CONCLUSION
 *  A code fix is being provided in PureApplication System
   version 2.2.0.0.
