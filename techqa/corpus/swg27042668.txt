Title: IBM Content Navigator for Microsoft Office known problem: If a user with IBM Content Navigator for Microsoft Office V2.0.2.2 selects an entry template created in IBM Content Navigator 2.0.3, it cannot be opened - United States

Text:
 PRODUCT DOCUMENTATION

ABSTRACT
 When IBM Content Navigator for Microsoft Office V2.0.2.2 is trying to select an entry template that was created in IBM Content Navigator 2.0.3, the hourglass icon is shown and the entry template is not selected. 

CONTENT
IBM Content Navigator for Microsoft Office V2.0.2 fix pack users who are pointing to IBM Content Navigator 2.0.3, see IBM Content Navigator 2.0.3 entry templates but cannot use them.

If users want to add items by using an entry template, they must select the entry template that was created from IBM FileNet Workplace XT.
