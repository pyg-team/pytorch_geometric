APAR STATUS
 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  When comparing members in Inspector, we see the following error
   if the logs when the name value has an '&' symbol.
   
   MemberCompare E
   com.initiatesystems.web.inspector.model.MemberCompareModel
   parseXmlString The reference to entity "M" must end with the ';'
   delimiter.
   
   
    
   
   

LOCAL FIX
 *  use workbench or the memcomp utility
