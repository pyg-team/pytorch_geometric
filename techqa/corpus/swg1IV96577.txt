APAR STATUS
 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  The default script behavior should be to generate a name of a
   computer based on the output of the 'hostname' command.
   After importing such disconnected scan package this name of the
   host is shown on web UI.
   During upgrade of already existing disconnected scanner
   installations, the existing hostname would be preserved
   (endpoint ID and computer.yml will not be regenerated).
