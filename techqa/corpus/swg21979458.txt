Title: IBM 【セキュリティ情報】IBM WebSphere MQ同梱のICU4Cにバッファオーバーフローの脆弱性 (CVE-2011-4599) - United States

Text:
 SECURITY BULLETIN

SUMMARY
 Managed File Transfer process controller機能のためにWebSphere MQが同梱しているICU4Cでは、_canonicalize 関数にスタックベースのバッファオーバーフローの脆弱性が存在します。 

VULNERABILITY DETAILS

脆弱性の詳細：

CVEID: CVE-2011-4599 [http://cve.mitre.org/cgi-bin/cvename.cgi?name=CVE-2011-4599]
DESCRIPTION: International Components for Unicode is vulnerable to a stack-based buffer overflow, caused by improper bounds checking by the _canonicalize( ) function. By supplying a negative len value, a remote attacker could overflow a buffer and execute arbitrary code on the system or cause the application to crash.
CVSS Base Score: 7.5
CVSS Temporal Score: See https://exchange.xforce.ibmcloud.com/vulnerabilities/71726 [https://exchange.xforce.ibmcloud.com/vulnerabilities/71726] for the current score
CVSS Environmental Score*: Undefined
CVSS Vector: (AV:N/AC:L/Au:N/C:P/I:P/A:P) 


AFFECTED PRODUCTS AND VERSIONS
影響のある製品・バージョン： 

IBM WebSphere MQ の以下のバージョンに影響があります。 

 * V8.0.0.3 およびそれ以前の保守レベル 
 * V7.5.0.5 およびそれ以前の保守レベル
