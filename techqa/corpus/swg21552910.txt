PROBLEM(ABSTRACT)
 How to enable verbose Garbage Collection and Heap Dumps for the agent servers and integration servers? 

SYMPTOM


How can a user enable verbose Garbage Collection (GC) and Heap Dumps for the agent servers and integration servers?


RESOLVING THE PROBLEM


In performance.properties file, set the following parameters to true to enable verbose GC and Heap Dumps respectively:

EnableVerboseGc=true
EnableHeapDump=true

Note: These properties cannot be set in customer_overrides.properties.
