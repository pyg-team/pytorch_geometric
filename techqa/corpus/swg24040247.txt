 DOWNLOADABLE FILES

ABSTRACT
 The IBM Datacap deployment automation scripts automate the installation of IBM Datacap components in a client/server environment. 

DOWNLOAD DESCRIPTION
The deployment scripts are applicable to IBM Datacap Versions 8.1, 9.0, and 9.0.1.

INSTALLATION INSTRUCTIONS
To deploy Datacap, download the following items: 

Deployment instructions The attached PDF document: DCDeploymentAutomation.pdf Deployment scripts The attached zip file: DatacapDeploymentScripts.zip Database client software You need the following files based on your database type:
 * DB2: v10.5fp5_ntx64_client.exe. See http://www-01.ibm.com/support/docview.wss?uid=swg21385217 [http://www-01.ibm.com/support/docview.wss?uid=swg21385217]. 
 * Microsoft SQL: msodbcsql.msi and MsSqlCmdLnUtils.msi. See https://msdn.microsoft.com/en-us/data/ff658533.aspx [https://msdn.microsoft.com/en-us/data/ff658533.aspx]. 
 * Oracle: win32_11gR2_client.zip. See http://www.oracle.com/technetwork/database/enterprise-edition/downloads/112010-win32soft-098987.html [http://www.oracle.com/technetwork/database/enterprise-edition/downloads/112010-win32soft-098987.html].
