ABSTRACT
 IBM Rational ClearQuest 8.0.1.1 has been made generally available and contains fixes to version 8.0.1 including all predecessor fix packs. 

DOWNLOAD DESCRIPTION

Please read ClearQuest Release Notes [http://www.ibm.com/support/docview.wss?&uid=swg27039507] for information about system requirements, new features, and other important release information.

Important Upgrade Information:
If you plan to upgrade ClearQuest 7.1.x directly to this release and you are running ClearQuest Web, please read technote 1586861 [http://www.ibm.com/support/docview.wss?&uid=swg21586861] prior to upgrading.

Please refer to 1572830 [http://www.ibm.com/support/docview.wss?&uid=swg21572830] prior to upgrading servers deploying web components. Specifically refer to the section labeled "Avoiding Problems".

If you plan to install this ClearQuest release or to upgrade to this release on Solaris 11.1, please read technote 1649246 [http://www.ibm.com/support/docview.wss?&uid=swg21649246] prior to that to understand why those operations may fail.


If you wish to apply the latest updates to your system, check the Fixes by Version for Rational ClearQuest [http://www.ibm.com/support/docview.wss?&uid=swg27036262] technote for more details.

Technote 1450027 [http://www.ibm.com/support/docview.wss?&uid=swg21450027] describes behavioral differences in ClearQuest releases (including all Fix Packs) between 7.1.1 and the current release.




TestFixes included and excluded in this Fixpack Technote 1574727 [http://www.ibm.com/support/docview.wss?&uid=swg21574727] lists TestFixes that are included in this release, as well as those that have not been incorporated into this release.
