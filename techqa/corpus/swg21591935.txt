Title: IBM Error occurring when running delRootIUnRelationships script - United States

Text:
DEL2CONF; DE; Deployment Engine; delRootIUnRelationships TECHNOTE (TROUBLESHOOTING)

PROBLEM(ABSTRACT)
 While running the delRootIUnRelationships script to remove a product from the Deployment Engine database, this error occurs: "The filename, directory name, or volume label syntax is incorrect." 

SYMPTOM
The following error message is being displayed: 

"The filename, directory name, or volume label syntax is incorrect."


CAUSE
Missing configuration


RESOLVING THE PROBLEM
 Before using the "delRootIUnRelationships" in the delRootIUnRelationships.cmd file 

please replace <DE_COMMON_LOCATION> with the correct DE location:

[/support/docview.wss?uid=swg21591935&amp;aid=1] 
For root install :

/usr/ibm/common/acsi

For Non-root install :

/home/<username>/.acsi_<username>

[/support/docview.wss?uid=swg21591935&amp;aid=2]
C:\Program Files\IBM\Common\acsi (if using 64-bit, use Program Files: x86)
