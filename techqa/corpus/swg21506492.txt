Title: IBM Invalid Date Time value entered when using timestamp to filter data in IBM InfoSphere Information Analyzer - United States

Text:
datetime TECHNOTE (TROUBLESHOOTING)

PROBLEM(ABSTRACT)
 Error when adding filter to timestamp datatype on virtual table:

Invalid DateTime value entered, cannot be converted to DateTime format 

RESOLVING THE PROBLEM
For invalid date time issue, please try using MM/DD/YYYY HH:MM:SS format.
For example: 10/19/2000 19:00:00
