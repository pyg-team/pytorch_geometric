Collecting Managing Server (MS) Data 
The following script collects system info, ITCAM doc, and WebSphere documentation. 
Note: The ID under which the script is run must have read access to the ITCAM and WebSphere files and directories. 

Windows (must be run in a Korn Shell) 

Location <MS_HOME>/bin/MS_Gather_Data.sh Usage cd <MS_HOME>/bin
./MS_Gather_Data.sh <pmr> <MsWasHome> <AppSvrName> Example ./MS_Gather_Data.sh 70023.900.000 
/dev/fs/C/PROGRA~1/WebSphere/AppServer/profiles/AppSvr01 server1 Output <MS_HOME>/70023.900.000.*.tar.gz 
Unix and Linux 

Location <MS_HOME>/bin/MS_Gather_Data.sh Usage cd <MS_HOME>/bin
./MS_Gather_Data.sh <pmr> <MsWasHome> <AppSvrName> Example ./MS_Gather_Data.sh 70023.900.000
/opt/WebSphere/AppServer/profiles/AppSvr01 server1 Output <MS_HOME>/70023.900.000.*.tar.gz
