Title: IBM Security Bulletin: Security vulnerabilities have been identified in multiple components shipped with IBM Intelligent Operations Center (June 2015) - United States

Text:
 SECURITY BULLETIN

SUMMARY
 Multiple components are shipped with IBM Intelligent Operations Center. Information about security vulnerabilities that affect some components has been published in security bulletins. 

VULNERABILITY DETAILS
Consult the following security bulletins for vulnerability details: 

 * Security Vulnerability in IBM WebSphere Application Server (CVE-2015-1920) [http://www.ibm.com/support/docview.wss?uid=swg21883573] 
 * Multiple vulnerabilities in IBM Java SDK affect WebSphere Application Server April 2015 CPU [http://www.ibm.com/support/docview.wss?uid=swg21902260]

AFFECTED PRODUCTS AND VERSIONS
 

Affected Product and Version(s) Product and Version shipped as a component IBM Intelligent Operations Center version 1.6.0.3  * IBM WebSphere Application Server interim fix (PI38302) to address security vulnerability CVE-2015-1920 [http://www.ibm.com/support/docview.wss?uid=swg21883573] 
 * IBM WebSphere Application Server (PI39866) - upgrade IBM WebSphere Application Server version 8 Java to IBM SDK 6R1 SR8 FP 4 
 * IBM WebSphere Application Server (PI39867) - upgrade IBM WebSphere Application Server version 7 Java to IBM SDK 6R1 SR16 FP 4

WORKAROUNDS AND MITIGATIONS
Download and install IBM Intelligent Operations Center version 1.6.0.3 interim fix PO04846 [http://www.ibm.com/support/docview.wss?uid=swg24040061]. 
Security fixes are not cumulative. You must install all earlier security fixes in addition to the current fix.
