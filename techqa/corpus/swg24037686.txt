ABSTRACT
 This document describes how to download IBM Integration Bus Manufacturing Pack V1.0 eImages from the Passport Advantage website. 

DOWNLOAD DESCRIPTION
Terminology 

IBM Integration Bus Manufacturing Pack V1.0 is supplied in an eImage. The eImage contains the Quick Start Guide and the product code. A separate eImage is also available that contains only the Quick Start Guide.

Related product documentation

For information about tasks you must complete before you install the product, and about the installation process itself, see the IBM Integration Bus Manufacturing Pack V1.0 product documentation, in the IBM Integration Bus V9.0 product documentation [http://www.ibm.com/support/knowledgecenter/SSMKHH_9.0.0/mapfiles/help_home_msgbroker.html].

The most up-to-date information can be found in the IBM Integration Bus Manufacturing Pack V1.0 release notes at http://www.ibm.com/support/docview.wss?&uid=swg27006913 [http://www.ibm.com/support/docview.wss?rs=171&uid=swg27006913].

Downloading the relevant eImages from the Passport Advantage website


 1. Find the required IBM Integration Bus Manufacturing Pack V1.0 images in the following eImages table. 
 2. Go to the "Download package" section of this document, and follow the link to the Passport Advantage website. 
 3. Sign in and download the images that you need. 
 4. Unpack each of the parts onto your computer by following the installation instructions in the product documentation.
