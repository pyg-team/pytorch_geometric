ERROR DESCRIPTION
 *  Environment:
   Microsoft Exchange Server Agent 6.2.3, 64 bit
   Microsoft Exchange Server 2007
   
   
   Problem Description:
   When Microsoft Exchange Server Agent 6.2.3, 64 bit is configured
   to collect mailbox details on Microsoft Exchange Server 2007.
   Agent crash is observed sometimes while fetching mailbox details
   attribute group data from portal.
   
   
   Detailed Recreation Procedure:
   Install and configure Microsoft Exchange Server agent 6.2.3, 64
   bit on Microsoft Exchange Server 2007 with Mailbox Details
   Collection enabled. Sometimes there are some garbage lines
   appended at the end of "MailBox_Details.log" file. If such
   garbage lines are there in "MailBox_Details.log" file then agent
   crashes while fetching mailbox details attribute group data from
   portal.
   
   
   Related Files and Output:
   1. Log file created by kexcma.exe named *kexcma*.log
   2. Mailbox Details log file "MailBox_Details.log"

 *  Exchange Server agent crashes while fetching the mailbox

   fetching the mailbox details attribute group data.
