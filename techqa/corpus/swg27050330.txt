Title: IBM PureApplication Software Release Note: Windows ITM OS agents are not updated to the ITM 6.3 release - United States

Text:
sw2240rlsnotes RELEASE NOTES

ABSTRACT
 The Windows ITM OS agents are not updated to the ITM 6.3 release when the external ITM 6.3 shared service is deployed. 

CONTENT
Problem: The Windows ITM OS agents are not updated to the ITM 6.3 release when the external ITM 6.3 shared service is deployed. The agent will remain on the ITM 6.2 release when the instance is restarted.




Resolution: The ITM 6.2 OS agent will continue to function while connected to the ITM 6.3 management server. No additional resolution steps are necessary.
