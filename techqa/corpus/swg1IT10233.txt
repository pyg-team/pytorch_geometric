APAR STATUS
 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  The elastic load balancing (ELB) binaries in the ELB plug-in
   included in the Foundation Pattern Type need to be updated to
   versions of the binaries that include the POODLE and LogJam
   security fixes.
   
   
    
   
   

LOCAL FIX

PROBLEM SUMMARY
 *  The elastic load blancing (ELB) plugin binaries in the
   Foundation Pattern Type need to be updated to versions that have
   security fixes for the following two bulletins:
   TLS padding vulnerability affects IBM PureApplication System
   (CVE-2014-8730)
   http://www-01.ibm.com/support/docview.wss?uid=swg21960375 [http://www-01.ibm.com/support/docview.wss?uid=swg21960375]
   
   Vulnerability in Diffie-Hellman ciphers affects IBM
   PureApplication System. (CVE-2015-4000)
   http://www-01.ibm.com/support/docview.wss?uid=swg21964801 [http://www-01.ibm.com/support/docview.wss?uid=swg21964801]
   
   
    
   
   

PROBLEM CONCLUSION
 *  The binaries have been updated in the Foundation Pattern Type
   version 2.1.3.2 that is being released with PureApplication
   System version 2.1.0.2.
   
   
    
   
   

TEMPORARY FIX
