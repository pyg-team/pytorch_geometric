Title: IBM Security Bulletin: A security vulnerability has been identified in IBM WebSphere Application Server shipped with WebSphere Commerce (CVE-2017-1151) - United States

Text:
 SECURITY BULLETIN

SUMMARY
 IBM WebSphere Application Server is shipped as a component of WebSphere Commerce. Information about a security vulnerability affecting IBM WebSphere Application Server was published in a security bulletin. 

VULNERABILITY DETAILS
Consult the security bulletin Privilege Escalation Vulnerability in WebSphere Application Server (CVE-2017-1151) [http://www-01.ibm.com/support/docview.wss?uid=swg21999293] for vulnerability details and information about fixes.

AFFECTED PRODUCTS AND VERSIONS
 

Principal Product and Version(s) Affected Supporting Product and Version WebSphere Commerce V8.0 IBM WebSphere Application Server  * Version 8.5.5
