CAUSE
PrimaryViews.xml file has encoding that's set incorrectly to ANSI. By default, this file should be set to UCS-2 Big Endian after XT installation.


RESOLVING THE PROBLEM
 

 1. Open the <XT install folder>FileNet/Config/WebClient/PrimaryViews.xml into Notepad ++. 
 2. Select Encoding menu. 
 3. Select Convert to UCS-2 Big Endian then save the change. 
 4. Stop and restart the Workplace XT application server
