Title: IBM Security Bulletin: Security vulnerabilities (CVE-2013-6747, CVE-2014-0963) have been identified in IBM DB2 shipped with IBM Security Privileged Identity Manager components - United States

Text:
PSIRT SECURITY BULLETIN

SUMMARY
 IBM Security Access Manager and IBM Security Identity Manager are shipped as components of IBM Security Privileged Identity Manager (ISPIM) and are affected by a vulnerability in IBM® DB2®. Information about the security vulnerabilities affecting IBM Security Access Manager and IBM Security Identity Manager have been published in the below security bulletins. 

VULNERABILITY DETAILS
Please consult the following security bulletins 
Security vulnerabilities have been identified in IBM® DB2® shipped with IBM Security Access Manager for Enterprise Single Sign-On (CVE-2013-6747, CVE-2014-0963) [http://www-01.ibm.com/support/docview.wss?uid=swg21674315] 
and 
Security vulnerabilities (CVE-2013-6747, CVE-2014-0963) have been identified in IBM® DB2® shipped with IBM Tivoli Identity Manager/IBM Security Identity Manager [http://www-01.ibm.com/support/docview.wss?uid=swg21674351] 
for vulnerability details.
