Does upgrading from ISAM for Mobile 8.0.1.X mean that my Oracle 12c R1 database is not supported any longer? 

CAUSE
The IBM Clarity Tool for IBM Security Access Manager for Advanced Access Control (ISAM AAC) specifies that only Oracle Database 11g Enterprise Edition R2 (11.2.0.1.X) is supported in the matrix.
ISAM for Mobile 8.0.1.X also lists Oracle Database 12c Release 1 (12.1.0.1.X) is supported for use as the External Runtime and Configuration Databases.


ANSWER
Development has confirmed that Oracle Database 12c Release 1 (12.1.0.1.X) is also supported at 9.0.0.X and 9.0.1.X as an available database for use as the External Runtime or Configuration Database.
