ERROR DESCRIPTION
 *  Environment: ITM 6.2.FP1, MS SQL Server Agent 6.2.0, No specific
   SQL server version and Windows OS required.
   Approver:  MJK
   Problem Description:  MS SQL Server Agent shows an incorrect job
   status for the SQL Server Job which is created but hasn't ever
   ran.  Therefore, the SQL Server Agent fires a situation for
   the job as failed, even though the job was never executed.
   Detailed Recreation Procedure:
   1. Go to TEP
   2. Right click on Navigator Item 'Job'
   3. click on Job detail workspace
   4. Check 'Job Detail' attribute in job detail view.
   
   
    
   
   

LOCAL FIX
 *  Fix will be included in SQL Server Agent 6.2.0 IF0004.
