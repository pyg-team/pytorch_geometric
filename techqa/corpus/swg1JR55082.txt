APAR STATUS
 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  When you try to view a report in the Report Viewer widget or
   preview a report in the Report Designer widget from Internet
   Explorer, you see a blank area where the report should be
   displayed.
   
   PRODUCTS AFFECTED
   IBM Business Monitor
   
   
    
   
   

LOCAL FIX

PROBLEM SUMMARY
 *  No additional information is available.
   
   
    
   
   

PROBLEM CONCLUSION
 *  A fix is available for IBM Business Monitor V8.5.6.0 that
   ensures Cognos reports can be viewed and previewed in the
   Business Monitor report widgets, even in Internet Explorer.
   On Fix Central (http://www.ibm.com/support/fixcentral), search
   for JR55082:
   1. Select IBM Business Monitor from the product selector, the
   installed version to the fix pack level, and your platform, and
   then click Continue.
   2. Select APAR or SPR, enter JR55082, and click Continue.
   
   When you download fix packages, ensure that you also download
   the readme file for each fix. Review each readme file for
   additional installation instructions and information about the
   fix.
