APAR STATUS
 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  In IBM Content Navigator, when a document is opened from the
   worklist view using the Daeja ViewONE Virtual viewer, selecting
   the 'Save Then Get Next' check box and clicking the 'Save'
   button does not fetch the next document. The save operation
   completes. Additionally, the arrow buttons on the top right to
   navigate to the previous and next items are disabled.
   
   
    
   
   

LOCAL FIX

PROBLEM SUMMARY
 *  When saving properties with the  Save Then Get Next" option
   enabled from the viewer window, it doesn t open the next
   document. Also, the navigator toolbar buttons on the viewer
   window are disabled.
   
   
    
   
   

PROBLEM CONCLUSION
 *  This has been fixed in IBM Content Navigator V3.0.1. With this
   fix, the next document is opened when properties are saved.
   Also, the navigator toolbar buttons are enabled.
