ERROR DESCRIPTION
 *  Rational Developer for System z - RDz - v9.5.1.1 fails with out
   of memory error
   when editing PLI program with a recursive include.
   
   1- Create Sample PLI project using menu
   File > New > Examples > Workstation PLI > PLI Sample1
   2- Create an include file include\loop1.inc like:
   
   %INCLUDE loop1;
   
   DCL msg1 CHAR(30) varying;
   
   3- Edit HelloApp.pli and add line:
   %include loop1;
   
   4- Observe that task downloading include files runs for ever and
   that RDz eclipse/java process consumes a lot of CPU on task
   manager.
   After some time, process disappears and javacore file generated
   shows error:
   
   Javacore file, heapdump and core dump files can be found and
   show error:
   1TISIGINFO     Dump Event "systhrow" (00040000) Detail
   "java/lang/OutOfMemoryError" "Java heap space" received
   
   
    
   
   

LOCAL FIX
 *  Remove the recursive include.
   
   
    
   
   

PROBLEM SUMMARY
 *  See error description.
