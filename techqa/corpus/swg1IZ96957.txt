PROBLEM SUMMARY
 *  The "ps time+" attribute is missing for Linux OS agent.
   
   The Monitoring Agent for Linux OS does not collect the "ps
   Time+" attribute but the Monitoring Agent for UNIX OS Agent does
   collects it.
   
   
    
   
   

PROBLEM CONCLUSION
 *  The "ps Time" attribute has been added to the process attribute
   group.
   
   
   The fix for this APAR is going to be included in the following
   maintenance vehicle:
