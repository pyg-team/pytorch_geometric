Title: IBM PI92264: The IBM z/OS Connect EE API toolkit is unable to share service projects in multiple workspaces. - United States

Text:
z/os SUBSCRIBE
You can track all active APARs for this component.



APAR STATUS
 * CLOSED AS UNREPRODUCIBLE IN NEXT RELEASE.
    
   
   

ERROR DESCRIPTION
 *  The IBM z/OS Connect EE API Toolkit is unable to share service
   projects in multiple workspaces. When referencing a service
   project from one Eclipse workspace and that location is in a
   different workspace an error occurs during the build process.
   
   For a project located at:
   C:\Users\IBM_ADMIN\.zosexplorer
   
   Project referenced at:
   C:\Users\IBM_ADMIN\.zosexplorer2
   
   The following error messages are issued when there is an
   attempt to build the project:
   
   Errors occurred during the build.
   Errors running builder 'Service Project Builder' on project
   'ivtnoService'.
   C:\Users\IBM_ADMIN\.zosexplorer2\ivtnoService\service.properties
   (The system cannot find the path specified.)
   C:\Users\IBM_ADMIN\.zosexplorer2\ivtnoService\service.properties
   (The system cannot find the path specified.)
   C:\Users\IBM_ADMIN\.zosexplorer2\ivtnoService\service.properties
   (The system cannot find the path specified.)
   C:\Users\IBM_ADMIN\.zosexplorer2\ivtnoService\service.properties
   (The system cannot find the path specified.)
   
   
    
   
   

LOCAL FIX
 *  Create a copy of the service project for the new workspaces
   where the service will be built.
   
   
    
   
   

PROBLEM SUMMARY
 *  ****************************************************************
   * USERS AFFECTED: All users of the IBM z/OS Connect EE API     *
   *                 toolkit feature.                             *
   ****************************************************************
   * PROBLEM DESCRIPTION: The IBM z/OS Connect EE API toolkit is  *
   *                      unable to share service projects in     *
   *                      multiple workspaces.                    *
   ****************************************************************
   The IBM z/OS Connect EE API toolkit has been updated to enable
   the sharing of service projects between multiple Eclipse
   workspaces.
   
   
    
   
   

PROBLEM CONCLUSION

TEMPORARY FIX

COMMENTS
 *  The reported problem has been resolved in the IBM z/OS Connect
   EE API toolkit feature V3.0.3.1
