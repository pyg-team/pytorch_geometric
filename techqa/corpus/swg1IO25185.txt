APAR STATUS
 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  IBM Content Navigator is downloading XSN files stored in a
   Content Manager Repository with a .dat extension
   
   
    
   
   

LOCAL FIX

PROBLEM SUMMARY
 *  When a user downloads a Microsoft InfoPath file, IBM Content
   Navigator appends the .dat file extension.
   
   
    
   
   

PROBLEM CONCLUSION
 *  This issue has been addressed in IBM Content Navigator V2.0.3
   Fix Pack 8 LA008.
