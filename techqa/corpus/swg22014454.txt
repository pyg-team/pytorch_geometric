Title: IBM Security Bulletin: A security vulnerability has been identified in IBM WebSphere Application Server shipped with IBM Operations Analytics Predictive Insights (CVE-2017-1741) - United States

Text:
 SECURITY BULLETIN

SUMMARY
 Websphere Application Server (WAS) is shipped as a component of IBM Operations Analytics Predictive Insights. Information about a security vulnerability affecting WAS has been published in a security bulletin. 

VULNERABILITY DETAILS
Please consult the security bulletin Information disclosure in WebSphere Application Server Admin Console (CVE-2017-1741) [http://www-01.ibm.com/support/docview.wss?uid=swg22012342] for vulnerability details and information about fixes.

AFFECTED PRODUCTS AND VERSIONS
 

Principal Product and Version(s) Affected Supporting Product and Version IBM Operations Analytics Predictive Insights version 1.3.6 Websphere Application Server 8.5.5.x IBM Operations Analytics Predictive Insights versions 1. 3.3, 1.3.5 Websphere Application Server 8.5.0.x This needs to be upgraded to minimum 8.5.5.2, noting this known issue [https://www.ibm.com/support/knowledgecenter/en/SSEKCU_1.1.2.1/com.ibm.psc.doc/tshoot/psc_ts_jazzsm_start_stop.html], before applying the fix mentioned in the linked bulletin. For older versions of IBM Operations Analytics Predictive Insights it is recommended to upgrade to 1.3.6 [/support/docview.wss?uid=swg22014454&amp;aid=1] GET NOTIFIED ABOUT FUTURE SECURITY BULLETINS
