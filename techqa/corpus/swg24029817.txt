InfoSphere Foundation Tools components include IBM InfoSphere FastTrack, IBM InfoSphere Information Analyzer, InfoSphere Data Architect, IBM InfoSphere Discovery for Information Integration, IBM InfoSphere Information Analyzer Exception Manager, IBM InfoSphere Business Glossary, IBM InfoSphere Business Glossary Anywhere, and IBM InfoSphere Metadata Workbench.

Built on an open architecture, these components help profile, model, define, map, and govern information that is spread across your enterprise, so your business can deliver the right information, to the right people, at the right time. These components are designed to be deployed in a heterogeneous IT environment to leverage existing IT investments. They work with any IBM or non-IBM data source, business intelligence tool or operating system, or in conjunction with the InfoSphere Foundation Tools comprehensive set of integration products.

Downloading the product

Your proof of entitlement directs you to your license file. This license file controls which of the product modules you can install and run.

To download the product:

 1. Go to the Find Downloads and Media [http://www.ibm.com/software/howtobuy/passportadvantage/paocustomer/sdma/SD] Web page and log in.
 2. Under the Download finder options, select the Find by part number search option.
 3. Search for each downloadable image by the part number. For example: Product description File name Part number InfoSphere Information Server v8.5 Quick Start Guide Multilingual InfoSvr_v8.5_QuickStartGuide.pdf CZG5PML 
 4. Download all of the required images.
 5. Extract each of the images into a single, temporary directory on your system. Use a file extracting utility that supports large file extractions.

If you purchase additional product modules or components after you install the InfoSphere Information Server suite, do not download the InfoSphere Information Server installation image again. However, you must update the license on the services tier computer before you can install the new modules. For more information, see the IBM Information Server Planning, Installation, and Configuration Guide.
