Title: IBM Security Bulletin: Security vulnerabilities have been identified in IBM WebSphere Application Server and IBM DB2 shipped with IBM Predictive Maintenance and Quality (CVE-2015-0138, CVE-2015-0159 and CVE-2014-6221). - United States

Text:
 SECURITY BULLETIN

SUMMARY
 IBM WebSphere Application Server and IBM DB2 are shipped as components of IBM Predictive Maintenance and Quality. Information about security vulnerabilities affecting IBM WebSphere Application Server and IBM DB2 has been published in security bulletins. 

VULNERABILITY DETAILS
Please consult the Security Bulletin: Vulnerabilities in GSKit affect IBM® DB2® (CVE-2015-0138, CVE-2015-0159 and CVE-2014-6221) [http://www-01.ibm.com/support/docview.wss?uid=swg21699543] for vulnerability details and information about fixes.

Please consult the Security Bulletin: Vulnerabilities in GSKit affect IBM WebSphere MQ (CVE-2015-0159, CVE-2015-0138 and CVE-2014-6221) [http://www-01.ibm.com/support/docview.wss?uid=swg21699055]for vulnerability details and information about fixes.
