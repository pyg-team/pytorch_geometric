Title: IBM CVE-2014-0160 and the IBMi - United States

Text:
 SECURITY BULLETIN

SUMMARY
 CVE-2014-0160 - Heartbleed Vulnerability 

This CVE does not effect System SSL. System SSL is accessed through GSKit, SSL_ and the Native IBM i JSSE interface.

This CVE does not effect IBMJSSE2.

This CVE does not effect the current versions of OpenSSL IBM i 6.1 (0.9.8y), 7.1 (0.9.8y).The issue was introduced into OpenSSL 1.0.1 which is newer than the IBM i supported versions.

The simple answer in all cases is that IBM i is not effected by CVE-2014-0160
