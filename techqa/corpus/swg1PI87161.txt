Title: IBM PI87161: CONFIGURATION OPTION TO PREVENT EMAIL ACTION FROM ADDING CURRENT VIRTUAL PORTAL CONTEXT PATH TO GENERATED URLS - United States

Text:
  A FIX IS AVAILABLE
Fixes integrated in WebSphere Portal 8.5.0.0 and V9.0 Combined Cumulative Fixes [http://www-01.ibm.com/support/docview.wss?uid=swg24037786]



APAR STATUS
 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  Email actions currently uses the value of the configuration
   property wcm.authoringui.url as the base URL for any generated
   links.
   
   By default, the base URL will be modified to include the current
   virtual portal context path if applicable.
   
   To prevent the email action from adding the current virtual
   portal context path to the generated URL, add the following
   configuration property to the WCM WCMConfigService resource
   environment provider within WAS.
   wcm.email.url.includeCurrentVirtualPortalContext=false
