ERROR DESCRIPTION
 *  Syslogs are not detected sometimes on SAP Version 731.
   
   Environment:
   TEPS/TEMS/TEMA: ITM6.23 FP3 + mySAP Agent 7.1 FP1
   Affected SAP Version : 731
   
   Detailed Recreation Procedure:
   1. Import 710 FP1 on SAP system having Sap_Version 731
   2. Create some(4-5 or more if needed) syslog
    a. Go to SE38 transaction
    b. Execute RSLG0014
    c. Specify I_WILL and MSG_ID
    d. Execute
   3. Check if these syslogs come under transaction SM21.
   4. Execute function module /ibmmon/itm_sys_log
   Some of the syslogs will not be collected by this function
   module.
   
   Approver:   Chaitanya Sai Kolloju
