Title: IBM IC92476: FOR VIRTUAL DATAPOWER, AN ATTEMPT TO COPY A SCRYPT IMAGE FILE ONTO A DEVICE MAY FAIL DURING FIRMWARE UPGRADE - United States

Text:
 SUBSCRIBE
You can track all active APARs for this component.



APAR STATUS
 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  Affected are Virtual DataPower appliances that used the
   5.0.0.0 OVA release on 30 November 2012 (build 220750).
   
   An attempt to copy a scrypt image file onto a device could fail
   during firmware upgrade due to insufficient ramdisk allocation
   to accommodate the upgrade.
   
   
    
   
   

LOCAL FIX

PROBLEM SUMMARY
 *  Affected is 'firmware upgrade' for Virtual DataPower
   installations that used the 5.0.0.0 OVA release on
   30 November 2012 (build 220750).
   
   Upgrade may fail due to insufficient ramdisk allocation. Fix
   increases the size of temporary:// and image:// for improved
   compatibility with future firmware.
   
   
    
   
   

PROBLEM CONCLUSION
 *  The fix is available in 5.0.0.8.
   
   For a list of the latest fix packs available, please see:
   http://www-01.ibm.com/support/docview.wss?uid=swg21237631 [http://www-01.ibm.com/support/docview.wss?uid=swg21237631]
