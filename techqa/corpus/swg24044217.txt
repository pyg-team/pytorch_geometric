Images via Fix Central Online 

To get the latest features and enhancements it is required to update IBM Spectrum Protect™ Plus from a previous version or release. Software patches and updates are delivered via Fix Central and installed by using the IBM Spectrum Protect Plus Administrative Console. 

The Fix Central Online website contains update files for IBM Spectrum Protect Plus for download. The HTTP link in the section "Download package" directs to the Fix Central Online website. 

The images available through Fix Central do not contain license enablement files. You must have a base level package installed to use these images.

The following files are available for download of IBM Spectrum Protect Plus 10.1.0.1 (interim fix: spp_10.1.0_patch1):

The Passport Advantage Online website has complete images of IBM Spectrum Protect Plus 10.1.0 for download. The HTTP link in the section "Download package" directs to the Passport Advantage Online website. 

The images available through Passport Advantage are the base level packages. For any software patches and updates, check Fix Central.

DOWNLOAD PACKAGE
The HTTP link in the Download Options column below directs you to the following sites: 

 * Passport Advantage Online website. Passport Advantage is a secure website that requires an account ID and password. It has all of the component and platform images associated with IBM Spectrum Protect Plus 10.1.0 available for download. 
   
   You can locate images on the Passport Advantage Online website by using the part number as the search query. For example, to locate the IBM Spectrum Protect Plus for VMware virtual appliance, use the search query CNMM0ML, and for IBM Spectrum Protect Plus for Hyper-V virtual appliance, use CNMM2ML.
   
 * Fix Central Online website. Fix Central is a secure website that requires you to log in using your IBM ID. You can acquire an IBM ID for free at Sign up to IBMid [https://www.ibm.com/account/us-en/signup/register.html]. Fix Central allows you to search, select, order, and download fixes to your system with a choice of delivery options. 
   
   You can locate images on the Fix Central Online website by using the product name, version and platform as the search query. For example use the "IBM Spectrum Protect Plus". 


Reminder: The images available through Fix Central do not contain license enablement files. You must have a base level package installed to use these packages.
