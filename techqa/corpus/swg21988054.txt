Hold Notice Atlas Request TECHNOTE (FAQ)

QUESTION
 Can we change Atlas request used by the Hold Notice to a different one after we create the notice? 

CAUSE
Wrong request was used while creating Hold Notice. The Notice is still in drafted status

ANSWER
No, you cannot switch the request used by Hold Notice to a different one after the notice is created. The Hold Notice with the wrong request can be deactivated. A new Hold Notice can be created using the correct request
