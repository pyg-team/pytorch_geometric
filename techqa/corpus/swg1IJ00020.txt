APAR STATUS
 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  We have TWSv9.2 BKM on Solaris with DWC on a linux.
   When i create a forecast plan on BKM an try to run a report on
   it, it
   is timing out and failing with below error, it used to work fine
   previously.
   
   "
   AWSUI2007E The following Tivoli Workload Scheduler engine error
   occurred during the running of the report:
   AWSJCO005E WebSphere Application Server has given the following
   error:
   CORBA NO_RESPONSE 0x4942fb01 Maybe; nested exception is:
   org.omg.CORBA.NO_RESPONSE: Request 3818996 timed out vmcid: IBM
   minor
   code: B01 completed: Maybe."
   
   
    
   
   

LOCAL FIX
 *  Had user remove two jobs that were defined to run on non-active
   workstations (that were not in the current plan)  Once this was
   done the forecast plan reports ran correctly.
