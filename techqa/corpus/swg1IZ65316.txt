ERROR DESCRIPTION
 *  Environment:  IBM Tivoli Monitoring Agents for System P V6.2
   Interim Feature 1:  AIX Premium Agent
   Problem Description:
   When running the AIX Premium agent on AIX 5.3 TL5 or TL6, SPMI
   does not supply all the metrics.  The result is that all
   metrics, even the ones SPMI does supply, are displayed as zero
   for LPAR CPU Utilization, LPAR Attributes and CPU Entitlement.
