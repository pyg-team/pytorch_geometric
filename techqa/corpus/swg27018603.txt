Step 4: Install WebSphere Portal 

The Quick installation scenario provides step-by-step instructions to help you quickly install a basic working instance of WebSphere Portal.
1. You can either start the installation process from the DVD or from the appropriate downloaded product code for your operating system. 

Navigate to the Setup directory and run the following commands: 

 * IBM i: 
 * To install locally from a QSH session of the IBM i console, run install.sh. 
 * To run the installer from the Microsoft® Windows® platform and install to a remote IBM i system, run install400.bat. 
 * Linux® and UNIX®: Run ./install.sh. 
 * Windows: Run install.bat.

2. Follow the instructions in the installation pages. 
3. To verify that WebSphere Portal is running, open the following URL in a browserhttp://example.com:port_number/wps/portal 
where example.com is the fully qualified host name of the system that is running WebSphere Portal and port_number is the port number that is displayed on the confirmation page. For example, http://www.ibm.com:10039/wps/portal.
4. Install any required fixes for WebSphere Application Server at this time. See Supported hardware and software for more information about required fixes that must be applied.
