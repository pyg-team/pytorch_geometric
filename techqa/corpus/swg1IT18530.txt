APAR STATUS
 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  When using Certificate Management (iKeyman) in PCOMM R12 and
   trying to Import a .P12 certificate we get Message: The command
   cannot complete because your JRE is using restricted policy
   files
   
   We are able to import .cer certificates fine.
   
   We do not have this problem using the PCOMM 6.0.11 version of
   Certificate Management.
   
   
    
   
   

LOCAL FIX

PROBLEM SUMMARY
 *  ****************************************************************
   * USERS AFFECTED:                                              *
   * Users of Certificate Management of IBM Personal              *
   * Communications.                                              *
   ****************************************************************
   * PROBLEM DESCRIPTION:                                         *
   * Importing of .P12 Certificate file into Certificate          *
   * Management KDB results in error message: "The command cannot *
   * complete because your JRE is using restricted policy files"  *
   * being displayed.                                             *
   ****************************************************************
   * RECOMMENDATION:                                              *
   ****************************************************************
   User starts Certificate Management utility of IBM Personal
   Communications V12.
   User imports .P12 Certificate into IBM Personal communications
   database.
   Error message "The command cannot complete because your JRE is
   using restricted policy files" is displayed.
   
   
    
   
   

PROBLEM CONCLUSION
 *  Upgrade to JRE version of 8.0.4.1 resolves the restricted JCE
   Policy files.
   
   Fix scheduled for IBM Personal Communications 12.0.2.0 Refresh
   Pack.
