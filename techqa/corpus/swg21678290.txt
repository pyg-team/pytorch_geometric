Title: IBM Starting or uninstalling Installation Manager results in "Installation data has incompatible version" error - United States

Text:
RDz; uninstalling; Rational Developer for system z; deinstall; uninstall; IM; manual; manually TECHNOTE (TROUBLESHOOTING)

PROBLEM(ABSTRACT)
 Starting IBM Installation Manager (IM) results in the error "Error restoring Information Manager state". 

SYMPTOM
The error message below is received when you try to start IM: 
"Error restoring Information Manager state"
"Installation data has incompatible version 1.7.2; expected 1.6.3.1. Newer version of the installation manager was used on this system." 


All attempts to uninstall IM fail.


CAUSE
Unknown


ENVIRONMENT
Windows 7



RESOLVING THE PROBLEM
Before you manually uninstall IM, you should first try to uninstall by using either the Windows Control Panel > Add/Remove Programs, or by using the uninstall.exe provided by IM. 

For more information on uninstalling IM, review the IM documentation topic Commands to Uninstall Installation Manager [http://pic.dhe.ibm.com/infocenter/install/v1r7/index.jsp?topic=%2Fcom.ibm.silentinstall12.doc%2Ftopics%2Fr_uninstall_cmd.html&resultof%3D%2522%2575%256e%2569%256e%2573%2574%2561%256c%256c%2522%2520%2522%2575%256e%2569%256e%2573%2574%2561%256c%2522%2520]. 

The uninstall commands are in the \\agent_data_directory\uninstalldirectory. See the Agent Data Location Information [http://pic.dhe.ibm.com/infocenter/install/v1r7/index.jsp?topic=%2Fcom.ibm.silentinstall12.doc%2Ftopics%2Fr_app_data_loc.html] to determine the location of your IM packages. 

 

If you must manually uninstall IM, follow these steps: 



Disclaimer This solution contains information about modifying the system registry. Before making any modifications to the Microsoft Registry Editor, it is strongly recommended that you make a backup of the existing registry. For more information describing how to back up the registry, refer to Microsoft Knowledge Base article 256986 [http://support.microsoft.com/kb/256986]

 

 1. Delete the following directories:  * C:\Program Files\IBM\Installation Manager - 64 bit 
     * C:\Program Files (X86)\IBM\Installation Manager - 32 bit
       
     * C:\ProgramData\IBM\Installation Manager - Administrators 
     * C:\Users\user\AppData\Roaming\IBM\Installation Manager - non-administrators
       
     * C:\ProgramData\Microsoft\Windows\Start Menu\Programs
       
       
    
    
 2. Remove the following Windows registry keys:

 * HKEY_LOCAL_MACHINE\SOFTWARE\IBM\Installation Manager 
 * HKEY_LOCAL_MACHINE\SOFTWARE\Wow6432Node\IBM\Installation Manager 
 * HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows\CurrentVersion\Uninstall\IBM InstallationManager
