APAR STATUS
 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  ODBC Connector LOB jobs with BIGINT as key aborts with
   incorrect syntax error due to invalid locator query generation.
   
   
   Message seen
   
   ODBC_Connector_1,0: ODBC function "SQLExecDirect" reported:
   SQLSTATE = HY000: Native Error Code = 102: Msg = [IBM
   (DataDirect OEM)][ODBC SQL Server Wire Protocol
   driver][Microsoft SQL Server]Incorrect syntax near '='.
   (CC_OdbcInputStream::open, file CC_OdbcInputStream.cpp, line
   121)
   
   
    
   
   

LOCAL FIX
 *  Use another supported dataype for primary key column in source
   table.
   
   Disable lob referenes, in which case you may need to increase
   the value of APT_DEFAULT_TRANSPORT_BLOCK_SIZE for the job.
   
   
    
   
   

PROBLEM SUMMARY
 *  ODBC Connector LOB jobs with BIGINT as key abort with invalid
   syntax error due to incorrect locator query generated.
   
   
    
   
   

PROBLEM CONCLUSION
 *  Code has been corrected generate the locator query properly.
   
   
    
   
   

TEMPORARY FIX
 *  Pass them inline or use another type as key column.
