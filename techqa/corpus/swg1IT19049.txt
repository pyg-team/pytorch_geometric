APAR STATUS
 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  The user observed an unexpected restart of DataPower on SSH
   connections closure (in this case for an SFTP connection).
   
   
    
   
   

LOCAL FIX

PROBLEM SUMMARY
 *  Affected is the use of DataPower as an SSH client including the
   use of SFTP.  After the fix, DataPower will no longer
   unexpectedly restart in rare instances.
   
   
    
   
   

PROBLEM CONCLUSION
 *  Fix is available in 7.0.0.18, 7.1.0.15, 7.2.0.12, 7.5.0.6,
   7.5.1.5 and 7.5.2.3
