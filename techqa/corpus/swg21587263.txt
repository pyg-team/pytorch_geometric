Title: IBM Configure MQ agent on AS/400 to use ephemeral IP.PIPE - United States

Text:
MQ AS/400 i5/OS iSeries ephemeral IP.PIPE 5724K3500 5724K3500TR TECHNOTE (FAQ)

QUESTION
 How do you configure the WebSphere MQ Monitoring agent on AS/400 to use the ephemeral IP.PIPE connection? 

ANSWER
Steps to configure MQ agent on AS/400 to use ephemeral IP.PIPE 

(Note: xxxxx below is the agent suffix id, e.g. 00001.) 

 

1. stop MQ agent suffix xxxxx 

 

2. edit the KMQTMP/KMSPAxxxxx(KBBENV) file 

e.g. by command .. 

EDTF FILE(KMQTMP/KMSPA00001) MBR(KBBENV) 

 

3. add the 'EPHEMERAL:Y' keyword in the KDC_FAMILIES environment variable 

e.g. 

KDC_FAMILIES=IP.PIPE EPHEMERAL:Y PORT:1918 

 

4. start MQ agent suffix xxxxx 

 

You just need to repeat the steps for each of the MQ agent instances
