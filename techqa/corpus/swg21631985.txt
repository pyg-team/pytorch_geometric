CAUSE
There are several known causes for this error. 

 * TIP: For more examples, see separate IBM Technote #2007891.


This Technote specifically relates to the scenario where the cause is that there is an existing user (already in the system) who already has the same name as the user who is being added. 
RESOLVING THE PROBLEM
Do not create duplicate users.
