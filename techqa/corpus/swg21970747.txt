QUESTION
 Is my N100x, N200x, N300x appliance compatible with 64bit Tivoli Storage Manager client? 

ANSWER
Yes, NPS 7.2.0.5 Adds support for Netezza backup and restore operations with a 64-bit Tivoli Storage Manager 7.1.x client. 

Please see fix id 96038 in the NPS 7.2.0.5 ReadMe file
