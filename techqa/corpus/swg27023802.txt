Title: IBM WebSphere Portal 7.0.0.2: Documentation of updates performed to portlets, theme files and changes to the database schema during the 7.0.0.2 upgrade - United States

Text:
7002eGA PRODUCT README

ABSTRACT
 This document describes which portlets and theme files are updated and which database schema files are changed during the IBM WebSphere Portal 7.0.0.2 upgrade. It is designed to help you plan your upgrade.
