ERROR DESCRIPTION
 *  The current version of the IBM Eclipse Help Server has a
   number of changes to address some potential security issues.
   
   
    
   
   

LOCAL FIX

PROBLEM SUMMARY
 *  Cross-site scripting and phishing vulnerability.
   
   The IBM Eclipse Help Server is bundled with the Tivoli
   Enterprise Portal Server.  During internal testing of this
   component, a cross-site scripting vulnerability and a phishing
   vulnerability were identified.
   
   
    
   
   

PROBLEM CONCLUSION
 *  The IBM Eclipse Help Server bundled with the Tivoli Enterprise
   Portal Server was upgraded to version 3.1.2.
   
   
   The fix for this APAR is contained in the following maintenance
   packages:
