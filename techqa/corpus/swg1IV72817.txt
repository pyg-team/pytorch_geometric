ERROR DESCRIPTION
 *  Environment:
   All Supported Windows Server OS Versions.
   Microsoft Active Directory Agent 6.3.1.2
   
   Problem Description:
   Agent shows wrong value for "DCA PDC Master","DCA RID Master"
   and
   "DCA Infrastructure Master" attributes of "Domain Controller
   Availability" attribute group, when DNS name and NetBIOS name of
   Domain Controller machine are different.
   
   Detailed Recreation Procedure:
   1.Install the Active directory Agent.
   2.Request the data for "Domain Controller Availability"
   attribute
   group, then check the value of "DCA PDC Master","DCA RID Master"
   and "DCA Infrastructure Master" attributes.
   
   Related Files and Output:
   <%CANDLE_HOME%>\TMAITM6_x64\logs\*k3zcma*.log
   <%CANDLE_HOME%>\TMAITM6\logs\*k3zcma*.log
   
   
    
   
   

LOCAL FIX

PROBLEM SUMMARY
 *  Agent shows wrong value for "DCA PDC Master","DCA RID Master"
   and "DCA Infrastructure Master" attributes of DCA attribute
   group.
   
   
    
   
   

PROBLEM CONCLUSION
 *  Now Agent shows correct value for "DCA PDC Master","DCA RID
   Master" and "DCA Infrastructure Master" attributes of DCA
   attribute group.
