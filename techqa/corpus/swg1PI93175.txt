ERROR DESCRIPTION
 *  Through currently unknown causes, behavior where the well-known
   types cache cannot be loaded due to duplicate well-known types
   existing in the same component is observed. Further
   investigation shows that several well-known type resources are
   referenced in multiple components but unused in any indexed
   artifacts and also not mapped to any configurations in the
   components in question. Because of this, this data is serving no
   purpose but still renders affected components completely useless
   due to broken well-known types caches.
