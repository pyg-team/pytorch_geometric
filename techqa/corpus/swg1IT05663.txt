APAR STATUS
 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  On the Workload Console when viewing a virtual system or
   applicaton instance status it displays the
   Health status as
     "Running and the health status is 'unknown'".
   
   After logging in to the virtual machine instance, you will
   see the error message:
   'ERROR CWZMO2025E: HTTP request to ITMOS agent failed with
   return code 401.'  in the console logs under
   /opt/IBM/maestro/agent/usr/servers/XXXXXXXX.XXXXXXXX/logs/
   
   
    
   
   

LOCAL FIX
 *  Log on the VMs of the instance, go to /opt/IBM/maestro/ITM/bin
   and execute the following commands:
   
   ./itmcmd agent -f stop all
   ./itmcmd agent -f start all
   
   
    
   
   

PROBLEM SUMMARY
 *  An error condition in the health status monitoring code was
   causing the "Running and the health status is 'unknown'" status.
   
   
    
   
   

PROBLEM CONCLUSION
 *  A code fix is being provided in PureApplication System version
   2.0.0.1 to detect this error condition and restart the agent.
   
   
    
   
   

TEMPORARY FIX
