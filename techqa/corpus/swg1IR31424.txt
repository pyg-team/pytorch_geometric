Title: IBM IR31424: AE IR28065 ** GROUP 0 GLOBAL BATCHES WHICH COMPLETE WILL CAUSE OTHER BATCHES OF NON 0 GLOBAL GROUPS TO END PREMATURELY. - United States

Text:
 
APAR STATUS
 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  1. Set up an event with 3 global batches, one batch with
   group 0 (primary ID), one batch with group 3 and one
      batch with group 5.
   2. Ensure all are active on the system, then change event
      controls to the current date and end the batch.
   3. Check PM and you will see that all 3 batches prematurely
      end.  No errors are logged.
   4. APAR IR28065 must be on to see this failure.
   
   
    
   
   

LOCAL FIX

PROBLEM SUMMARY
 *  GROUP 0 GLOBAL BATCHES BROUGHT TO COMPLETION WILL CAUSE OTHER
   BATCHES OF NON 0 GLOBAL GROUPS TO END PREMATURELY.
   
   
    
   
   

PROBLEM CONCLUSION
 *  ADDED CODE TO MODULE EALPGLOP.J86 TO CHECK FOR GL 0 GLOBAL
   BEFORE REMOVING.
   HOW TO RECREATE THE PROBLEM:
   1. Set up an event with 3 global batches, one batch with
      group 0 (primary ID), one batch with group 3 and one
      batch with group 5.
   2. Ensure all are active on the system, then change event
      controls to the current date and end the batch.
   3. Check PM and you will see that all 3 batches prematurely
      end.  No errors are logged.
   4. Apar IR28065 must be on to see failure.
