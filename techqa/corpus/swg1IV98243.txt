LOCAL FIX
 *  Workaround:
   1.  Try ITCAM for DB2 installation after executing below
   command,
     setarch $(uname -m) --uname-2.6
   2.  Perform a silent installation with a response file
   containing the following entry:
        PREREQ_CHECK=NO

 *  RHEL 7.3 is not supported for DB2 agent 710 release

 *  The fix for this issue has been delievered in DB2 agent
   7.1.1 release, included in ITCAM for Application 7.2.1.2 BOM
   refresh release.
