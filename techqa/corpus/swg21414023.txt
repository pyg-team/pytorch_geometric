Title: IBM How to make LDAP authentication case insensitive in Jazz repository - United States

Text:
RTC; Team Concert; Integrations; LDAP; Security; Authentication; Jazz team server; access; import; case sensitive TECHNOTE (FAQ)

QUESTION
 How can you make LDAP authentication case insensitive in the Jazz repository for IBM Rational Team Concert? 

CAUSE
Jazz repository credentials information is case sensitive while information in LDAP might be case insensitive. Sometimes it may be required to use case insensitive authentication in Jazz repository.

ANSWER
In order to turn on case insensitive authentication:


 1. Log in to the Jazz server Web UI as an administrator 
 2. Go to the Advanced Properties section:
    
    
    [/support/docview.wss?uid=swg21414023&amp;aid=1]
    
 3. Change "Use case insensitive user ID matching" property to true:
    
    [/support/docview.wss?uid=swg21414023&amp;aid=2]
    
 4. Restart the Jazz server
