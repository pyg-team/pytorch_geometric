Title: IBM Forms 8.2.1 requires a WebSphere Application Server patch to work properly - United States

Text:
 TECHNOTE (TROUBLESHOOTING)

PROBLEM(ABSTRACT)
 IBM Forms requires a WebSphere Application Server patch to avoid security vulnerabilities and run properly. 

CAUSE
To avoid a class conflict and the security vulnerability, you must apply a patch to WebSphere Application Server (APAR PI50993 [http://www-01.ibm.com/support/docview.wss?uid=swg21969251]) before updating IBM Forms to version 8.2.1. 

If you attempt to run IBM Forms Server - Webform Server without applying the WebSphere Application Server patch, you might see the following error:

 * Security Bulletin: Potential security vulnerabilities relating to IBM Forms Server using Apache HttpComponents (CVE-2014-3577, CVE-2012-6153) [http://www-01.ibm.com/support/docview.wss?uid=swg21961713]

 * Security Bulletin: Apache HTTPComponents vulnerabilities in WebSphere Application Server (CVE-2012-6153, CVE-2014-3577) [http://www-01.ibm.com/support/docview.wss?uid=swg21969251]

RESOLVING THE PROBLEM
Download and apply the PI50993 update from WebSphere Application Server [http://www-01.ibm.com/support/docview.wss?uid=swg21969251].
