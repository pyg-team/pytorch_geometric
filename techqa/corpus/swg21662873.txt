Title: IBM Security Bulletin: Fix available for Unrestricted File Upload Security Vulnerability in IBM WebSphere Portal (CVE-2013-6722) - United States

Text:
vulnerability; security; unrestricted; upload SECURITY BULLETIN

SUMMARY
 A fix that blocks unrestricted file upload is available for a security vulnerability in IBM WebSphere Portal. 

VULNERABILITY DETAILS
CVEID: CVE-2013-6722 [http://cve.mitre.org/cgi-bin/cvename.cgi?name=CVE-2013-6722] 

DESCRIPTION: 
A security vulnerability in the 'Registration/Edit My Profile Portlet' has been identified. A remote unauthorized attacker could upload arbitrary files to the server and execute denial of service attacks.

CVSS:
CVSS Base Score: 5.8
CVSS Temporal Score: See https://exchange.xforce.ibmcloud.com/vulnerabilities/89235 [https://exchange.xforce.ibmcloud.com/vulnerabilities/89235] for the current score
CVSS Environmental Score*: Undefined
CVSS Vector: (AV:N/AC:M/Au:N/C:N/I:P/A:P)

AFFECTED PRODUCTS AND VERSIONS
WebSphere Portal 8 (Fix Pack levels 8.0.0.0 and 8.0.0.1)
WebSphere Portal 7 (Fix Pack levels 7.0.0.1 and 7.0.0.2)

REMEDIATION/FIXES
The recommended solution is to apply PI07013 as soon as practical.
Fix: Apply Interim Fix PI07013 or a Cumulative Fix containing PI07013.

For 8.0.0 through 8.0.0.1
▪ Upgrade to Fix Pack 8.0.0.1 with Cumulative Fix 9 (CF09) and then apply Interim Fix PI07013 [http://www.ibm.com/eserver/support/fixes/fixcentral/swg/quickorder?productid=WebSphere%20Portal&brandid=5&apar=PI07013]
(Combined Cumulative Fixes for WebSphere Portal 8.0.0.1: http://www-01.ibm.com/support/docview.wss?uid=swg24034497 [http://www-01.ibm.com/support/docview.wss?uid=swg24034497])

For 7.0.0.1 through 7.0.0.2
▪ Upgrade to Fix Pack 7.0.0.2 with Cumulative Fix 27 (CF27)
(Combined Cumulative fixes for WebSphere Portal 7.0.0.2: http://www.ibm.com/support/docview.wss?uid=swg24029452 [http://www.ibm.com/support/docview.wss?uid=swg24029452])

WORKAROUNDS AND MITIGATIONS
Disable access to the 'Registration/Edit My Profile Portlet' using Portal Access Control.
Apply the workaround until the APAR is installed.
