ABSTRACT
 This document describes how to download and assemble the IBM Records Manager V8.5 components from the Passport Advantage Online Web site. 

DOWNLOAD DESCRIPTION


[http://www.ibm.com/i/c.gif] 

Downloading the product  1. From a browser, navigate to the Find Downloads and Media [ http://www.ibm.com/software/howtobuy/passportadvantage/paocustomer/sdma/SDMA?PO=DOWNLOAD_FIND_BY_PART_NUMBERl] page on the IBM Passport Advantage Online Web site. You must log on to continue. 
 2. Select the Find by part number search option under Download finder options. 
 3. Search for the following part numbers and download the images.
    Application Part Number IBM Records Manager V8.5 CR9HFML IBM DB2® Enterprise Server Edition V9.5 for IBM Records Manager V8.5 CR9HGML IBM WebSphere® Application Server V7.0 for IBM Records Manager V8.5 CR9HHML 
    
 4. Unpack the part into a temporary directory on your system. For AIX, first unpack the .gz file, and then enter the command cpio -idcmv <*.cpio to unpack the .cpio file, where *.cpio is the name of the package. 
 5. Unpack the Quick Start Guide part C1Y9SML.zip (included in the IBM Records Manager part CR8KRML) to a single, temporary directory on your system. Open the guide using Acrobat Reader for an overview of what components are needed to install the product. 
 6. Follow the instructions in Planning and Installing IBM Records Manager [http://publib.boulder.ibm.com/infocenter/irm/v8r5m0/topic/com.ibm.installingirm.doc/rminst0100.htm] to install the product. For the latest information, see the IBM Records Manager V8.5 Release Notes [http://www.ibm.com/support/docview.wss?rs=817&uid=swg27014789]. 
 7. The IBM Records Manager V8.5 Toolkit can be downloaded from the IBM Software ftp site: ftp://public.dhe.ibm.com/software/db2recordmanager/IRM8.5/Toolkit/ [ftp://public.dhe.ibm.com/software/db2recordmanager/IRM8.5/Toolkit/]. The IBM Records Manager V8.5 Toolkit includes a sample database package and other resources to help you integrate with or extend IBM Records Manager. The Toolkit can also help you develop solutions that are compliant with the United States Department of Defense (DoD) 5015.02 standard.
