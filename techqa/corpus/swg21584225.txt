Title: IBM Studio - Error when opening templates - exception has been thrown by the target of an invocation - United States

Text:
 TECHNOTE (TROUBLESHOOTING)

PROBLEM(ABSTRACT)
 When trying to open templates in studio (locally or from server) the following error occurs : "exception has been thrown by the target of an invocation" 

SYMPTOM
Exception has been thrown by the target of an invocation [System.RuntimeMethodHandle._InvokeMethodFast()]
- Access to the path '[path to Clarity Studio]\[Clarity Studio folder]\ClarityStudio.Settings' is denied.@[System.IO.__Error.WinIOError()]


CAUSE
The Studio user does not have read/write permissions on the Clarity Studio folder contained in the error message

ENVIRONMENT
Clarity Studio 7.x


Clarity Server 7.x


DIAGNOSING THE PROBLEM
-Open studio
-Load any template/report



RESOLVING THE PROBLEM
The user must have full permissions (read and write) set on the Studio folder
