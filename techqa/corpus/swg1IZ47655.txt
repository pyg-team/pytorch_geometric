ERROR DESCRIPTION
 *  Problem: No data seen in ErrorLog Alert workspace
   
   Approver: MJK
   Compid: 5724B96MO
   
   Environment:  All windows versions
   
   Microsoft SQL Server Agent 6.2.0.
   
   Microsoft SQL Server 2000 Enterprise Edition SP4
   
   Problem Description:
   --------------------
   ErrorLog Alert workspace does not show data
   on Tivoli Enterprise Portal for Microsoft SQL Server for
   Non-English version.
   This happens because Microsoft SQL Server  agent fails to parse
   MSSQL Server ERRORLOG which contains Non-English messages.
   
   Detailed Recreation Procedure:
   ------------------------------
   1. Install Microsoft SQL Server 2000 SP4 Non-English Version
   2. Go to TEP
   3. click on ErrorLog Alert  workspace
   4. Check Attributes values in ErrorLog Alert Detail view.
   
   Related Files and Output:
   -------------------------
   Non English Microsoft SQL Server ErrorLog file  at location:
   
   C:\Program Files\Microsoft SQL
   Server\<%SQLInstanceId%>\MSSQL\LOG\ERRORLOG

PROBLEM SUMMARY
 *  NO DATA SEEN IN ERRORLOG ALERT WORKSPACE.
