ABSTRACT
 IBM WebSphere Application Server Version 7 Fix Pack 1, also known as Version 7.0.0.1, for an installed test environment extension. 

DOWNLOAD DESCRIPTION
This update applies IBM WebSphere Application Server Version 7 Fix Pack 1, also known as Version 7.0.0.1, to the WebSphere Application Server v7.0 Test Environment extension included with Rational Software Delivery Platform products. 




For details of this update such as known issues, view the WebSphere Application Server 7.0.0.1 readme at http://www-1.ibm.com/support/docview.wss?rs=180&uid=swg27014017 [http://www-1.ibm.com/support/docview.wss?rs=180&uid=swg27014017].

PREREQUISITES
You must have the WebSphere Application Server v7.0 Test Environment extension installed with one of the following products: 

 * IBM Rational Application Developer for WebSphere Software, v7.5.x 
 * IBM Rational Software Architect for WebSphere Software, v7.5.x


You need up to 1.4 GB of disk space to install the fix pack. You also need 3.5 GB of disk space to download and extract the installation files. INSTALLATION INSTRUCTIONS


Important: By default, you do not need to download the compressed files that contain the update; the update is downloaded and installed by IBM Installation Manager directly from the IBM update repository.


For instructions on installing this update using Installation Manager, see the topic "Updating Installed Product Packages" in the information center for Rational Application Developer for WebSphere Software [http://publib.boulder.ibm.com/infocenter/radhelp/v7r5/topic/com.ibm.rad.install.doc/topics/t_update.html] or Rational Software Architect for WebSphere Software [http://publib.boulder.ibm.com/infocenter/rsawshlp/v7r5m0/topic/com.ibm.xtools.installation.rsaws.doc/topics/t_update.html].


To download and install the update from the compressed files, complete the following steps: 


 1. 
 2. Download the update files listed in the download table below.
 3. 
 4. Extract the compressed files in an appropriate directory. For example, choose to extract to C:\temp\update
 5. 
 6. 
 7. Add the update's repository location in IBM Installation Manager:
     1.  
     2.  Start IBM Installation Manager.
     3.  
     4.  On the Start page of Installation Manager, click File > Preferences, and then click Repositories. The Repositories page opens.
     5.  
     6.  On the Repositories page, click Add Repository.
     7.  
     8.  In the Add repository window, browse to or enter the file path to the diskTag.inf file, which is located in the disk1 sub-directory in the directory where you extracted the compressed files and then click OK. For example, enter C:\temp\updates\disk1\diskTag.inf.
     9.  
     10. Click OK to close the Preference page.
     11. 
    
    
    
 8. Install the update as described in the the topic "Updating Installed Product Packages" in the information center for Rational Application Developer for WebSphere Software [http://publib.boulder.ibm.com/infocenter/radhelp/v7r5/topic/com.ibm.rad.install.doc/topics/t_update.html] or Rational Software Architect for WebSphere Software [http://publib.boulder.ibm.com/infocenter/rsawshlp/v7r5m0/topic/com.ibm.xtools.installation.rsaws.doc/topics/t_update.html]. 
 9.
