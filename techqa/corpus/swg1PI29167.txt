PROBLEM SUMMARY
 *  ****************************************************************
   * USERS AFFECTED:  All users of IBM WebSphere Application      *
   *                  Server Liberty Core                         *
   ****************************************************************
   * PROBLEM DESCRIPTION: NullPointerException intermittently     *
   *                      prevents EBA application starting       *
   *                      successfully.                           *
   ****************************************************************
   * RECOMMENDATION:                                              *
   ****************************************************************
   A change in the threading for EBA application start makes an
   existing potential for NullPointerException more likely.  The
   cause of the NullPointerException is in Apache Felix bundle
   resolver.
   
   
    
   
   

PROBLEM CONCLUSION
 *  This APAR resolves the work item by correcting the programming
   error that results in the NullPointerException.
   
   The fix for this APAR is currently targeted for inclusion in
   fix pack 8.5.5.5. Please refer to the Recommended Updates page
   for delivery information:
