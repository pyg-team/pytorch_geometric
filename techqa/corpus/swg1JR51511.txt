LOCAL FIX
 *  Disable Java 2 security.
   
   
    
   
   

PROBLEM SUMMARY
 *  When Java 2 security is enabled, the IBM BPM infrastructure
   application doesn't have enough privileges to access the session
    workspace, so you cannot access Process Center from a web
   browser or Process Designer.
   
   
    
   
   

PROBLEM CONCLUSION
 *  A fix is available for IBM BPM V8.0.1.3 that allows you to
   access Process Center from a web browser or Process Designer
   when Java 2 security is enabled.
   
   On Fix Central (http://www.ibm.com/support/fixcentral), search
   for JR51511:
   
   1. Select IBM Business Process Manager with your edition from
   the product selector, the installed version to the fix pack
   level, and your platform, and then click Continue.
   2. Select APAR or SPR, enter JR51511, and click Continue.
   3. Repeat steps 1 and 2 for JR51770 [http://www-01.ibm.com/support/docview.wss?uid=swg1JR51770].
   
   When you download fix packages, ensure that you also download
   the readme file for each fix. Review each readme file for
   additional installation instructions and information about the
   fix.
