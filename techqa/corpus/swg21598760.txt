QUESTION
 What to collect for IBM support to help analyze and understand a TBSM event issue? 

ANSWER
Update PMR, Provide the following items to allow support to analyze a TBSM problems with events:

1) Provide a concise problem description that includes things like:
a timeline, sequence of events, any error messages issued, and a scenario on how to recreate the problem you encountered. 

2) Send TBSM version information as outlined in Version Technote [http://www.ibm.com/support/docview.wss?uid=swg21509592]

3) Can you recreate the Issue? if so and your TBSM system is:
TBSM 4.2.1 FP3 (or greater) OR TBSM 6.1 FP1 (or greater) OR any version of TBSM 6.1.1, Then please turn on event logging: (for other TBSM versions leave default trace levels):
