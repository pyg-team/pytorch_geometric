ABSTRACT
 This new gateway will be ready for downloading on July 20, 2017. 

CONTENT
 

This gateway is written to support the forwarding of events from the Netcool/OMNIbus ObjectServer to the Cloud Event Management service.

You can download the installation package from the IBM Passport Advantage website: 
www.ibm.com/software/howtobuy/passportadvantage/pao_customers.htm [http://www.ibm.com/software/howtobuy/passportadvantage/pao_customers.htm]

You can access the latest version of the guide written for this gateway on the IBM Knowledge Center: www-01.ibm.com/support/knowledgecenter/SSSHTQ/omnibus/gateways/cemgw/wip/concept/cemgw_intro.html [http://www-01.ibm.com/support/knowledgecenter/SSSHTQ/omnibus/gateways/cemgw/wip/concept/cemgw_intro.html]
