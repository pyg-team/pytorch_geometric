ERROR DESCRIPTION
 *  When IMS backend server sends null response to DataPower via IMS
   Connect DataPower might time out.

PROBLEM SUMMARY
 *  Affected are Datapower configurations using IMS Connect.
   
   A message shorter than expected received from the backend is
   ignored and ultimately times out.
