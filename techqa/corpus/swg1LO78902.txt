APAR STATUS
 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  Steps to reproduce.
   1. Create a document with a RichText field on a form and set it
   to render as HTML on the web.  Set the field STORAGE as "Store
   as HTML and MIME".
   2. Create a document on the web and save it.
   3. Checking the field mime format it is stored as "text/plain".
   4. Change the RichText field to Java Applet or JavaScript and
   repeat creating a document. The field will be stored in
   "text/html".
   
   
    
   
   

LOCAL FIX
 *  Do not use HTML rendering to store data in "text/html" format.
   
   
    
   
   

PROBLEM SUMMARY
 *  Fixed an issue where setting a richtext field to render on the
    web as "HTML" and storage as "HTML and MIME" causes it to save
    as Text/Plain and not Text/HTML
   
   
    
   
   

PROBLEM CONCLUSION
 *  Fixed an issue where setting a richtext field to render on the
    web as "HTML" and storage as "HTML and MIME" causes it to save
