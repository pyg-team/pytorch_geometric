Title: IBM LO70445: HTML TAGS SHOWING UP IN PREVIEW OF MESSAGES ON DEVICE - United States

Text:
 SUBSCRIBE
You can track all active APARs for this component.



APAR STATUS
 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  With some custom mail templates emails preview on the device are
   showing up as HTML text vs plain text.  And in some cases these
   messages are too long and cause the rest of the message not to
   be down loaded properly.
   
   
    
   
   

LOCAL FIX
 *  Apply IBM template vs customers custom template
   
   
    
   
   

PROBLEM SUMMARY
 *  Customer reported that some emails would display HTML markup in
   the preview of e-mail.  The preview should show plain text only,
   no HTML.
   
   
    
   
   

PROBLEM CONCLUSION
 *  Problem only happened for certain documents or custom mail
   templates.  The Lotus Traveler server was updated to handle
   these scenarios correctly.
   
   
    
   
   

TEMPORARY FIX
 *  This fix is included in Lotus Traveler 8.5.3 Upgrade Pack 1
   Interim Fix 1 and will be in all future releases.  Please see
   this technote for the latest maintenance release information.
   
   http://www.ibm.com/support/docview.wss?uid=swg24019529 [http://www.ibm.com/support/docview.wss?uid=swg24019529]
