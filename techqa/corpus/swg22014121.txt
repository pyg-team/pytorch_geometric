Title: IBM Security Bulletin: Multiple Security Vulnerabilities have been Identified in IBM WebSphere Application Server Shipped with IBM WebSphere Portal - United States

Text:
 SECURITY BULLETIN

SUMMARY
IBM WebSphere Application Server is shipped as a component of IBM WebSphere Portal. Information about security vulnerabilities affecting IBM WebSphere Application Server has been published in security bulletins.

VULNERABILITY DETAILS
Please consult the security bulletins for IBM WebSphere Application Server published at

 * Security Bulletin: Potential Privilege Escalation in WebSphere Application Server Admin Console (CVE-2017-1731) [https://www-01.ibm.com/support/docview.wss?uid=swg22012345]

 * Security Bulletin: Security vulnerability in Apache Commons FileUpload used by WebSphere Application Server (CVE-2016-1000031) [https://www-01.ibm.com/support/docview.wss?uid=swg22011428]
 * Security Bulletin: Multiple Vulnerabilities in IBM® Java SDK affects WebSphere Application Server January 2018 CPU [https://www-01.ibm.com/support/docview.wss?uid=swg22013818]
 * Security Bulletin: Information disclosure in WebSphere Application Server Admin Console (CVE-2017-1741) [https://www-01.ibm.com/support/docview.wss?uid=swg22012342]
 * Security Bulletin: Denial of Service in Apache CXF used by WebSphere Application Server (CVE-2017-12624) [https://www-01.ibm.com/support/docview.wss?uid=swg22013597]
 * Security Bulletin: Security Bulletin: Information disclosure in IBM HTTP Server (CVE-2018-1388) [https://www-01.ibm.com/support/docview.wss?uid=swg22014196]

 * Security Bulletin: Information Disclosure in WebSphere Application Server (CVE-2017-1743) [https://www-01.ibm.com/support/docview.wss?uid=swg22013601]
