Title: IBM Error "At least one of the requested properties couldn't be found: ITXFormTemplate" when creating IBM FileNet P8eForms form policy - United States

Text:
eForms; p8eF; Workflow Policy; Policy Creation TECHNOTE (TROUBLESHOOTING)

PROBLEM(ABSTRACT)
 Attemtps to create IBM FileNet P8eForms form policy may result in the following error:

At least one of the requested properties couldn't be found: ITXFormTemplate 

SYMPTOM
Cannot create form policies. See error below: 

[/support/docview.wss?uid=swg21398497&aid=1] [/support/docview.wss?uid=swg21398497&aid=1]


CAUSE
The property "Form Policy" is missing from class "Form Template"


ENVIRONMENT
Microsoft Windows Server 2003, IBM WebSphere 6.1, Oracle 10g, eForms for P8 4.0.2



RESOLVING THE PROBLEM
Here is the procedure to resolve the issue: 

 1. Browse to the Object Store and click on "Document Class" 
 2. Click on "Form Template" 
 3. Choose "ITX Form Template", right click and select "Properties" 
 4. Click on "Property Definitions" Tab and Click "Add/Remove" 
 5. Select "Show Object Type" and Select "Form Policy" and move to the right 
 6. Click "Ok" 
 7. Select the "Form Policy" property and choose "Edit" 
 8. Click on "More" Tab and do the following:  1. Click "Hidden" check box 
     2. Set the Settability to: "Read/Write" 
     3. Set the Required Class to: "Form Policy" 
     4. Set the Reflective Property to: "Form Template" 
     5. Set the Delete Action to: "None" 
     6. Choose "Ok"
    
    
 9. Choose "Apply" and then Click "OK"
