Decision Validation Services DOWNLOADABLE FILES

ABSTRACT
 This cumulative fix pack updates Decision Validation Services V7.1.1 (7.1.1, 7.1.1.1 or 7.1.1.3) to V7.1.1.4. 

DOWNLOAD DESCRIPTION
This cumulative fix pack updates Decision Validation Services V7.1.1 (7.1.1, 7.1.1.1 or 7.1.1.3) to V7.1.1.4.


The fix pack is not compatible with Decision Validation Services V7.0.x and V7.1.0. Update to 7.1.1 to be able to apply this fix pack.

For details on problems fixed in Fix Pack 4, refer to the Fix list for V7.1 [http://www-01.ibm.com/support/docview.wss?uid=swg27019787#7114] document.

This fix pack is entitled. Only customers with Decision Validation Services V7.1.1 can download this fix pack.

PREREQUISITES
 

 * WebSphere ILOG Decision Validation Services V7.1.1 or newer, 
 * WebSphere ILOG JRules V7.1.1.4, 
 * and WebSphere ILOG Rule Team Server V7.1.1.4

INSTALLATION INSTRUCTIONS
Download and execute the fix pack installer.
