APAR STATUS
 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  The cause of the problem is that the thread context class loader
   is set
   to an application class loader that has overriden the XML parser
   implementation.  The application code calls SIB, which calls SM,
   which
   calls EMF to read an XML document, which uses the incompatible
   XML
   parser implementation, which causes a ClassCastException.
   
   
    
   
   

LOCAL FIX
 *  non
