ABSTRACT
 The SupportPac includes performance charts showing performance measurements to present the performance capabilities of XMS .Net which is available with WebSphere MQ 7.1 on Windows. 

DOWNLOAD DESCRIPTION
This SupportPac includes: 


• Performance measurements with figures and tables to present the performance capabilities of XMS .Net with WebSphere MQ V7.1.
• Performance measurements for XMS .Net using connection modes as managed and unmanaged client and bindings with WebSphere MQ 7.1 with different scenarios.
• Performance comparison using MQ V7 client features.
• Tuning guidelines for better performance when using XMS .Net with WebSphere MQ 7.1

Possible Uses
This SupportPac is designed for people who:
• Will be designing and implementing solutions using WebSphere MQ XMS .Net on Windows.
• Want to understand the performance limits of WebSphere MQ/XMS .Net on Windows.
• Want to understand how to tune WebSphere MQ/XMS .Net on Windows.

Skill Level Required
The reader should have a general awareness of XMS .Net and WebSphere MQ in order to make best use of this SupportPac.
