APAR STATUS
 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  When you select Business Process Rules Manager on the Server
   launch menu in the Servers tab, a message appears saying that
   the application is not installed on the server.
   
   PRODUCTS AFFECTED:
   IBM Integration Designer
   
   
    
   
   

LOCAL FIX
 *  Open the Business Process Rules Manager in a browser by using
   the URL http://<hostname>[:<port>]/br [http://<hostname>[:<port>]/br]
   
   
    
   
   

PROBLEM SUMMARY
 *  No additional information is available.
   
   
    
   
   

PROBLEM CONCLUSION
 *  A fix is available for IBM Integration Designer V8.5.5 that
   enables the Business Process Rules Manager menu item.
   
   On Fix Central (http://www.ibm.com/support/fixcentral), search
   for JR52804:
   1. Select IBM Integration Designer with your edition from the
   product selector, the installed version to the fix pack level,
   and your platform, and then click Continue.
   2. Select APAR or SPR, enter JR52804, and click Continue.
   
   When you download fix packages, ensure that you also download
   the readme file for each fix. Review each readme file for
   additional installation instructions and information about the
   fix.
   
   
    
   
   

TEMPORARY FIX
 *  Open the Business Process Rules Manager in a browser by using
   the URL http://<hostname>[:<port>]/br [http://<hostname>[:<port>]/br]
