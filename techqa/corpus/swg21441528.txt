Title: IBM Information Server Version 8.1 installation and upgrade fails on Linux Red Hat or SUSE 64 bit with RegistrationCommand.sh error - United States

Text:
redhat InformationServer TECHNOTE (TROUBLESHOOTING)

PROBLEM(ABSTRACT)
 Information Server Version 8.1 installation and upgrade fails with Error: ServiceException: (error code = 2; message = "EASBApplication13007: 
/opt/IBM/InformationServer/ASBNode/bin/RegistrationCommand.sh command 
did not return 0, returned 1 

SYMPTOM
Installation fails with Error: ServiceException: (error code = 2; message = "EASBApplication13007: 
/opt/IBM/InformationServer/ASBNode/bin/RegistrationCommand.sh command 
did not return 0, returned 1 


CAUSE
Version 6.0.2 Java SDK 1.4.2 cumulative fix pack was not obtained prior to installation

DIAGNOSING THE PROBLEM
Verify the java that is being used by issuing the following command:
which java 

 * Verify the Java SDK version by using the following command: 

/opt/IBM/WebSphere/AppServer/java/bin/java -fullversion The correct reported version is one of the following results: 

 * Linux cue graphic [/support/docview.wss?uid=swg21441528&amp;aid=1]J2RE 1.4.2 IBM build j9xa64142-20080130 (SR10)

RESOLVING THE PROBLEM
Use the Update Installer to install both Fix Pack 27 to the application server and the Java SDK for the application server by using the instructions that follow. You can also refer to detailed instructions available at http://www.ibm.com/support/docview.wss?rs=180&uid=swg24018618 [http://www.ibm.com/support/docview.wss?rs=180&uid=swg24018618] 

a. Obtain Fix Pack 27 for version 6.0.2 of the application server: 

 * Linux cue graphic [/support/docview.wss?uid=swg21441528&amp;aid=2]6.0.2-WS-WAS-LinuxX64-FP00000027.pak [ftp://public.dhe.ibm.com/software/websphere/appserv/support/fixpacks/was60/refreshpack2/cumulative/cf60227/LinuxX64/6.0.2-WS-WAS-LinuxX64-FP00000027.pak]

b. Obtain the version 6.0.2 Java SDK 1.4.2 cumulative fix pack:  * Linux cue graphic [/support/docview.wss?uid=swg21441528&amp;aid=3]6.0.2-WS-WASJavaSDK-LinuxX64-FP00000027.pak [ftp://public.dhe.ibm.com/software/websphere/appserv/support/fixpacks/was60/refreshpack2/cumulative/cf60227/LinuxX64/6.0.2-WS-WASJavaSDK-LinuxX64-FP00000027.pak]

c. Stop the application server processes as described previously. 
d. From the /opt/IBM/WebSphere/UpdateInstaller directory, run the installation command: 
./update.sh 
e. Specify both the fix pack and the Java SDK as the available maintenance packages to install.
