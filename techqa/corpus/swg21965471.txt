Title: IBM TCTUI1003E error when selecting dataset to configure Impact widget in DASH - United States

Text:
 TECHNOTE (TROUBLESHOOTING)

PROBLEM(ABSTRACT)
 After upgrading to JazzSM 1.1 FP2. This error occurred when selecting a dataset when configuring DASHBoard Application Services Hub (DASH) widgets:
"TCTUI1003E An error has occurred communicating with the server." 

SYMPTOM
When selecting the "Edit" -> "Select a Dataset", it will error out with the 'TCTUI1003E" error.


CAUSE
Software bug in JazzSM 1.1 FP02.

ENVIRONMENT
Attempting to select a Dataset when configuring an Impact Widget in DASH
