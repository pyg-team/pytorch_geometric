 DETAILED SYSTEM REQUIREMENTS

ABSTRACT
 This page contains links to detailed system requirements for IBM InfoSphere Information Server Pack for SAP Applications. 

CONTENT
IBM provides you with a Software Product Compatibility Reports (SPCR) tool where you can search and find complete details of supported operating systems, system requirements, prerequisites, and optional supported software for IBM products. The SPCR tool links to information about the minimum product levels that must be installed before opening a problem report with IBM Technical Support. The software versions and any maintenance levels listed in the tool have been explicitly verified as working.
