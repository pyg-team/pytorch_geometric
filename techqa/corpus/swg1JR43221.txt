PROBLEM SUMMARY
 *  ****************************************************************
   * USERS AFFECTED:  Users of BPM Express, BPM Standard, and     *
   *                  BPM Advanced.                               *
   ****************************************************************
   * PROBLEM DESCRIPTION: Coach containing either Document        *
   *                      Attachment/Viewer control, when         *
   *                      previewing the coach results in HTML    *
   *                      rendering error.                        *
   ****************************************************************
   * RECOMMENDATION:                                              *
   ****************************************************************
   Code did not catch error when handling the user id and
   password associated with document attachment/viewer controls.
   Exception is thrown because at preview time user id and
   password is not yet available.
   
   
    
   
   

PROBLEM CONCLUSION
 *  To correct this functionality, a "try/catch" clause was added
   to the code to properly handle these variables.
