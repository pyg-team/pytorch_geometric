Title: IBM How to enable TLS 1.2 for IBM Rational DOORS Web Access? - United States

Text:
DWA; TLSv1.2; TLS; Transport Layer Security; TLS configuration; DWA TLS TECHNOTE (FAQ)

QUESTION
 How to enable TLS 1.2 (Transport Layer Security) for IBM Rational DOORS Web Access (DWA) product? 

CAUSE
The documentation link suggests to enable DWA for TLS as follows:
https://www.ibm.com/support/knowledgecenter/en/SSYQBZ_9.6.1/com.ibm.rational.dwa.install.doc/topics/t_configureSSL.html [https://www.ibm.com/support/knowledgecenter/en/SSYQBZ_9.6.1/com.ibm.rational.dwa.install.doc/topics/t_configureSSL.html]

However, this enables the TLS but not specific to TLS 1.2.

ANSWER
To enable TLS 1.2 version: 

 1. Open server.xml file in Notepad 
 2. Add following lines:
    
    
    sslProtocol="TLS" 
    sslEnabledProtocols="TLSv1.2,TLSv1.1"
    
    
 3. Save and Close the File 
 4. Re-start DWA
