APAR STATUS
 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  Unable to update HATS to the latest fixpack when it is installed
   in the same package group as RDz.
   Swt beans present in HATS 9.5 is not matching with the swt beans
   added in latest RDz.
   com.ibm.etools.terminal.beans used in HATS is 3.0.3.201503261213
   and the same plugin used in latest RDz is 3.0.4.201706130919.
   They are not compatible so it is not allowing HATS update to
   the latest fixpack.
   
   
    
   
   

LOCAL FIX

 *  ****************************************************************

   ****************************************************************
   * RECOMMENDATION:                                              *
   ****************************************************************
   
   
    
   
   

PROBLEM CONCLUSION
