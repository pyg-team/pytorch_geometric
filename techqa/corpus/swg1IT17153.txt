Title: IBM IT17153: INSTABILITY OF MQ CLIENT CONNECTIONS WHEN SHARING CONVERSATIONS IS ENABLED - United States

APAR STATUS
 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  Enabling Sharing Conversations on the DataPower MQ client can
   results in connection instability, leading to unexpected MQ
   2009 & 2059 Reason Code events.
   
   
    
   
   

LOCAL FIX
 *  Turn off Sharing Conversations.
   
   
    
   
   

PROBLEM SUMMARY
 *  When using sharing conversation, users might find unexpected
   MQRC 2009 and 2059 reason codes.
   
   
    
   
   

PROBLEM CONCLUSION
 *  Fix is available in 7.2.0.10
   
   For a list of the latest fix packs available, please see:
   http://www-01.ibm.com/support/docview.wss?uid=swg21237631 [http://www-01.ibm.com/support/docview.wss?uid=swg21237631]
