ABSTRACT
 This Interim Fix provides product corrections, enhancements and security updates for IBM Cognos Disclosure Management 10.2.4. 

DOWNLOAD DESCRIPTION
Cognos Disclosure Management 10.2.4 Interim Fix 10 is an update of IBM Cognos Disclosure Management 10.2.4 for existing IBM Cognos Disclosure Management customers. This Interim Fix addresses security vulnerabilities and includes important product corrections detailed in the Problem Solved section below.


Please refer to the following Security Bulletin for more details: 
Cognos Disclosure Management file upload bypass (CVE-2015-5014) [http://www-01.ibm.com/support/docview.wss?uid=swg21967228] 

Additionally, a new capability for SaaS offerings to connect to On Prem HFM data sources has been introduced, more details in this Technote [http://www-01.ibm.com/support/docview.wss?uid=swg27046820]. This Interim Fix contains also third party updates, most of them around the report generation engine.

PREREQUISITES
This Interim Fix requires that Cognos Disclosure Management 10.2.4 is already installed. An existing installation may optionally include earlier interim fixes for Cognos Disclosure Management 10.2.4.

Our interim fixes are cumulative by design. Hence Cognos Disclosure Management 10.2.4 Interim Fix 10 includes all updates from earlier 10.2.4 interim fixes.
