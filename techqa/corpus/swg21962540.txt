Title: IBM Security Bulletin: A security vulnerability has been identified in WebSphere Application Server shipped with IBM Tivoli Service Automation Manager (CVE-2015-4000) - United States

Text:
CVE-2015-4000 SECURITY BULLETIN

SUMMARY
 WebSphere Application Server is shipped as a component of Tivoli Service Automation Manager. Information about a security vulnerability that affects WebSphere Application Server has been published in the security bulletin. 

VULNERABILITY DETAILS
Review Security Bulletin: Vulnerability with Diffie-Hellman ciphers may affect IBM WebSphere Application Server (CVE-2015-4000) [http://www.ibm.com/support/docview.wss?uid=swg21957980] for vulnerability details.

AFFECTED PRODUCTS AND VERSIONS
 

Principal Product and Versions Affected Supporting Product and Version Tivoli Service Automation Manager V7.2.1 through V7.2.4 WebSphere Application Server V6.1.0.0 through 6.1.0.47. REMEDIATION/FIXES
Review Security Bulletin: Vulnerability with Diffie-Hellman ciphers may affect IBM WebSphere Application Server (CVE-2015-4000) [http://www.ibm.com/support/docview.wss?uid=swg21957980] for information about fixes.
