Title: IBM FileNet Enterprise Manager displays the error: "A network error occurred when processing a Web service request or response." - United States

Text:
 TECHNOTE (TROUBLESHOOTING)

PROBLEM(ABSTRACT)
 An error occurs on Windows 8 and Windows 2012 when FileNet Enterprise Manager tries to connect through SSL to Content Platform Engine on WebSphere Application Server 7. 

SYMPTOM
FileNet Enterprise Manager displays the following error: "A network error occurred when processing a Web service request or response. See the nested exception for specific details. The underlying connection was closed: An unexpected error occurred on a send. Authentication failed because the remote party has closed the transport stream."


CAUSE
This issue occurs due to an issue in the IBM Java SDK that is shipped with IBM WebSphere Application Server 7.

ENVIRONMENT
FileNet Enterprise Manager on Windows 8 and Windows 2012

DIAGNOSING THE PROBLEM
FileNet Enterprise Manager connects successfully to a Content Platform Engine server that uses Microsoft Web Services Extensions (WSE) but gets an error when using SSL to connect to a Content Platform Engine server.

RESOLVING THE PROBLEM
Maintain the WebSphere application server version 7 minimum fixpack and interm fix level to WAS 7.0.0.27 + IF PM80757 to resolve the issue.
