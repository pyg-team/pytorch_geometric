Title: IBM System requirements for IBM TXSeries for Multiplatforms V8.1 on Microsoft Windows - United States

Text:
 NEWS

ABSTRACT
 The following tables provide you with a listing of supported IBM and vendor-acquired software for use with IBM TXSeries for Multiplatforms V8.1 on Microsoft Windows. 

CONTENT

This document provides information on IBM and vendor-acquired software that is compatible with TXSeries for Multiplatforms V8.1. The minimum maintenance levels are listed. TXSeries is not tested with every maintenance level of every product. In general, you can install and run with updates to supported product levels, if those updates are forward compatible. If you use later maintenance levels and experience problems, test with the levels specified in this document before contacting your IBM support representative. 

For the supported software that is listed, refer to the respective product website for their system requirements and specifications. 

Note: 

 * TXSeries is a 32-bit product. All libraries linked with TXSeries must be 32-bit libraries. Most vendors offering 64-bit versions of their products also provide 32-bit client libraries. On 64-bit operating systems, TXSeries runs in 32-bit compatibility mode. 
 * IBM recommends that you maintain your TXSeries system environments at the latest fix pack level. See TXSeries for Multiplatforms support policy statement [http://www.ibm.com/support/docview.wss?uid=swg21663762] for more details.
