Title: IBM Importing an SAP IDOC or DXOB with WebSphere Transformation Extender results in a cannot open ttmaker.log error. - United States

Text:
Importer SAP IDOC DXOB ttmaker TECHNOTE (TROUBLESHOOTING)

PROBLEM(ABSTRACT)
 WebSphere® Transformation Extender (WTX) generates a "Cannot open ttmaker.log" error when importing an SAP® IDOC or DXOB. 

CAUSE
The incorrect version of SAP was selected in the WTX SAP IDOC or DXOB Importer, Select Versions screen,

ENVIRONMENT
Windows

RESOLVING THE PROBLEM
Review the contents of the IDOC or DXOB being imported and if each field has a definition for CHARACTER_FIRST, then when you are presented with the "Select the version of SAP that created this data:"in the SAP IDOC or DXOB Importer, Select Versions screen, select either Enterprise or mySAP ERP"




If each field has a definition for BYTE_FIRST, select a version of 4.x, 4.5x or 4.6x
