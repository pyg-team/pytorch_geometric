Title: IBM CTGWA2057E Error when running tipcli.Export command - United States

Text:
TIPL2IMPEXP; TIPL2; CTGWA2057E; error; tipcli; export; Cognos TECHNOTE (TROUBLESHOOTING)

PROBLEM(ABSTRACT)
 Performing an Export using the tipcli script fails with severe errors as those listed below. 

SYMPTOM
SEVERE: CTGWA2057E Plug-in "com.ibm.tivoli.reporting.advanced.cognos. upgrade.exporter.ExportPlugin" failed during migration, performing rollback. 

Check logfile /opt/IBM/tivoli/tipv2/profiles/TIPProfile/logs/tipcli.log for more messages.


DIAGNOSING THE PROBLEM
Check the tipcli log for errors like these: 

WARNING: Plug-in 
'com.ibm.tivoli.reporting.advanced.cognos.upgrade.exporter.ExportPlugin' 
failed during migration, performing rollback. 
Throwable occurred: 
com.ibm.tivoli.nc.migration.plugin.MigrationPluginException: 
com.ibm.tivoli.reporting.advanced.cognos.upgrade.TCRPluginException: 
error while running export 
at 
com.ibm.tivoli.reporting.advanced.cognos.upgrade.tip.TIPPluginAdapter.mi 
grate(TIPPluginAdapter.java:69) 
.... 
Caused by: 
com.ibm.tivoli.reporting.advanced.cognos.upgrade.TCRPluginException: 
error while running export 
at com.ibm.tivoli.reporting.advanced.cognos.upgrade.exporter.CognosDBEx 

RESOLVING THE PROBLEM
Run the tipcli script with this parameter to exclude the TCRExportPlugins: 

$ ./tipcli.sh Export --username tipadmin --password <password> --excludePlugins TCRExportPlugins 


TCR does not support the tipcli import/export functions and its plugin must be excluded during import/export if TCR is deployed in the source and target TIP installs. 

For further details on how to export and import TCR related data, please see the section "TIP Migration" starting on page 13 of this guide:
