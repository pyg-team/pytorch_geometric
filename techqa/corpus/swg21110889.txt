PROBLEM(ABSTRACT)
 Your CICS region fails with the following messages at start up after you apply CICS maintenance:

DFHAP1203I Language Environment is being initialized. 
DFHSR0601 Program interrupt occurred with system task III in control.

A system dump shows that an abend 0C4 occurred in Language Environment (LE) module CEEZLOD. 

SYMPTOM
DFHAP1203I and DFHSR0601

CAUSE
There were two libraries in the RPL concatenation containing LE module CEEPLPKA. 

RESOLVING THE PROBLEM
Remove LE module CEEPLPKA from all but the CEE.SCEERUN library.
