ITCAM FOR APPLICATION DIAGNOSTICS COMPONENTS
 The Managing Server provides detailed ("deep dive") diagnostic information through the Visualization Engine, a component of the Managing Server.

The WebSphere Applications Agent provides monitoring, diagnostics, and server management for IBM WebSphere servers.

The J2EE Agent provides monitoring, diagnostics, and server mgmt for IBM WebSphere CE, JBoss, Apache Tomcat, WebLogic, SAP NetWeaver, Oracle Application Server, J2SE.

The HTTP Servers Agent provides monitoring and server management for Apache, IBM HTTP Server, and Sun Java System Web Server (Sun ONE Web Server) servers.
