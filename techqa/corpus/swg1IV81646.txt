ERROR DESCRIPTION
 *  Problem description and business impact
   TDWC V9.3 with FP1 on AIX, I observed that when you click close
   on a
   'monitor jobs on multiple engines' task the application takes
   you to an
   active task list. This is also evident when you open multiple
   'monitor
   jobs on multiple engines' tasks. This was observed on IE,
   Firefox and
   Chrome.

 *  N/A

 *  The Dynamic Workload Console java code will be fixed in order to
   allow the user to close multiple tasks at the same time.
