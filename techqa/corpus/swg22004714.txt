Title: IBM Security Bulletin: IBM API Connect is affected by an information disclosure vulnerability (CVE-2017-1379). - United States

Text:
 SECURITY BULLETIN

SUMMARY
 IBM API Connect has addressed the following vulnerability. 

VULNERABILITY DETAILS
CVEID: CVE-2017-1379 [http://cve.mitre.org/cgi-bin/cvename.cgi?name=CVE-2017-1379]
DESCRIPTION: IBM API Connect could allow a remote attacker to obtain sensitive information, caused by improper handling of requests to the Developer Portal.
CVSS Base Score: 5.3
CVSS Temporal Score: See https://exchange.xforce.ibmcloud.com/vulnerabilities/127002 [https://exchange.xforce.ibmcloud.com/vulnerabilities/127002] for the current score
CVSS Environmental Score*: Undefined
CVSS Vector: (CVSS:3.0/AV:N/AC:L/PR:N/UI:N/S:U/C:L/I:N/A:N) 

AFFECTED PRODUCTS AND VERSIONS
 

Affected Product Affected Versions IBM API Connect 5.0.0.0-5.0.7.1 
REMEDIATION/FIXES
The recommended solution is to apply either of the following Developer Portal iFixes containing APAR LI79677. 

Product VRMF APAR Remediation / First Fix IBM API Connect 5.0.0.0 - 5.0.6.2 LI79677 V5.0.6.2_iFix3 [http://www.ibm.com/support/fixcentral/swg/quickorder?parent=ibm%7EWebSphere&product=ibm/WebSphere/IBM+API+Connect&release=5.0.6.2&platform=All&function=fixId&fixids=5.0.6.2-iFix3-APIConnect-Portal-20170609-2204,5.0.6.2-iFix3-APIConnect-Portal-20170609-2204.ova&includeSupersedes=0&source=fc] IBM API Connect 5.0.7.0 - 5.0.7.1 LI79677 V5.0.7.1_iFix1 [http://www.ibm.com/support/fixcentral/swg/quickorder?parent=ibm%7EWebSphere&product=ibm/WebSphere/IBM+API+Connect&release=5.0.7.0&platform=All&function=fixId&fixids=*5.0.7.1-iFix1*&includeSupersedes=0&source=fc]
