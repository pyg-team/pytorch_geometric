APAR STATUS
 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  Using SFTP to query for file that does not exist (STAT command),
   DataPower returns error code 4 (Failure) instead of 2 (Not
   Found).
   
   
    
   
   

LOCAL FIX

PROBLEM SUMMARY
 *  Affected is the use of a Multi-Protocol Gateway with a
   SFTP Server Front Side Handler as a proxy to a SFTP backend.
   
   After a query for file that doesn't exist (STAT command)
   DataPower returns error code 4 (Failure) instead of 2 (Not
   Found).
   
   
    
   
   

PROBLEM CONCLUSION
 *  Fix is available in 6.0.0.6 and 6.0.1.2.
   
   For a list of the latest fix packs available, please see:
   http://www-01.ibm.com/support/docview.wss?uid=swg21237631 [http://www-01.ibm.com/support/docview.wss?uid=swg21237631]
