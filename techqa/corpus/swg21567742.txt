The error in Workplace: 

Checkout is not supported for this document


CAUSE
When the document was created, the Supports Versioning option was disabled on the document class


RESOLVING THE PROBLEM
Look at the properties for the document class to see if the Supports Versioning option is enabled. If it is disabled, you can enabled it and all new documents of that document class will be versionable and can be checked out. 

For all existing documents, the versioning_enabled property will need to be modified to allow checkout.
