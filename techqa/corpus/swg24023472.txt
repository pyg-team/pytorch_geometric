ABSTRACT
 This document explains how to download the IBM® Tivoli® Netcool® Performance Manager Version 1.2.1 Tier 2 Technology Pack from Passport Advantage®. 

DOWNLOAD DESCRIPTION
Tivoli Netcool Performance Manager is a bundled product consisting of Tivoli Netcool/Proviso® 4.4.3 and Tivoli Netcool Performance Manager 1.2.1 - Wireless Component, plus additional Technology Packs for each product.





After logging into the IBM Passport Advantage Online [http://www.ibm.com/software/howtobuy/passportadvantage/pao_customers.htm] Web site, click Software download & media access, then click Download finder, then select Find by product description and type in Netcool. Expand the Tivoli Software eAssembly, and select the image that you want to download from that list. All Tivoli Netcool Performance Manager Version 1.2.1 technology pack tier 2 download images are described below. 

More information 
The IBM Tivoli Netcool Performance Manager library of publications are available from this Web site: Tivoli Network Management Information Center [http://publib.boulder.ibm.com/infocenter/tivihelp/v8r1/index.jsp] 

Downloading and assembling the product 

 1. Download all of the parts that you need. The parts are listed by platform in the tabbed sections of this document. 
 2. Unpack each of the parts onto your system. 
 3. Follow the instructions in the product installation guides to install the technology pack.
