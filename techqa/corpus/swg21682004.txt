Text:
PMR 18365077724; 5724C71AOTR TECHNOTE (TROUBLESHOOTING)

PROBLEM(ABSTRACT)
 Application Support Mismatch warning message gets displayed on TEP after support upgrade to Active Directory agent 6.3.1.1 IF1 

SYMPTOM
Application Support Mismatch warning message gets displayed on TEP


CAUSE
 

Following warning message gets displayed on TEP after support upgrade to Active Directory Agent 6.3.1.1 IF1 (6.3.1.1-TIV-ITM_ADO-IF0001)

[/support/docview.wss?uid=swg21682004&amp;aid=1]
