After uninstalling the Application Server Fix Pack 7.0.0.9 it works again. 

SYMPTOM
Any email functionality, for example, the password reset mail functionality, for a Web 2.0 enabled sample store (for example, the Madison store) does not work. 

RESOLVING THE PROBLEM
There are two options to resolve this problem. 

1. Apply WebSphere Commerce Fix Pack 7.0.0.1 if upgrading WebSphere Application Server to Version 7.0.0.9 is required, See link below :

IBM - WebSphere Commerce Version 7.0.0.1 Fix Pack [http://www.ibm.com/support/docview.wss?uid=swg24026649] 

OR

2. Request and apply APAR JR36472 while remaining on WebSphere Commerce V7.0.

Note: The minimum required level of WebSphere Application Server is 7.0.0.3. It is recommended to upgrade to WebSphere Commerce V7.0.0.1 when running WebSphere Commerce with WebSphere Application Server 7.0.0.9.
