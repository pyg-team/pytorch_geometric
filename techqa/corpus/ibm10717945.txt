Downloading the release Sign in to Passport Advantage [http://www.ibm.com/software/howtobuy/passportadvantage/pao_customers.htm]. For the full release, search for part number CJ3DZML for production use and CJ3E0ML for trial use. If you don't need the full release, you can instead search for any of the eImages that make up the full release. The eImage part numbers are given in the following table. If you install IBM MQ Version 9.1 eAssembly images on a UNIX or Linux-based platform, you must use GNU tar (also known as gtar) to unpack any tar images.
