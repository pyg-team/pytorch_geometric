However, recompileLOBT_WC_demo.log is empty


CAUSE
recompileLOBT.xml is required to recompile the LOBTools project.
This does not exist if no Feature Packs were installed on the environment

ENVIRONMENT
WebSphere Commerce v7 with no Feature Packs
This issue has only been seen with CF4 which requires a LOBTools recompile

DIAGNOSING THE PROBLEM
Check the following directory to see if recompileLOBT.xml exists:
CommerceServer70/components/management-center/openlaszlo/xml

RESOLVING THE PROBLEM
Copy the file from an environment where the file exists or contact IBM Support for a copy of this file.
Place the file in the CommerceServer70/components/management-center/openlaszlo/xml directory and rerun the installation
