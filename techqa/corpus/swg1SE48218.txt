APAR STATUS
 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  An error dialog with the message "'Downloding copybooks for
   /directory/file.sqb' has encountered a problem."  The directory
   and file names in the message will point to the file that
   encountered the problem. The message details will conntain:
   
   An internal error occurred during:
   "Downloading copybooks for /directory/file.sqb"
   java.lang.NullPointerException
   
   After the AIX COBOL project is built for the first time the
   problem will happen every time the .sqb file is parsed. The
   .sqb file is parsed when the source file is opened and after
   every change in the editor.
   
   
    
   
   

LOCAL FIX

PROBLEM SUMMARY
 *  ****************************************************************
   * USERS AFFECTED: RD Power users that use AIX COBOL projects.  *
   ****************************************************************
   * PROBLEM DESCRIPTION: Error occurs when editing source in     *
   *                      AIX COBOL projects.                     *
   ****************************************************************
   * RECOMMENDATION:                                              *
   ****************************************************************
   An error dialog with a message starting with 'Downloading
   copybooks for' interrupts editing source in an AIX COBOL
   project.
   
   
    
   
   

PROBLEM CONCLUSION
 *  This problem is fixed in the RD Power 8.0.3 Refresh Pack.
