CONTENT
"SCAPM V7.7.0.1 is a composite offering containing the following products:
- SCAPM 7.7
- SCM AI 1.2 (including APM UI running on MIN, the Lightweight Infrastructure)
- SCAPM for Mobile Applications 7.7.0.1 (running on MIN, the Lightweight Infrastructure)


Install SmartCloud Application Performance Management for Mobile Applications V7.7.0.1 with the Application Performance Management UI running on the MIN lightweight infrastructure. You cannot install SmartCloud Application Performance Management for Mobile Applications V7.7.0.1 or its upgrade, SmartCloud Application Performance Management for Mobile Applications V7.7.0.2 onto a IBM Tivoli Manager (with TEMS and TEPS) deployment.
