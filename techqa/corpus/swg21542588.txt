Title: IBM BAPI ERROR: INVALID_CHARACTER_ERR: An invalid or illegal XML character (SCI93603) - United States

Text:
STERLINGPRI TECHNOTE (TROUBLESHOOTING)

PROBLEM(ABSTRACT)
 BAPI ERROR: INVALID_CHARACTER_ERR: An invalid or illegal XML character (SCI93603) 

SYMPTOM
BAPI ERROR: INVALID_CHARACTER_ERR: An invalid or illegal XML character


RESOLVING THE PROBLEM
add 2 lines in the sap.properties and sap.properties.in files : 



SAP.SapBAPIsylvain.XMLSpecialCharHandling = Yes 



SAP.SapBAPIsylvain.XMLNodesSpecialHandling = Yes where SapBAPIsylvain is then name of the instance of the sap suite adapter you have configured.

Fact


Gentran Integration Suite SAP R3 BAPI Adapter Linux
Release 
When calling 
BAPI_MATERIAL_SAVEDATA function
we get an illegal xml caracter sent back by 
SAP. SAP creates an XMl file based on column names in database tables. this 
function returns an XML element of 2STEP_ 
it is not allowed to have the first 
char as a numbner in an xml tag. By adding a property in the sap.properties 
file
a function renames ALL illegal XML tags to _tagname so that our XML 
parser can then parse the file correctly.
