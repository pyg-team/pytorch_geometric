Title: IBM How can I obtain info about tracking logins and logouts for our users? - United States

Text:
information server; audit; logon TECHNOTE (FAQ)

QUESTION
 When I look at the user ids in the audit log, ISauditLog0.log, I see "InformationServerSystemUser". How can we see individual users? 

CAUSE
The System User logins that occur will vary depending on the products you have installed. Below is a list of predefined System Users:
InformationServerSystemUser
InformationAnalyzerSystemUser
DataStageSystemUser
BusinessGlossarySystemUser
LoggingAgentSystemUser
WebServiceSystemUser
CASSystemUser
ORAsyncSystemUser



ANSWER
These predefined system users login to perform various tasks. You can suppress LOGIN and LOGOUT audit events from these System User logins if you wish by configuring: 

audit.event.LOGIN = INFO 
audit.event.LOGOUT = INFO

This can be set by using the iisAdmin command.

On Unix/Linux:
cd IS_install_path/ASBServer/bin
./iisAdmin.sh -set -key com.ibm.iis.isf.audit.event.LOGIN -value INFO
./iisAdmin.sh -set -key com.ibm.iis.isf.audit.event.LOGOUT -value INFO

On Windows:
cd IS_install_path\ASBServer\bin
cd IS_install_path\ASBServer\bin
iisAdmin.bat -set -key com.ibm.iis.isf.audit.event.LOGIN -value INFO
iisAdmin.bat -set -key com.ibm.iis.isf.audit.event.LOGOUT -value INFO

To see a listing of the current settings execute:
On Unix/Linux: ./iisAdmin.sh -display

or on Windows: iisAdmin.bat -display
