ABSTRACT
 IBM WebSphere Application Server fixes are required for several V7.5.1 Fix Pack 2 products.

Installation instructions


Note: If you have Internet access and you are using the IBM Installation Manager user interface, then these updates are found and downloaded for you as part of the IBM Business Monitor, WebSphere Enterprise Service Bus, or IBM Business Process Manager V7.5.1 Fix Pack 2 update. 

1. Download and install WebSphere Application Server Network Deployment V7.0.0 Fix Pack 31 from the "Download package" table in this document. Use IBM Installation Manager to install the fix pack.

WebSphere Application Server Network Deployment V7.0.0 Fix Pack 31 is required for V7.5.1 Fix Pack 2 for the following products: 

 * IBM Business Monitor 
 * IBM Business Process Manager Standard 
 * IBM Business Process Manager Express 
 * IBM Business Process Manager Advanced 
 * IBM Business Process Manager Advanced â€“ Process Server 
 * WebSphere Enterprise Service Bus


For WebSphere Application Server Network Deployment V7.0.0 Fix Pack 31, use one of the following methods to install the fix pack:  * All supported operating systems and architectures
   The following images includes the WebSphere Application Server Network Deployment offerings for all supported operating systems and architectures:  * was.7.0.0.31.fp.all_Part1of2.zip 
    * was.7.0.0.31.fp.all_Part2of2.zip
      
      Note: You must extract both images to the same location. 
   
   
 * Specific operating systems and architectures
   The following files contain sliced repositories that you can use to install WebSphere Application Server Network Deployment V7.0.0 Fix Pack 31 for specific operating systems and architectures. If you use this method, you do not need to download the repository files for all supported operating systems and architectures. To upgrade WebSphere Application Server Network Deployment on a specific operating system and architecture, download only the file that matches your operating system and architecture:  * was.nd.7.0.0.31.fp.aix.zip 
    * was.nd.7.0.0.31.fp.win.zip 
    * was.nd.7.0.0.31.fp.linux.x86.zip 
    * was.nd.7.0.0.31.fp.linux.s390.zip 
    * was.nd.7.0.0.31.fp.linux.ppc.zip 
    * was.nd.7.0.0.31.fp.solaris.sparc.zip 
    * was.nd.7.0.0.31.fp.solaris.x86.zip
   
   

2. Download and then install the following fix packs for the WebSphere Application Server Feature Packs:  * WebSphere Application Server V7.0 Feature Pack for XML V1.0.0 Fix Pack 15 [ftp://public.dhe.ibm.com/software/websphere/downloads/xml/service/1.0/fixpacks/xml.1.0.0.15.fp.zip]
   
   
 * WebSphere Application Server V7.0 Feature Pack for Service Component Architecture (SCA) V1.0.1 Fix Pack 17 [ftp://public.dhe.ibm.com/software/websphere/downloads/sca/service/1.0/fixpacks/sca.1.0.1.17.fp.zip]
   
   
 * WebSphere Application Server V7.0 Feature Pack for XML V1.0.0 import repositories [ftp://public.dhe.ibm.com/software/websphere/downloads/xml/import/xml.1.0.import.zip]
   
   
 * WebSphere Application Server V7.0 Feature Pack for Service Component Architecture (SCA) V1.0.1 import repositories [ftp://public.dhe.ibm.com/software/websphere/downloads/sca/import/sca.1.0.import.zip]
   
   
   
   Note: The WebSphere Application Server Version 7.0 Feature Pack for Service Component Architecture (SCA) does not apply to IBM Business Monitor, to IBM Business Process Manager Express, or to IBM Business Process Manager Standard.
