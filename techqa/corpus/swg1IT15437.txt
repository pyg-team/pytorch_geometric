APAR STATUS
 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  When logged on to WebGUI of the Datapower appliance using
   Firefox browser, it is possible to modify the webgui html code
   to allow renaming of files on the appliance to include invalid
   characters.
   
   
    
   
   

LOCAL FIX

PROBLEM SUMMARY
 *  Affected is the misuse of WebGUI.
   
   When logged on to WebGUI of the appliance using
   Firefox browser, it is possible to modify the WebGUI html code
   to allow renaming of files on the appliance to include invalid
   characters.
   
   
    
   
   

PROBLEM CONCLUSION
 *  Fix is available in 7.2.0.9, 7.5.0.3 and 7.5.1.2
   
   For a list of the latest fix packs available, please see:
   http://www-01.ibm.com/support/docview.wss?uid=swg21237631 [http://www-01.ibm.com/support/docview.wss?uid=swg21237631]
