Title: IBM Steps to add an authorization code to the license manager (LM) Running on Linux Solaris or AIX - United States

Text:
activate authorization code license manager Linux Solaris AIX TECHNOTE (FAQ)

QUESTION
 How do you manually activate a concurrent authorization code on the operating systems of Linux, Solaris, or AIX? 

ANSWER
Here are the steps to add an authorization code to the license manager (LM) running on Linux, Solaris, or AIX:

1) Log in as root
2) Change the directory to where you have installed the license manager. Defaults are listed below:

[License Manager Default installation locations for the license manager running on Linux, Solaris, or AIX]
[Version] [Location]
9.5.0 and above /opt/IBM/SPSS/Concurrent Licensing Tools/9.5.0/License Manager
8.3.0 /opt/IBM/SPSS/Concurrent Licensing Tools/9.0/License Manager

3) Activate the Authorization Code(s) which will update the license file (lservrc) with the new license(s) code(s) -> ./licenseactivator <authcode>

4) Stop the LM --> ./lsrvdown localhost

5) Start the LM ---> ./lserv <-switches>
