Title: IBM IC85268: CERTIFICATE DETAILS ARE NOT DISPLAYED WHEN SUBJECTKEYID IS EMPTY - United States

Text:
 SUBSCRIBE
You can track all active APARs for this component.



APAR STATUS
 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  Certificates with an empty Subject Key Identifier
   (SubjectKeyID) may cause no information to be displayed,
   after clicking the certificate details button from the WEBGUI.
   The certificate details page will be blank.
   
   Likewise, if using the dp:get-cert-details() function, the
   results will yield an error.
   
   
   To determine if the certificate has an invalid SubjectKeyID,
   view the certificate from your workstation to see:
   
   Subject Key Identifier:    Information Not available
   
   
    
   
   

LOCAL FIX

PROBLEM SUMMARY
 *  Affected are customers using the dp:get-cert-details() extension
   function or the certificate details button from the WebGUI  for
   a certificate containing an s an empty SubjectKeyId.
   
   
    
   
   

PROBLEM CONCLUSION
 *  The fix is available in 3.8.2.15, 4.0.1.13, 4.0.2.9 and 5.0.0.3
