Title: IBM SRVE0260E error is seen when trying to login to the MS Visualization Engine. - United States

Text:
SRVE0260E SRVE0200E itcamdata LandingServlet VE login MS TECHNOTE (TROUBLESHOOTING)

THIS DOCUMENT APPLIES ONLY TO THE FOLLOWING LANGUAGE VERSION(S):
 US English 

PROBLEM(ABSTRACT)
 SRVE0260E error is seen when trying to login to the MS Visualization Engine. 

SYMPTOM
This is the full error stack trace seen when trying to login. 

[/support/docview.wss?uid=swg21608029&aid=1] [/support/docview.wss?uid=swg21608029&aid=1]


CAUSE
This is caused because the 'itcamdata' application is not started in your WAS admin console.


ENVIRONMENT
ITCAM for AD 7.1



RESOLVING THE PROBLEM
To resolve the problem, please follow these steps. 

1. Stop your Managing Server using the am-stop.sh script.. 

2. Login to your WAS Admin console and start the itcamdata application. If its already running, stop and restart it. (See screenshot below.) 

3. Wait few seconds and then start the Managing Server using the am-start.sh script.
