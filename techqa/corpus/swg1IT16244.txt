APAR STATUS
 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  Datapower service (i.e MPGW, WSP, etc) is configured to connect
   to backend server through an HTTP Proxy server (as configured
   in the Proxy Policy of the user agent) using the HTTP CONNECT
   method.
   However, if the HTTP CONNECT request to the proxy server,
   receives a 5xx response, Datapower might issue the following SSL
   error:
   20160711T163325.760Z [0x8120002f][ssl][error]
   ssl-client(PMYClientProfile): tid(11573565)[10.x.x.x]
   gtid(11573565): SSL library error: error:1408F10B:SSL
   routines:SSL3_GET_RECORD:wrong version number
   
   
    
   
   

LOCAL FIX

PROBLEM SUMMARY
 *  With the fix the device will now log the actual HTTP response
   code received if there is an error.
   
   
    
   
   

PROBLEM CONCLUSION
 *  The fix is available in 7.0.0.16, 7.1.0.13, 7.2.0.10, 7.5.0.4,
   7.5.1.3 and 7.5.2.1
