ABSTRACT
 This patch provides an update to the for IBM Tivoli Monitoring (ITM) portal server component. 

DOWNLOAD DESCRIPTION
This fix upgrades the embedded WebSphere Application Server (eWAS) which is shipped as part of the IBM Tivoli Monitoring portal server, to version 8.0.00.12 plus additional Interim Fixes referred to as Interim Fix Block 4. The Interim Fixes included as part of Interim Fix Block 4 include:



PI76781: SHIP JAVA 6/26 SR8 FP41 FOR WAS V8.0.0.X
PI73367:Potential cross-site scripting vulnerability in WebSphere Application Server Admin Console
PI73519: Potential Denial of Service in WebSphere Application Server with SOAP connectors

PREREQUISITES
The prerequisite for this fix is IBM Tivoli Monitoring, version 6.3.0 Fix Pack 2 (6.3.0-TIV-ITM-FP0002) or later.
