ERROR DESCRIPTION
 *  After applying FP 4 for Cognos 8.3 client is getting the
   following error in their portal page:
   
   " CCL-PLG-0001 Failed to load the pluggable shared library "
   
   This error only shows up when client have a portal page with
   Shortcut reports in "Cognos navigator" side and next to it in
   "Cognos Viewer" client have a default reports set.
   
   The default report shows up properly when click the tab for the
   first time after log in. But when client click on any other
   shortcut report to view in Cognos viewer, the error shows up in
   a yellow bar.
   
   Tested the issue with Cognos 8.3 RTM and client tested with
   Cognos 8.3 SP3, issue is not re-producible. Error only
   re-producible in Cognos 8.3 FP4.
   
   
    
   
   

LOCAL FIX

PROBLEM CONCLUSION
 *  UV was not retrieving the correct CM object. We were querying CM
   using "navSelectionId" instead of "display_id".
