CAUSE
Anti-virus software preventing some of the installation scripts from performing their intended functions.

ENVIRONMENT
Windows

DIAGNOSING THE PROBLEM
Error messages displayed such as 'Product com.ibm.websphere.dtx.product.designstudio could not
be found' or 'No application id has been found' when trying to start the Design Studio.

RESOLVING THE PROBLEM
Disable anti-virus software during the product install.
