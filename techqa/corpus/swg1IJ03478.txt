APAR STATUS
 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  User accidentally dragged DASH Menu Item and it transforms into
   a floating window that cannot be closed.
   
   Steps to reproduce:
   1. Login to DASH
   2. Click any of the menu
   3. Click a menu item and drag from left to right
   4. Notice that the floating window cannot be closed.
   
   This issue is observed in all DASH versions starting from DASH
   3.1.0.x to 3.1.3.x
   
   
    
   
   

LOCAL FIX
 *  Workaround:  Refresh the current page by F5, then the floating
   window will disappear.
   
   
    
   
   

PROBLEM SUMMARY
 *  Reported issue is not reproducible with DASH 3.1.3 CP6
   
   
    
   
   

PROBLEM CONCLUSION
