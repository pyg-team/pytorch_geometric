ERROR DESCRIPTION
 *  When opening a simulation the page keep loading for ever. In
   the server side you can see the stack trace below. The reason
   is that in the database table VSREPORT the latest report has
   its ERRORSCENARIOCOUNT to null.
   In the SimulationHelper.getSuccessRate() we have the following
   code:
   
   if (IlrReportStatusKind.STARTED_LITERAL.equals(status)
                   ||
   IlrReportStatusKind.STARTING_LITERAL.equals(status)
                   ||
   IlrReportStatusKind.FAILED_LITERAL.equals(status)) {
         return Double.valueOf(0d);
   }
   Integer nbTests = report.getEstimatedScenarioCount();
   if (nbTests == null || nbTests == 0) {
        return Double.valueOf(0d);
   }
   int nbErrors = report.getErrorScenarioCount();
   return Double.valueOf((double)(nbTests - nbErrors)/nbTests);
   
   The getErrorScenarioCount() return an Integer, with java
   unboxing we got the NPE getErrorScenarioCount().intValue()
