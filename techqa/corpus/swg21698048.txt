Title: IBM Measuring last access time for pages in DASH - United States

Text:
DASHL2; DASHL2CONF; metrics ; page; access; measure; analytics; last time TECHNOTE (FAQ)

QUESTION
 How can I generate information of the last time each page was accessed in Dashboard Application Service Hub (DASH) component of Jazz for Service Management (JazzSM) application? 

ANSWER
This is available in DASH 3.1.1 and here are the steps:
1) Create a page.

2) Add a table widget to the page.

3) Edit the table widget and search for analytics dataset.. Select Dashboard Access Console Analytics dataset...

[/support/docview.wss?uid=swg21698048&amp;aid=1]
4) Select OK and save the page..

The table will show last access time for each page, listing the page id.
