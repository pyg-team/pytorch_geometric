APAR STATUS
 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  An XMLAccess content-node locate without using "uniquename"
   attribute for "wps.content.root" fails.
   
   The following valid XMLAccess file fails to execute:
   
   <request xmlns:xsi="http://www.w3.org/2001/XMLSchema-instance
   xsi:noNamespaceSchemaLocation="PortalConfig_6.1
   .0.2.xsd"
   type="export">
   <portal action="locate">
   <content-node action="export"
   objectid="6_000000000000000000000000A0"/>
   </portal>
   </request>
   
   
    
   
   

LOCAL FIX
 *  Add the uniquename attribute explicitly.
