ICN Client Server Browser debugging TECHNOTE (FAQ)

QUESTION
 How do you generate logs for IBM Content Navigator (ICN) client and ICN server? 

ANSWER
To gather logs from both ICN client and server, do the following:
1. IBM Content Navigator server logging: 
Go to Settings tab in the administration tool, in the Logging sub-tab, set the level of logging information to write to the server log file.
Logging is written to the IBM WebSphereÂ® Application Server SystemOut.log file.

2. IBM Content Navigator client logging:
On the client PC enable Java Script logging by accessing ICN with the following URL: 

http://<server:port>/navigator/?debug=true&useConsole=false&logLevel=4 

Ex: http://cm1:9080/navigator/?debug=true&useConsole=false&logLevel=4 

or specify the desktop that is not the default desktop: 

http://<server>:<port>/navigator/?debug=true&useConsole=false&logLevel=4
/?desktop=<desktop name> 

Ex: http://cm1:9080/navigator/?debug=true&useConsole=false&logLevel=4/?deskt
op=Sample 

A console window will appear. 
Duplicate the problem, then copy and paste the console log into a text file.

3. Browser Debugging:
Hit F12 on your browser. This will open a console at the bottom of the browser. 
Reproduce the issue. 
Select the console tab and right click on the content to select all.
Copy the output from the console log to a file.
