Title: IBM Security Bulletin: A security vulnerability has been identified in IBM Websphere Message Broker, IBM Integration Bus, IBM SPSS Data Access Pack, and IBM SPSS Modeler shipped with IBM Predictive Maintenance and Quality (CVE-2014-3566) - United States

Text:
 SECURITY BULLETIN

SUMMARY
 IBM Websphere Message Broker, IBM Integration Bus, IBM SPSS Data Access Pack, and IBM SPSS Modeler are shipped as components of IBM Predictive Maintenance and Quality. Information about a security vulnerability affecting IBM Websphere Message Broker, IBM Integration Bus, IBM SPSS Data Access Pack and IBM SPSS Modeler have been published in security bulletins. 

VULNERABILITY DETAILS

Please consult the security bulletin: Vulnerability in SSLv3 affects IBM SPSS Data Access Pack (CVE-2014-3566) [http://www-01.ibm.com/support/docview.wss?uid=swg21688151] for vulnerability details.

Please consult the security bulletin: Vulnerability in SSLv3 affects IBM SPSS Modeler (CVE-2014-3566) [http://www-01.ibm.com/support/docview.wss?uid=swg21688163] for vulnerability details.

Please consult the security bulletin: Vulnerability in IBM Websphere Message Broker and IBM Integration Bus are affected by SSLv3 Vulnerability (CVE-2014-3566) [http://www-01.ibm.com/support/docview.wss?uid=swg21687678] for vulnerability details.
