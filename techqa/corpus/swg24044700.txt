INSTALLATION INSTRUCTIONS
 

====================================================================

Installation, migration, upgrade and configuration information:
---------------------------------------------------------------

* Hardware and software requirements:

Refer to the Netcool/Impact 6.1.1 Release Notes for a full 
description of all the hardware and software requirements.


* Dependencies:

This Interim Fix must be installed on each server(s) that is
running the Netcool/Impact server. 


This Interim Fix requires Netcool/Impact 6.1.1 FP0005
to be already installed.


* Special considerations:

1) The Netcool/Impact and GUI server(s) need to be stopped, prior to
installing this Interim Fix. 

* Extracting this Interim Fix:
Windows and Non-Windows

1. Copy the file 6.1.1.5-TIV-NCI-IF0002.zip to a 
temporary location on your Netcool/Impact and GUI server.
2. Unzip the file on each server that you need to install on.




* Installing this Interim Fix:

[Interim Fix] Refers to the directory where this Interim Fix
resides (after you have extracted it from the zip file).

for example: /tmp/6.1.1.5-TIV-NCI-IF0002

On the Netcool/Impact server system:

Navigate to the [Interim Fix] directory where the files
were extracted.

Install the Interim Fix with the same user that was used to 
install Netcool/Impact 6.1.1 Base release.

All Netcool/Impact GUI servers
===================================================
1.) Stop the Netcool/Impact and GUI server(s)
2.) Open a command prompt and change to the unzip directory 
3.) Set the NC_HOME environment variable to locate the Netcool/Impact
6.1.1 installation:
for example: /opt/IBM/tivoli

4.) Backup and move all commons-fileupload*.jar under following directory to other place
$NC_HOME/impact/lib3p
$NC_HOME/impact/uilib
$NC_HOME/tipv2/profiles/ImpactProfile/installedApps/ImpactCell/NCI.ear 
$NC_HOME/tipv2/systemApps/isclite.ear
$NC_HOME/tipv2/profiles/TIPProfile/installedApps/TIPCell/isc.ear 
$NC_HOME/tipv2/profiles/TIPProfile/installedApps/TIPCell/authnsvc_ctges.ear

5.) Copy extracted commons-fileupload-1.3.3.jar file to same place as Step 4.)

6.) Start the Netcool/Impact and GUI server(s)
