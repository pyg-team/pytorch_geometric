Title: IBM TM1Web 10.1.1  Making Tm1web Grid line borders more visible - United States

Text:
 TECHNOTE (TROUBLESHOOTING)

THIS DOCUMENT APPLIES ONLY TO THE FOLLOWING LANGUAGE VERSION(S):
 English 

PROBLEM(ABSTRACT)
 TM1web 10.1.1 grid lines are not as visible as in the earlier release 

SYMPTOM
Grid lines in TM1web 10.1.1 are not visible 


CAUSE
Border line code within the TM1CubeViewer.css file (specifically .DataCell and .CVHearderCell blocks) are not provisioned

ENVIRONMENT
TM1Web 10.1.1

RESOLVING THE PROBLEM
1. Locate the TM1CubeViewer.css file in the tm1_64\tm1web\css folder


 2. The following areas will be modified to include new lines of code:

.CVHeaderCell to include border:1px solid #CAD9F0;
.DataCell to include border:1px solid #CAD9F0;

Configuration including new lines of code will look as follows:
.DataCell
{
font-size: 8pt;
font-family: Tahoma, 'Trebuchet MS', Arial, Verdana, Helvetica,
sans-serif;
text-align: right;
padding-right: 3px;
padding-left: 3px;
margin:0px;
border:1px solid #CAD9F0 //This line is to be added
}
.
.CVHeaderCell /*** bad0f1 ***/
{
padding-right: 3px;
padding-left: 3px;
/*font-weight: bold;*/ /* CHANGE: Removing Text bold style */
font-size: 8pt;
font-family: Tahoma, 'Trebuchet MS', Arial, Verdana, Helvetica,
sans-serif;
color: #333333;
/* These are the header cells of the each column of data */
background-color: #E7E5E5; 
border-right: 1px solid #E2E2E2;
border-left-width: 0px;
border-bottom: 1px solid #E2E2E2;
border-top-width: 0px;
margin:0px;
border:1px solid #CAD9F0; // This line is to be added
}

3. Save the file
4. Login to TM1web 
5. Select any view 
6. Now the grid lines will more more visible as in the earlier release
