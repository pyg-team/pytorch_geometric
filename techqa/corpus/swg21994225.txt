Title: IBM Security Bulletin: Multiple vulnerabilities in IBM® Java SDK affect WebSphere Application Server October 2016 CPU shipped with WebSphere Commerce (CVE-2016-5573, CVE-2016-5597) - United States

Text:
 SECURITY BULLETIN

SUMMARY
 IBM® Java™ SDK included with WebSphere Application Server is shipped as a component of WebSphere Commerce. Information about multiple security vulnerabilities affecting IBM Java SDK has been published in a security bulletin. 

VULNERABILITY DETAILS
See the security bulletin Multiple vulnerabilities in IBM® Java SDK affects WebSphere Application Server October 2016 CPU (CVE-2016-5573, CVE-2016-5597) [http://www-01.ibm.com/support/docview.wss?uid=swg21993440] vulnerability details and information about fixes.

AFFECTED PRODUCTS AND VERSIONS
 

Principal Product and Version(s) Affected Supporting Product and Version WebSphere Commerce V8.0 IBM WebSphere Application Server  * Version 8.5.5 

WebSphere Commerce V7.0 IBM WebSphere Application Server  * Version 7
