 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  When configuring the Sybase agent, the agent cannot recognize
   the Sybase server name correctly.
   
   Detailed Recreation Procedure:
   1. Edit <Sybase_ASE_Home>/install/RUN_<SID>, add a blank
   character immediately after "-s ".
   2. Re-start the Sybase db server with command "startserver -f
   RUN_<SID>"
   3. Configure the monitoring agent, list active server.
   
   Related Files and Output: N/A
