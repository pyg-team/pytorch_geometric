ERROR DESCRIPTION
 *  hi
   When login on to Web Editor, The system hangs on
   the logon screen.
   Then if you go to the LUM server and check the
   BASIC License Tool, all of the ST Editor
   Licenses are used by the same user; the user
   that was trying to log onto the Web Client. To
   see the detailed list of Editor Users, go to
   Selected Open as Detail Concurrent User Tab.
   To fix this, we have to do the following:
   1. Restart the LUM services
   2. On the Web Client, open IE and go to
   Tools Internet Options Delete Cookies &
   Clear Files.
   3. Reset IIS on the Web Server
   Thnaks
   Baruch
   ** IMPORTANT NOTE: Please refer to Bugs:
   BID-74156 Sometimes Web Editor cannot find WED
   licenses and takes all available EDR or NAV
   licenses on LUM serve - and  BID-75963  1 Web
   Editor and 1 Navigator license taken when the
   connection to database fails
   .
