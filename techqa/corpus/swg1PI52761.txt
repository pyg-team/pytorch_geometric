ERROR DESCRIPTION
 *  In RDz v9.1.1, the user is starting up the Integrated Debugger
   using SVC 252. He receives a message:
   "AQECM116E - Current SVC 252 is AQESVC03".
   
   Since this is an informational message, the message id should
   not end with E, as this causes extra work
   for the user to respond to  the message.
