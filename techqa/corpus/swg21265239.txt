ABSTRACT
 The IBM Rational License Server is an application that serves floating license keys to the IBM Rational products running on remote client machines or on the same machine as the license server. The license server is built with FLEXlm, a third-party component developed by Flexera. This technote outlines how Rational delivers the license server and the platforms it supports.

Note for Rational License Key Server v8.1.x: 

 * Rational License Key Server v 8.1.x is compatible with all shipping version Rational Products that use Flex based licensing.
   
   
 * You cannot install Rational License Key Server version 8.1.x on the same server that has Rational License Key Server 7.1.x or earlier installed.

How do I download the Rational License Server? 

The latest version of the Rational License Server (on Windows, Linux, and UNIX platforms) is available as one of the components you can download from your Passport Advantage [https://www-112.ibm.com/software/howtobuy/softwareandservices]account when you purchase any of the products listed above. 

For example. if you bought IBM Rational Application Developer, you will see the Rational License Server as one of the items you can download from the Rational Application Developer "eAssembly". The license server is also available on CD when you order physical media. 

For Rational Build Forge, the license server is included in the Build Forge download itself. 

If you are unable to download the Rational License Key Server 8.1.x through your Passport Advantage [https://www-112.ibm.com/software/howtobuy/softwareandservices]account, you can also download it from Fix Central. [http://www-01.ibm.com/support/docview.wss?uid=swg27035665] 

Guidance on how to obtain new and renewal licenses from the License Key Center can be found in Technote 21250433 [http://www-01.ibm.com/support/docview.wss?uid=swg21250433] or Technote 21250403 [http://www-01.ibm.com/support/docview.wss?uid=swg21250403]. 

If you are using tokens, contact your IBM sales representative for guidance on the number of tokens required.

Notes and known limitations


 1. Rational License Server is a 32-bit application and runs in 32 bit mode on 64 bit operating systems.
    
    
 2. Earlier versions of FLEXlm were implemented on AIX and HP-UX in order to maintain support for older versions of the operating system
    
    
 3. lmutil fails on Windows 2008 and Windows 7 OS. 
    
    WORKAROUND: Edit the hosts file in the system32 folder (C:\Windows\System32\Drivers\etc\hosts) to contain the IP address and the machine name (for example, 9.X.X.X hostname)
    
    
 4. Dongles are not supported on Windows 2008 and Windows 7.
