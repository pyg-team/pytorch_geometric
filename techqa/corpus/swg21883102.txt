Title: IBM Security Bulletin:  Multiple security vulnerabilities in IBM SmartCloud Cost Management shipped with IBM Cloud Orchestrator Enterprise and IBM SmartCloud Orchestrator Enterprise - United States

Text:
cloud orchestrator ; security; CVE-2015-0138; CVE-2015-2808; CVE-2015-1927; CVE-2015-1885; CVE-2015-4000; CVE-2015-4938; CVE-2015-1932; CVE-2015-0138; CVE-2015-2808; CVE-2015-1920; CVE-2015-7450; CVE-2015-2017 SECURITY BULLETIN

SUMMARY
 IBM SmartCloud Cost Management is shipped as a component of IBM Cloud Orchestrator Enterprise and IBM SmartCloud Orchestrator Enterprise. Information about security vulnerabilities affecting IBM SmartCloud Cost Management has been published in a security bulletin. 

VULNERABILITY DETAILS
Consult the security bulletins for IBM SmartCloud Cost Management for vulnerability details and information about fixes. 

 * Security Bulletin: HTTP response splitting has been identified in IBM WebSphere Application Server Liberty Profile shipped with SmartCloud Cost Management and Tivoli Usage Accounting Manager (CVE-2015-2017) [http://www-01.ibm.com/support/docview.wss?uid=swg2C1000121]

 * Security Bulletin: A security vulnerability has been identified in IBM WebSphere Application Server shipped with SmartCloud Cost Management and Tivoli Usage Accounting Manager (CVE-2015-7450) [http://www-01.ibm.com/support/docview.wss?uid=swg2C1000003]
   
 * Security Bulletin: A security vulnerability has been found in IBM WebSphere Application Server 8.5.5.6 shipped with Tivoli Usage and Accounting Manager/SmartCloud Cost Management (CVE-2015-1927) [http://www.ibm.com/support/docview.wss?uid=swg21964651]
   
   
 * Security Bulletin: A security vulnerability has been identified in WebSphere Application Server shipped with Tivoli Usage and Accounting Manager/SmartCloud Cost Management. (CVE-2015-1932) [http://www.ibm.com/support/docview.wss?uid=swg21965064]
   
   
 * Security Bulletin: A security vulnerability has been found in IBM WebSphere Application Server 8.5.5.6 shipped with Tivoli Usage and Accounting Manager/SmartCloud Cost Management (CVE-2015-1885) [http://www.ibm.com/support/docview.wss?uid=swg21964504]
   
   
 * Security Bulletin: A security vulnerability has been identified in WebSphere Application Server shipped with Tivoli Usage and Accounting Manager/SmartCloud Cost Management (CVE-2015-4000) [http://www.ibm.com/support/docview.wss?uid=swg21964499]
   
   
 * Security Bulletin: A security vulnerability has been identified in WebSphere Application Server shipped with Tivoli Usage and Accounting Manager/SmartCloud Cost Management. (CVE-2015-4938) [http://www.ibm.com/support/docview.wss?uid=swg21964864]
   
   
 * Security Bulletin: Security vulnerabilities have been identified in IBM® DB2® shipped with SmartCloud Cost Management (SCCM/TUAM) (CVE-2013-6747, CVE-2014-0963) [http://www.ibm.com/support/docview.wss?uid=swg21675921]
   
   
 * Security Bulletin: Tivoli Usage and Accounting Manager / SmartCloud Cost Management (CVE-2015-1920) [http://www.ibm.com/support/docview.wss?uid=swg21957821] 
   
   
 * Security Bulletin: Vulnerability in RC4 stream cipher affects Tivoli usage and Accounting Manager / SmartCloud Cost Management (CVE-2015-2808, CVE-2015-0138 ) [http://www.ibm.com/support/docview.wss?uid=swg21883107]
