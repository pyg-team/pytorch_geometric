APAR STATUS
 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  The IBM MQ installation contains a Java Runtime Environment
   (JRE), the version of which is not the latest available.
   
   
    
   
   

LOCAL FIX

PROBLEM SUMMARY
 *  ****************************************************************
   USERS AFFECTED:
   The JRE which is embedded into the IBM MQ installation is used
   to run the following Java components of the product:
   
     IBM MQ Managed File Transfer
     IBM MQ Explorer
     IBM MQ Extended Reach (MQXR) service
     IBM MQ Advanced Message Queueing Protocol (AMQP)
   
   
   Platforms affected:
   AIX, Linux on Power, Linux on S390, Linux on x86-64, Linux on
   zSeries, Solaris SPARC, Solaris x86-64, Windows
   
   ****************************************************************
   PROBLEM DESCRIPTION:
   This APAR updates the Java Runtime Environment (JRE) supplied
   with IBM MQ 9.0 LTS.
   
   
    
   
   

PROBLEM CONCLUSION
 *  This APAR updates the JRE embedded into the IBM MQ 9.0 LTS
   release to the following version:
   
     8.0.5.10
   
   on the following platforms:
   
     AIX
     Linux (x86-64)
     Linux (PPC64LE)
     zLinux (s390x)
     Solaris (SPARC)
     Solaris (x86-64)
     Windows (64-bit)
   
   The JRE for HP-UX has NOT been updated under this APAR.
