APAR STATUS
 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  If the user removes the external IBM Installation Manager
   Repository and does not remove the corresponding URL from the
   vsys.im plugin, the PureSystems Manager service60 will crash
   and the system becomes unusable.
   
   The event generated is:
   CWZIP9604W: Service60 failed. The high availability component
   will attempt to recover the service.
   
   
    
   
   

LOCAL FIX

PROBLEM SUMMARY
 *  The code has been updated to check for the proxy host and
   check that the external repository is available before
   proceeding with Installation Manager operations.
   
   
    
   
   

PROBLEM CONCLUSION
 *  A code fix is being provided in PureApplication System
   version 2.1.2.1 in the Foundation Pattern Type version
   2.1.5.2 and PureApplication System version 2.2.0.0
   in the Foundation Pattern Type version 2.1.6.0.
