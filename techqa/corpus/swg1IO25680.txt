APAR STATUS
 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  Edit button is missing in the properties info pane after
   upgrading to IBM Content Navigator V2.0.3 Fix Pack 8.
   This issue applies to IBM Content  Navigator V2.0.3 Fix Pack 8
   and Content Manager OnDemand environments.
   
   
    
   
   

LOCAL FIX

PROBLEM SUMMARY
 *  When working with cross-repository search results, the selected
   document cannot be edited in the Document Info pane.
   
   
    
   
   

PROBLEM CONCLUSION
 *  This has been addressed in IBM Content Navigator V2.0.3.8
   LA014, V3.0.1 LA004, and V3.0.2 LA001.
