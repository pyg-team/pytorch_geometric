Title: IBM Documentation corrections for the Planning and Installing IBM Content Manager for z/OS Version 8.4.3 Guide - United States

Text:
 PRODUCT README

ABSTRACT
 This technical support document contains significant corrections for the Planning and Installing IBM Content Manager for z/OS 8.4.3 and later documentation that could affect product installation or configuration. 

CONTENT
Documentation corrections


ICMACLPrivExit [http://pic.dhe.ibm.com/infocenter/cmgmt/v8r4m0/topic/com.ibm.installingcmzos.doc/dcmiz049.htm]
You can use the ICMACLPrivExit exit routine to determine whether the user has the authority to perform the requested function.
Should be read as:
You can use the ICMACLPrivExit exit routine to determine if a user has the authorization to perform functions on specific items.

ICMGenPrivExit [http://pic.dhe.ibm.com/infocenter/cmgmt/v8r4m0/topic/com.ibm.installingcmzos.doc/dcmiz050.htm]
You can use the ICMGenPrivExit exit routine to determine if a user has the authorization to perform functions on specific items.
Should be read as:
You can use the ICMGenPrivExit exit routine to determine whether the user has the authority to perform the requested function.
