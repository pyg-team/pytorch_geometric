Title: IBM Security Bulletin: Vulnerability in RC4 stream cipher affects WebSphere Application Server shpped with IBM Security Identity Manager (CVE-2015-2808) - United States

Text:
PSIRT SECURITY BULLETIN

SUMMARY
 WebSphere Application Server is shipped as a component of IBM Security Identity Manager. Information about the RC4 "Bar Mitzvah" Attack for SSL/TLS affecting WebSphere Application Server has been published in a security bulletin. 

VULNERABILITY DETAILS

CVEID: CVE-2015-2808 [http://cve.mitre.org/cgi-bin/cvename.cgi?name=CVE-2015-2808]

DESCRIPTION: Please consult the security bulletin Vulnerability in RC4 stream cipher affects WebSphere Application Server [http://www-01.ibm.com/support/docview.wss?uid=swg21701503] for vulnerability details and information about fixes.

AFFECTED PRODUCTS AND VERSIONS
 

Principal Product and Version(s) Affected Supporting Product and Version IBM Security Identity Manager version 6.0 WebSphere Application Server version 7.0 IBM Tivoli Identity Manager version 5.1 WebSphere Application Server versions 6.1 and 7.0 GET NOTIFIED ABOUT FUTURE SECURITY BULLETINS
