Title: IBM Security Bulletin: Password disclosure via trace log vulnerability in IBM MQ Managed File Transfer (CVE-2017-1795) - United States

Text:
 SECURITY BULLETIN

SUMMARY
 Trace files generated by IBM MQ Managed File Transfer commands display passwords in plain text. 

VULNERABILITY DETAILS
CVEID:CVE-2017-1795 [http://cve.mitre.org/cgi-bin/cvename.cgi?name=CVE-2017-1795]
DESCRIPTION:IBM MQ Managed File Transfer could allow a local user to obtain highly sensitive information via trace log files generated by its commands.
CVSS Base Score: 4.4
CVSS Temporal Score: Seehttps://exchange.xforce.ibmcloud.com/vulnerabilities/137042 [https://exchange.xforce.ibmcloud.com/vulnerabilities/137042]for the current score
CVSS Environmental Score*: Undefined
CVSS Vector: (CVSS:3.0/AV:L/AC:L/PR:H/UI:N/S:U/C:H/I:N/A:N) 

AFFECTED PRODUCTS AND VERSIONS
IBM MQ Managed File Transfer v8.0.0.0 through v8.0.0.8 

IBM MQ Managed File Transfer v9.0.0.0 through v9.0.0.2 (LTS Release) 

IBM MQ Managed File Transfer v9.0.1, v9.0.2, v9.0.3 and v9.0.4 (CD Release)

REMEDIATION/FIXES

Apply fix pack 8.0.0.9 [http://www-01.ibm.com/support/docview.wss?uid=swg22015103]

Apply fix pack 9.0.0.3 [http://www-01.ibm.com/support/docview.wss?uid=swg24044508]
