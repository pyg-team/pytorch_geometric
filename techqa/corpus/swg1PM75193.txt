LOCAL FIX

PROBLEM SUMMARY
 *  ****************************************************************
   * USERS AFFECTED:
   *
   * ITCAM for AD v7.1 users who use port consolidator to
   * communicate with the ITCAM Managing Server (to reduce the
   * number of ports that need to be opened through the fire wall).
   *
   ****************************************************************
   *
   * PROBLEM DESCRIPTION:
   *
   * When the Port consolidator and the JVM containing the ITCAM
   * Data Collector are started before the Managing server is
   * started, the monitored JVM never connects to the Managing
   * server.  Only when the Managing server is started first, does
   * the DC connect correctly.
   *
   ****************************************************************
