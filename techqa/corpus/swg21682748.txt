Title: IBM Security Bulletin: A security vulnerability has been identified in IBM Websphere shipped with IBM Security Access Manager for Enterprise Single Sign-On (CVE-2014-4263, CVE-2014-4244) - United States

Text:
PSIRT SECURITY BULLETIN

SUMMARY
 IBM Websphere is shipped as a component of IBM Security Access Manager for Enterprise Single Sign-On (ISAM ESSO). Information about two security vulnerabilities affecting IBM Websphere has been published in a security bulletin. 

VULNERABILITY DETAILS
Two security vulnerabilities have been identified in IBM WebSphere shipped with IBM Security Access Manager for Enterprise Single Sign-On (CVE-2014-4263 , CVE-2014-4244). Please consult the security bulletin "Security Bulletin: Multiple vulnerabilities in current IBM SDK for Java for WebSphere Application Server July 2014 CPU" [http://www-01.ibm.com/support/docview.wss?uid=swg21680418] for vulnerability details and information regarding fixes.
