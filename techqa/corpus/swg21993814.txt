Title: IBM Security Bulletin: Multiple OpenSSL and Non-OpenSSL vulnerabilities have been identified in IBM SDK for Node.js included in Rational Software Architect and Rational Software Architect for WebSphere Software - United States

Text:
 SECURITY BULLETIN

SUMMARY
 IBM SDK for Node.js shipped with Rational Software Architect Designer and Rational Software Architect Designer for Websphere Software .OpenSSL vulnerabilities were disclosed on September 22 and 26, 2016 by the OpenSSL Project. OpenSSL is used by IBM SDK for Node.js. IBM SDK for Node.js has addressed the applicable CVEs, plus three additional vulnerabilities unrelated to the OpenSSL release. 

VULNERABILITY DETAILS
Refer to the security bulletin listed in the Remediation/Fixes section

AFFECTED PRODUCTS AND VERSIONS
IBM Rational Software Architect v9.1 - 9.1.2.2 

 IBM Rational Software Architect for WebSphere Software v9.1 - 9.1.2.2 

IBM Rational Software Architect Designer v9.5 - 9.5.0.2 

 IBM Rational Software Architect Designer for WebSphere Software v9.5 - 9.5.0.2

REMEDIATION/FIXES
Refer to the following security bulletins for vulnerability details and information about fixes addressed by IBM SDK Node.js which is shipped with Rational Software Architect and Rational Software Architect for Websphere Software.
