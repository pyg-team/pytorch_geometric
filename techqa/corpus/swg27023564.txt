Title: IBM System requirements for IBM Atlas Suite V6.0.1.x - United States

Text:
 PRODUCT DOCUMENTATION

ABSTRACT
 To deploy IBM Atlas Suite, you need, at a minimum, a database server and client, an application server, and a Web browser. This document lists the technical specifications for these platforms, as well as additional, optional, components

IBM Atlas Suite comprises:
- Atlas eDiscovery Process Management
- Disposal and Governance Management for IT
- Global Retention Policy and Schedule Management 

CONTENT
To deploy IBM Atlas Suite, you need, at a minimum, a database server and client, an application server, and a web browser. Technical specifications for these platforms, as well as optional components, are given in the following sections. IBM Atlas Suite has been validated on systems with the characteristics listed in these sections. These system requirements assume a 25-user environment. 




The specifications listed here are guidelines; actual memory and disk space requirements depend on a number of factors. Consult your IBM ECM Client Technical Professional to determine the appropriate hardware configuration for your environment. 

The database and application server software are not provided with the IBM Atlas Suite distribution.

PST Extractor Platform
Atlas PST Extractor is an optional application that extracts and indexes Outlook PST and MSG files that are included as part of a collection. If your collections do not include these files, you do not need to install PST Extractor. 

PST Extractor runs on a Windows machine with the following characteristics: 

Operating System Windows 2008 Server, Windows 7 Software .NET framework version 2.0 (included with Windows 2008 Server) Microsoft Outlook 2007 or 2010 with a default profile
or
Microsoft Exchange Storage A file system with a shared directory that is accessible to the Atlas application server and the PST Extractor host. 
Policy Federation and Policy Distribution Connectors
IBM Atlas eDiscovery Process Management and IBM Global Retention Policy and Schedule Management provide frameworks that let you use connect Atlas to external eDiscovery and Records Management systems through Policy Federation (eDiscovery) and Policy Distribution (Records Management) Connectors. The Connectors are separate applications that let you automate the preservation and collection of evidence, and the application of retention policies to a Records Management system.
