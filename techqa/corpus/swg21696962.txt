Title: IBM 【セキュリティ情報】WebSphere Transformation Extender with Launcher Hypervisor Edition [for RHEL] GNU C library (glibc)の脆弱性(CVE-2015-0235) - United States

Text:
 SECURITY BULLETIN

SUMMARY
 WebSphere Transformation Extender with Launcher Hypervisor Edition [for RHEL] で提供されている Red Hat Enterprise Linux (RHEL) の GNU C library (glibc) に脆弱性が存在します。(CVE-2015-0235).


VULNERABILITY DETAILS
脆弱性の詳細：
CVEID:CVE-2015-0235 [http://cve.mitre.org/cgi-bin/cvename.cgi?name=CVE-2015-0235] 
内容：GNU C library (glibc)の__nss_hostname_digits_dots() 関数にバッファオーバフローの脆弱性が存在します。
細工したホスト名を gethostbyname などの関数の引数に渡すことにより、バッファオーバーフローが発生します。
この脆弱性を悪用された場合、アプリケーションの異常終了や、任意のコードが実行可能になる可能性があります。
この脆弱性は、"Ghost"と呼ばれています。 

CVSS Base Score: 7.6
CVSS Temporal Score: See https://exchange.xforce.ibmcloud.com/vulnerabilities/100386 [https://exchange.xforce.ibmcloud.com/vulnerabilities/100386] for the current score
CVSS Environmental Score*: Undefined
CVSS Vector: (AV:N/AC:H/Au:N/C:C/I:C/A:C)

AFFECTED PRODUCTS AND VERSIONS
影響のある製品・バージョン： 

次の製品に影響があります。

製品: WebSphere Transformation Extender with Launcher Hypervisor Edition [for RHEL]

バージョン:
* 8.4.0.0 - 8.4.0.5
* 8.4.1.0 - 8.4.1.2

REMEDIATION/FIXES
修正策：

Fix Central から interim fix for APAR PI34228 [http://www.ibm.com/support/fixcentral/swg/selectFixes?parent=ibm~WebSphere&product=ibm/WebSphere/WebSphere+Transformation+Extender&release=All&platform=All&function=aparId&apars=PI34228] を導入してください。 [http://www.ibm.com/support/fixcentral/swg/selectFixes?parent=ibm~WebSphere&product=ibm/WebSphere/WebSphere+Transformation+Extender&release=All&platform=All&function=aparId&apars=PI34228]


FIXを導入後、glibcを用いたプロセスを再開始する必要があります。
システムの多くのプロセスがglibcを使う場合、FIX導入後、システムを再起動することを推奨いたします。

オペレーティングシステムを含むglibcの脆弱性を確認し、適切な緩和と改善措置をとるために、全ての環境を見直すよう、推奨いたします。詳細はあなたのオペレーティングシステム・プロバイダーに連絡してください。
