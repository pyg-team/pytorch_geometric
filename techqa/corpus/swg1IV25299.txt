LOCAL FIX

PROBLEM SUMMARY
 *  The problem is caused when the process that generated the core
   dump is badly damaged and the resulting core dump file is
   corrupted.
   
   
    
   
   

PROBLEM CONCLUSION
 *  This defect will be fixed in:
   7.0.0 SR3
   6.0.1 SR4
   6.0.0 SR12
   .
   The jextract tool has been updated to avoid going out of memory
   in this way. Depending on the corruption to the core dump
   jextract may still fail to read it but should exit with an error
   message instead.
