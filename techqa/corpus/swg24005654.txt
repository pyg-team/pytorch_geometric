DOWNLOAD DESCRIPTION
This document contains a link to a patch that resolves problems when using Lotus EasySync Pro® 4.2.1 for synching with devices that run Windows Pocket PC 2003® software.

NOTE: ONLY apply this patch to EasySync Pro 4.2.1 that links to devices using a Pocket PC 2003. This patch is NOT for use with any version of EasySync Pro prior to 4.2.1. This issue was fixed in EasySync Pro 4.2.2. 

For more information, refer to IBM's Upgrade Central web site at http://www-306.ibm.com/software/lotus/support/upgradecentral/index.html [http://www-306.ibm.com/software/lotus/support/upgradecentral/index.html].

INSTRUCTIONS FOR A NEW EASYSYNC PRO INSTALL

1. Install EasySync Pro.

2. After installation but BEFORE the first synchronization, copy the setup.arm.cab file into the Program Files\ Common Files\ XCPCSync\ Translators\ PocketPC directory (this replaces the existing .cab file of the same name).

3. Proceed with the first synchronization.

INSTRUCTIONS FOR AN EXISTING EASYSYNC PRO INSTALL
If EasySync Pro is already installed, and was already used for synchronization with the device, you must uninstall the current file loaded on the Pocket PC device, as follows: 

1. Copy the setup.arm.cab file into the Program Files\ Common Files\ XCPCSync\ Translators\ PocketPC directory (this replaces the existing .cab file of the same name).

2. On the Pocket PC, from the Windows TaskBar, choose Start --> Settings --> System panel --> Remove Programs. Highlight EasySync Pro and delete it.

3. Once EasySync Pro is removed from the Pocket PC, launch a full sync from EasySync Pro. The new .cab file will automatically be copied to and installed on the device.
