ABSTRACT
 This document contains links (URLs) to published technical support articles and information pertaining to high availability (HA) and recovery. Use the information in these links to help you get more value from WebSphere Process Server version 7.x and WebSphere Enterprise Service Bus (WebSphere ESB) version 7.x. 

CONTENT
This information is intended to help you support and administer your WebSphere Process Server and WebSphere ESB installations. 

Document title (click the title for link) Abstract Configuring high availability and workload sharing of service integration [http://www.ibm.com/support/knowledgecenter/SSAW57_7.0.0/com.ibm.websphere.nd.multiplatform.doc/info/ae/ae/tjt0023_.html?lang=en] This WebSphere Application Server knowledge center topic provides information on configuring HA and workload sharing of service integration.

It provides descriptions for the following tasks:
 * Creating a policy for messaging engines 
 * Configuring a policy for messaging engines 
 * Configuring shared durable subscriptions

Service Integration Bus Explorer - frequently asked questions (FAQ) [http://www.alphaworks.ibm.com/tech/sibexplorer/faq] This FAQ document provides information on Service Integration Bus Explorer.

Service Integration Bus Explorer is a stand-alone tool that allows more natural navigation and monitoring of the messaging components of a Service Integration Bus. Transactional high availability [http://www.ibm.com/support/knowledgecenter/SSAW57_7.0.0/com.ibm.websphere.nd.doc/info/ae/ae/cjta_trans_ha.html?lang=en]
This WebSphere Application Server knowledge center topic provides conceptual information on transactional HA.

High availability of the transaction service enables any server in a cluster to recover the transactional work for any other server in the same cluster. This facility forms part of the overall WebSphere Application Server HA strategy. Configuring transaction properties for peer recovery [http://publib.boulder.ibm.com/infocenter/wasinfo/v7r0/topic/com.ibm.websphere.nd.multiplatform.doc/info/ae/ae/tjta_cfgpeer.html] This WebSphere Application Server knowledge center topic provides information on configuring transaction properties for peer recovery.

It provides descriptions for the following tasks:
 * Configuring manual peer recovery for the transaction service. 
 * Configuring automated peer recovery for the transaction service 
 * Disabling file locking
