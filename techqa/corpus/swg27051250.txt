ABSTRACT
The IBM License Metric Tool and BigFix Inventory Support team hosted an Open Mic webcast about How to install and configure Disconnected Scanner (IBM iSeries) 

CONTENT
IBM iSeries - Disconnected Scanner - How to install and configure
Topics covered, which included two demonstrations:

 * Disconnected scanner package
 * Scalability
 * Scan frequency
 * Limitations
 * Disconnected scanner requirements for IBM i
 * Adding a new data source for disconnected scans
 * Preparing installation files for disconnected scans for IBM i
 * Installing the scanner and gathering initial data on IBM i systems
 * Running software scans and gathering scan results on IBM i systems
 * Importing scan results from IBM i systems to IBM License Metric Tool
