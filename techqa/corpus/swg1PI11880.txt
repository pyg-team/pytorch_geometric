APAR STATUS
 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  Associating child object to an unlocked parent and locked parent
   at same time causes "missing locale" to appear in OP-04050
   error message
   
   This is the full error message that appears:
   
   OP-04050: Association of the child object type "missing
   en_US:_<Chinese chars>" to the locked parent object type
   "missing:en_US:_<Chinese Chars>" is not allowed.  Please contact
   your administrator.
   
   
   The following are some generic details to replicate the issue.
   
   Prerequisites:
   
   6.2.1 environment with Solutions installed
   
   Ensure that this Administration Setting contains SOXProcess in
   it's list of objects
   /OpenPages/Applications/GRCM/Locks/Display Lock Button
   
   1. Open a supported browser.
   2. Go to the OpenPages URL.
   3. Login with an administrator id.
   4. Navigate to Organization, Business Entities on top level
   menu.
   5. To create a new entity,  Click "Add New" , fill in the "Name
   Field" with E1 and hit the "Create" button.
   6. Click on E1 to access the detail page.
   7. In left pane, click on Processes,
   8. Click on Actions, Add a new Process on the Process List pane
   9. Enter a Process name, P1, in the name field and click Save
   button
   10. Repeat steps (8 and 9)  2 more times to create processes P2
   and P3
   11. Click on P3 in the List Pane and then click on Actions, Lock
   this Process on the Context Pane
   12. Click on P1 in the List Pane and click on P1 in the Context
   pane to display the Detail View
   13. On left pane, click on Risks
   14. Click on Actions , Add a new Risk
   15. Enter a Risk name, R1,  in the name field and click Save
   button
   
   
   Steps to Reproduce:
   1. Open a supported browser.
   2. Go to the OpenPages URL.
   3. Login with an administrator id.
   4. Click on Assessments, Risk on top navigation menu
   5. Click on R1 and then click Processes under Parents section in
   left pane
   6. Click on Actions, Associate an existing Process on the
   Process List pane
   7. Select P2 and P3 using the checkbox and click on "Associate
   Selected" button
   8. Confirm that the error message appears with the "missing
   locale" message embedded.
   
   Expected Results:
   
   The correct error message below should appear:
   
   OP-04050: Association of the child object type "Risk" to the
   locked parent object type "Process" is not allowed.  Please
   contact your administrator.
   
   Actual Results:
   
   The incorrect error message below appears:
   
   OP-04050: Association of the child object type "missing
   en_US:_<Chinese chars>" to the locked parent object type
   "missing:en_US:_<Chinese Chars>" is not allowed.  Please contact
   your administrator.
   
   Error Message:
   
   No entries in APP logs occur at this time.
