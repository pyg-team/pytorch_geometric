Title: IBM Security Bulletin: API Connect is affected by a vulnerability by which an authenticated user could generate an API token - United States

Text:
 SECURITY BULLETIN

SUMMARY
 API Connect has addressed the following vulnerability. 

An authenticated user could be allowed to generate an API token when not subscribed to the application plan. 

VULNERABILITY DETAILS
CVEID: CVE-2017-1555 [http://cve.mitre.org/cgi-bin/cvename.cgi?name=CVE-2017-1555]
DESCRIPTION: IBM API Connect could allow an authenticated user to generate an API token when not subscribed to the application plan.
CVSS Base Score: 4.3
CVSS Temporal Score: See https://exchange.xforce.ibmcloud.com/vulnerabilities/131545 [https://exchange.xforce.ibmcloud.com/vulnerabilities/131545] for the current score
CVSS Environmental Score*: Undefined
CVSS Vector: (CVSS:3.0/AV:N/AC:L/PR:L/UI:N/S:U/C:N/I:L/A:N)
