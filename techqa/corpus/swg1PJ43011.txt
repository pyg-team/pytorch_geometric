 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  Description of Problem (full details)
   Using Internet Explorer 11 with Case Manager Builder, the
   browser might
   hang and become unresponsive during a logout operation.
   
   Steps to Reproduce:
   1  Open Case Manager Builder in Internet Explorer 11 (not using
   Compatibility View mode).
   2  Logon to Case Manager Builder application.
   3  Logout of Case Manager Builder and click Ok on the dialog.
   
   Actual Results:
   Browser hangs or long running script error message in the bottom
   of the IE browser.
   
   Expected Results:
   Browser would not hang and would return to the logon page.
   
   
    
   
   

LOCAL FIX
 *  Close the browser instance, restart it, and return to the Case
   Manager
   Builder logon page.
