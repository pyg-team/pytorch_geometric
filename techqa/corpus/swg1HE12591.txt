 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  When the CSV to import people into the scope of a request
   contains IDs that already exist in scope, three symptoms are
   seen -
   a) the first time the import will fail when clicking on "Add
   and Close"
   b) subsequent attempts to import will throw an error message on
   the screen but will succeed
   c) the people in scope page needs to be manually refreshed in
   order to see the newly imported users
   
   
    
   
   

LOCAL FIX
 *  Use the "Add" button to do the import. Manually refresh the
   page to see the imported users.
