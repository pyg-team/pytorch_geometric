CAUSE
The Service Username was not a member of the PE administrator group. 


RESOLVING THE PROBLEM
The error indicates that the Service Username is not a member of the PE administrator. 

 1. Check the active directory to make sure that the supplied user in Service Username field is a member of the PE Administrator group. 
 2. If the Service Username is not a member of the PE administrator group, find out a user that is a member of the PE administrator group, and use that user to configure in Service Username field. See reference screen shot below. 


[/support/docview.wss?uid=swg21668624&aid=1] [/support/docview.wss?uid=swg21668624&aid=1] Note that if the user is newly added to PE administrator group in the active directory, it may take up to 4 hours before the user group membership gets updated in PE database. If you need to get it to update right away to avoid the error, log onto vwtool as the PE administrator user and run flushusercache command follow by the environment i f m y command from within PE vwtool utility. See the reference link below.
