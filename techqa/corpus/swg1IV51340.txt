APAR STATUS
 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  After installing Oracle Java Update 45, or IBM Java 7 SR6 FP1,
   or IBM Java 6 SR15 FP1  the product console:
   
          1. Displays warning:
       "This application will run with unrestricted access which
   may put your computer and personal information at risk. Run this
    application only if you trust the location and publisher above"
   
       2. After clicking RUN button nothing happens and Product
   Console is not opened.
   
   In the java webstart console log the following error appears;
   
   Exception in thread "AWT-EventQueue-2"
   java.lang.IllegalArgumentException: Wrong address
    at
   com.collation.gui.util.ServerNameExtractor.extractServerName(Ser
   verNameExtractor.java:32)
    at com.collation.gui.client.GuiMain.runIt(GuiMain.java:503)
   
   
   or
   Security warning indicates "JAR file manifest does not contain
   the Permissions attribute"
   
   And Java console log exception shows "Missing required
   Permissions manifest attribute in main jar ... guiclient.jar"
   
   
    
   
   

LOCAL FIX
 *  Install an earlier level of Java
   
   
    
   
   

PROBLEM SUMMARY
 *  Latest Oracle Java 7 update 45 or IBM Java SR6 FP1 or IBM
   Java 6 SR15 FP1 does not allow TADDM Discovery Management
   Console
