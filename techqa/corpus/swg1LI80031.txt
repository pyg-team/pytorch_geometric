APAR STATUS
 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  Currently the Chinese characters are showing as "?" symbols in
   the received notification mails configured using SEND MAIL
   activity.After trying with different MIME attributes by adding
   UTF-8/.., customer observed that Send Mail output activity
   showing chinese character but when we see in Email body showing
   '????'.
   
   
    
   
   

LOCAL FIX

PROBLEM SUMMARY
 *  Chinese characters are not interpreted since the content is not
   passed as UTF-8.
   
   
    
   
   

PROBLEM CONCLUSION
 *  The fix is targeted for inclusion if WebSphere Cast Iron fixpack
   V7.5.1.1, V7.5.2.1
