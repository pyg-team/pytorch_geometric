Title: IBM How to recreate the nzevent templates after a nzsystem -reinit - United States

Text:
 TECHNOTE (FAQ)

QUESTION
 I can not create nzevents from templates, the command 'nzevent show -template' shows nothing 

CAUSE
After a nzsystem -reinit there are no event templates to copy from. 

ANSWER
A system reinit does not automatically recreate the event templates. You have to run a SQL script to do this. As the 'nz' user run the following: 

[nz@h1 ~]$ nzevent show -template
No event rules

[nz@h1 ~]$ nzsql system -f /nz/kit/bin/adm/tools/create_event_templates.sql
INSERT 7 

[nz@h1 ~]$ nzevent show -template
