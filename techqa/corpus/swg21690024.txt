Title: IBM Unable to start Rulerunner as a Service in IBM Datacap Taskmaster Capture - United States

Text:
rulerunner service start TECHNOTE (TROUBLESHOOTING)

PROBLEM(ABSTRACT)
 Unable to start Rulerunner in Windows Services or by using Rulerunner Manager, and no Rulerunner logs are generated. 

SYMPTOM
 

 * When trying to run Rulerunner from Windows Services it does not start and Windows Event Viewer reports "The Datacap Rulerunner Service service terminated unexpectedly", and no Rulerunner logs are generated. 
 * When trying to run Rulerunner from Datacap Rulerunner Manager, the count down reaches zero and an error states that it timed out. 
 * When running Rulerunner from CMD on the same Rulerunner service account, the logs are generated and Rulerunner works.


CAUSE
The Rulerunner service account is being blocked from creating a COM component.

RESOLVING THE PROBLEM
 

 1. Add the Rulerunner service account to Datacap folder security for full control
    http://www.ibm.com/support/knowledgecenter/SSZRWV_8.1.0/com.ibm.dc.install.doc/dcsrv006.htm [http://www-01.ibm.com/support/knowledgecenter/SSZRWV_8.1.0/com.ibm.dc.install.doc/dcsrv006.htm] 
 2. Add the Rulerunner service account to the Datacap folder share properties for full control
    http://www.ibm.com/support/knowledgecenter/SSZRWV_8.1.0/com.ibm.dc.install.doc/dcsrv010.htm [http://www-01.ibm.com/support/knowledgecenter/SSZRWV_8.1.0/com.ibm.dc.install.doc/dcsrv010.htm] 
 3. Add Rulerunner service account to the "Distributed COM Users" group policy
    Located in 64-bit at Computer Management > System Tools > Local Users and Groups > Groups
    Located in 32-bit at Server Manager > Configuration > Local Users and Groups > Groups
