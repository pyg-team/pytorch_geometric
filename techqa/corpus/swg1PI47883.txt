ERROR DESCRIPTION
 *  When using FireFox browser, you will see the following issues in
   the Component Template page.
   
   Steps to reproduce:
   
   1. Select Configuration > Environment Property Definition
   2. Add Property
   3. Set name= test
   4. Set Pattern= +//d
   5. Set value= 1234
   
   The page will not reload after you save--it will appear to hang.
   
   
   Setting he pattern to: /d+ you will see the expected
   error message:
   
   Please correct the following errors before submitting this form:
   Default value for property test does not follow the required
   pattern
