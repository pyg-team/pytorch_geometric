Title: IBM Datacap Rulerunner service will not start - United States

Text:
rulerunner service start TECHNOTE (TROUBLESHOOTING)

PROBLEM(ABSTRACT)
 When starting the Datacap Rulerunner service, using Rulerunner Manager, the service does not start. After 30 seconds, it times out while still displaying the Starting Service. 

SYMPTOM
If attempting to start the service through Windows Services, it reports that the service cannot start. No Rulerunner or Windows Event log entries are generated.


CAUSE
If Rulerunner does not have any threads configured, it will shut itself down.

RESOLVING THE PROBLEM
Using Datacap Rulerunner Manager, go to the Workflow tab and configure at least one thread.
