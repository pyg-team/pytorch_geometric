Title: IBM Weblogic Application Console Does Not Come Up Properly for OpenPages - United States

Text:
OpenPages WebLogic Console TECHNOTE (TROUBLESHOOTING)

PROBLEM(ABSTRACT)
 Attempting to log into the WebLogic Console cannot be performed because the Console is not being displayed correctly. The login page displays an 'Error 404--Not Found' message. 

SYMPTOM
Attempting to log into the WebLogic Console cannot be performed because the Console is not being displayed correctly. The login page displays an 'Error 404--Not Found' message.


CAUSE
A corruption has occurred in the deployment of the WebLogic application.

DIAGNOSING THE PROBLEM
 The WebLogic Console login page displays an 'Error 404--Not Found' message.

RESOLVING THE PROBLEM
Clear the WebLogic cache (see TechNote # 1591907 for steps on clearing the application cache).

RELATED INFORMATION
