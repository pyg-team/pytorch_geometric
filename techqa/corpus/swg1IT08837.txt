Title: IBM IT08837: TYPE 8436: UPGRADE TO RAID FIRMWARE IS NEEDED - United States

APAR STATUS
 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  An upgrade to the RAID firmware is needed.  Issuing "boot image"
   to install this firmware upgrade on Type 8436 appliance
   platforms might take an additional few minutes for the RAID
   firmware upgrade.  The increase is one time for the initial
   "boot image" only.
   
   
    
   
   

LOCAL FIX

PROBLEM SUMMARY
 *  Affected are all users of the RAID storage on an IBM DataPower
   Gateway (Type 8436).
   
   CLI command "show raid-logical-drive" might show "writeThrough"
   Write Policy in effect instead of "writeBack".
   
   
    
   
   

PROBLEM CONCLUSION
 *  Fix is available in 7.1.0.5.
   
   For a list of the latest fix packs available, please see:
   http://www-01.ibm.com/support/docview.wss?uid=swg21237631 [http://www-01.ibm.com/support/docview.wss?uid=swg21237631]
