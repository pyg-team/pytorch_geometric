ERROR DESCRIPTION
 *  IBM DataPower Gateways
   
   Firmware Level : 7.0.1.x
   
   Problem description:
   After upgrade to 7.0.1.x and later fix packs, user experience a
   WebGUI problem in Microsoft Internet Explorer browser. The
   "Object" section of the navigation view does not expand.
   
   Same issue doesn't be seem in Firefox and Chrome browser.
   
   This issue will be considered for inclusion in a future release
   of DataPower.
   
   
    
   
   

LOCAL FIX
 *  There are 2 workarounds for this WebGUI issue.
   
   1. Clear Internet Explorer browser cache
      (1). Select Tools (with the Gear Icon) > Safety > Delete
   browsing history.
      (2). Uncheck Preserve Favorites website data, and check both
   Temporary Internet Files and Cookies, then click Delete.
   
   
   2. In some instances, user might still problem after refresh the
   cache and applying Internet Explorer browser patches. The
   problem could be document compatibility mode in Internet
   Explorer. To verify this, user can try to change the document
   compatibility mode in Internet Explorer.
      Press F12 to launch development tools, then press CTRL|8, and
   change the document mode from 8 to 9. Changes made to the
   Document Mode in F12 Developer Tools apply only for the duration
   of the browser session. When you close the browser and return
   to the site, it will be in the default mode again and the
   problem will return. This problem is specific to Internet
   Explorer and customers can use FireFox or Chrome if this option
   is available.
   
   
    
   
   

PROBLEM SUMMARY
 *  A DataPower WebGUI user using Internet Explorer 10 or 11 may
   not be able to expand the Object left navigation.
   
   Fix will be available in next major releases.
   
   
    
   
   

PROBLEM CONCLUSION

TEMPORARY FIX
