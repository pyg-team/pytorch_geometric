Title: IBM Object Server を起動できません - United States

Text:
failed; storage system; processing loop; Japanese; 日本語 TECHNOTE(トラブルシューティング)

問題(概要)
 ストレージ・システムの開始に失敗しましました。 

症状
Object Server を起動できず、以下のエラーがログファイルに見つかります。 

Error: E-OBJ-102-020: Failed to start the storage system. (2:No such file or directory)
Error: E-IPC-005-001: OpenServer - Error: 16334/10/0: SRV_START イベントにより srv_run がアボートされました。
Error: E-STK-102-014: [nstk]: BOB: Failed to start server processing loop. (-19:General failure)


解決方法
$OMNIHOME/bin にディレクトリを変更後、新たなDBを生成させるように nco_dbinit -server BOB を実行します。 

この実行が成功したら、BOBという名前の Object Server を起動できるようになります。
