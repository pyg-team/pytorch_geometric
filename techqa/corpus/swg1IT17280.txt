APAR STATUS
 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  Possible execution of arbitrary code with elevated privileges
   (CVE-2016-2177)
   
   Possible access to sensitive information, caused by an error in
   DSA implementation (CVE-2016-2178)
   
   Possible memory corruption due to printing overly large
   untrusted certificate of CRL (CVE-2016-2182)
   
   SWEET32 attack on 64 bit block cipher algorithms [DES]
   (CVE-2016-2183)
   
   Possible denial of service from maliciously sending of multiple
   excessively large OCSP Status Request extensions (CVE-2016-6304)
   
   Theoretical denial of service due to memory corruption
   (CVE-2016-6306)
   
   
    
   
   

LOCAL FIX
 *  CVE-2016-2183: Disabling of Triple-DES CBC cipher suites
   Others: None.
   
   
    
   
   

PROBLEM SUMMARY
 *  DataPower might be susceptible to vulnerabilities in SSL
   announced  in September, 2016.
   
   Possible execution of arbitrary code with elevated privileges
   (CVE-2016-2177)
   Possible access to sensitive information, caused by an error in
   DSA implementation (CVE-2016-2178)
   Possible memory corruption due to printing overly large
   untrusted certificate of CRL (CVE-2016-2182)
   SWEET32 attack on 64 bit block cipher algorithms [DES]
   (CVE-2016-2183)
   Possible denial of service from maliciously sending of multiple
   excessively large OCSP Status Request extensions (CVE-2016-6304)
   Theoretical denial of service due to memory corruption
   (CVE-2016-6306)
   
   
    
   
   

PROBLEM CONCLUSION
 *  Fix is available in 7.0.0.16, 7.1.0.13, 7.2.0.10, 7.5.0.4,
   7.5.1.3 and 7.5.2.1
   
   For a list of the latest fix packs available, please see:
   http://www-01.ibm.com/support/docview.wss?uid=swg21237631 [http://www-01.ibm.com/support/docview.wss?uid=swg21237631]
