Title: IBM The bootstrapProcessServerData command returns a null pointer exception when configuring an IBM Business Process Manager (BPM) Advanced-only deployment environment - United States

Text:
 TECHNOTE (TROUBLESHOOTING)

PROBLEM(ABSTRACT)
 The bootstrapProcessServerData command aborts with a NullPointerException message in the wsadmin.traceout file when it is run for an IBM Business Process Manager Advanced-only deployment environment.

CAUSE
An IBM Business Process Manager Advanced-only deployment environment is configured for deploying Business Process Choreographer and SCA modules. This configuration is equivalent to a WebSphere Process Server configuration and does not contain the IBM Business Process Manager Standard components. There is no need to run the bootstrapProcessServerData command for an IBM Business Process Manager Advanced-only deployment environment.



RESOLVING THE PROBLEM
The bootstrapProcessServerData command is not required for an IBM Business Process Manager Advanced-only deployment environment.
