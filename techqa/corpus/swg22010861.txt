Title: IBM Fix Pack 8.0.0.8 for IBM MQ V8 - United States

Text:
8.0.0.8; csd8008; fix pack; fixpack NEWS

ABSTRACT
 Fix Pack 8.0.0.8 for IBM MQ V8 is now available. 

CONTENT
Fix Pack 8.0.0.8 is the eighth maintenance for the IBM MQ V8 product first shipped in May 2014 on the following platforms: 

 * AIX 
 * HP-UX Itanium 
 * IBM i 
 * Linux on POWER 
 * Linux on Power (Little Endian) 
 * Linux on x86-64 
 * Linux on zSeries s390x 
 * Solaris on SPARC 
 * Solaris on x86-64 
 * Windows 
 * z/OS (JMS feature only) 
 * IBM MQ Appliance M2000/M2001 
 * IBM MQ Advanced V8.0 Virtual System Pattern Type for IBM PureApplication on AIX 
 * IBM MQ Advanced V8.0 Virtual System Pattern Type for IBM PureApplication on Linux
   

The links in the download section below link to the downloads for MQ V8.0.0.7 on all applicable platforms. You will be asked to register before being presented with a set of downloadable files. 

Prerequisites 

Install this fix pack on top of MQ V8.0.0.0 , MQ V8.0.0.1, MQ V8.0.0.2, MQ V8.0.0.3, MQ V8.0.0.4, MQ V8.0.0.5, MQ V8.0.0.6 or MQ V8.0.0.7. 

The supported environments for MQ V8.0 can be found here [http://www.ibm.com/support/docview.wss?uid=swg27041395].
