Title: IBM Running an mqsilist command results in a Java OutOfMemoryError - United States

Text:
 TECHNOTE (TROUBLESHOOTING)

PROBLEM(ABSTRACT)
 You run the mqsilist command in IBM Integration Bus (IIB) or WebSphere Message Broker (WMB) and receive an error message (BIP8075E) that indicates a Javaâ„¢ out of memory error has occurred. 

SYMPTOM
The error includes the following text: 

BIP8075E: A Java exception was caught from class 'com/ibm/broker/config/util/MQSIList.main'. The text is 'java.lang. OutOfMemoryError: Failed to create a thread: retVal -1073741830, errno 11'. 
A Java exception was caught via the JNI. Check preceding messages for related errors and retry the operation. If the exception still occurs, contact your IBM support center. 


CAUSE
The value of the parameter "data ulimit" is restricting the memory that is assigned to the mqsilist process.


RESOLVING THE PROBLEM
To resolve this problem, complete the following steps: 

 * 
 * 
 * ulimit -d unlimited 
 *
