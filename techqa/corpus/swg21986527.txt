Title: IBM Does TBSM 6.1.1 support JazzSM 1.1.3? - United States

Text:
 TECHNOTE (FAQ)

QUESTION
 Does TBSM 6.1.1 support JazzSM 1.1.3.0? 

ANSWER
Yes, TBSM 6.1.1 Fixpack 4 supports JazzSM 1.1.3.0 and has been certified for use with JazzSM 1.1.3.0.
However, there are some known issues with using JazzSM 1.1.3 currently. Please see the technotes below for more information.

1) JazzSM 1.1.3 upgrade with Installation Manager issue
