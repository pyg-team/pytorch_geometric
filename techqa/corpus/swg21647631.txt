PROBLEM(ABSTRACT)
 Can the configtemplate.sh script be run in silent mode? 

ENVIRONMENT
IBM Tivoli Composite Application Agent for WebSphere Applications, Version 7.2

DIAGNOSING THE PROBLEM
 ITCAM Agent for WebSphereÂ® Applications provides a configtemplate script to create an ITCAM server template and to add it to the WebSphere template repository. Running configtemplate script presents prompts that requires responses that must be entered manually from the command line. 

RESOLVING THE PROBLEM
This script cannot be run in "silent mode" because a response file is not provided for configtemplate.sh script. Currently, command-line is the only option available.
