 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  On Windows platform, after the installation of the Microsoft fix
   KB2998527, needed to update the OS timezone tables accordingly
   to the recent changes for Russia, the Cognos scheduled reports
   are executed one hour earlier than expected.
   It occurs when the selected timezone is set to (GMT+3:00)
   Russia, Moscow.
   This is a defect in Cognos engine, documented with Cognos APAR
   PI31932 [http://www-01.ibm.com/support/docview.wss?uid=swg1PI31932]
