Title: IBM IZ19822: ADVANCED PING TOOL DOESN'T WORK WITHOUT HAVING CORRECT LIBRARY PATH & PERMISSIONS FOR A NON-ROOT USER - United States

ERROR DESCRIPTION
 *  When running NGF as non-root user, On the Webtop/Topoviz server
   following steps are required for Advanced ping tool to function
   correctly :
   
   1.  Add $PRECISION_HOME/platform/solaris2/lib to the non-root
   user LD_LIBRARY_PATH.
   
   2. Add $PRECSION_HOME/platform/solaris2/lib to the trusted
   list and default search path (using crle command)
   
   3. Add setuid in the permissions for the executables in the
   $NCHOME/guifoundation/webapps/ncp_webtools/WEB-INF/bin directory
   (chmod u+s)
   
   The Advanced Ping tool works having above permissions & path
   in place.
   
   This need to be documented for the tool to work correctly for
   non-root users.

   When running the NGF as a non-root user, the following
   permissions and paths must be set on the Webtop/Topoviz server
   for Advanced ping tool to function correctly.
   
   1.  Add /precision/platform/solaris2/lib to the non-root user
   LD_LIBRARY_PATH.
   
   2. Add /precision/platform/solaris2/lib to the trusted list and
   default search path (using crle command).
   
   3. Add setuid in the permissions for the executables in the
   /guifoundation/webapps/ncp_webtools/WEB-INF/bin directory. Use
   the command chmod u+s.
