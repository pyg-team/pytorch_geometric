DOWNLOAD DESCRIPTION


This document describes how to download the installation images for IBM Tivoli Storage Manager for Storage Area Networks Version 6.2 using the Passport Advantage Online Web site [http://www.ibm.com/software/howtobuy/passportadvantage/pao_customers.htm]. 

The IBM Tivoli Storage Manager for Storage Area Networks extension allows SAN-connected Tivoli Storage Manager servers and Tivoli Storage Manager client computers to make maximum use of their direct network connection to storage by performing backup/restore and archive/retrieve data transfers over the SAN instead of the LAN. 

More information

To access a wide variety of technical resources for this product, see the Tivoli Storage Manager V6.2 Information Center [http://publib.boulder.ibm.com/infocenter/tsminfo/v6r2/index.jsp].

Space requirements for AIX

 * 28.1 GB of disk space to store the downloadable part 
 * 32.7 GB of disk space to unpack the part 

 * Disk space for the installed product:  * AIX: 105 MB in /opt, 2 MB in /var, and 28 MB in /tmp 
   
   Downloading and assembling the product
   
    1. Download the part that you need. The parts are listed by platform in the tabbed sections of this document. 
    2. Unpack each downloaded part. 
    3. Follow the instructions in the IBM Tivoli Storage Manager for Storage Area Networks Quick Start Guide [http://www.ibm.com/support/docview.wss?&uid=swg27018163] to install the product.
