LOCAL FIX

PROBLEM SUMMARY
 *  ****************************************************************
   USERS AFFECTED:
   This issue affects all users of the runmqras command who use
   WebSphere MQ v7.0.1.12 on Windows platforms.
   
   
   Platforms affected:
   Windows
   
   ****************************************************************
   PROBLEM DESCRIPTION:
   In WebSphere MQ v7.0.1.12, a change was made to the runmqras
   facility (a tool that automates the task of collecting
   diagnostic information) to print progress information to the
   console while the command was running. However, this change
   included some incorrect command parsing logic while determining
   the progress information to display that resulted in an uncaught
   java.util.regex.PatternSyntaxException that caused the command
   to terminate and not to generate the expected output.
   
   
    
   
   

PROBLEM CONCLUSION
 *  The runmqras facility has been updated to correct the command
   parsing error which caused the
   java.util.regex.PatternSyntaxException to be thrown.
   
   ---------------------------------------------------------------
   The fix is targeted for delivery in the following PTFs:
   
   Version    Maintenance Level
   v7.0       7.0.1.14
