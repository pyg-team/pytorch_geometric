ABSTRACT
 The End of Marketing (EOM) date for CICS Transaction Server for z/OS (CICS TS) V3.2 was 30 September 2010. This document contains a link to the software withdrawal announcement letter that contains the EOM announcement for CICS TS V3.2. This document also contains information about upgrading to the latest releases of CICS TS. 

CONTENT
