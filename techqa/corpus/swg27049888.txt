ABSTRACT
 This document provides access to the detailed system requirements (DSR) for the supported operating systems, hardware and software requirements, and other related information for CICS Transaction Server for z/OS (CICS TS) 5.4, CICS TS Value Unit Edition (VUE) 5.4, and CICS TS Developer Trial 5.4.

For a list of supported and out of service high-level programming languages and compilers, see the High-level language support [https://www.ibm.com/support/knowledgecenter/SSGMCP_5.4.0/reference/compilers/hll_support.html] topic in the CICS TS V5.4 product documentation [https://www.ibm.com/support/knowledgecenter/SSGMCP_5.4.0/home/welcome.html]. 


CICSPlex SM 
If you intend to connect a CICS TS V5.4 CICSPlex System Manager Address Space (CMAS) to a previous version of CICS TS CMAS, you need to apply the PTF(s) for the following APAR(s) to each downlevel environment.  * PI81778 [http://www.ibm.com/support/docview.wss?uid=swg1PI81778] for CICS TS V4.1 and V4.2 
 * PI81780 [http://www.ibm.com/support/docview.wss?uid=swg1PI81780] for CICS TS V5.1, V5.2 and V5.3


The PTF(s) from the following APAR(s) are also required if you intend to use CICS HTTP event processing (EP) Adapters to emit events consumable by the Decision Server Insights component of IBM Operational Decision Manager in a CICS TS V5.4 MAS.  * PI55133 [http://www.ibm.com/support/docview.wss?uid=swg1PI55133] for CICS TS V5.1 and V5.2 
 * PI55134 [http://www.ibm.com/support/docview.wss?uid=swg1PI55134] for CICS TS V5.3 


CICS Explorer 
CICS Explorer V5.4 or later is required for CICS TS V5.4.
