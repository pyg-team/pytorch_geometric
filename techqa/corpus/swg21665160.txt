RESOLVING THE PROBLEM
To resolve the problem, complete the following steps: 

 1. Restart IBM Business Process Manager from the administrative console. To restart it, click Servers > All servers > SingleClusterServer1. 
    
    
 2. Stop the Deployment Manager by running the /opt/ibm/BPM/v8.5/profiles/DmgrProfile/bin/stopManager.sh command.
    
    
 3. Restart the Deployment Manager by running the /opt/ibm/BPM/v8.5/profiles/DmgrProfile/bin/startManager.sh command.



You can access the Administrative Console for IBM Business Process Manager using the following URL syntax: https://<Central Server 4>:9043/ibm/console 
Replace the <Central Server 4> variable with the address of the Center Server 4 in your environment.
