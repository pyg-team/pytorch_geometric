Title: IBM Tivoli Netcool/OMNIbus_GUI 8.1.0 Fix Pack 2 Interim Fix 1, 8.1.0.2-TIV-NCOMNIbus_GUI-IF0001 - United States

Text:
OMNIbus_GUI 8.1.0 Fix Pack 2 Interim Fix 1; OMNIbus_GUI 8.1.0.2.IF0001; 8.1.0.2.IF0001; Web GUI 8.1.0.2.IF0001; Web GUI 8.1.0 Fix Pack 2 Interim Fix 1; Web GUI 8.1.0 FP0002 IF0001 DOWNLOADABLE FILES

ABSTRACT
 This Interim Fix addresses problems reported in IBM Tivoli Netcool/OMNIbus Web GUI V8.1.0.2 

DOWNLOAD DESCRIPTION
Fixes that are included in Interim Fix 0001.


APAR IV71023 

 * 
 * Related event portlets fails to handle the request when the historical events have double quotes in the EVENTIDENTITY field 
 * 
 * When the historical events have double quotes in the EVENTIDENTITY field, it fails to show any of the related event configurations 


APAR IV71022  * 
 * Unable to view seasonal or historical events events using event viewer 
 * 
 * The feature in the Event Viewer that lets the user configure whether to view seasonal or historical events in the Event Viewer edit mode no longer works 


APAR IV71284  * 
 * Historical data with < and > in the summary fields does not show in related event configuration and seasonal report results 
 * 
 * When the historical data contains either the less than or the greater than sign in the summary field of the event, It fails to show up correctly in the table of events 


APAR IV70332  * 
 * WebGUI applets code sign certificate about to expire 
 * 
 * WebGUI applets code sign certificate will expired on Jun 28, 2015. The renew certificate need to be SHA-256 (SHA-2) due to deprecation of SHA-1 by major browsers (e.g. Microsoft Internet Explorer and Mozilla Firefox) starting Jan 1, 2017. 

PREREQUISITES
This IF contains fixes that are only applicable when using Netcool/OMNIbus Web GUI V8.1.0 Fix Pack 2.

INSTALLATION INSTRUCTIONS
Installation, migration, upgrade and configuration information: 
* Hardware and software requirements:

Refer to the Netcool/OMNIbus Web GUI 8.1.0 Release Notes for a full 
description of all the hardware and software requirements.

* Dependencies:

This Interim Fix must be installed on each instance of the OMNIbus Web GUI
installation when the load balanced environment is configured. 

This Interim Fix requires Netcool/OMNIbus Web GUI 8.1.0 FP0002 
to be already installed.


* Special considerations:

1) A Backup of the files will be made and placed in:

[NCHOME]/backup/install_IF/8.1.0.2_IF0001

2) The Netcool/OMNIbus Web GUI need to be stopped, prior to
installing this Interim Fix. 

* Extracting this Interim Fix:
Windows and Non-Windows

1. Copy the file OMNIbusWebGUI_8.1.0.2_IF0001.zip to a 
temporary location on your Netcool/OMNIbus Web GUI server.
2. Unzip the file on each server that you need to install on.

* Installing this Interim Fix:

[Interim Fix] Refers to the directory where this Interim Fix
resides (after you have extracted it from the zip file).

for example: /tmp/OMNIbusWebGUI_8.1.0.2_IF0001

On the Netcool/OMNIbus Web GUI system:

Navigate to the [Interim Fix] directory where the files
were extracted.

Install the Interim Fix with the same user that was used to 
install Netcool/OMNIbus Web GUI 8.1.0 Base release.

Netcool/OMNIbus Web GUI servers
===================================================
1.) Stop the Netcool/OMNIbus Web GUI server(s)
2.) Open a command prompt and change to the unzip directory 
3.) To install Interim Fix 0001,
If you are on a Windows operating system, run the following
command from a prompt: 

install_IF.bat <DASH_HOME>
Note: DASH_HOME is the path to the JazzSM/ui directory

If you are on a Unix operating system, run the following
command from a prompt:

./install_IF.sh <DASH_HOME>

NOTE: There is no GUI mode for this Interim Fix. 

The above script execution should complete with the
statement "BUILD SUCCESSFUL"
4.) Make sure you clear your browser cache
5.) Start the Netcool/OMNIbus Web GUI server


* Verify the installation

1) Login to OMNIbus Web GUI and verify the version on the welcome page.
It must say 'Version 8.1.0.2_IF0001'


* Uninstalling this Interim Fix

Please note: This Interim Fix must be uninstalled on each OMNIbus Web GUI
server(s)


On the Netcool/OMNIbus Web GUI system:

Uninstall the Interim Fix with the same user that was used to 
install OMNIbus Web GUI 8.1.0.2 IF0001.


1.) Stop the OMNIbus Web GUI server(s)
2.) Navigate to the backup directory where the files were backed up
by 8.1.0.2 IF0001 installer:
[NCHOME]/backup/install_IF/8.1.0.2_IF0001

For example: /opt/IBM/netcool/omnibus_webgui/backup/install_IF/8.1.0.2_IF0001

3.) Issue the following command:
./uninstall_IF.sh <DASH_HOME> (Unix) 
uninstall_IF.bat <DASH_HOME> (Windows)

Where DASH_HOME is the directory where DASH is installed.
For example: ./uninstall.sh /opt/IBM/JazzSM/ui

During the uninstall the user is asked to confirm they wish to
uninstall. Type y for yes and n for no (followed by the Enter
key!)

The above script execution should complete with the statement
"BUILD SUCCESSFUL"
4.) Make sure you clear your browser cache
5.) Start the OMNIbus Web GUI Server

* Verify Uninstallation

1) Login to OMNIbus Web GUI and verify the version on the welcome page.
It must say 'Version 8.1.0.2'

===================================================
Additional Information

The installation log can be found under
[USER_HOME]/WebGUI_uninstall_8.1.0.2_IF0001.log
