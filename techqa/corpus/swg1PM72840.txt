APAR STATUS
 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  After installing CF01 customer are no longer able to select
   folder or document from the IBM Connections server via CMIS
   using the federated documents feature in WCM richtext Editor
   or PZN federated documents collection.
   
   The following can be found in the log:
   EJCBD0021E: The URI
   [https://<host>/connections/files/basic/cmis/my/servicedoc?
   filter=cmis%3AbaseTypeId%2Ccmis%3Aname%2Ccmis%3AlastModification
   Date%2Ccm
   is%3AcreationDate%2Ccmis%3AcontentStreamLength%2Ccmis
   %3AcontentStreamMimeType%2Ccmis%3AlastModifiedBy%2Ccmis%3Acreate
   dBy&hpaa.
   slotid=w3connections_shared&hpaa.authtype=http-
   basic&hpaa.sso=false] and parameters []] could not be
   processed:
   
   [EJCBD0022E: Access to the URL
   [https://<host>/connections/files/basic/cmis/my/servicedoc?
   filter=cmis%3AbaseTypeId%2Ccmis%3Aname%2Ccmis%3AlastModification
   
   Date%2Ccmis%3AcreationDate%2Ccmis%3AcontentStreamLength%2Ccmis
   %3AcontentStreamMimeType%2Ccmis%3AlastModifiedBy%2Ccmis%3Acreate
   dBy]
   resulted in status code [404] and the message [<HTML>
   <TITLE>404 - Not Found</TITLE><BODY><h1>
   404 Not Found</h1></BODY></HTML>].]].
