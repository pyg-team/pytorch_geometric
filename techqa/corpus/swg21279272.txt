QUESTION
How does the logging level set in the Configuration Document and the notes.ini file parameter "log_mailrouting=" work in Lotus DominoÂ®? Which value takes precedence?

ANSWER
The notes.ini parameter, "log_mailrouting=", and the logging level in the Configuration Document (Router/SMTP --> Advanced --> Controls) have the same functionality.
The parameter, "log_mailrouting=", may be set to 10 (minimal), default 20 (normal), 30 (informational), 40 (Verbose).

What happens if this parameter is set along with the logging level?

A change on the logging level in the Configuration Document will NOT change the value set in the "log_mailrouting=" parameter. The notes.ini parameter takes precedence over the logging level in the Configuration Document.

Therefore, if the notes.ini parameter is set, that is the value logged by Domino.
