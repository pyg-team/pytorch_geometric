APAR STATUS
 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  MQ Error Code 2010 is visible when sharing conversation is used
   even if MXMSGML of qmgr's channel is greater than mq-qm's
   Maximum Message Size. The same condition works when non-sharing
   conversation is used.
   
   
    
   
   

LOCAL FIX
 *  Use the Best practices that defines the following conditions.
   
   MAXMSGL attribute of qmgr =< Channel =< Queue =< mq-qm's
   Maximum Message Size
   
   
    
   
   

PROBLEM SUMMARY
 *  MQRC 2010 will be reported, when the setting of the maximum size
   of MQ message in DataPower is bigger than the setting of MQ
   Server.
   
   
    
   
   

PROBLEM CONCLUSION
 *  Fix is available in 7.0.0.17, 7.1.0.14, 7.2.0.11, 7.5.0.5,
   7.5.1.4 and 7.5.2.2

   http://www-01.ibm.com/support/docview.wss?uid=swg21237631 [http://www-01.ibm.com/support/docview.wss?uid=swg21237631]
