LOCAL FIX
 *  JazzSm 1.1.0.3 supports 'Windows Server 2012 R2 Standard'
   
   Please perform the following steps:
   
   1) Exit from Launchpad, go to
   <JazzSM_image>/PrereqScanner/Windows
   
   2) In file 'FAS_01010002.cfg' replace 'intel.cpu=2.93GHz' with
   'intel.cpu=[sev:WARN]2.93GHz'
   
   3) In latest files 'ODP_01010002.cfg', 'DSH_03010002.cfg',
   'FRS_01010002', 'FAS_01010002','ESS_01010002',
    replace 'OS Version=regex{Microsoft Windows Server............'
    with 'OS Version=[sev:WARN]regex{Microsoft Windows
   Server...........'
   
   this will add warning for OS version and intel.cpu properties
   and install proceeds to next step.
   
   4) Invoke Launchpad and start the installation again.
