Title: IBM Security Bulletin: A security vulnerability has been identified in IBM DB2, IBM SPSS Modeler, and IBM WebSphere MQ shipped with IBM Predictive Maintenance and Quality (CVE-2015-0159). - United States

Text:
 SECURITY BULLETIN

SUMMARY
 IBM DB2, IBM SPSS Modeler and IBM WebSphere MQ are shipped as components of IBM Predictive Maintenance and Quality. Information about a security vulnerability affecting IBM DB2, IBM SPSS Modeler and IBM WebSphere MQ has been published in their respective security bulletins. 

VULNERABILITY DETAILS
Please consult the Security Bulletin: Vulnerabilities in GSKit affect IBM® DB2® (CVE-2015-0138, CVE-2015-0159 and CVE-2014-6221) [https://www-304.ibm.com/support/docview.wss?uid=swg21699543] for vulnerability details and information about fixes.

Please consult the Security Bulletin: Vulnerabilities in GSKit affect IBM SPSS Modeler (CVE-2015-0159, CVE-2015-0138, CVE-2014-6221) [https://www-304.ibm.com/support/docview.wss?uid=swg21699585] for vulnerability details and information about fixes.

Please consult the Security Bulletin: Vulnerabilities in GSKit affect IBM WebSphere MQ (CVE-2015-0159, CVE-2015-0138 and CVE-2014-6221) [https://www-304.ibm.com/support/docview.wss?uid=swg21699055] for vulnerability details and information about fixes.
