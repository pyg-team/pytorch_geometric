Title: IBM Connections Support Statement for Microsoft Internet Explorer 11 (IE11) - United States

Text:
IE11; IE 11; internet explorer; microsoft internet explorer; IBM connections; Lotus connections TECHNOTE (FAQ)

QUESTION
Microsoft has announced the end of support for all versions of Internet Explorer prior to Internet Explorer 11 effective January 12, 2016: 

http://www.microsoft.com/en-us/WindowsForBusiness/End-of-IE-support [www.microsoft.com/en-us/WindowsForBusiness/End-of-IE-support] 

How is IBM Connections on-premises support for Internet Explorer affected by this announcement?


ANSWER
Internet Explorer 11 is tested and supported with Connections 5.0* and Connections 5.5. There are currently no plans to backport support for Internet Explorer 11 to earlier Connections releases. 

For a list of supported browsers per release, please refer to IBM Connections System Requirements: http://www.ibm.com/support/docview.wss?uid=swg27012786 [http://www.ibm.com/support/docview.wss?uid=swg27012786]

For further specifics on support handling of untested versions of browsers (non-IBM products), please refer to the IBM Connections Support Statement for each release:
http://www.ibm.com/support/docview.wss?uid=swg27019736 [http://www.ibm.com/support/docview.wss?uid=swg27019736]
