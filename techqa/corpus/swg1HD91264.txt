APAR STATUS
 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  Export name which hooked script is displayed in the user
   defined menu.
   Scenario :
   1. Start Export tool
   2. Create new export
   3. Set any query and any script
   4. Strat Script maintenance tool
   5. Open "User Defined" panel
   ==> Export name is automatically added in the user deinfed
   command. Also Export name is displayed in the user defined
   menu on the editor
   Note: Even if the user removed this export definition on the
   export tool, the user cannot delete it from the user defined
   command.
   Expectation:
   export name should not be appeared on the user defined
   command.
   If it is impossible, the user should be able to delete
   unnecessary user defined command which was automatically
   added by smarteam.
