ERROR DESCRIPTION
 *  Problem Description :
   
    The MIB of v4.0.2.0 indicates that
    dpStatusSystemUsageTableTable is obsoleted.
   
    dpStatusSystemUsageTableTable OBJECT-TYPE
      SYNTAX      SEQUENCE OF DPStatusSystemUsageTableEntry
      MAX-ACCESS  not-accessible
      STATUS      obsolete
      DESCRIPTION
          "Component level system usage.
          Note: this is obsoleted and is replaced by
    dpStatusSystemUsage2Table as of release 4.0.1"
   
      ::= { dpStatus 201 }
   
    ITCAM Agent for WebSphere DataPower Appliance v6.3 requests
    component level system usage from OID
    1.3.6.1.4.1.14685.3.1.201.1.1-6.
    This is obsoleted starting from firmware v4.0.1 released June
    30,2011.  From firmware v4.0.1, there is no data returned for
    these OIDs from the DataPower device.
   
    Because of this..
    the "Performance Object Status" workspace showed Error Code "NO
    INSTANCE RETURNED" of the SystemUsageTable attribute group.
   
   Approver : ZW
