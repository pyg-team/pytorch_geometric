Title: IBM Attributes that can be inserted using adduser.bat or adduser.sh - United States

Text:
TPAEINST; TPAEADMIN TECHNOTE (FAQ)

QUESTION
 adduser.bat (Windows) or adduser.sh (Unix) is a command line utility located in (default) C:\IBM\SMP\maximo\tools\maximo\ (Windows) /opt/IBM/SMP/maximo/tools/maximo/ (Unix). It can be used to add users to the MAXUSER table from a formatted list saved as an xml file. The example attributes in AddUser.xml are just a few of the attributes that can be added. 

ANSWER
The syntax for adduser is


adduser[.bat][.sh] [-d(logfile dir)] [-i(filename)] [-o(filename)] 
-d(logfile dir) If using the -l parameter, will output log file to the specified directory. If not specified, outputs to the Maximo tools log directory, which normally is C:\IBM\SMP\maximo\tools\maximo\log\ (Windows) /opt/IBM/SMP/maximo/tools/maximo/log (Unix) . -i(filename) File name specifying the users to be added. If not specified, the file name is assumed to be AddUser.xml in the C:\IBM\SMP\maximo\tools\maximo directory (Windows) /opt/IBM/SMP/maximo/tools/maximo/ (Unix). -o(filename) If using the -l parameter, the file name for the log. If not specified, log file name is "ConfigDB" + timestamp + ".log". 
Each line in the XML file users for input is formatted following the example below. 
 * 
 * <USER username="LIBERI" password="liberi" firstname="Diane" lastname="Liberi" pagepin="maximo57@hotmail.com" />



You must specify all of the required attributes plus any optional attributes desired. 

The passwordcheck attribute need not be specified; it will be set equal to the value of the password attribute
