Title: IBM The File Event is missing from the Audit Definitions of the Folder class in Administration Console for Content Platform Engine (ACCE) version 5.2.1.0 - United States

Text:
 TECHNOTE (TROUBLESHOOTING)

PROBLEM(ABSTRACT)
 The File Event is missing from the Audit Definitions of the Folder class in Administration Console for Content Platform Engine (ACCE) version 5.2.1.0 

SYMPTOM
Unable to either audit or subscribe the File Event. The "File Event" is triggered by the creation of a ReferentialContainmentRelationship (RCR) object which is either filing an object in a folder or creating a subfolder. 


CAUSE
The option to choose the File Event is missing from the Audit Definitions of the Folder class in Administration Console for Content Platform Engine (ACCE) version 5.2.1.0.

ENVIRONMENT
IBM FileNet Content Platform Engine 5.2.1.0

DIAGNOSING THE PROBLEM
In the Administration Console for Content Platform Engine (ACCE) administration tool.


 1. Navigate down to <P8 Domain> / Object Store / <object store> / Data Design / Classes / Folder
    
    
 2. On the Folder Class Definition screen, change to the Audit Definitions tab
    
    
 3. Click on the New button.
    
    
 4. The Event pull down is missing the File Event choice

RESOLVING THE PROBLEM
The File Event issue is fixed in the 5.2.1.2-P8CPE-ALL-FP002 fix pack.
