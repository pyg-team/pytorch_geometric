Title: IBM PJ39254: CONTENT ENGINE INSTALLER IGNORES OPTIONS AND INSTALLS ALL SOFTWARE WHEN SELECTING .NET CLIENT ONLY INSTALL. - United States

Text:
 SUBSCRIBE
You can track all active APARs for this component.



APAR STATUS
 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  When selecting .Net client only for install, the Content Engine
   (CE) installer ignores the option and installs all files and
   desktop administrative icons.
   
   
   Steps to Reproduce:
   1 Install CE 4.5.1 GA for .net only
   2 Install CE 4.5.1.6 for .net only
   3 This occurs with the GUI and silent installers.
   
   Actual Results:
   All admin icons and all CE files installed on the client.
   
   Expected Results:
   Only .net client should install when only .net client is
   selected.
   
   
    
   
   

LOCAL FIX

PROBLEM SUMMARY
 *  ****************************************************************
   * USERS AFFECTED:                                              *
   * Users upgrading to a fixpack below 4.5.1.7-P8CE-FP007 with   *
   * the component chosen as IBM FileNet .NET Clients (excluding  *
   * FileNet Enterprise Manager)                                  *
   ****************************************************************
   * PROBLEM DESCRIPTION:                                         *
   * Users upgrading to a fixpack below 4.5.1.7-P8CE-FP007 with   *
   * the component chosen as IBM FileNet .NET Clients (excluding  *
   * FileNet Enterprise Manager), would result in all the         *
   * components getting installed (Content Engine Server, .Net    *
   * Clients, FileNet Enterprise Manager,Tools). This was due to  *
   * an incorrect assignment of the set of components getting     *
   * installed.                                                   *
   ****************************************************************
   * RECOMMENDATION:                                              *
   * Upgrade to product version 4.5.1.7-P8CE-FP007 or later       *
   ****************************************************************
   
   
    
   
   

PROBLEM CONCLUSION
 *  Users upgrading to a fixpack below 4.5.1.7-P8CE-FP007 with the
   component chosen as IBM FileNet .NET Clients (excluding FileNet
   Enterprise Manager), would result in all the components getting
   installed (Content Engine Server, .Net Clients, FileNet
   Enterprise Manager,Tools). This was due to an incorrect
   assignment of the set of components getting installed.
   
   In 4.5.1.7-P8CE-FP007 and higher, this problem has been
   corrected.
