APAR STATUS
 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  Problem Abstract:
   Inconsistent Request Times and Rates summary Cognos reports
   
   ---------------------------------
   Users Affected:
   All users of ITCAM Cognos Reports
   
   ---------------------------------
   Problem Description:
   Inconsistentency in the ITCAM for AD 7.2 Cognos Model; the name
   of the insertable object is different for the Hourly
   Summarization. For this it is called:
   [Request Times and Rate]
   For all other types (Detailed, Daily,etc)  it is called:
   [Request Times and Rates]
   - note the addition of 's' on Rates
   
   A change is required to the name of the 'Hourly Request Times
   and Rate' object in the Consolidated and Query Views to allow
   the creation of full summary reports including Hourly data
   
   ---------------------------------
   Keyword/Symptom:
   INCORR
   
   ---------------------------------
   Component ID:
   5724Y92WD
   
   ---------------------------------
   Severity:
   2
   
   ---------------------------------
   Local Fix/Workaround:
   (1) Create two reports - one for Hourly data and one for all the
   other query types
   (2) Use the Cognos FM tool to manually correct the name of the
   object
   
   
    
   
   

LOCAL FIX
 *  (1) Create two reports - one for Hourly data and one for all the
   other query types
   (2) Use the Cognos FM tool to manually correct the name of the
   object
