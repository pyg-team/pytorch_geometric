Title: IBM Security Bulletin: Information regarding security vulnerability in IBM SDK for Java that is shipped with IBM WebSphere Application Server and addressed by Oracle CPU October 2013 - United States

Text:
 SECURITY BULLETIN

SUMMARY
 Multiple security vulnerabilities exist in the IBM SDK for Java that is shipped with IBM WebSphere Application Server and is included in the products that are listed in this document. 

VULNERABILITY DETAILS
WebSphere Lombardi Edition shipped with a version of IBM WebSphere Application Server, which is shipped with an IBM SDK for Java that is based on the Oracle Java Development Kit (JDK). Oracle has released the October 2013 critical patch updates (CPU), which contain security vulnerability fixes and the IBM SDK for Java has been updated to incorporate the critical patch updates.
The following vulnerabilities are applicable to WebSphere Application Server and for WebSphere Lombardi Edition

Vulnerability fixes for other advisories are included in the IBM SDK for Java, but IBM WebSphere Application Server and WebSphere Lombardi Edition are not vulnerable to them. You need to evaluate your own code to determine if you are vulnerable. Refer to the Reference section for more information on the advisories that are not applicable to WebSphere Application Server.

AFFECTED PRODUCTS AND VERSIONS
WebSphere Lombardi Edition 7.1.x, 7.2.0

REMEDIATION/FIXES
Install WebSphere Application Server interim fixes as appropriate for your current WebSphere Application Server version as described in Security Bulletin: Multiple vulnerabilities in current IBM SDK for Java for WebSphere Application Server October 2013 CPU [http://www.ibm.com/support/docview.wss?uid=swg21655990].
