Title: IBM Getting error message when browsing an object store in FileNet Workplace XT - United States

Text:
Workplace XT Addons TECHNOTE (TROUBLESHOOTING)

PROBLEM(ABSTRACT)
 Select an object store to browse in FileNet Workplace XT returns an error, The search failed. Ask your administrator to review the error logs and find the possible cause of the search failing.

CAUSE
This issue is caused by some Add-Ons missing on the object store with the problem. The below list is some of the missing Add-Ons:


 * 5.2.1 Workplace (Base, Templates, Access Roles, etc) Extensions 
 * 5.2.1 Workplace XT Extensions 
 * 5.2.1 Stored Search Extensions 

RESOLVING THE PROBLEM
Install the required Add-Ons to the object store from the IBM Administration Console for Content Engine (ACCE): 

 1. Log on to ACCE, right-click on the problem object store and select Install Add-on Features 
 2. Select the necessary Add-Ons from the 'Select the add-on features to install' list for Workplace, Workplace XT and Stored Search then click OK to go through the install wizard.
