Title: IBM IV10135: UNDER CERTAIN CIRCUMSTANCES WORKMANAGER-BASED NAVIGATION CAN CAUSE DEADLOCKS ON BPEDB TABLES - United States

APAR STATUS
 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  Under certain circumstances Workmanager-based navigation can
   cause deadlocks on BPEDB tables.
   .
   When using Workmanager-based navigation, there is a chance of
   getting deadlocks on BPEDB tables. In particular there were
   deadlocks observed that involved the process instance table and
   the process context table of the BPEDB.
   
   
    
   
   

LOCAL FIX

PROBLEM SUMMARY
 *  When WorkManager based navigation is used, in rare situations
   deadlocks can occur during
   navigation of long running business processes.
   This IFix resolves these deadlocks.
   
   
    
   
   

PROBLEM CONCLUSION
 *  Problem fixed
   iFix available on top of V6203
