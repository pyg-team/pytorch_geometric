Title: IBM Running tests with more than 5 users on Rational Performance Tester results in " No License keys available for this 10 user run " error - United States

Text:
RTW; RPT; licenses; Cannot run tests for more than 5 users; Rational virtual tester pack licenses. TECHNOTE (TROUBLESHOOTING)

PROBLEM(ABSTRACT)
 Attempts to run tests for more than 5 virtual users on IBM Rational Performance Tester using IBM Rational Test Workbench licenses results in error "No license keys available for this 10 user run" 

SYMPTOM
You receive the following error message: 

image
[/support/docview.wss?uid=swg21609592&amp;aid=1]

CAUSE
You do not have Virtual Tester Pack licenses which is not included in the Rational Test Workbench suite licenses



DIAGNOSING THE PROBLEM
Verify if the Rational Performance Tester is being run with Rational Test Workbench licenses.



RESOLVING THE PROBLEM
In order to resolve the reported problem, apply the steps below: 

 1. Navigate to IBM Rational License Key Center [https://licensing.subscribenet.com/control/ibmr/index] website
    
    
 2. Download Rational Virtual Tester Pack licenses if already procured
    
    Else, contact your IBM Sales Representative for assistance
    
    
 3. If download is successful, import the following on the Rational License Key Server:
     * IBM Test Work bench Floating Licenses 
     * IBM Rational Virtual Tester Pack Floating Licenses.
