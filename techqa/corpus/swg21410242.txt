QUESTION
 What diagnostic information should be collected to help IBM Support troubleshoot RBM (Role-Based Management) login problems on the Datapower Gateway Appliance? 

ANSWER
The following information describes the documentation needed by IBM support to investigate the problem. Gather as much information listed in this document as possible. 
1. Describe the problem: 

 * 
 *   * Is this an existing environment or a new setup? 
    * Are there any recent changes that may have caused the problem to occur?
   
   

2. Enable RBM Debug Logging 
 * From CLI:
 * co; 
   loglevel debug 
   set var://system/map/debug 3 
   exit 
   
 * From WebGUI:
 * Control Panel -> Troubleshooting -> Logging 
   Log Level = Debug 
   Enable RBM Debug logging = ON 
   Click 'Set Log Level'
