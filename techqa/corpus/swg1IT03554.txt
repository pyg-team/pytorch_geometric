ERROR DESCRIPTION
 *  User scenario:
   
   1. Deploy a virtual system pattern with a schedule deployment
      "Run until" of 1 day
   2. Deploy a virtual system pattern with a schedule deployment
      of "Run indefinitely"
   3. Try to stop the virtual system with the run time set
   4. Try to stop the virtual system with no run time set
   
   You will notice that the stop task for the virtual system with
   the run time set is hung.
   
   
    
   
   

LOCAL FIX

PROBLEM SUMMARY
 *  The issue is caused by a task that is called before the instance
   status changes to the "Stopped" state has a dependency on a
   delete task that is scheduled to run in the future.
   
   
    
   
   

PROBLEM CONCLUSION
 *  The solution was to remove the dependency on scheduled delete
   tasks.
