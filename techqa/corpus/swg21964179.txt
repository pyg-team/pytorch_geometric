 TECHNOTE (FAQ)

QUESTION
 What link option should be used for AIX Micro Focus COBOL stored procedure? 

ANSWER
The COBOL routine will be loaded by the DB2 server process, so you need to pass -y option to compile to Self-contained Callable Shared Object. 

For more information on MF COBOL embedded applications, please refer the documentation from Micro Focus.
http://documentation.microfocus.com/help/index.jsp [http://documentation.microfocus.com/help/index.jsp]
