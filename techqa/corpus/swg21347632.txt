Title: IBM Error when trying to start the Controller batch process - United States

Text:
 TECHNOTE (TROUBLESHOOTING)

PROBLEM(ABSTRACT)
 Administrator logs onto Controller application server, and launches IBM Cognos 8 Controller Configuration. 

Administrator opens 'Batch Services' section, and highlights Process 'Controller Batch Process 1'. Administrator clicks on green 'play' button to start service. Administrator receives error message.

CAUSE
The Administrator has mistakenly tried to launch Controller Configuration on the wrong server (e.g. server#1).

ENVIRONMENT
Distributed Controller environment, with some Controller components on server #1, and other components on server#2. For example, server#1 = Cognos gateway (and/or reporting server) whilst server#2 = 'main' Controller application server (also known as WSS / COM+ server).

RESOLVING THE PROBLEM
Logon to the correct server (e.g. server#2) and launch the Controller Configuration application from there instead.
