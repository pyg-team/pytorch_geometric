Title: IBM 【セキュリティ情報】 WebSphere Application Server で使用される Apache CXF におけるサービス運用妨害 (DoS) の脆弱性 (CVE-2017-12624) - United States

Text:
 SECURITY BULLETIN

SUMMARY
 WebSphere Application Server で使用されている Apache CXF には、サービス運用妨害 (DoS) の状態にされる脆弱性が存在します。

最新の情報については下記の文書（英語）もご参照ください。
Security Bulletin: Denial of Service in Apache CXF used by WebSphere Application Server (CVE-2017-12624)
https://www.ibm.com/support/docview.wss?uid=swg22013597 [https://www.ibm.com/support/docview.wss?uid=swg22013597] 

VULNERABILITY DETAILS
 

CVEID: CVE-2017-12624 [http://cve.mitre.org/cgi-bin/cvename.cgi?name=CVE-2017-12624]
説明: Apache CXF はサービス運用妨害 (DoS) に対して脆弱です。特別に細工されたメッセージ添付ヘッダーを用いることで、リモートの攻撃者はこの脆弱性を悪用して JAX-WS および JAX-RS サービスの応答を停止できる可能性があります。
CVSS 基本スコア: 5.3
CVSS 一時スコア: 現在の一時スコアについては
https://exchange.xforce.ibmcloud.com/vulnerabilities/135095 [https://exchange.xforce.ibmcloud.com/vulnerabilities/135095] をご参照ください。
CVSS 環境スコア*: 未定義
CVSS Vector: (CVSS:3.0/AV:N/AC:L/PR:N/UI:N/S:U/C:N/I:N/A:L) [/support/docview.wss?uid=swg22014471&amp;aid=1] AFFECTED PRODUCTS AND VERSIONS
この脆弱性は、WebSphere Application Server の下記のバージョンおよびリリースに影響します。 

 * Liberty 
 * バージョン 9.0

REMEDIATION/FIXES
各製品に対応した 各APAR を含む個別修正モジュール、Fix Pack、または PTF をできる限り早く適用いただくことをお薦めします。 
WebSphere Application Server Liberty で JAX-WS を使用している場合: 

対応策  * Fix Pack 17.0.0.3 以降へアップデートし、個別修正モジュール PI92494 [https://www.ibm.com/support/docview.wss?uid=swg24044631] を適用します。

-- または --  * Fix Pack 18.0.0.1 以降へアップデートします。
   

※Fix Pack 18.0.0.1 は、2018年3月16日にリリース予定です。 (2018/03/12 時点) 

WebSphere Application Server traditional で JAX-RS を使用している場合: バージョン 対応策 V9.0.0.0 ～ 9.0.0.6  * Fix Pack 9.0.0.4 以降へアップデートし、個別修正モジュール PI92492 [https://www.ibm.com/support/docview.wss?uid=swg24044541] を適用します。

-- または --  * Fix Pack 9.0.0.7 以降へアップデートします。
   

※Fix Pack 9.0.0.7 は、2018年3月16日にリリース予定です。 (2018/03/12 時点) [/support/docview.wss?uid=swg22014471&amp;aid=1] GET NOTIFIED ABOUT FUTURE SECURITY BULLETINS
