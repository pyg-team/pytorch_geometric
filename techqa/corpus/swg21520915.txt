PROBLEM(ABSTRACT)
 LAPP013I An invalid keyword value was found 

SYMPTOM
LAPP013I 
An invalid keyword value was found 
Invalid usage in this statement

CAUSE
SYSOPTS string has to be enclosed by double quotes.

RESOLVING THE PROBLEM
Code SYSOPTS="aaa 'bbb' ccc"
