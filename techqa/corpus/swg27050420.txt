 1.  End of support notification: IBM Security SiteProtector System 3.1.1 [http://www.ibm.com/support/docview.wss?uid=swg3g491809c98841g92&acss=danl_4723_web]

 2.  
    End of support notification: IBM Security SiteProtector System 3.0.0 [http://www.ibm.com/support/docview.wss?uid=swg3a133332k87727q77&acss=danl_4723_web] 
    IBM Security SiteProtector System 3.0.0 will reach End of Support on 2022-12-30. For more information about this product, please visit the IBM software support lifecycle site. 
 3.  Security Bulletin: IBM Security Network Intrusion Prevention System is affected by a vulnerability in OpenSSL (CVE-2016-2183) [http://www.ibm.com/support/docview.wss?uid=swg22007331&acss=danl_4723_web] 
    IBM Security Network Intrusion Prevention Systems has addressed the following vulnerability in OpenSSL. The vulnerability is known as the SWEET32 Birthday attack (CVE-2016-2183). 
 4.  Security Bulletin: IBM Security Network Protection is affected by a vulnerability in coreutils (util-linux) [http://www.ibm.com/support/docview.wss?uid=swg22003046&acss=danl_4723_web] 
    A security vulnerability has been discovered in coreutils (util-linux), which is used by IBM Security Network Protection. 
 5.  Security Bulletin: Multiple vulnerabilities in IBM Java Runtime affect IBM Security Network Protection [http://www.ibm.com/support/docview.wss?uid=swg22003341&acss=danl_4723_web] 
    There are multiple vulnerabilities in IBM® Runtime Environment Java™ Version 7 used by IBM Security Network Protection. IBM Security Network Protection has addressed the applicable CVEs. 
 6.  Security Bulletin: IBM Security Network Protection is affected by Vulnerabilities in GNU Bash [http://www.ibm.com/support/docview.wss?uid=swg22003045&acss=danl_4723_web] 
    Security vulnerabilities have been discovered in GNU Bash, which is used by IBM Security Network Protection. IBM Security Network Protection has addressed the applicable CVEs. 
 7.  Security Bulletin: Multiple vulnerabilities in IBM Java SDK affect IBM Security Network Protection [http://www.ibm.com/support/docview.wss?uid=swg22007918&acss=danl_4723_web] 
    There are multiple vulnerabilities in IBM® SDK Java™ Technology Edition, Version 7 Service Refresh 10 Fix Pack used by IBM Security Network Protection. These issues were disclosed as part of the IBM Java SDK updates in July 2017. 
 8.  Security Bulletin: IBM Security Network Protection is affected by a vulnerability in glibc [http://www.ibm.com/support/docview.wss?uid=swg22005764&acss=danl_4723_web] 
    IBM Security Network Protection has addressed the following vulnerability: A security vulnerability has been discovered in glibc.
