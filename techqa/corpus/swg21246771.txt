PROBLEM(ABSTRACT)
 In IBM® Rational® Software Architect (RSA) v6.0.1.1 on Linux®, the following error message appears when you select Window > Preferences > Java > Code Review.
Internal Error: An SWT Error has occurred. You are recommended to exit the workbench. Subsequent errors may happen and may terminate the workbench without warning. See .log file for more details. Do you want to exit the workbench?

(SWT, or Standard Widget Toolkit, is an Eclipse toolkit for Java™ developers that defines a common, portable, user interface API that uses the native widgets of the underlying operating system.)
