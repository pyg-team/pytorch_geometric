ERROR DESCRIPTION
 *  And analyze job of a change to a table with an associated
   auxiliary table may end with the following error if the
   auxiliary table was created with a comment:
   ADB8052E A change was found for schema.aux_table_name but no
   base definition was found.
   
   
    
   
   

LOCAL FIX
 *  Process the change outside of Change Management.
   
   
    
   
   

PROBLEM SUMMARY
 *  ****************************************************************
   * USERS AFFECTED: Users of DB2 Administration Tool for         *
   *                 z/OS who use ALT to change the table         *
   *                 space CCSID from EBCDIC to UNICODE.          *
   *                 The table space has a table which has        *
   *                 a LOB column and the related auxiliary       *
   *                 table has a comment.                         *
   ****************************************************************
   * PROBLEM DESCRIPTION: Analyzing a change which has a COMMENT  *
   *                      ON statement for an auxiliary table     *
   *                      failed with error ADB8052E /            *
   *                      msgadb8052e.                            *
   ****************************************************************
   * RECOMMENDATION:                                              *
   ****************************************************************
   When performing the ALT command to change the table space
   CCSID from EBCDIC to UNICODE, the table has an auxiliary
   table which has COMMENT. The registered change is analyzed
   and fails with error ADB8052E / msgadb8052e:
   
   ADB8052E A change was found for xxx.xxxx but no base
   definition was
   
   and information message ADB8057I / msgadb8057i:
   
   ADB8057I Object details at the time of error:
    Operation : 50
    Row type : TB
    Object qualifier: DB
    Object name : xxxxx
   
   Procedure : merge_files
    ADBCCME - Error(s) found. Please review output
    ADBC002E - Severe error. Merge terminates
