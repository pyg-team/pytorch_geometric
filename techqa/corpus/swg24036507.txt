Title: IBM Operational Decision Manager V8.5.1 Mod Pack - United States

 DOWNLOADABLE FILES

ABSTRACT
 This cumulative refresh pack updates Operational Decision Manager V8.5 to V8.5.1. 

DOWNLOAD DESCRIPTION
This refresh pack is entitled. Only customers who have purchased Operational Decision Manager V8.5 can download this refresh pack.


 
Find out what's new in this release at What's new in IBM Operational Decision Manager [http://www.ibm.com/support/docview.wss?uid=swg21992623].

How to install the mod pack
If you are not installing both components, skip the instructions regarding Decision Center or Decision Server. However, if you already have both components installed, you must update both with the mod pack. Otherwise you receive an error during the update installation.


 1.  Download 8.5.0-WS-ODM_DC-<OS>-RP001.zip. Change <OS> to match your platform: AIX, HPUX, LIN_PPC, LIN_S390, LIN_X86, OS400, SOL_SPARC, SOL_X86, or WIN. 
 2.  Extract the archive to a new folder named WDC. 
 3.  Download 8.5.0-WS-ODM_DS-<OS>-RP001.zip. Change <OS> to match your platform: AIX, HPUX, LIN_PPC, LIN_S390, LIN_X86, OS400, SOL_SPARC, SOL_X86, or WIN. 
 4.  Extract the archive to a new folder named WDS. 
 5.  Launch IBM Installation Manager. 
 6.  In the toolbar menu, open File > Preferences and do the following: 
 7.  Select repositories. 
 8.  Click Add Repository. 
 9.  Browse to the extracted folder WDC\updates. 
 10. Select the file repository.config. 
 11. Click Add Repository. 
 12. Browse to the extracted folder WDS\updates. 
 13. Select the file repository.config. 
 14. Make sure both repositories are checked in the repository list Apply, and click OK to close. 
 15. Click Update to update your installation. 
 16. Follow the instructions in the installation manager to complete the installation.
