Title: IBM Current Date function for DB2, SQL Server and Oracle - United States

Text:
TPAEMSSQL; TPAEDB2; getdate; TPAEDATABASE; TPAEORACLE TECHNOTE (FAQ)

QUESTION
 What would be the equivalent of MSSQL getdate() function for Maximo running on DB2 or Oracle? 

ANSWER
The getdate() MSSQL function can be replaced by value from DB2 CURRENT DATE special register. On Oracle, sysdate is used to retrieve the current date. 

EXAMPLE : MSSQL and transdate <= ( getdate() - 35) 

 DB2 and transdate <= ( CURRENT DATE - 35 DAYS) 

 Oracle and transdate <= ( sysdate - 35)
