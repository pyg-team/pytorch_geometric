Title: IBM Security Bulletin: A security vulnerability has been identified in IBM WebSphere Application Server shipped with IBM Endpoint Manager for Remote Control (CVE-2015-1283) - United States

Text:
 SECURITY BULLETIN

SUMMARY
 IBM WebSphere Application Server is shipped as a component of IBM Endpoint Manager for Remote Control. Information about a security vulnerability affecting IBM HTTP Server used by IBM WebSphere Application Server has been published in a security bulletin. 

VULNERABILITY DETAILS
Please consult the security bulletin 'Denial of service may affect IBM HTTP Server (CVE-2015-1283)' [http://www-01.ibm.com/support/docview.wss?uid=swg21964428] for vulnerability details and information about fixes.

AFFECTED PRODUCTS AND VERSIONS
 

Principal Product and Version(s) Affected Supporting Product and Version IBM Endpoint Manager for Remote Control version 9.1.0 and 9.0.1 IBM WebSphere Application Server version 8.5.5.5 IBM Tivoli Endpoint Manager for Remote Control 8.2.1 IBM WebSphere Application Server 7.0.0.37 REMEDIATION/FIXES
Please consult the security bulletin 'Denial of service may affect IBM HTTP Server (CVE-2015-1283)' [http://www-01.ibm.com/support/docview.wss?uid=swg21964428] for vulnerability details and information about fixes.

WORKAROUNDS AND MITIGATIONS
None
