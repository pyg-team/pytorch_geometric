 SECURITY BULLETIN

SUMMARY
 IBM HTTP Server and IBM Tivoli/Security Directory Server are shipped as components of WebSphere Commerce. Information about a security vulnerability affecting IBM HTTP Server and IBM Tivoli/Security Directory Server have been published in two security bulletins. 

VULNERABILITY DETAILS
Please consult the security bulletins TLS padding vulnerability affects IBM HTTP Server (CVE-2014-8730) [http://www-01.ibm.com/support/docview.wss?uid=swg21692502] and TLS padding vulnerability affects IBM Tivoli/Security Directory Server (CVE-2014-8730) [http://www-01.ibm.com/support/docview.wss?uid=swg21692906] for vulnerability details and information about fixes.

AFFECTED PRODUCTS AND VERSIONS
 

Principal Product and Version(s) Affected Supporting Product and Version WebSphere Commerce V6.0 and V7.0 IBM HTTP Server all versions and releases.

IBM Tivoli/Security Directory Server versions 6.0, 6.1, 6.2, 6.3 and 6.3.1. GET NOTIFIED ABOUT FUTURE SECURITY BULLETINS
