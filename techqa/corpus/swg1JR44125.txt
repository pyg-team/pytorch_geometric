ERROR DESCRIPTION
 *  When the table action is set to Truncate and user defined SQL is
   provided as "Truncate table <TableName>", the ODBC Connector job
   is failing with the following error while connecting to the DB2
   Database.
   ODBC Info:  SQLSTATE = HY000: Native Error Code = -428: Msg =
   [IBM(DataDirect OEM)][ODBC DB2 Wire Protocol driver][UDB DB2 for
   Windows, UNIX, and Linux]Unknown error: SQLCODE -428
   
   
    
   
   

LOCAL FIX
 *  Use the DB2 Connector stage.
   
   
    
   
   

PROBLEM SUMMARY
 *  ODBC Connector job is failing while executing the truncate table
   action and connecting to DB2 Database.
