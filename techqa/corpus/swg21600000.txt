Title: IBM myfilegateway UI displays the blank page when user authentication is done via SSO. - United States

Text:
 TECHNOTE (TROUBLESHOOTING)

PROBLEM(ABSTRACT)
 myfilegateway UI displays the blank page when user authentication is done via SSO. SSO authentication is done via Apache Reverse proxy. 

SYMPTOM
Issue with Internet Explorer only. No issues with firefox.


RESOLVING THE PROBLEM
Set the following settings in Internet Explorer: 

Tools -> Internet options -> Advanced -> Under Security, check the option "Allow active content to run in files on my computer.
