Title: IBM Security Bulletin: Security vulnerabilities have been identified in IBM Websphere Message Broker and IBM Integration Bus shipped with IBM Predictive Maintenance and Quality (CVE-2014-3566, CVE-2014-3568). - United States

Text:
 SECURITY BULLETIN

SUMMARY
 IBM Websphere Message Broker and IBM Integration Bus is shipped as a component of IBM Predictive Maintenance and Quality. Information about security vulnerabilities affecting IBM Websphere Message Broker and IBM Integration Bus has been published in a security bulletin. 

VULNERABILITY DETAILS
Please consult the security bulletin IBM Websphere Message Broker and IBM Integration Bus are affected by SSLv3 Vulnerability (CVE-2014-3566 and CVE-ID:VE-2014-3568) [http://www-01.ibm.com/support/docview.wss?uid=swg21687678&myns=swgws&mynp=OCSSNQK6&mynp=OCSSHL3F&mynp=OCSSKM8N&mync=E] for vulnerability details and information about fixes.
