ERROR DESCRIPTION
 *  When customer sets Usebasicnotes=1 to change Notes 8.5 Standard
   to Notes 8.5
   Basic there is an issue attending instant messaging meetings for
   invited people
   when Windows XP Slipstream and Internet explorer are used.
   
   -  Create an instant message meeting from File, Instant
   Messaging. Start
   Instant Chat Meeting.
   -  Add some people to invite to the meeting
   - Meeting reservations mostly show that Internet Explorer 7 will
   not load. In
   some tests it was shown that it loaded but then hanged. If
   Firefox is used the
   browser loads to start the meeting.
   
   
    
   
   

LOCAL FIX
 *  If Windows XP SP2 is used Internet Explorer 7 loads with a
   prompt to accept a certificate.
   
   
   Adjusting Internet Explorer Security -
   
   Internet Explorer.  From the menu, select Internet Options ->
   Advanced  Tab ->
   Security.  Enable these settings.
   
   Allow active content from CDs to run on My Computer
   Allow active content to run in files on My Computer
