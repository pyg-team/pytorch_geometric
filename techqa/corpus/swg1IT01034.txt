APAR STATUS
 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  When creating a new self-sign certificate using Crypto Tools
   with Export Private Key set to OFF (default), WebGUI hangs and
   does not display the successful creation message.  However, the
   key and certificate are successfully generated in the location
   specified.
   
   
    
   
   

LOCAL FIX
 *  Set Export Private Key to ON.
   
   
    
   
   

PROBLEM SUMMARY
 *  Affected is the use of Crypto Tools to generate a key and
   certificate. WebGUI might not show the results and appears to
   not complete.
   
   
    
   
   

PROBLEM CONCLUSION
 *  Fix is available in 6.0.0.6 and 6.0.1.2.

   http://www-01.ibm.com/support/docview.wss?uid=swg21237631 [http://www-01.ibm.com/support/docview.wss?uid=swg21237631]
