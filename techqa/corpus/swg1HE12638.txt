ERROR DESCRIPTION
 *  Exception occurs when stubbing SharePoint documents with links.
   The document was migrated to IBM FileNet P8 but the migration
   Information
   is missing from document link.
   Sometimes in post processing task, while adding ICC columns to
   SharePoint, an exception "a duplicate field was found" is
   thrown. This exception might occur at the time of upgrade or
   migration of library.
   Due to this exception, some ICC columns may not get added to
   the library. Link creation will be successful but the P8
   information will be missing from the document. The following
   exception is logged:
   "Exception occurred after link creation - link document
   will not be removed and task will not be failed in case request
   to delete original document succeeds. If original document
   persists, consult documentation for alternatives."

PROBLEM SUMMARY
 *  Exception occurs when stubbing SharePoint documents with links.
   The document was migrated to IBM FileNet P8 but the migration
   Information is missing from document link.
   Sometimes in post processing task, while adding ICC columns to
   SharePoint an exception "a duplicate field was found" is thrown.
   This exception might occur at the time of upgrade or migration
   of library.
   Due to this exception :
   - some ICC columns may not get added to the library.
   - link creation will be successful but the p8 information will
   be missing from the document
   - exception is logged with error as " Exception occurred after
   link creation - link document will not be removed and task will
   not be failed in case request to delete original document
   succeeds. If original document persists, consult documentation
   for alternatives."
   
   
    
   
   

PROBLEM CONCLUSION
 *  Starting with IBM Content Collector 4.0.1 FP5 release, the issue
    is fixed by better exception handling.
   In case of any exception occurred during archival process,
   archival will be rolled back. Original document will be
   persisted and stub link will not be created.
