PROBLEM CONCLUSION
 *  Problem Summary:
   Unable to create syndicator / subscriber pair when
   store.puma_default.user.fbadefault.filter is not set to uid
   
   
   Detailed Problem Description:
   Unable to create syndicator / subscriber pair when
   store.puma_default.user.fbadefault.filter is not set to uid.
   
   Problem Analysis:
   The code did not use the puma.lookupAttribute when creating the
   syndicator.
   
   Problem Solution:
   Code has been updated to use the puma.lookupAttribute if the
   default lookup fails.
   
   Affected Users:
   All users
