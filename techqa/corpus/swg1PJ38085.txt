Title: IBM PJ38085: Leading spaces in property values do not display correctly in the Workplace XT UI when rendering as read-only text. - United States

Text:
 SUBSCRIBE
You can track all active APARs for this component.



APAR STATUS
 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  Spaces on the document title, folder name and other string
   properties are not displaying in Workplace XT when rendering as
   read-only text or labels.
   
   How long has the problem been occurring (recent changes)? :
   Not recent changes
   
   Does this occur on more than one station / server? : y
   
   Is there a workaround? : n
   
   Can the problem be replicated on an internal system?  Y
   
   Steps to Reproduce:
   1  Create a document or folder with a document title where the
   value has spaces in front, middle or at the end.
   2  Browse to the folder that contains the document or folder.
   
   
   Actual Results:  The document or folder is displayed with the
   spaces stripped and it looks out of place in the sorting order.
   
   Expected Results:  The spaces are displayed in Workplace XT UI
   
   
    
   
   

LOCAL FIX

PROBLEM SUMMARY
 *  ****************************************************************
   * USERS AFFECTED:                                              *
   * Users entering spaces in the front and/or middle of the      *
   * value for string properties including the document title and *
   * folder name.                                                 *
   ****************************************************************
   * PROBLEM DESCRIPTION:                                         *
   * Following behaviors are observed when the user enters spaces *
   * in the front and/or middle for a string property.            *
   *                                                              *
   * 1) Spaces in the front is missing when displayed.            *
   * 2) Only one space is shown even if there are multiple spaces *
   * entered between words.                                       *
   ****************************************************************
   * RECOMMENDATION:                                              *
   * Upgrade to the Workplace XT fix pack 1.1.4.9.                *
   ****************************************************************
   
   
    
   
   

PROBLEM CONCLUSION
 *  The Workplace XT was not handling the spaces properly while
   displaying the values.
   This has been addressed the Workplace XT fix pack 1.1.4.9 to
   display the spaces properly whether they're in the front or
   middle of the value.
   
   
    
   
   

TEMPORARY FIX
