STERLINGPRI TECHNOTE (FAQ)

QUESTION
 Why did I received message CSPA200E on first time connection to a remote NT node using Secure+? (SCI31142) 

ANSWER


Verify you have imported the key the remote sent. Check the Secure+ parmfile for the remote node involved. The Auth.Rmt Key and Sig Rmt Key fields should be filled in. If they are not, import the keys from the remote node and save the parmfile.
