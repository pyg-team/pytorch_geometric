Title: IBM 【セキュリティ情報】 WebSphere MQ Internet Pass-ThruのIBM Java Runtime の複数の脆弱性の影響  (CVE-2015-7575) - United States

Text:
 SECURITY BULLETIN

SUMMARY
 IBM WebSphere MQ Internet Pass-Thruで提供されているIBM® Runtime Environment Java™ Technology Editions バージョン7にセキュリティ脆弱性が存在します。
これらの問題は、2016年1月にIBM Java SDK更新の一部として公開されました。この修正には、"SLOTH"と呼ばれる脆弱性への対応を含みます。 

VULNERABILITY DETAILS
脆弱性の詳細：
CVEID: CVE-2015-7575 [http://cve.mitre.org/cgi-bin/cvename.cgi?name=CVE-2015-7575]
DESCRIPTION: The TLS protocol could allow weaker than expected security caused by a collision attack when using the MD5 hash function for signing a ServerKeyExchange message during a TLS handshake. An attacker could exploit this vulnerability using man-in-the-middle techniques to impersonate a TLS server and obtain credentials. This vulnerability is commonly referred to as “SLOTH”. 
CVSS Base Score: 7.1
CVSS Temporal Score: See https://exchange.xforce.ibmcloud.com/vulnerabilities/109415 [https://exchange.xforce.ibmcloud.com/vulnerabilities/109415] for the current score
CVSS Environmental Score*: Undefined
CVSS Vector: (CVSS:3.0/AV:N/AC:L/PR:N/UI:R/UI:U/C:H/I:L/A:N)

AFFECTED PRODUCTS AND VERSIONS
影響のある製品・バージョン： 

次の製品に影響があります。 

 * IBM WebSphere MQ Internet Pass-Thru 2.1.0.2 and all previous levels

WORKAROUNDS AND MITIGATIONS
回避策/軽減策： 
CVE-2015-7575の回避策/軽減策

Java 7 およびそれ以降を使用されている場合、/jre/lib/security/java.security ファイルのMD5アルゴリズムを無効化することで問題に対処することができます。(両方の手順が必要です。)


 * jdk.certpath.disabledAlgorithms プロパティに MD5 を追加します。

例: jdk.certpath.disabledAlgorithms=MD2, RSA keySize < 1024, MD5 

 * jdk.tls.disabledAlgorithms プロパティに MD5withRSA を追加します。

例:jdk.tls.disabledAlgorithms=SSLv3, RC4, DH keySize < 768, MD5withRSA GET NOTIFIED ABOUT FUTURE SECURITY BULLETINS
