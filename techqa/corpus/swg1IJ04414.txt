APAR STATUS
 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  Fixlets, Tasks and Baselines having a 'Relevance Group' and
   exported using REST API cannot be re-imported.
   
   The 'Relevance Group' can be created for example from the
   console using the "Computers which match the condition below"
   section in the 'Relevance' tab  of Fixlets, Tasks and
    Baselines.
   
   Fixlets, Tasks and Baselines without the 'Relevance are not
   affected by this issue.
   
   The impacted BigFix version are 9.5.7 and higher.
   
   
    
   
   

LOCAL FIX
 *  Export Fixlets, Tasks and Baselines with a 'Relevance Group'
   section using the console
   
   
    
   
   

PROBLEM SUMMARY
 *  Fixed in IBM BigFix 9.5 (fixpack 9)
   General availability Q2 2018 (release schedule subject to
   change).
   
   
    
   
   

PROBLEM CONCLUSION
 *  This is resolved in:
   IBM BigFix 9.5 (fixpack 9)
