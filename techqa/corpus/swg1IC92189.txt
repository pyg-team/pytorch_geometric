APAR STATUS
 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  A refresh certificate action, either manual or initiated by a
   shutdown of the ISAM client, might result in a notification
   that the client certificate has expired with the message 'The
   certificate has expired or the date is invalid'.
   
   
    
   
   

LOCAL FIX

PROBLEM SUMMARY
 *  For customers with IBM Security Access Manager (ISAM), the
   certificate refresh action might fail to refresh expired
   certificates.
   
   
    
   
   

PROBLEM CONCLUSION
 *  Fix is available in 5.0.0.10 and 6.0.0.2.
   
   For a list of the latest fix packs available, please see:
   http://www-01.ibm.com/support/docview.wss?uid=swg21237631 [http://www-01.ibm.com/support/docview.wss?uid=swg21237631]
