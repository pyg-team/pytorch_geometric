Title: IBM Global Catalog Host is required when configuring Active Directory Server failover using a failover list in P8 - United States

Text:
 TECHNOTE (FAQ)

QUESTION
 Is the Global Catalog Host (Active Directory) entry optional when configuring Active Directory Server failover with a failover list in a P8 environment 

ANSWER
This entry is not optional but is required when configuring failover using a failover list. If you do not configure the Global Catalog Host entry Active Directory Server failover will fail.
