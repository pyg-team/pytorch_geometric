APAR STATUS
 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  Steps to reproduce:
   1.  Create a Choice list with 2 string items.
   2.  Create a property template (myProp) do not assign the Choice
   list.
   3.  Create a folder class (myFolderClass) and associate the
   property template created.
   4.  Click on the Property template in the property definition.
   5.  There are four tab, click 'More' and from drop down, choose
   the choice list created.
   6.  Save and Close.
   7.  Create an an instance of the folder class created in step 2.
   
   Note the following error:
   FNRCE0010E: E_BAD_VALUE: The value given for a property or list
   element lies outside the permitted range or value set, or
   exceeds the maximum length allowed. The value () specified for
   property myFolderClass.myProp is not within the range of
   permitted values.
   
   
    
   
   

LOCAL FIX
 *  Assign the choice list when creating the property template.
   
   
    
   
   

PROBLEM SUMMARY
 *  ****************************************************************
   * USERS AFFECTED:                                              *
   * ACCE users with 5.2.1.7-P8CPE-IF001 and prior                *
   ****************************************************************
   * PROBLEM DESCRIPTION:                                         *
   * E_BAD_VALUE occurs when creating an instance of a document   *
   * class containing a choice list assigned to a property        *
   * definition                                                   *
   *                                                              *
   * Steps to reproduce:                                          *
   * 1.  Create a Choice list "CL" with 2 string items. (str1,    *
   * str2)                                                        *
   * 2.  Create a property template (myProp) and do not assign    *
   * the Choice list.                                             *
   * 3.  Create a folder class (myFolderClass) and associate the  *
   * property template created. (Target -> Data Design -> Classes *
   * -> Folder -> Right click "New Class").                       *
   *                                                              *
   * To associate the Property Template:  Click on                *
   * "myFolderClass" and click on "Add" to add "myProp" to this   *
   * from my "property definition" tab.                           *
   * 4.  Click on the Property template (myProp) in the property  *
   * definition.                                                  *
   * 5.  There are four tab, click 'More' and from drop down,     *
   * choose the choice list (CL) created. (Add default value in   *
   * this screen "str1")                                          *
   * 6.  Save and Close.                                          *
   * 7.  Create an an instance of the folder class                *
   * (myFolderClass) created in step 2.                           *
   *                                                              *
   * Target -> Browse -> Root Folder -> Right click -> Copy       *
   * Object Reference (This is to paste, while creating the       *
   * instance of the folder)                                      *
   *                                                              *
   * myFolderClass -> Actions -> create instance -> click on      *
   * "Paste Object" from drop down for "Parent" input.            *
   * Give any "Folder Name". for example "test1".                 *
   * Next -> Finish.                                              *
   ****************************************************************
   * RECOMMENDATION:                                              *
   * Upgrade to 5.2.1.7-P8CPE-IF002 or 5.5.0.0-P8CPE or higher.   *
   ****************************************************************
   
   
    
   
   

PROBLEM CONCLUSION
 *  Issue is bcz of missing default value for choice list while
   property template is associated with the folder class.
   Now warning/alert dialog is added if it is missing during the
   time of creation.
   Fixed in  5.2.1.7-P8CPE-IF002 and 5.5.0.0-P8CPE and higher.
