QUESTION
 What is the meaning of CSPA008I? (SCI68616) 

ANSWER
If Auto Update is disabled, check the authentication previous key pair expiration date for both nodes. Check the update history log on both nodes for the last change to the record. Verify the authentication public key is correct for both nodes.
