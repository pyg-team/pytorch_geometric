Title: IBM (参考) IBM Forms Server が使用している Apache HTTPComponents の潜在的な脆弱性への対応 (CVE-2014-3577, CVE-2012-6153) - United States

Text:
 SECURITY BULLETIN

SUMMARY
 IBM Forms Server は、情報取得に関する脆弱性が存在する Apache HttpComponents のオープン・ソース・ライブラリを利用しています。 

VULNERABILITY DETAILS

CVEID: CVE-2012-6153 [http://cve.mitre.org/cgi-bin/cvename.cgi?name=CVE-2012-6153]
説明： Apache HTTPComponent には、サーバーのホスト名が証明書のサブジェクトの共通名 (CN) または subjectAltName フィールドのドメイン名と一致することに対する不完全な修正により、リモート攻撃者からなりすまし攻撃を受ける脆弱性が存在します。
特殊に細工された証明書を含む Web サイトを参照するように被害者を仕向けることで、攻撃者は SSL サーバーを偽装する中間者攻撃 (man-in-the-middle) 手法でこの脆弱性を悪用する可能性があります。

CVSS Base Score: 4.3
CVSS Temporal Score: 現時点のスコアについては ISS X-Force Database [https://exchange.xforce.ibmcloud.com/vulnerabilities/95328] を参照してください。
CVSS Environmental Score*: Undefined
CVSS Vector: (AV:N/AC:M/Au:N/C:N/I:P/A:N)


CVEID: CVE-2014-3577 [http://cve.mitre.org/cgi-bin/cvename.cgi?name=CVE-2014-3577]
説明： Apache HTTPComponent には、サーバーのホスト名と証明書のサブジェクトの共通名 (CN) または subjectAltName フィールドのドメイン名との検証に失敗することを利用し、リモート攻撃者からなりすまし攻撃を受ける脆弱性が存在します。
特殊に細工された証明書を含む Web サイトを参照するように被害者を仕向けることで、攻撃者は SSL サーバーを偽装する中間者攻撃 (man-in-the-middle) 手法でこの脆弱性を悪用する可能性があります。

CVSS Base Score: 4.3
CVSS Temporal Score: 現時点のスコアについては ISS X-Force Database [https://exchange.xforce.ibmcloud.com/vulnerabilities/95327] を参照してください。
CVSS Environmental Score*: Undefined
CVSS Vector: (AV:N/AC:M/Au:N/C:N/I:P/A:N)

AFFECTED PRODUCTS AND VERSIONS
IBM Forms Server 4.0.*
IBM Forms Server 8.0.*
IBM Forms Server 8.1
IBM Forms Server 8.2

製品 VRMF APAR 対応策 IBM Forms Server 4.0.0.* LO85829 以下をダウンロードし導入してください： 4.0.0.2-FormsServer-WFS-IF0002 [http://www.ibm.com/support/fixcentral/swg/quickorder?product=ibm/Lotus/Forms+Server&function=fixId&fixids=4.0.0.2-FormsServer-WFS-IF0002&includeRequisites=1] IBM Forms Server 8.0.0.* LO85829 以下をダウンロードし導入してください： 8.0.0-FormsServer-WFS-IF0002 [http://www.ibm.com/support/fixcentral/swg/quickorder?product=ibm/Lotus/Forms+Server&function=fixId&fixids=8.0.0-FormsServer-WFS-IF0002&includeRequisites=1] IBM Forms Server 8.0.1.* LO85829 IBM Forms Server 8.1.* LO85829 以下をダウンロードし導入してください： 8.1.0-FormsServer-WFS-IF0002 [http://www.ibm.com/support/fixcentral/swg/quickorder?product=ibm/Lotus/Forms+Server&function=fixId&fixids=8.1.0-FormsServer-WFS-IF0002&includeRequisites=1] IBM Forms Server 8.2.* LO85829 以下をダウンロードし導入してください：8.2.0-FormsServer-WFS-IF0001 [http://www.ibm.com/support/fixcentral/swg/quickorder?product=ibm/Lotus/Forms+Server&function=fixId&fixids=8.2.0-FormsServer-WFS-IF0001&includeRequisites=1] [http://www.ibm.com/support/fixcentral/swg/quickorder?product=ibm/Lotus/Forms+Server&function=fixId&fixids=4.0.0.2-FormsServer-WFS-IF0002&includeRequisites=1]
