Rule Team Server DOWNLOADABLE FILES

ABSTRACT
 This document describes how to download IBM WebSphere ILOG Rule Team Server V7.1.1 eAssembly images from IBM Passport Advantage Online. 

DOWNLOAD DESCRIPTION


 WebSphere ILOG Rule Team Server plays an important role in the WebSphere ILOG Business Rule Management System (BRMS), serving as the central workspace for enabling business users to collaborate business rules. 

Contents

 This document provides a list of eAssembly images, and describes how to download the images from IBM Passport Advantage Online. The eAssembly images include the latest fix pack for ILOG Rule Team Server V7.1.1.5. 

More information

 For more information about this product, visit the WebSphere ILOG Rule Team Server home page [http://www.ibm.com/software/integration/business-rule-management/rule-team-server/support/index.html]. 

Downloading and assembling the product

 1. With your browser, go to the Find Downloads and Media [http://www.ibm.com/software/howtobuy/passportadvantage/paocustomer/sdma/SDMA?PO=DOWNLOAD_FIND_BY_PART_NUMBER] page on the IBM Passport Advantage Online web site.
 2. Sign in using your IBM ID.
 3. Select the Find by part number search option under Download finder options.
 4. Search for each downloadable image by part number (refer to the list below).
 5. Download all of the parts you require.
 6. Unpack each part into a separate directory on your computer.
 7. Follow the instructions in the Quick Start Guide (CZM2QML) to install the product.
