Title: IBM Rulerunner Manager does not show the application Workflows and Tasks in IBM Datacap - United States

Text:
 TECHNOTE (TROUBLESHOOTING)

PROBLEM(ABSTRACT)
 The workflows and tasks are not present when selecting an application in Rulerunner Manager to populate a thread with tasks. 

RESOLVING THE PROBLEM
Re-import the keys: 

 1. Login to the Rulerunner Service account on the Operating System 
 2. Open a Windows Command Prompt (CMD) 
 3. Navigate to Taskmaster folder:
    Ex.: cd C:\Datacap\Taskmaster 
 4. run dcskey.exe with the 'i' flag:
    .\dcskey.exe -i
    or
    dcskey.exe /i
