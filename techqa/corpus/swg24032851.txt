ABSTRACT
 WebSphere eXtreme Scale Client Version 8.5 Fix Pack 1 includes performance improvements and fixes that were found internally. 

DOWNLOAD DESCRIPTION
You can download the IBM WebSphere eXtreme Scale Client Version 8.5 Fix Pack 1. With this download, you can extract the .zip file from this page, and install one of the following downloads using IBM Installation Manager:




 * Fix Pack 1. Includes fixes that were found since Version 8.5 was released. 
 * Full installation. Includes the Version 8.5 Client and the improvements from the fix pack .


For more information about the client, [http://publib.boulder.ibm.com/infocenter/wxsinfo/v7r1m1/topic/com.ibm.websphere.extremescale.doc/rxsfeatures.html]see Installing WebSphere eXtreme Scale Client [http://publib.boulder.ibm.com/infocenter/wdpxc/v2r1/topic/com.ibm.websphere.datapower.xc.doc/tinstclient.html] for more information about installing the client. Complete the following instructions to download the client images.
