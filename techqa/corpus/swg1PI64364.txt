ERROR DESCRIPTION
 *  Using Rational Developer for System z (RDz) host version
   9.1.1.3, if multiple users debug a CICS transaction using
   Integrated Debugger at the same time, the debug session for one
   of the users may hang. After the hang, the following binder
   error and a  CICS abend may also be observed:
   
   IEW2974T C009 REGION TOO SMALL TO ESTABLISH BINDER ENVIRONMENT
   
   
    
   
   

LOCAL FIX
 *  Debug the CICS transaction one user at a time
