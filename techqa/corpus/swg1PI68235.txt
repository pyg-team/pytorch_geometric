APAR STATUS
 * CLOSED AS UNREPRODUCIBLE.
    
   
   

ERROR DESCRIPTION
 *  When using IE11 with Compatibility View Mode disabled,
   
   If you execute a Cognos Report Studio, Chart type source report
   with Drill Through definition to a target report using prompt
   value passing, when you  hover over a chart intersection value,
   and right click on the mouse button to open the pop-up. Access
   the Go To menu, the associated target Drill-Through reports
   not displayed, nor while clicking on the Related Links are
   the target reports shown.
   
   
    
   
   

LOCAL FIX
 *  Enable Compatibility View mode within each Internet Explorer 11
   web browser. Use other web browsers e.g. Mozilla Firefox or
   Google Chrome.
