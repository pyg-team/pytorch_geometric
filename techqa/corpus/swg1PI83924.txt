ERROR DESCRIPTION
 *  In IBM Doors Next Generation (DNG), when a user attempts to
   import a subsequent ReqIF file it fails with the following
   errors:
   
   Errors importing standalone artifacts. x of x artifacts had
   errors.
   Error importing Artifact: null
   ReqIF ID    _1852b1a-989c-47s0-b830-89488f2e6c03
   Error importing Artifact: null
   ReqIF ID    _170d5257-0daf-40a2-84c5-c9458e43a14v
   ---
   --
   -
   
   
   Steps to Reproduce:
   1. Create a new DNG Project Area.
   2. Import the first ReqIf file.
   3. Confirm that this imports successfully with no errors.
   4. Import the second ReqIF file into the same Project Area.
   
   
   Expected Results:
   
   The ReqIF file imports successfully.
   
   Actual Results:
   
   The ReqIF import fails with "Errors importing standalone
   artifacts. x of x artifacts had errors"
   
   For more information, refer to the following workitem on
   jazz.net:
   https://jazz.net/jazz03/resource/itemName/com.ibm.team.workitem. [https://jazz.net/jazz03/resource/itemName/com.ibm.team.workitem.]
   WorkItem/111569
