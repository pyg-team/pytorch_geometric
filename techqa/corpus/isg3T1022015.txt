Title: IBM Security Bulletin: GNU C library (glibc) vulnerability affects PowerKVM (CVE-2015-0235) - United States

Text:
 SECURITY BULLETIN

SUMMARY
 GNU C library (glibc) vulnerability that has been referred to as GHOST affects PowerKVM. 

VULNERABILITY DETAILS
CVEID: CVE-2015-0235 [http://cve.mitre.org/cgi-bin/cvename.cgi?name=CVE-2015-0235]
DESCRIPTION: The gethostbyname functions of the GNU C Library (glibc) are vulnerable to a buffer overflow. By sending a specially crafted but valid hostname argument, a remote attacker could overflow a buffer and execute arbitrary code on the system with the privileges of the targeted process or cause the process to crash. The impact of an attack depends on the implementation details of the targeted application or operating system. This issue is being referred to as the "Ghost" vulnerability.

CVSS Base Score: 6.8
CVSS Temporal Score: See http://xforce.iss.net/xforce/xfdb/100386 [http://xforce.iss.net/xforce/xfdb/100386] for the current score
CVSS Environmental Score: Undefined
CVSS Vector: (AV:N/AC:M/Au:N/C:P/I:P/A:P) 

AFFECTED PRODUCTS AND VERSIONS
PowerKVM 2.1

REMEDIATION/FIXES
Fix is made available via Fix Central https://ibm.biz/BdEnT8 [https://ibm.biz/BdEnT8] in PowerKVM 2.1.1.1-36.0 SP1 and all later fix packs. See the README at http://delivery04.dhe.ibm.com/sar/CMA/OSA/04xig/0/README [http://delivery04.dhe.ibm.com/sar/CMA/OSA/04xig/0/README] for prerequisite fixes and instructions.
