ERROR DESCRIPTION
 *  Add the field "fully qualified file name" field to fileinfo for
   the Linux OS Agent
   
   
    
   
   

LOCAL FIX
 *  n/a
   
   
    
   
   

PROBLEM SUMMARY
 *  The Monitoring Agent for Linux OS File Information attribute
   group does not have a unique field to specify both path and file
    name.
   
   
    
   
   

PROBLEM CONCLUSION
 *  Add a field name "Fully Qualified File Name" that allows
   specifying path and file name in a unique field.

   vehicle:
