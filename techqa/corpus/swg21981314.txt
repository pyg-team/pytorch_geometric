Title: IBM Security Bulletin: Potential security vulnerability in IBM WebSphere Application Server may affect Tivoli Netcool Performance Manager (CVE-2016-0306) - United States

Text:
 SECURITY BULLETIN

SUMMARY
 IBM WebSphere Application Server is shipped as a component of Tivoli Netcool Performance Manager. Information about a potential security vulnerability in IBM WebSphere Application Server if FIPS 140-2 is enabled. 

VULNERABILITY DETAILS
CVEID: CVE-2016-0306 [http://cve.mitre.org/cgi-bin/cvename.cgi?name=CVE-2016-0306]
DESCRIPTION: IBM WebSphere Application Server could provide weaker than expected security, caused by the improper TLS configuration. A remote attacker could exploit this vulnerability to obtain sensitive information using man in the middle techniques.
CVSS Base Score: 3.7
CVSS Temporal Score: See https://exchange.xforce.ibmcloud.com/vulnerabilities/111423 [https://exchange.xforce.ibmcloud.com/vulnerabilities/111423] for the current score
CVSS Environmental Score*: Undefined
CVSS Vector: (CVSS:3.0/AV:N/AC:H/PR:N/UI:N/S:U/C:L/I:N/A:N) 
Please consult the security bulletins below for vulnerability details: 

http://www-01.ibm.com/support/docview.wss?uid=swg21981086 [http://www-01.ibm.com/support/docview.wss?uid=swg21981086] http://www-01.ibm.com/support/docview.wss?uid=swg21979231 [http://www-01.ibm.com/support/docview.wss?uid=swg21979231]
