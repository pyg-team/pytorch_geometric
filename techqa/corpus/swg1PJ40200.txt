APAR STATUS
 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  When Single Sign-On (SSO) using Kerberos or SPNEGO is enabled,
   the Deploy Solution task in Case Manager Administration Client
   generates the following error:
   
   The IBM Case Manager REST protocol returned the 401 response
   code. Use Enterprise Manager to look at the deployment logs in
   ICM > IBM Case Manager > Solution Deployments > SSO. The
   following response was returned:
   <html><head><title>An NTLM Token was
   received.</title></head><body>Your browser configuration is
   correct, but you have not logged into a supported Microsoft(R)
   Windows(R) Domain. <p>Please login to the application using the
   normal login page.</html>
   
   The deployment error log could not be found.
   
   
    
   
   

LOCAL FIX
 *  Deploy Solution in CMAC cannot be used with SSO enabled.  To
   deploy in the production environment, disable SSO.
   
   
    
   
   

PROBLEM SUMMARY
 *  ****************************************************************
   * USERS AFFECTED:                                              *
   * System Administrators who want to run Case Manager           *
   * Administration Client's Deploy Solution task.                *
   ****************************************************************
   * PROBLEM DESCRIPTION:                                         *
   * See Error Description                                        *
   ****************************************************************
   * RECOMMENDATION:                                              *
   * To fix this problem, apply IBM Case Manager V5.1.0.1 Interim *
   * Fix 1 or later releases.                                     *
   ****************************************************************
   
   
    
   
   

PROBLEM CONCLUSION
 *  The fix for this problem is provided in IBM Case Manager
   V5.1.0.1 Interim Fix 1 and later releases.
   
   A fix was made in the Case Manager Administration Client (CMAC)
   such that when Single Sign-On (SSO) using Kerberos/SPNEGO is
   enabled, the Deploy Solution task in Case Manager Administration
   Client succeeds.
