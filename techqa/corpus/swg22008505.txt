Title: IBM Error 'SSL Init Failed' is encountered when posting to HTTPS from ITX. - United States

Text:
WTX ITX SSL Failed Sterling HTTPS TECHNOTE (TROUBLESHOOTING)

PROBLEM(ABSTRACT)
 Error 'SSL Init Failed' is encountered when posting to HTTPS from ITX within Sterling Integrator (SI). 

SYMPTOM
Error 'SSL Init Failed' is encountered when posting to HTTPS from ITX.


CAUSE
The IBM Transformation Extender/WebSphere Transformation Extender(ITX/WTX) Environment variables have not been set.

ENVIRONMENT
Running a ITX/WTX map on Sterling Integrator (SI) and the ITX/WTX setup script has not been run.

DIAGNOSING THE PROBLEM
Running a ITX/WTX map on Sterling Integrator (SI) using HTTPS adapter a 'SSL Init failed' error is encountered.

RESOLVING THE PROBLEM
Run the ITX/WTX setup script (. ./setup) to set the ITX/WTX environment variables.
