Title: IBM z/OS V1R13 Communications Server: AT-TLS currency with System SSL - United States

Text:
 PRODUCT DOCUMENTATION

ABSTRACT
 This document describes the z/OS V1R13 Communications Server AT-TLS support for TLS v1.2 and related features. 

CONTENT
z/OS V1R13 Communications Server Application Transparent Transport Layer Security (AT-TLS) with PM62905 applied provides toleration for the following functions that System SSL provides: 

 * Renegotiation (RFC 5746) in z/OS V1R12 
 * TLS v1.2 (RFC 5246) in z/OS V1R13 with OA39422 applied 
 * AES GCM Cipher Suites (RFC 5288) in z/OS V1R13 with OA39422 applied
