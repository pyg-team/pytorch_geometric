Title: IBM Security Bulletin: Vulnerabilities in OpenSSL affect IBM FileNet System Monitor/IBM Enterprise Content Management System Monitor (CVE-2014-3567) - United States

Text:
 SECURITY BULLETIN

SUMMARY
 OpenSSL vulnerabilities along with SSL 3 Fallback protection (TLS_FALLBACK_SCSV) were disclosed on October 15, 2014 by the OpenSSL Project. OpenSSL is used by IBM FileNet System Monitor/IBM Enterprise Content Management System Monitor. IBM FileNet System Monitor/IBM Enterprise Content Management System Monitor has addressed the applicable CVE and included the SSL 3.0 Fallback protection (TLS_FALLBACK_SCSV) provided by OpenSSL. 

VULNERABILITY DETAILS
CVE-ID: CVE-2014-3567 [http://cve.mitre.org/cgi-bin/cvename.cgi?name=CVE-2014-3567]
DESCRIPTION: OpenSSL is vulnerable to a denial of service, caused by a memory leak when handling failed session ticket integrity checks. By sending an overly large number of invalid session tickets, an attacker could exploit this vulnerability to exhaust all available memory of an SSL/TLS or DTLS server.

CVSS Base Score: 5.0
CVSS Temporal Score: See https://exchange.xforce.ibmcloud.com/vulnerabilities/97036 [https://exchange.xforce.ibmcloud.com/vulnerabilities/97036] for more information
CVSS Environmental Score*: Undefined
CVSS Vector: (AV:N/AC:L/Au:N/C:N/I:N/A:P)

AFFECTED PRODUCTS AND VERSIONS
IBM FileNet System Monitor v4.5.0
IBM Enterprise Content Management System Monitor v5.1.0
IBM Enterprise Content Management System Monitor v5.2.0

REMEDIATION/FIXES
For IBM FileNet System Monitor v4.5.0, upgrade to v4.5.0.3 Interim Fix 2.
For IBM Enterprise Content Management System Monitor v5.1.0, upgrade to v5.1.0.3 Interim Fix 5.
For IBM Enterprise Content Management System Monitor v5.2.0, upgrade to v5.2.0.0 Interim Fix 3.

Fixes can be downloaded from www.ibm.com/support/fixcentral [www.ibm.com/support/fixcentral] site. Search for "IBM Enterprise Content Management System Monitor" and select appropriate installed version.

If you need further assistance, please contact IBM Support.
