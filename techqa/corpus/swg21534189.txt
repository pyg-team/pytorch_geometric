QUESTION
 HOW TO: Code a process to invoke a shell script on a remote UNIX (SCI83375) 

ANSWER


This process will copy a file to UNIX and then kick off a shell script on UNIX that will submit a process to do another copy:



UNIXRUNT PROCESS SNODE=UNIX.NODE -
SNODEID=(unixid,password)
STEP01 -
COPY FROM (DSN=SOURCE.FILE -
DISP=SHR -
PNODE -
) -
CKPT=5K -
COMPRESS -
TO (DSN="/home/unix/destination.file" -
DISP=RPL -
SNODE -
)
IF (STEP01 = 0) THEN
STEP02 RUN TASK SNODE (PGM=UNIX) -
SYSOPTS="/home/unix/unixcopy.sh"
EIF



where the submitted shell script unixcopy.sh:



set -v
ndmcli -x << EOJ<BR>submit<BR>unixcopy process snode=DEST.NODE snodeid=(DESTID,PASSWORD)<BR>step1 copy to (file=DEST.FILE<BR> space=(CYL,(50,10),RLSE)<BR> snode)<BR> from (file=/home/unix/source.file<BR> pnode)<BR>pend;</P><br/><br/>
