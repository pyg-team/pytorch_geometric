Title: IBM 【セキュリティ情報】: WebSphere Message Broker と IBM Integration Bus 情報漏えいの脆弱性(CVE-2017-1126) - United States

Text:
 SECURITY BULLETIN

SUMMARY
 IBM Integration Bus と WebSphere Message Broker に権限のないユーザーがソフトウェアバージョンなどの情報を得てしまう情報漏えいのセキュリティ脆弱性が見つかりました。 

VULNERABILITY DETAILS
最新の情報については、下記URLの文書（英語）をご参照ください。
WebSphere Message Broker and IBM Integration Bus are affected by information disclosure vulnerability(CVE-2017-1126)
http://www-01.ibm.com/support/docview.wss?uid=swg22008470 [http://www-01.ibm.com/support/docview.wss?uid=swg22008470]

CVEID: CVE-2017-1126
DESCRIPTION:IBM Integration Bus と WebSphere Message Broker に権限のないユーザーがソフトウェアバージョンなどの更なる攻撃を引き起こしてしまう危険性のある情報を得ることができてしまうという情報漏えいのセキュリティ脆弱性が見つかりました。

CVSS Base Score: 5.3
CVSS Temporal Score: See https://exchange.xforce.ibmcloud.com/vulnerabilities/121341 [https://exchange.xforce.ibmcloud.com/vulnerabilities/121341] for the current score
CVSS Environmental Score*: Undefined
CVSS Vector: (CVSS:3.0/AV:N/AC:L/PR:N/UI:N/S:U/C:L/I:N/A:N)

AFFECTED PRODUCTS AND VERSIONS
下記のバージョンにおいて影響を受けます。

IBM Integration Bus V9.0.0.0 - V9.0.0.8 and V10.0.0.0 - V10.0.0.9

WebSphere Message Broker V8.0.0.0 - V8.0.0.8
