APAR STATUS

ERROR DESCRIPTION
 *  When trying to install a new ICC version or fixpack on an
   already existing system the installation can fail with an error
   such as:
   
   Install File: C:\Program
   Files(x86)\IBM\ContentCollector\AFUWeb\ewas\bin\WASService.exe
                            Status: Fehler
    Zusatzliche Hinweise: Fehler - Flexeraak4: C:\Program Files
   (x86)\ibm\ContentCollector\AFUWeb\ewas\bin\WASService.exe
   (Access is denied.)
   
   As part of the installation process the WASService.exe file can
   be deleted but a new one can not be created and the
   installation can not proceed from this point
   
   
    
   
   

LOCAL FIX
 *  If an existing copy of the WASService.exe file is copied from a
   previous installation the upgrade can continue
