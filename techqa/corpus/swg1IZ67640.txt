   should be entered for the truststore.  The sentence should read:
   
   "Complete path and file name to the VMware VI agent certificate
   database, which should be:"
   
   
    
   
   

LOCAL FIX

   just an example.

   "Complete path and file name to the VMware VI agent certificate
   database, which should be:"

   be found by going to
   http://www.ibm.com/software/sysmgmt/products/support/ then
   clicking on 'search' on the Support tab, entering the DCF record
   number in the search terms box and clicking 'Search'.'
