PROBLEM(ABSTRACT)
 You receive an abend 0C1 in your CICS Transaction Server for z/OS (CICS TS) region while running a transaction in Debug Tool. 

CAUSE
Installation of Debug Tool [http://www.ibm.com/software/awdtools/debugtool/support/] resources missing resource group.

RESOLVING THE PROBLEM
Add resource group sample groupname of EQA to the list which is specified in the SIT GRPLIST(xxx) parameters. then restart CICS with a COLD or INITIAL start. 


Note: Ensure module CEEEV006 is loaded during CICS start-up. The EQA was installed after CICS was already up so the environment was not properly initialized.
