Title: IBM ICN menu appears greyed out when assigned to a XT entry template - United States

Text:
ICN; WorkplaceXT Extension Addon; CE; ICN Menu TECHNOTE (TROUBLESHOOTING)

PROBLEM(ABSTRACT)
 When selecting a menu option in IBM Content Navigator (ICN), which is assigned to an IBM FileNet P8 Workplace XT (XT) entry template, it appears greyed out. 

CAUSE
XT Extensions AddOn had not been added to the system when the object store was originally created.

DIAGNOSING THE PROBLEM
Attempting to enable the ICN entry template under Settings\Configuration Parameters tab causes the following error to appear: 

[/support/docview.wss?uid=swg21698998&aid=1] [/support/docview.wss?uid=swg21698998&aid=1]


RESOLVING THE PROBLEM
 

 * Go to IBM FileNet P8 Content Engine Enterprise Manager (FEM) and add the Workplace XT Extensions AddOn. 
 * Menu option assigned to XT entry template now appears correctly.
