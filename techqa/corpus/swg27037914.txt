Title: IBM Best Practices for Implementing IBM Content Manager OnDemand for z/OS in a Parallel Sysplex - United States

Text:
CMOD parallel sysplex WHITE PAPER

ABSTRACT
 This document explains how Content Manager OnDemand (CMOD) V8.5 and V9.0 for z/OS can be set up in a parallel sysplex envrionment. 

CONTENT
This example environment is set up to support CMOD development internal performance testing and testing to validate or replicate customer implementations. The information contained in this document represents a working CMOD Parallel Sysplex configuration which can be used as a guideline when implementing CMOD in a Parallel Sysplex.
