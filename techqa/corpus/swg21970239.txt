Title: IBM Microsoft® Windows Patch KB3042058 の適用後に IBM Sterling B2B Integrator and IBM Sterling File Gateway が起動しない事象 - United States

Text:
 フラッシュ(ALERTS)

概要
 Windows の更新プログラム KB3042058 を適用した後 IBM Sterling B2B Integrator および IBM Sterling File Gateway が起動に失敗することがあります。 

本文
IBM Sterling B2B Integrator および IBM Sterling File Gateway を 

 * MicrosoftR Windows OS 
 * SQL Server database 
 * Oracle JDK 1.6

の環境で稼働させている場合、2015年10月にリリースされたMicrosoftR Windows の更新プログラム KB3042058 ( https://technet.microsoft.com/en-us/library/security/3042058.aspx [https://technet.microsoft.com/en-us/library/security/3042058.aspx]) 
を適用すると IBM Sterling B2B Integrator および IBM Sterling File Gateway が以下のエラーにより起動しなくなる事象が報告されています。 

ERROR 000310100002 UTIL.FRAME_JDBC.ERR_ConnectionFactory_create [ConnectionFactory].create() 
ERROR [1445255601635] The driver could not establish a secure connection to SQL Server by using Secure Sockets Layer (SSL) encryption. Error: "SQL Server returned an incomplete response. The connection has been closed.". 
ERRORDTL [1445255601635]com.microsoft.sqlserver.jdbc.SQLServerException: The driver could not establish a secure connection to SQL Server by using Secure Sockets Layer (SSL) encryption. Error: "SQL Server returned an incomplete response. The connection has been closed.". 


このエラーはSQL Serverが稼働しているOSとOracle JDKのセキュリティ・プロバイダーの暗号の不整合によるものです。 

当該エラーには2つの対応策があります。 
1. IBM Sterling B2B Integrator 5.2.4.2以降のFix Packを適用してください。これによりIBM Sterling B2B IntegratorはIBM Java SDK 7.0を使用するようになり、このWindowsの更新プログラムによる影響を受けなくなります。できる限りこの対応策を実施いただくことを推奨します。 

2. 回避策：もしFix Packを適用できない場合は、以下の手順によってこの事象を回避することが可能です。 

--------------------------------------------- 
1. SQL Serverが稼働しているWindowsサーバーに管理者としてログオンします。 
2. コマンドプロンプトで「gpedit.msc」を実行してグループポリシーエディターを開きます。 
3. 「コンピューターの構成」 > 「管理用テンプレート」 > 「ネットワーク」と展開し、「SSL構成設定」をクリックします。 
4. 「SSL暗号の順位」をダブルクリックします。 
5. 「有効」が選択されていることを確認します。 
6. 「SSL暗号」で設定されている内容をコピーしてテキストファイルに保存し、バックアップをとります。 
7. 「SSL暗号」に以下の内容を設定します。（注：改行やスペースは入れずすべてを1行で設定してください。） 
 * 


8. 「適用」「OK」をクリックします。 
9. 変更を反映するためWindowsを再起動し、それからIBM Sterling B2B Integratorを再起動します。
