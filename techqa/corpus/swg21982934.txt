Title: IBM Security Bulletin: Two ReDoS vulnerabilities in modules included in the Node.js npm tool - United States

Text:
 SECURITY BULLETIN

SUMMARY
 Two ReDoS vulnerabilities in modules included in the Node.js npm tool shipped by IBM Rational Application Developer for WebSphere Software. 

VULNERABILITY DETAILS
CVEID: CVE-2016-2515 [http://cve.mitre.org/cgi-bin/cvename.cgi?name=CVE-2016-2515]
DESCRIPTION: Node.JS hawk is vulnerable to a denial of service, caused by an error in the regular expression implementation. An attacker could exploit this vulnerability using an overly long header or URI to cause the application to hang.
CVSS Base Score: 5.3
CVSS Temporal Score: See https://exchange.xforce.ibmcloud.com/vulnerabilities/110819 [https://exchange.xforce.ibmcloud.com/vulnerabilities/110819] for the current score
CVSS Environmental Score*: Undefined
CVSS Vector: (CVSS:3.0/AV:N/AC:L/PR:N/UI:N/S:U/C:N/I:N/A:L) 

CVEID: CVE-2016-2537 [http://cve.mitre.org/cgi-bin/cvename.cgi?name=CVE-2016-2537]
DESCRIPTION: Node.js is vulnerable to a denial of service, caused by an error in the regular expression implementation. An attacker could exploit this vulnerability using a regular expression to block the event loop and cause the application to hang.
CVSS Base Score: 5.3
CVSS Temporal Score: See https://exchange.xforce.ibmcloud.com/vulnerabilities/110870 [https://exchange.xforce.ibmcloud.com/vulnerabilities/110870] for the current score
CVSS Environmental Score*: Undefined
CVSS Vector: (CVSS:3.0/AV:N/AC:L/PR:N/UI:N/S:U/C:N/I:N/A:L)

AFFECTED PRODUCTS AND VERSIONS
IBM Rational Application Developer for WebSphere Software v9.1 and v9.5

REMEDIATION/FIXES
 

Rational Application Developer 9.1.x and 9.5.x PI60929  * Apply IBM SDK for Node.js Version 1.1 release updated equivalent to the Joyent Node.js API version 0.10.44 [https://developer.ibm.com/node/sdk/#v11] to the Cordova platform in the product.
   
   Installation instructions for applying the update to the Cordova platform in the product can be found here:
   
   Upgrading the IBM SDK for Node.js used by Cordova [http://www.ibm.com/support/docview.wss?uid=swg21684946]
