About Fix Pack 16.0.0.2 
New fix pack numbering is introduced that Fix pack 16.0.0.2 for WebSphere Application Server Liberty is the first of a series of common Liberty levels that apply to both Version 8.5 and Version 9.0 of WebSphere Application Server on all supported platforms. 

For update Version 8.5 Liberty offerings, use Fix pack 16.0.0.2 download page [http://www-01.ibm.com/support/docview.wss?uid=swg24042163]. 

For Version 9 fresh install, download from PPA [http://www.ibm.com/support/docview.wss?uid=swg27038624].

Prerequisites 
 * Check WebSphere Application Server detailed system requirements [http://www-01.ibm.com/support/docview.wss?rs=180&uid=swg27006921], IBM WebSphere SDK Java Technology Edition 7.1 Supporting Software [http://www-01.ibm.com/support/docview.wss?rs=180&uid=swg27042161] and IBM WebSphere SDK Java Technology Edition 8 Supporting Software [http://www-01.ibm.com/support/docview.wss?uid=swg27045339] before proceeding with the install.


 * Fix pack 8.5.5.9 or any lower version 8.5.5 fixpack of the product needs to be installed first before you can proceed with Fix Pack 16.0.0.2 installation for version 8.5 offerings. 
   
   
 * Download and install the latest available IBM Installation Manager [http://www.ibm.com/support/docview.wss?uid=swg24020212]IBM Installation Manager download website [http://www-947.ibm.com/support/entry/portal/Recommended_fix/Software/Rational/IBM_Installation_Manager].

Space requirements 
The amount of disk space required varies with the number of features or products installed. If you are installing the product using Installation Manager, the installation summary panel indicates the approximate amount of disk space required based on the features and products you have selected. For a Fix Pack, you need less than 500MB of free disk space.
