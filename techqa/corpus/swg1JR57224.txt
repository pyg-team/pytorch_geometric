LOCAL FIX

PROBLEM SUMMARY
 *  The Process Monitor page in the Process Admin console requires
   the instrumentation framework to work correctly. However, the
   page is not enabled to deal with the option to disable
   instrumentation, which was introduced with interim fix JR55797 [http://www-01.ibm.com/support/docview.wss?uid=swg1JR55797].
   As a result, the page fails with the indicated error.
   
   PRODUCTS AFFECTED
   IBM Business Process Manager (BPM) Advanced
   IBM BPM Standard
   IBM BPM Express
   
   
    
   
   

PROBLEM CONCLUSION
 *  The Process Monitor page code was changed to show an error
   message that indicates that the instrumentation framework is
   disabled.
   
   The fix is currently targeted for inclusion in IBM BPM 8.5.7
   cumulative fix 2017.03.
