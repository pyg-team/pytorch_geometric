APAR STATUS
 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  Extended Oracle Database agent v6.31 FP2 data collector process
   with name krzclient (or krzstart on some platforms) consumes
   more and more memory. The amount of memory leak depends on the
   amount of data requests.
   
   Detailed Recreation Procedure:
   1) Install and configure Extended Oracle Database agent v6.31
   FP2.
   2) Start the agent.
   3) Observe memory consumption of data collector process with
   name krzclient (or krzstart on some platforms)
   
   
    
   
   

LOCAL FIX

PROBLEM SUMMARY
 *  The collector process - krzclient (or krzstart on some
   platforms) of the Oracle Database Extended agent version 6.31
   Fix pack 2 has memory leak. It only occurs on version 6.31 Fix
   pack 2.
   
   
    
   
   

PROBLEM CONCLUSION
 *  6.3.1.2-TIV-ITM_KRZ-IF0002 includes fix of the APAR. Apply
   6.3.1.2-TIV-ITM_KRZ-IF0002 or later to resolve the problem.
