APAR STATUS
 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  "IBMMON/ITM_SYS_LOG is invoked every 10 minutes and it generates
   ABAP  Dump like :
   Function Module "/IBMMON/ITM_SYS_LOG" was called with parameter
   "ITM_R3_LOG_RECORD".This parameter is not defined.
   Syslog information are available from INS and GRP subnodes, but
   problem only occurs while collecting data for :GRP subnode.
   Problem occurs with MySAP Agent 620 IF15 and IF16.
   
   
    
   
   

LOCAL FIX
 *  You can temporary disable data collection for ITM_SYS_LOG follow
   ing those steps:
   SAP Side :
   1) Run transaction code SE16
   2) Enter the table name /IBMMON/ITM_CNFG
   3) Push function key F5 or click "Create Entries" button.
   4) Add the entry : /IBMMON/ITM_SYS_LOG with value NO.
   5) Click Save
   .
   No changes on agent side.
