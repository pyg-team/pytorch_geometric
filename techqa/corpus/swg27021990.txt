 The Product Documentation Library contains books that tell you how to install, configure, develop, and use the IBM Atlas Suite V6 products and frameworks. You can download each book individually by area or access the entire library from the product media. 

 

 RELEASE NOTES AND INSTALLATION
 * IBM Atlas Suite V6.0.x Features, Software Integration, and Release Notes [http://www.ibm.com/support/docview.wss?uid=swg27024529] lists and describes the enhancements, software dependencies, and issue resolutions that are part of the IBM Atlas Suite 6.0.x releases (including Fix Packs). 
 * IBM Atlas Suite V6.0.1 Administration: Installing and Upgrading the System [http://www.ibm.com/support/docview.wss?uid=swg27024073] tells you how to install or upgrade to the full version of IBM Atlas Suite 6.0 (6.0.1), and how to apply Fix Pack 3 (6.0.1.3). 



 CONFIGURING, POPULATING, AND MONITORING THE SYSTEM
 * IBM Atlas Suite V6.0 Administration: Configuring the System [http://www.ibm.com/support/docview.wss?uid=swg27024076] tells you how to configure a fresh deployment of IBM Atlas Suite, and provides reference documentation for Components and Timer Tasks. 
 * IBM Atlas Suite V6.0 Administration: Alerts [http://www.ibm.com/support/docview.wss?uid=swg27024077] provides reference documentation for the Atlas Alert notifications, and tells you how to customize the messages that are displayed in the Alerts. 
 * IBM Atlas Suite V6.0 Administration: Populating Atlas with Business Data [http://www.ibm.com/support/docview.wss?uid=swg27024078] tells you how to import your business data (Organizations, Persons, Legal Matters, and so on) into Atlas. 
 * IBM Atlas Suite V6.0 Administration: Monitoring the System [http://www.ibm.com/support/docview.wss?uid=swg27024606] tells you how to monitor the health of the Atlas applications. 



 CUSTOM REPORT DEVELOPMENT
 * IBM Atlas Suite V6.0 Development: Creating Custom Reports [http://www.ibm.com/support/docview.wss?uid=swg27024079] tells you how to design your own BIRT reports, and provides a specification of the Reporting Views that bring data out of the IBM Atlas Suite database for use in your reports. 



 EDISCOVERY CONFIGURATION AND DEVELOPMENT
IBM Atlas Suite V6.0 Administration and Development: Advanced eDiscovery Features [http://www.ibm.com/support/docview.wss?uid=swg27024081] tells you how to: 

 * Develop, configure, and use Drop Boxes. 
 * Export documents that were collected as part of a legal matter. 
 * Set up a web service that augments the set of custodians who are involved in a legal matter. 



 EDISCOVERY POLICY FEDERATION DEVELOPMENT
IBM Atlas Suite V6.0 Administration and Development: eDiscovery Policy Federation Framework and SDK [http://www.ibm.com/support/docview.wss?uid=swg27024082] tells you how to develop and use eDiscovery Policy Federation Connectors. 



 GLOBAL RETENTION MANAGEMENT CONFIGURATION DEVELOPMENT
IBM Atlas Suite V6.0 Administration and Development: Installing, Configuring, and Developing Retention Policy Applications and Connectors [http://www.ibm.com/support/docview.wss?uid=swg27024080] tells you how to: 

 * Install and configure the Retention for Employees Portal. 
 * Read the file that the system creates when you export retention policies. 
 * Install and configure the Policy Distribution connectors. 



 USER GUIDES
 * IBM Atlas Suite V6.0 User Guides: Data Sources [http://www.ibm.com/support/docview.wss?uid=swg27024084] provides information about the data repositories (email servers, hard drives, filing cabinets, and so on) that you define in IBM Atlas Suite. 
 * IBM Atlas Suite V6.0 User Guides: Reports (including the Retention for Employees Portal [http://www.ibm.com/support/docview.wss?uid=swg27024083]) explains the layout and contents of the Reports that are available through the Atlas application and the Retention for Employees Portal.
