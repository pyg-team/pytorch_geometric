APAR STATUS
 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  The user observed DataPower failed to Import a file from an
   export that had an identical match for path/filename if one
   would remove the folder "/" slashes.
   For example, the Import contained
   - "local:///aFolder/aFile.txt"
   - "local:///aFolderaFile.txt"
   and then only one of these got imported.
   
   
    
   
   

LOCAL FIX
 *  Manual upload of affected files.
   
   
    
   
   

PROBLEM SUMMARY
 *  Affected is importing of files.
   
   Files of form aFolder/aFile.txt and aFolderaFile.txt
   are not handled correctly until this fix. This might result in
   some files not getting imported.
   
   
    
   
   

PROBLEM CONCLUSION
 *  Fix is available in 7.0.0.14, 7.1.0.11, 7.2.0.8, 7.5.0.2 and
   7.5.1.2
   
   For a list of the latest fix packs available, please see:
   http://www-01.ibm.com/support/docview.wss?uid=swg21237631 [http://www-01.ibm.com/support/docview.wss?uid=swg21237631]
