APAR STATUS
 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  Users of the DB2 Administration Tool for z/OS utilities.
   Perform online reorg on TS will pop-up message 'ADB788E The
   object name specified contains invalid characters.'
   
   
    
   
   

LOCAL FIX
 *  Remove any special characters from the Map Table name.
   
   
    
   
   

PROBLEM SUMMARY
 *  ****************************************************************
   * USERS AFFECTED: Users of the DB2 Administration Tool for     *
   *                 z/OS.                                        *
   ****************************************************************
   * PROBLEM DESCRIPTION: When displaying tablespaces on panel    *
   *                      ADB21S 'Table Spaces', specifying the   *
   *                      'UTIL-Utility Selections' line command  *
   *                      on a tablespace, selecting the 'OO -    *
   *                      online reorg' utility option on panel   *
   *                      ADB2US 'Table Space Utilities',         *
   *                      entering a SHRLEVEL value of 'C' on     *
   *                      panel ADB2USOO 'Online Reorg Mapping    *
   *                      Table', and entering a mapping table    *
   *                      with an underscore in the name, DB2     *
   *                      Admin advances to panel ADB2USO1        *
   *                      'Specify Utility Options - REORG        *
   *                      TABLESPACE' and further processing      *
   *                      causes the error message ADB788E        *
   *                      to be displayed:                        *
   *                                                              *
   *                      ADB788E The object name specified       *
   *                      contains invalid characters.            *
   ****************************************************************
   * RECOMMENDATION:                                              *
   ****************************************************************
   Panel ADB2USO1 incorrectly checks the syntax of the mapping
   table name.
   
   
    
   
   

PROBLEM CONCLUSION
 *  Panel ADB2USO1 is modified to allow the underscore in the
   mapping table name.  The underscore can not be the first
   character of the name.
