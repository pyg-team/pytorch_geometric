PROBLEM SUMMARY
 *  ****************************************************************
   * USERS AFFECTED:  Users of WebSphere eXtreme Scale HTTP       *
   *                  session replication.                        *
   ****************************************************************
   * PROBLEM DESCRIPTION: The eXtreme Scale client HTTP session   *
   *                      component did not correctly retrieve    *
   *                      the bootstrap port of the application   *
   *                      server JVMs.                            *
   ****************************************************************
   * RECOMMENDATION:                                              *
   ****************************************************************
   HTTP session applications are unable to connect to catalog
   domains containing JVMs for WebSphere Application Server,
   including deployment manager or application servers. This
   causes a NoRouteToHostException when the application attempts
   an initial client connection.
   
   
    
   
   

PROBLEM CONCLUSION
 *  The eXtreme Scale client HTTP session component has been
   updated to correctly use the JVM bootstrap port for WebSphere
   Application Server.
