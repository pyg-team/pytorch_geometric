APAR STATUS
 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  Customer found using mvscan unable to delete the empty folder at
   following structure, if they set mvscan source folder as
   Folder1, and images folder under Folder4, Rulerunner is able to
   pick up the images under image folder, but the images folder can
   not be deleted after mvscan action being executed.
   
   
   Folder1/Folder2/Folder3/Folder4/images
   
   But if customer either set image source folder as
   Folder1/Folder2 or put images folder under Folder3, the folder
   can be properly deleted after Rulerunner process images per
   mvscan ruleset.
