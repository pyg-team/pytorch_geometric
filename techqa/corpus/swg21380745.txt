Title: IBM java.lang.UnsatisfiedLinkError when installing IBM Rational ClearCase 7.1 on Solaris - United States

Text:
RATLC01276762; CQ7.1; CQ71; installation; java.lang.UnsatisfiedLinkError: no swt-gtk-3346 or swt-gtk in swt.library.path; java.library.path or the jar file; java.lang.UnsatisfiedLinkError; swt-gtk-3346; swt-gtk; swt.library.path; UnsatisfiedLinkError; 3346; swt-gtk; gtk TECHNOTE (TROUBLESHOOTING)

PROBLEM(ABSTRACT)
 An error might occur, when you install IBM Rational ClearCase 7.1 through the IBM Installation Manager on a Sun Solaris 10/x86 computer. 

SYMPTOM
The error message is: 

 * 
 * java.lang.UnsatisfiedLinkError: no swt-gtk-3346 or swt-gtk in swt.library.path, java.library.path or the jar file


CAUSE
The installation through the graphical interface is unavailable on Solaris 10/x86. Therefore the operating system does not support the interactive install through the Installation Manager. This is a product limitation.



RESOLVING THE PROBLEM
You can use the IBM Installation Manager in silent mode under Solaris. 


For more information about install in silent mode, visit the IBM Installation Manager Information Center [http://publib.boulder.ibm.com/infocenter/install/v1r2/index.jsp]. Also the web documents in the section 'Related Information' contain more information.
