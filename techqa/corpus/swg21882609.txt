Title: IBM Security Bulletin: A security vulnerability has been identified in IBM DB2, IBM HTTP Server and IBM WebSphere MQ shipped with IBM Predictive Maintenance and Quality (CVE-2014-8730). - United States

Text:
 SECURITY BULLETIN

SUMMARY
 IBM DB2, IBM HTTP Server and IBM WebSphere MQ are shipped as components of IBM Predictive Maintenance and Quality. Information about a security vulnerability affecting IBM DB2, IBM HTTP Server and IBM WebSphere MQ has been published in security bulletins. 

VULNERABILITY DETAILS
Please consult the Security Bulletin: TLS padding vulnerability affects IBM WebSphere MQ (CVE-2014-8730) [http://www-01.ibm.com/support/docview.wss?uid=swg21692552] for vulnerability details and information about fixes.
Please consult the Security Bulletin: TLS padding vulnerability affects IBM® DB2® LUW (CVE-2014-8730) [http://www-01.ibm.com/support/docview.wss?uid=swg21692618] for vulnerability details and information about fixes.

Please consult the Security Bulletin: TLS padding vulnerability affects IBM HTTP Server (CVE-2014-8730) [http://www-01.ibm.com/support/docview.wss?uid=swg21692502] for vulnerability details and information about fixes.
