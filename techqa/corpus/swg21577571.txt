QUESTION
 How do you find out which release of IBM Explorer for z/OS (z/OS Explorer) and CICS tools plug-ins for CICS Explorer are compatible with the different releases of CICS Explorer? How do you know what release of the Eclipse SDK is required for your release of z/OS Explorer and CICS Explorer? 

CAUSE
Certain releases of z/OS Explorer and CICS tools plug-ins for CICS Explorer will only work with certain releases of CICS Explorer.

ANSWER
Click a link below to be taken to the corresponding compatibility reference table. Use the table to determine the correct combination of 'Explorer' release and CICS tools plug-in release. 

 * IBM Explorer for z/OS (z/OS Explorer)
 * CICS Configuration Manager for z/OS (CICS CM)
 * CICS Deployment Assistant for z/OS (CICS DA)
 * CICS Interdependency Analyzer for z/OS (CICS IA)
 * CICS Performance Analyzer for z/OS (CICS PA)
 * CICS Transaction Gateway for z/OS and Multiplatforms (CICS TG)
 * CICS VSAM Transparency for z/OS (CICS VT)
 * IBM Session Manager for z/OS (ISM)


Key: 

 * N denotes No, not supported at this release. 
 * Y denotes Yes, supported by this release and further fixpacks at this level. 
 * Vv.r.m.f denotes version.release.modification-level.function-level. This refers to the release or fixpack that introduced support for this CICS Explorer release, and continued support by further fixpacks at this level.








CICS Explorer is compatible with the operating environments described on the associated Detailed System Requirements for CICS Transaction Server [http://www.ibm.com/support/docview.wss?uid=swg27006382] web page. Click on your release of CICS Transaction Server, then on the next web page click on the link for CICS Explorer in the list of products. 
Each new release of CICS Explorer and CICS tools plug-ins for CICS Explorer are hosted on the z/OS Explorer download page [https://developer.ibm.com/mainframe/] and also the existing CICS Explorer download page [http://www.ibm.com/support/docview.wss?rs=1083&uid=swg24033579]. For download packages for previous releases of CICS Explorer, see the Downloads for older releases of IBM Explorer for z/OS and IBM CICS Explorer [http://www.ibm.com/support/docview.wss?uid=swg24031508] web page.

For information about getting maintenance for the CICS Explorer, see ordering maintenance for IBM CICS Explorer [http://www.ibm.com/support/docview.wss?uid=swg21380083].









IBM Explorer for z/OS (z/OS Explorer) 

Users of CICS Explorer V5.4 should use z/OS Explorer V3.1, available in the Mainframe Developer Center [https://developer.ibm.com/mainframe/]. [https://developer.ibm.com/mainframe/]

Users of CICS Explorer V5.3 should use z/OS Explorer V3.0.
Users of CICS Explorer V5.2 or V5.1.1 should use z/OS Explorer V2.1.0.5, or later fix packs.

If Installation Manager was used to install both z/OS Explorer and CICS Explorer then install the corresponding fix packs for CICS Explorer and z/OS Explorer at the same time. For more details see the instructions in the associated APAR documentation.
