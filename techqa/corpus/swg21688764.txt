Title: IBM Known Limitation: IBM Case Manager 5.2.x and Microsoft Internet Explorer versions prior to 11.0.10 - United States

Text:
icm 521; if002; 5210-if2 TECHNOTE (TROUBLESHOOTING)

PROBLEM(ABSTRACT)
 The Case Manager v5.2.x minimum required (minor) version of Internet Explorer 11 is 11.0.10 (with IE-11 update KB2962872). 

SYMPTOM
If you use an older minor version of Internet Explorer 11 as your browser, you might experience unexpected behavior in Case Manager. 

For example, in Case Manager Client, you might not be able to view cases.


RESOLVING THE PROBLEM
Use Internet Explorer 11 minor version 11.0.10 (with IE-11 update KB2962872) or the latest available update version.

RELATED INFORMATION
 Using earlier minor versions of IE 10 / 11 cause issues [http://www.ibm.com/support/docview.wss?uid=swg27043221]
