   Affected SAP machines (OS): All OSes
   
   Problem Description: MAI alerts for SAP not appearing in ITM
   workspace
      for customer's SAP Solution Manager 7.1 SP 12
      when email notification is not configured.
   
   Detailed Recreation Procedure:
      Configure mySAP agent on SAP Solution Manager with SP12.
      Do not configure the email notification.
      This will not show any data on TEP for MAI alerts.

   EMAIL notification in SAP solution manager

   the MAI alerts only it is greater that or equal version
   of Service Pack 10 SAP solution manager 7.1
