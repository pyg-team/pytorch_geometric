APAR STATUS
 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  In the clould management console (CMC) user interface, when
   defining servers in clusters, or updating SMTP properties, if a
   server which has a domain name extension of 9 or more characters
   is entered, the user interface rejects this entry. For example:
   mydomainname.ninechars is invalid and the user interface
   displays an error message stating "The value entered is not
   valid" and does not allow you to save this value.
   
   
    
   
   

LOCAL FIX
 *  Use IP address instead of hostname
