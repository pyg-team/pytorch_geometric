QUESTION
 Where can I find and how do I add the Data File Drivers (ODBC Drivers) for SPSS Statistics? 

ANSWER
The Data File Drivers (ODBC Drivers) for SPSS Statistics can be found at IBM Passport Advantage. 

After signing in, choose "Software Downloads and media access", and then "Download finder". Select your entitled product (SPSS Statistics) and specify your language and platform. Finally, click the "+" to expand the download options and select the IBM SPSS Statistics Data File Drivers. 

Once downloaded, read the documentation for complete and platform specific installation. 

In Windows, run the appropriate executable file and follow the instructions in the wizard. 

Next, go to Administrative Tools > Data Sources (ODBC). Next, select "Add" and choose the appropriate SPSS Statistics driver. The driver will now be available for use within SPSS Statistics.
