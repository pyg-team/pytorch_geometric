Title: IBM Error CRIMC1020E zip com.ibm.urbancode.urelease.swg 6.1.0.20140422 not found - United States

Text:
CRIMC1020E; com.ibm.urbancode.urelease.swg 6.1.0.20140422; not found TECHNOTE (TROUBLESHOOTING)

PROBLEM(ABSTRACT)
 IBM Installation Manager could not find the required file "com.ibm.urbancode.urelease.swg 6.1.0.20140422" when installing IBM UrbanCode Release. 

CAUSE
Installation Manager could not find the required files because you installed UrbanCode Release as a Trial package. 

RESOLVING THE PROBLEM
To resolve the issue, use the following instruction. 

1. Go to IBM Passport Advantage > Find IBM UrbanCode Release 6.1.0 Multiplatform Multilingual eAssembly (CRRL8ML) > Download a UrbanCode Release package that works with your system platform

2. Extract the file to a temp directory 

3. Stop your UrbanCode Release server 

4. Launch Installation Manager > Add UCR 6.1.0.x disktag.inf and UCR 6.1.0.0 disktag.inf to IBM Installation Manager's repository 

[/support/docview.wss?uid=swg21686470&aid=1] [/support/docview.wss?uid=swg21686470&aid=1]

5. Apply > OK > Update

6. Follow the Update Wizard.
