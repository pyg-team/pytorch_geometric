ERROR DESCRIPTION
 *  When using any of the standard Java Applets in legacy design
   elements with Java 7 Update 45, will fail with errors like the
   following:
   Missing Application-Name: manifest attribute for: [url]
   Missing Permissions manifest attribute for: [url]
   This is due to required updates in the Manifest in order to
   work, as detailed here:
   http://docs.oracle.com/javase/tutorial/deployment/jar/secman.htm [http://docs.oracle.com/javase/tutorial/deployment/jar/secman.htm]
   l
   To the end user they see a Java warning with "Application:
   UNKNOWN" which can be misinterpreted as an untrusted applet.
   
   
    
   
   

LOCAL FIX
 *  Use an earlier version of Java that does not make this check,
   or user education on the "Application: UNKNOWN" message.
   
   
    
   
   

PROBLEM SUMMARY
 *  Added information to the manifest files provided with the
    Domino applets to name the applets as well as set their
    privilege level.
