Performance considerations for Human Task Management list widgets with IBM Integration Designer (1499983) [http://www.ibm.com/support/docview.wss?uid=swg21499983&acss=dakc] 
Version: 7.5 
If you are familiar with the performance characteristics of WebSphere Process Server, you might experience an out-of-the-box performance degradation. This degradation might occur when you use Human Task Management list widgets, such as Tasks, Processes and Task Definitions, in Business Space on a new installation of IBM Business Process Manager Version 7.5 run time. The same performance considerations apply if you use the Human Task Management widgets with only business process defintion (BPD) processes and human services on a single application server, or a cluster. 

Installing IBM Integration Designer and the test environment with an existing IBM Rational Application Developer 8.0.x installation might result in unexpected options (1500111) [http://www.ibm.com/support/docview.wss?uid=swg21500111&acss=dakc] 
Version: 7.5 
When you simultaneously install IBM Integration Designer and the test environment on a system with an existing IBM Rational Application Developer 8.0.x installation, IBM Installation Manager only displays the option for you to select the same package group in which Rational Application Developer 8.0.x is installed. 

Cleaning up after profile creation failed during the test environment installation (1506588) [http://www.ibm.com/support/docview.wss?uid=swg21506588&acss=dakc] 
Versions: 7.5, 7.5.0.1, 7.5.1, 7.5.1.1 and 8.0 
After a profile creation failure, the system might contain partially configured profile and database tables, which have to be cleaned up before a new profile can be created. How can I clean up the system before re-creating a profile? 

Installation manager cannot create profile with remote DB2 on z/OS (1508354) [http://www.ibm.com/support/docview.wss?uid=swg21508354&acss=dakc] 
Version: 7.5 
When using a remote IBM DB2 on z/OS, the Installation Manager for IBM Integration Designer cannot create the profile for IBM Business Process Manager V7.5 successfully. 

Collect troubleshooting data for IBM Integration Designer installation problems (1566753) [http://www.ibm.com/support/docview.wss?uid=swg21566753&acss=dakc]
