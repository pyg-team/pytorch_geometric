APAR STATUS
 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  On PureApplication System v2.0, similar messages as seen below
   may be encountered when attempting to test the Call Home
   feature.
   
   CWZIP3534E Call Home connection test did not complete
   successfully, http error = 0
   
   CWZIP3509E The Call Home request failed while submitting a
   request to create a new service ticket. ECC Exception error
   received: WSWS3713E: Connection to the remote host
   [2002:90b:717:1::1000]:443 failed.Received the following error:
   Network is unreachable
   
   
    
   
   

LOCAL FIX

PROBLEM SUMMARY
 *  The error messages in the Call Home test function was caused by
   problems processing the Call Home configuration files used to
   connect to the IBM Problem Management Reporting system.
   
   
    
   
   

PROBLEM CONCLUSION
 *  A code fix is being provided in PureApplication System version
   2.0.0.1 to fix the Call Home configuration file issue.
