ABSTRACT
 IBM Security Guardium Support hosted this Open Mic on November 30, 2017 discussing configuring Guardium ATAP and EXIT to collect database traffic. 

CONTENT
This Open Mic discussed the usage of ATAP and EXIT. ATAP allows interception of encrypted traffic at the database server application level, and EXIT functionality making possible to intercept any traffic while eliminating the need for loading KTAP module into the kernel.
