Title: IBM No impact of CVE-2014-0160 ("HeartBleed") on MyISS portal and the IBM Security License Key and Download Center - United States

Text:
heartbleed; HeartBleed; CVE-2014-0160; OpenSSL; flaw; 1669815; XPU 30.040; xpu 30.040; password; passwords; user name; username; vulnerability; Flexera; DLC; LKC; download center; Download Center; License Key Center; license key center; myiss; MyISS; Myiss; 1.0.1; ssl; SSL; Flexera; flexera; IBM Security TECHNOTE (FAQ)

QUESTION
 Do you need to change your password for the MyISS portal and/or the IBM Security License Key and Download Center for protection from CVE-2014-0160 ("Heartbleed")? 

CAUSE
See http://web.nvd.nist.gov/view/vuln/detail?vulnId=CVE-2014-0160 [http://web.nvd.nist.gov/view/vuln/detail?vulnId=CVE-2014-0160] for specifics regarding this flaw. 

ANSWER
Confidential information such as user names and passwords on the MyISS portal or the IBM Security License Key and Download Center (also knows as LKC or DLC) were not affected by this OpenSSL vulnerability. As such there is not need to change the password unless the password is going to expire or has already expired. The password expiration is a system requirement and not related to this vulnerability. 

For detailed information on product coverage, please see related Technote 1669815 [http://www-01.ibm.com/support/docview.wss?uid=swg21669815] - SiteProtector and related products vulnerability status for CVE-2014-0160 "HeartBleed" OpenSSL flaw.
