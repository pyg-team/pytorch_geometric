Title: IBM WebSphere Application Server for z/OS servant ABEND when reading data by using a ReadListener on an upgraded Servlet 3.1 request - United States

Text:
WASV9RNotes TECHNOTE (TROUBLESHOOTING)

PROBLEM(ABSTRACT)
 Due to suspected memory corruption, a servant region for WebSphere Application Server for z/OS might end abnormally when it reads data by using a ReadListener interface API on an upgraded Servlet 3.1 request. 

SYMPTOM
Memory corruption can cause various symptoms, depending on what storage is overlaid. Your SYSOUT job log might contain a symptom similar to the following example:
