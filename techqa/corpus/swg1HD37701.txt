ERROR DESCRIPTION
 *  Japanese character in exported XML file of clash
   becomes     garbage
   Scenario:
   1. File -> Open any CATProduct
   2. Go to Space Analysis workbench if it's necessary
   3. Click Clash icon
   Type: Clearance + Contact + Clash 25mm

   sel all products
