ERROR DESCRIPTION
 *  Error Description&#65496;
   Customer reported the following issue which I was able to
   reproduce.
   
   If spell check and auto-signature are enabled in iNotes,
   creating an email using the numbered list would cause formating
   issue upon clicking Send.
   
   I was able to reproduce this on 853 using the Firefox 10.0.3
   and Firef12 browser.  This problem was not present in the IE9
   and Chrome 18.
   
   Steps to reproduce:
   
   Log into iNotes and enable auto Spell Check and Signature.
   Signature can be Plain text, Rich text or Notes.
   
   - Create new mail message from iNotes 8.5.3
   
   - Added list of names in memo
   Hilda
   Loc
   Brandon
   Joshua
   Powell
   
   - Highlight the list and select number format
   
   - Click on spell check icon and the format of numbered list is
   modified.
