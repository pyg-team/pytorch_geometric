ABSTRACT
 This document explains how to exploit WebSphere MQSeries and the MQSeries Link for SAP R/3 to avoid the need for files. 

CONTENT
There are two files associated with this topic. The PDF document describes the MQSeries Link for SAP R/3. The zip file contains 'C' and Windows 'dll' files that are Exits for the MQ Link for SAP R/3 (inbound and outbound servers). They need to be installed in the correct MQ Link directories and the MQ Link for SAP R/3 config should be updated.
