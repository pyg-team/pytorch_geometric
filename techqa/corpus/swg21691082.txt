Title: IBM Internet Explorer 11 Enterprise Mode is not supported by IBM Connections Docs - United States

Text:
 TECHNOTE (FAQ)

QUESTION
Is Internet Explorer 11 Enterprise mode supported by IBM Connections Docs?

ANSWER
Internet Explorer Enterprise Mode is not supported by IBM Connections Docs. The IE 11 Enterprise Mode causes the browser to emulate Internet Explorer 8 which is not a supported browser for IBM Connections Docs. 

For additional information see What is Enterprise Mode? [http://msdn.microsoft.com/en-us/library/dn640687.aspx]
