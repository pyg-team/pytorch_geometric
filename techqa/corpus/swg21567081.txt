Title: IBM Tivoli Logfile Agent WINEVENTLOGS value for setup and forwarded event logs. - United States

Text:
WINEVENTLOGS; 'Log File'; Agent; tecad_logfile.conf TECHNOTE (FAQ)

QUESTION
 What is the setting for monitoring setup and forwarded event logs by the ITM/Omnibus Logfile Agent on a Windows 2008 R2 64 bit system? 

ANSWER
Within the .conf file specified to be used by the Logfile Agent;
UseNewEventLogAPI=y 
WINEVENTLOGS=Setup,ForwardedEvents
