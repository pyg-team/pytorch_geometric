Title: IBM Security Bulletin: Open Source Apache CXF Vulnerablities affect IBM Tivoli Application Dependency Discovery Manager (TADDM) (CVE-2017-12624) - United States

Text:
 SECURITY BULLETIN

SUMMARY
 Vulnerabilities in Open Source Apache CXF affect IBM Tivoli Application Dependency Discovery Manager 

VULNERABILITY DETAILS

CVE-ID: CVE-2017-12624 [https://exchange.xforce.ibmcloud.com/vulnerabilities/135095]
DESCRIPTION: Apache CXF is vulnerable to a denial of service. By using a specially crafted message attachment header, a remote attacker could exploit this vulnerability to cause the AX-WS and JAX-RS services stop responding.
CVSS Base Score: 5.3
CVSS Temporal Score: https://exchange.xforce.ibmcloud.com/vulnerabilities/135095 [https://exchange.xforce.ibmcloud.com/vulnerabilities/135095] for more information
CVSS Environmental Score*: Undefined
CVSS Vector: (CVSS:3.0/AV:N/AC:L/PR:N/UI:N/S:U/C:N/I:N/A:L)

AFFECTED PRODUCTS AND VERSIONS
TADDM 7.3.0.2 - 7.3.0.4
