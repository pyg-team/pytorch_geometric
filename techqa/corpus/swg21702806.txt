Title: IBM sa agent does not report data from Solution Manager - United States

Text:
5724B97SOTR TECHNOTE (TROUBLESHOOTING)

PROBLEM(ABSTRACT)
 SAP agent functions on this SAP systems for all the subagents, except "Solution Manager." 
/IBMMON/ITM_MAIALRT_INX is not returning any results. 

DIAGNOSING THE PROBLEM
Confirm that ALRT_TIMESTAMP in the table /ibmmon/itm_alix, is the same as solution manager alert inbox (check in solman_workcenter in SAP GUI). 

 * Confirm that time on the agent machine and SAP are in sync. 

RESOLVING THE PROBLEM
The fix for this is provided in 7.1.1.1-TIV-ITM_SAP-IF0001 

 * Apply IBM Tivoli Composite Application Manager Agent for SAP Applications 7.1.1.1, Multilingual, Multiplatform (CN0MJML ) fix pack 7.1.1-TIV-ITM_SAP-FP0001 
 * And then apply 7.1.1.1-TIV-ITM_SAP-IF0001 

RELATED INFORMATION
