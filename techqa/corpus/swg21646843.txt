Title: IBM Unable to load several users from LDAP to IBM Connections EMPINST database via TDI scripts - United States

Text:
java.util.MissingResourceException: Can't find resource for bundle java.util.PropertyResourceBundle; key err_method_findEntry TECHNOTE (TROUBLESHOOTING)

PROBLEM
The Default Lotus Connection Assembly lines are being used to populate the IBM Connections EMPINST database from LDAP via TDI scripts. This process was successful for 448 users, it is failing for 12 users. Why?

SYMPTOM
Twelve users cannot be added to IBM Connections from LDAP using default TDI scripts


CAUSE
Duplicate mail IDs caused this issue. 

ENVIRONMENT
Connections version is 3.0.1.1 CR3; IC40; IC45; IC50
