ERROR DESCRIPTION
 *  Problem Description:
   
   
   ITCAM Extended Agent calculated wrong value for attribute of
   Buffer Cache Hit Ratio. This is due to the agent uses wrong
   parameter 'physical reads' from Oracle performance view
   v$systat. The correct parameter should be 'physical reads
   cache'.
   
   
   
   Detailed Recreation Procedure:
   
   Install and configure agent, then go to the navigator view from
   Node: Cache to Workspace: Buffer Cache, then check the value of
   '% Buffer Cache Hit'.
