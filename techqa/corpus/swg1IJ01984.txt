APAR STATUS
 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  When monitoring jobs using the DWC plan view, automatic refresh
   does not work if the TWS engine time and Browser time are not
   synchronized, and the TWS engine time is earlier than the
   Browser, for example TWS engine time is 11:34 and Browser time
   is 11:36.
   
   
    
   
   

LOCAL FIX
 *  Ensure TWS engine and Browser system times are synchronized.
   
   
    
   
   

PROBLEM SUMMARY
 *  PLAN VIEW DOES NOT REFRESH AUTOMATICALLY IF BROWSER AND TWS
   ENGINE TIME ARE NOT SYNCHRONZIED
   
   When monitoring jobs using the DWC plan view, automatic refresh
   does not work if the TWS engine time and Browser time are not
   synchronized, and the TWS engine time is earlier than the
   Browser, for example TWS engine time is 11:34 and Browser time
   is 11:36.
