Title: IBM There is no data in the Application Health workspace for ITCAM - United States

Text:
Application Health TEP ITCAM TECHNOTE (TROUBLESHOOTING)

THIS DOCUMENT APPLIES ONLY TO THE FOLLOWING LANGUAGE VERSION(S):
 US English 

PROBLEM(ABSTRACT)
 ITCAM TEP Application Health workspace is empty 

SYMPTOM
No applications are showing in the Application Health workspace in the TEP for ITCAM for WebSphere


CAUSE
Applications were discovered but application monitoring was not enabled for any applications.

DIAGNOSING THE PROBLEM
Access the Application Registry workspace in the TEP and verify if you see application monitoring is set to disabled for any or all of the applications listed. 

RESOLVING THE PROBLEM
Access the TEP, select the Navigator item "Application Health". Right click on this navigator item and select workspaces and then the workspace "Application Registry". Right click on the applications that are presented and have a status of disabled or discovered in the view then select "take action". Select the take action "Set_Application_Monitoring". All of the values for the take action will be filled in automatically. Change the value Mon_Statusfrom 0 to 1 to enable Application monitoring. You should now see the applications in the "Application Health" workspace.
