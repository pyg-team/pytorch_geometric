ERROR DESCRIPTION
 *  PureApplication System
   -
   CMM-77777703:Node Node % message: Expansion card 1, (Mezz Exp 1
   Fault) critical event does not generate a call home problem
   report as expected.
   
   
    
   
   

LOCAL FIX
 *  The following REST request will change their CMM event to a call
   home.
   POST /admin/resources/event_rules
   
   With this data:
   
   {
       "description": "change CMM-77777703:Node Node % message:
   Expansion card 1, (Mezz Exp 1 Fault) critical. to a call home",
       "search": "CMM-77777703:Node Node % message: Expansion card
   %, \\(Mezz Exp % Fault\\) critical.",
       "type": 8,
       "category": "CallHome"
   }
   
   
    
   
   

PROBLEM SUMMARY
 *  The event rules metadata has been updated so that this event
   generates a Call Home problem report.
   
   
    
   
   

PROBLEM CONCLUSION
 *  A code fix is being provided in PureApplication System and
   BlueMix Local System version 2.2.3.0.
