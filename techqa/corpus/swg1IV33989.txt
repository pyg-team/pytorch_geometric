ERROR DESCRIPTION
 *  The following are the Steps to make it happenning in the Out Of
   the Box
   Maxiomo versio 7.1.1.11
   1. Create a new workorder.
   2. Select the location REPAIR from the location value list
   3. Click on the ASSET field, Maximo will fill in an asset 11470
   for
   this location.
   4. Delete the asset from the ASSET field
   5. Click on any other field (e.g Configuration Item)
   5. Maximo will return the following error
   
   BMXAA4214E - An unknown error has occurred.
   Please contact your system administrator for assistance.
   Please confirm that this is a bug in Maximo and let us know if
   there is
   any workaround that we can apply to fix it.
