How can this issue be solved?



CAUSE
The Java Heap size is not set high enough.

RESOLVING THE PROBLEM
Follow the steps discussed in http://publib.boulder.ibm.com/infocenter/irm/v8r5m0/index.jsp?topic=/com.ibm.installingirm.doc/rminst0232.htm [http://publib.boulder.ibm.com/infocenter/irm/v8r5m0/index.jsp?topic=/com.ibm.installingirm.doc/rminst0232.htm]


These steps help increase the JVM max heap size from the default of 1 GB.
