Title: IBM Netcool/OMNIbus Integrations Release Notice - Netcool/OMNIbus Gateway for SNMP (nco-g-snmp-7_0) - United States

Text:
release notice; downloads; Gateway for SNMP; nco-g-snmp NEWS

ABSTRACT
 A new version of the Netcool/OMNIbus Gateway for SNMP will be available to download from July 28, 2016. 

CONTENT
 

This version of the gateway has been updated to add the following functionality:

 * RFE 20320: Forwarding configurable IP addresses within SNMP traps
 * RFE 76266: Gateway now able to log PDU Request ID values 

The Netcool/OMNIbus V8.1 version of this gateway supports the following Operating Systems:

AIX, Linux, Solaris, Windows, and zLinux.

You can download the installation package from the IBM Passport Advantage website: 
www.ibm.com/software/howtobuy/passportadvantage/pao_customers.htm [http://www.ibm.com/software/howtobuy/passportadvantage/pao_customers.htm]

You can access the latest version of the guide written for this gateway on the IBM Knowledge Center:

www-01.ibm.com/support/knowledgecenter/#!/SSSHTQ/omnibus/gateways/snmpgw/wip/concept/snmpgw_intro.html [http://www-01.ibm.com/support/knowledgecenter/#!/SSSHTQ/omnibus/gateways/snmpgw/wip/concept/snmpgw_intro.html]

Netcool/OMNIbus V8.1

If you are running Netcool/OMNIbus V8.1, a single package contains versions of the gateway for all supported Operating Systems. To obtain this new package, use the Find by part number field to search for part number: CND8GEN
