APAR STATUS
 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  The following error message is displayed when using the
   Liberty SSL
   
   [ERROR ? ? ? ? ] CWWKO0801E: Unable to initialize SSL connection
   Unauthorized access was denied or security settings have
   expired. Exception is javax.net.ssl.SSLHandshakeException: no
   cipher suites in common.
   
   This error message points the problem to something like
   password issue, but real problem is the
   keystore is lack of personal certificate.
   
   From the error message, there is not a clear idea on which part
   may cause the problem.
   
   
    
   
   

LOCAL FIX

   * USERS AFFECTED:  All users of IBM WebSphere Application      *
   *                  Server Liberty Profile                      *
   ****************************************************************

   *                      CWWKO0801E.                             *
   ****************************************************************
   * RECOMMENDATION:                                              *
   ****************************************************************
   Improve the user action for message
   CWWKO0801E.
   
   
    
   
   

PROBLEM CONCLUSION
 *  Updated the messages user action to suggest one of the
   possible problems is a missing certificate.
