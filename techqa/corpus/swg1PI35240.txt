PROBLEM CONCLUSION
 *  ANALYSIS:
   The "ReportItems" report is used to determine the number of
   Failed Items for the syndication pair to be displayed in the
   table.
   
   When this report is missing a NullPointerException is thrown
   when attempting to calculate the number of failed items for the
   pair, causing rendering of the Syndication Portlet to fail.
   
   SOLUTION:
   Calculation of the number of failed items for a syndication pair
   
   has been updated to only attempt the calculation if the required
   
   "ReportItems" report exists.
