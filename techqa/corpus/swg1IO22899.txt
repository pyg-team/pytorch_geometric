APAR STATUS
 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  When IBM Content Navigator for MS Office is used to search
   documents using teamspace search, and a stored template defined
   in Workplace XT is used, and refined, the refinements are not
   applied.
   
   This results in an incorrect set of documents returned from the
   search.
   
   
    
   
   

LOCAL FIX
 *  Redefine the Search template using IBM Content Navigator or use
   general search, i.e. not the teamspace search.
