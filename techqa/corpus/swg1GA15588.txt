APAR STATUS
 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  When capturing traffic from SQL Server 2014 DB, DB Username
   shows blank for traffic using DB_PROTOCOL TDS and
   DB_PROTOCOL_VERSION 9.0.
   
   
    
   
   

LOCAL FIX
 *  Install sniffer patch 4046.
   
   
    
   
   

PROBLEM SUMMARY
 *  DB user blank for MS SQL server 2014.
   
   Cause: Several clients use TLS v1.2 but only v1.0 and v1.1 is
   supported.
   
   Solution: Support for TLS v1.2 is added
   
   
    
   
   

PROBLEM CONCLUSION
 *  Fix is available in v9 sniffer patch v4046 or higher patch
   level.
