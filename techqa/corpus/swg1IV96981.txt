APAR STATUS
 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  Details:  This APAR will be the place holder for any issue raise
   during the the certification. This APAR includes:
   
   1. Removal of the Java Alcatel5620SamCollector support for NSP
   NFM-P Java
   Messaging System (JMS). The Java Alcatel5620SAMCollector has
   removed support for these
   features due to concerns raised in the 2016 certification. (JMS
   codes is not removed, but
   commented out in case someone needs to use it)
   2. High Availability (HA) features. This feature rely on the JMS
   in receiving the notification
   of the HA activity. Since JMS feature is removed, HA feature
   also need to be removed
   3. Null pointer exception for MPLS interface (adding checking
   for  terminatedPortPointer & nodeId)
   4. New Nokia MPR 9500 device support
