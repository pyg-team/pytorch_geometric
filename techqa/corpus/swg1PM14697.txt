APAR STATUS
 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  Views are missing in TEP for ITCAM for Application Diagnostics
   7.1. The following error is displayed:
   
   javax.xml.parsers.FactoryConfigurationError:Provider
   org.apache.xerces.jaxp.DocumentBuilderFactoryImpl not found
   
   The exception is happening because the verion of Java being used
   (SUN JRE 1.5) isn't setting up the property
   javax.xml.parsers.DocumentBuilderFactory.
   
   The property defaults to its IBM value instead of the Sun value,
   which is what is causing the problem.
   
   We need to modify the code to enforce that
   javax.xml.parsers.DocumentBuilderFactory is properly set at
   runtime.
   
   
    
   
   

LOCAL FIX
 *  In the meantime, please try the following workaround in file
   /tivoli/ibm/itm6.2/aix536/cw/kwj_tepui.jar.jnlp
   
     <resources>
       <j2se version="1.5+" />
       <property name="javax.xml.parsers.DocumentBuilderFactory"
   value="com.sun.org.apache.xerces.internal.jaxp.DocumentBuilderFa
   ctoryImp
   "/>
       <jar href="classes/kwj_tepui.jar"/>
     </resources>
   
   
   Please note that this workaround will only work with TEP
   WebStart clients using Sun's JVM.
