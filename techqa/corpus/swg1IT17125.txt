APAR STATUS
 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  The user upgraded to a fixpack level that lifts HSM component
   firmware from level 1.2 to 2.2. At that 2.2 level, the user
   failed to import new keys into the HSM.
   
   
    
   
   

LOCAL FIX
 *  Downgrade HSM component firmware level to 1.2 as described at
   http://www.ibm.com/support/docview.wss?uid=swg21984581. [http://www.ibm.com/support/docview.wss?uid=swg21984581.]
   
   
    
   
   

PROBLEM SUMMARY
 *  Importing of private keys into a HSM module fails after firmware
    upgrade.
   
   
    
   
   

PROBLEM CONCLUSION
 *  Fix is available in 7.0.0.16, 7.1.0.13, 7.2.0.10, 7.5.0.4,
   7.5.1.3 and 7.5.2.0
   
   For a list of the latest fix packs available, please see:
   http://www-01.ibm.com/support/docview.wss?uid=swg21237631 [http://www-01.ibm.com/support/docview.wss?uid=swg21237631]
