ERROR DESCRIPTION
 *  Problem Description:
   kd4agent is 32-bit process. On AIX, the virtual address space
    of 32-bit process is 256M segments. The kd4agent process will
    crash when it tries to allocate more space than the limit. The
    core file is truncated to 256M.
   
   Detailed Recreation Procedure:
    1) Run ITCAM for SOA agent process on AIX when processing huge
       data from data collector(s).
    2) The kd4agent process crashes with core file being truncated
       to 256M.
    3) No data available on ITM server from the agent.
