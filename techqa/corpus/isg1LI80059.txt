 * CLOSED AS NEW FUNCTION.
    
   
   

ERROR DESCRIPTION
 *  Wave currently supports TLS V 1.2 with other communications, but
   only TLS V 1.0 with the connection to LDAP for the GUI logon.
   TLS V1.1 and V1.2 support is needed for that connection.
   
   
    
   
   

LOCAL FIX

PROBLEM SUMMARY
 *  IBM Wave is supporting only up to TLS 1.0 when connecting to
   LDAP for the GUI logon.
   
   
    
   
   

PROBLEM CONCLUSION
 *  Support for TLS 1.1 and TLS 1.2 provided with this APAR.  APAR
   interim fix available on request, otherwise it is planned to b
   e included with IBM Wave V 1.2 FP 11.
