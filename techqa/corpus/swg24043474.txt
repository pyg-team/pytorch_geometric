Title: IBM PI73197:Enable Java 8 Support for EJBDeploy on WebSphere Application Server - United States

Text:
PI73197; PI73197; PI73197 DOWNLOADABLE FILES

ABSTRACT
 Enable Java 8 Support for EJBDeploy on WebSphere Application Server 8.5.5.9 or later.
NOTE: After this fix is applied, EJBDeploy can no longer be used with Java 6 

DOWNLOAD DESCRIPTION
PI73197 resolves the following problem:

ERROR DESCRIPTION: 
Java 8 is not supported for EJBDeploy on WebSphere Application Server 8.5.5.9 or later 

LOCAL FIX: 

PROBLEM SUMMARY

USERS AFFECTED:
All users of IBM WebSphere Application Server

PROBLEM DESCRIPTION:
EJBDeploy does not run on Java 8

RECOMMENDATION:
None

EJBDeploy does not run on Java 8

PROBLEM CONCLUSION: 
The fix for this APAR is targeted for inclusion in 2Q2018.
