Title: IBM Security Bulletin: Vulnerabilities have been identified in IBM WebSphere Application Server shipped with IBM Tivoli Network Manager IP Edition - United States

Text:
 SECURITY BULLETIN

SUMMARY
 Websphere Application Server (WAS) is shipped as a component IBM Tivoli Network Manager IP Edition. Information about a security vulnerability affecting WAS has been published in a security bulletin. 

VULNERABILITY DETAILS
Please consult following security bulletins in WebSphere Application Server for vulnerability details and information about fixes. 

 * Potential cross-site scripting in the Admin Console for WebSphere Application Serve (CVE-2016-8934) [http://www-01.ibm.com/support/docview.wss?uid=swg21992315] 
 * Potential Information Disclosure vulnerability in WebSphere Application Server (CVE-2016-5986) [http://www-01.ibm.com/support/docview.wss?uid=swg21990056] 
 * There is a potential information disclosure in WebSphere Application Server using malformed SOAP requests on WebSphere Application Server. [http://www-01.ibm.com/support/docview.wss?uid=swg21991469] 
 * Certificates will need to be converted to use SHA256withRSA in WebSphere Application Server [http://www-01.ibm.com/support/docview.wss?uid=swg21959568] 
 * Multiple vulnerabilities in IBM® Java SDK affects WebSphere Application Server October 2016 CPU (CVE-2016-5573, CVE-2016-5597) [http://www-01.ibm.com/support/docview.wss?uid=swg21993440] 
 * Multiple vulnerabilities in IBM® Java SDK affects WebSphere Application Server July 2016 CPU (CVE-2016-3485) [http://www-01.ibm.com/support/docview.wss?uid=swg21988339] 
 * Redirect HTTP traffic vulnerability may affect IBM HTTP Server (CVE-2016-5387) [http://www-01.ibm.com/support/docview.wss?uid=swg21988019] 
 * Apache Commons FileUpload Vulnerability affects WebSphere Application Server (CVE-2016-3092) [http://www-01.ibm.com/support/docview.wss?uid=swg21987864] 
 * Vulnerabilities in Apache Struts affects IBM WebSphere Application Server [http://www-01.ibm.com/support/docview.wss?uid=swg21985995] 
 *  [http://www-01.ibm.com/support/docview.wss?uid=swg21985995]Potential denial of service with SIP Services (CVE-2016-2960) [http://www-01.ibm.com/support/docview.wss?uid=swg21984796] 
 * Bypass security restrictions in WebSphere Application Server (CVE-2016-0385) [http://www-01.ibm.com/support/docview.wss?uid=swg21982588] 
 * HTTP Response Splitting in WebSphere Application Server (CVE-2016-0359) [http://www-01.ibm.com/support/docview.wss?uid=swg21982526] 
 * Potential security vulnerability in IBM WebSphere Application Server if FIPS 140-2 is enabled [http://www-01.ibm.com/support/docview.wss?uid=swg21979231] 
 * Multiple Denial of Service vulnerabilities with Expat may affect IBM HTTP Server [http://www-01.ibm.com/support/docview.wss?uid=swg21988026]
