Title: IBM Adapter connector project migration fails in IBM Integration Designer V8.5 - United States

Text:
migration; integration designer; adapter; jca; connector; JR48037 TECHNOTE (TROUBLESHOOTING)

PROBLEM(ABSTRACT)
 When the "Migrate connector project" wizard is invoked on an Adapter Connector project version prior to V7.5.0.5, which is the version of Adapters shipped with IBM Integration Designer V8.5, the migration fails. 

SYMPTOM

CAUSE
By default IBM Integration Designer V8.5 is bundled with Javaâ„¢ 7. The Adapter migration code expects the class com.ibm.jvm.util.ByteArrayOutputStream ,which is available in Java 6 but not in Java 7. 
Business Process Manager runtime V8.5 only supports Java 6.


RESOLVING THE PROBLEM
Refer to APAR JR48037. Contact IBM support for the migration fix.
