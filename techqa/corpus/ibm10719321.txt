Title: IBM Encrypt the internal perimeter server connection? - United States

Text:
 HOW TO

SUMMARY
Is it possible to configure the connection between Sterling B2B Integrator and the remote perimeter server?

STEPS
It is not possible to encrypt this connection in the current product (current release: 5.2.6.3_6).

Note that this is a solely internal connection. Remote partners and users do not connect to the perimeter server or use this port. Connections using protocols such as FTP, HTTP, OFTP2 and Connect:Direct can be encrypted.
