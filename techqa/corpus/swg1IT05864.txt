APAR STATUS
 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  For some use cases, occasional performace reduction or intervals
   of extended transaction processing latencies might be observed
   with firmware 6.0.0 and higher.
   
   
    
   
   

LOCAL FIX
 *  See TechNote "IBM DataPower firmware version 6.0.x Enhanced
   Memory Management feature" at
   http://www.ibm.com/support/docview.wss?uid=swg21688030 [http://www.ibm.com/support/docview.wss?uid=swg21688030]
   
   
    
   
   

PROBLEM SUMMARY
 *  Affected are Datapower firmware releases 6.0.0 and up.
   
   In some use cases, especially ones with large dynamic of memory
   usage, the appliance might manifest occasional or permanent
   slowdown. The slowdown might be significant.
   
   
    
   
   

PROBLEM CONCLUSION
 *  Fix is available in 6.0.0.12, 6.0.1.8 and 7.0.0.5.
   
   For a list of the latest fix packs available, please see:
   http://www-01.ibm.com/support/docview.wss?uid=swg21237631 [http://www-01.ibm.com/support/docview.wss?uid=swg21237631]
   
   
    
   
   

TEMPORARY FIX
 *  For 6.0.1 and 7.0.0 firmware, see the following technote:
   http://www-01.ibm.com/support/docview.wss?uid=swg21688030 [http://www-01.ibm.com/support/docview.wss?uid=swg21688030]
