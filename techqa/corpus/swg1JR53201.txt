APAR STATUS
 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  IBM WebSphere Integration Developer and IBM Integration Designer
   can be affected by the RC4 "Bar Mitzvah" attack
   (CVE-2015-2808).
   
   PRODUCTS AFFECTED:
   IBM Integration Designer
   WebSphere Integration Developer
   
   
    
   
   

LOCAL FIX
 *  For IBM Integration Designer V8.5.0.1, V8.5.5, and V8.5.6, see
   "Security Bulletin: Vulnerability in RC4 stream cipher affects
   IBM Integration Designer and WebSphere Integration Developer
   (CVE-2015-2808)"
   (http://www.ibm.com/support/docview.wss?uid=swg21882974) for the
    workaround.
   
   
    
   
   

PROBLEM SUMMARY
 *  No additional information is available.
   
   
    
   
   

PROBLEM CONCLUSION
 *  A fix is available for the the latest fix pack of all supported
   releases of WebSphere Integration Developer and IBM Integration
   Designer that disables RC4 ciphers at the Java runtime level.
   
   On Fix Central (http://www.ibm.com/support/fixcentral), search
   for JR53201:
   
       1. Select IBM WebSphere Integration Developer or IBM
   Integration Designer from the product selector, the installed
   version to the fix pack level, and your platform, and then click
   Continue.
       2. Select APAR or SPR, enter JR53201, and click Continue.
   
   When you download fix packages, ensure that you also download
   the readme file for each fix. Review each readme file for
   additional installation instructions and information about the
   fix.
   
   
    
   
   

TEMPORARY FIX
 *  For IBM Integration Designer V8.5.0.1, V8.5.5, and V8.5.6, see
   "Security Bulletin: Vulnerability in RC4 stream cipher affects
   IBM Integration Designer and WebSphere Integration Developer
   (CVE-2015-2808)"
   (http://www.ibm.com/support/docview.wss?uid=swg21882974) for the
    workaround.
