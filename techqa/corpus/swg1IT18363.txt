ERROR DESCRIPTION
 *  PureApplication System
   -
   After a firmware update to PureApplication System version
   2.2.2.1, the customer received the following error while trying
   to test the Call Home function:
   CWZIP3549E The Call Home request failed while submitting a
   heartbeat request to IBM. Could not retrieve IBM PureApplication
   shared serial.
   
   
    
   
   

LOCAL FIX

PROBLEM SUMMARY
 *  A metadata property that the Call Home function uses
   was not being found, causing the error. Code is being
   added to work around the missing property.
   
   
    
   
   

PROBLEM CONCLUSION
 *  A code fix is being provided in PureApplication System and
   BlueMix Local System version 2.2.3.0.
