ERROR DESCRIPTION
 *  Page data is not generated when the page type is changed through
   batch structure in verify client of Datacap Navigator. It's
   regression in Datacap 9.0.1. This fix will be contained to
   Datacap 9.0.1 fp1.

PROBLEM CONCLUSION
 *  It's regression in Datacap 9.0.1. It is fixed in Datacap 9.0.1
   fp1. When customer change page type in batch tree, the field
