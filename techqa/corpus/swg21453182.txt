 TECHNOTE (FAQ)

QUESTION
 How to invoke a WSDL using an external tool after applying self signed certificate? 

CAUSE
While invoking a method in a WSDL using an external tool, it will give an message stating "The hostname in the certificate is invalid or does not match".

ANSWER
Generate and apply self signed certificate


 1. When creating the certificate please use the data ip address 
 2. Once the certificate is created, you will need at add it in the browser while doing it you will need to store the certificate in Trusted Root Certification Authorities.  *  By clicking on View Certificate Install certificate On Certificate Import wizard need to click on next Select the second option "Place all certificates in the following store click on Browse and then select Trusted Root Certification Authorities Once done click ok and then finsh, You will get a warning message to validate the certificate and shall as to install the certificate please click on Yes. (see attached screenshot: Invoking a WSDL.jpg) 
    
    
 3. Once done you can invoke the WSDL using the external tool
