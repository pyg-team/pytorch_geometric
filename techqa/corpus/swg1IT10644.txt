APAR STATUS
 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  On downgrading DataPower XI52 firmware via WebGUI, the user
   observed an uncorrect device type being mentioned in a pop-up
   warning message. ("Firmware downgrade from 'XI50.x.x.x.x' to
   version 'XI50.y.y.y.y' might affect existing configuration.")
   
   
    
   
   

LOCAL FIX

PROBLEM SUMMARY
 *  DataPower during some upgrade/downgrade paths will display XI50
   where XI52 should be displayed.
   
   
    
   
   

PROBLEM CONCLUSION
 *  Fix is available in 6.0.1.13, 6.0.0.17, 7.0.0.10, 7.1.0.7 and
   7.2.0.1
   
   For a list of the latest fix packs available, please see:
   http://www-01.ibm.com/support/docview.wss?uid=swg21237631 [http://www-01.ibm.com/support/docview.wss?uid=swg21237631]
