Title: IBM Enterprise Content Management System Monitor (ECM SM) running the NetworkPing Monitor - United States

Text:
 TECHNOTE (FAQ)

QUESTION
 IBM Enterprise Content Management System Monitor (ECMSM) is running the NetworkPing Monitor but why is the monitor reporting status as "not_ok"? 

CAUSE
If the System Administrators my disable the Network Ping, the system cannot ping itself.
The command "ping <IP Address>" returns "Request timed out."

ANSWER
If Network Ping fails from the command line, check with the Windows System Administrator. 

Sometimes it is disabled for Network Security reasons.
