APAR STATUS
 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  When using a SFTP Server Front Side Handler with a default
   directory that does not end in a slash, DataPower might
   unexpectedly and immediately restart when receiving certain
   combinations of requests from an SFTP client.
   
   
    
   
   

LOCAL FIX

PROBLEM SUMMARY
 *  Affected is the use of the SFTP Server Front Side Handler.  When
    using a SFTP Server Front Side Handler with a default directory
    that does not end in a slash, DataPower may unexpectedly and
   immediately reload when receiving certain combinations of
   requests from an SFTP client.
   
   
    
   
   

PROBLEM CONCLUSION
 *  Fix is available in 7.1.0.21, 7.2.0.18, 7.5.0.12, 7.5.1.11,
   7.5.2.11 and 7.6.0.4
   For a list of the latest fix packs available, please see:
   http://www-01.ibm.com/support/docview.wss?uid=swg21237631 [http://www-01.ibm.com/support/docview.wss?uid=swg21237631]
