Title: IBM How to Find Out if CHGASPACT was Successful - United States

Text:
 TECHNOTE (TROUBLESHOOTING)

PROBLEM(ABSTRACT)
 This document explains how to find out if CHGASPACT was successful or not. 

RESOLVING THE PROBLEM
 

The following VLOGs are created and can be used to determine if the quiesce was successful or if it failed:

JO cuts a VL0D00218B which indicates a successful quiesce. 
JO cuts a VL0D00218A indicates a failed quiesce.

To view the VLOGS, you should do the following:

1. Issue the STRSST command.
2. Select Option 1 - Start a service tool. 
3. Select Option 5 - Licensed Internal Code log. 
4. Select Option 1 - Display Entries.
5. Specify major code of 0D00 and minor code of 218B or 218A.
