 DOWNLOADABLE FILES

ABSTRACT
 This fix upgrades the IBM HTTP Server (IHS) that is provided with IBM Tivoli Monitoring portal server version 6.30.


DOWNLOAD DESCRIPTION
This fix upgrades the embedded IBM HTTP Server (IHS) which is shipped as part of the IBM Tivoli Monitoring portal server, to version 8.0.0.11 plus additional Interim Fixes referred to as Interim Fix Block 1. It addresses the security vulnerability as documented in the Security bulletin: http://www.ibm.com/support/docview.wss?uid=swg21970056 [http://www.ibm.com/support/docview.wss?uid=swg21970056]
