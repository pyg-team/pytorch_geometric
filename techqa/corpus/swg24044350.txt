PROBLEM DESCRIPTION: 
Ship Java 8 SR5 FP6 for WebSphere Application Server traditional bundled Java 8 

RECOMMENDATION:
Apply this i-fix to your SDK to upgrade your SDK 8 to SDK8 SR5 FP6.
NOTE:
This iFix can ONLY be applied on:
a) WebSphere Application Server traditional bundled Java 8 V855.11 and above with bundled Java 8 installed


PROBLEM CONCLUSION: 
Applying this apar will upgrade your SDK 8 to SDK8 SR5 FP6 (Oracle 1.8.0_u151 ).

The fixes in this Java SDK are found here:
https://www.ibm.com/developerworks/java/jdk/fixes/8/#SR5FP6 [https://www.ibm.com/developerworks/java/jdk/fixes/8/#SR5FP6]

When downloading the fix from fix central, please also download the readme.txt with instructions for installation.

This Java SDK level was minimally tested with WebSphere 8.5.5.12.
