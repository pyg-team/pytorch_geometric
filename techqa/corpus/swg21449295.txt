Title: IBM HTTP Server: SSL handshake failure, no website found for IP address [X.X.X.X] - United States

Text:
 TECHNOTE (TROUBLESHOOTING)

PROBLEM
HTTP Server: SSL handshake failure, no website found for IP address

SYMPTOM
Above error message seen after implementing SSL.


CAUSE
In this case, it was found that the issue was caused due to Incorrect .kyr file mentioned in the Internet sites document on the server. 

Below error message was seen on Domino server, after implementing SSL:
HTTP Server: SSL handshake failure, no website found for IP address [X.X.X.X]


RESOLVING THE PROBLEM
 

After correcting the SSL Key file name field in the Internet sites document, and restarting the HTTP task, the above issue got resolved.
