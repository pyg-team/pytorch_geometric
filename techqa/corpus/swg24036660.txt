INSTALLATION INSTRUCTIONS
 The two common ways to install this update are as follows: 


If the product is already installed on your system, you can install this update by using the Update Packages wizard in IBM Installation Manager. Note: you will be prompted for an IBM id when using the IBM update repository. If you do not have an IBM id, you can create one at https://www.ibm.com/account/profile/us?page=reg [https://www.ibm.com/account/profile/us?page=reg]. For general instructions on installing an update, see the Installing > Installing Rational Developer for i > Updating installed product packages section in the Information Center [http://pic.dhe.ibm.com/infocenter/iadthelp/v9/index.jsp]. If you are installing the product for the first time or if you are installing an additional instance of it, you can install this update at the same time that you install the product. Click Check for Other Versions and Extensions on the Install page of the Installing Packages wizard in IBM Installation Manager. 

By default, the update is installed directly from the IBM update repository; however, it is also available for download below. If you download the update, unzip DISK1 and DISK2 to a local folder. Note: unzip both zips to the same local folder. Then register the folder as a repository in IBM Installation Manager. See the Installing > Installing Rational Developer for i > Preinstallation tasks > Installation repositories > Setting repository preferences in IBM Installation Manager section in the Information Center [http://pic.dhe.ibm.com/infocenter/iadthelp/v9/index.jsp]. Once it is registered, you can perform the update. For general instructions on installing an update, see the Installing > Installing Rational Developer for i > Updating installed product packages section in the Information Center [http://pic.dhe.ibm.com/infocenter/iadthelp/v9/index.jsp].

If you have multiple developer machines, see How to install Rational Developer for i at the latest level on multiple developer machines [http://www-01.ibm.com/support/docview.wss?uid=swg21668005].

KNOWN SIDE EFFECTS
 Review the Known problems section of the Rational Developer for i 9.0 [http://www.ibm.com/support/docview.wss?uid=swg24034857#KNOWN]. 


As part of IBM i 7.1 Technology Refresh 7, a major enhancement to RPG IV is being announced: Free-form control, file-declaration, data-declaration, and procedure statements. The enhancement is provided by 7.1 RPG Compiler PTF SI51094. 

With the Rational Developer for i 9.0.1 update, the product has incorporated support for the RPG IV enhancement. This includes updates to ILE RPG syntax checking, program verification, editor tokenization, and the Outline view. 

There is some functionality which does not fully support the language changes such as the New Procedure wizard, the New D-specification wizard, the New Java Method Call wizard, and certain content assist cases.

HELPFUL HINTS FOR OBTAINING TECHNICAL ASSISTANCE:
Before you contact IBM Rational Software Support, gather the background information that you need to describe the problem. When you describe a problem to an IBM software support specialist, be as specific as possible and include all relevant background information so that the specialist can help you solve the problem efficiently. To save time, know the answers to these questions: 

 * What software versions were you running when the problem occurred? 
 * Do you have logs, traces, or messages that are related to the problem? 
 * Can you reproduce the problem? If so, what steps do you take to reproduce it? 
 * Is there a workaround for the problem? If so, be prepared to describe the workaround. 

If you have helpful information to diagnose or identify the problem on your system, you can provide this data by following the instructions to exchange information with IBM Technical Support [http://www.ibm.com/software/support/exchangeinfo.html].
