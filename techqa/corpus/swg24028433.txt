ABSTRACT
 This document describes how to download IBM WebSphere MQ File Transfer Edition Version 7.0.3 eAssembly images from the Passport Advantage website. 

DOWNLOAD DESCRIPTION


Terminology

WebSphere MQ File Transfer Edition Version 7.0.3 is supplied in an eAssembly. Contents

This document provides a list of the installation images available in the eAssembly, and describes how to download the eAssembly images from the Passport Advantage website. 

Related product documentation 



For information about the installation process, refer to the Installing WebSphere MQ File Transfer Edition [http://www.ibm.com/support/knowledgecenter/SSEP7X_7.0.4/com.ibm.wmqfte.doc/installing_planning.htm] information supplied in the product documentation. [http://publib.boulder.ibm.com/infocenter/wmqfte/v7r0/topic/com.ibm.wmqfte.install.doc/installing_planning.htm] 

Downloading the relevant eAssembly images from the Passport Advantage website

 1. To get started with WebSphere MQ File Transfer Edition, download the following eAssembly images:  * WebSphere MQ File Transfer Edition V7.0.3 Quick Start Guide for Multiplatform Multilingual
     * WebSphere MQ File Transfer Edition V7.0.3 Server for your platform
     * WebSphere MQ File Transfer Edition V7.0.3 Documentation and Tools for your platform
     * WebSphere MQ V7.0.1 for your platform
    
    
    
 2. Go to the Download package table below and follow the link to the Passport Advantage website. 
 3. Sign in to download all of the images that you need. 
 4. Unpack each of the parts onto your system.
