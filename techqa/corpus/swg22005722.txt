Title: IBM Security Bulletin: Weaker than expected security in IBM API Connect Developer Portal (CVE-2017-6922) - United States

Text:
 SECURITY BULLETIN

SUMMARY
 IBM API Connect Developer Portal could allow a remote attacker to bypass security restrictions, caused by the failure to restrict access to the private file system. 

VULNERABILITY DETAILS
CVEID: CVE-2017-6922 [http://cve.mitre.org/cgi-bin/cvename.cgi?name=CVE-2017-6922]
DESCRIPTION: Drupal could allow a remote attacker to bypass security restrictions, caused by the failure to restrict access to the private file system. By sending a specially-crafted request, an attacker could exploit this vulnerability to access the private file system.
CVSS Base Score: 5.3
CVSS Temporal Score: See https://exchange.xforce.ibmcloud.com/vulnerabilities/127568 [https://exchange.xforce.ibmcloud.com/vulnerabilities/127568] for the current score
CVSS Environmental Score*: Undefined
CVSS Vector: (CVSS:3.0/AV:N/AC:L/PR:N/UI:N/S:U/C:N/I:L/A:N)
