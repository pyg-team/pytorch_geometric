ERROR DESCRIPTION
 *  Impact Finter analyzer throws an error if the field's value has
   a '
   For example:
   Filter: Node = 'O\'Neil'
   When you click on the analyzer, it throws an error.
   
   
    
   
   

LOCAL FIX

PROBLEM SUMMARY
 *  Event Anlayer in the Event Reader throws an excpetion if the
   restriction filter has a single qout. For example:
   if you create an event reader with a restriction filter mapping
   as following:
   Node = 'O\'Nile';
   Then click on Analyze, it will throw an exception
