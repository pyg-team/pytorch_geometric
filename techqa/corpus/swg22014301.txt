 TECHNOTE (FAQ)

QUESTION
 What is the proper upgrade order of prerequisite software and IBM® Content Manager Enterprise Edition? Which software combination and upgrade path is supported? What issues might occur when upgrading software? 

ANSWER
Changing prerequisite software can require special procedures or a certain sequence of steps to remain in a supported IBM® Content Manager Enterprise Edition configuration. It requires an understanding of the prerequisite software maintenance procedures within the context of an IBM Content Manager V8.6 system. The information provided in the PDF attachment encapsulates the knowledge needed to apply maintenance for various prerequisite software components. This document discusses the following issues: 

 * Upgrading operating systems 
 * Upgrading DB2 database software 
 * Upgrading IBM Content Manager Enterprise Edition from V8.4.5.x to V8.6
