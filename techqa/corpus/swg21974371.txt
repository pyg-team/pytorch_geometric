Title: IBM JazzSM fail security scan - United States

Text:
WAS_PASSWORD fsconfig.xml TECHNOTE (FAQ)

QUESTION
 JazzSM default password resulting in fail security scan. 

ANSWER

To avoid security issue, please remove the dummy password in fsconfig.xml file and run the scan.

Edit '/opt/IBM/JazzSM/install/fsconfig.xml' and make password empty.

<property name="WAS_PASSWORD" value="" /> 

Removing default password will not be harmful.
