ERROR DESCRIPTION
 *  Snapshot to Excel failed with following error message: Message
   from webpage: Ein Fehler ist aufgetreten
   
   
   Environment:
   Windows 2008 R2
   TM1 10.2.2 FP2 IF1024
   Excel 2013
   IE11
   German environment
   
   Steps to reproduce the behavior
   
   1. open TM1 Merck model (StartCockpit will open)
   2. select and mark Preselection for report
   3. push button Yearly Planning ( Yearly Planning FCI 2015 - CH
   sheet open)
   4. select element Lowest level in picklist Drilldown hierachy -
   Product
   5. select element Lowest level in picklist Drilldown hierachy -
   Country
   6. select element NO in picklist Zero Suppresion
   7. push button Refresh Planning Maks
   8. sheet will be refreshing and you will see all elements
   9. take snapshot to Excel from current sheet view (export more
   than 7100 rows's)
   
   effect:
   after few time (in my test after 5:20 minute) you will get the
   error message
   "Message from webpage: Ein Fehler ist aufgetreten"
   
   
   Tests with no effects:
   I modified java heap to 4 GB
   modified TM1Web_config.xml file
   delete all IE temp/cookies
   
   
   Test with Firefox and Chrome
   Snapshot works in Firefox and Chrome, but did take a long time
   (more than 8 minutes)
   
   
   I made more test with IF1029.
   After insttall IF1029 I made the same tests.
   The Snapshot works and takes 2 minutes, second test snapshot
   takes less than 2 minutes

   with Excel or can save the snapshot
