PROBLEM SUMMARY
 *  ****************************************************************
   * USERS AFFECTED:  Users utilizing outbound web service in     *
   *                  BPM Express, Standard and Advanced          *
   ****************************************************************
   * PROBLEM DESCRIPTION: Additional array objects may be         *
   *                      serialized in outbound web service      *
   ****************************************************************
   * RECOMMENDATION:                                              *
   ****************************************************************
   Invoking an outbound web service with a signature of the
   operation containing array objects, the resulting SOAP request
   may contain unnecessary elements.  The server receiving the
   request may experience an error indicating that there are
   unexpected elements in the request.
   
   
    
   
   

PROBLEM CONCLUSION
 *  The ifix resolves the generation of these additional list
   elements in the serialized outbound soap request.
   
   This fix has the following prerequisites:
   
   Process Designer fix JR43607.
   JR43607 has prerequisites on JR43049 and JR43570. Please
   install prereq fixes JR43049, JR43570 and JR43607 prior to
   installing this fix.
