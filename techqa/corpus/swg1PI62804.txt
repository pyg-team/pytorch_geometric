PROBLEM SUMMARY
 *  ****************************************************************
   * USERS AFFECTED: PDTools Studio and Aqua users.               *
   ****************************************************************
   * PROBLEM DESCRIPTION: This APAR resolves various problems for *
   *                      PDTools Studio and Aqua users.          *
   *                      1. File Manager problems:               *
   *                         - Check if IMS subsystems have       *
   *                           canonical config before freeing.   *
   *                         - Fixed issue where IMS records      *
   *                           could not be deleted.              *
   *                         - Now show more progress during      *
   *                           export to Desktop for Base/DB2.    *
   *                         - Fixed issue where after clicking   *
   *                           save in DB2 template editor        *
   *                           dialog,                            *
   *                           the title changes to ...{1}.       *
   *                         - Fixed IMS Extract Wizard not       *
   *                           working for V12 host.              *
   *                         - Fixed bad TD names causing         *
   *                           unhandled exceptions               *
   *                      2. Fault Analyzer problems:             *
   *                         - Improved Fault Entry list          *
   *                           performance on linux.              *
   *                         - Fixed Invalid Thread exception     *
   *                           when opening analytics.            *
   *                         - Improve handling of user           *
   *                           cancellation requests.             *
   *                         - Add an indicator to drill down     *
   *                           charts so it is possible to        *
   *                           differentiate a regular vs drill   *
   *                           down chart.                        *
   *                         - The ChartEditor is not activated   *
   *                           again if already open, but not in  *
   *                           focus.                             *
   *                      3. Problem Determination Tools Common   *
   *                         Component problems:                  *
   *                         - Improved security certificate      *
   *                           management so they are not lost if *
   *                           Eclipse crashes.                   *
   *                         - Improved memory efficiency of      *
   *                           Lookup View.                       *
   *                         - Resolved issue where double        *
   *                           clicking a file node no longer     *
   *                           works.                             *
   ****************************************************************
   * RECOMMENDATION: Please upgrade to Aqua and install the       *
   *                 latest updates.                              *
   *                                                              *
   *                 Users remaining on PDTools Studio should     *
   *                 install release v13.1.0.23 (or later).       *
   ****************************************************************
