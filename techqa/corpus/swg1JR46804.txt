PROBLEM SUMMARY
 *  ****************************************************************
   * USERS AFFECTED:  Users who run BPM in cluster environment.   *
   ****************************************************************
   * PROBLEM DESCRIPTION: Process Designer can't reconnect to     *
   *                      Process Center when server has been     *
   *                      restarted (in cluster environment).     *
   ****************************************************************
   * RECOMMENDATION:                                              *
   ****************************************************************
   Process Designer creates many ORB instances, it is not
   supported by WebSphere Application Server WLM. This problem
   only happens in cluster environment.
   
   
    
   
   

PROBLEM CONCLUSION
 *  Code logic has been changed during Process Designer
   initialization to avoid creating many ORB instances.
   
   JR46804 for BPM v8.0.1.1 is available in Fix Central.
   Installation instruction can be downloaded along with
   the JR46804 iFix file.
