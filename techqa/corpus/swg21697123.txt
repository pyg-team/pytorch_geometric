Title: IBM The workflow definition class is not visible when adding workflows to  IBM FileNet Content Manager repositories. - United States

Text:
VWVersion; property identifier; class Document TECHNOTE (FAQ)

QUESTION
 How to enable workflow definition class for repositories in IBM Content Navigator (ICN). 

ANSWER
Open the administration tool in ICN. 

 * Click Repositories. 
 * Open and connect to the IBM FileNet Content Manager repository where you want users to add workflow definitions. 
 * On the Configuration Parameters tab, set Display workflow definition class to Yes. This setting enables users to add workflow definitions to the repository.
