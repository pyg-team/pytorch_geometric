Title: IBM Change the background color of summary cells in a PowerPlay Enterprise Server XLS export file - United States

Text:
 TECHNOTE (TROUBLESHOOTING)

PROBLEM(ABSTRACT)
 Is it possible to change the background color of summary cells in a PowerPlay XLS export file? 

RESOLVING THE PROBLEM
Yes, you can do this by making a change to the ...cognos/cer4/ppserver/excel_styles.css file.

Modify the following sections to change the color of the summary label row, col and cell values. Save the CSS file and do a XLS export in PowerPlay Enterprise server again.

.summaryrowlabelone
{
background:#9999cc;
font-weight: bold;
color: black;
text-align: left;
padding-left:12px;
mso-char-indent-count:1;
}

.summaryrowlabeltwo
{
background:#e3e3e3;
font-weight: bold;
color: black;
text-align: left;
padding-left:12px;
mso-char-indent-count:1;
}

.summarycollabelone
{
background:#9999cc;
font-weight: bold;
color: black;
text-align: center;
}

.summarycollabeltwo
{
background:#e3e3e3;
font-weight: bold;
color: black;
text-align: center;
}


.summarycellone
{
background-color:#9999cc;
font-weight: bold;
text-align: right;
}

.summarycelltwo
{
background:#e3e3e3;
font-weight: bold;
text-align: right;
}
