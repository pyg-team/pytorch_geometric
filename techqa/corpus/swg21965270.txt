Title: IBM "Call was rejected by callee. (Exception from HRESULT: 0x80010001 (RPC_E_CALL_REJECTED))" errors when using Trial Balance with Drilldown - United States

Text:
 TECHNOTE (TROUBLESHOOTING)

PROBLEM(ABSTRACT)
 User launches the 'main' Controller client (CCR.EXE). User clicks "Group - Reports - Trial Balance with Drilldown". User clicks 'Send to Application'. Error #1 occurs.

User clicks OK, and the values appear. User clicks 'Send to Application' again. Error #2 occurs.

CAUSE
Known limitation (reference APAR PI48680) of Controller, when some third-party Excel add-ins are registered/enabled simultaneously with the Controller Excel addin.


ENVIRONMENT
Client device has a third-party (non-IBM) Excel add-in installed/registered. 

 

Example: 

This problem has been reproduced with the third-party software module 'Fire.Sys' enabled.


DIAGNOSING THE PROBLEM
Inside Excel, check to see what Excel add-ins are registered by using the 'Add-ins' menu: 

[/support/docview.wss?uid=swg21965270&aid=3] [/support/docview.wss?uid=swg21965270&aid=3]

Inside 'Manage' click 'Go':
[/support/docview.wss?uid=swg21965270&amp;aid=4]


RESOLVING THE PROBLEM
Fix: 

Upgrade to Controller 10.2.1 Fix Pack 2 (10.2.5120.72) or later. 

 

Workaround: 

Disable the third-party Excel add-in:
[/support/docview.wss?uid=swg21965270&amp;aid=5]
