ABSTRACT
 A new version of the Netcool/OMNIbus Gateway Java Support Package will be available to download from March 12, 2015. 

CONTENT
 

This support package provides Java infrastructure support for Java-based gateways.

You can download this package from the IBM Passport Advantage website:

www.ibm.com/software/howtobuy/passportadvantage/pao_customers.htm [http://www.ibm.com/software/howtobuy/passportadvantage/pao_customers.htm]

Netcool/OMNIbus V8.1

The Netcool/OMNIbus V8.1 version of this package supports the following Operating Systems:

AIX, Linux, Solaris, Windows and zLinux.

If you are running Netcool/OMNIbus V8.1, a single package contains versions of the gateway support library for all supported Operating Systems. To obtain this new package, use the Find by part number field to search for part number: CN4FVEN

Netcool/OMNIbus V7.3.0, 7.3.1 or 7.4.0

If you are running Netcool/OMNIbus V7.3.0, 7.3.1 or 7.4.0, a separate package is supplied for each Operating System. To obtain this new package, use the Find by part number field to search for the part number that corresponds to your Operating System as shown in the table below.
