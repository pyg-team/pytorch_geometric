OVERVIEW
IBM Watson Explorer Version 10.0.0.3 is a fix pack that addresses security issues. See the following Security Bulletins for more information: 

 * Security Bulletin: Vulnerability in OpenSSL affects Watson Explorer (CVE-2016-0702) [http://www.ibm.com/support/docview.wss?uid=swg21982340] 
 * Security Bulletin: Vulnerability in OpenSSL affects Watson Explorer (CVE-2016-2107) [http://www.ibm.com/support/docview.wss?uid=swg21986054] 
 * Security Bulletin: libarchive vulnerability affects Watson Explorer (CVE-2016-1541) [http://www.ibm.com/support/docview.wss?uid=swg21988311] For additional information about IBM Watson Explorer Version 10.0, see the Release Notes [http://www.ibm.com/support/docview.wss?uid=swg27042989].
