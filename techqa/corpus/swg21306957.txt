QUESTION
 In the Eclipse™ debug perspective, the variable detail pane holds a limited number of characters. How do you change that limit? 

ANSWER
Note: This technote also applies to other IBM® Rational® Eclipse-based products. 

This task involves two steps. 

 

 1. Open the Configure Details Area dialog box.
    
 2. Change the default limit value.




Step 1. Open the Configure Details Area dialog box. 

The following actions apply to Eclipse 3.2.x and the Eclipse 3.2.x Rational products. 
 * In the debug perspective you left-click the down arrow in the view's top right corner (1). 
 * Click Detail pane... from the context menu (2). See screenshot below for more details.

Step 2: Change the default limit value. 
 * Set the appropriate number of characters in the dialog box Configure Details Pane.
