dsrdoc; software requirements; system requirements PRODUCT DOCUMENTATION

ABSTRACT
 The list below identifies supported hardware and software options for the IBM Lotus Forms Server with Workflow 3.5.1 software suite. These system requirements apply to the entire software suite which contains the following two products: Alphalogix Tasica Forms Director 3.4 and IBM Lotus Forms Server 3.5.1.2.
