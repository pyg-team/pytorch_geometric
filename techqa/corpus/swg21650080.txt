Title: IBM WebSphere Transformation Extender Integration Flow Designer may crash when importing an exported XML file that is missing optional elements - United States

Text:
WTX; msdimport; Integration Flow Designer; PM92311 TECHNOTE (TROUBLESHOOTING)

PROBLEM(ABSTRACT)
 WebSphere Transformation Extender (WTX) Integration Flow Designer or msdimport command line utility crashes when importing a system definition from an exported XML file which does not contain optional elements ServerName, type and typetree. 

CAUSE
Integration Flow Designer or msdimport command line utility was not handling the optional elements ServerName, type and typetree properly resulting in crashes during the import.

RESOLVING THE PROBLEM
Please contact IBM WebSphere Transformation Extender support to request a patch for APAR PM92311. The fix will be included in next release.
