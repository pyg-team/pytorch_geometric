Title: IBM Security Bulletin: Multiple security vulnerabilities have been identified in IBM HTTP Server shipped with WebSphere Commerce (CVE-2015-1283, CVE-2015-3183, CVE-2015-4947) - United States

Text:
 SECURITY BULLETIN

SUMMARY
 IBM HTTP Server is shipped as a component of WebSphere Commerce. Information about multiple security vulnerabilities affecting IBM HTTP Server have been published in multiple security bulletins. 

VULNERABILITY DETAILS
For vulnerability details and information about fixes, consult the following security bulletins:

Security Bulletin: Denial of service may affect IBM HTTP Server (CVE-2015-1283) [http://www-01.ibm.com/support/docview.wss?uid=swg21964428]

Security Bulletin: HTTP Request smuggling vulnerability may affect IBM HTTP Server (CVE-2015-3183) [http://www-01.ibm.com/support/docview.wss?uid=swg21963361]

Security Bulletin: Stack Buffer overflow may affect IBM HTTP Server (CVE-2015-4947) [http://www-01.ibm.com/support/docview.wss?uid=swg21965419]

AFFECTED PRODUCTS AND VERSIONS
 

Principal Product and Version(s) Affected Supporting Product and Version WebSphere Commerce V6.0 and V7.0 IBM HTTP Server for WebSphere Application Server  * Version 8.5 and 8.5.5 
 * Version 8 
 * Version 7 
 * Version 6.1
