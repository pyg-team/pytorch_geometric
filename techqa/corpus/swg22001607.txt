Title: IBM StoredIQ Dataserver login page is not coming up in the browser - United States

Text:
 TECHNOTE (FAQ)

QUESTION
 Why is the StoredIQ Dataserver login page not coming up in the browser? 

CAUSE
This is caused by missing or incorrect hostname in the /etc/hosts file. When you restart the apache service, you will see an error message - cannot reliably
determine the fully qualified sever domain name


ANSWER
 

 1. Login to the appliance as root 
 2. Edit the /etc/hosts file 
 3. Update the correct hostname and save changes 
 4. Restart the appliance
