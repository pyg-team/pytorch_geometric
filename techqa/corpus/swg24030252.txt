Title: IBM WebSphere Business Monitor V7.0 Fix Pack 4 (7.0.0.4) - United States

Text:
7.0.0.4; Fix Pack; Fix Pack 4 DOWNLOADABLE FILES

ABSTRACT
 Fix Pack 4 is now available for WebSphere Business Monitor V7.0. 

DOWNLOAD DESCRIPTION
V7.0 Fix Pack 4 is a cumulative fix for IBM WebSphere Business Monitor V7.0, first shipped in July 2011.


The links in the download section below link to the download packages for V7.0 Fix Pack 4 on all applicable platforms.


For a list of fixes included in this Fix Pack, please refer to Fix/Update list for WebSphere Business Monitor V7.0 [http://www.ibm.com/support/docview.wss?rs=802&uid=swg27019404]. Fixes for Business Space are also included; please refer to Fix/Update list for Business Space powered by WebSphere: Version 7.0 [http://www.ibm.com/support/docview.wss?rs=802&uid=swg27019647] (fixes in all releases prior to and including V7.0 Fix Pack 4 for Business Space are included in this Fix Pack).

In addition to fixes, the following features are included in V7.0 Fix Pack 4.



Feature Description Updated Information Center The IBM WebSphere Business Monitor 7.0 Information Center [http://publib.boulder.ibm.com/infocenter/dmndhelp/v7r0mx/topic/com.ibm.btools.help.monitor.doc/home/home.html] has been updated for V7.0.0.4.

If you have a locally installed version of the documentation, see Updating the local information center [http://publib.boulder.ibm.com/infocenter/dmndhelp/v7r0mx/topic/com.ibm.btools.help.monitor.inst.doc/inst/ic_update.html]. Alphablox Version 9.5.7.2 WebSphere Business Monitor V7.0.0.4 updates Alphablox from Version 9.5.7 to Version 9.5.7.2. Windows 7 support WebSphere Business Monitor V7.0.0.4 includes support for the Windows 7 operating system. (Note: This support was originally included in V7.0.0.2.) Microsoft SQL Server support SQL Server is supported as a database for WebSphere Business Monitor in V7.0.0.4. (Note: This support was originally included in V7.0.0.2.)See MSDN Library: Microsoft SQL Server [http://msdn.microsoft.com/en-us/library/bb545450.aspx] for details.

Also refer to this technote [http://www.ibm.com/support/docview.wss?uid=swg21445276] which describes a solution for an issue with service integration buses failing to start with SQL Server. Red Hat Enterprise Linux Version 6.0 support Red Hat Enterprise Linux (RHEL) Advanced Edition Version 6.0 is supported by WebSphere Business Monitor V7.0.0.4. WebSphere Portal Version 7 support for portlet-based dashboards The portlet-based dashboards in WebSphere Business Monitor V7.0.0.4 support WebSphere Portal V7. (Note: This support was originally included in V7.0.0.2.) PREREQUISITES
 

 * Please ensure that your WebSphere Business Monitor environment meets the system requirements documented at: http://www.ibm.com/support/docview.wss?uid=swg27008414 [http://www-01.ibm.com/support/docview.wss?uid=swg27008414] 
 * This Fix Pack can only be installed on top of IBM WebSphere Business Monitor V7.0.0.0, including any maintenance deliverables prior to this one. 
 * Important: WebSphere Business Monitor V7.0 Fix Pack 4 includes an update for Business Space. If you have installed Business Space V7.0 Interim Fix 1 for WebSphere Business Monitor V7.0, and you have not yet applied any WebSphere Business Monitor V7.0 fix packs, you must install Business Space V7.0 Interim Fix JR36142 [http://www.ibm.com/support/docview.wss?rs=802&uid=swg24026356] prior to installing V7.0 Fix Pack 4. 
 * WebSphere Application Server (the instance on which WebSphere Business Monitor is installed) must be at Version 7.0.0.17 or higher. (Note: The installation instructions linked below provide steps for upgrading WebSphere Application Server to Version 7.0.0.17.) 
 * The WebSphere Application Server V7.0 Feature Pack for XML must be at Version 1.0.0.7 or higher. (Note: The installation instructions linked below provide steps for upgrading the WebSphere Application Server V7.0 Feature Pack for XML to Version 1.0.0.7.)
