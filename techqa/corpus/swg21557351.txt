RESOLVING THE PROBLEM


 Resolution:- User might face this issue during a fresh installation and the root cause is that a table or view might be missing in the database. Follow these steps to get rid of the error

1) Run the dbverify. It wil generate the SQL script for database at <Install_dir>\bin folder 2) Execute the auto generated SQL script on database 3) Finally run the factory setup again.
