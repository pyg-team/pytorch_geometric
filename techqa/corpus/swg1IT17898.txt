APAR STATUS
 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  In scenarios with many SSH connect/disconnct cycles, the
   appliance might restart, generating a backtrace.
   
   
    
   
   

LOCAL FIX

PROBLEM SUMMARY
 *  Affected are DataPower configurations using SSH access to CLI
   management interface.
   
   In scenarios with very many SSH connect / disconnect cycles, the
   device might restart, generating a backtrace.
   
   
    
   
   

PROBLEM CONCLUSION
 *  The fix is available in 7.0.0.17, 7.1.0.14, 7.2.0.11, 7.5.0.5,
   7.5.1.4 and 7.5.2.2

   http://www-01.ibm.com/support/docview.wss?uid=swg21237631 [http://www-01.ibm.com/support/docview.wss?uid=swg21237631]
