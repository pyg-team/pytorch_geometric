 DOWNLOADABLE FILES

ABSTRACT
 The MIB file describes the format of the Simple Network Management Protocol (SNMP) messages sent by SAN Volume Controller, Storwize V7000 and Storwize V7000 Unified, V5000 & V3x00. 

DOWNLOAD DESCRIPTION
Use this MIB file to configure a network management program to receive SNMP event notifications sent from a SAN Volume Controller, Storwize V7000 or Storwize V7000 Unified, V5000, V3500 or V3700 system. 

This MIB file is suitable for use with SNMP messages from all versions of SAN Volume Controller, Storwize V7000 and Storwize V7000 Unified, V5000, V3500 or V3700 software.
