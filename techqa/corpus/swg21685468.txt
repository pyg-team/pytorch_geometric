QUESTION
 Where can I find a quick start guide to deploy an Object Server failover pair for OMNIbus 7.4? 

ANSWER
The following document provides a quick deployment guide to get OMNIbus up and running in 10 steps. The example deployment configures two Object Servers as a failover pair with a bi-directional Object Server gateway configured to run under the Process Agent. The example configuration is completed on Linux, but the same steps can be applied to all operating systems.
