Title: IBM Error: 452 Put failed : Could not add batch to database, disk may be full - United States

Text:
STERLINGPRI TECHNOTE (TROUBLESHOOTING)

PROBLEM(ABSTRACT)
 Error: 452 Put failed : Could not add batch to database, disk may be full. 
There is a request to a device or address that does not exist. 

RESOLVING THE PROBLEM
Check that a file can be added using cmuadd or one of the batches can be viewed from the mailbox using cmuextract command. If unable to view and an error was receive 'read error or a message indicating there is request to a device or address that does not exist', problem is disk. Contact your UNIX administrator to check volumes and run fsck on filesystem.
