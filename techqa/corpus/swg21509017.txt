Title: IBM Error BMWMB0039E when logging into Business Space - United States

Text:
 TECHNOTE (TROUBLESHOOTING)

PROBLEM(ABSTRACT)
 All users logging into Business Space receive error message: BMWMB0039E: Fail to load page. Please contact Lotus Mashups administrator. 

SYMPTOM
When logging into Business Space, users see the following message: 

BMWMB0039E: Fail to load page. Please contact Lotus Mashups administrator.

The SystemErr.log contains the following message:
[8/9/11 12:01:23:078 PDT] 0000001d SystemErr R com.ibm.websphere.ce.cm.StaleConnectionException: [jcc][t4][2043][11550][3.52.110] Exception java.net.ConnectException: Error opening socket to server localhost/127.0.0.1 on port 50,000 with message: Connection refused: connect. ERRORCODE=-4499, SQLSTATE=08001DSRA0010E: SQL State = 08001, Error Code = -4,499

The SystemOut.log contains many SQL errors, including the following:
[8/9/11 12:01:24:406 PDT] 0000001d UserDataAcces E com.ibm.bspace.manager.services.dbhelper.UserDataAccess getUserData(userdn='null,' currentUserdn='uid=admin,o=defaultWIMFileBasedRealm,' currentUserid='admin,' includeEveryone=true) CWMFN4008E: Problem retrieving user personalization data from database.
com.ibm.websphere.ce.cm.StaleConnectionException: [jcc][t4][2043][11550][3.52.110] Exception java.net.ConnectException: Error opening socket to server localhost/127.0.0.1 on port 50,000 with message: Connection refused: connect. ERRORCODE=-4499, SQLSTATE=08001DSRA0010E: SQL State = 08001, Error Code = -4,499


CAUSE
After user credentials have been verified, Business Space data and user data are retrieved from the database. If there is any problem retrieving the data, error BMWMB0039E will be returned.


RESOLVING THE PROBLEM
Verify the following: 

1) the database is running. 

2) database user id/password has not expired. 

3) database license has not expired. 

4) database can be accessed from the Business Space server. 

5) test if the connection of jdbc/mashupDS in WAS console > Resources > JDBC > Data sources is successful.
