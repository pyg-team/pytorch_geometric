APAR STATUS
 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  Applications that use Outbound HTTP Connections in a clustered
   environments can cause following error messages:
   
   WASSession ETMBuffWrapper getBytes write object exception. e=
   java.io.NotSerializableException:
   com.ibm.wps.outbound.cookiestore.CookieInstanceImpl
   
   These errors occur if the following conditions occur:
   1. A clustered environment has been set up.
   2. An application utilizes the Outbound HTTP Connection
   Service.
   3. A policy rule has been defined in the outbound HTTP
   connection profile for which a cookie handling rule
   "store-in-session" is set.
   4. The application connects to the remote server using the
   outbound HTTP connection service. The HTTP response of the
   remote site returns a cookie for which the handling type
   "store-in-session" is defined.
   Subsequent HTTP requests using Outbound Connection Service may
   cause the error message listed above.
