APAR STATUS
 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  DataPower mq-qm object may report MQ error code 2009, when
   DataPower is trying to end the conversation and the connection
   is getting marked as closed before the last receiving task is
   done.
   
   
    
   
   

LOCAL FIX

PROBLEM SUMMARY
 *  The MQ connection might be broken in an unexpected way.
   
   
    
   
   

PROBLEM CONCLUSION
 *  Fix is available in 7.2.0.11, 7.5.0.5, 7.5.1.4 and 7.5.2.2

   http://www-01.ibm.com/support/docview.wss?uid=swg21237631 [http://www-01.ibm.com/support/docview.wss?uid=swg21237631]
