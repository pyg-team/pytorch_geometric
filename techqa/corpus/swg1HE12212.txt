ERROR DESCRIPTION
 *  New available feature in Atlas 6.0.3 affects when and how a
   given custodian can be removed from the scope of a Hold
   Notice.Custodians listed in a Request attached to a
   Hold Notice will be prevented from both Removal and Release
   while that Notice is in Draft Status. To remove a custodian
   from the Notice while in Draft Status, the user must first
   Stop Reminders for the custodian--which has the effect of
   removing them from the Notice.
   
   
    
   
   

LOCAL FIX
 *  Use Stop Reminders button
