ERROR DESCRIPTION
 *  PureApplication System V2.1.0.1
   
   The user had Call Home enabled on a PureApplication System
   version 2.1.0.0 system and upgraded to PureApplication System
   version 2.1.0.1. The Call Home function of the system stopped
   working.
   
   An error is displayed in the Events page:
   CWZIP3523E IBM PureApplication System is not able to
   communicate with IBM for Call Home events
   
   The Test Connection operation fails with the error:
   CWZIP3534E Call Home connection did not complete successfully,
   http error = 404
   
   
    
   
   

LOCAL FIX

PROBLEM SUMMARY
 *  Code changes introduced in the Call Home function in
   PureApplication System version 2.1.0.1 were not handled
   correctly in the upgrade scenario where Call Home was already
   configured in a system running version 2.1.0.0.
   
   
    
   
   

PROBLEM CONCLUSION
 *  A code fix has been provided in PureApplication System version
   2.1.0.2.
   
   
    
   
   

TEMPORARY FIX
