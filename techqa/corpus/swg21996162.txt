ABSTRACT
 The Summarization and Pruning Agent has been updated to support HTTP and HTTPS connections to the Tivoli Enterprise Portal Server. 

CONTENT
This technote documents changes from the current online documentation for Configuring the Summarization and Pruning Agent (JDBC Connection) [http://www.ibm.com/support/knowledgecenter/en/SSTFXA_6.3.0/com.ibm.itm.doc_6.3fp2/install/sp_config.htm] for selection of TEPS protocol (APAR IV81519 [http://www.ibm.com/support/docview.wss?uid=swg1IV81519]). 
The screenshot below shows the new fields (update to Figure 1 in documentation).

The Protocol choices now include HTTP and HTTPS in addition to the default IIOP.
