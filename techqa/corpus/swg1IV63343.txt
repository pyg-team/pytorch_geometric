ERROR DESCRIPTION
 *  If a log source extension (LSX) is created with a "Matcher" and
   no associated "field" entry, then uploaded for use into the UI,
   it will cause some DSM extensions not to load and in turn cause
   parsing to fail for the log source(s) that are using that LSX.
