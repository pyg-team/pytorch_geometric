APAR STATUS
 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  If an SFTP Server Front Side Handler is reconfigured by changing
   host-private-key, it goes to the down state.
   
   
    
   
   

LOCAL FIX
 *  Save the new configuration and shutdown-restart the DataPower
   appliance to bring the SFTP Server Front Side Handler to the up
   state.
   
   
    
   
   

PROBLEM SUMMARY
 *  Affected are DataPower installations configuring an SFTP Server
   front side handler with a host-private-key. The configured front
   side handler is not applied as op-state up without an appliance
   restart.
   
   
    
   
   

PROBLEM CONCLUSION
 *  Fix is available in 6.0.0.16, 6.0.1.12, 7.0.0.9, 7.1.0.6 and
   7.2.0.0.
   
   For a list of the latest fix packs available, please see:
   http://www-01.ibm.com/support/docview.wss?uid=swg21237631 [http://www-01.ibm.com/support/docview.wss?uid=swg21237631]
