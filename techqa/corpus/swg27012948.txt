Space requirements 

 * 800 MB of disk space to store the required downloadable parts plus 900 MB of disk space to unpack the parts. 
 * 1 GB (1000 MB) of disk space to install the product without profiles and 1.25 GB (1250 MB) for a cell installation. 
 * 30 MB of space in the tmp directory to run the installation program. 
 * For more detailed information, see the WebSphere Application Server detailed system requirements [http://www.ibm.com/support/docview.wss?rs=180&uid=swg27006921] document.


Downloading and assembling the product  1. Navigate your browser to the Find Downloads and Media [http://www-112.ibm.com/software/howtobuy/passportadvantage/paocustomer/sdma/SDMA?PO=DOWNLOAD_FIND_BY_PART_NUMBER] Web page on the IBM Passport Advantage Online Web site. 
 2. Select the Find by part number search option under Download finder options. 
 3. Search for each downloadable image by part number. 
 4. Download all of the required parts. 
 5. Use a file extracting utility that supports large file extractions to unpack each of the parts into a single, temporary directory on your system. 
 6. Open the Quick Start Guide, which is located on the Quick Start CD. Follow step three through step nine of the Quick Start Guide to access the documentation to help with product installation.
