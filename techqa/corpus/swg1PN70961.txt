Title: IBM PN70961: COBOL VERSION OF DFHTCTTE IS CODED AS PL/I STYLE COMMENT RESULTSIN COMPILER ERROR WHEN ANY COBOL PROG INCLUDING IT IS COMPILED. - United States

APAR STATUS
 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  Correct copyright statement in COBOL version of DFHTCTTE the
   cobol version of DFHTCTTE is currently coded as a PL/I style
   comment which results in compiler error when any cobol program
   including it is compiled.  Ditto for DFHJCADS.
   Build process has added invalid COPYRIGHT statement to DFHTCTTE
   and DFHJCADS COBOL copybooks.
   
   
    
   
   

LOCAL FIX

PROBLEM SUMMARY
 *  ****************************************************************
   * USERS AFFECTED: CICS/VSE R230 users of the Cobol versions    *
   *                 of copybooks DFHTCTTE and DFHJCADS.          *
   ****************************************************************
   * PROBLEM DESCRIPTION: Compiler errors result when compiling   *
   *                      the Cobol versions of copybooks         *
   *                      DFHTCTTE and DFHJCADS.                  *
   ****************************************************************
   * RECOMMENDATION:                                              *
   ****************************************************************
   The COBOL versions of copybooks DFHTCTTE and DFHJCADS are
   currently coded with PL/1 style comments for the COPYRIGHT
   header statements. Any COBOL program including these Cobol
   copybooks will therefore compile with errors.
   
   
    
   
   

PROBLEM CONCLUSION
 *  DFHTCTTE and DFHJCADS have had their COPYRIGHT header
   changed from PL/I style to Cobol style comments.
   
   Keywords: PTM 74316 PTM74316 PL1 PLI
   
   
    
   
   

TEMPORARY FIX
