APAR STATUS
 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  The Script Action button on the work page menu can disappear
   after a page refresh.  Subsequent page refresh can sometimes
   bring back the button.  The button will appear and disappear at
   random with subsequent refresh on the same page.  You have to
   reload the "role" to get the button back.  Clicking another tab
   and returning to the original tab does not restore the button.
   
   The new Script Action button can be empty or have code in it,
   the result is the same.
   
   On ICM 5.2.1.1 this has been reproduced on the following FireFox
   versions:
   17.0.11
   24.8.1
   31.5.0
   36.0
   37.0
   
   This issue was not reproducible with IE or Chrome.
   
   
    
   
   

LOCAL FIX
 *  There are a couple of ways to mitigate this issue
   1. This issue was not reproduced with IE or Chrome
   2. Change the page, via Page Designer, to a two row
   configuration with the Toolbar on top, and the In-baskets on the
   bottom.  Remove any/all OOTB buttoms from the Toolbar.
