ABSTRACT
 Rational Agent Controller, version 8.3.2 

DOWNLOAD DESCRIPTION
IBM Rational Agent Controller is a daemon that allows client applications to launch and manage local or remote applications and provides information about running applications to other applications. You must install Rational Agent Controller separately in the target machine before you can use the following tools: 

 * Profiling tools to profile your applications. 
 * Component testing tool to run test cases. 
 * Tools for remote profiling or enabling Java code coverage statistic collection on the following versions of WebSphere Application Server :  * Version 8.0 beta (Java 1.6 applications). 
    * Version 7.0 (Java 1.6 applications) 
    * Version 6.1 (Java 1.5 applications) 
    * Version 6.0 (Java 1.4 applications)
      Note on WebSphere Application Server v6.0 support:
      Profiling is only supported for Windows and Linux operating systems. For profiling on WebSphere Application Server version 6.0 with other supported operating systems, use Rational Agent Controller version 7.x [http://www.ibm.com/support/docview.wss?rs=2042&uid=swg27013420].
      
   
   

For details on other available versions of Agent Controller, see http://www-01.ibm.com/support/docview.wss?uid=swg27013420 [http://www-01.ibm.com/support/docview.wss?uid=swg27013420].
