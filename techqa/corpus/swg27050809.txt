Infrastructure Security Support Newsletter 
January 2018 

This newsletter is intended to help you, a valued IBM client who has contacted technical support, be proactive in the support and administration of your software. We hope you find it useful!  New for you 

 

 1.  January Super Tuesday XPU 3801.04165 Release [https://exchange.xforce.ibmcloud.com/search/xpu%203801.04165] 
    PAM update 3801.04165 contains 12 new events, 21 new moderate event responses, and 20 new aggressive event responses. 
 2.  XPU 3801.09174 Release [https://exchange.xforce.ibmcloud.com/search/xpu%203801.09174] 
    XPU 3801.09174 Release: PAM update 3801.09174 contains 4 new events, 2 new moderate event responses, and 2 new aggressive event responses. 
 3.  FLASH: Central Processor Unit (CPU) Architectural Design Flaws [http://www-01.ibm.com/support/docview.wss?uid=swg22012320&acss=danl_5063_web] 
    IBM Security X-Force is aware of the CPU vulnerability disclosed by Google. In response to the disclosure of vulnerabilities, the IBM X-Force has raised the current internet threat level to AlertCon 2. 
 4.  FLASH: Security Bulletin: IBM Security SiteProtector System is affected by Apache HTTP Server Vulnerabilities [http://www.ibm.com/support/docview.wss?uid=swg22009368&acss=danl_5063_web] 
    IBM Security SiteProtector System has addressed the following vulnerabilities in Apache HTTP Server.

 Get help 

 

 1.  New online Support experience! [http://www.ibm.com/mysupport?acss=danl_5063_web] 
    In order to serve you better, IBM Security has launched a new online Support experience. We understand that you may have other IBM products, and this change may require you to temporarily access different portals for different products. IBM will continue to onboard products quickly so that you can have a simple, consistent experience. There will be no change to the process of contacting Support through phone or e-mail. 
 2.  My Notifications [http://www.ibm.com/software/support/einfo.html?acss=danl_5063_web] 
    My Notifications is an IBM Support email subscription or RSS feed notification service. You select the products of interest to you, and you opt to be notified in a daily or weekly digest. You will be notified immediately when the PSIRT team announces new product vulnerabilities and solutions, or when Flash Alerts are published. 
 3.  IBM customer forum for Infrastructure Security products [http://www.ibm.com/support/docview.wss?uid=swg21902372&acss=danl_5063_web] 
    The dW Answers forum in IBM developerWorks offers a place to ask questions and view answers to questions already posed. You may filter the forum using the tags SiteProtector, XGS, and GX. When asking a question please ensure you apply the appropriate tag to the question. 
 4.  Fix Central [http://www.ibm.com/support/fixcentral?acss=danl_5063_web] 
    Fix Central provides fixes and updates for your system's software, hardware, and operating system. Many interim patches and updates can be found here.
