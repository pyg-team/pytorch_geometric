PROBLEM(ABSTRACT)
 Recommended settings and known issues are documented below for Continuous Data Protection for Files and FastBack for Workstations 

RESOLVING THE PROBLEM
 If a user experiences a browser error: clear the error, then refresh the browser. This will usually resolve any browser error. 
The product was designed to be optimally viewed with the following settings: 

 *  1024 X 768 resolution maximize the browser window remove extra tool bars from the browser 


The Activity Report window must be manually refreshed.  *

 * On Windows XP/2000, the path is 
   %AppData%\Mozilla\Firefox\Profiles\ xxxxxxxx.default\, where xxxxxxxx is a random 
   string of 8 characters. Browse to C:\Documents and Settings\[User 
   Name]\Application Data\Mozilla\Firefox\Profiles\. 
   
   %AppData% is a shortened expression for the Application Data path on Windows 
   2000/XP. To use it, click Start -> Run, type %AppData%, and then press Enter. You 
   will be directed to the appropriate folder, which is normally C:\Documents and 
   Settings\ <user_name>\Application Data.
