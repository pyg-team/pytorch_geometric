ERROR DESCRIPTION
 *  Can be reproduced in the Business Console with any decision
   service in DGF mode, for example in AutoQuote:
   - Select the ADMINISTRATION tab
   - Select the Project Security sub-tab
   - Click the Edit button on the AutoQuote row
   - Under 'Security', select 'Enforce security', select a group,
   click Done
   - Expand AutoQuote/Initial Release/Projects
   - Click the Edit button on any sub-project, for example
   AutoInsuranceQuotingBOM
   - Click 'Security settings are inherited from main' and select
   'Override security for this branch', click Done
   - Click the Edit button again on the same sub-project (for ex.
   AutoInsuranceQuotingBOM)
   ==> The dialog still shows 'Security settings are inherited
   from main'.
