PROBLEM SUMMARY
 *  ****************************************************************
   * USERS AFFECTED: ITCAM for SOA Interaction Detail topology    *
   * users on all platforms.                                      *
   ****************************************************************
   * PROBLEM DESCRIPTION:                                         *
   * If a service port namespace is longer than 64 characters the *
   * BPM code attempts to insert the full length string into a    *
   * smaller buffer and this throws an exception.                 *
   *                                                              *
   * The fix limits the length of a namespace query that was      *
   * causing database errors and preventing users from viewing    *
   * certain interaction detail topologies.                       *
   ****************************************************************
   * RECOMMENDATION: Apply fix when it is available.              *
   ****************************************************************
