LOCAL FIX
 *  Because the Ad Hoc group description is built with the following
   pattern
   
   "Automatically created for "" + <activity Name> + " activity in
   "" + <instance display name> + " process"
   
   revise the activity, instance display name, or process so that
   the  description length equates to 255 or fewer characters.
   
   
    
   
   

PROBLEM SUMMARY
 *  The Ad Hoc group description is built with the following
   pattern:
   
   "Automatically created for "" + <activity Name> + " activity in
   "" + <instance display name> + " process"
   
   Sometimes, this description can be longer than 255 characters,
   which causes the exception.
   
   
    
   
   

PROBLEM CONCLUSION
 *  A fix is/will be available for IBM BPM V8.0.1.3 that restricts
   entries in the Description column to 255 or fewer characters.
   
   On Fix Central (http://www.ibm.com/support/fixcentral), search
   for JR54713:
   
   1. Select IBM Business Process Manager with your edition from
     the product selector, the installed version to the fix pack
     level, and your platform, and then click Continue.
   
   2. Select APAR or SPR, enter JR54713, and click Continue.
   
   When you download fix packages, ensure that you also download
   the readme file for each fix. Review each readme file for
   additional installation instructions and information about the
   fix.
