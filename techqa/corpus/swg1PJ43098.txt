APAR STATUS
 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  When trying to import a solution template using IBM Case Manager
   administration client after one previous attempt, the Import
   Solution Template wizard does not allow the user to proceed as
   the Next button appears disabled (or grayed out). Additionally
   the user may also see a field for Project Area with a drop-down
   (or select) control. The project area field should not normally
   appear when importing a solution template.
   
   
    
   
   

LOCAL FIX
 *  The following workaround can be employed to proceed with the
   import Solution Template action.:
   1) Close the Import Solution Template wizard with the disabled
   'Next' button.
   2) Logout of the IBM Case Manager administration client.
   3) Log back in to IBM Case Manager administration client.
   4) Attempt the ?Import? action from the Solution Templates list
   view again.
   Now the Import Solution Template wizard should initialize
   correctly.
   
   
    
   
   

PROBLEM SUMMARY
 *  ****************************************************************
   * USERS AFFECTED:                                              *

   * Manager administration client.                               *
   ****************************************************************
   * PROBLEM DESCRIPTION:                                         *
   * See Error Description                                        *
   ****************************************************************
   * RECOMMENDATION:                                              *
   * Update to 5.2.1.2-ICM-FP002 or later.                        *
   ****************************************************************
   
   
    
   
   

PROBLEM CONCLUSION
 *  Fixed in 5.2.1.2-ICM-FP002. With this fix, the next button for
   an import dialog will no longer be disabled after an initial
   import.
