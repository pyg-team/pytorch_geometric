Title: IBM Upgrading or Uninstalling ClearQuest or Rational ProjectConsole on a machine with Rational Performance Tester fails to delete cqjni.jar - United States

Text:
RN7.1; RN7.1.1 TECHNOTE (TROUBLESHOOTING)

PROBLEM(ABSTRACT)
 You upgrade or uninstall IBM Rational ClearQuest or IBM Rational ProjectConsole on a machine with IBM Rational Performance Tester installed. The upgrade or uninstall fails when deleting cqjni.jar. 

SYMPTOM
The install log contains the entry: 


Error uninstalling.
Deleting files from C:\Program Files\IBM\RationalSDLC
Failed to delete C:\Program Files\IBM\RationalSDLC\ClearQuest\cqjni.jar



CAUSE
The IBM Rational Agent Controller service loads cqjni.jar which causes the operating system to lock this file. The Rational Agent Controller service is a Rational Performance Tester service. Installation Manager cannot delete a locked file during an upgrade or uninstall.



RESOLVING THE PROBLEM
This issue has been addressed in 7.1.1.2 version. 

To resolve the problem, take the following steps:

Option 1: Service Control Manager 

 1. In the Start Menu, click Run. 
 2. Type services.msc and press Enter. 
 3. Right-click on the IBM Rational Agent Controller service. 
 4. Click Stop. 
 5. Close the Service Control Manager. 
 6. If you are upgrading Rational ClearQuest or Rational ProjectConsole, restart the upgrade. If you have performed an uninstall, delete cqjni.jar.
    
    


Option 2: Using sc.exe via the command line Open a command prompt and execute: 

sc stop "IBM Rational Agent Controller"

Note: This command is available only if you have installed the Microsoft Win32Â® Software Development Kit.
