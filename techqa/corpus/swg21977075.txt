Title: IBM 統合パッケージをインストールするための Installation Manager 使用方法 - United States

Text:
listIU.sh; nco_id; install manager; IMHOME; installation mananger; probes; gateways; Japanese; 日本語 技術情報(FAQS)

質問
 imcl コマンドを使用して、probe や gateway のパッケージをどのようにインストールするのでしょうか。 

原因
Netcool/OMNIbus 8.1 はパッケージのインストールに Installation Manager を使用します。

回答
Installation Manager のコマンド imcl はダウンロードされた統合パッケージのインストールに使用できます。 

 * imcl のある場所を確認するにはNetcool/OMNIbus のユーザーで nco_id コマンドを使用します。



nco_id

IBM Tivoli Netcool/OMNIbus - 8.1.0

 * imclコマンドを使用するために IMHOME を設定します。 

export IMHOME 
IMHOME=/opt/nrv81/IBM/InstallationManager/eclipse 
 * 以下のコマンドを使用して、インストール済みパッケージを表示します。


$IMHOME/tools/imcl listInstalledPackages -features -long 
$IMHOME/tools/imcl listInstalledPackages listInstalledPackages 
 * 製品名とパッケージの場所を指定してプローブ/ゲートウェイをインストールします。

$IMHOME/tools/imcl -c install com.ibm.tivoli.omnibus.integrations. nco-p-socket -repositories 
./ im-nco-p-socket-10_0.zip -installationDirectory /opt/nrv81/tivoli/netcool 
 * -version を使用して [例：nco_p_socket] 製品のインストールを確認します。


$NCHOME/omnibus/probes/nco_p_socket -version 関連情報
