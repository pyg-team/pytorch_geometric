ERROR DESCRIPTION
 *  When importing a gateway with debug probe attached via XML
   management interface,
   the gateway will be correctly imported, but the probe data will
   not.
   
   
    
   
   

LOCAL FIX
 *  Use webGUI import instead.
   
   
    
   
   

PROBLEM SUMMARY
 *  Affected is import of export with Probe capture via XML
   management interface.
   
   When importing the gateway will be correctly imported, but the
   probe data will not.
