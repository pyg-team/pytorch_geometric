Title: IBM PM43260; 8.0: IHS admin start-up is "automatic" despite choosing "manual" - United States

Text:
PM43260; PM43260; PM43260 DOWNLOADABLE FILES

ABSTRACT
 IBM HTTP Server (IHS) admin start-up is "Automatic" despite choosing "manual" in the Plug-in Configuration Tool (PCT) 

DOWNLOAD DESCRIPTION
PM43260 resolves the following problem:

ERROR DESCRIPTION: 
In WebSphere Application Server v8.0, using the Plug-in Configuration Tool (PCT) to create an IBM HTTP Server Admin Service. If you choose the start-up should be "manual", the PCT sets it to be "automatic" anyway. 

This problem only happens on non-English systems. 

LOCAL FIX: 
Work-around is to change the start-up to "manual" after the IHS Admin Service has been created by PCT. 

PROBLEM SUMMARY

USERS AFFECTED:
All users of IBM WebSphere Customization Toolbox (with Plug-ins Configuration Tool).
