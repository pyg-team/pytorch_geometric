Title: IBM Security Bulletin: Multiple security vulnerabilities have been identified in IBM WebSphere Application Server shipped with IBM WebSphere Portal - United States

Text:
 SECURITY BULLETIN

SUMMARY
 IBM WebSphere Application Server is shipped as a component of IBM WebSphere Portal. Information about security vulnerabilities affecting IBM WebSphere Application Server has been published in security bulletins. 

VULNERABILITY DETAILS
Please consult the security bulletins for IBM WebSphere Application Server published at 

 * Security Bulletin: Information disclosure in WebSphere Application Server (CVE-2017-1381) [http://www-01.ibm.com/support/docview.wss?uid=swg22004792] 
 * Security Bulletin: Cross-site scripting vulnerability in Admin Console for WebSphere Application Server (CVE-2017-1380) [http://www-01.ibm.com/support/docview.wss?uid=swg22004786] 
 * Security Bulletin: WebSphere Application Server may have insecure file permissions (CVE-2017-1382) [http://www-01.ibm.com/support/docview.wss?uid=swg22004785] 
 * Security Bulletin: Weaker than expected security in WebSphere Application Server (CVE-2017-1504) [http://www-01.ibm.com/support/docview.wss?uid=swg22006803] 
 * Security Bulletin: Potential security vulnerability in the WebSphere Application Server Admin Console (CVE-2017-1501) [http://www-01.ibm.com/support/docview.wss?uid=swg22006810] 
 * Security Bulletin: Multiple Security Vulnerabilities in IBM HTTP Server (CVE-2017-7679, CVE-2017-7668, CVE-2017-3167) [http://www-01.ibm.com/support/docview.wss?uid=swg22005280]
