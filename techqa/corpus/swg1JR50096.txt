ERROR DESCRIPTION
 *  The implemented 'Asynchronous Suspected Duplicate'
   is working fine with the base services but with custom services
   the following errors are generated.
   Note: The ControlExtensionProperty object is used to provide
   additional control properties in the main request that triggers
   the 'Asynchronous Suspected Duplicate'.
   The error message found in the logs is similar to the following:
   "00000105 ConnectionEve W
   J2CA0206W: A connection error occurred.  To help determine the
   problem, enable the Diagnose Connection Usage option on the
   Connection Factory or Data Source.
   ...00000105 ConnectionEve A
   J2CA0056I: The Connection Manager received a fatal connection
   error from the Resource Adapter for resource
   JMS$AsynchronousWork_QCF$JMSManagedConnection@10. The exception
   is: com.ibm.msg.client.jms.DetailedMessageFormatException:
   JMSCC0052: An exception occurred while serializing the object:
   'java.io.NotSerializableException:
   com.dwl.base.ControlExtensionProperty'.
   It was not possible to serialize the object because of the
   exception.
   Examine the linked exception for details of the
   error.:java.io.NotSerializableException:
   com.dwl.base.ControlExtensionProperty
   ...
   Caused by: java.io.NotSerializableException:
   com.dwl.base.ControlExtensionProperty
           at
   java.io.ObjectOutputStream.writeObject0(ObjectOutputStream.java:
   1184)
           ... 52 more
   ... 00000105 TCRMCoreParty E   1056
   DIERR 49322 Failed to send asynchronous suspect processing work
   message."
