ERROR DESCRIPTION
 *  In configuration files, you can specify the hostname for
   images.  This is used to offload images to a secondary server
   away from the main process server.
   
    <web-images>
        <prefix>SERVER</prefix>
    </web-images>
   
   This function does not work in Business Process Manager V7.5. It
   was previously working in WebSphere Lombardi Edition V7.2.
