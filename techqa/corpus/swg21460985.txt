Title: IBM Creating or starting a dynamic view reports error: cleartool: Error: Unable to create directory "view": Unrecognized status "2009252814" (from a newer server?) - United States

Text:
create view; startview; unrecognized status; dynamic; clearcase; Unable to create directory "view; Unrecognized status "2009252814"; from a newer server? TECHNOTE (TROUBLESHOOTING)

PROBLEM(ABSTRACT)
 This technote describes an error, Unable to create directory "view": Unrecognized status "2009252814" (from a newer server?), that can occur when creating or starting an IBM Rational ClearCase dynamic view. 

SYMPTOM
 

When creating or starting a dynamic view hosted on an affected system the following error will be received:


cleartool: Error: Unable to create directory "view": Unrecognized status "2009252814" (from a newer server?)


This will sometimes occur after a bad install of the software.



CAUSE
 

The multiversion filesystem (MVFS) on the machine starting the view is at the incorrect version. 


DIAGNOSING THE PROBLEM
 

Running a Cleartool -verall will show that the MVFS version does not match the expected installed version. 


Further details about this output can be found in the technote 1380024 [https://www.ibm.com/support/docview.wss?uid=swg21380024]
