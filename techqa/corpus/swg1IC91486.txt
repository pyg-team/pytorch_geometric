Title: IBM IC91486: RUNNING MQ WITH SHARING CONVERSATION AND SSL KEY REPOSITORY, CIPHER SPECS FAIL AND MQ-QM SHOWS AS DOWN. - United States

Text:
 SUBSCRIBE

APAR STATUS
 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  Running MQ with Sharing Conversation while testing SSL might
   cause Cipher Specs to fail;
   MQ Queue Manager shows as down when using sharing conversation
   (10) on SSL channel connect Queue Manager.
   
   
    
   
   

LOCAL FIX

PROBLEM SUMMARY
 *  Users of both SSL Key Repository and Sharing
   Conversation in an MQ Queue Manager might see this
   problem.
   
   An MQ Queue Manager using both SSL Key Repository and Sharing
   Conversation cannot establish the MQ connection successfully.
   
   
    
   
   

PROBLEM CONCLUSION
 *  The fix is available in 4.0.2.14, 5.0.0.9 and 6.0.0.0.
   
   
   For a list of the latest fix packs available, please see:
   http://www-01.ibm.com/support/docview.wss?uid=swg21237631 [http://www-01.ibm.com/support/docview.wss?uid=swg21237631]
