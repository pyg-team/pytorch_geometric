Title: IBM Why is the Java EE Connector (J2C) Tools feature not available in the Rational Software Architect  for install? - United States

Text:
J2C; Java Connector Tools; J2C Tools; IMS; JCA Projects TECHNOTE (TROUBLESHOOTING)

PROBLEM(ABSTRACT)
 This technote provides information about why the Java EE Connector (J2C) Tools option does not show up in IBM Rational Application Developer or Rational Software Architect for install. 

CAUSE
The Java EE Connector (J2C) Tools feature is only available in the Rational Application Developer for WebSphere Software and Rational Software Architect for WebSphere Software. It is not available in the Rational Application Developer Standard Edition or Rational Software Architect Standard Edition,

ENVIRONMENT
windows, Rational Application Architect, Rational Application Developer

DIAGNOSING THE PROBLEM
Check which edition of the Rational Software Architect or Rational Application Developer you have.

RESOLVING THE PROBLEM
Download the Rational Software Architect for WebSphere Software or the Rational Application Developer for WebSphere Software.
