ERROR DESCRIPTION
 *  The IBM Tivoli Monitoring Agent Builder Version 6.2.3.1 User's
   Guide does not list Red Hat Enterprise Linux 6.0 x86_64 (64-bit)
   as a supported operating system for the Agent Builder.
   
   It also does not document the prerequisite libraries that must
   be installed. 32 bit and 64 bit versions of the following rpms
   are required on Red Hat Enterprise Linux version 6 64 bit
   systems:
   
   - gtk2
   - libXtst
   - compat-libstdc++-33
   - glibc
   - libgc
   - libXp
   
   
    
   
   

LOCAL FIX

PROBLEM SUMMARY
 *  The IBM Tivoli Monitoring Agent Builder Version 6.2.3.1 User's
   Guide does not document Red Hat Enterprise Linux 6.0 x86_64
   (64-bit) as a supported operating system for the Agent Builder.
   
   It also does not document that the prerequisite libraries that
   must be installed.
   
   The Linux Agent builder requires the installation of the latest
   32-bit versions of the following packages:
   libstdc++
   libgcc
   libXp
   gtk2
   
   Further, RedHat Enterprise Linux requires:
   compat-libstdc++-33
   libXtst.
   
   
    
   
   

PROBLEM CONCLUSION
 *  Technote #1600903
   http://www-01.ibm.com/support/docview.wss?uid=swg21600903 [http://www-01.ibm.com/support/docview.wss?uid=swg21600903]
   was published.
   
   The next version of the Agent Builder User's Guide,
   Chapter 3. Installing the Tivoli Monitoring Agent Builder,
   Requirements section will also contain this information.
