Title: IBM Manually Attaining X-Force Content Updates for the ISAM for Web Appliance Family - United States

Text:
Licence TECHNOTE (FAQ)

QUESTION
 How can I manually attain and install the X-Force Content updates for my IBM Security Access Manager for Web, Advanced Access Control, or Federation appliance? 

CAUSE
The networking does not allow the appliance to reach the predefined update server at 'ibmxpu.flexnetoperations.com:443' to retrieve the automatic updates from the IBM ISS Default Licence and Update Server.

ANSWER
You can manually attain the X-Force Content from the IBM Security License Key and Download Center using the following steps : 



A) I have an account for the 'IBM Security License Key and Download Center' : 
1) Navigate to the following URI : 
https://ibmss.flexnetoperations.com/control/isdl/product?child_plneID=123870 [https://ibmss.flexnetoperations.com/control/isdl/product?child_plneID=123870]
You will be prompted to authenticate when accessing this URI.
2) Select the 'Update' hyperlink next to the desired 'Version' of X-Force Content you'd like to download'
3) The desired packages for download will be of the format : 
IBM Security Content Update (PAM) <version>
IBM Security Content Update (PAM) <version> Readme
4) Use the checkbox to select your package and then complete the download using the button at the bottom of the page
5) Once you've downloaded the update and associated Readme, you will update the X-Force Content in a similar manner to Appliance Firmware Updates : 
http://www-01.ibm.com/support/knowledgecenter/SSPREK_8.0.1/com.ibm.isamw.doc_8.0.1/admin/task/alps_installing_updates.html?lang=en [http://www-01.ibm.com/support/knowledgecenter/SSPREK_8.0.1/com.ibm.isamw.doc_8.0.1/admin/task/alps_installing_updates.html?lang=en]
6) You can verify your version by navigating to 'Manage System Settings -> Updates and Licensing -> Overview' from the appliance LMI



B) If you do not have an account for the 'IBM Security License Key and Download Center' : 
1) Use the following URI to register a guest account : https://ibmss.flexnetoperations.com/control/ibms/memberRegistration [https://ibmss.flexnetoperations.com/control/ibms/memberRegistration]
2) Use the following URI to set/reset your password : https://ibmss.flexnetoperations.com/control/ibms/password [https://ibmss.flexnetoperations.com/control/ibms/password]
3) Follow the outline above in A) to attain your fixes.
