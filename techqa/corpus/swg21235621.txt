Title: IBM WDI z/OS Error occurs 11041 - A database error occurred while attempting to open table "EDIENU32.EDIPSBI" - United States

Text:
WDI Z/OS 11041 EDIENU32 EDIPSBI SQL0204N ODBC SQLSTATE 42704 ERROR OPENING TABLE TECHNOTE (TROUBLESHOOTING)

PROBLEM(ABSTRACT)
 When exporting the trading partner profile from WDI 3.2.1 z/OS, error occurs 11041 - A database error occurred while attempting to open table "EDIENU32.EDIPSBI". ODBC return code is -1. Description: SQL0204N "EDIENU32.EDIPSBI" is an undefined name SQLSTATE=42704. Internal Codes: State: S0002,Native:-204, Origin:[IBM][CLI Driver][DB2] 

CAUSE
Hold actions were not run after applying PTFs to WDI z/OS host system.

RESOLVING THE PROBLEM
Check to see if all the hold actions were done that were required for any PTF's that were applied to the z/OS host system. Specifically check UK05042 where the table EDIPSBI needs to be created.
