abstract text
Attempts to deploy the interactive channel in SSL enabled environment results in below error on UI even when correct credentials are provided: "Authentication failed. You may have entered invalid login information or the server group is not available or the server group was restarted." Below error is seen in Campaignweb.log when deployment action is in process: WARN actions.AuthenticateAndDeployAction [167] - Failed authenticating with remote server group, instance URL: https://&lt;host&gt;:&lt;port&gt;
