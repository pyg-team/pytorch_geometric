APAR STATUS
 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  IBM Content Collector cannot restore all email documents into a
   Lotus Domino 8.x.x mailbox that is located on an AS/400 system.
   For some documents, the following error is reported in the trace
    log:
   
   "The content of the note could not be written to the file:
   Error code returned by the operating system: 10035"
   
   
    
   
   

LOCAL FIX
 *  N/A
   
   
    
   
   

PROBLEM SUMMARY
 *  If a Domino email document that has been archived by IBM Content
    Collector has an attachment that is larger than 16 MB, the
   document cannot be restored into a Domino mailbox in a mail
   environment on an AS/400 system.
   
   
    
   
   

PROBLEM CONCLUSION
 *  Starting with IBM Content Collector 2.0.0 Fix Pack 5, documents
   with attachments larger than 16 MB can be restored successfully
   to a Domino email system that runs on an AS/400 system.
