ERROR DESCRIPTION
 *  Image overlay color is not persisting in DotEdit app when batch
   is put on hold and then reopened in DotEdit.
   
   User uses an Image overlay ( red background image) for a
   particular page type so that users can easily know which task is
   currently running. Tests shows that the background changes from
   red to white for the same image after the batch is put on hold.
