ERROR DESCRIPTION
 *  At TDWC 8.6.0.2 level, when querying jobstreams with the TDWC
   the same jobstreams
   are displayed multiple times if you select all statuses.
   When you starting from the   Welcome tdwc panel choose:
   workload scheduler -> workload -> monitor -> Monitor Job Streams
   Task
   (z/OS): MonitorJobstreams_1
   and then checkmark: all Status ( Waiting, Successful, Running,
   Error,
   Canceled),
   then the problem occurs.
   The error occurs only when you select all statuses and
   for Multiple Engines Task Monitor.
   
   
    
   
   

LOCAL FIX

PROBLEM SUMMARY
 *  At TDWC 8.6.0.2 level, when querying jobstreams with the TDWC
   the same jobstreams
   are displayed multiple times if you select all statuses.
   When you starting from the   Welcome tdwc panel choose:
   workload scheduler -> workload -> monitor -> Monitor Job Streams
   When performing a multiple engine query against a z/OS
   engine selecting all job status as filter the result
   contains multiple instance of the same jobstream.
   
   The error occurs only when you select all statuses and
   for Multiple Engines Task Monitor.
