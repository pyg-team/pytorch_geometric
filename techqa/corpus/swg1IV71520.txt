ERROR DESCRIPTION
 *  Approver: Hua Ming Zhong
   
   Environment:
   All windows versions
   All Microsoft SQL Server versions
   Microsoft SQL Server Agent 06.31.02.00
   
   Problem Description: Collector is crashing in Table Detail
   attribute group as query is failing for negative table id.
   
   Following is the reason for collector crashing:
   
   1: During table detail collection if the table id is negative,
   the table size query fails and since the table name is not
   handled properly
   collector crashes while accessing the table name.
   
   Impact: Collector crashes, agent cannot collect data.
   
   Detailed Re-creation Procedure:
   Not Reproducible
   Note: table id can be negative for "tempdb" database, in other
   cases we don't see a reason why it should be negative.
   Hence it cannot be reproduce at will.
   
   Related Files and Output:
   None
