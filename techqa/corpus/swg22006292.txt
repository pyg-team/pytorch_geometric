Title: IBM DataPower Gateway version 7.6.0.8 is now available - United States

Text:
 TROUBLESHOOTING

PROBLEM
The firmware for IBM DataPower Gateway version 7.6.0.8 is now available for download.

RESOLVING THE PROBLEM
Important: IBM is not releasing the v7.6.0 VHD images on IBM Passport Advantage, but you can download the v7.5.2 VHD images. To upgrade DataPower Gateway for XenServer, use a v7.6.0.1 or later scrypt4 firmware image from IBM Fix Central. For additional information, refer to IBM DataPower Gateway for Citrix XenServer v7.6.0.0 is incompatible on initial release [http://www-01.ibm.com/support/docview.wss?uid=swg22006283].


Refer to the Fix Packs [http://www.ibm.com/support/docview.wss?uid=swg24043675] for details on the Authorized Program Analysis Reports (APARs) included in this version. See also:

 * Link to the Fix Central site [http://www.ibm.com/support/fixcentral/]to begin the download process.
 * New to Fix Central? If so, see Fix download method for IBM DataPower Gateways. [http://www.ibm.com/support/docview.wss?uid=swg21376557]
 * IBM DataPower Gateways Firmware Support Lifecycle [http://www.ibm.com/support/docview.wss?uid=swg21246298] for reference.
