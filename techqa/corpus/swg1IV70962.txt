APAR STATUS
 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  When using the Data Item GUI editor to edit the Filterstat
   field a data item in the mm_windows table, the changes take
   place, however the "FILTERSTAT" value (depending on the Impact
   version) either is cutoff or after editing converts it in the
   Filterstat field to HTML <entity> numbers or values.
   
   Impact 5.1.1, 6.1 and 7.1 all behave incorrectly when it comes
   to editing the Filterstat value directly from within the
   mm_windows table via the data item editor.
   
   
    
   
   

LOCAL FIX
 *  This will be fixed in Impact 7.1 with the next 7.1 Fix Pack
   release.
   
   Problem needs to be corrected in Impact 5.1.1 and 6.1.
   
   Appears to work correctly on Impact 6.1.1.
