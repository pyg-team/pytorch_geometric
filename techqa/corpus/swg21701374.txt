Title: IBM Security Bulletin: A security vulnerability has been identified in WebSphere Application Server shipped with IBM Service Delivery Manager (CVE-2015-0138) - United States

Text:
CVE-2015-0138 SECURITY BULLETIN

SUMMARY
 IBM WebSphere Application Server is shipped as a component of with IBM Service Delivery Manager. Information about a security vulnerability affecting WebSphere Application Server has been published in a security bulletin. 

VULNERABILITY DETAILS
Review security bulletin Security Bulletin: Vulnerability with RSA Export Keys may affect IBM WebSphere Application Server (CVE-2015-0138) [http://www-01.ibm.com/support/docview.wss?uid=swg21698613] for vulnerability details and information about fixes.

AFFECTED PRODUCTS AND VERSIONS
 

Principal Product and Version(s) Affected Supporting Product and Version IBM Service Delivery Manager V7.2.1 WebSphere Application Server V6.1.0.0 through 6.1.0.47. IBM Service Delivery Manager V7.2.2 WebSphere Application Server V6.1.0.0 through 6.1.0.47. IBM Service Delivery Manager V7.2.4 WebSphere Application Server V6.1.0.0 through 6.1.0.47.
