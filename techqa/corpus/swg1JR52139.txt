APAR STATUS
 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  When you upgrade the IBM WebSphere Application Server base of
   IBM Business Process Manager (BPM) V8.0.1.3 from V8.0.0.9 to
   V8.0.0.10, you might see an exception with message
   java.lang.NoSuchMethodError:
   com.ibm.ffdc.impl.Ffdc.getProvider()  because of the security
   changes in WebSphere Application Server V8.0.0.10 .

PROBLEM CONCLUSION
 *  A fix is available for IBM BPM V8.0.1.3 that removes the
   WebSphere Application Server V8.0.0.10 dependency from the
   IBM BPM code. As a result, you no longer receive the
   java.lang.NoSuchMethodError.
   
   On Fix Central (http://www.ibm.com/support/fixcentral), search
   for JR52139:
   
        1.Select IBM Business Process Manager with your edition
   from the product selector, the installed version to the fix
   pack level, and your platform, and then click Continue.
        2.Select APAR or SPR, enter JR52139, and click Continue.
   
   
   When you download fix packages, ensure that you also download
   the readme file for each fix. Review each readme file for
   additional installation instructions and information about the
   fix.
