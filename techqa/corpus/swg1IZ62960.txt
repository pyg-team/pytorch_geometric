   Operation's Console for TSA AM and TSA MP, version 3.1
   
   
   Description:
   The current list of supported web browsers doesn't include the
   later versions which do work with the Operations Console and
   which we will support use of TSAAM and TSAMP with. The
   documentation needs to be updated with the additional
   browsers we will allow.

PROBLEM SUMMARY
 *  ****************************************************************
   * USERS AFFECTED: System Automation Application Manager 3.1
   * users
   ****************************************************************
   * PROBLEM DESCRIPTION:
   * The Administrators and Users Guide does not describe current
   * list of supported web browser for the Integrated Solutions
   * Console. The documentation currently only shows:
   *
   * Configuring your Web browser for Integrated Solutions Console
   * The following minimum versions of Web browsers are supported:
   * - Microsoft Internet Explorer V6.0 SP1
   * - Mozilla V1.7.8
   * - Firefox 1.5
   ****************************************************************
   * RECOMMENDATION:
   ****************************************************************
   
   
    
   
   

PROBLEM CONCLUSION
 *  The Administrator's and User's Guide of System Automation
   Application Manager (SA AM) will be changed to address this
   issue.
   .
   Chapter 22. Using Integrated Solutions Console for System
   Automation Application Manager will be adjusted with the
   following information about supported web browsers:
   
   Configuring your Web browser for Integrated Solutions Console
   The following versions of Web Browsers are supported:
   - Microsoft Internet Explorer V6.0 SP1
   - Microsoft Internet Explorer V7.0
   - Firefox 2.0.x
   - Firefox 3.0.x
   .
   Web Browsers other than Firefox and Internet Explorer are not
   supported. Other versions of Firefox and Internet Explorer than
   the versions listed above may work too, but are not supported.
   In particular there are known problems with the following
   version of Internet Explorer:
   - Microsoft Internet Explorer V8.0
   .
   The updated documentation will be made available with fixpack 5
   of System Automation Application Manager.
   .
   Additional Search Keywords.
