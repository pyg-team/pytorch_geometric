ERROR DESCRIPTION
 *  The customer tried to do Virtual Scan on Datacap Navigator using
   "Dynamic Web TWAIN HTML5 Scan SDK", which is added in Datacap
   9.0 FP3, and found that the error occurred in the case that the
   specific Japanese character is included in the file name. The
   file name is "????Updated.tif", and the specific character is
   "?"(SJIS: 0x905C). The second byte '0x5C' is suspected.
