ABSTRACT
 This readme describes the above referenced Refresh Pack, including installation-related instructions, prerequisites and co-requisites, and a description of the enhancements. 


Readme file for: Initiate Master Data Service
Product/Component Releases: 10.1


CONTENT
This fix contains updated data direct drivers, java as well as security fixes for Inspector and Web Reports. Details about the security fixes are available in the security bulletins below. 

Download location: 

Download location fixes from Fix Central http://www.ibm.com/support/fixcentral/ [http://www.ibm.com/support/fixcentral/] 


For Initiate Master Data Service choose:
Select Information Management -> Initiate Master Data Service-> 



Product/Component Name: File Name: Initiate Master Data Service 10.1.031518_IM_Initiate_MasterDataService_ALL_Refresh Pack 
Prerequisites: 

To ensure that all components in an IBM Initiate Master Data Service installations are synchronized, you must install the latest fixes on each component. IBM Support supports only installations whose components are all at the same release level.
