APAR STATUS
 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  Filter values repeat for filter in either a Report Studio or a
   Workspace Advanced report if there is another filter with After
   Auto Aggregation set
   
   
    
   
   

LOCAL FIX
 *  Remove the After Auto Aggregation setting on the other filter
