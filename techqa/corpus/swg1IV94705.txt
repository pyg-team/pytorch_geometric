   Problem Description:
   Sybase agent logs an error for KOYLOCKS cursor as Col_function
   failed to load.
   The cursor uses LOCKTYPE attribute and fails if column length of
   LOCKTYPE is greater than 256.
   
   Detailed Re-creation Procedure:
   1. Install sybase agent on Solaris or AIX system.
   2. Configure sybase agent with the IBM Tivoli Monitoring.
   3. Enable detailed trace and visit "Locking" workspace from TEP.
   4. Check collector log file for KOYLOCKS attribute group.
   
   
   Related Files and Output:
   Failure error for KOYLOCKS cursor with below error in collector
   logs
   PLO3105E (153556) Col_function failed for load, cur = KOYLOCKS,
   load = NUMLOCKS
