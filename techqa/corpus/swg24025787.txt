Title: IBM PM00610; 7.0.0.7 SSO LTPATOKEN IS PROCESSED INCORRECTLY WHEN REQUIRES SSL IS SET - United States

Text:
PM00610; PM00610; PM00610 DOWNLOADABLE FILES

ABSTRACT
 SSO LTPATOKEN IS PROCESSED INCORRECTLY WHEN "REQUIRES SSL" IS ENABLED 

DOWNLOAD DESCRIPTION
PM00610 resolves the following problem:

ERROR DESCRIPTION: 
User specifies that the single sign-on function is enabled only when requests are made over HTTPS Secure Sockets Layer (SSL) connections. Once this function is enabled in the console, WebSphere Application Server is not setting the secure flag as true. 

LOCAL FIX: 
NONE

PREREQUISITES
None
