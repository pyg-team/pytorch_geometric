 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  After clicking Add button in the Add Employees and other add
   related dialog boxes in People in Scope page or Hold Notice
   Recipients page, the dialog box is closed as soon as one entry
   is selected and user is not able to continue adding
   multiple recipients.
   
   
    
   
   

LOCAL FIX

PROBLEM SUMMARY
 *  After clicking Add button in the Add Employees and other add
   related dialog boxes in People in Scope page or Hold Notice
   Recipients page, the dialog box is closed and user is not able
   to continue adding recipients.
   
   
    
   
   

PROBLEM CONCLUSION
 *  It is rare that only one recepient is added at a time through
   the Add Employees and other add dialog boxes. So this issue was
   fixed by changing the functionality of the Add dialog to remain
   open after adding currently selected recepients. A separate Add
   and Close button was added to close the dialog after adding the
   selected recepients.
   
   This fix is available from Fix Central in Atlas 6.0.3 iFix 2
