Title: IBM Security Bulletin: Fix Available for LDAP Injection Vulnerability in IBM WebSphere Portal (CVE-2015-7472) - United States

Text:
 SECURITY BULLETIN

SUMMARY
 A fix is available for an LDAP injection vulnerability in IBM WebSphere Portal (CVE-2015-7472). 

VULNERABILITY DETAILS
CVEID: CVE-2015-7472 [http://cve.mitre.org/cgi-bin/cvename.cgi?name=CVE-2015-7472]
DESCRIPTION: IBM WebSphere Portal is vulnerable to LDAP injection. A remote attacker could send specially-crafted requests, which could allow the attacker to view, add, modify or delete information in the user repository.
CVSS Base Score: 7.2
CVSS Temporal Score: See https://exchange.xforce.ibmcloud.com/vulnerabilities/108430 [https://exchange.xforce.ibmcloud.com/vulnerabilities/108430] for the current score
CVSS Environmental Score*: Undefined
CVSS Vector: (CVSS:3.0/AV:N/AC:L/PR:N/UI:N/S:C/C:L/I:L/A:N) 

AFFECTED PRODUCTS AND VERSIONS
 

 * IBM WebSphere Portal 8.5 
 * IBM WebSphere Portal 8.0 
 * IBM WebSphere Portal 7.0 
 * IBM WebSphere Portal 6.1

For unsupported versions IBM recommends upgrading to a fixed, supported version of the product. REMEDIATION/FIXES
Apply Interim Fix PI53426 [http://www.ibm.com/eserver/support/fixes/fixcentral/swg/quickorder?productid=WebSphere%20Portal&brandid=5&apar=PI53426] or a Cumulative Fix containing PI53426 [http://www.ibm.com/eserver/support/fixes/fixcentral/swg/quickorder?productid=WebSphere%20Portal&brandid=5&apar=PI53426].
