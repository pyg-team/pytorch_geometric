ERROR DESCRIPTION
 *  Analysis (GSA) : If no analysis licence is activated,
   CATIA is ABEND when activating an analysis
   representation
   .
   Scenario :
   1. Start CATIA with only generic licence activated
   2. Open 'sample14.CATProduct'. (WB: Product
   Structure)
   2. MB3 on 'Surface1' and select 'Representations/
   Manage Representations'.
   3. In the 'Manage representation' window,
   select 'Shape 2' and click 'Activate'. Then
   click 'close'
   4. No shape is displayed because analysis licence is not
   activated. This is normal
   5. Close the CATProduct. CATIA is abend
   .
