Title: IBM Oracle 32-bit Instant Client setup for Oracle Database 12c to be used with IBM Rational Synergy 7.2.1.5 interim fix 001 or later versions - United States

Text:
Oracle12c; Instant Client; Oracle 32-bit instant client; IBM Rational Synergy 7.2.1.5 interim fix 001 TECHNOTE (FAQ)

QUESTION
 Oracle 32-bit Instant Client versions for Oracle Database 12c to be used with IBM Rational Synergy 7.2.1.5 iFix001 or later. 

CAUSE
IBM Rational Synergy requires Oracle 32-bit instant clients.

ANSWER
IBM Rational Synergy Versions 7.2.1.5 iFix001 or later, supports Oracle Database 12c with the following Oracle 32-bit Instant Client versions. 

Oracle Database Platform Oracle Instant Client Database 12c AIX basic-11.2.0.1.0-aixppc32.zip Database 12c Solaris basic-11.2.0.1.0-solaris-sparc32.zip Database 12c Linux basic-10.2.0.5.0-linux.zip 
For Oracle 32-bit Instant Client Installation details please refer to existing documentation [http://www.ibm.com/support/knowledgecenter/SSRNYG_7.2.1/com.ibm.rational.synergy.install.unix.oracle.doc/topics/s_t_igu_oracle_patch_install.html] RELATED INFORMATION
