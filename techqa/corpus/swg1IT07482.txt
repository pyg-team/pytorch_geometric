Title: IBM IT07482: DATAPOWER LOG TARGET TYPE "SYSLOG TCP" DOES NOT CONTAIN GLOBAL TRANSACTION ID IF ENABLED - United States

APAR STATUS
 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  Using DataPower, the user observed that all log target types but
   "Syslog TCP" do contain Global Transaction ID ("gtid") if
   enabled.
   
   
    
   
   

LOCAL FIX

PROBLEM SUMMARY
 *  Affected is the use of "Syslog TCP" log target type.
   
   Global Transaction ID is not shown.
   
   
    
   
   

PROBLEM CONCLUSION
 *  Fix is available in 7.0.0.8 and 7.1.0.5.
   
   For a list of the latest fix packs available, please see:
   http://www-01.ibm.com/support/docview.wss?uid=swg21237631 [http://www-01.ibm.com/support/docview.wss?uid=swg21237631]
