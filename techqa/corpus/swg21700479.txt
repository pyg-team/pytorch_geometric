Title: IBM Security Bulletin: Log viewer vulnerability affects IBM PureApplication System (CVE-2014-6190) - United States

Text:
 SECURITY BULLETIN

SUMMARY
 Log viewer vulnerability affects IBM PureApplication System. 

VULNERABILITY DETAILS
CVEID: CVE-2014-6190 [http://cve.mitre.org/cgi-bin/cvename.cgi?name=CVE-2014-6190]
DESCRIPTION: 
Defined system users without proper permissions can access the log viewer functionality by entering the log page URLs in their browser.

CVSS Base Score: 5.0
CVSS Temporal Score: See https://exchange.xforce.ibmcloud.com/vulnerabilities/98561 [https://exchange.xforce.ibmcloud.com/vulnerabilities/98561] for the current score
CVSS Environmental Score*: Undefined
CVSS Vector: (AV:N/AC:L/Au:N/C:P/I:N/A:N) 

AFFECTED PRODUCTS AND VERSIONS
 

 * IBM PureApplication System V1.0 
 * IBM PureApplication System V1.1 
 * IBM PureApplication System V2.0

REMEDIATION/FIXES
The solution is to apply the following IBM PureApplication System version fixes. 
The PureSystems Managers on all IBM PureApplication Systems types are affected. Upgrade the IBM PureApplication System to the following interim fix levels:


 * IBM PureApplication System V1.0
   Upgrade to IBM PureApplication System V1.0.0.4 Interim Fix 11 
   
 * IBM PureApplication System V1.1
   Upgrade to IBM PureApplication System V1.1.0.5 Interim Fix1
   
 * IBM PureApplication System V2.0
   Upgrade to IBM PureApplication System V2.0.0.1

WORKAROUNDS AND MITIGATIONS
