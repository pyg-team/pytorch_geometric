ERROR DESCRIPTION
 *  Datacap navigator start panel not displaying MESSAGE variable
   error message

   box with the contents of the MESSAGE variable is displayed when
   validation fails in the start batch panel.  However, in
   Navigator, only the message "Task profile failed" is displayed
