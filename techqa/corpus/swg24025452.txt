Performance Report PDF for WebSphere MQ File Transfer Edition V7.0.1 for System Z. 

This SupportPac includes performance measurements, with figures and tables to present the data. Analysis of the results and recommendations for WebSphere MQ File Transfer Edition are also included.

This SupportPac is intended for people who will be designing and implementing solutions using WebSphere MQ File Transfer Edition V7.0.1 on System Z. The SupportPac demonstrates how a user can make the most of their hardware when used with WebSphere MQ File Transfer Edition on System Z.
