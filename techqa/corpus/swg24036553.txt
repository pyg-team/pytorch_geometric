Title: IBM CYNCR9101E pop-up error occurs during ITCAM for AD Managing Server 7.1 installation when using Oracle 12C - United States

Text:
Support for Oracle 12C; CYNCR9101E DOWNLOADABLE FILES

ABSTRACT
 When using existing Oracle 12C database, pop-up error CYNCR9101E occurs during the ITCAM for AD Managing Server 7.1 installation 

DOWNLOAD DESCRIPTION
This is a fix for the pop-up error CYNCR9101E during the Managing Server installation. If you have installed Oracle 12C and then try to install the Managing Server, you will see the following error in a pop-up window after you select the JDBC path and hit next. 

CYNCR9101E The classes12.jar file, classes12.zip file or ojdbc14.jar file was not found in the specified JDBC path.
.
.
The specified JDBC path must be a directory containing the classes12.jar file, classes12.zip file or ojdcbc14.jar file. 

DOWNLOAD PACKAGE
Download the ojdbc14.jar from the following link. 

http://www.oracle.com/technetwork/database/enterprise-edition/jdbc-10201-088211.html [http://www.oracle.com/technetwork/database/enterprise-edition/jdbc-10201-088211.html] 

Copy it in the ORACLE_HOME Directory/jdbc/lib directory.
For example on Windows 2012 R2, the directory looks like C:\app\ctora\product\12.1.0\dbhome_1\jdbc\lib

Note: You can also find this jar in the Oracle 11 or previous versions.
