APAR STATUS
 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  Datapower configurations using NFS log targets might cause a
   DataPower restart when the corresponding application domain is
   restarted.
   
   
    
   
   

LOCAL FIX

PROBLEM SUMMARY
 *  Affected are configurations using NFS log targets in an
   application domain.
   When the domain is restarted or the log target object set to
   admin-state disabled, the appliance may restart, generating a
   backtrace.
   
   
    
   
   

PROBLEM CONCLUSION
 *  The fix is available in 7.1.0.20, 7.2.0.17, 7.5.0.11, 7.5.1.10,
   7.5.2.10 and 7.6.0.3.
   For a list of the latest fix packs available, please see:
   http://www-01.ibm.com/support/docview.wss?uid=swg21237631 [http://www-01.ibm.com/support/docview.wss?uid=swg21237631]
