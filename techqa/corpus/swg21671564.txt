Title: IBM A NullPointerException message occurs while accessing the BPC Explorer in IBM Business Process Manager (BPM) - United States

Text:
beforePhase; BPC Explorer; NullPointerException. setNames TECHNOTE (TROUBLESHOOTING)

CAUSE
The Configuring administrative and application security [http://www.ibm.com/support/knowledgecenter/SSFPJS_7.5.1/com.ibm.wbpm.admin.doc/topics/tsec_enablingglobal.html?cp=SSFPJS_7.5.1&lang=en] topic mentions the following information: 

Important: Application security is required by IBM Business Process Manager and must not be turned off in the administrative console. Application security is also enabled by default, and is in effect only when administrative security is enabled.


RESOLVING THE PROBLEM
To resolve this issue, enable the administrative security and application security of IBM Business Process Manager.
