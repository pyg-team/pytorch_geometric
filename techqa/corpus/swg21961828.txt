Title: IBM Accessing Rational Team Concert admin page results in error 'CRJAZ1114I: The required service owner could not be found' - United States

Text:
CRJAZ1114I The required service owner for "com.ibm.team.repository.service.internal.IServletConfigurationService" could not be found. TECHNOTE (TROUBLESHOOTING)

PROBLEM(ABSTRACT)
 Attemps to access IBM Rational Team Concert admin page results in error "CRJAZ1114I The required service owner for com.ibm.team.repository.service.internal.IServletConfigurationService could not be found" 

CAUSE
Websphere Application Server (WAS) class cache is still holding on to previous version of classes

ENVIRONMENT
WebSphere Application Server

DIAGNOSING THE PROBLEM
Below error exists in SystemOut.log : 

000000b1 ServletWrappe E com.ibm.ws.webcontainer.servlet.ServletWrapper service Uncaught service() exception thrown by servlet equinoxbridgeservlet: java.lang.VerifyError: JVMVRFY007 final method overridden; class=net/jazz/ajax/service/internal/ProxyService, method=service(Ljava/lang/String;Ljava/lang/String;Ljavax/servlet/http/HttpServletRequest;Ljavax/servlet/http/HttpServletResponse;)V, pc=0


RESOLVING THE PROBLEM
Apply the steps below in order to clear the OSGi cache and JVM class cache: 




 1. Stop the server before clearing the cache
    
    
 2. Run the following script in each profile to clear the OSGi class cache:
    
    <WebSphere_HOME>\profiles\profile_name\bin\osgiCfgInit.bat
    
    
 3. Run the following script to clear the JVM's class cache:
    
    <WebSphere_HOME>\bin\clearClassCache.bat
    
    
 4. Clear the contents of directory javasharedresources
    
    Path:C:\Windows\System32\config\systemprofile\AppData\Local\javasharedresources
