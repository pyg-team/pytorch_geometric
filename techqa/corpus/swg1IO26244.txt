APAR STATUS
 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  When running a Workplace XT search that queries multiple object
   stores, the repository assigned to items in the search results
   doesn't always match the repository where they are stored.
   
   
    
   
   

LOCAL FIX

PROBLEM SUMMARY
 *  When running a Workplace XT search that queries multiple object
   stores, the repository assigned to items in the search results
   doesn't always match the repository where they are stored.
   
   
    
   
   

PROBLEM CONCLUSION
 *  With this fix in IBM Content Navigator V2.0.3.8 IF019, V3.0.2
   IF006, V3.0.3 IF004, and V3.0.4, items that are returned by the
   search are assigned the repository where they are stored.
