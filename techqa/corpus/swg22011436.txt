Title: IBM MustGather: Logon errors in the IBM Case Manager client - United States

Text:
MustGather case manager client logon TECHNOTE (TROUBLESHOOTING)

PROBLEM(ABSTRACT)
 If you are experiencing logon errors in the IBM Case Manager client, this MustGather will assist you in collecting the data necessary to diagnose and resolve the issue. 

SYMPTOM
The IBM case Manager client displays a logon error message.


DIAGNOSING THE PROBLEM
The following questions help narrow down the problem area. Performing the tests in this section and providing detailed information to IBM will help in troubleshooting the problem efficiently. 

 * Are there any recent configuration changes (e.g. patching, migration, SSL, SSO, etc.)? 
 * How many users are affected? Are there specific users who are or who are not affected? 
 * Can the affected users successfully log into another Enterprise Content Manager application?  * For example, you can test the following if the user has access:  * another IBM content Navigator desktop; 
       * ACCE in the Content Platform Engine.
