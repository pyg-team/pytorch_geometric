APAR STATUS
 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  When you use an AJAX Service for autocompletion in a coach, the
   AJAX service uses environment variables that is set in the
   process app or toolkit for the autocompletion instead of the
   current value set in the Procss Admin Console when the service
   runs.
   
   PRODUCTS AFFECTED:
   IBM Business Process Manager (BPM) Advanced
   IBM BPM Standard
   IBM BPM Express

 *  No additional information is available.

   environment variable value set via the Process Admin Console.
