Title: IBM Actions issued by situations are failed. - United States

Text:
KRAKGE005; "Automation request rejected"; "no security authorization credential" TECHNOTE (TROUBLESHOOTING)

PROBLEM(ABSTRACT)
 Actions issued by situations are failed and the following message is logged LG0 log file of OS agent: 
"KRAKGE005 Automation request rejected. User <username> has no security authorization credential"
The message is logged for any action issued by any situation. 

CAUSE
Refer the following technote: [http://www-01.ibm.com/support/docview.wss?uid=swg21631465] 

- IBM Tivoli Monitoring 6.3 Take action command fails with message "automation request rejected" [http://www-01.ibm.com/support/docview.wss?uid=swg21631465]
http://www-01.ibm.com/support/docview.wss?uid=swg21631465 [http://www-01.ibm.com/support/docview.wss?uid=swg21631465]


DIAGNOSING THE PROBLEM
Check version of ITM components. 

This issue can happen with OS Agent is 6.2.3 or earlier, if Agent Framework is 6.3.


RESOLVING THE PROBLEM
Downgrade version of the ITM 6.3 components to 6.2.3 or earlier.
