 XML message bus probe : How can the integration be used? 

ANSWER
Support's Guide to the Message Bus Integration covers how to configure the XML [Message Bus] Probe and Gateway for use with the Apache MQ server, as well as the general usage of the probe and gateway [see the Related Information section] 

General guidance for the HTTP transformers and WebSphere JNDI lookups are included.
