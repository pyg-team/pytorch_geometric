Title: IBM JR47582: SERVER STATUS IS NOT CORRECT - United States

Text:
  A FIX IS AVAILABLE
Version 8.5.0 Fix Pack 1 for the IBM Business Process Manager products [http://www-01.ibm.com/support/docview.wss?uid=swg24035702]


SUBSCRIBE
You can track all active APARs for this component.



APAR STATUS
 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  Create a new process server cluster DE env, select environment
   type as stage. After creating the DE and start the new DE, open
   Process center to check the environment type, it is Development,
   NOT stage.
   
   
    
   
   

LOCAL FIX
 *  Change Stage to Staging in 99Local.
   *****************
   <server-name>TEST1</server-name>
   <server-description>~SERVER_DESCRIPTION~</server-description>
   <environment-type>Stage</environment-type>
   *****************
   
   
    
   
   

PROBLEM SUMMARY
 *  ****************************************************************
   * USERS AFFECTED:  IBM Business Process Manger V8.5 standard   *
   *                  and                                         *
   *                  advanced edition users, specifically users  *
   *                  of the Deployment Environment wizard in     *
   *                  the administrative console                  *
   ****************************************************************
   * PROBLEM DESCRIPTION: After using the Deployment              *
   *                      Environment wizard to install a         *
   *                      process server as a stage, the user     *
   *                      sees the deployment environment         *
   *                      incorrectly listed as an environment    *
   *                      in Process Center.                      *
   ****************************************************************
   * RECOMMENDATION:                                              *
   ****************************************************************
   After creating a process server deployment stage using the
   Deployment Environment wizard and then opening Process Center
   to check the environment type, the user sees that the process
   server is listed as an environment, not a stage.
   
   
    
   
   

PROBLEM CONCLUSION
 *  FIX AVAILABILITY
   A fix for IBM BPM V8.5.0.0 is available on Fix Central:
   http://www.ibm.com/support/fixcentral. Search for JR47582.
   Then see the installation and dependency information in the
   readme.txt file, which is available with the fix package as a
   separate download.
