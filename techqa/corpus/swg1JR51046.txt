PROBLEM SUMMARY
 *  ****************************************************************
   * USERS AFFECTED:  IBM Business Process Manager (BPM)          *
   *                  Advanced                                    *
   *                  IBM BPM Standard                            *
   *                  IBM BPM Express                             *
   ****************************************************************
   * PROBLEM DESCRIPTION: When checking task data of an           *
   *                      external activity in Process Admin      *
   *                      Console -> Process Inspector, an        *
   *                      CWTBG0019E error occurs and the data    *
   *                      cannot be retrieved.                    *
   ****************************************************************
   * RECOMMENDATION:                                              *
   ****************************************************************
   When a user expands the Data field of a process instance
   within Process Inspector, the following error is displayed in
   a pop-up window:
   "CWTBG0019E: Unexpected exception during execution. Exception
   information: 'For input string: "null"'."
   The corresponding exception stack may be seen in the logs:
   Caused by:
   com.ibm.processinspector.rest.ProcessAdminRestException:
   com.ibm.processinspector.rest.ProcessAdminRestException
   at
   com.ibm.processinspector.rest.TaskActionImpl.getData(TaskActionI
   mpl.java:174)
   at
   com.ibm.processinspector.server.handlers.ActivityInstanceDataHan
   dler.getData(ActivityInstanceDataHandler.java:123)
   at
   com.ibm.processinspector.server.handlers.ActivityInstanceDataHan
   dler.handleRequest(ActivityInstanceDataHandler.java:96)
   ... 37 more
   The issue occurs due to a null process instance ID being used
   in a REST API request made by Process Inspector.
   The following HTTP URL request may be seen in the browser's
   web console or fiddle logs:
   https://<host>:<port>/rest/bpm/wle/v1/process/null [https://<host>:<port>/rest/bpm/wle/v1/process/null]
   The problem only occurs when the task is on an External
   Activity.
   
   
    
   
   

PROBLEM CONCLUSION
 *  A fix is available for IBM BPM v8.5.5.0.  The error no longer
   occurs as the proper process instance ID is now being passed.
   
   On Fix Central (http://www.ibm.com/support/fixcentral), search
   for JR51046:
   1.Select IBM Business Process Manager with your edition from
   the product selector, the installed version to the fix pack
   level, and your platform, and then click Continue.
   2.Select APAR or SPR, enter JR51046, and click Continue.
   When you download fix packages, ensure that you also download
   the readme file for each fix. Review each readme file for
   additional installation instructions and information about the
   fix.
