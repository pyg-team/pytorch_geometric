ERROR DESCRIPTION
 *  Problem Description:
   Created folder is not displayed on Japanese until refreshing by
   Web browser.
   It occurs when installing Language pack with add mode.
   
   Environment:
   
   8.0.x
   
   Steps to reproduce:
   
   1. Configure Japanese environment by installing Language Pack
   with Add mode.
   2. Set "Preferred alternate name languages:" to English and
   Japanese in server setting. It is a key to reproduce.
   3. Restart HTTP task
   4. Create a DWA user, then login via web browser. (IE6 or IE7 or
   FireFox)
   5. Create an appropriate folder, then whole contents of mail
   view is reloaded, but created folder is not displayed.
   
   
    
   
   

LOCAL FIX
 *  - Refresh by web browser
   - Or, remove Japanese from the above option in server setting if
   user do not use alternate name. Then, clear Web browser's
   cache.
