ABSTRACT
 This is the first fix pack after ITCAM Agents for WebSphere Messaging Version 7.3.0 GA 

DOWNLOAD DESCRIPTION



 * This fix pack fixes the APARs of previous product versions. 
 * This fix pack provides support for IBM Integration Bus V10 and several other enhancements(See readme). 
 * This fix pack is certificated with RHEL 7, IBM iSeries v7.2.
