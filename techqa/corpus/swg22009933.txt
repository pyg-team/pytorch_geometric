Title: IBM Cognos for Connections install fails with error CRIMA1217E - United States

Text:
cognos install TECHNOTE (TROUBLESHOOTING)

PROBLEM
The installation of Cognos fails when attempting to perform a gui installation .


SYMPTOM
The Cognos installation will fail with the following errors:
Within the installation manager logs:
<message>Error during "install" phase: 
CRIMA1217E: A problem occurred during the execution of the ../Cognos/ant_execution.xml file. 

Explanation: A package has an issue that cannot be resolved by Installation Manager. 

Within the cognos-setup.log: 
Error: Failed to find file 
"../shared/install/Cognos/bi/linux/setup.csp". Unable to continue. 



CAUSE
The customer was performing the install from the shared directory which introduced file/directory access issues that impacted the installation process causing it to fail.

ENVIRONMENT
IBM Connections 6
Linux platform

DIAGNOSING THE PROBLEM
Enable the following traces within the log.properties file prior to reproducing the installation failure:
InstallOperation=DEBUG
InstallOperation/pause.on_error=true

RESOLVING THE PROBLEM
The customer was performing the install from the shared directory as revealed by the path specified in the cognos-setup.log: 
../shared/install/Cognos/bi/linux/setup.csp 


performing the install with the files on the local system, non shared location resolved the issue
