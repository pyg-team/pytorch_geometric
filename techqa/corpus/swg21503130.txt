Title: IBM An exception occurs for WebSphere Enterprise Service Bus (WESB) when connecting to a Microsoft SQL database through JDBC driver 1.2 - United States

Text:
jdbc; BMWSD0429E TECHNOTE (TROUBLESHOOTING)

PROBLEM(ABSTRACT)
 The Microsoft SQL JDBC driver fails to connect to the Microsoft SQL database. 

SYMPTOM
You might encounter the following error: 

BMWSD0429E: An SQLException was encountered while communicating with the database. Error code: 0, SQL State: 08S01, Message: The driver could not establish a secure connection to SQL Server by using Secure Sockets Layer (SSL) encryption. Error: The TDS protocol stream is not valid..DSRA0010E: SQL State = 08S01, Error Code = 0



RESOLVING THE PROBLEM
To resolve this issue, upgrade to Microsoft SQL JDBC driver Version 2.0. For more information, see the Microsoft SQL Server forum [http://social.msdn.microsoft.com/Forums/en-US/sqldataaccess/thread/6a0263ab-e35f-41b3-b4ca-fcdb1cfc8ac6].
