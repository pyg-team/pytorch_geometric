Title: IBM PK58509: THE JRE PATH UNDER INSTALL IMAGE, WHICH IS DESCRIBED IN THE VERS ION 6.1 INFORMATION CENTER, IS INCORRECT - United States

Text:
 SUBSCRIBE
You can track all active APARs for this component.



APAR STATUS
 * CLOSED AS DOCUMENTATION ERROR.
    
   
   

ERROR DESCRIPTION
 *  The installation instructions in the Information Center for
   WebSphere Application Server Version 6.1 incorrect states the
   following information:
   
   Step 3
   If a problem occurs with the Java Runtime Environment (JRE),
   force the installation to use the correct JRE.
    UNIX :-   ./install -is:javahome disc_mount_point/WAS/java/jre
    WIndows:-   install -is:javahome disc_drive:\WAS\java\jre
   
   The correct path should be-
   AIX, HP-UX, Linux, Solaris operating systems:
   ./install -is:javahome disc_mount_point/downloaded_directory/
   JDK/jre.pak/repository/package.java.jre/java/jre
   
   Windows operating systems:
   install -is:javahome disc_drive\downloaded_directory\JDK\
   jre.pak\repository\package.java.jre\java\jre
   
   Where:
   downloaded_directory is the name of the directory in which you
   downloaded the image.
   
   This incorrect information was found in the "Installing the
   base product with full installation options" topic in the
   information center.
   
   
    
   
   

LOCAL FIX

PROBLEM SUMMARY
 *  ****************************************************************
   * USERS AFFECTED:  This APAR affects users who are installing  *
   *                  WebSphere Application Server Version 6.1    *
   *                  and having difficulties with the Java       *
   *                  Runtime Environment (JRE).                  *
   ****************************************************************
   * PROBLEM DESCRIPTION: The "Installing the base product with   *
   *                      full installation options" topic in     *
   *                      the Information Center for WebSphere    *
   *                      Application Server V6.1 specifies the   *
   *                      JRE location in the following           *
   *                      commands:                               *
   *                      For the AIX, HP-UX, Linux, and          *
   *                      Solaris operating systems: ./install    *
   *                      -is:javahome                            *
   *                      disc_mount_point/WAS/java/jre           *
   *                                                              *
   *                      For the Windows operating systems:      *
   *                      install -is:javahome                    *
   *                      disc_drive:WASjavajre                   *
   *                                                              *
   *                      However, the JRE cannot be found.       *
   ****************************************************************
   * RECOMMENDATION:                                              *
   ****************************************************************
   The location of the JRE is incorrect in the previously listed
   install commands.
