 The list below identifies the supported releases of IBM Decision Optimization Center from which you can select specific detailed system requirements. 

CONTENT
The detailed system requirements information is available through the Software Product Compatibility Reports [http://publib.boulder.ibm.com/infocenter/prodguid/v1r0/clarity/index.jsp] web site. This web site provides you will the ability to dynamically generate operating system, prerequisite, server virtualization environment, translation, end of service, and detailed system requirements reports for your specific product, release, and operating system. 


This document describes the minimum product levels that need to be installed before opening a problem report with IBM Decision Optimization Center. Because other products frequently ship fixes, updates, and new releases, testing every configuration is not possible. In general and unless otherwise stated, you can install and run with updates to supported products if those updates are forward compatible. 

Here are the detailed system requirements for IBM Decision Optimization Center version 3.8 and later. Refer to the Software Product Compatibility Report [https://www.ibm.com/software/reports/compatibility/clarity/softwareReqsForProduct.html] if you are looking for versions 3.7 or 3.6, which were released under the name IBM ILOG ODM Enterprise.
