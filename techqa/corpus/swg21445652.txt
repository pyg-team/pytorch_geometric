Title: IBM Starting IBM Software Delivery Platform version 8.0 products with a Floating license may lead to: "JVM terminated. Exit code=8096" - United States

Text:
J9VMThread; checking license status; Exit code=8096; RATLC00423544; PM22221 TECHNOTE (TROUBLESHOOTING)

PROBLEM(ABSTRACT)
 Attempts to use floating license support with IBM Software Delivery Platform version 8.0 products such as Rational Software Architect for Websphere, Rational Software Architect or Rational Application Developer version 8.0 results in the error "JVM terminated. Exit code=8096" or simply a crash of the application after seeing the Progress dialog "Checking license status..." for a while.

CAUSE
This issue was identified as a product defect under APAR PM22221. 


The IBM Rational License Server version is older than v8.1.1. 

The version 8.0 of Rational Software Architect for WebSphere, Rational Software Architect and Rational Application Developer requires Rational License Server version v8.1.1 or higher.


RESOLVING THE PROBLEM
To resolve the problem, install a Rational License server v8.1.1 and use IBM Installation Manager to reconfigure floating license support. 

Follow the steps outlined in the InfoCenter topic "Configuring client computers to use floating keys using Installation Manager" [http://publib.boulder.ibm.com/infocenter/rational/v0r0m0/index.jsp?topic=/com.ibm.rational.license.doc/topics/t_config_client_float_keys.html] or read technote 1403623 [http://www.ibm.com/support/docview.wss?uid=swg21403623].
