   PROBLEM DESCRIPTION:
   ================
   Portal server fails to start when using SSL certificates that
   use the SHA-256 signature algorithm.
   
   RECREATE INSTRUCTIONS:
   ==================
   Create a self-signed or root CA signed certificate using the
   SHA-256 signature algorithm. Install in ITM keyfile database as
   the default.
