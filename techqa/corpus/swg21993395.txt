Title: IBM Knowledge Collection: PureData System for Analytics - Query History Data Collection - United States

Text:
 RECOMMENDED RESOURCES

ABSTRACT
 A single stop shop of resources and documents to help you configure and setup Query History Data Collection on an IBM PureData System for Analytics appliance. 

CONTENT
Capturing query history data on a Pure Data for Analytics appliance is a key DBA task needed to understand the usage trends and patterns along with gathering information regarding object access, query runtimes, access plans, troubleshoot long running queries etc. 
This document intends to capture various sources of published information on configuring/setting up query history data that would allow end users greater self sufficiency and flexibility in administering the PDA appliance. 

Below are various resources available that explain configuring query history in detail as well as some best practices around it.
