Title: IBM Wrap table column text in list tab - United States

Text:
TPAEICD; wrap; table; col; length; line; feed TECHNOTE (FAQ)

QUESTION
 How to wrap table column text in list tab? 

ANSWER
You can wrap table column text according to number of characters by using property wraplength in the tablecol tag e.g.

<tablecol dataattribute="description" id="results_showlist_tablebody_3" sortable="false" width="200" wraplength="100"/>

This will wrap the text once it reaches 100 characters into a new line.
