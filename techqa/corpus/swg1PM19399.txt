LOCAL FIX

PROBLEM SUMMARY
 *  ****************************************************************
   * USERS AFFECTED:  Users of the WebSphere MQ messaging provider*
   *                  for WebSphere Application Server Version 7.0*
   ****************************************************************
   * PROBLEM DESCRIPTION: The WebSphere MQ JCA Resource Adapter   *
   *                      within WebSphere Application            *
   *                      Server Version 7.0 requires updating    *
   ****************************************************************
   * RECOMMENDATION:                                              *
   ****************************************************************
   The WebSphere MQ JCA Resource Adapter shipped within WebSphere
   Application Server Version 7.0 is used by all applications
   connecting to WebSphere MQ from within the application server.
   
   WebSphere MQ fix pack 7.0.1.3 includes a number of updates to
   the WebSphere MQ JCA resource adapter that apply to WebSphere
   Application Server users.
   
   
    
   
   

PROBLEM CONCLUSION
 *  The fix for this APAR updates the WebSphere MQ JCA resource
   adapter shipped with WebSphere Application Server.
   With the fix applied, the following log entry should be seen
   on all servers on all nodes (except those where the 'Update
   resource adapter...' button has been used to manually update
   the resource adapter):
   WMSG1703I: RAR implementation version 7.0.1.3-k701-103-100812
   
   The fix for this APAR is currently targeted for inclusion in
   fix pack 7.0.0.13.  Please refer to the Recommended Updates
   page for delivery information:
   http://www.ibm.com/support/docview.wss?rs=180&uid=swg27004980 [http://www.ibm.com/support/docview.wss?rs=180&uid=swg27004980]
