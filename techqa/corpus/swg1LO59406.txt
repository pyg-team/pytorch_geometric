Title: IBM LO59406: UNABLE TO INSTALL CLIENT ON NOKIA DEVICE - United States

Text:
 SUBSCRIBE
You can track all active APARs for this component.



APAR STATUS
 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  Customer reported that on some Nokia devices, installation fails
   when pressing next key after filling in server name, username
   and password.
   
   
    
   
   

LOCAL FIX

PROBLEM SUMMARY
 *  Special characters in the host name of the Traveler server or
   proxy server would cause LMI on Nokia to fail to download the
   Lotus Traveler client.
   
   
    
   
   

PROBLEM CONCLUSION
 *  The LMI client for Nokia was updated to handle the unexpected
   characters.
   
   
    
   
   

TEMPORARY FIX
 *  This fix will be included in Lotus Notes Traveler 8.5.2.3 and
   any future releases.  See this technote for latest maintenance
   information.
   http://www.ibm.com/support/docview.wss?uid=swg24019529 [http://www.ibm.com/support/docview.wss?uid=swg24019529]
