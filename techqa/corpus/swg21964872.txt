Title: IBM Security Bulletin: Security vulnerabilities have been identified in multiple components shipped with IBM Intelligent Operations Center (August 2015) - United States

Text:
 SECURITY BULLETIN

THIS DOCUMENT APPLIES ONLY TO THE FOLLOWING LANGUAGE VERSION(S):
 Language Independent 

SUMMARY
 Multiple components are shipped with IBM Intelligent Operations Center. Information about security vulnerabilities affecting some components has been published in security bulletins. 

VULNERABILITY DETAILS
Consult the following security bulletins for vulnerability details: 

 
IBM WebSphere Application Server: 

 * Security Bulletin: Multiple vulnerabilities in IBM® Java SDK affect WebSphere Application Server July 2015 CPU [http://www-01.ibm.com/support/docview.wss?uid=swg21962931] 
 * Security Bulletin: Multiple Security Vulnerabilities fixed in IBM WebSphere Application Server 8.0.0.11 [http://www.ibm.com/support/docview.wss?uid=swg21963275]


Java:  * Security Bulletin: Multiple vulnerabilities in current releases of the IBM® SDK, Java™ Technology Edition [http://www-01.ibm.com/support/docview.wss?uid=swg21962302]


IBM Tivoli Directory Server:  * Security Bulletin: Multiple Vulnerabilities fixed in IBM Security Directory Server [http://www-01.ibm.com/support/docview.wss?uid=swg21960659] 
 * Security Bulletin: Vulnerabilities in OpenSSL affect IBM Security Directory Server (CVE-2015-1788) [http://www-01.ibm.com/support/docview.wss?uid=swg21961111] 
 * Security Bulletin: Vulnerability in RC4 stream cipher affects IBM Tivoli/Security Directory Server (CVE-2015-2808) [http://www-01.ibm.com/support/docview.wss?uid=swg21883604]


IBM WebSphere MQ:  * Security Bulletin: Multiple vulnerabilities in IBM Java SDK affect IBM WebSphere MQ (CVE-2014-4263, CVE-2014-4244, CVE-2014-3068) [http://www-01.ibm.com/support/docview.wss?uid=swg21694094] 
 * Security Bulletin: IBM WebSphere MQ XR WebSockets listener is vulnerable to reflected cross-site scripting (CVE-2015-0176) [http://www-01.ibm.com/support/docview.wss?uid=swg21699549] 
 * Security Bulletin: Queue manager repository manager integrity/abend caused by multiple transmit queue records (CVE-2015-0189) [http://www-01.ibm.com/support/docview.wss?uid=swg21883457] 
 * Security Bulletin: IBM WebSphere MQ potential denial of service vulnerability in command server (CVE-2014-4771) [http://www-01.ibm.com/support/docview.wss?uid=swg21696120]


IBM SPSS Modeler:  * Security Bulletin: Multiple vulnerabilities in IBM Java SDK affect IBM SPSS Modeler (CVE-2015-4000, CVE-2015-0478, CVE-2015-0488) [http://www-01.ibm.com/support/docview.wss?uid=swg21960194] 

IBM DB2:  * DB2 Version 10.1 Fix Pack 5 [http://www-01.ibm.com/support/docview.wss?uid=swg21610582#5]

REMEDIATION/FIXES
Download and install IBM Intelligent Operations Center version 1.6.0.3 multiple security fixes (August 2015) [http://www-01.ibm.com/support/docview.wss?uid=swg24040526]. 
Security fixes are not cumulative. You must install all earlier security fixes in addition to the current fix.
