DEL2; remove components deployment engine DE registry TECHNOTE (FAQ)

QUESTION
 How to remove installed components from Deployment Engine registry 

ANSWER
To remove an installed component from DE registry use below steps. Please take a backup of DE registry before making any changes. See below technote for details:

http://www-01.ibm.com/support/docview.wss?uid=swg21395223 [http://www-01.ibm.com/support/docview.wss?uid=swg21395223]


1. If it is a base package, then following command can be run to perform uninstall of base package. You can use de_lsrootiu.sh command to get discriminant, UUID and version (on Windows platform use de_lsrootiu.bat and manageIU.bat script). These scripts can be found in <DE-HOME>/bin dir.
