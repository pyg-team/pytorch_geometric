PROBLEM SUMMARY
 *  ****************************************************************
   * USERS AFFECTED: Users of WebSphere default messaing, who     *
   *                 have promoted a server with installed        *
   *                 applications into a cluster.                 *
   ****************************************************************
   * PROBLEM DESCRIPTION: CWSIV0773E exceptions are thrown        *
   *                      indicating that the messaging engine    *
   *                      is not started.                         *
   ****************************************************************
   * RECOMMENDATION:                                              *
   ****************************************************************
   During messaging engine configuration reload (message
   CWSID0019I), a javax.resource.ResourceException is thrown
   indicating that the messaging engine is not started. After
   this error has occurred, the messaging engines starts
   successfully.  The error was due to the method that implements
   the reload expecting the messaging engine to be running, when
   this is not always the case.
   
   
    
   
   

PROBLEM CONCLUSION
 *  The method that handles the reload has been updated to handle
   the case where the messaging engine is not running.
