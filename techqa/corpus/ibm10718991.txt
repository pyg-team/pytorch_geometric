Title: IBM Security Bulletin: IBM BladeCenter Advanced Management Module (AMM) is affected by a vulnerability in GNU C Library (CVE-2017-12133) - United States

Text:
 SECURITY BULLETIN

SUMMARY
IBM BladeCenter Advanced Management Module (AMM) has addressed the following vulnerability in GNU C Library.

VULNERABILITY DETAILS
CVEID: CVE-2017-12133 [http://cve.mitre.org/cgi-bin/cvename.cgi?name=CVE-2017-12133]
DESCRIPTION: GNU C Library (aka glibc or libc6) could allow a remote attacker to conduct spoofing attacks, caused by a flaw in the DNS stub resolver. An attacker could exploit this vulnerability to perform off-path DNS spoofing attacks.
CVSS Base Score: 5.3
CVSS Temporal Score: See https://exchange.xforce.ibmcloud.com/vulnerabilities/131622 [https://exchange.xforce.ibmcloud.com/vulnerabilities/131622] for the current score
CVSS Environmental Score*: Undefined
CVSS Vector: (CVSS:3.0/AV:N/AC:L/PR:N/UI:N/S:U/C:N/I:L/A:N)
