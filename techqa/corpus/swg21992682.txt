 * Software requirements:
   The minimum hardware configuration for Windows workstations must be compatible with that stated by Microsoft for the respective operating systems. The following platforms are supported for this release of the IBM Spectrum Protect for Workstations Client:
   Microsoft Windows 8.1 (x86-64)
   Microsoft Windows 10 (x86-64)
   Microsoft Windows Server 2012 (x64)
   Microsoft Windows Server 2012 R2 (x64)
   Microsoft Windows Server 2016 (x64) 
   
 * Supported Browsers
   The IBM Spectrum Protect for Workstations Client GUI supports the following browsers:
   Microsoft/Internet Explorer V11
   Firefox/Mozilla V45, V47
   Google Chrome V52, V53
