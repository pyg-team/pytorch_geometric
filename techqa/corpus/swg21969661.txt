Title: IBM Collecting Data: Read first for OMNIbus 8.1 Probe Integrations on Windows - United States

Text:
Probe; probes; MustGather; OMNIbus; 8.1 TECHNOTE (FAQ)

QUESTION
 Read first : On OMNIbus 8.1, what information are required for troubleshooting probe problems on Windows? 

ANSWER
Gather Netcool/OMNIbus version and the installed software on OMNIbus 8.1:
1. Output of following commands: 

[/support/docview.wss?uid=swg21969661&amp;aid=1]

imcl.exe listInstalledPackages -features -long

2. Send a copy of following file: 

[/support/docview.wss?uid=swg21969661&amp;aid=2]

install_files\InstallationManager\configuration\config.ini 


Gather Probe Integrations information:


 * MustGather details for probe integrations on Windows [http://www-01.ibm.com/support/docview.wss?uid=swg21643313]


Please note: use of the listIU.cmd command to gather Netcool/OMNIbus version is for OMNIbus prior to 8.1. For 8.1 use the imcl.exe command from the first section of this technote (Gathering Netcool/OMNIbus version and the installed software on OMNIbus 8.1).
