Title: IBM Cannot open FileNet Image Services Resource Adapter viewer after Java 7 update25 upgrade? - United States

Text:
ISRA; JRE; Image Service Resource Adaptor TECHNOTE (TROUBLESHOOTING)

PROBLEM(ABSTRACT)
 Cannot open FileNet Image Services Resource Adapter (ISRA) viewer after Java 7 update25 upgrade? 

SYMPTOM
When I opening ISRA viewer, web browser returns 'Application Blocked for Security' message box: 
Failed to validate certificate. The application will not be executed.


CAUSE
New Java Update security requirement blocks ISRA viewer please refer to Oracle technotes:
http://docs.oracle.com/javase/7/docs/technotes/guides/jweb/no_redeploy.html [http://docs.oracle.com/javase/7/docs/technotes/guides/jweb/no_redeploy.html]
http://www.java.com/en/download/help/revocation_options.xml [http://www.java.com/en/download/help/revocation_options.xml]

RESOLVING THE PROBLEM
You can upgrade Java 7 update 40 and ISRA viewer (Daeja) version v4.0.48 to solve this problem.
