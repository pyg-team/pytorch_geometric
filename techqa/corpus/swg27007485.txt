ABSTRACT
 This page provides links to the recommended maintenance packages that are available for IBM WebSphere Enterprise Service Bus. 

CONTENT
Software maintenance updates for WebSphere Enterprise Service Bus are supplied in the form of cumulative fix bundled packages, called Fix Packs or Refresh Packs. These are summarized in the tables below, organized by release.

The packages are cumulative, so that each new package has all the fixes contained within the previous package, in addition to the new fixes. Therefore, you should always install the latest package to bring your installation up to the specified version.

A refresh pack will take a previous version of the product at the same version and release level, and upgrade it to the named modification level.
