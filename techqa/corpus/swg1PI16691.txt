APAR STATUS
 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  With IE9 or IE10, the 'Insert Link' button and also the 'Insert
   Link to Remote Document' button does not work when
   editing a rich text element using inplace editing.
   
   The following Javascript error will be seen:
   SCRIPT438: Object doesn't support property or method
   'createRange' File: RTEFieldActions.js
   
   The issue affects only 8001 CF09, 8001 CF10 and 8001 CF11 and is
   specific to inplace editing only.
   
   
    
   
   

LOCAL FIX
 *  - use IE8 compatibility mode
   - use a different browser
   - use the normal inline editing
   - or use the standing Authoring portlet to edit the rich text
