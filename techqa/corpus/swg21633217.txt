Title: IBM Process Designer might hang and close automatically when editing the Common section in IBM Business Process Manager (BPM) - United States

Text:
 TECHNOTE (TROUBLESHOOTING)

PROBLEM(ABSTRACT)
 When you have Microsoft Internet Explorer 6.0 or 7.0 installed in your environment, IBMProcess Designer hangs and closes automatically when you edit the Common section.

Note: The issue occurs even if Microsoft Internet Explorer V6.0 or 7.0 is not the default web browser in your environment.


CAUSE
Microsoft Internet Explorer V6.0 or 7.0 is not the official supported version for IBM Business Process Manager Version 8.0.1. IBM Process Designer has an underlying dependency on the Internet Explorer browser regardless of the system default browser.


RESOLVING THE PROBLEM
To resolve this issue, upgrade the web browser to Microsoft Internet Explorer V8.0 or 9.0, which are the official supported versions for IBM Business Process Manager Version 8.0.1.
