Text:
gethostbyname(); gethostbyname security TECHNOTE (FAQ)

QUESTION
 Is Tivoli Netcool OMNIbus impacted by the GNU C Library (glibc) Vulnerability (CVE-2015-0235)? 

CAUSE
 Red Hat Product security has reported a vulnerability in the glibc library whereby the gethostbyname() function calls can be exploited.

ANSWER
The GNU C Library (glibc) vulnerability , also called "GHOST" vulnerability, will affect on the next components of IBM Tivoli Netcool OMNIbus products. 

- OMNIbus 
- WebGUI

To resolve this issue, it would be necessary for customers to patch their glibc to the latest version per the operating system guidance. 

Additional information about CVE-2015-0235, and steps necessary for remediation, can be found at this URL:

https://access.redhat.com/articles/1332213 [https://access.redhat.com/articles/1332213]
