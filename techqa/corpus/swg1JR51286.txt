APAR STATUS
 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  By using IBM Business Process Manager (BPM) and IBM WebSphere
   Lombardi Edition, you can import and export process applications
    and toolkits. Although this functionality is available only to
   authorized users, the actual server side code accepts requests
   from lower privileged users.
   
   IBM BPM Advanced
   IBM BPM Standard
   IBM BPM Express
   WebSphere Lombardi Edition
   
   
    
   
   

LOCAL FIX

PROBLEM SUMMARY
 *  Authenticated users can invoke project actions, such as import,
   export or generate report with insufficient authorization
   checks.
   
   
    
   
   

PROBLEM CONCLUSION
 *  A fix is available for IBM BPM and WebSphere Lombardi Edition
   that corrects the authorization checks.
   
   On Fix Central (http://www.ibm.com/support/fixcentral), search
   for JR51286:
   
   1.Select IBM Business Process Manager with your edition from the
   product selector, the installed version to the fix pack level,
   and your platform, and then click Continue.
   2.Select APAR or SPR, enter JR51286, and click Continue.
   
   When you download fix packages, ensure that you also download
   the readme file for each fix. Review each readme file for
   additional installation instructions and information about the
   fix.
