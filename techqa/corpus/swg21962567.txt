Title: IBM Security Bulletin: Fix Available for Denial of Service Vulnerability in IBM WebSphere Portal (CVE-2015-1943) - United States

Text:
 SECURITY BULLETIN

SUMMARY
 A fix is available for a denial of service vulnerability in IBM WebSphere Portal (CVE-2015-1943). 

VULNERABILITY DETAILS
CVEID: CVE-2015-1943 [http://cve.mitre.org/cgi-bin/cvename.cgi?name=CVE-2015-1943]
DESCRIPTION: IBM WebSphere Portal is vulnerable to a denial of service. By sending a malicious request, a remote attacker could exploit this vulnerability to cause the consumption of all CPU or memory resources to cause a denial of service.
CVSS Base Score: 5
CVSS Temporal Score: See https://exchange.xforce.ibmcloud.com/vulnerabilities/103138 [https://exchange.xforce.ibmcloud.com/vulnerabilities/103138] for the current score
CVSS Environmental Score*: Undefined
CVSS Vector: (AV:N/AC:L/Au:N/C:N/I:N/A:P) 

AFFECTED PRODUCTS AND VERSIONS
IBM WebSphere Portal 8.5
IBM WebSphere Portal 8.0
IBM WebSphere Portal 7
IBM WebSphere Portal 6.1

REMEDIATION/FIXES
The recommended solution is to apply PI39617 [http://www.ibm.com/eserver/support/fixes/fixcentral/swg/quickorder?productid=WebSphere%20Portal&brandid=5&apar=PI39617] as soon as practical.
Fix: Apply an Interim Fix or a Cumulative Fix containing PI39617 [http://www.ibm.com/eserver/support/fixes/fixcentral/swg/quickorder?productid=WebSphere%20Portal&brandid=5&apar=PI39617].

For 8.5.0 

 * Upgrade to Cumulative Fix 06 (CF06).
   (Combined Cumulative Fixes for WebSphere Portal 8.5.0.0: http://www-01.ibm.com/support/docview.wss?uid=swg24037786 [http://www-01.ibm.com/support/docview.wss?uid=swg24037786])


For 8.0.0 through 8.0.0.1  * Upgrade to Fix Pack 8.0.0.1 with Cumulative Fix 17 (CF17).
   (Combined Cumulative Fixes for WebSphere Portal 8.0.0.1: http://www-01.ibm.com/support/docview.wss?uid=swg24034497 [http://www-01.ibm.com/support/docview.wss?uid=swg24034497])


For 7.0.0 through 7.0.0.2  * Upgrade to Fix Pack 7.0.0.2 with Cumulative Fix 29 (CF29) and then apply the Interim Fix PI39617 [http://www.ibm.com/eserver/support/fixes/fixcentral/swg/quickorder?productid=WebSphere%20Portal&brandid=5&apar=PI39617].
   (Combined Cumulative fixes for WebSphere Portal 7.0.0.2: http://www.ibm.com/support/docview.wss?uid=swg24029452 [http://www.ibm.com/support/docview.wss?uid=swg24029452])


For 6.1.5.0 through 6.1.5.3  * Upgrade to Fix Pack 6.1.5.3 with Cumulative Fix 27 (CF27) and then apply the Interim Fix PI47714 [http://www.ibm.com/eserver/support/fixes/fixcentral/swg/quickorder?productid=WebSphere%20Portal&brandid=5&apar=PI47714].
   (Cumulative fixes for WebSphere Portal 6.1.5.3: http://www-01.ibm.com/support/docview.wss?uid=swg24023835 [http://www-01.ibm.com/support/docview.wss?uid=swg24023835])


For 6.1.0.0 through 6.1.0.6  * Upgrade to Fix Pack 6.1.0.6 with Cumulative Fix 27 (CF27) and then apply the Interim Fix [http://www.ibm.com/eserver/support/fixes/fixcentral/swg/quickorder?productid=WebSphere%20Portal&brandid=5&apar=PI39617] PI47714 [http://www.ibm.com/eserver/support/fixes/fixcentral/swg/quickorder?productid=WebSphere%20Portal&brandid=5&apar=PI47714].
   (Cumulative fixes for WebSphere Portal 6.1.0.6: http://www-01.ibm.com/support/docview.wss?uid=swg24023835 [http://www-01.ibm.com/support/docview.wss?uid=swg24023835])
