ABSTRACT
 This document describes how to download and install IBM Netcool Network Management V9.2 using the Passport Advantage website. 

DOWNLOAD DESCRIPTION
 

Netcool Network Management provides network discovery and event and network management, including identification and resolution of root-causes and system, network, and service failures, and configuration management capabilities for network devices. Inventory and connectivity data is integrated with IBM Tivoli Netcool/OMNIbus event data. Network management staff can visualize and understand the layout of complex networks and the impact of network events and failures, and work with events to locate, diagnose, and solve network problems in real time.

Netcool Network Management consists of the following products: IBM Tivoli Netcool/OMNIbus V7.4, IBM Tivoli Network Manager IP Edition V3.9, and IBM Tivoli Netcool Configuration Manager V6.4.

More information

The libraries of publications for the products in Netcool Network Management are available from this website: IBM Tivoli Network Management Information Center [http://www.ibm.com/support/knowledgecenter/SSNFET].

Downloading and assembling the product

 1. Navigate your browser to the Find Downloads and Media [http://www.ibm.com/software/howtobuy/passportadvantage/paocustomer/sdma/SDMA?PO=DOWNLOAD_FIND_BY_PART_NUMBER] page on the IBM Passport Advantage Online website. You must log on to continue.
 2. Select the Find by part number search option under Download finder options. 
 3. Search for each downloadable image by part number (parts are listed and described by operating system in the tabbed section of this document).
 4. Download all of the parts that you need.
 5. Unpack each of the parts into a single temporary directory on your system. If you want to install Tivoli Netcool/OMNIbus as part of the Network Manager installation process, you must download the Tivoli Netcool/OMNIbus installation package to a specific location and point the Network Manager installer to that location. Do not extract or rename the Tivoli Netcool/OMNIbus installation package.
 6. Open the Quick Start image and follow the instructions in the Quick Start Guide to install the base product.

Netcool Configuration Manager V6.4 is not supported on Linux zSeries operating systems. Tivoli Netcool/OMNIbus V7.4 and Network Manager IP Edition V3.9 are supported on Windows operating systems.
