Title: IBM The Property Definition window in Administrative Console for Content Engine (ACCE) does not show the same properties as in FileNet Enterprise Manager (FEM). - United States

Text:
 TECHNOTE (FAQ)

QUESTION
 The Property Definition window ((from Document Class > Property Definitions tab > [property link])) in Administrative Console for Content Engine (ACCE) does not show the same properties as in FileNet Enterprise Manager (FEM). How do I view the properties of a Property Definition in ACCE? 

ANSWER
Property Definition properties can be viewed as follows:
1. Using ACCE, open the Class Definition
2. Click Properties tab
3. Open Property Definitions drop down and select property to modify.
4. ACCE will open the property in separate tab listing all the properties.
