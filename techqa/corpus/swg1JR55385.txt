APAR STATUS
 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  When you use the failed event manager to query failed events,
   a folder is created under the wstemp deployment manager folder
   after each query. These folders are not deleted afterward.

 *  A fix is available for IBM BPM V8.5.5.0 that ensures the
   temporary folder is deleted after the query completes.

   for JR55385:

   the product selector, the installed version to the fix pack
   level, and your platform, and then click Continue.
   2.Select APAR or SPR, enter JR55385, and click Continue.
