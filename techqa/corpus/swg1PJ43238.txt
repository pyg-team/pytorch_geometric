APAR STATUS
 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  The Finish Button is disabled when running the "Create Files
   that List Properties and Document Types" wizard from IBM Case
   Manager configuration tool.
   
   Steps to reproduce:
   1. Launch Case Manager configuration tool
   2. Click on Tools -> List Property Templates and document types
   3. On the wizard page "Create Files that List Properties and
   Document Types", enter data click "Back" and then "Next"
   4. Choose the "Target Object Store", the Finish Button is
   disabled despite all the fields being filled up.
   
   
    
   
   

LOCAL FIX
 *  Delete or append additional characters in the "Output Directory"
   and/or "File name prefix" field
   
   
    
   
   

PROBLEM SUMMARY
 *  ****************************************************************
   * USERS AFFECTED:                                              *
   * Users using the create properties and document types option  *
   * from the IBM Case Manager configuration tool                 *
   ****************************************************************
   * PROBLEM DESCRIPTION:                                         *
   * See Error Description                                        *
   ****************************************************************
   * RECOMMENDATION:                                              *
   * It is recommended to apply the 5.2.1.2-ICM-FP002 or later GA *
   * patch.                                                       *
   ****************************************************************
   
   
    
   
   

PROBLEM CONCLUSION
 *  This issue is resolved in the 5.2.1.2-ICM-FP002 GA patch.  The
   code has been changed to enable the "Finish" button when all of
   the required fields are populated.
