Title: IBM B2B Advanced Communication User Interface Unexpectedly Timing Out - United States

Text:
timeout; b2bac TECHNOTE (TROUBLESHOOTING)

PROBLEM(ABSTRACT)
 While working in the IBM B2B Advanced Communications (B2BAC) user interface (UI) random timeouts are experienced 

SYMPTOM
When the user is kicked out of the UI they are returned to the login screen. The timeouts are random, sometimes within a few minutes/seconds regardless is the user is idle or actively working

As of release 1.0.0.5 there is no user adjustable timeout setting

The informational member log may contain timeout exceptions such as:
