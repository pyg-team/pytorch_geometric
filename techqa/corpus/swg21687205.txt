Title: IBM Unable to open MS office documents in Daeja Viewer Pro - United States

Text:
Daeja viewer; Microsoft office; MS office; Daeja Viewer Pro; content_redir.properties TECHNOTE (TROUBLESHOOTING)

PROBLEM(ABSTRACT)
 Unable to open the Microsoft Office documents in Daeja Viewer Pro 

SYMPTOM
An error occurs when the content_redir.properties properties file does not have the content to open MS office files.


CAUSE
If you are going to be using ViewONE Pro to view and annotate PDF and other document types, then you need to open the content_redir.properties properties file and add additional lines for each new file type that you wish to view in the viewer.

ENVIRONMENT
There are no environment issues that cause this error.

DIAGNOSING THE PROBLEM
Install a specific version of Daeja viewer and try to open any MS office documents. An error occurs at that time.

RESOLVING THE PROBLEM
Add the lines in the content_redir.properties properties file.

Example of a modified .properties file.

# MS Office 2007 format
#
application/vnd.openxmlformats-officedocument.wordprocessingml.document=/WcmJavaViewer.jsp?id={OBJECT_ID}&{JSP_QUERY_STRING}
#
application/vnd.openxmlformats-officedocument.spreadsheetml.sheet=/WcmJavaViewer.jsp?id={OBJECT_ID}&{JSP_QUERY_STRING}
#
application/vnd.openxmlformats-officedocument.presentationml.presentation=/WcmJavaViewer.jsp?id={OBJECT_ID}&{JSP_QUERY_STRING}
