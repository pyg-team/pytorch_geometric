PROBLEM CONCLUSION
 *  THIS PROBLEM WILL BE FIXED ON CATIA VERSION V5R21 GA level.
   Additional Closure Information:
   INCIDENT DIAGNOSIS:
   A parameter named '.' (dot) was accepted but Knowledge parser
   does not support it. Special character dot is not enclosed
   within single quotes while renaming a user parameter.
   .
   CORRECTION EXPLANATION:
   A parameter named '.' (dot) is now enclosed within single
   quotes.
