Title: IBM IV33539: MULTIBYTES CHARS WAS CUT OFF INCORRECTLY IN MESSAGE TEXT ATTRIB. - United States

APAR STATUS
 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  When a multiple bytes character like Japanese character is
   overlaid before and behind 78th bytes in Message Text
   attribute of System Log attribute group, the character is cut
   off at the middle of the character incorrectly.  Then, the
   character is garbled.
   
   As the result, the event isn't formed correctly in Netcool EIF
   probe and it cannot handle the event.
   
   
   ENVIRONMENT:
   SAP Agent 6.2 IF15 / AIX 7.1 / SAP 7.0.2
   ITM 6.2.3 FP1
