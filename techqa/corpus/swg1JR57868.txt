LOCAL FIX
 *  Remove leading underscores from the alias of every searchable
   variable in all of your business process definitions (BPDs) and
   delete every process instance that was created using these BPDs
   regardless of its state.
   
   
    
   
   

PROBLEM SUMMARY
 *  When you apply the search optimization, a table is generated
   based on the data in the LSW_BPD and LSW_BPD_INSTANCE_VARIABLES
   tables that holds a column for each variable that is marked
   'Visible in Process Portal' in IBM Process Designer. If the
   alias of one of these variables starts with an underscore, the
   table creation SQL statement fails.
   
   
    
   
   

PROBLEM CONCLUSION
 *  A fix will be available in IBM BPM V8.5.7 cumulative fix 2017.06
    that removes leading underscores from variable aliases before
   creating the SQL statement.
