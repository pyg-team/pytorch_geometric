ERROR DESCRIPTION
 *  Environment:
   TAMIT 7.2
   
   Problem Description:
   With TAMIT 7.2 a new feature moving assets between GL accounts
   is provided. But when we are moving asset to a new site of
   different organization, GL Account does not get updated when
   asset is moved from one site to a different site in different
   orgainzation. Asset.GLAccount does not get updated correctly
   neither.
   
   Asset.GLAccount gets updated correctly before as well as after
   hotfix when asset is moved WITHIN same site
   Asset.GLAccount does not get updated correctly neither before
   the hotfix nor after the hot fix when asset is moved to
   DIFFERENT Site
   
   
    
   
   

LOCAL FIX
 *  N/A
   
   
    
   
   

PROBLEM SUMMARY
 *  ****************************************************************
   * USERS AFFECTED: Those trying to move assets using the move   *
   *                 asset dialog                                 *
   ****************************************************************
   * PROBLEM DESCRIPTION: When trying to move an asset, the GL    *
   *                      Account does not get moved properly.    *
   *                      You can fill in a new one, but the      *
   *                      moved asset will not have the changed   *
   *                      value.  Also, when switching sites in   *
   *                      the dialog, the GL Account field was    *
   *                      not being blanked.                      *
   *                                                              *
   ****************************************************************
   * RECOMMENDATION:                                              *
   *                                                              *
   *                                                              *
   *                                                              *
   ****************************************************************
   When trying to move an asset, the GL Account does not get moved
   properly.
   
   
    
   
   

PROBLEM CONCLUSION
 *  A lookup that was being done on siteid should have been using
   newsite.  Also, code was added to blank the 'to GLAccount'
   field when choosing a new location.
