APAR STATUS
 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  CE Health Page shows object store as available (healthy) when
   the database server is down.
   
   Steps to Reproduce:
   1  Hit the CPE Health Page and confirmed that the status for the
   GCD and object stores are correct
   2  Shutdown the database server
   3  Hit the CPE Health Page again and select the GCD and confirm
   that it display the correct message:
         *  The Global Configuration database is currently not
   available.  A file-based, read-only, image of the Global
   Configuration database will be used
   4  However, when selecting the object stores, no errors or
   messages are displayed.  The CPE Health Page displays the object
   stores as "healthy"
