APAR STATUS
 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  PMDESC and PMVALUE attributes of Database
   Summary, Process Summary, Problem Summary Workspaces
   were omitted previously. When these attribute groups
   were enabled for Historical Data Collection, PMDESC and
   PMVALUE attributes used to have junk values in the
   warehouse database.
   
   Note: problem is similar to IY92671 in SQL Server Agent.
   
   Approved by: SHM
   
   
    
   
   

LOCAL FIX

PROBLEM SUMMARY
 *  The PMDESC and PMVALUE attributes of Database Summary, Process
   Summary, Problem Summary Workspaces were omitted previously.
   When these attribute groups were enabled for Historical Data
   Collection, the PMDESC and PMVALUE attributes used to have junk
   values in the warehouse database.
