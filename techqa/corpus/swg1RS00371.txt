Title: IBM RS00371: BOM UPDATE SHOULD DISPLAY AN ERROR PANEL WHEN AN EXCEPTION IS THROWN DURING XOM CLASS LOADING - United States

APAR STATUS
 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  BOM update does not report and display error in display panel
   when an exception is caught during the XOM class loading.
   
   
    
   
   

LOCAL FIX
 *  If the issue is related to missing dependent java classes due to
   XOM refactoring, the missing classes have to be re-packaged
   back to the XOM jar or packaged the missing classes in a jar and
   define as required libraries to the XOM. One can do that in Rule
   project preference (at Java XOM section)
   
   
    
   
   

PROBLEM SUMMARY
 *  BOM Update should display an error panel when an exception is
   thrown during XOM class loading
   
   
    
   
   

PROBLEM CONCLUSION
 *  Source code has been modified to fix this problem.
