ERROR DESCRIPTION
 *  The copy operation removes the content of RTE and HTML
   components After applying 6.1.0.3 + CF29
   
   Steps to reproduce the issue.
   
   (1) Create two sites and other artifacts.
   (2) Copy a content from one sitearea which has some RTE element

   (4) Similar is for HTML element.

PROBLEM SUMMARY
 *  The copy operation removes the content of RTE and HTML
   components After applying 6.1.0.3 + CF29

   (4) Similar is for HTML element.
