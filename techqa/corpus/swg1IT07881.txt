APAR STATUS
 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  PureApplication System  V2.0.0.1
   -
   During a PureApplication System health check for an upgrade
   the "Run System Diagnostics" shows the following as errors:
   [Date time UTC] Rackmmmm-mod serial
   Chassis *  CMMs:
   Errors were found during verification.
   [Date time UTC] Rackmmmm-mod serial
   Chass * CMM *
   (mm[*]): The productName is Chassis Mgt Module, instead of
   Chassis Management Module.
   
   
    
   
   

LOCAL FIX

PROBLEM SUMMARY
 *  The module verification code was checking for a specific module
   name as well as a required version. The name between fix levels
   can change and make the health check fail.
   
   
    
   
   

PROBLEM CONCLUSION
 *  The verification code has been updated to only check the version
   level in PureApplication System version 2.1.0.0 interim fix 1.
