Title: IBM LO72396: "DOCUMENT.GETITEMVALUEDATETIMEARRAY" METHOD RETURNS VECTOR CONTAINING A SINGLE NULL VALUE - United States

ERROR DESCRIPTION
 *  The "Document.getItemValueDateTimeArray" method returns a
   vector containing a single NULL value.
   
   The "Document.getItemValue" method when used on the same field
   returns a vector containing two elements, both are type
   "lotus.domino.local.DateTime".
   
   
   STEPS TO REPRODUCE:
   1.  Download the attached file.
   2.  Place the file in the Notes or Domino data directory.
   3.  Open the database in the Notes client.
   4.  Open the Java Debug Console.
       >  Tools :: Show Java Debug Console
   5.  Run the agent "Test".
   
   
   SAMPLE AGENT OUTPUT:
   Document.getItemValueDateTimeArray method:
   element @ 0:  null
   
   Document.getItemValue method:
   type of lotus.domino.local.DateTime
   element @ 0:  10/23/2012 12:20:35 PM CDT
   type of lotus.domino.local.DateTime
   element @ 1:  10/30/2012 12:20:35 PM CDT
   
   
    
   
   

LOCAL FIX
 *  Use the "Document.getItemValue" method instead.
