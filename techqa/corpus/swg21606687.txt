Title: IBM Which Concurrent License Manager installation do I need to download? - United States

Text:
 TECHNOTE (FAQ)

QUESTION
 The download page for Modeler's Sentinel License manager has three different downloads. What contents are in each download? 

ANSWER
1. IBM SPSS Concurrent Licensing Tools 9.5 - Hardware Key Microsoft Windows English (CI8Z6EN)
This is the driver required to use Dongles. Some licenses, not in North America, require a dongle. If you are unsure if you have a dongle license, please contact your sales representative.

2. IBM SPSS Concurrent Licensing Tools 9.5 - License Manager Microsoft Windows English (CI8Z7EN)
This is the Sentinel License Manager which is required when using network(concurrent) Licenses. 

To install: Right click on downloaded spss_clt_9.5_licmgr_win.exe and run as administrator. InstallAnywhere will prepare to install, will start the IBM SPSS Concurrent Licensing Tools installation and finishes with a request to enter the Concurrent authorization code. 
Note: The default path for the Sentinel License Manager has changed, its default path is: \Program Files\IBM\SPSS\Concurrent Licensing Tools. 
3. IBM SPSS Concurrent Licensing Tools 9.5 - License Manager Administrator Microsoft Windows English (CI8Z8EN) 
This is just the Wlmadmin tool which is a GUI tool used to administer Sentinel License Manager. This is typically used by customers who install the Sentinel License Manager on UNIX and want to use the Windows GUI Administer Tool.
