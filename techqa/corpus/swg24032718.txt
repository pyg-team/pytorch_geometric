 This document describes how to download and assemble the Sterling Order Management version 9.2 components using the Passport Advantage Web site. 

DOWNLOAD DESCRIPTION


IBM Sterling Order Management requires a tiered environment, using a combination of application server, Web server, and database server software. You install IBM Sterling Order Management in a test environment, configure it for your business needs, and deploy to a production environment. Because the product includes several solutions, planning the installation involves considering your deployment and security architecture, your performance needs, and any requirements for high availability.

The product includes base software and other components. Installing the base software is a required first step. Afterwards, install and configure other components as provided by your licensing agreement. 

 * Base software:  * IBM Sterling Selling and Fulfillment Foundation 
    * IBM Sterling Distributed Order Management 
    * IBM Sterling Global Inventory Visibility 
    * IBM Sterling Reverse Logistics 
    * IBM Sterling Logistics Management 
    * IBM Sterling Supply Collaboration (requires additional licensing) 
    * IBM Sterling Delivery and Service Scheduling (requires additional licensing) 
    * IBM Sterling Business Center 
   
   
 * Components included with all licensing agreements:  * IBM Sterling Order Management: Selling and Fulfillment Suite: Business Intelligence 
    * IBM Sterling Order Management: Selling and Fulfillment Suite: Business Intelligence Operational Reports 
   
   
 * Components included separately based on licensing agreement:  * IBM Sterling Call Center 
    * IBM Sterling Store Inventory Management 
    * IBM Sterling Store 
    * IBM Sterling Mobile Framework for Internal Users (for use with Store Associate, Field Sales, and Order Management Administrator) 
   
   

The general process outlined below describes a typical installation for IBM Sterling Order Management. 

 * Evaluate your hardware and system configuration. 
 * Create installation, testing and deployment plans. 
 * Install one of these databases:  * DB2 
    * Oracle 
   
   
 * Install and configure one of the following application servers:  * WebSphere 
    * JBoss 
    * Oracle WebLogic 
   
   
 * Install and configure software for IBM Sterling Order Management. Installing the base software is a required first step. Afterwards, install and configure other components as provided by your licensing agreement.

Downloading and assembling the product

 1. Navigate your browser to the Find Downloads and Media [http://www.ibm.com/software/howtobuy/passportadvantage/paocustomer/sdma/SDMA?PO=DOWNLOAD_FIND_BY_PART_NUMBER] page on the IBM Passport Advantage Online Web site. You must log on to continue. 
 2. Select the Find by part number search option under Download finder options. 
 3. Search for each downloadable image by part number (parts are listed and described by platform, below) 
 4. Download all of the parts you need. 
 5. Unpack each of the parts into a single, temporary directory on your system. 
 6. Open the Quick Start image and follow the instructions in the Quick Start Guide to install the product.
