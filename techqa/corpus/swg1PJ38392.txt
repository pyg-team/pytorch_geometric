   1 On FEM, add Audit Log settings for an event "Checkout" for an

LOCAL FIX

PROBLEM SUMMARY
 *  ****************************************************************
   * USERS AFFECTED:                                              *
   * non-English users of the History page                        *
   ****************************************************************
   * PROBLEM DESCRIPTION:                                         *
   * On the History page, when the browser is set to use a        *
   * non-English language, performing a search for events with    *
   * criteria using the "Initiated By" field results in an        *
   * unexpected exception error when clicking on the Search       *
   * button.                                                      *
   ****************************************************************
   * RECOMMENDATION:                                              *
   * Upgrade to Workplace XT fix pack 1.1.4.9.                    *
   ****************************************************************
   
   
    
   
   

PROBLEM CONCLUSION
 *  In Workplace XT fix pack 1.1.4.9, searching for events by the
   
   
   "Initiated By" field has been fixed when the browser language
   
   
   is non-English.
