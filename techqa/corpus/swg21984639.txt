Title: IBM 【セキュリティ情報】 IBM Integration Bus および WebSphere Message BrokerへのIBM Java Runtime の複数の脆弱性の影響 - United States

Text:
 SECURITY BULLETIN

SUMMARY
 WebSphere Message Brokerで提供されているIBM® Runtime Environment Java™ Technology Editions 6.0.16.21(およびそれ以前の保守レベル)と、IBM Integration Busで提供されているIBM® Runtime Environment Java™ Technology Editions 7.0.9.31(およびそれ以前の保守レベル) もしくは、7.1.3.31(およびそれ以前の保守レベル)のIBM Java Runtime Environmentコンポーネントに複数のセキュリティ脆弱性が存在します。
これらの問題は、2016年4月にIBM Java SDK更新の一部として公開されました。
