Title: IBM Security Bulletin: IBM OpenPages GRC Platform Web Applications are not vulnerable to (CVE-2017-5638) - United States

Text:
 SECURITY BULLETIN

SUMMARY
 IBM OpenPages GRC Platform Web Applications are not vulnerable to the Apache Struts 2 vulnerability CVE-2017-5638 

VULNERABILITY DETAILS
IBM OpenPages GRC Platform Web Applications are NOT vulnerable to the Apache Struts 2 vulnerability (CVE-2017-5638). 
Please refer to https://cwiki.apache.org/confluence/display/WW/S2-045 [https://cwiki.apache.org/confluence/display/WW/S2-045] for more information on CVE-2017-5638.

AFFECTED PRODUCTS AND VERSIONS
IBM OpenPages versions 7.0 through 7.3

REMEDIATION/FIXES
None

WORKAROUNDS AND MITIGATIONS
None
