 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  Error when clicking on magnifying glass for an attribute of
   lookup table type in InfoSphere MDM Server for PIM.
   
   Steps to reproduce:
   * Ensure rich_search_default_view_indexed_only parameter in
   common.properties is set to true.
   
   1. Create primary spec with at least one attribute of type
   lookup table
   2. Create a catalog
   3. Click to create an item and click on magnifying glass to view
   lookup table entries.
   
   The error below appears in the UI:
