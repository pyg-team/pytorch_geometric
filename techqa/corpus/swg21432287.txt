 TECHNOTE (FAQ)

QUESTION
 What are the most current instructions for migrating to Lotus Quickr 8.5 for WebSphere Portal?

CAUSE
From time to time, the migration instructions need updating.

ANSWER
The product documentation is a wiki that can be updated and available to the public in real time. We chose to update the migration instructions in the product documentation. You need to re-read the migration section here from time to time to make sure you are using the latest instructions. 

http://www.lotus.com/ldd/lqwiki.nsf/dx/Migrating_qp85 [http://www.lotus.com/ldd/lqwiki.nsf/dx/Migrating_qp85]
