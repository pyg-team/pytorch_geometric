QUESTION
 The INDEXREC parameter is present in both the 'Database Configuration' Setting and the 'Database Manager Configuration' Setting. Which setting takes precedence ? 

ANSWER
This parameter indicates when the database manager attempts to rebuild invalid indexes, and whether or not any index build is redone during rollforward or high availability disaster recovery (HADR) log replay on the standby database. 
When the INDEXREC is set to 'SYSTEM' in the database configuration file (This setting is only valid for the database configuration setting) then the setting specified for INDEXREC in the database manager configuration file take precedence.

For any other setting of the database configuration setting takes precedence over the database manager configuration setting.
