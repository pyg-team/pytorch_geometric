Title: IBM How to clear WebSphere 6.1 temp cache - United States

Text:
Websphere; 6.1; clear; cache; appserver; temp; TPAEWEBSPHERE TECHNOTE (TROUBLESHOOTING)

PROBLEM(ABSTRACT)
 This document shows the correct directories to delete when clearing WebSphere 6.1 cache 

SYMPTOM
New changes, such as fix pack, hot fix or xml changes do not take affect after deploying new maximo.ear file.


RESOLVING THE PROBLEM
Clear the WebSphere cache to make changes take affect by performing the following. 

Add this procedure of clearing WebSphere cache every time you rebuild and redeploy the ear files.



From the file structure delete the following folders/files 
a. ibm\websphere\appserver\profiles\<app server profile>\temp\<node>\<app server> folder 
i.e. ibm\websphere\appserver\profiles\ctgappsrv01\temp\ctgnode01\mxserver 
b. ibm\websphere\appserver\profiles\<app server profile>\wstemp, delete files in this folder 
c. ibm\websphere\appserver\profiles\<app server profile>\tranlog\<cell>\<node>\<app server> folder 
i.e. ibm\websphere\appserver\profiles\ctgappsrv01\tranlog\ctgcell01\ctgnode01\mxserver 
d. ibm\websphere\appserver\profiles\<app server profile>\logs\<app server> folder 
e. ibm\websphere\appserver\profiles\<app server profile>\logs\ffdc folder Note: This path is from a standard Maximo 7 install using defaults from the guide, ctgNode01 and MXServer may be named something different if you did not use the default naming.
