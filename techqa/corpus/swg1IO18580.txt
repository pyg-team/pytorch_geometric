ERROR DESCRIPTION
 *  (FileNet P8) Searches created in IBM FileNet Workplace XT with
   hidden "Search in" criteria result in blank search results when
   run in IBM Content Navigator. The search results are empty
   because IBM Content Navigator is not able to run the search when
    all of the search in folders for the search are set to hidden.
   
   
    
   
   

LOCAL FIX
 *  The search can be run if at least one search-in folder is not
   hidden.
   
   
    
   
   

PROBLEM SUMMARY
 *  Searches created in FileNet Workplace XT with hidden "Search in"
    criteria result in blank search results when run in Content
   Navigator.
   
   
    
   
   

PROBLEM CONCLUSION
 *  Searches will run and display results as expected.
