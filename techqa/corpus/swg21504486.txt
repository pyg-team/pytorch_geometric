Title: IBM "Standard Error 76 ... Could not find a part of the path" when preview report if launch Excel first - United States

Text:
 TECHNOTE (TROUBLESHOOTING)

PROBLEM(ABSTRACT)
 User launches Excel. User clicks "Add-ins" - "Controller" - "Log on". User chooses database connection, and logs onto Controller. Afterwards, user launches the 'main' Controller program (ccr.exe). Inside the main program, user clicks "Reports - Run". User selects a report, and clicks "Preview". An error message appears.

CAUSE
Code production problem in Controller 10.1.384 and earlier.



RESOLVING THE PROBLEM
Fix: 

Upgrade to Controller 10.1.387 or later.

Workaround:
Do not launch Excel before launching the 'main' Controller program (CCR.EXE). 

 * Instead, allow Excel to be launched via the action of clicking "Reports - Run" inside Controller.
