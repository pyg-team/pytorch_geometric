Title: IBM Security Bulletin: Cognos Disclosure Management file upload bypass (CVE-2015-5014) - United States

Text:
 SECURITY BULLETIN

SUMMARY
 A specific usage scenario of IBM Cognos Disclosure Management could allow a remote attacker to obtain sensitive information. An attacker could send a specially crafted executable file to a CDM client machine by using a man-in-the-middle mechanism. 

VULNERABILITY DETAILS
CVEID: CVE-2015-5014 [http://cve.mitre.org/cgi-bin/cvename.cgi?name=CVE-2015-5014]
DESCRIPTION: IBM Cognos Disclosure Management could allow a remote attacker to obtain sensitive information and gain access to the system by tricking a user to executing an uploaded file. An attacker could send a specially-crafted executable file to a CDM client machine by using a man-in-the-middle mechanism.
CVSS Base Score: 7
CVSS Temporal Score: See https://exchange.xforce.ibmcloud.com/vulnerabilities/106427 [https://exchange.xforce.ibmcloud.com/vulnerabilities/106427] for the current score
CVSS Environmental Score*: Undefined
CVSS Vector: (CVSS:3.0/AV:L/AC:H/PR:N/UI:R/S:U/C:H/I:H/A:H) 

AFFECTED PRODUCTS AND VERSIONS
Cognos Disclosure Management 10.2.4 and previous

REMEDIATION/FIXES
Cognos Disclosure Management 10.2.4 Interim Fix 10 [http://www-01.ibm.com/support/docview.wss?uid=swg24041006]

WORKAROUNDS AND MITIGATIONS
None
