Title: IBM Security Bulletin: Multiple security vulnerabilities have been identified in WebSphere Application Server shipped with IBM Omni-Channel Marketing products suite - United States

Text:
 SECURITY BULLETIN

SUMMARY
 IBM WebSphere Application Server variants are shipped with IBM Omni-Channel Marketing products suite. Information about security vulnerabilities affecting WebSphere Application Server have been published in security bulletins. 

VULNERABILITY DETAILS
1. Potential security vulnerability in WebSphere Application Server MQ JCA Resource adapter (CVE-2016-0360) 

2. Potential security vulnerability in WebSphere Application Server Administrative Console (CVE-2017-1137) 

3. Cross-site request forgery in WebSphere Application Server (CVE-2017-1194) 

4. Privilege Escalation Vulnerability in WebSphere Application Server (CVE-2017-1151) 


For more details refer to the security bulletin listed in the Remediation/Fixes section.
