Title: IBM® Cloud Orchestrator V2.5.0.2 extends support for newer versions of IBM® PowerVC - United States

Text:
 NEWS

ABSTRACT
 IBM® Cloud Orchestrator V2.5.0.2 with IBM® Cloud Manager for OpenStack V4.3.0.6 interim fix 4 extends its support for newer versions of IBM PowerVC. 

CONTENT
IBM Cloud Orchestrator V2.5.0.2 and IBM Cloud Manager with OpenStack V4.3 now supports the following managed versions of PowerVC:


 * IBM Cloud Manager with OpenStack V4.3 GA to Fix Pack 3 supports PowerVC 1.2.3.x and 1.2.2.x 
 * IBM Cloud Manager with OpenStack V4.3 Fix Pack 4 to Fix Pack 6 interim fix 3 supports PowerVC 1.3.0.x and 1.2.3.x 
 * IBM Cloud Manager with OpenStack V4.3 Fix Pack 6 interim fix 4 and later supports PowerVC 1.3.2.x, 1.3.1.x, 1.3.0.x, and 1.2.3.x


For the IBM Cloud Manager with OpenStack V4.3 interim fixes, go to IBM Fix Central [https://www-945.ibm.com/support/fixcentral/swg/selectFixes?] and download interim fix 4 for fix pack 6 - 4.3.0.6-IBM-CMWO-IF004.
