LOCAL FIX
 *  Edit the xmlaccess export document which failed to import.
   
   Locate the following resource-type section with the name
   "UDDI_REGISTRY".
   
   <resource-type ...  name="UDDI_REGISTRY" >
   
   Remove this resource-type section and save the document.
   Retry to import based on the corrected xml document.
