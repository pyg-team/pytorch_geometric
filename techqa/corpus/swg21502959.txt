QUESTION
 What is the definition of Users Roles that can be created on the Managing Server? 

ANSWER
Although the ITCAM for AD guides do not provide a great deal of information defining the pre-defined roles, here is basic information with respect to Problem Determination. All user roles can perform functions for Availability and Performance Analysis(Reports). The exceptions are the following, 
ADMINISTRATION 

1. Administrator (access to everything) 

 *  Admins can do everything the with the ITCAM VE. 


2. Operator  *  The operator can only administer the following, 
 *  Web Server Administration(View Only) 
 *  Managing Server Self Diagnosis 
 *  Can view whether or not all the MS components are up and running. 

3. User  *  The only function the regular User can do is Web Server Administration (View Only) 


PROBLEM DETERMINATION 

1. Administrator (access to everything):
