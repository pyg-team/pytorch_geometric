Title: IBM BMT-MD-003/DPR-ERR-2008 importing metadata in Cognos BI Framework Manager - United States

Text:
 TECHNOTE (TROUBLESHOOTING)

PROBLEM(ABSTRACT)
 The error message displayed in Framework Manager is:
BMT-MD-0003 DPR-ERR-2008 The dispatcher is unable to process the request. The request is directed to an unknown service name: www.ibm.com/xmlns/prod/cognos/contentManagerService/201301/. DPR-ERR-2082 An error has occurred. Please contact your administrator. The complete error has been logged by CAF with SecureErrorID: ... 

CAUSE
Incompatible product versions.

DIAGNOSING THE PROBLEM
Check the \cognos\cmplst.txt files from Cognos BI Server and Framework Manager


Product versions don't match.


RESOLVING THE PROBLEM
It is required that the system be running the same version of Cognos BI Server and Framework Manager. Install the correct product version.
