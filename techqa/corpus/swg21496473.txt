PROBLEM(ABSTRACT)
 Get Sun motif library errors due to missing fonts during Tomcat DC configuration. 

SYMPTOM

Bringing up the DC GUI gives these errors: 

[/support/docview.wss?uid=swg21496473&aid=1] [/support/docview.wss?uid=swg21496473&aid=1]


CAUSE
This is caused due to missing Xming fonts.


ENVIRONMENT
Xming X server on a local PC.



RESOLVING THE PROBLEM
 

You will see the above errors if you're using Xming X server and are trying to start the X server to remotely configure the Tomcat server. 

Install the correct Xming fonts before starting up your Xming server. See screenshot below. You will not be able to start the DC config using the Xming server if you don't have the right fonts installed.
