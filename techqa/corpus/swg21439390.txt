PROBLEM(ABSTRACT)
 This document contains a comprehensive list of recommended, generally available (GA) fixes for IBM Tivoli Composite Application Manager (ITCAM) for Application Diagnostics releases. 

RESOLVING THE PROBLEM
The recommended fixes are below. 

When installing the recommended fixes, please keep in mind that: 

 1. Base code must be installed prior to installing any fixes. The base code level of ITCAM for Application Diagnostics is 7.1. 
 2. All fix packs (FPs) are cumulative which means they include all of the fixes included in the previous fix packs along with those new fixes identified for the latest fix pack listed below. Interim fixes (iFixes) are not cumulative unless specifically stated.
