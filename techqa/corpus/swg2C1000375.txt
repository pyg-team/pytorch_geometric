Title: IBM Security Bulletin: Multiple Vulnerabilities in Apache Tomcat affects IBM UrbanCode Deploy (CVE-2017-5647, CVE-2017-5650) - United States

Text:
security; vulnerability; psirt; CVE-2017-5647; CVE-2017-5650 SECURITY BULLETIN

SUMMARY
 Previous releases of IBM UrbanCode Deploy are affected by multiple vulnerabilities in Apache Tomcat. 

VULNERABILITY DETAILS
CVE-ID: CVE-2017-5647 [http://cve.mitre.org/cgi-bin/cvename.cgi?name=CVE-2017-5647]
Description: Apache Tomcat could allow a remote attacker to bypass security restrictions, caused by the failure to properly enforce security constraints that are defined by annotations of Servlets in certain cases. An attacker could exploit this vulnerability to bypass security constraints to access restricted resources.
CVSS Base Score: 7.5
CVSS Temporal Score: https://exchange.xforce.ibmcloud.com/vulnerabilities/139475 [https://exchange.xforce.ibmcloud.com/vulnerabilities/139475] for more information
CVSS Environmental Score*: Undefined
CVSS Vector: (CVSS:3.0/AV:N/AC:L/PR:N/UI:N/S:U/C:N/I:H/A:N)

CVE-ID: CVE-2017-5650 [http://cve.mitre.org/cgi-bin/cvename.cgi?name=CVE-2017-5650]
Description: Apache Tomcat could allow a remote attacker to bypass security restrictions, caused by the failure to properly enforce security constraint definitions that contain a URL pattern of "" (the empty string) that exactly maps to the context root. An attacker could exploit this vulnerability to bypass security constraints to access restricted resources.
CVSS Base Score: 7.5
CVSS Temporal Score: https://exchange.xforce.ibmcloud.com/vulnerabilities/139476 [https://exchange.xforce.ibmcloud.com/vulnerabilities/139476] for more information
CVSS Environmental Score*: Undefined
CVSS Vector: (CVSS:3.0/AV:N/AC:L/PR:N/UI:N/S:U/C:N/I:H/A:N) 

AFFECTED PRODUCTS AND VERSIONS
All fixpacks of IBM UrbanCode Deploy 6.1 - 6.1.3.6 and IBM UrbanCode Deploy 6.2 - 6.2.4.0 are affected.

REMEDIATION/FIXES
Upgrade to IBM UrbanCode Deploy 6.2.4.1 [https://www-945.ibm.com/support/fixcentral/swg/selectFixes?parent=ibm~Rational&product=ibm/Rational/IBM+UrbanCode+Deploy&release=6.2.4.0&platform=All&function=all] or later. If it is not possible to upgrade to 6.2.4.1, upgrade to IBM UrbanCode Deploy 6.1.3.7 [https://www-945.ibm.com/support/fixcentral/swg/selectFixes?parent=ibm%7ERational&product=ibm/Rational/IBM+UrbanCode+Deploy&release=6.1.3&platform=All&function=all].
