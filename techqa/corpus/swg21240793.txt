CAUSE
You are not using the correct level of Javaâ„¢ with CICS TS V3.1. Refer to the CICS Program Directory for CICS TS V3.1 [http://publib.boulder.ibm.com/infocenter/cicsts/v3r1/topic/com.ibm.cics.ts31.doc/pdf/dfha9b00.pdf] which states that you need 5655-D35 IBM Software Developer Kit for z/OS, Java 2 Technology Edition V1.4.

RESOLVING THE PROBLEM
Upgrade your level of Java to IBM Software Developer Kit for z/OS, Java 2 Technology Edition V1.4. 


You also need the PTF for Java for z/OS SDK V1.4.1 (SR1) APAR PQ79281. 

You can check the level of Java you are running by entering the following command from OMVS:

java -fullversion
