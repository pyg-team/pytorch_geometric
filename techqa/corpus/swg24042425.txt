INSTALLATION INSTRUCTIONS
There are two methods of installing an interim fix using Installation Manager: 

1. Download and unzip the interim fix file, then point Installation Manager at the root of the unzipped file and press Update (following the instructions in the wizard).

2. Start Installation Manager, ensuring that you enable the connection to Passport Advantage within the Installation Manager Preferences, then press Update and follow the instructions for remote installation of the interim fix in the wizard.
