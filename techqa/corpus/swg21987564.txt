Title: IBM Oracle Database configuration parameters required to deploy Atlas application - United States

Text:
Atlas Oracle TECHNOTE (FAQ)

QUESTION
 What are the Oracle Database configuration parameters required to deploy Atlas application? 

CAUSE
Some queries run for a long time and timeout

ANSWER
 

 1. Oracle Text is required 
 2. Database must use the Unicode (AL32TUF8) character set. 
 3. Hierarchical queries must be enabled by adding alter system set "_allow_level_without_connect_by"=true to the initialization parameter. 
 4. alter system set "_replace_virtual_columns" = false; 
 5. Container database is not supported (Only for Oracle 12c)
