ERROR DESCRIPTION
 *  cics process hang issue at MF COBOL exit handler
   
   
    
   
   

LOCAL FIX
 *  fix
   
   
    
   
   

PROBLEM SUMMARY
 *  When CICS Application Server(CICSAS) process running Micro Focus
    COBOL transation and lead to some COBOL runtime error, CICSAS
   process is not terminating and listing as defunct (from ps
   command output).
   This further resulted in the region hang when all CICSAS process
   listed as defunct.
   
   
    
   
   

PROBLEM CONCLUSION
 *  Linking to the multi threaded MF COBOL libraries while creating
   COBOL language (runtime) method lead to this problem.
   Changed from multi threaded to single thread MF COBOL libraries
   for creating COBOL language (runtime) method.
   
   
    
   
   

TEMPORARY FIX
 *  Kill all CICSAS processes which are listing as defunct to avoid
   region hang.
