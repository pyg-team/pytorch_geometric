Title: IBM Security Bulletin: Multiple Vulnerabilities in IBM Java SDK and IBM Java Runtime affect IBM Workload Manager (CVE-2018-2633 CVE-2018-2603 CVE-2018-2579 CVE-2018-2637 and CVE-2018-2588) - United States

Text:
TWS CVE-2018-2633 CVE-2018-2603 CVE-2018-2579 CVE-2018-2637 CVE-2018-2588 SECURITY BULLETIN

THIS DOCUMENT APPLIES ONLY TO THE FOLLOWING LANGUAGE VERSION(S):
 US English 

SUMMARY
 There are multiple vulnerabilities in IBM® SDK Java™ Technology Edition and IBM® Runtime Environment Java™ that are used by IBM Workload Scheduler. These issues were disclosed as part of the IBM Java SDK updates in January 2018.

AFFECTED PRODUCTS AND VERSIONS
IBM Workload Scheduler is potentially impacted by the listed vulnerabilities since they potentially affect secure communications between eWAS and subcomponents.

The affected version is:
Tivoli Workload Scheduler Distributed 8.6.0 
Tivoli Dynamic Workload Console 8.6.0

REMEDIATION/FIXES
IBM has provided patches for all embedded WebSphere versions. 

Follow the instructions in the link below to install the fixes for eWAS 7.0.0.39 that is embedded in TWS 8.6 fixpack 04 :

http://www-01.ibm.com/support/docview.wss?uid=swg22013818 [http://www-01.ibm.com/support/docview.wss?uid=swg22013818]

The fixes can be applied on top of TWS version 8.6 only after TWS 8.6 fixpack 04 has been installed.

For unsupported versions, releases or platforms IBM recommends upgrading to a fixed, supported version/release/platform of the product.
