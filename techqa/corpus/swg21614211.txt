QUESTION
 Alcatel 5620 SAM Probe : How are the EMS jar files defined? 

CAUSE
Probe does not use the correct jar files

ANSWER
The latest non-native probe script [nco_jprobe] uses the PROGRAM name to locate a probe specific environment configuration file, in $NCHOME/omnibus/probes/java. 

The recommendation is to create a PROGRAM folder in this directory to hold the Alcatel 5620 SAM version specific jar files. 


Support's Guide to the Alcatel 5620 SAM Probe includes an additional method to allow multiple versions of the Alcatel 5620 SAM probe to be run on the same server without having to specify the version specific probe name.
