Title: IBM CTGIN2275E: WebSphere thin admin client install verification completed with errors - United States

Text:
CTGIN2275E; websphere; thin; admin; client; ADMN0022E; access; denied; insufficient; empty; credentials; TPAEWEBSPHERE TECHNOTE (TROUBLESHOOTING)

PROBLEM(ABSTRACT)
 Receiving error installing fixpack, CTGIN2275E: WebSphere thin admin client install verification completed with errors
"ADMN0022E: Access is denied for the 
getProcessType operation on Server MBean because of insufficient or 
empty credentials." 



SYMPTOM
unable to proceed with fix pack installation


RESOLVING THE PROBLEM
Log into Maximo, go to the System Properties Application, search for WAS.AdminPassword and change the value to the WebSphere admin password and this should be updated in the install.propteries file as well located at C:\ibm\smp\etc
