APAR STATUS
 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  In Ja_JP locale of AIX, run tacmd listworkspaces throw
   java.io.UnsupportedEncodingException:
       at sun.io.Converters.getConverterClass(Converters.java:318)
       at sun.io.Converters.newConverter(Converters.java:371)
     The workspaces can be listed but some with blank name.
   
   
    
   
   

LOCAL FIX
 *  Run the command in a terminal of LANG=C
   
   
    
   
   

PROBLEM SUMMARY
 *  tacmd commands throw UnsupportedEncodingException on Ja_JP
   locale.
   
   
   On Ja_JP locale on AIX, running tacmd listworkspaces throws the
   following exception:
   
   java.io.UnsupportedEncodingException
   at sun.io.Converters.getConverterClass(Converters.java:318)
   at sun.io.Converters.newConverter(Converters.java:371)
   The workspaces are listed, but some have a blank name.
   
   
    
   
   

PROBLEM CONCLUSION
 *  The code has been changed to support the Ja_JP locale and others
    locale with the ICU standard for all the tacmd commands that
   interact with the portal server.
   
   
   The fix for this APAR is contained in the following maintenance
   packages:
   
      | fix pack | 6.3.0-TIV-ITM-FP0005
