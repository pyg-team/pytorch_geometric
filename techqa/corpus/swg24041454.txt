ABSTRACT
 This is the first interim fix pack after ITCAM Agents for WebSphere Messaging Version 7.3.0 FP01 

DOWNLOAD DESCRIPTION
This interim fix provides support of IBM Integration Bus Version 10.0.0.2 and refreshes the Java certificate of application support files.

PREREQUISITES
1. ITCAM Agents for WebSphere Messaging Version 7.3.0 Fix Pack 1 must be installed. 


 See the link below for details:
http://www-01.ibm.com/support/docview.wss?uid=swg24040581 [http://www-01.ibm.com/support/docview.wss?uid=swg24040581]

2. Make sure that the software installed in your environment is compatible with this interim fix.
