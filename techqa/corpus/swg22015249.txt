Title: IBM Restore issue with Content Collector - United States

Text:
 TECHNOTE (TROUBLESHOOTING)

PROBLEM(ABSTRACT)
 When trying to restore the archived email, receiving an internal processing error coming from ICC Notes Client. 

ENVIRONMENT
Content Collector for Email 

Note 8.5


DIAGNOSING THE PROBLEM

RESOLVING THE PROBLEM
Followed steps to create SSL certificate 


1. Run CMD prompt as administrator
Executed afu_ewas_exchange_cert.bat to create new SSL certificate

2. Executed afu_ewas_backup_cert.bat to take backup of new SSL certificate.

3. Verified new SSL certificate by ikeyman.exe.

4. Tested Restore functionality on Lotus domino and it work as expected.
