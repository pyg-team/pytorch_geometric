ABSTRACT
 This document describes how to download IBM® InfoSphere™ Federation Server Developer Edition Version 10.1 from the IBM Passport Advantage® Web site. 

DOWNLOAD DESCRIPTION
 



IBM® IBM InfoSphere Federation Server provides innovative virtualization technology which enables you to access multiple diverse data sources in real time as if they were a single source. 

More information

For product release information, see the Release notes [http://publib.boulder.ibm.com/infocenter/db2luw/v10r1/topic/com.ibm.swg.im.iis.prod.fed.rel.info.doc/topics/iiypifrinf-reln-v10ga.html].

For additional product information, see the InfoSphere Federation Server Information Center [http://publib.boulder.ibm.com/infocenter/db2luw/v10r1/topic/com.ibm.swg.im.iis.db.prod.fed.nav.doc/dochome/iiypfnav_dochome.html].

Space requirements

For more information on disk space requirements, see the Hardware and software requirements [ http://publib.boulder.ibm.com/infocenter/db2luw/v10r1/topic/com.ibm.swg.im.iis.prod.install.core.doc/topics/iiypisrq-sys.html] for InfoSphere Federation Server. 

Downloading the product

 1. Download all of the parts that you need, both required and optional. The parts are described by platform in the tabbed sections of this document.
 2. Unpack each of the parts into a single temporary directory on your system.
 3. Follow the instructions in the Quick Start Guide to install the product.
