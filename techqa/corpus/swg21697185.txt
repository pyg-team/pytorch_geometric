Title: IBM Case Manager Administrator Client 5.2: Get error when applying Security Configuration to a Solution: The security manifest could not be applied. FNRAM9052 - United States

Text:
Security Manifest TECHNOTE (TROUBLESHOOTING)

PROBLEM(ABSTRACT)
 IBM Case Manager Administrator Client 5.2: Get error when applying Security Configuration to a Solution: The security manifest could not be applied. FNRAM9052 

SYMPTOM
Get error when applying Security Configuration to a Solution: 

The security manifest could not be applied. FNRAM9052

The security manifest of the solution could not be applied because of the following error: FNRPA0087E An unexpected error occurred.

Try the request again. If the problem persists, contact your system administrator.

Review the associated message and the IBM Content Navigator server log.


CAUSE
The version of Content Platform Engine (CPE) is mismatched between CPE and IBM Content Navigator


ENVIRONMENT
Content Platform Engine
