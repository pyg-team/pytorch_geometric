LOCAL FIX
 *  Technote to be published.
   
   
    
   
   

PROBLEM SUMMARY
 *  ****************************************************************
   * USERS AFFECTED:  All users of WebSphere Application Server   *
   ****************************************************************
   * PROBLEM DESCRIPTION: When using the Update Installer to      *
   *                      install the 6.0.2.27 or 6.0.2.29 JDK    *
   *                      fixpack for the Plugins the install     *
   *                      fails.                                  *
   ****************************************************************
   * RECOMMENDATION:                                              *
   ****************************************************************
   Plugins AIX ppc64 bit did not contain the correct OS platform
   architecture which caused the JDK fixpack for the Plugins
   install to fail.
   
   
    
   
   

PROBLEM CONCLUSION
 *  Plugins AIX ppc64 bit corrected to contain the correct OS
   platform architecture
   
   The fix for this APAR is currently targeted for inclusion in
   fixpack 6.0.2.31.  Please refer to the Recommended Updates
   page for delivery information:
   http://www.ibm.com/support/docview.wss?rs=180&uid=swg27004980 [http://www.ibm.com/support/docview.wss?rs=180&uid=swg27004980]
