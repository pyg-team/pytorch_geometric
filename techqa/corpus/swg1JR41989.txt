 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  Error Description&#65496;
   The node agents make multiple calls to the Dmgr to poll the BLA
   status causing excessive CPU usage. The fix is to make those
   calls locals instead of polling the dmgr.

   n/a
