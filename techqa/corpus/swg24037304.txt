ABSTRACT
 This document describes how to download the TDM solutions for Siebel CRM and Siebel CRM with data masking option of IBM® InfoSphere® Optim™ Solution for Oracle Applications v11.3. 

DOWNLOAD DESCRIPTION
You can download the test data management solutions for Siebel CRM and Siebel CRM with data masking portions of the v11.3 IBM InfoSphere Optim Solution for Oracle Applications using the Passport Advantage website.


IBM InfoSphere Optim Test Data Management Solution for Siebel CRM and Optim Test Data Management Solution for Siebel CRM with Data Masking Option help you to build realistic and right-sized non-production databases. They allow you to easily extract a selected subset of data from your production Siebel CRM database, mask sensitive data, and insert the data into a separate Siebel instance, avoiding the expense and effort of maintaining multiple clones of your production database. By copying data from your production database to a separate instance of Siebel CRM, you can furnish your Siebel testing environments, establish or refresh your training environment, and support demonstrations of your Siebel software.

PREREQUISITES
You must install IBM InfoSphere Optim Solution v11.3 before you install the Optim Test Data Management Solution for Siebel CRM or the Optim Test Data Management Solution for Siebel CRM with Data Masking Option. 

For information about hardware and software requirements for the Optim Test Data Management solution for Siebel CRM and the Optim Test Data Management Solution for Siebel CRM with Data Masking Option, see the Detailed System Requirements at http://www.ibm.com/support/docview.wss?&uid=swg27041753 [http://www.ibm.com/support/docview.wss?&uid=swg27041753].

INSTALLATION INSTRUCTIONS
For general installation instructions, see the Quick Start Guide.  4. Download all required parts.
 5. Use a file extracting utility that supports large file extractions to extract each of the parts into a single, temporary directory on your system.
 6. Follow the instructions in the Quick Start Guide to install the product.
