Title: IBM WebSphere Application Server および IBM HTTP Server におけるApache Struts 2脆弱性 (CVE-2017-5638) の影響について - United States

Text:
 SECURITY BULLETIN

SUMMARY
 IBM WebSphere Application Server および IBM HTTP Server は、Apache Struts 2 の脆弱性（CVE-2017-5638） の影響を受けません。 

VULNERABILITY DETAILS
最新の情報については、下記URLの文書（英語）もご参照ください。
IBM WebSphere Application Server is not vulnerable to CVE-2017-5638
http://www.ibm.com/support/docview.wss?uid=swg22000122 [http://www.ibm.com/support/docview.wss?uid=swg22000122]

WebSphere Application Server(Liberty含む) と IBM HTTP Serverは、Apache Struts 2 脆弱性(CVE-2017-5638)の影響を受けません。

(ご注意) お使いのアプリケーションでApache Struts 2 を利用している場合、CVE-2017-5638の影響を受ける可能性があります。
アプリケーションが脆弱性の影響を受けるかどうかは、以下のApache Struts 2 サイト（英語）をご参照ください。
Apache Struts 2 Documentation "S2-045"
https://cwiki.apache.org/confluence/display/WW/S2-045 [https://cwiki.apache.org/confluence/display/WW/S2-045]
