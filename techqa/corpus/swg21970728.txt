Title: IBM Does OpenPages with Weblogic support SHA-256 encryption? - United States

Text:
 TECHNOTE (FAQ)

QUESTION
 How can SHA-256 encryption be utilized for OpenPages installation on Weblogic Application Server? 

ANSWER
Customers using WebLogic Application Server for their OpenPages installation can make use of SHA-256 encryption, however this will require the customer to download and install the Application IF3 patch. 

For further information, please see the following link: 


http://www.ibm.com/support/docview.wss?uid=swg24040412 [http://www.ibm.com/support/docview.wss?uid=swg24040412]
