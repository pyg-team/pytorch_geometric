Title: IBM Cognos report fails to load, RSV-CM-0008 The specification signature is invalid. - United States

Text:
RSV-CM-0008; RSV-SRV-0042; "signature is invalid"; RSReportService TECHNOTE (TROUBLESHOOTING)

PROBLEM(ABSTRACT)
 Cognos report fails to display data, returns error RSV-CM-0008. 

DIAGNOSING THE PROBLEM
> Open IBM Cognos Configuration GUI 

> Under the navigator item "Local Configuration" choose 
-> Local Config -> Security -> Cryptography -> Cognos 

Confirmed assigned values for the "Supported Ciphersuites".


RESOLVING THE PROBLEM
Click in the "Value" field for the item "Supported Ciphersuites". When the "Value" field is highlighted, a pencil icon will appear. 

 
From the "Available Values" section, choose these 6 values adding them to the "Current Values" section, in this order.

RSA-RSA-DES(168)CBC3-SHA 
DH-RSA-DES(168)CBC3-SHA 
DH-DSS-AES(128)CBC-SHA 
DH-DSS-DES(168)CBC3-SHA 
DH-RSA-AES(128)CBC-SHA 
RSA-RSA-AES(128)CBC-SHA
