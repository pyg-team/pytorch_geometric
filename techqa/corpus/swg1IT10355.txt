APAR STATUS
 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  The 'x-dp-cache-key' header unique cache key is used for 'fixed'
   document cache policy types, but not for 'protocol based' cache
   policy types.
   
   
    
   
   

LOCAL FIX

PROBLEM SUMMARY
 *  Affected is the us of document cache.
   
   'x-dp-cache-key' header unique cache key is not used for
   'protocol based' document cache policy types.
   
   
    
   
   

PROBLEM CONCLUSION
 *  The fix is available in 6.0.0.17, 6.0.1.13, 7.0.0.10, 7.1.0.7
   and 7.2.0.1
   
   For a list of the latest fix packs available, please see:
   http://www-01.ibm.com/support/docview.wss?uid=swg21237631 [http://www-01.ibm.com/support/docview.wss?uid=swg21237631]
