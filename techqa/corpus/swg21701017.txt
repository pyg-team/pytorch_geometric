The problem persists even after command deployConsole remove and deployConsole install. 

CAUSE
The “iscliete.ear” file got corrupted. 

RESOLVING THE PROBLEM
You need to restore the isclite.ear from the backup to the state when the ISC was working. 

Another thing you can do is to replace the iscliete.ear file if you have another exact working environment. The file isclite.ear is located at app_server_root/systemApps.
