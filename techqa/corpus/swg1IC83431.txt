Title: IBM IC83431: NEW REPLICATION ERROR MESSAGE NEEDED - United States

Text:
AIX SUBSCRIBE

APAR STATUS
 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  Replication ends with completion status FAILURE, but no reason
   given.  A new replication message ANR1936E needs to be created
   to address the reasons for the failure.
        ANR1936E Replication of xxxx data for file space xxx, (fsid
   xx) on node xxxx, failed with reason:  reason.
   
   Customer/L2 Diagnostics (if applicable)
   During testing of an e-fix defect 82128 was encountered.  The
   failure was due to a GRC_COMM_ERROR
   
   *NOTE*
   Examples of new ANR1936E replication failure messages:
   ANR1936E Replication of SPACE MANAGEMENT data for file space
   /lspace (fsId 1) on node NODE3 failed with reason: Item not
   found.
   ANR1936E Replication of ARCHIVE data for file space /lspace
   (fsId 1) on node NODE3 failed with reason: Transaction failed.
   ANR1936E Replication of BACKUP data for file space /lspace (fsId
   1) on node NODE3 failed with reason: Authentication Failure.
   ANR1936E Replication of ARCHIVE data for file space /lspace
   (fsId 1) on node NODE3 failed with reason: Communication
   Failure.
   ANR1936E Replication of SPACE MANAGEMENT data for file space
   /lspace (fsId 1) on node NODE3 failed with reason: Server
   internal error.
   ANR1936E Replication of BACKUP data for file space /lspace (fsId
   1) on node NODE3 failed with reason: Reason code 4000.
   
   Platforms affected:
   Server 6.3 Unix/Linux/Windows
   
   Initial Impact: Low
   
   Additional Keywords: ZZ63 repl
