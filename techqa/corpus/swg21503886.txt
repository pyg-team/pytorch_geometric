CAUSE
Controller application server's COM+ Windows user is not a member of the local 'Administrators' group on the application server.



RESOLVING THE PROBLEM
Add the Controller application server's COM+ Windows user account to the local Windows 'Administrators' group on the application server. Afterwards, obtain a short period of downtime and reboot the application server.
