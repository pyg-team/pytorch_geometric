Title: IBM InfoMan Web Connector and Web Access compatibility on z/OS V2.1 and higher - United States

Text:
infoman web wai connector migrate domino apache TECHNOTE (FAQ)

QUESTION
 Is the InfoMan Web Connector (WAI) and the InfoMan Web Connector for OS/390 compatibile with z/OS V2.1 and higher. 

CAUSE
As of z/OS V2.1, the IBM HTTP Web server will NO longer be powered by Domino, as of z/OS V2.1, it will be powered by Apache.

ANSWER
You MUST migrate your IBM HTTP server to run under Apache. 

Prior to z/OS V2.1, the IBM HTTP Web server was powered by Domino, but from z/OS V2.1 and higher, it will be powered by Apache. Make sure to migrate your IBM HTTP Web Server from Domino to Apache.
