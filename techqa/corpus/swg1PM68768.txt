PROBLEM SUMMARY
 *  ****************************************************************
   * USERS AFFECTED:  All IBM WebSphere Application Server,       *
   *                  Feature Pack for Modern Batch users         *
   ****************************************************************
   * PROBLEM DESCRIPTION: Duplicate key exception in scheduler    *
   *                      log.                                    *
   ****************************************************************
   * RECOMMENDATION:                                              *
   ****************************************************************
   com.ibm.websphere.ce.cm.DuplicateKeyException: DB2 SQL Error:
   SQLCODE=-803,SQLSTATE=23505, SQLERRMC=JCMCJIX;0000014201,
   DRIVER=3.61.84 trigger when there are no entries in the
   scheduler database and multiple jobs are submitted to the
   scheduler at the same time.
   
   
    
   
   

PROBLEM CONCLUSION
 *  Multiple operations concurrently try to initialize the database
   table, causing DuplicateKeyException. This has been corrected.
   
   The fix for this APAR is currently targeted for inclusion in
   fix pack 1.0.0.5. Please refer to the Recommended Updates
   page for delivery information:
   http://www.ibm.com/support/docview.wss?rs=180&uid=swg27004980 [http://www.ibm.com/support/docview.wss?rs=180&uid=swg27004980]
