APAR STATUS
 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  In IBM Content Navigator, if you edit an entry template and set
   the field width to 400 pixels, for instance, when you modify a
   property of a document in teamspaces, after you click on Save,
   you get a pop up message if you go to another document.
   
   
    
   
   

LOCAL FIX
 *  Click Save twice
   
   
    
   
   

PROBLEM SUMMARY
 *  In IBM Content Navigator, if you edit an entry template and set
   the field width to 400 pixels, for instance, when you modify a
   property of a document in teamspaces, after you click on Save,
   you get a pop up message if you go to another document.
   
   
    
   
   

PROBLEM CONCLUSION
 *  This issue has been fixed in IBM Content Navigator V2.0.3 Fix
   Pack 8 LA004 and V3.0 LA001
