Collecting general data for all problems
A complete description of the problem, including the following: 

 * WebSphere Message Broker version, release, and maintenance level 
 * Operating system version, release, and maintenance level 
 * Related products version, and release levels if applicable 
 * Components installed 
 * Number of machines (include operating systems) 
 * Number of brokers (include operating systems) 
 * Details of any User Name Servers on V6.1 and V6.(This is not applicable for V7 or V8) 
 * Is the failure occurring in more than one address space?Back to top
   
   


Collecting troubleshooting data for this problem 
Required (MustGather) data:  * Broker component joblog or screen capture showing the issue.
   See: z/OS broker joblogs [http://publib.boulder.ibm.com/infocenter/wmbhelp/v8r0m0/index.jsp?topic=/com.ibm.etools.mft.doc/an20101_.htm]


Optional (MustGather) data:  * Refer to any documentation that accompanied the Plugin/SupportPac to ensure it is installed and setup properly.Back to top
