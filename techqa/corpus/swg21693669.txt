Title: IBM 【セキュリティ情報】IBM WebSphere Commerce　TLS パディング脆弱性の影響 (CVE-2014-8730) - United States

Text:
 SECURITY BULLETIN

SUMMARY
 WebSphere Commerceに同梱しているIBM HTTP Server とIBM Tivoli/Security Directory Serverに関して、TLS パディング脆弱性 (CVE-2014-8730)セキュリティ速報(security bulletins) が公開されました。


VULNERABILITY DETAILS
次のセキュリティ速報(security bulletins)を参照し、対応を実施してください。
TLS padding vulnerability affects IBM HTTP Server (CVE-2014-8730) [http://www-01.ibm.com/support/docview.wss?uid=swg21692502] a
TLS padding vulnerability affects IBM Tivoli/Security Directory Server (CVE-2014-8730) [http://www-01.ibm.com/support/docview.wss?uid=swg21692906] 

AFFECTED PRODUCTS AND VERSIONS
次の製品に影響があります。 

主要な製品・バージョン 影響のある製品とバージョン WebSphere Commerce V6.0 and V7.0 IBM HTTP Server (すべてのバージョン、リリース).

IBM Tivoli/Security Directory Server versions 6.0, 6.1, 6.2, 6.3 and 6.3.1. GET NOTIFIED ABOUT FUTURE SECURITY BULLETINS
 Subscribe to My Notifications [ http://www-01.ibm.com/software/support/einfo.html] to be notified of important product support alerts like this.
