Title: IBM Why am I getting "arsload: could not connect to server to establish log id" message when trying to run arsload daemon on Content Manager OnDemand for Multiplatforms? - United States

Text:
 TECHNOTE (TROUBLESHOOTING)

PROBLEM(ABSTRACT)
 Why am I getting "arsload: could not connect to server to establish log id" message when trying to run arsload daemon on Content Manager OnDemand for Multiplatforms? 

SYMPTOM
Error message : "arsload: could not connect to server to establish log id"


RESOLVING THE PROBLEM
 1. Check ars.ini file. 

 2. Check HOST Parameter 

 3. If the HOST Parameter is blank, replace it with a fully qualified host name of IP address. 

 4. Run arsload daemon.
