APAR STATUS
 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  When a non admin user with contributor access to the library
   tries to move content between site areas, an authorization
   exception occurs.
   
   
    
   
   

LOCAL FIX
 *  The permission checks for the move operation should not include
   checking whether the user has editor access on the top library
   level.

   level.
