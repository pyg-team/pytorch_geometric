Title: IBM LO56169: ALL DAY EVENT MAY EXTEND ONE DAY BEFORE OR AFTER. - United States

Text:
 SUBSCRIBE
You can track all active APARs for this component.



APAR STATUS
 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  Creating all day event on Nokia device the event may extend one
   day before or one day after when synced to the server calendar.
   
   
    
   
   

LOCAL FIX
 *  NA
   
   
    
   
   

PROBLEM SUMMARY
 *  All day event may appear off by one day on Nokia client.
   
   
    
   
   

PROBLEM CONCLUSION
 *  Nokia does not handle timezone fields well, it works better
   without specifying timezone and let Nokia handle as it normally
   does.
   
   
    
   
   

TEMPORARY FIX
 *  This fix will be included in 8521 and any future releases.  See
   this technote for latest maintenance info:
   http://www.ibm.com/support/docview.wss?uid=swg24019529 [http://www.ibm.com/support/docview.wss?uid=swg24019529]
