APAR STATUS
 * CLOSED AS DOCUMENTATION ERROR.
    
   
   

ERROR DESCRIPTION
 *  In the IBM Tivoli Monitoring for Virtual Environments
   information center, version 7.1, the existing Prerequisites
   topic for VMware VI agent mentions that the VMware VI agent
   supports IBM Tivoli Monitoring 6.2.2 Fix Pack 4 or later. The
   existing content must be updated to mention that the VMware
   agent supports IBM Tivoli Monitoring 6.2.2 Fix Pack 2 or later.
   Note: IBM Tivoli Monitoring 6.2.2 Fix Pack 3 does not support
   the following features:
   ? Remote deployment of VMware VI agent.
   ? Remote configuration of VMware VI agent instances.
   
   
    
   
   

LOCAL FIX
 *  This content modification in the Prerequisites topic for VMware
   VI agent will be communicated through a Technote and a blog in
   Service Management Connect (SMC).
