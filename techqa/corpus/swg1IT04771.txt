APAR STATUS
 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  There are conditions when network errors might cause an
   appliance to restart if MQ shared conversation is configured.
   
   
    
   
   

LOCAL FIX
 *  Resolve any network issues between the DataPower appliance and
   the WebSphere MQ server.
   
   
    
   
   

PROBLEM SUMMARY
 *  Customers who use the shared conversation function of MQ with
   an unstable network are likely to see this problem.
   
   DataPower might restart with MQ using a network which is
   experiences errors.
   
   
    
   
   

PROBLEM CONCLUSION
 *  Fix is available in 5.0.0.18, 6.0.0.10, 6.0.1.6 and 7.0.0.3.
