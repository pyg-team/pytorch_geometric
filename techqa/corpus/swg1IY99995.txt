Title: IBM IY99995: REPORTS UNDER 'MY FOLDERS' ARE NOT GETTING MIGRATED FROM TCR 1.3 TO TCR 2.1.1 - United States

Text:
 SUBSCRIBE
You can track all active APARs for this component.



APAR STATUS
 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  Problem description: When a user migrates from TCR 1.3 to TCR
   2.1.1, the migration does not include any reports which is part
   of 'My Folders'. Only 'Public Folders' reports' are migrated.
   This causes loss of reports for the user.
   Reproduce steps:
   a) Install TCR 1.3 and navigate to TIP Console -> Common
   Reporting
   b) Create reports/report packages under 'My Folders'
   c) Now try installing TCR 2.1.1 with the 'Upgrade' option
   d) Upgrade is successful but in TIP Console -> Common Reporting,
   no reports are migrated under 'My Folders'
   
   
    
   
   

LOCAL FIX
 *  NA
   
   
    
   
   

PROBLEM SUMMARY
 *  While exporting data from TCR 1.3 to TCR 2.1.1, the data/reports
   under 'My Folders' are not exported/migrated. This causes
   inconvinience to the customers who have data/reports under 'My
   Folders'.
   
   
    
   
   

PROBLEM CONCLUSION
 *  This APAR and the interim fix - 2.1.1.0-TIV-TCR-IF0004 in Fix
   Central provides a fix for the above described issue. Please
   follow the readme file for the procedure to install and get the
   'My Folders' into the new TCR 2.1.1
   
   
    
   
   

TEMPORARY FIX
