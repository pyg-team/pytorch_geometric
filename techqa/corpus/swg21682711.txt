Title: IBM Jazz for Service Management Version 1.1.2.1 integration services do not support the 64-bit version of Installation Manager - United States

Text:
jazzsm1121relnotes TECHNOTE (TROUBLESHOOTING)

PROBLEM(ABSTRACT)
 Failure while un-installing Jazz for Service Management Version 1.1.2.1 integration services using 64-bit version of Installation Manager. 

CAUSE
While using 64-bit version of Installation Manager, it checks each package for 64-bit support. If a package does not support the 64-bit version, you receive a warning message.

RESOLVING THE PROBLEM
To avoid this warning, it is recommended to use a 32-bit version of Installation Manager to install JazzSM
