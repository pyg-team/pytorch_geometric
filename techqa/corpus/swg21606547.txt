Title: IBM Weblogic J2EE Data Collector Machine Name white space - United States

Text:
ITCAM for J2EE CAMJ2WLDC CAMJ2INST TECHNOTE (TROUBLESHOOTING)

PROBLEM(ABSTRACT)
 ITCAM for J2EE Weblogic Data Collector fails configuration when Weblogic Machine Name contains whites spaces(ie. blanks). 

SYMPTOM
Unable to configure the J2EE DC on Weblogic when the Weblogic Machine name has white spaces in the name. For example "EPR Admin Machine". Only after it was "cloned" to "EPR_Admin_Machine" would the configuration run successfully.


CAUSE
Fixed Bugs List - Oracle WebLogic Server 11g Release 1 (10.3.1) (Doc ID982209.1) 

8170534 General 10.3.1 The ServerStartMBean.Arguments value validator does not permit the use of quotes in the value. This is important to escape space characters. The -Dweblogic.serverStart.allowQuotes=true system option can be used to allow quotes in the value. 

RESOLVING THE PROBLEM
Fixed Bugs List - Oracle WebLogic Server 11g Release 1 (10.3.1) (Doc ID
982209.1)
