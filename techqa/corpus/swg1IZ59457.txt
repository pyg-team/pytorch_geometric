ERROR DESCRIPTION
 *  **********************************************
   **********************************************
   Title: Fails to connect SAP managed resource if password of SAP
   account to connect includes "+".
   
   Problem Description:
   When user defined SAP service, he set a password that included
   "+" (plus) in "Password" field of "SAP CONNECTION DETAILS" tab.
   Then he tried to perform "Test Connection", it failed to connect
   to the SAP.
   
   He tried to directly logon to SAP using the password, he could
   successfully logon without any errors, so the password is
   correct.
   
   The password he used is "K9%+09GA".
   When he checked ibmdi.log, the password seemed to be recognized
   as "K9% 09GA".
