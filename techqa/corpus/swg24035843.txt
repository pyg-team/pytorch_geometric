Installation Instructions:
-----------------------------------------------
1) Stop Collaboration and Deployment Services.

2) Backup any existing packages located at "[SERVER_INSTALL_DIR]\staging\"
by copying them to a different location.
[SERVER_INSTALL_DIR] is typically something like "C:\Program Files\IBM\SPSS\Deployment\4.2.1\Server".

3) Extract the interim fix file to a temporary directory.

4) Run the IBM SPSS Collaboration and Deployment Services Package Manager* install all of
the packages from the temporary directory. For example, to install all of the packages using the command-line version of 
the package manager, use the following package manager command:
install 'C:\temp\configWeb.package' 'C:\temp\cradapter.package' 'C:\temp\jmxhttp.package' 'C:\temp\language-management.package' 'C:\temp\notification.package' 'C:\temp\PROCESS.package' 'C:\temp\scoring.package' 'C:\temp\search.package' 'C:\temp\security.package' 'C:\temp\securityWeb.package'

5) Remove the temporary directory.

6) Start Collaboration and Deployment Services.

Uninstall Instructions:
-----------------------------------------------
1) Stop Collaboration and Deployment Services.

2) Reapply the backup packages using the Collaboration and Deployment Services Package Manager*.

3) Start Collaboration and Deployment Services.


* Instructions for Package Manager can be found in the documentation, located at
"[SERVER_INSTALL_DIR]\documentation\en".
See the section "Package management" in the "IBM SPSS Collaboration and Deployment Services Repository 4.2 
Installation and Configuration Guide".
