APAR STATUS
 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  In the WebGUI, when the following Web Service Proxy settings are
   added or modified via Objects > Service Configuration > Web
   Service Proxy:
   
     - Process Messages Whose Body is Empty
     - Probe Transaction History
   
   the changes might be lost if the Web Service Proxy is changed
   using Services > Web Service Proxy > Edit Web Service Proxy.
   This is also the case when editing the service using the Control
   Panel.
   
   
    
   
   

LOCAL FIX
 *  Use other methods to edit the service such as the Objects menu
   or SOMA.
   
   
    
   
   

PROBLEM SUMMARY
 *  "Process Messages Whose Body is Empty" and "Transaction History"
    configuration reverts to default values upon configuring the
   Web Service Proxy from the Control Panel link.
   
   
    
   
   

PROBLEM CONCLUSION
 *  Fix is available in 5.0.0.16, 6.0.0.8, 6.0.1.4, and 7.0.0.2.
   
   For a list of the latest fix packs available, please see:
   http://www-01.ibm.com/support/docview.wss?uid=swg21237631 [http://www-01.ibm.com/support/docview.wss?uid=swg21237631]
