原因
この問題は障害 APAR PM22221 として登録されています。


IBM Rational License Server のバージョンが v8.1.1 より古いことが原因です。


バージョン 8.0 の Rational Software Architect for WebSphere, Rational Software Architect, Rational Application Developer は Rational License Server v8.1.1 以降を使用する必要があります。


解決方法
この問題を解決するためには、 Rational License server v8.1.1 をインストールし、IBM Installation Manager でフローティング･ライセンスを認識させる必要があります。 

Information Center にある 「Installation Manager を使用してフローティング キーを使用するようにクライアントを設定する」 [http://publib.boulder.ibm.com/infocenter/rational/v0r0m0/topic/com.ibm.rational.license.doc/topics/t_config_client_float_keys.htm] もしくは英文技術文書 1403623 [http://www-01.ibm.com/support/docview.wss?&uid=swg21403623]を参照してください。
