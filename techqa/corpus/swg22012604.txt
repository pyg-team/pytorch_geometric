Title: IBM Security Bulletin: Cross-site scripting vulnerability in Business Space affects IBM Business Process Manager, WebSphere Process Server, and WebSphere Enterprise Service Bus (CVE-2018-1384) - United States

Text:
 SECURITY BULLETIN

SUMMARY
 Cross-site scripting vulnerability in Business Space allows a remote attacker to inject script. 

VULNERABILITY DETAILS
CVEID:CVE-2018-1384 [http://cve.mitre.org/cgi-bin/cvename.cgi?name=CVE-2018-1384]
DESCRIPTION:IBM Business Process Manager is vulnerable to cross-site scripting. This vulnerability allows users to embed arbitrary JavaScript code in the Web UI thus altering the intended functionality potentially leading to credentials disclosure within a trusted session.
CVSS Base Score: 5.4
CVSS Temporal Score: Seehttps://exchange.xforce.ibmcloud.com/vulnerabilities/138135 [https://exchange.xforce.ibmcloud.com/vulnerabilities/138135]for the current score
CVSS Environmental Score*: Undefined
CVSS Vector: (CVSS:3.0/AV:N/AC:L/PR:L/UI:R/S:C/C:L/I:L/A:N)
