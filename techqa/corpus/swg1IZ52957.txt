ERROR DESCRIPTION
 *  On SAP system where the database table TSTCT contains
   millions of rows and where the number of active users
   is high, ITM V6.x MySAP agent processing can have a negative
   impact on SAP system performance because of the high number
   of queries performed on table TSTCT.
   
   
    
   
   

LOCAL FIX

PROBLEM SUMMARY
 *  When SAP Sysetm has many active users and sessions, the ABAP
   code will frequently to access TSTCT table, it causes high db
   workload. the fix is to cache some key records of TSTCT TABLE to
   reduce the workload on the TSTCT table.
   
   
    
   
   

PROBLEM CONCLUSION

TEMPORARY FIX
 *  It has been fixed in 6.2.0-TIV-ITM_SAP-IF0007 which has been
   released at July 1st.
