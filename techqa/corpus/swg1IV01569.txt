ERROR DESCRIPTION
 *  Tested the Following on Maximo 7117 and 7118
   
   
   Go to Inventory app
   Press enter on the List Tab to see all items
   Go to Select Action and Choose Inventory Adjustment> Physical
   Count
   Now select the Calendar besides the Physical Count Date field
   and Choose
   any Date, when you press on OK
   The Screen refreshes and updates the Count Date accordingly,
   however if
   you do not notice the Date being updated and press on REFRESH
   you will
   receive the following error
   
   BMXAA4214E - An unknown error has occurred.
   Please contact your system administrator for assistance.
   
   Client did notice that it does update the Count Date on the
   screen and
   is okay with Removing the REFRESH button at this point
   
   Tested in Maximo 7.5 and when choosing the Date and pressing on
   OK, it
   does NOT refresh and the REFRESH button needs to be pressed in
   order to
   see the count Date Change
   
   
    
   
   

LOCAL FIX
 *  Remove the REFRESH BUTTON
