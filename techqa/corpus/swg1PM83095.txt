Text:

Fixes integrated in WebSphere Portal 8.0.0.0 Cumulative Fixes [http://www-01.ibm.com/support/docview.wss?uid=swg24033155]

APAR STATUS
 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  With managed pages turned off, no wcm item can be created in a
   VP.
   
   Authoring in VPs should work with manage pages turned off,
   however items should appear in the base portal.
   
   
    
   
   

LOCAL FIX
 *  WCM will use the base Portal with authoring in a VP when manage
   pages is turned off.
   
   
    
   
   

PROBLEM SUMMARY
 *  Cannot create content in VP when managed pages is off.
   
   With managed pages turned off, no wcm item can be created in a
   VP.
   
   
    
   
   

PROBLEM CONCLUSION
 *  Authoring in VPs should work with manage pages turned off,
   however items should appear in the base portal.
   
   WCM will use the base Portal with authoring in a VP when manage
   pages is turned off.
