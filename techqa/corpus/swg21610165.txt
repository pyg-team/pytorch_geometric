Title: IBM Netcool/OMNIbus Integrations Release Notice - IBM® Tivoli® Netcool/OMNIbus Gateway for TSRM (nco_g_tsrm-5_0) - United States

Text:
release notice; downloads; nco-g-tsrm; TSRM Gateway NEWS

ABSTRACT
 A new version of the Netcool/OMNIbus Gateway for TSRM will be available to download from March 12, 2015. 

CONTENT
 

This version of the gateway has been updated to add the following functionality:

 * Added 64-bit support on selected platforms. 

It also incorporates fixes for the following APARs:

 * IV38115: Prevent gateway from crashing when an unexpected journal KeyField is processed. 
 * IV61582: Prevent gateway crashing when Gate.TSRM.FilterRepeatedValues is enabled. 
 * IV69306: Gateway for TSRM core dumps with an empty props file.

Netcool/OMNIbus V8.1

If you are running Netcool/OMNIbus V8.1, a single package contains versions of the gateway for all supported Operating Systems. To obtain this new package, use the Find by part number field to search for part number: CN4FTEN
