Title: IBM IC91310: FOR 4195 DATAPOWER APPLIANCES, AN ATTEMPT TO COPY A SCRYPT IMAGE FILE ONTO A DEVICE MAY FAIL DURING FIRMWARE UPGRADE - United States

Text:
 SUBSCRIBE
You can track all active APARs for this component.



APAR STATUS
 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  For 4195 DataPower Appliances, an attempt to copy a scrypt
   image file onto a device could fail
   during firmware upgrade due to insufficient ramdisk allocation
   accommodate the upgrade.
   
   
    
   
   

LOCAL FIX

PROBLEM SUMMARY
 *  Affected is 4195 firmware upgrade.
   
   Upgrade may fail due to insufficient ramdisk allocation. Fix
   increases the size of temporary:// and image:// for improved
   compatibility with future firmware.
   
   
   APAR IC92476 [http://www-01.ibm.com/support/docview.wss?uid=swg1IC92476] affects Virtual DataPower appliance upgrades for
   the same problem.
   
   
    
   
   

PROBLEM CONCLUSION
 *  The fix is available in 4.0.1.17, 4.0.2.13 and 5.0.0.8.
   
   For a list of the latest fix packs available, please see:
   http://www-01.ibm.com/support/docview.wss?uid=swg21237631 [http://www-01.ibm.com/support/docview.wss?uid=swg21237631]
   
   
    
   
   

TEMPORARY FIX
