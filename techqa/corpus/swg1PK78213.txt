APAR STATUS
 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  If logout is configured to use a custom redirect url, a session
   timeout does not redirect to the session timeout error page
   (ErrorSessionTimeout.jsp) anymore. A session timeout should
   lead
   in all cases to the session timeout error page.

 *  If logout is configured to use a custom redirect url, a session
   timeout does not redirect to the session timeout error page

   in all cases to the session timeout error page.
