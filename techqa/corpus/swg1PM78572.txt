PROBLEM SUMMARY
 *  ****************************************************************
   * USERS AFFECTED:  All users of IBM WebSphere Application      *
   *                  Server on Solaris platform                  *
   ****************************************************************
   * PROBLEM DESCRIPTION: SSL connection failure after            *
   *                      upgrading to 7.0.0.23, 8.0.0.3 and      *
   *                      8.5.0.0 due to Elliptical Curve         *
   *                      Cryptography (ECC) suites.              *
   ****************************************************************
   * RECOMMENDATION:                                              *
   ****************************************************************
   This APAR is PM68915 [http://www-01.ibm.com/support/docview.wss?uid=swg1PM68915] (for IBM JVM) equivalent fix for Sun JVM.
   When PM68915 [http://www-01.ibm.com/support/docview.wss?uid=swg1PM68915] was created, there was no issue found with Sun
   JVM. However, as Sun JVM upgrades, we started see the same
   issue.  This APAR provides the same fix for WebSphere
   Application Server running with Sun JVM.
   Please refer to
   http://www-01.ibm.com/support/docview.wss?uid=swg1PM68915 [http://www-01.ibm.com/support/docview.wss?uid=swg1PM68915]
   for details of this APAR.
