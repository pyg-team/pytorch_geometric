Rule Team Server DOWNLOADABLE FILES

ABSTRACT
 This fix pack updates Rule Team Server V7.1 (7.1.0, 7.1.1, 7.1.1.1 or 7.1.1.3) to V7.1.1.4. 

DOWNLOAD DESCRIPTION
This fix pack updates Rule Team Server for Java and Rule Team Server for .NET: 

 * Rule Team Server for Java: This release updates V7.1.1 (7.1.1, 7.1.1.1 or 7.1.1.3) to V7.1.1.4. It does not update earlier versions of Rule Team Server. Users of V7.0 or V7.1.0 should update to V7.1.1 in order to install V7.1.1.4. The update is supplied as an incremental installer on Passport Advantage and Fix Central. 
 *  Rule Team Server for .NET: This release updates earlier versions to V7.1.1.4. The update is available as a full product installer on Passport Advantage.


For details on problems fixed in Fix Pack 4, refer to the Fix list for V7.1 [http://www-01.ibm.com/support/docview.wss?uid=swg27019171#7114] document. 

This fix pack is entitled. Only customers who have purchased Rule Team Server V7.1 can download this fix pack. PREREQUISITES
Java customers: WebSphere ILOG Rule Team Server for Java V7.1.1

INSTALLATION INSTRUCTIONS
Execute the product installer.
