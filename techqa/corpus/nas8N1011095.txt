Resolution: 
This type of error can be caused when the OSGI cache becomes damaged. The OsgiCfgInit script can be used to clean the cache. 

The instructions for WAS v7 Express are as follows:

1. Issue the STRQSH command.
2. Type cd /qibm/proddata/websphere/appserver/v7/express/bin
3. osgicfgInit

You should receive a message that the cache was cleaned.
