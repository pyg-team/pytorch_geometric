Title: IBM Security Bulletin: IBM WebSphere MQ vulnerability in CHLAUTH with adopted credentials (CVE-2014-4793) - United States

Text:
 SECURITY BULLETIN

SUMMARY
 There is a vulnerability in IBM WebSphere MQ 8.0.0.0 where CHLAUTH rules do not correctly block client connections that match a rule after adopting context using CONNAUTH. 

VULNERABILITY DETAILS
CVE-ID: CVE-2014-4793 [http://cve.mitre.org/cgi-bin/cvename.cgi?name=CVE-2014-4793]
CVSS Base Score: 3.5
CVSS Temporal Score: See http://xforce.iss.net/xforce/xfdb/95208 [http://xforce.iss.net/xforce/xfdb/95208] for the current score
CVSS Environmental Score*: Undefined
CVSS Vector: (AV:N/AC:M/Au:S/C:N/I:P/A:N)

IBM WebSphere MQ could allow an authenticated attacker to bypass security restrictions and access a queue manager that they should not have privileges for.

AFFECTED PRODUCTS AND VERSIONS
IBM WebSphere MQ V8.0.0.0

REMEDIATION/FIXES
For IBM WebSphere MQ for distributed platforms, apply Fix Pack 8.0.0.1 [http://www-01.ibm.com/support/docview.wss?uid=swg21682233] or later. 

For IBM WebSphere MQ for z/OS apply PI19991 [http://www-01.ibm.com/support/docview.wss?uid=swg1PI19991].

WORKAROUNDS AND MITIGATIONS
None known
