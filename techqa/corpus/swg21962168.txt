QUESTION
 imcl : How can you install a probe or gateway package? 

CAUSE
Netcool/OMNIbus 8.1 uses the Installation Manager to install packages

ANSWER
 

The Installation Manager command imcl can be used to install downloads integration packages. 

 

 * To determine the location of imcl use the nco_id command as the Netcool/OMNIbus user:



nco_id

 * Set IMHOME to use the imcl command 

export IMHOME 
IMHOME=/opt/nrv81/IBM/InstallationManager/eclipse 
 * List the installed packages using these commands


$IMHOME/tools/imcl listInstalledPackages -features -long 
$IMHOME/tools/imcl listInstalledPackages listInstalledPackages 
 * Install probes and gateway packages by defining the dot product name and package location


$IMHOME/tools/imcl -c install com.ibm.tivoli.omnibus.integrations. nco-p-socket -repositories 
./ im-nco-p-socket-10_0.zip -installationDirectory /opt/nrv81/tivoli/netcool 
 * Check the product [nco_p_socket] is installation using -version


$NCHOME/omnibus/probes/nco_p_socket -version
