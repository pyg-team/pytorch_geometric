PROBLEM SUMMARY
 *  ****************************************************************
   * USERS AFFECTED:                                              *
   ****************************************************************
   * PROBLEM DESCRIPTION:                                         *
   ****************************************************************
   * RECOMMENDATION:                                              *
   ****************************************************************
   The command line -exportInstallData option does not work. The
   following workarounds are possible.
   
   1. Use the Installation Manager in the graphical user interface
   mode and chose Help ? Export Data for Problem Analysis.
   
   2. Update the Installation Manager to version 1.4 or later.
   Verify that the File ? Prefernces ? Updates 'Search for
   Installation Manager updates' is checked.  Verify that File ?
   Preferences ? Repositories ? 'Search for service repositories
   during install and update' is checked.  Return to the main panel
   and click the 'Update' button to update the IM to the latest
   version.
   
   3. Zip the contents of the following folder: C:\Documents and
   Settings\All Users\Application Data\IBM\Installation Manager
   manually.
   
   
    
   
   

PROBLEM CONCLUSION
 *  This problem has been resolved in Installation Manager version 1
   .4 and later.
