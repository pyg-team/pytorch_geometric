 TECHNOTE (TROUBLESHOOTING)

PROBLEM(ABSTRACT)

If user clicks OK, the "Mapping Table" screen appears successfully.

CAUSE
Code production issue (defect) in Controller 8.5.x.

The error is triggered when the mapping table - view contains job names longer than 50 characters.


RESOLVING THE PROBLEM
Fix: 

Upgrade to Controller 10.1 or later.

Workaround 

 * Click "Maintain - Jobs - Define". 
 * Select the jobs with long names, and rename them to be shorter than 50 characters.
