APAR STATUS
 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  This APAR adds new function to allow selection of either MQ Base
   or MQ Advanced when using the IBM MQ Virtual System Pattern Type
   to create a virtual system pattern on the IBM PureApplication
   System.
   
   
    
   
   

LOCAL FIX

PROBLEM SUMMARY
 *  ****************************************************************
   USERS AFFECTED:
   This affects users of IBM Pure Application System
   
   
   Platforms affected:
   MultiPlatform
   
   ****************************************************************
   PROBLEM DESCRIPTION:
   Installation of the IBM MQ Advanced components was not optional
   and MQ Advanced was installed by default.
   
   
    
   
   

PROBLEM CONCLUSION
 *  IBM MQ Pattern now includes an option so that the user may elect
   not to install the IBM MQ Advanced components.
