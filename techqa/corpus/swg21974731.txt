Title: IBM NoClassDefFoundError while running LogAnalyzer - United States

Text:
 TECHNOTE (FAQ)

QUESTION
 We tried running the Performance Log Analyzer as per the following technote: http://www-01.ibm.com/support/docview.wss?uid=swg21969917 [http://www-01.ibm.com/support/docview.wss?uid=swg21969917]. But when we try to run the utility, we get the following java.lang.NoClassDefFoundError: com/ibm/misc/BASE64Encoder



Why are we getting this and how may we circumvent it? 

CAUSE
The prerequisite for the tool is IBM J9 JRE 1.7 and from the error, it seems the correct JRE version was not installed. Full error stack:
