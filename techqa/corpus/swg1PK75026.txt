ERROR DESCRIPTION
 *  When using Websphere Portal 6.0.1.4 to connect to WCM content
   running on WCM 6.1.0.0 using a WCM 6.1 Remote Rendering Portlet,
   you try to set or change the User Profile through the
   AJPECatSelect module and it fails with IWKPL1179X (unable to
   retreive category) and IWKPY1015X (Unauthorised access) errors
   shown in SystemOut.log file.

 *  CATEGORY ID NOT RECOGNIZED IN TAXONOMY WHEN ATTEMPTING TO UPDATE
   USER PROFILES, unauthorized access in the SystemOut.log

 *  Changed the level of access that WCM uses to check access to the
   category.
