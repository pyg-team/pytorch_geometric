ERROR DESCRIPTION
 *  Description:
   
   The OpenPages documentation currently only includes information
   on configuring SSL for Cognos on Apache and IIS.  We need to
   include information for configuring SSL for Cognos with IBM HTTP
   Server (IHS).
   
   Page 523 of the OpenPages Administrators Guide starts the
   section for Configuration SSL with IIS/Apache on Cognos.  We
   need similar information in the guide for Cognos SSL on IHS.
   http://public.dhe.ibm.com/software/data/cognos/documentation/ope
   npages/en/7.1.0/administrators_guide.pdf [http://public.dhe.ibm.com/software/data/cognos/documentation/openpages/en/7.1.0/administrators_guide.pdf]
   
   Prerequisites:
   
   OpenPages 7.1 Administrators Guide
   
   Expected Results:
   
   If we support IHS we should have steps in the OpenPages
   documentation on how to configure SSL with this product.  This
   information is available for both IIS and Apache already.
   
   Actual Results:
   
   There is no information related to configuring SSL on IHS for
   Cognos.
   
   
    
   
   

LOCAL FIX
 *  Workaround:
   
   There are two pieces of documentation available that go over
   configuration Cognos with SSL on IHS:
   http://www-01.ibm.com/support/docview.wss?uid=swg21179559 [http://www-01.ibm.com/support/docview.wss?uid=swg21179559]
   
   http://www-01.ibm.com/support/knowledgecenter/SSEP7J_10.2.1/com. [http://www-01.ibm.com/support/knowledgecenter/SSEP7J_10.2.1/com.]
   ibm.swg.ba.cognos.inst_cr_winux.10.2.1.1.doc/c_configuringthessl
   protocol.html%23ConfiguringtheSSLProtocol
