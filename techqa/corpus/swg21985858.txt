Title: IBM Security Bulletin: XML External Entities Injection Vulnerability in IBM Traveler (CVE-2016-3039) - United States

Text:
 SECURITY BULLETIN

SUMMARY
 IBM Traveler is vulnerable to a denial of service caused by an XML External Entity Injection (XXE) error when processing XML data. 

VULNERABILITY DETAILS
 

CVEID: CVE-2016-3039 [http://cve.mitre.org/cgi-bin/cvename.cgi?name=CVE-2016-3039] 
DESCRIPTION: IBM Traveler is vulnerable to a denial of service, caused by an XML External Entity Injection (XXE) error when processing XML data. A remote attacker could exploit this vulnerability to expose highly sensitive information or consume all available memory resources.

AFFECTED PRODUCTS AND VERSIONS
 IBM Traveler 8.5.3 
IBM Traveler 9.0
IBM Traveler 9.0.1 

REMEDIATION/FIXES
Traveler 9.0.1.12 and later versions have been updated to address the vulnerability. See APAR LO89357 [http://www.ibm.com/support/docview.wss?uid=swg21700212] for details. It is recommended that all Traveler servers be upgraded to Traveler 9.0.1.12 or later. Refer to the following technote for the latest Traveler maintenance: Index of recommended maintenance for IBM Traveler [http://www-01.ibm.com/support/docview.wss?uid=swg24019529] 


For versions no longer supported such as IBM Traveler 8.0.1, 8.5.0, 8.5.1 and 8.5.2, IBM recommends upgrading to a fixed, supported version of the product.

WORKAROUNDS AND MITIGATIONS
