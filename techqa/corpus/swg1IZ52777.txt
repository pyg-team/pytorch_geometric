LOCAL FIX
 *  export LANG=C, then execute the tacmd command.
   
   
    
   
   

PROBLEM SUMMARY
 *  The tacmd bulkexportsit command crashed with situations having
   Japanese characters.  This issue occured when setting
   LANG=ja_JP.utf8 on Linux systems only.
   
   
    
   
   

PROBLEM CONCLUSION
 *  The buffer size was adjusted to allow for the Japanese character
   set.
