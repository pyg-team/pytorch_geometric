Title: IBM NullPointerException when updating  Installation Manager from 1.1.1 to 1.4.2 - United States

Text:
NullPointerException; NPE; com.ibm.cic.agent.internal.core.Director.computeIUsForPrepare; ERROR; PM30992 TECHNOTE (TROUBLESHOOTING)

PROBLEM(ABSTRACT)
 When using IBM Installation Manager 1.1.1 to Update an existing product, Installation Manager prompts you that it needs to update itself to the latest version. When it attempts to update to IM 1.4.2, you receive a NullPointerException.

CAUSE
This has been identified as a product defect under APAR PM30992



RESOLVING THE PROBLEM
Until APAR PM30992 is resolved, you will need to install version 1.4.2 manually. 

Follow the steps under the "Resolving the Problem" section of the technote 1290142: What to do when IBM Installation Manager fails to startup or hangs or is unresponsive [http://www.ibm.com/support/docview.wss?uid=swg21290142].
