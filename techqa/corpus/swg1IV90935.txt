ERROR DESCRIPTION
 *  The Web applications do not connect to the backup ITNM server
   when it becomes primary during failover.
   Some Web applications, such as the MIB Browser and MIB Grapher,
   do not function when the backup server is in primary mode.
   
   This limitation exsits in ITNM v3.8. still hold true for 3.9 -
   4.2, but missing in the official Document.
   
   The change has already been made in the 4.2 FP2 documentation
   and will be visible once that goes live.
   
   Need Docs team to have it added to 3.9, 4.1.1,4.2 Document.
