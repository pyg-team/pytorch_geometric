Title: IBM IZ58684: SELF SERVICE SEARCH SOLUTIONS APPLICATION, SEARCHING ON DATABASE RESERVED TERMS SUCH AS NT, RT WILL RESULT IN AN ORA-20000 ERROR - United States

APAR STATUS
 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  In the Search Solutions application under Self Service Module if
   you search using Oracle reserved words it will result in a
   System Message Box pointing to an ORA-20000: Oracle Text error
   this message box can not be cleared, you must close
   the browser to clear message box.
   
   
   Go To - Self Service -> Service Requests -> Search Solution
   application.
   
   In the Solution Description Field type -> Windows NT
   
   Results in:
   
   BMXAA4210E - Database error number 29902 has occurred.
   ORA-29902: error in executing ODCIIndexStart() routine
   ORA-20000: Oracle Text error:
   DRG-50901: text query parser syntax error on line 1, column 8
   
   When you click OK nothing happens you have to close the
   browser to continue.
   
   
    
   
   

LOCAL FIX
 *  Use Quoted on any of the reserved terms.
   
   Instead of searching for Windows NT, use Windows "NT"
