ILOG JRules DOWNLOADABLE FILES

ABSTRACT
 This document describes how to download WebSphere ILOG JRules V7.1.1.4 eAssembly images from IBM Passport Advantage Online. 

DOWNLOAD DESCRIPTION


WebSphere ILOG JRules is a business rule management system (BRMS) that enables both business users and developers to manage the rules that drive their business. 

Note

V7.1.1.4 is not compatible with JRules V7.0.x and V7.1.0. Users of these versions should update to V7.1.1 before installing V7.1.1.4. 

Contents

 This document provides a list of the eAssembly images for JRules V7.1.1 and V7.1.1.4, and describes how to download the images from IBM Passport Advantage Online. 

More information

 For further information about this product, see the WebSphere ILOG JRules home page [http://www.ibm.com/software/integration/business-rule-management/jrules/support/index.html]. 

Downloading and assembling the product

 1. Navigate your browser to the Find Downloads and Media [http://www.ibm.com/software/howtobuy/passportadvantage/paocustomer/sdma/SDMA?PO=DOWNLOAD_FIND_BY_PART_NUMBER] page on IBM Passport Advantage Online.
 2. Sign in using your IBM ID.
 3. Select the Find by part number search option under Download finder options.
 4. Search for each downloadable image by part number (parts are listed below).
 5. Download the parts that you require. For each V7.1.1 part, download the matching V7.1.1.4 part in order to install V7.1.1.4. 
 6. Unpack each of the parts into a separate directory on your system.
 7. Follow the instructions in the Quick Start Guide (CZLX4ML) to install the product.
