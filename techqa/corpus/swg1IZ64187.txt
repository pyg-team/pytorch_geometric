ERROR DESCRIPTION
 *  Product: Exchange Agent
   Release: 6.2.0
   Severity: 2
   Approver: Steven Barnett
   Compid: 5724B98EO
   
   
   In Exchange Agent under reachability workspace the values in
   
   reachability test runtime and server time Stamp does not match.
   The
   value in
   "Reachability test runtime" is retrieved in GMT format where as
   the
   server
   "Time Stamp" value is retrieved in Local time format.
   
   Detailed Recreation Procedure:
   Install and configure Microsoft Exchange Monitoring agent 6.2.0.
   Select Server navigation item and then open Reachability
   workspace.
   Observe the values of reachability test run time field and
   timestamp
   field under Reachability workspace.
   
   Related Files and Output:
   We can also Observe the output in
   MailBox_Reachability_Results.log
   at the location <%CANDLE_HOME%>\IBM\ITM\TMAITM6\logs.
