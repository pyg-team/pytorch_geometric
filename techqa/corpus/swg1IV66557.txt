APAR STATUS
 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  After running migrate-export many times over a period of time,
   users are
   unable to log in to the portal client. The portal server log
   shows many
   messages like the following:
   
   ERRMSG: Warning: 100 active connections. Retry #4 on datasource
   'TEPS'
   
   RECREATE INSTRUCTIONS:
   - start the portal server
   - run the migrate-export script
   - from db2 prompt, enter 'list applications'
   - result will show open connections from KfwServices
   - run migrate-export script again
   - run 'list applications' again
   - result will show an increase of one in open connections from
   KfwServices
   - if run 100 times, the 100 active connections error will appear
   
   
    
   
   

LOCAL FIX

PROBLEM SUMMARY
 *  Users are unable to log in to the portal client.  The portal
   server log shows many messages similar to the following:
   
   ERRMSG: Warning: 100 active connections.  Retry #4 on datasource
   'TEPS'
   
   The problem may start during an execution of the migrate-export
   script.
   
   
    
   
   

PROBLEM CONCLUSION
 *  Fixed an issue where the portal server was not releasing
   connections to the portal server database.
   
   The fix for this APAR is contained in the following maintenance
   packages:
