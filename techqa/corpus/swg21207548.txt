arsdoc; arsdoc query; query; -I; requires an argument TECHNOTE (FAQ)

QUESTION
 When using arsdoc query -I to retrieve load IDs, I get the following error message: Option >-I< requires an argument. 

ANSWER
Starting in Version 8.3 of IBM® DB2® Content Manager OnDemand, the arsdoc query parameter, -I, requires you to specify whether to search or create a load ID:
-I <type> Append the LoadId to each record in the file
f Search the System Log for the LoadId
p Create the LoadId from document information

For example, you might enter the following query:

arsdoc query -h ARCHIVE -u admin -p password -f "Credit Card Statements" -i "where account = '000-000-000'" -I p
