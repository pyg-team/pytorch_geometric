Title: IBM PureApplication System is affected by security vulnerabilities (CVE-2015-0197, CVE-2015-0198, CVE-2015-0199) - United States

Text:
 SECURITY BULLETIN

SUMMARY
 Security vulnerabilities have been identified in General Parallel File System V4.1, V3.5, and V3.4 which affect IBM PureApplication System. 

VULNERABILITY DETAILS
CVEID: CVE-2015-0197 [http://cve.mitre.org/cgi-bin/cvename.cgi?name=CVE-2015-0197]
DESCRIPTION: IBM General Parallel File System could allow a local attacker which only has a non-privileged account to execute programs with root privileges.
CVSS Base Score: 6.9
CVSS Temporal Score: See https://exchange.xforce.ibmcloud.com/vulnerabilities/101224 [https://exchange.xforce.ibmcloud.com/vulnerabilities/101224] for the current score
CVSS Environmental Score*: Undefined
CVSS Vector: (AV:L/AC:M/Au:N/C:C/I:C/A:C)

CVEID: CVE-2015-0198 [http://cve.mitre.org/cgi-bin/cvename.cgi?name=CVE-2015-0198]
DESCRIPTION: IBM General Parallel File System may not properly authenticate network requests and could allow an attacker to execute programs remotely with root privileges.
CVSS Base Score: 9.3
CVSS Temporal Score: See https://exchange.xforce.ibmcloud.com/vulnerabilities/101225 [https://exchange.xforce.ibmcloud.com/vulnerabilities/101225] for the current score
CVSS Environmental Score*: Undefined
CVSS Vector: (AV:N/AC:M/Au:N/C:C/I:C/A:C)

CVEID: CVE-2015-0199 [http://cve.mitre.org/cgi-bin/cvename.cgi?name=CVE-2015-0199]
DESCRIPTION: IBM General Parallel File System allows attackers to cause kernel memory corruption by issuing specific ioctl calls to a character device provided by the mmfslinux kernel module and cause a denial of service.
CVSS Base Score: 6.9
CVSS Temporal Score: See https://exchange.xforce.ibmcloud.com/vulnerabilities/101226 [https://exchange.xforce.ibmcloud.com/vulnerabilities/101226] for the current score
CVSS Environmental Score*: Undefined
CVSS Vector: (AV:L/AC:M/Au:N/C:C/I:C/A:C)

AFFECTED PRODUCTS AND VERSIONS
This vulnerability affects IBM PureApplication Server and the GPFS pattern type included in the following:

IBM PureApplication System V2.0
IBM PureApplication System V2.1

The solution is to upgrade the IBM PureApplication System to the following fix levels and update deployments that implement the GPFS pattern type using the GPFS Pattern Interim Fix:
