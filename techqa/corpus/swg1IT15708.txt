APAR STATUS
 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  The mq-qm object fails with MQ error of 2009/2059 when SSL
   connections are used in the mq-qm configurations involving slow
   networks between appliance and MQ server.
   
   
    
   
   

LOCAL FIX
 *  Disable SSL connections.
   
   
    
   
   

PROBLEM SUMMARY
 *  Affected is use of MQ Queue Manager object with SSL.
   
   MQ-QM may fail to work in slow networks.
   
   
    
   
   

PROBLEM CONCLUSION
 *  Fix is available in 7.0.0.14, 7.1.0.11, 7.2.0.8, 7.5.0.2 and
   7.5.1.2
   
   For a list of the latest fix packs available, please see:
   http://www-01.ibm.com/support/docview.wss?uid=swg21237631 [http://www-01.ibm.com/support/docview.wss?uid=swg21237631]
