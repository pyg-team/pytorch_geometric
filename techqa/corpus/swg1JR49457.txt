Title: IBM JR49457: YOU CANNOT SELECT THE INSTANCES WIDGET TEXT WHEN YOU USE CHROME OR SAFARI - United States

APAR STATUS
 * CLOSED AS FIXED IF NEXT.
    
   
   

ERROR DESCRIPTION
 *  When you try to use the Chrome or Safari browsers with the
   Instances widget, you cannot select the contents of a data cell,
    which prevents you from copying business data to an external
   application because you cannot select the contents of the cell.
   
   PRODUCT AFFECTED:
   IBM Business Monitor
   
   
    
   
   

LOCAL FIX
 *  Use another browser, such as Firefox, to select and copy text
   from the Instances widget to an external application.
   
   
    
   
   

PROBLEM SUMMARY
 *  No additional information is available.
   
   
    
   
   

PROBLEM CONCLUSION
 *  A fix is included in IBM Business Monitor V8.0.1 Fix Pack 3
   (V8.0.1.3) and Business Monitor V8.5.5.0 that enables you to
   select and copy Instances widget text when you use either the
   Chrome or Safari browser.
   
   
    
   
   

TEMPORARY FIX
 *  Not applicable
