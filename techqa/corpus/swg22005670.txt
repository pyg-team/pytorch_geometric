Title: IBM Security Bulletin: Content Collector for Email is affected by an Incorrect redirection for an OAuth authorization request in WebSphere Application Server both Traditional and Liberty - United States

Text:
 SECURITY BULLETIN

SUMMARY
 Content Collector for Email has addressed the following vulnerability. IBM WebSphere Application Server is vulnerable to cross-site request forgery which could allow an attacker to execute malicious and unauthorized actions transmitted from a user that the website trusts. 

VULNERABILITY DETAILS
CVEID:CVE-2017-1194 [http://cve.mitre.org/cgi-bin/cvename.cgi?name=CVE-2017-1194]
DESCRIPTION:IBM WebSphere Application Server is vulnerable to cross-site request forgery which could allow an attacker to execute malicious and unauthorized actions transmitted from a user that the website trusts.
CVSS Base Score: 4.3
CVSS Temporal Score: Seehttps://exchange.xforce.ibmcloud.com/vulnerabilities/123669 [https://exchange.xforce.ibmcloud.com/vulnerabilities/123669]for the current score
CVSS Environmental Score*: Undefined
CVSS Vector: (CVSS:3.0/AV:N/AC:L/PR:N/UI:R/S:U/C:N/I:L/A:N) 

AFFECTED PRODUCTS AND VERSIONS
IBM Content Collector for Email v3.0
IBM Content Collector for Email v4.0
IBM Content Collector for Email v4.0.1

REMEDIATION/FIXES
 

Product VRM Remediation IBM Content Collector for Email 3.0 Use IBM Content Collector for Email 4.0.1.6 Interim Fix 001 [https://www-945.ibm.com/support/fixcentral/swg/selectFixes?product=ibm%2FInformation+Management%2FContent+Collector&fixids=4.0.1.5-IBM-ICC-IF001&source=SAR&function=fixId&parent=Enterprise%20Content%20Management] IBM Content Collector for Email 4.0 Use IBM Content Collector for Email 4.0.1.6 Interim Fix 001 [https://www-945.ibm.com/support/fixcentral/swg/selectFixes?product=ibm%2FInformation+Management%2FContent+Collector&fixids=4.0.1.5-IBM-ICC-IF001&source=SAR&function=fixId&parent=Enterprise%20Content%20Management] IBM Content Collector for Email 4.0.1 Use IBM Content Collector for Email 4.0.1.6 Interim Fix 001 [https://www-945.ibm.com/support/fixcentral/swg/selectFixes?product=ibm%2FInformation+Management%2FContent+Collector&fixids=4.0.1.5-IBM-ICC-IF001&source=SAR&function=fixId&parent=Enterprise%20Content%20Management] Follow the steps in the readme file in the 4.0.1.6 interim fix 001 to install the interim fix applicable to your version.
