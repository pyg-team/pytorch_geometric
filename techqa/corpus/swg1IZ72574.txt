LOCAL FIX

PROBLEM SUMMARY
 *  Intermittent problems with events that are lost between IBM
   Tivoli Monitoring and the EIF probe.  Usually the events are
   correctly sent to the Object Server, but some events seem to get
   lost between management server and the EIF probe.
   
   
    
   
   

PROBLEM CONCLUSION
 *  The issue was caused by an outdated TEC EIF SDK which was
   missing the most current maintenance.  The code was modified to
   link with the current TEC EIF SDK.
   
   The fix for this APAR is contained in the following maintenance
   packages:
   
     | fix pack | 6.2.2-TIV-ITM-FP0003
