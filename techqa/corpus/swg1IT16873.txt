APAR STATUS
 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  DataPower intermittent and unexpected restart with MQ sharing
   conversation. It is timing related issue and does not happen all
   the time.
   It is also triggered when DataPower enabling sharing
   conversation but MQ server disable sharing conversation.
   
   
    
   
   

LOCAL FIX

PROBLEM SUMMARY
 *  Affected is the use of MQ sharing conversation.
   
   When disabling
   sharing conversation in MQ server but enabling
   sharing conversation in DataPower, the appliance might be
   reloaded.
   
   
    
   
   

PROBLEM CONCLUSION
 *  Fix is available in 7.0.0.16, 7.1.0.13, 7.2.0.10, 7.5.0.4,
   7.5.1.3 and 7.5.2.1
   
   For a list of the latest fix packs available, please see:
   http://www-01.ibm.com/support/docview.wss?uid=swg21237631 [http://www-01.ibm.com/support/docview.wss?uid=swg21237631]
