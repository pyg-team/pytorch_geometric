APAR STATUS
 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  The "tacmd Help" file contains incorrect information about some
   tacmd commands that are not available until ITM version 6.3.
   The file contains information about the following commands, but
   these commands are not valid in any ITM version prior to 6.3:
   
   tacmd addsdainstalloptions
   tacmd deletesdainstalloptions
   tacmd editsdainstalloptions
   tacmd listsdainstalloptions
   
   
    
   
   

LOCAL FIX
 *  Do not use these commands prior to ITM 6.3.
   
   
    
   
   

PROBLEM SUMMARY
 *  The tacmd "help" option displays information about commands that
   are not available until IBM Tivoli Monitoring version 6.30.  The
   "help" option displays information about the following commands,
   but these commands are not supported in IBM Tivoli Monitoring
   releases prior to 6.30:
   
   tacmd addsdainstalloptions
   tacmd deletesdainstalloptions
   tacmd editsdainstalloptions
   tacmd listsdainstalloptions
   
   
    
   
   

PROBLEM CONCLUSION
 *  References to these commands have been removed from tacmd help
   file.
   
   The fix for this APAR is contained in the following maintenance
   packages:
   
     | fix pack | 6.2.3-TIV-ITM-FP0004
