Title: IBM Sterling Order Management application deployment causes WebSphere Application Server admin console screen to go blank - United States

Text:
WAS; multiple; jar TECHNOTE (TROUBLESHOOTING)

PROBLEM(ABSTRACT)
 Attempts to deploy IBM Sterling Order Management application causes the WebSphere Application Server admin console right panel to go blank. 

SYMPTOM
On clicking any option on the left side panel of WAS Admin console you can see a white screen on right side on Admin console. The below exception is recorded in WebSphere Application Server SystemOut.log

CAUSE
Conflicting version of JAR are used by WebSphere Application Server Admin console application and Sterling OMS application.



DIAGNOSING THE PROBLEM
WAS application server admin console works fine without Sterling Application.



RESOLVING THE PROBLEM
To resolve, set the server class loader policy to Multiple and restart the application server.
