1) I stopped the DM engine on the endpoint ( Domino server ). 

wdmcmd -e os390 -stop 

2) Deleted the LCFNEW folder on the endpoint \LCF\DAT\1 folder. 

3) Reconfigured my DM profile Ignore Task as follow: 


Ignore Tasks = 
Task_Name = ntask 
Task_nice_name = SAV Process 


Same as before.

on the endpoint before it can receive a new profile. Otherwise; it keeps using the old profile.
