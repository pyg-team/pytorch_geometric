ERROR DESCRIPTION
 *  After upgrading ICC 2.2 to ICC 4.0.1 FP4,  folder search doesn't
   work even after following the instructions properly.
   Search works only after disabling customlabel.properties file
   and manually tying folder name.
   When user select Inbox from Folder dropdown, it returns zero
   results.  Whereas, when user types "Inbox" in the Folder
   dropdown, it  returns some results.
   The setting for folder normalized is missed during the upgrade
   process.
   
   The problem was detected initially while working with the
   following environment:
   ICC 4.0.1 FP4
   ICC OE 4.0.1 FP4
   CE 5.1.0.4 (5.1 Fix Pack 4)
    ICC CSSS 4.0.1.3
