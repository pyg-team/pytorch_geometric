APAR STATUS
 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  When conducting a firmware upgrade, if the internal space is
   exhausted, the display *Validation failed* as well as 'Firmware
   validation error'.
   
   
    
   
   

LOCAL FIX
 *  A full device reboot is necessary to resolve the internal space
   exhaustion (co;flash;shutdown reboot)
   
   
    
   
   

PROBLEM SUMMARY
 *  Affected is firmware upgrade.
   In some instances, attempting to upgrade firmware may produce an
   error of "validation failed" instead of the correct error
   message.
   
   
    
   
   

PROBLEM CONCLUSION
 *  Fix is available in 7.0.0.14, 7.1.0.11, 7.2.0.8, 7.5.0.2 and
   7.5.1.0
   For a list of the latest fix packs available, please see:
   http://www-01.ibm.com/support/docview.wss?uid=swg21237631 [http://www-01.ibm.com/support/docview.wss?uid=swg21237631]
