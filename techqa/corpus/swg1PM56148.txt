LOCAL FIX
 *  Local Fix&#65496;
   Edit the jcr.install_cfg.xml file located in the following
   directory:
   
   <PortalServer_root>/jcr/wp.content.repository.install/config/inc
   ludes/
   
   Change the following line (line 1324) from:
   
   <setvariable name="jcr.database.schema" value="${jcr.dbname}" />
   
   to:
   
   <setvariable name="jcr.database.schema" value="${jcr.DbSchema}"
   />
   
   
   Also, set the correct value in icm.properties for jcr.Schema.
   Then rerun the upgrade.
