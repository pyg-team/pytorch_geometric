PROBLEM SUMMARY
 *  ****************************************************************
   USERS AFFECTED:
   Users of the IBM MQ .NET v8 Redistributable client
   
   
   Platforms affected:
   Windows
   
   ****************************************************************
   PROBLEM DESCRIPTION:
   The logic to locate the MQ .NET runtime library (amqmdnet.dll)
   when running a .NET application has changed with
   resdistributable clients. A coding error within this change
   meant that the MQ .NET client was unable to locate runtime
   library when the environment was not associated with a full MQ
   installation.
   
   
    
   
   

PROBLEM CONCLUSION
 *  The MQ .NET redistributable client has been updated to correctly
   locate the runtime library from the client directory if a full
   MQ installation is not present.
