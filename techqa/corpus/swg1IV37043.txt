ERROR DESCRIPTION
 *  ENVIRONMENT: TDWC 8.6 FP2 and TWS 8.6 FP2 on the same machine in
   separate eWAS servers.
   
   Recreate:
   1. Login to TDWC's Self Service Catalog:
   https://<hostname>:<WC_defaulthost_secure_port>/TWSWebUI/mobile [https://<hostname>:<WC_defaulthost_secure_port>/TWSWebUI/mobile]
   
   Attempt to submit a service.  If the submit button is active and
   you're able to submit then goto step 2.  Otherwise you should
   see that there is a pop up with "* Error message" and a [Close]
   button.
   
   2. Login to TIP as the same user that was logged into the SSC.
   Then logout.
   
   3. Attempt to submit a service in the TWSWebUI/mobile
   session and there should be a pop up with "* Error message".
   
   Workaround:
   1. Login to TIP and access any task under "Tivoli Workload
   Scheduler"
   2. Go back to SSC session with the TIP session still open and
   attempt to submit.  The submit button should be available.
