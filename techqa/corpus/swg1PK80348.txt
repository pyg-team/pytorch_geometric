APAR STATUS
 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  When a RMI / IIOP Client attempts to connect to WebSphere with
   a REVOKED userid or EXPIRED password, the following CORBA error
   is sent back to the client:
   .
   org.omg.CORBA.NO_PERMISSION:   vmcid: 0x49424000  minor code:
   300 completed: Maybe
   
   RACF returns to WebSphere specific reson codes for revoked id or
   expired password, so WebSphere should be able to pass this info
   to the client
   
   Expired password:
   SAF product returnCode=8,
   SAF product reasonCode=24.
   
   Revoked userid:
   SAF product returnCode=8,
   SAF product reasonCode=28.

PROBLEM SUMMARY
 *  ****************************************************************
   * USERS AFFECTED: All users of WebSphere Application Server    *
   *                 V7.0 for z/OS                                *
   ****************************************************************
   * PROBLEM DESCRIPTION: A couple of CORBA minor codes are       *
   *                      introduced to indicate more granular    *
   *                      cause of login error.                   *
   ****************************************************************
   * RECOMMENDATION:                                              *
   ****************************************************************
   Following new CORBA minor codes are implemented when Local OS
   user registry (System Authorization Facility (SAF) user
   registry) is used.
   
   Minor code value (in hex):  0x49424310
   Minor code name:            RevokedUserid
   Exception type:             NO_PERMISSION
   Minor code description:     This code occurs when the user
   registry returns an error which indicates user id is revoked
   or user access to group is revoked.
   Retry performed:            Yes
   
   Minor code value (in hex):  0x49424311
   Minor code name:            ExpiredPassword
   Exception type:             NO_PERMISSION
   Minor code description:     This code occurs when the user
   registry returns an error which indicates password or password
   phrase is expired.
   Retry performed:            Yes
