Title: IBM PK20022: "WEBSPHERE V5.1 RUNTIME" IS MISSING SAS.JAR FILE WHICH IS PRESENT IN "WASV6.0 RUNTIME" , THIS CAUSES BUILD TIME ERRORS. - United States

Text:

APAR STATUS
 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  THe "Websphere v5.1runtime" library packaged by RAD Application
   server toolkit , is missing a file - sas.jar.
   If a project is targetted for WASv5.1or Portalv5.1 , and used
   any classes from this sas.jar file in its code ( like
   WSSEcurityException Class) it will get build path erros because
   this class cannot be resolved.
   
   
    
   
   

LOCAL FIX
 *  Manually add the Appserver\sas.jar in the project build path in
   RAD .
   
   
    
   
   

PROBLEM SUMMARY
 *  The Websphere v5.1 runtime classpath library does not
   contain th
   e sas.jar file.
   
   
    
   
   

PROBLEM CONCLUSION
 *  The sas.jar from the v5.1 WebSphere Test Environment is now
   included in the list of JAR files found on the WebSphere
   Application Server runtime classpath library.
   Delivered in Rational Application Developer, V6.0.1.1
   iFix002
