Title: IBM Security Bulletin: API Connect is affected by a Cross Frame Scripting vulnerability CVE-2017-1551 - United States

Text:
 SECURITY BULLETIN

SUMMARY
 API Connect has addressed the following vulnerability. 

IBM API Connect could allow a remote attacker to hijack the clicking action of the victim. 

VULNERABILITY DETAILS
CVEID: CVE-2017-1551 [http://cve.mitre.org/cgi-bin/cvename.cgi?name=CVE-2017-1551]
DESCRIPTION: IBM API Connect could allow a remote attacker to hijack the clicking action of the victim. By persuading a victim to visit a malicious Web site, a remote attacker could exploit this vulnerability to hijack the victim's click actions and possibly launch further attacks against the victim.
CVSS Base Score: 5.4
CVSS Temporal Score: See https://exchange.xforce.ibmcloud.com/vulnerabilities/131291 [https://exchange.xforce.ibmcloud.com/vulnerabilities/131291] for the current score
CVSS Environmental Score*: Undefined
CVSS Vector: (CVSS:3.0/AV:N/AC:L/PR:L/UI:R/S:C/C:L/I:L/A:N)
