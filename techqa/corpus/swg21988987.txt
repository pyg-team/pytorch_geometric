Title: IBM Security Bulletin: Multiple security vulnerabilities have been identified in IBM WebSphere Application Server shipped with WebSphere Sensor Events products (Java Technology Edition CPU July 2016 - Includes Oracle July 2016 CPU + CVE-2016-3485) - United States

Text:
 SECURITY BULLETIN

SUMMARY
 IBM WebSphere Application Server is shipped as a component of WebSphere Sensor Events and related products. Oracle released the July 2016 critical patch updates which contain multiple fixes for security vulnerabilities in the IBM Java Development Kit that is included with IBM WebSphere Application Server.
