ERROR DESCRIPTION
 *  Storage Devices Utilization view incorrectly shows several
   service queue and wait queue metrics as value of zero.  The
   problem is in the aixDataProvider and can be seen by tracing the
   data provider.
   
   ENVIRONMENT:
   
   IBM Tivoli Monitoring for System p - AIX Premium Agent 6.2.2.1
   
   
   
   RECREATE INSTRUCTIONS:
   Start the AIX Premium agent and view the
   Storage -> 'MPIO Storage Information' -> 'Storage Devices
   Utilization' view. Set the window refresh to every 60 seconds.
   Check for the service queue and wait queue metrics such as Avg
   Request in WaitQ MS', 'Min Request in WaitQ MS', 'Max Request in
   WaitQ MS', 'Avg WaitQ Size', 'Avg ServiceQ Size', and 'ServiceQ
   Full per Sec' . They would all show up as zeros.
   
   LOCAL FIX: None.
