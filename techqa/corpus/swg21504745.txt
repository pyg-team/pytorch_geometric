Title: IBM IE9 and Opera 11 display wrong height of Dojo Dialog - United States

Text:
Dojo; Opera; IE; Rich UI; RATLC01814535; RBD; 10089; RATLC01522750 TECHNOTE (TROUBLESHOOTING)

PROBLEM(ABSTRACT)
 Attempts to develop EGL rich UI applications with IBM Rational Business Developer and Dojo Dialog result in incorrect height when you use the mentioned browsers. 

CAUSE
Microsoft Internet Explorer 9 and Opera 11 are not yet supported Web browser. See Rational Business Developer, Version 8.0.1.1 release information [http://www.ibm.com/support/docview.wss?uid=swg27021152] and List of supported Web Browser [http://www-01.ibm.com/support/docview.wss?uid=swg27021056#Linux_Web_Browsers_ww]s. 

Only Dojo 1.7 supports these web browser, while EGL uses Dojo 1.5.


ENVIRONMENT
 

 * Microsoft Internet Explorer 9 
 * Opera 11

RESOLVING THE PROBLEM
The problem has been fixed in fixpack 8.0.1.2. This fix includes Dojo 1.6.1. Update to 8.0.1.2 or newer.
