APAR STATUS
 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  After configuring Portal with Oracle 11g, the Helper Class
   defined for in each of the WebSphere Appliction Server data
   sources is set for Oracle 9i.  It should be Oracle 11g.
   
   
    
   
   

LOCAL FIX
 *  Manually set the Helper Class after completing database-transfer
   
   
    
   
   

PROBLEM SUMMARY
 *  Database-transfer to Oracle 11g does not set up the Oracle 11g
   helper class for the datasources
   
   
    
   
   

PROBLEM CONCLUSION
 *  Updated scripts to set the Oracle 11g Helper classes for
   database-transfer and connect-database if Oracle 11g is being
   used.
