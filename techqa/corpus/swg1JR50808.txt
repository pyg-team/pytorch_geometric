APAR STATUS
 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  If the BPM install path, e.g. /opt/IBM/BPM/v85, contains the DE
   name, e.g. "BPM", the "BPMConfig -create -de xxx" command will
   fail when generating the DB upgrade sql files.
   
   
    
   
   

LOCAL FIX
 *  use a different Deployment Environment name
