ERROR DESCRIPTION
 *  When attempting to save a new Form Template Policy, 50
   warnings occur if document title in the base doc class is
   marked required
   Expanded description:
   In Enterprise Manager, select the Form Policy class property
   Select Property Definitions  in property tab
   Click Inherited Properties
   Select Document Title then do Edit
   Select More Tab
   Check the Value required check box.
   Log on Workplace
   Add a Form Template (ITX) file to the object store
   From Author Advanced Tools, create an ordinary document Entry
   Template
   From Author Advanced Tools, create a Form Template Policy
   In the policy wizard:
   select the ITX file added previously
   select the entry template added previously
   proceed to save the policy:
   select a folder, press next to get to the policy properties
   screen and...
   50 more browser windows will eventually popup on the screen,
   each complaining that the document title property is required
   but not been filled in.
