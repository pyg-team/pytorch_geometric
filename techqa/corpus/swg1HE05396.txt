Title: IBM HE05396: IBM Content Collector indexer for text search does not correctly process Notes items of type RFC822 text list - United States

IBM Content Collector Text Search Support 2.2.0.1 for IBM CommonStore [http://www-01.ibm.com/support/docview.wss?uid=swg24030014]

APAR STATUS
 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  For some Domino email, Notes items of type RFC822 text list are
   not processed correctly by the IBM Content Collector (ICC)
   Indexer, causing plain text content of these items to not be
   added to the full text index.
   
   
    
   
   

LOCAL FIX

PROBLEM SUMMARY
 *  For some Domino email, Notes items of type RFC822 text list were
   not processed correctly by the Content Collector indexer for
   text search. Therefore, plain text content of these items was
   not added to the full text index.
   
   
    
   
   

PROBLEM CONCLUSION
 *  Starting with this fix, the Content Collector indexer for text
   search processes Notes items of type RFC822 text list properly.
