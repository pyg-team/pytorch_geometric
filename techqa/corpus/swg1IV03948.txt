LOCAL FIX

PROBLEM SUMMARY
 *  After activating a new configuration profile generated by WSRR
   Studio, the following message appears in the web UI menu bar
   for MQ object types:
   
   "Error: Message Key not found"
   
   Also, the following exceptions appear in the SystemOut log:
   
   MissingResourceException: Can't find resource for bundle
   java.util.PropertyResourceBundle, key
   menubar.SM63.actions.create.MQ.Destination
   
   java.util.MissingResourceException: Can't find resource for
   bundle java.util.PropertyResourceBundle, key
   navigationtree.unified.SM63.concepts.Topic
   
   
    
   
   

PROBLEM CONCLUSION
 *  The GEP produced by WSRR Studio contained some missing resources
   and incorrect resource references.
   
   These have now been corrected and the fix is targeted for
   inclusion in fix pack 1 for WSRR v7.5.
