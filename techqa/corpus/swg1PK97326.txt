APAR STATUS
 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  Problem: When a fileResource is deleted from a Content on
   syndicator server, the deletion is not syndicated to the
   subscriber.
   
   This only affects deletions. If the fileResource is updated or
   replaced with a new fileResource, the change is syndicated as
   expected.
   
   Also, the content in which the fileResource is embedded will
   update as expected.
   
   
   There is no exception correlated to this issue.
   
   
    
   
   

LOCAL FIX
 *  NA
   
   
    
   
   

PROBLEM SUMMARY
 *  File and images resources attached to content when deleted
   on syndicator server are not deleted on subscriber server after
   syndication.
   
   
    
   
   

PROBLEM CONCLUSION
 *  Code added on subscriber to update controllables on subscriber
   server if file and image resources are removed on syndicator
   server.
