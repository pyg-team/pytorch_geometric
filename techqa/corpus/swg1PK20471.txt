LOCAL FIX

PROBLEM SUMMARY
 *  APAR PK14896 [http://www-01.ibm.com/support/docview.wss?uid=swg1PK14896] introduced a change in the way ORB initializes
   ServerRequestImpl. The processing done there would not
   work in "Connection less" mode which is how ORB operates in
   z/OS.
   
   
    
   
   

PROBLEM CONCLUSION
 *  A check has been made to enable the code changes done in
   PK14896 [http://www-01.ibm.com/support/docview.wss?uid=swg1PK14896] only in "Connection" mode. This averts the problem
   described in this APAR.
   This APAR would be available in 142 SR5.
   .
   To obtain the fix:
   Install build 20060303 or later
