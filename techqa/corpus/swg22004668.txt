Title: IBM Viewing IBM Case Manager widgets and IBM Case Monitor Dashboard in bidirectional languages - United States

Text:
 TECHNOTE (FAQ)

QUESTION
 How can I view IBM Case Manager widgets or IBM Case Monitor Dashboard in a bidirectional language such as Hebrew or Arabic? 

ANSWER
To view IBM Case Manager widgets or IBM Case Monitor Dashboard in a bidirectional language: 

 1. Set the locale to the appropriate language in your browser.
    
 2. In Case Manager Client or IBM Case Monitor Dashboard, click the user ID drop-down menu and select Change Language and Locale Settings and select the Enable bidirectional support check box.
