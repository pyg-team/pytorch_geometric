 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  When a network error occurs, DataPower MQ Components (MQ Front
   Side Handler and MQ url-open) might not detect the network error
   immediately.
   
   
    
   
   

LOCAL FIX

PROBLEM SUMMARY
 *  Affected is use of MQ sharing conversation.
   
   
   DataPower MQ Components (MQ Front Side Handler and MQ url-open)
   might not detect a network error immediately.
   
   
    
   
   

PROBLEM CONCLUSION
 *  Fix is available in 5.0.0.14, 6.0.0.6, and 6.0.1.2.
