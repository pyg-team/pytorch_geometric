Title: IBM Unread mail indication is not consistent with preferences setting for mail851.ntf - United States

Text:
 TECHNOTE (TROUBLESHOOTING)

PROBLEM
After searching the inbox in Notes 8.5.1, unread documents appear as Bold red rather than unbold/normal red. 

Steps to reproduce
1. Ensure you have Notes 8.5.1 and the mail file is based on mail851.ntf template

2. Set Preferences - Fonts and Colors- Unread mail indication to "Plain red text". Click OK

[/support/docview.wss?uid=swg21413406&amp;aid=1]

3. Mark some messages as unread. 

Expected result: Unread mail is indicated as plain red text
Actual result: Unread mail is actually indicated as bold red text

4. Now close and then reopen Notes. 

Now you get the expected result: Unread mail is indicated as plain red text.

5. Switch to any folder (for instance All documents) and then switch back to Inbox folder.

Expected result: Unread mail is indicated as plain red text
Actual result: Unread mail is again indicated as bold red text


RESOLVING THE PROBLEM
 

This issue is fixed in 8.5.1 Fix Pack 2 and 8.5.2.

Fix details: SPR# JTHN7VSR7R [http://www.lotus.com/ldd/r5fixlist.nsf/SPR/JTHN7VSR7R] 

Refer to the Upgrade Central [http://www.ibm.com/software/lotus/support/upgradecentral/index.html] site for details on upgrading Notes/Domino.
