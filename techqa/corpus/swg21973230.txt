Title: IBM (参考) IBM WebSphere Portal で使用されている IBM Java SDK における複数の脆弱性 (2015 年 10 月 IBM Java SDK CPU) - United States

Text:
 SECURITY BULLETIN

SUMMARY
 IBM WebSphere Portal のコンポーネントとして同梱されている IBM Java SDK について、2015 年 10 月の IBM Java SDK CPU にて複数の脆弱性が報告されました。 

VULNERABILITY DETAILS

2015 年 10 月の IBM Java SDK CPU にて報告された脆弱性の詳細と修正に関する情報は、以下の文書を参照してください。

(英文)「Security Bulletin: Multiple vulnerabilities in IBM® Java SDK affect WebSphere Application Server October 2015 CPU」(Technote #1969620) [http://www.ibm.com/support/docview.wss?uid=swg21969620]

この文書では、オラクル社より 2015 年 10 月 Critiral Patch Update (CPU) の一部として公開された Java SE の問題と、IBM SDK Java Technology Edition に影響する追加の脆弱性について説明します。IBM Java SDK には他のセキュリティー勧告も含まれますが、WebSphere Portal にはそれらの脆弱性の影響はありません。

お客様が開発されたコードに関しても脆弱性の影響を受けるかご確認ください。WebSphere Portal には影響しない勧告についての詳細は、下記の Reference セクションを参照してください。

CVEID:CVE-2015-4872 [http://cve.mitre.org/cgi-bin/cvename.cgi?name=CVE-2015-4872]
説明:Security コンポーネントに関連する不特定の問題により、完全性に部分的な影響を及ぼす可能性のある脆弱性が存在します。機密性・可用性には影響しません。

CVSS Base Score: 5
CVSS Temporal Score: 現時点のスコアについては ISS X-Force Database [https://exchange.xforce.ibmcloud.com/vulnerabilities/107361] を参照してください。
CVSS Environmental Score*: Undefined
CVSS Vector: (AV:N/AC:L/Au:N/C:N/I:P/A:N) 

CVEID:CVE-2015-4840 [http://cve.mitre.org/cgi-bin/cvename.cgi?name=CVE-2015-4840]
説明:2D コンポーネントに関連する Oracle Java SE の不特定の問題により、リモート攻撃者が機密情報を取得する可能性があります。

CVSS Base Score: 5
CVSS Temporal Score: 現時点のスコアについては ISS X-Force Database [https://exchange.xforce.ibmcloud.com/vulnerabilities/107353] を参照してください。
CVSS Environmental Score*: Undefined
CVSS Vector: (AV:N/AC:L/Au:N/C:P/I:N/A:N)

CVEID:CVE-2015-4734 [http://cve.mitre.org/cgi-bin/cvename.cgi?name=CVE-2015-4734]
説明:JGSS コンポーネントに関する不特定の問題により、リモート攻撃者が機密情報を取得することが可能であるという問題があります。

CVSS Base Score: 5
CVSS Temporal Score: 現時点のスコアについては ISS X-Force Database [https://exchange.xforce.ibmcloud.com/vulnerabilities/107356] を参照してください。
CVSS Environmental Score*: Undefined
CVSS Vector: (AV:N/AC:L/Au:N/C:P/I:N/A:N) 

CVEID: CVE-2015-5006 [http://cve.mitre.org/cgi-bin/cvename.cgi?name=CVE-2015-5006]
説明:Java Security コンポーネントには、攻撃者がシステムに物理的にアクセスすることにより、Kerberos クリデンシャルキャッシュから機密情報を取得することが可能であるという問題があります

CVSS Base Score: 4.6
CVSS Temporal Score: 現時点のスコアについては ISS X-Force Database [https://exchange.xforce.ibmcloud.com/vulnerabilities/106309] を参照してください。
CVSS Environmental Score*: Undefined
CVSS Vector: (CVSS:3.0/AV:P/AC:L/PR:N/UI:N/S:U/C:H/I:N/A:N)

IBM Java SDK に最新の修正を適用することを推奨します。

以下の文書を参照し、WebSphere Application Server のバージョンに合わせ SDK をアップグレードしてください。
(英文)「Security Bulletin: Multiple vulnerabilities in IBM® Java SDK affect WebSphere Application Server October 2015 CPU」(Technote #1969620) [http://www.ibm.com/support/docview.wss?uid=swg21969620]
