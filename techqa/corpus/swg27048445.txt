ABSTRACT
 Members of the IBM License Metric Tool (ILMT) support team shared information on installation of the BigFix Client for ILMT and IBM BigFix Inventory. 

CONTENT
The discussion of the installation of BES Client on various platforms using different methods included these topics:


• Installing the client manually
• Installing the client with MSI
• Using Software Distribution Tools
• Using Group Policies
• Using Login Scripts
• Embedding in a Common Build
• Using the Client Deploy Tool
