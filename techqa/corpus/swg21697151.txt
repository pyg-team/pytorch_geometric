Title: IBM Security Bulletin: One vulnerability in IBM FileNet Content Manager and IBM Content Foundation (CVE-2014-8917) - United States

Text:
Dojo; PSIRT 2498; XSS; Cross Site Scripting SECURITY BULLETIN

SUMMARY
 A security vulnerability exists in IBM FileNet Content Manager and IBM Content Foundation. See the Vulnerability Details for more information.


VULNERABILITY DETAILS
CVEID: CVE-2014-8917 [http://cve.mitre.org/cgi-bin/cvename.cgi?name=CVE-2014-8917]
DESCRIPTION: 
IBM Web applications using the IBM Dojo Toolkit and providing the dojox/form/resources/fileuploader.swf, dojox/form/resources/uploader.swf, dojox/av/resources/audio.swf or dojox/av/resources/video.swf files might be subject to XSS.
For IBM FileNet Content Manager and IBM Content Foundation, this includes the Administration Console for Content Platform Engine (ACCE) and any custom applications built on the IBM Content Navigator framework. (Security bulletins for the IBM Content Navigator application are published separately.)

CVSS Base Score: 4.3
CVSS Temporal Score: See https://exchange.xforce.ibmcloud.com/vulnerabilities/99303 [https://exchange.xforce.ibmcloud.com/vulnerabilities/99303] for the current score
CVSS Environmental Score*: Undefined
CVSS Vector: (AV:N/AC:M/Au:N/C:N/I:P/A:N) 


AFFECTED PRODUCTS AND VERSIONS
FileNet Content Manager 5.2.0, 5.2.1
IBM Content Foundation 5.2.0, 5.2.1
