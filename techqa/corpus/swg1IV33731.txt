ERROR DESCRIPTION
 *  Environment: TEPS/TEMS/TEMA: ITM7.10  + mySAP Agent 7.1
   affected SAP machines (OS): AIX
                SAP Database: Oracle
   
   
   Problem Description:
   Situation for ICM monitoring not working when ICM status is 'Not
   Running'.
   
   
   Detailed Recreation Procedure:
   1. Start SAP Agent.
   2. Stop the ICM service.
   3. Check status value on ICM Monitor Details Workspace(as zero).
   4. Situation with formula Status == 0 not getting triggered at
   any time.
   
   
   Root cause:
   
   When ICM service is not running , then  ABAP function module
   gives ABAP dump and fails to  handle this shutdown situations.
   Instead of sending 0 value it is terminated with
   "RAISE_EXCEPTION" ICM_ERROR.
   Here situation monitoring ICM shutdown does not gets any value
   so fails to fire alarms.
