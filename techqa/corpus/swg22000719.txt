Title: IBM Security Bulletin: IBM Jazz for Service Management (Jazz SM) is affected by a code execution vulnerability in IBM Tivoli Common Reporting (TCR) (CVE-2016-5983) - United States

Text:
 SECURITY BULLETIN

SUMMARY
 A code execution vulnerability has been discovered in IBM Cognos Business Intelligence installed by IBM Tivoli Common Reporting (TCR). TCR is included in IBM Jazz for Service Management (JazzSM). IBM has addressed the applicable CVE. 

VULNERABILITY DETAILS
CVEID: CVE-2016-5983 [http://cve.mitre.org/cgi-bin/cvename.cgi?name=CVE-2016-5983]
DESCRIPTION: IBM WebSphere Application Server could allow remote attackers to execute arbitrary Java code with a serialized object from untrusted sources.
CVSS Base Score: 7.5
CVSS Temporal Score: See https://exchange.xforce.ibmcloud.com/vulnerabilities/116468 [https://exchange.xforce.ibmcloud.com/vulnerabilities/116468] for the current score
CVSS Environmental Score*: Undefined
CVSS Vector: (CVSS:3.0/AV:N/AC:H/PR:L/UI:N/S:U/C:H/I:H/A:H) 

AFFECTED PRODUCTS AND VERSIONS
Tivoli Common Reporting 3.1 

Tivoli Common Reporting 3.1.0.1 

Tivoli Common Reporting 3.1.0.2 

Tivoli Common Reporting 3.1.2 

Tivoli Common Reporting 3.1.2.1 

Tivoli Common Reporting 3.1.3

REMEDIATION/FIXES
The recommended solution is to apply the fix for versions listed as soon as practical.
