ERROR DESCRIPTION
 *  STEPS TO REPRODUCE
   1. Install Eclipse Mars (4.5.2)
   2. Start Eclipse and add the Rhapsody 8.1.5 eclipse directory as
   a repository via Help > Install New Software
   3. Install the IBM(R) Rational(R) Rhapsody(R) Workflow
   Integration plugin only
   4. Create a new Rhapsody project in Eclipse, add a simple output
   statement.
   5. Generate the code and add a new Eclipse Cpp project when
   prompted.
   6. Build the code and run.
   7. Go to File menu > Switch Workspace > other and select the
   current workspace.
   8. Eclipse will attempt to restart.
   9. Intermittently Eclipse will fail to restart while the Eclipse
   process will remain in the process explorer. Therefore, user
   must manually terminate and restart the Eclipse process.
