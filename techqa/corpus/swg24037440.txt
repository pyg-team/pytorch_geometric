ABSTRACT
 This document describes how to download the PeopleSoft data growth and application retirement portion of the IBM® InfoSphere® Optim™ Solution for Oracle Applications 11.3. 

DOWNLOAD DESCRIPTION
You can download the PeopleSoft portion of the 11.3 Optim Solution for Oracle Applications using the Passport Advantage website.


The IBM InfoSphere Optim Data Growth Solution for PeopleSoft Enterprise includes the core Optim technology, definitions needed to drive archive and restore processing, PeopleSoft forms for archiving PeopleSoft data, and code for native displays of archived data. 

The Application Retirement Solution is used to archive PeopleSoft data prior to retiring or decommissioning the PeopleSoft application.

PREREQUISITES
You must install the IBM InfoSphere Optim Solution v11.3 before you install the solution for PeopleSoft.

For information about hardware and software requirements for the Optim Solution for PeopleSoft , see the Detailed System Requirements at http://www.ibm.com/support/docview.wss?uid=swg27041757 [http://www.ibm.com/support/docview.wss?uid=swg27041757].

INSTALLATION INSTRUCTIONS


For general installation instructions, see the Quick Start Guide. For detailed instructions, see the Optim Data Growth Solution for PeopleSoft Enterprise Installation and Configuration Guide. Both documents are included in the download package.

DOWNLOAD PACKAGE


IBM InfoSphere Optim supports processing across a combination of platforms and database management systems, but requires a 32-bit DBMS client to connect to a data source.

After proof of entitlement, you are allowed to download any or all of the images listed here. To obtain the download images:

 1. Navigate to the Find Downloads and Media [http://www.ibm.com/software/howtobuy/passportadvantage/paocustomer/sdma/SDMA?P0=DOWNLOAD_FIND_BY_PART_AND_DESCRIPTION] Web page on the IBM Passport Advantage Web site.
 2. Under Download finder options, select the Find by part number search option.
 3. Search for each downloadable image by download name.
 4. Download all required parts.
 5. Use a file extracting utility that supports large file extractions to extract each of the parts into a single, temporary directory on your system.
 6. Follow the instructions in the Quick Start Guide to install the product.
