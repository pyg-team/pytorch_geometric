 DOWNLOADABLE FILES

ABSTRACT
 This fix updates the IBM Java(R) Runtime Environment for the portal server to address documented security vulnerabilities 

DOWNLOAD DESCRIPTION
This fix updates the IBM Java Runtime Environment (JRE) which the portal server distributes to clients. It addresses security vulnerabilities as documented in the Security bulletin: http://www-01.ibm.com/support/docview.wss?uid=swg21640206 [http://www-01.ibm.com/support/docview.wss?uid=swg21640206].


The vulnerabilities addressed by this fix exist when the affected JRE is installed on systems running the Tivoli Enterprise Portal Browser client or Java WebStart client. 

The affected JRE is installed on a system when logging into the IBM Tivoli Enterprise Portal using the Browser client or WebStart client and a JRE at the required level does not exist. The portal provides an option to download the provided JRE to the system. 

This fix provides updated JRE packages for the portal which can be downloaded by new client systems. Once the fix has been installed on the portal server, instructions in the README can be used to update the portal clients.

There are three fixes provided:
