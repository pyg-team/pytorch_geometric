REQUIRED: DOWNLOAD YOUR REPLICATION ENGINES
On Linux, UNIX, and Windows, the Q Replication V10.2.1 engine is part of DB2 for Linux, UNIX, and Windows V10.5. Choose one of these options: 

 * If DB2 Version 10.5 is already installed on your system, you only need to activate the license for Q Replication (see activation kits below).
 * If you need to install DB2 V10.5, download the InfoSphere Federation Server V10.5 program for your operation system and use its installation program to add DB2 V10.5 to your system. You must also install the latest Federation Server fix pack, available from IBM Fix Central [http://www-933.ibm.com/support/fixcentral/].

Note: If you are replicating to federated targets, you must also download InfoSphere Federation Server.
