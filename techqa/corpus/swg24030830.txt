WebSphere ILOG BRMS JRules DOWNLOADABLE FILES

ABSTRACT
 This cumulative fix pack updates JRules V7.1.1 (7.1.1, 7.1.1.1, 7.1.1.2 or 7.1.1.3) to V7.1.1.4. 

DOWNLOAD DESCRIPTION
This cumulative fix pack updates JRules V7.1.1 (7.1.1, 7.1.1.1, 7.1.1.2 or 7.1.1.3) to V7.1.1.4. 


This fix pack does not work directly with earlier versions of JRules. Users of V7.0 and V7.1.0 should update to V7.1.1 in order to install V7.1.1.4.

This fix pack is entitled. Only customers who have purchased JRules V7.1.1 can download this fix pack.

PREREQUISITES
WebSphere ILOG JRules V7.1.1

INSTALLATION INSTRUCTIONS
Back up your current installation before installing the new release. Download and execute the V7.1.1.4 product installer.
