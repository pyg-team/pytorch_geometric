Title: IBM Number of hosts in ping probe - United States

Text:
ping probe; hosts; netcool; omnibus TECHNOTE (FAQ)

QUESTION
 Is there a hard limit as to the number of hosts a ping probe can support? 

ANSWER
From code perspective, there is no hard limit to number of host the Ping 
probe can monitor. Therefore, it is up to the user to set how many host 
to be monitored based on their environment scenario (memory allocation, 
adequate network bandwidth).
