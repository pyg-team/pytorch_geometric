Title: IBM PM01303: MISLEADING MESSAGES ISSUED AGAINST CLASSIC SCA MODULES IN EAR PACKAGES - United States

Text:
 SUBSCRIBE
You can track all active APARs for this component.



APAR STATUS
 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  Misleading messages issued against classic Service Component
   Architecture (SCA) modules in Enterprise Archive file (EAR)
   packages. They should be removed.
   
   
    
   
   

LOCAL FIX

PROBLEM SUMMARY
 *  ****************************************************************
   * USERS AFFECTED:  Users of WebSphere Application Server v7    *
   *                  Feature Pack for Service Component          *
   *                  Architecture (SCA) Refresh Pack v1.0.1.0    *
   ****************************************************************
   * PROBLEM DESCRIPTION: Misleading messages issued against      *
   *                      classic SCA modules in EAR              *
   *                      packages.                               *
   ****************************************************************
   * RECOMMENDATION:                                              *
   ****************************************************************
   When installing a classic Service Component Architecture (SCA)
   module in an Enterprise Archive (EAR) package, the following
   warning message is issued:
   
   CWSAM0001W: Package does not contain SCA artifacts. BLA
   containing only such packages will not be runnable.
   
   
    
   
   

PROBLEM CONCLUSION
 *  This warning message is misleading to users who install
   classic SCA modules. The warning message is now removed for
   the classic SCA module installation case.
   
   APAR PM01303 is currently targeted for inclusion in Service
   Level (Fix Pack) 1.0.1.1 of WebSphere Application Server v7
   Feature Pack for Service Component Architecture (SCA) Refresh
   Pack. Please refer to the Recommended Updates page for
   delivery information:
   http://www.ibm.com/support/docview.wss?rs=180&uid=swg27004980 [http://www.ibm.com/support/docview.wss?rs=180&uid=swg27004980]
   
   
    
   
   

TEMPORARY FIX
