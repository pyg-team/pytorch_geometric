APAR STATUS
 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  Errors are not always reported in a clear manner for xsd compile
   errors. This is particularly true when a regular expression
   contains an invalid escaped character and an MbXSDInternalError
   is reported.
   
   Errors similar to the following may be observed:
   
   BIP5024E: The creation and preprocessing of the XML schemas in
   message set 'schemaMset.xsdzip' has failed.
   
   BIP4395E: Java exception:
   'com.ibm.broker.schemadeploy.MbXSDInternalError'; thrown from
   class name: 'com.ibm.broker.schemadeploy.MbXSDDeployDirector',
   method name: 'compileMessageSetSchemas', file:
   'MbXSDDeployDirector.java', line: '517'

 *  N/A
