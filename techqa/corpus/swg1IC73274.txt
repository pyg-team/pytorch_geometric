APAR STATUS
 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  WebSphere MQ (WMQ) Advanced Messaging Security (AMS) v7.0.1
   generates errors when used with WMQ v6 client. WMQ v7 Client
   is successful. AMS loads mqic.dll. It should load mqic32.dll
   if the client is running WMQ v6.
   Log file shows
   DRQDM2004E   The IBM WebSphere MQ Advanced Message Security C
   client interceptor could not load the WebSphere MQ client
   library. The error message was: error code 126.
   AMS trace shows
   Client interceptor initialization failed.
   
   
    
   
   

LOCAL FIX

PROBLEM SUMMARY
 *  WebSphere MQ Advanced Message Security (MQAMS) v7.0.1 generates
   errors when used with WMQ v6 client on Ms Windows as MQAMS
   attempts to load mqic.dll instead of mqic32.dll.
   Log file shows:
   DRQDM2004E   The IBM WebSphere MQ Advanced Message Security C
   client interceptor could not load the WebSphere MQ client
   library. The error message was: error code 126.
   Trace shows:
   Client interceptor initialization failed.
   
   
    
   
   

PROBLEM CONCLUSION
 *  The fix for this APAR will be contained in the fix pack 1.
