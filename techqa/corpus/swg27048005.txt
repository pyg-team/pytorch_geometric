ABSTRACT
 First launched in Milton Keynes, U.K in 1999, acquired by IBM in 2013, Daeja ViewONE enables your organization to consistently view documents across your desktop and mobile devices that can display documents in hundreds of formats, including TIF, BMP, JPG, GIF, PDF, Excel, and PNG, either natively or through add-on modules. 

This document features the most requested documents as well as those identified as valuable in helping answer your questions related to Daeja. The list will be updated periodically to reflect new trends and issues.

    This technical note explains why the user is not able to see any document or image displayed and gets an error "Unable to generate page due to Repository Access Error". 
 3.  Why does IBM Daeja ViewONE Professional take a long time to load? [http://www-01.ibm.com/support/docview.wss?uid=swg21976232&acss=danl_2848_web] 
    You will see a spinning Java logo rather than the expected viewer. This is eventually replaced by the viewer which then attempts to load and render the selected document. 
 4.  How to disable navigation thumbnail view in ViewONE Virtual? [http://www-01.ibm.com/support/docview.wss?uid=swg21970539&acss=danl_2848_web] 
    By default, IBM Daeja ViewONE Virtual displays a set of small thumbnail icons on the right hand side of the viewer. This view can obscure the detail in the displayed document, so this technical note explains how can this view be disabled. 
 5.  Why does the file "/viewone/v1files/viewone.js" not exist while setting up HTML code to run IBM Daeja ViewONE Virtual? [http://www-01.ibm.com/support/docview.wss?uid=swg21963953&acss=danl_2848_web]

    This technical note explains the use of the event handler properly in IBM Daeja ViewONE viewer. 
 7.  How to resolve the error "Unable to generate page due to repository access error" when viewing a document in IBM Daeja ViewONE? [http://www-01.ibm.com/support/docview.wss?uid=swg21974498&acss=danl_2848_web] 
    The error occurs in IBM Content Navigator (ICN) when viewing a document with either ViewONE Pro with Streamer enabled for any streamed document (typically PDF and TIF) or with ViewONE Virtual for any document type. This technical note explains how to resolve it.
