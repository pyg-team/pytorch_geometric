ABSTRACT
 The IBM Security Support team shared information on using IBM License Metric Tool (ILMT) to run a report successfully. Les Popow provided a thorough demonstration. 

CONTENT
This session was a live demonstration of the tasks you need to accomplish in order to produce an accurate audit report. It assumed the user has a successful installation of the product, and is intended to address any confusion in the proper usage of ILMT.
