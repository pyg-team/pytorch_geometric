QUESTION
 What is the IBM Rational ClearCase (CC) 7.1 equivalent to installing by siteprep in pre-7.1 versions? 

ANSWER
 

ClearCase 7.1 uses Installation Manager and siteprep is no longer available.

Silent install using a response file is a similar method to using siteprep.

The steps below point to the appropriate sections of the ClearCase 7.1 installation documentation and will help you to install ClearCase 7.1 (siteprep style).



 1. Verifying and extracting electronic images [https://publib.boulder.ibm.com/infocenter/cchelp/v7r1m0/index.jsp?topic=/com.ibm.rational.clearcase.cc_ms_install.doc/topics/t_extract_verify_files.htm]
