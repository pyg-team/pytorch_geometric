Title: IBM JR36678: WHEN THE HIERARCHY WINDOW ON THE LEFT NAVIGATION PANE IS SMALL THE CATEGORY NAME IS PARTIALLY VISIBLE. - United States

Text:
 SUBSCRIBE
You can track all active APARs for this component.



APAR STATUS
 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  When the hierarchy window on the left navigation pane of
   InfoSphere MDM Server for PIM is small the category name is
   partially visible.
   
   Steps to reproduce:
   1. Login to a InfoSphere MDM Server for PIM instance.
   2. Make sure one or more hierarchy appear on the left navigation
   pane.
   3. Resize the left navigation pane until the horizontal scroll
   bar appears for any of the hierarchy window.
   Note: Make sure there are multiple categories under a hierarchy
   that has the horizontal scroll bar
   
   Actual Result:
   The category name that shows up right above the horizontal
   scroll bar is partially visible.
   
   Expected Result:
   The complete category name appear on the hierarchy
   window.
   
   
   Keywords: browser, user interface
   
   
    
   
   

LOCAL FIX
 *  Resize the left navigation pane to the larger size.
