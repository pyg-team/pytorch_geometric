MP48: WebSphere MQ for iSeries V7.1 - Performance Evaluations DOWNLOADABLE FILES

ABSTRACT
 This SupportPac provides a performance evaluation for WebSphere MQ for iSeries Version 7.1. 

DOWNLOAD DESCRIPTION
This SupportPac includes: 
• Performance charts showing release highlights 
• Performance measurements with figures and tables to present the performance capabilities of WebSphere MQ V7.1 
• Interpretation of the results and their implications for designing or sizing WebSphere MQ configurations.

Possible Uses
This SupportPac is designed for people who: 
• Will be designing/implementing solutions using WebSphere MQ V7.1
• Want to understand the performance limits of WebSphere MQ on the iSeries platform 
• Want to understand how to tune WebSphere MQ on iSeries

Skill Level Required
The reader should have a general awareness of the iSeries operating system and of WebSphere MQ in order to make best use of this SupportPac. 

New in this Release
• Capacity information added in Chapter 6.
