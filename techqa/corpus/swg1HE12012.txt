APAR STATUS
 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  When creating or editing a hold rules template, if you check the
   "Include Virtual Interview" box and attempt to add
   questionnaires, you will only be able to add them from the first
   page that is displayed. When you click on "Next" or on "2", "3"
   etc for other pages, the page comes back to the Initial Notice
   page and does not allow any questionnaires from ensuing pages to
   be chosen.
   
   
    
   
   

LOCAL FIX
 *  De-select the "Confirmation Required" check-box and select it
   agian. This will take you to the page you selected prior with
   the questionnaires of interest.
   
   
    
   
   

PROBLEM SUMMARY
 *    When you assign an interview questionnaire to a hold notice
   rules template, you cannot select any questionnaire past the
   first page. When you click Next, the Initial Notice page is
   displayed.
   
   
    
   
   

PROBLEM CONCLUSION
 *  This fix is available now from FixCentral in fixpack 6.0.1.7.
