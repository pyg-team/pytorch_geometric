Title: IBM PK71784; 7.0: Invalidkeyexception: SHA1withRSA.2 with hw crypto - United States

Text:
PK71784; PK71784; PK71784 DOWNLOADABLE FILES

ABSTRACT
 This interim fix supports the cryptographic provider in a multi-threaded environment. The hardware cryptography function in IBM® WebSphere® Application Server Version 7.0. 

DOWNLOAD DESCRIPTION
PK71784 resolves the following problem:

ERROR DESCRIPTION:? 
InvalidKeyException: SHA1withRSA.2 with HW crypto 

LOCAL FIX:? 

PROBLEM SUMMARY:?

USERS AFFECTED:
All users of WebSphere Application Server using Hardware Cryptography.

PROBLEM DESCRIPTION:
This interim fix supports the cryptographic provider in a multi-threaded environment.

The hardware cryptography function in IBM WebSphere Application Server Version 7.0 causes a key exception in a multi-threaded environment. The IBMJCEProvider cryptographic provider issues an exception that states "the key is not a valid key."

RECOMMENDATION:
None

Hardware Cryptography fails in a multithreaded environment which using JDK 6.0 ibmpkcs11. Symptom may include invalid key exception.

PROBLEM CONCLUSION:? 


The fix for this APAR is currently targeted for inclusion in fixpack 7.0.0.1.
