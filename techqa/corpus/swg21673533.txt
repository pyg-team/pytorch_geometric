Title: IBM Receiving NullPointerException when clicking Test Connection button in Cast Iron Studio - United States

Text:
 TECHNOTE (TROUBLESHOOTING)

PROBLEM(ABSTRACT)
 You receive a NullPointerException when clicking the Test Connection button in Cast Iron Studio endpoint configuration without providing proxy User Name and Password. 

RESOLVING THE PROBLEM
Cast Iron Studio connection test requires values for both proxy User Name and Password when Connect via a Proxy Server is checked in endpoint configuration. To connect to a proxy which allows anonymous login at connection test, give random values to User Name and Password.
