ABSTRACT
 The WebSphere Message Broker V6.0 Client for WebSphere Service Registry and Repository provides a means of querying the service registry. 

DOWNLOAD DESCRIPTION
PLEASE NOTE:
This SupportPac has been superseded by IA9Q. Please use the link in the download table to access the webpage and code.

The WebSphere Service Registry and Repository (WSRR) is the repository of record for the services meta-data for the ESB. This SupportPac provides WMB standard interfaces (nodes) for accessing WSRR, provides three levels of caching, and WSRR synchronized cache update.

The IA9L SupportPac (WMB Client for WSRR) has been largely superseded by a product extension SupportPac, IA9Q. The IA9Q is a Category 3 SupportPac and will be merged into the product base in the WMB 6.1 timeframe. The IA9Q SupportPac is the recommended WMB Client for WSRR, to access it click on the link below in the downloads table.

The IA9L SupportPac will be updated in the near future, as most of its functionality became IA9Q. Once updated IA9L will contain only the pattern of use nodes, SRxxxxVirtualService nodes. 

POSSIBLE USES
Implementation of a services registry, as part of a Service Oriented Architecture, where WebSphere Message Broker has been selected as the Enterprise Services Bus.

SKILL LEVEL REQUIRED
Normal WebSphere Message Broker development skills.
