Title: IBM InfoSphere Identity Insight v8.1 system requirements when running on IBM AIX - United States

Text:
 DETAILED SYSTEM REQUIREMENTS

ABSTRACT
 Update to AIX system requirements for v8.1. POWER8® is supported. 

CONTENT
System requirements when running on IBM AIX




The following list identifies the products that are supported when IBM® InfoSphere® Identity Insight runs on the AIX® operating system.

Table 1. System requirements when running on IBM AIX 

 

Operating Systems  *  IBM AIX 7.1L IBM AIX 6.1L 

Hardware Requirements  *  POWER8® (64-bit) POWER7® (64-bit) POWER6® (64-bit) 

Java™ The following is installed with the product:  *  IBM 64-bit Java Runtime Environment for AIX, Java Technology Edition, Version 6 

Client Java Requirements AIX is not a supported client platform. Any supported platform client machines connecting to the Configuration Console or the Visualizer must have the SUN Java SE Runtime Environment (JRE) Version 6 installed. Databases  *  IBM DB2® Database for Linux, UNIX, and Windows 9.7 IBM DB2 Database for Linux, UNIX, and Windows 9.5 IBM Informix® 11.70 Oracle 11g Release 1 (11.1.0.6, 11.1.0.7) Oracle 11g Release 2 (11.2.0.1, 11.2.0.2, or greater) 

Database Clients  *  DB2 client v9.7 when connecting to IBM DB2 Database for Linux, UNIX, and Windows 9.7 DB2 client v9.7 or v9.5 when connecting to IBM DB2 Database for Linux, UNIX, and Windows 9.5 IBM Informix client products, Version 3.50 and greater. Oracle 11g Release 1 (11.1.0.6) client when connecting to Oracle 11g Release 1 (11.1.0.6). Oracle 11g Release 1 (11.1.0.7) client when connecting to Oracle 11g Release 1 (11.1.0.7). Oracle 11g Release 2 (11.2.0.1) client when connecting to Oracle 11g Release 2 (11.2.0.1). 

Java Database Connectivity (JDBC) Clients for the Configuration Console and the Visualizer  *  DB2 client v9.7 JDBC driver v9.7 when connecting to IBM DB2 Database for Linux, UNIX, and Windows 9.5 or 9.7. For the exact driver version, see IBM DB2 JDBC Driver Versions [http://www.ibm.com/support/docview.wss?uid=swg21363866]. DB2 client v9.5 JDBC driver v9.5 when connecting to IBM DB2 Database for Linux, UNIX, and Windows 9.5. For the exact driver version, see IBM DB2 JDBC Driver Versions [http://www.ibm.com/support/docview.wss?uid=swg21363866]. IBM Informix JDBC Driver Version 3.70 Oracle 11g Release 1 (11.1.0.6) JDBC drivers when connecting to Oracle 11g Release 1 (11.1.0.6). Oracle 11g Release 1 (11.1.0.7) JDBC drivers when connecting to Oracle 11g Release 1 (11.1.0.7). Oracle 11g Release 2 (11.2.0.1) JDBC drivers when connecting to Oracle 11g Release 2 (11.2.0.1). 

Web Browsers  *  Mozilla Firefox 3.5 and 3.6 (no beta versions) 

Message Queuing Software  *  IBM WebSphere® MQ 6.0.2.11 

Other  *  IBM C++ Runtime Environment Components for AIX, For more information about this requirement, review this support information: http://www-01.ibm.com/support/docview.wss?uid=swg24025181 [http://www-01.ibm.com/support/docview.wss?uid=swg24025181]
