Title: IBM Progressive tag in redirection URL for some Atlas reports - United States

Text:
 TECHNOTE (FAQ)

QUESTION
 Why do we need to specify progressive tag in redirection URL for some Atlas reports? 

CAUSE
Some Atlas reports like "Schedule to Data Source Mapping Exceptions" report throw an exception " Cannot fetch the next data row" . As a work around we specify progressive tag in redirection URL for the report 

ANSWER
 

 1. Login to Atlas 
 2. Navigate to Admin > Manage Reports 
 3. Edit the report 
 4. In the Redirection URL field specify http://<host_name>:<Port>/AtlasReports/progressive 
 5. Click Save and Close
