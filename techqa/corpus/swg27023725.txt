Installing 
The following questions and answers refer to installation. 


 
FAQ: Where can I find the software and hardware requirements for DVS?
Answer: See the WebSphere ILOG JRules Business Rule Management System Version 7.1, hardware and software requirements [http://www.ibm.com/support/docview.wss?uid=swg27018135]. 


 
FAQ: Where can I find information about downloading DVS? 
Answer: See the [http://www-01.ibm.com/support/docview.wss?uid=swg27018135]WebSphere ILOG JRules BRMS V7.1 Download document [http://www.ibm.com/support/docview.wss?uid=swg24030938].

Usage 
The following questions and answers refer to usage. 



FAQ: Where can I find information about using DVS?
Answer: See the online documentation [http://publib.boulder.ibm.com/infocenter/brjrules/v7r1/index.jsp].


 
FAQ: How can I cluster Decision Validation Services?
Answer: See Clustering Decision Validation Services [http://www.ibm.com/support/docview.wss?uid=swg21425487].


 
FAQ: Can I use the multiple-execution server to distribute the execution of tests?
Answer: Yes. See Distribute Decision Validation Service tests on different Rule Execution Server instances [https://www.ibm.com/support/docview.wss?uid=swg21502095].


 
FAQ: How do I prevent DVS rule executions from being persisted in the Decision Warehouse? 
Answer: See [http://www-01.ibm.com/support/docview.wss?uid=swg27018135]Configure Decision Validation Services not to use Decision Warehouse [http://www.ibm.com/support/docview.wss?uid=swg21441394]. 


 
FAQ: How do I use spaces as valid input values in a Decision Validation Services (DVS) Excel spreadsheet? 
Answer: See Spaces as valid input values in a Decision Validation Services (DVS) Excel spreadsheet [http://www-01.ibm.com/support/docview.wss?uid=swg21677735].

Troubleshooting 
The following questions and answers refer to troubleshooting. 


 
FAQ: Where can I find support information for DVS?
Answer: Go to the online support center [http://www.ibm.com/support/entry/portal/Overview/Software/WebSphere/WebSphere_ILOG_Decision_Validation_Services].


 
FAQ: I do not have access to Test Suite or scenarios in Rule Team Server and the Decision Warehouse tab is not enabled, What should I do?
Answer: See Enabling Decision Validation Services and Decision Warehouse in Rules Team Server and Rule Execution Server [http://www.ibm.com/support/docview.wss?uid=swg21440249].


 
FAQ: How can I improve the performance of DVS and Decision Warehouse?
Answer: The document Improve Performance of Decision Validation Services (DVS) and Decision Warehouse [http://www.ibm.com/support/docview.wss?uid=swg21588859] addresses how to improve performance when you are conducting DVS tests and using Decision Warehouse.
