Title: IBM Security Bulletin: Multiple vulnerabilities in IBM Java SDK affect WebSphere Service Registry and Repository Java 2015 CPU - United States

Text:
 SECURITY BULLETIN

SUMMARY
 There are multiple vulnerabilities in IBM® SDK Java™ Technology Edition that is shipped with IBM WebSphere Service Registry and Repository. These issues were disclosed as part of the IBM Java SDK updates in January 2015. 

VULNERABILITY DETAILS
To fix the WebSphere Service Registry and Repository server, please apply the fix indicated in the WebSphere Application Server bulletin at http://www.ibm.com/support/docview.wss?uid=swg21695362 [http://www-01.ibm.com/support/docview.wss?uid=swg21695362] 

If you wish to also apply a fix to WebSphere Service Registry and Repository Studio, please either contact IBM support for a fix, or replace Studio's bundled JRE with the updated JRE version 6 SR16-FP2. The fixed JRE can be downloaded from https://www.ibm.com/developerworks/java/jdk/ [https://www.ibm.com/developerworks/java/jdk/]

AFFECTED PRODUCTS AND VERSIONS
WebSphere Service Registry and Repository 6.3, 7.0, 7.5, 8.0, 8.5 

WebSphere Service Registry and Repository Studio 8.5
