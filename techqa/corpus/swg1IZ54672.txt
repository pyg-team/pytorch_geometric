LOCAL FIX

PROBLEM SUMMARY
 *  Once the deadlock has occurred, the only way to recover is to
   stop and restart the database and the AppServer. You may need to
   force the database shutdown.
   
   This fix improves import and load performance so that the chance
   of a Transaction Timeout exception is reduced.

 *  Apply this fix to improve import and load performance.
