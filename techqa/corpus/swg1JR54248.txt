LOCAL FIX
 *  Exception when upgrading a dmgr profile without any DE
   configured
   
   
    
   
   

PROBLEM SUMMARY
 *  The IBM BPM profile upgrade tries to access the truststore for
   BlueWorksLive, which exists only if a deployment environment is
   configured.
   
   
    
   
   

PROBLEM CONCLUSION
 *  A fix is included in IBM BPM V8.5.6.0 Cumulative Fix 1 that
   allows profiles with no configured IBM BPM deployment
   environments to be upgraded. To get this cumulative fix, go to
   http://www.ibm.com/support/docview.wss?uid=swg27046290 [http://www.ibm.com/support/docview.wss?uid=swg27046290]
