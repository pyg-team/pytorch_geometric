PROBLEM SUMMARY

USERS AFFECTED:
All IBM WebSphere Application Server version 6.1 users on IA64 hardware running the HPUX operating system.

PROBLEM DESCRIPTION:
The original APAR PK26726 delivered a fix for crash in the deployment manager running on HP-UX/IA64 when using the administrative console to do administration. It was then uncovered that installing WebSphere Application Server service packs would cause PK26726 to be uninstalled. This was deemed to be an issue for customers. This APAR provides a repackaged PK26726 which will not be automatically uninstalled when WebSphere Application Server service packs are installed.

None
