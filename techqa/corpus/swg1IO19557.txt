Title: IBM IO19557: ADD NEEDED DST TIME ZONES - United States

Text:
 
APAR STATUS
 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  A user would like to set the following time zones that have DST
   rules which are not currently available in System Configuration
   for 4690 OS:
     * GMT - Greenwich Mean Time or
       WET - Western European Time or
       BST - British Summer Time
     * AZOT - Azores Time which observes
     * EET - Eastern European Time
   
   
    
   
   

LOCAL FIX

PROBLEM SUMMARY
 *  There were no options to define the Daylight time configuration
   for the following timezones :
      * British Summer Time
      * Azores Summer Time
      * Eastern European Summer Time
   
   
    
   
   

PROBLEM CONCLUSION
 *  Three new Time zones were added:
   .
      GMT0BST  - British Summer Time :
        DST begins the last Sunday in March, 1:00 AM
        DST ends the last Sunday in October, 2:00 AM
   .
      AZO1DST  - Azores Summer Time :
        DST begins the last Sunday in March, 12:00 Midnight
        DST ends the last Sunday in October, 1:00 AM
   .
      EET-2DST - Eastern European Summer Time  :
        DST begins the last Sunday in March, 3:00 AM
        DST ends the last Sunday in October, 4:00 AM
   
   
    
   
   

TEMPORARY FIX
