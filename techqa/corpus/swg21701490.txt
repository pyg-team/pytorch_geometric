Title: IBM Security Bulletin: A security vulnerability has been identified in WebSphere MQ shipped with WebSphere Remote Server (CVE-2015-0159, CVE-2015-0138 and CVE-2014-6221) - United States

Text:
 SECURITY BULLETIN

SUMMARY
 WebSphere MQ is shipped as a component of WebSphere Remote Server. Information about a security vulnerability affecting WebSphere MQ has been published in a security bulletin. 

VULNERABILITY DETAILS
For vulnerability details, see the security bulletin Vulnerabilities in GSKit affect IBM WebSphere MQ (CVE-2015-0159, CVE-2015-0138 and CVE-2014-6221). [http://www.ibm.com/support/docview.wss?uid=swg21699055]
