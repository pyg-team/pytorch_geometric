Title: IBM Data Access Plan fails with com.spss.adv.dataService. AnalyticDataServiceException: Data Service request caused SQL exception - United States

Text:
QEQMC0132I E3008 ODBC Driver E3318 MySQL Data Access Plan TECHNOTE (TROUBLESHOOTING)

PROBLEM(ABSTRACT)
 Attempts to create a Data Access Plan in IBM SPSS Collaboration and Deployment Services result in the error: 



com.spss.adv.dataService.

AnalyticDataServiceException: Data Service request caused SQL exception

(java.sql.SQLNonTransientConnectionException: [SPSSOEM][MySQL JDBC

Driver]Error establishing socket to host and port: host:port

Reason: Connection timed out: connect) Please see the server log to

find more detail regarding exact cause of the failure 

SYMPTOM
com.spss.adv.dataService.
AnalyticDataServiceException: Data Service request caused SQL exception
(java.sql.SQLNonTransientConnectionException: [SPSSOEM][MySQL JDBC
Driver]Error establishing socket to host and port: hops.ucsd.edu:3306.
Reason: Connection timed out: connect) Please see the server log to
find more detail regarding exact cause of the failure


CAUSE
This environment was using a VM
WebSphere / C&DS were on the VM
Modeler was on another VM
MySQL database was on a separate VM

ENVIRONMENT
C&DS 8.0
Modeler 18
MySQL 5.6

DIAGNOSING THE PROBLEM
the error in the log is the following
3432 0x00000ec8 INFO SDLSessionServer.ClientError AEQMC0132I: E3008:
IM002 [Microsoft][ODBC Driver Manager] Data source name not found and
no default driver specified (IM002 [Microsoft][ODBC Driver Manager]
Data source name not found and no default driver specified)
2016-10-27 10:19:08,286 3432 0x00000ec8 INFO SDLSessionServer.
ClientError AEQMC0132I: E3318: Failed to connect to datasource: xxxx.xxxx.xxx;UID=xxxxx;PWD=***** xxxx.xxxx.xxx;UID=cognosbi;PWD=*****)

RESOLVING THE PROBLEM
Ensure that the firewall between WebSphere/C&DS and the database (MySQL) is open.
