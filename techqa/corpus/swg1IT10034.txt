ERROR DESCRIPTION
 *  According to RFC 6749, Refreshing an Access Token requires the
   following parameters:
      grant_type=refresh_token  and  refresh_token=<refresh_token>
   
   However, when sending a request to Refresh an Access Token with
   the following parameters:
      grant_type=refresh_token  and code=<code>
   
   it does not result in an error.
   
   
    
   
   

LOCAL FIX

PROBLEM SUMMARY
 *  Affected is Oauth refresh token processing.
   
   Oauth refresh token requests with missing required parameters
   might not return an error code as expected.
   
   
    
   
   

PROBLEM CONCLUSION
 *  Fix is available in 6.0.0.18, 6.0.1.14, 7.0.0.11, 7.1.0.8
   and 7.2.0.3

   http://www-01.ibm.com/support/docview.wss?uid=swg21237631 [http://www-01.ibm.com/support/docview.wss?uid=swg21237631]
