APAR STATUS
 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  With custom OAuth processing, if an authorization request is
   sent to Datapower containing a state parameter, but without a
   response_type parameter, DataPower sends the error and
   error_description parameters in the redirect, but the "state"
   parameter is missing.
   According to Oauth RFC 6749, the "state" parameter is REQUIRED
   if a "state" parameter was present in the client authorization
   request.  The exact value received from the client.
   
   
    
   
   

LOCAL FIX

PROBLEM SUMMARY
 *  Affected is custom OAuth processing.
   
   If an authorization request is sent to Datapower containing a
   state parameter, but without a response_type parameter,
   DataPower sends the error and error_description parameters in
   the redirect, but the "state" parameter is missing.
