Title: IBM IT14284: PUREAPPLICATION SYSTEM: SQL ERROR OCCURS WHEN SAVING AN EVENT FILTER IN THE CONSOLE - United States

APAR STATUS
 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  The user attempted to create a filter with 6 types on the Events
   page in the console.
   
   1. Go to System > Event
   2. Click the Filter icon next to the dropdown box
   3. In the Build Filter windows,
       a. Click + icon (Add filter) to create new filter
       b. Enter some name for the Add filter field
       c. Click the Type dropdown box, un-select the "Select
          all" option to clear out all the pre selected options
       d. Then select the following 9 options:
   
            Chassis Management Module
            Chassis Network Switch
            Chassis SAN Switch
             Flex Chassis
             Management Node
             PureApplication Service
             PureApplicaiton System
             PureApplication Systems Manager
             Switch
   
        e. Click Save
   
   You get the following error:
   CWPZC9021E: failed; Caused by:
   com.ibm.db2.jcc.am.SqlDataException: DB2 SQL Error:
   SQLCODE=-302, SQLSTATE=22001, SQLERRMC=null, DRIVER=4.14.113
   
   
    
   
   

LOCAL FIX

PROBLEM SUMMARY
 *  The DB2 table column width for the event filters was too small.
   
   
    
   
   

PROBLEM CONCLUSION
 *  A code fix is being provided in PureApplication System version
   2.2.0.0.
   
   
    
   
   

TEMPORARY FIX
