Title: IBM PI92081: UPDATE LIBRARY IDS MODULE AND UPDATE-WCM-LIBRARY-ID TASK SHOULD SUPPORT DELETED ITEMS - United States

Text:
  A FIX IS AVAILABLE
Fixes integrated in WebSphere Portal 8.0.0.1 Combined Cumulative Fixes [http://www-01.ibm.com/support/docview.wss?uid=swg24034497]



APAR STATUS
 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  The update library IDs module has been modified to support
   
   deleted items. ie:
   
   /wps/wcm/myconnect?MOD=updatelibraryids&library=TestLibrary
   
   
   
   The update-wcm-library-id ConfigEngine task, which is executed
   
   during migration, will also now process and update deleted
   
   items.
   
   
    
   
   

LOCAL FIX

PROBLEM SUMMARY
 *  The update library IDs module has been modified to support
   deleted items. ie:
   /wps/wcm/myconnect?MOD=updatelibraryids&library=TestLibrary

   during migration, will also now process and update deleted
