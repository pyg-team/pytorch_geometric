PROBLEM SUMMARY
 *  ****************************************************************
   * USERS AFFECTED:  All administrative console users of IBM     *
   *                  WebSphere Application Server V7.0           *
   *                  configuring server transports               *
   ****************************************************************
   * PROBLEM DESCRIPTION: PatternSyntaxException in               *
   *                      SystemOut.log on the administrative     *
   *                      console server transports panel         *
   *                      causing no validation to occur          *

   Address exclude fields is not correct.
