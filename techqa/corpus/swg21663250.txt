Title: IBM Security Bulletin: IBM SPSS SamplePower vsflex8l ActiveX Control ComboList Property Remote Code Execution Vulnerability (CVE-2013-6724) - United States

Text:
 SECURITY BULLETIN

SUMMARY
 There is security vulnerability with an ActiveX control shipped by IBM SPSS SamplePower Version 3.0.1. This is corrected in IBM SPSS SamplePower 3.0.1 IF1. 

VULNERABILITY DETAILS
VULNERABILITY DETAILS: 
CVE IDs: CVE-2013-6724 [http://cve.mitre.org/cgi-bin/cvename.cgi?name=CVE-2013-6724] 

 

DESCRIPTION: 

The latest available Interim Fix installer for IBM SPSS SamplePower provides a fix for getting or setting the "drop-down" list capability when editing a cell value. 

As of January 30, 2014, IBM has not received any reports of customer issues related to these security vulnerabilities. 

 

CVSS:
Using the Common Vulnerability Scoring System (CVSS) v2, the security ratings for these issues are:

CVSS Base Score: 7.5
CVSS Temporal Score: See https://exchange.xforce.ibmcloud.com/vulnerabilities/89279 [https://exchange.xforce.ibmcloud.com/vulnerabilities/89279] for the current score
CVSS Environmental Score*: Undefined
CVSS Vector: (AV:N/AC:L/Au:N/C:P/I:P/A:P)





ACKNOWLEDGEMENT 

These vulnerabilities were reported to IBM by Bluesea, working with ZDI. 




AFFECTED PRODUCTS AND VERSIONS
IBM SPSS SamplePower for Windows V3.0.1


REMEDIATION/FIXES
The recommended solution is to download and install the IBM SamplePower 3.0.1 FP1 and apply the Interim Fix 3.0.1-IM-C30SAMP-WIN32-IF01 . 

Fix* VRMF APAR How to acquire fix 3.0.1-IM-C30SAMP-WIN32-IF01 3.0.1.1 PI09800 3.0-IM-S3SAMPC-WIN32-FP001-IF001 [http://www-933.ibm.com/support/fixcentral/swg/selectFixes?parent=SPSS&product=ibm/Information+Management/SPSS+SamplePower&release=3.0.0.1&platform=Windows+32-bit,+x86&function=all] Fix Pack: In order to remove the old version of the Control, the user should un-install the current release, download the available Sample Power 3.0.1 Fix Pack 1, install and re-license with your current Authorization Code. 

1 Download the Interim Fix from IBM Fix Central (3.0.1-IM-C30SAMP-WIN32-IF01) 

2. Install the IBM SamplePower 3.0.1. FP1 on top of your existing installation if not already completed. 

3. Install the Interim Fix 3.0.1-IM-C30SAMP-WIN32-IF01 for SamplePower 3.0.1 on top of your existing installation. 

3. Reboot if necessary.
