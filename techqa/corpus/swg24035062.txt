ABSTRACT
 Cognos Disclosure Management 10.2.1 FP1 provides important corrections and enhancements for Cognos Disclosure Management 10.2.1 products and should be applied to all installations. 

DOWNLOAD DESCRIPTION
This Fix Pack is an update of IBM Cognos Disclosure Management 10.2.1 for existing IBM Cognos Disclosure Management customers. New features have been added to this Fix Pack. Task lists, OLAP Suppression, Chart generation improvements, migration tool for FSR queries, HFM tagging and taxonomy migration are new enhancements. In addition, this Fix Pack includes fixes addressing defects. 

PREREQUISITES
This Fix Pack requires an existing installation of Cognos Disclosure Management 10.2.1.

INSTALLATION INSTRUCTIONS
For environments that have UAC (User Access Control) enabled, please run Update.exe (Server) using the 'Run as Administrator' option.
