ERROR DESCRIPTION

   trace log indicates error:

   password are correct. Consult the trace and message logs for
   additional information.
   Nov 28, 2016 1:57:55 PM com.ibm.tivoli.tuam.integrator.main.
   CIMSIntegrator execute
   SEVERE: AUCIN0305E The CollectorInput stage failed during the
   following process: Initialization. Refer to the log messages
   and trace files to determine the problem and take the
   appropriate actions. If the problem persists, contact IBM
   Software Support.
   FINE: NOTE  ==>Service error occured while attempting to
   connect to the web service:
   com.sun.xml.internal.ws.client.ClientTransportException:
   HTTP transport error: javax.net.ssl.SSLHandshakeException:
   Remote host
   closed connection during handshake
   
   
    
   
   

LOCAL FIX
 *  we changed the startJobRunner.sh
   added -Dhttps.protocol=TLSv1 it to the JAVA_OPTS string (all on
   1 line):
     JAVA_OPTS="${JAVA_OPTS} -Xms50M -Xmx512M
   -Dhttps.protocol=TLSv1"

 *  SCCM vmware job file gives ssl handshake exception for
