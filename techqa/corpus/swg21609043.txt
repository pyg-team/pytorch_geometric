 TECHNOTE (FAQ)

QUESTION
 How can I have the Javadoc for the WebSphere Operational Decision Management API available in Eclipse when I write Java code? 

ANSWER
When installing WODM, the Javadoc and documentation are not installed locally by default. For offline referencing, you must download the documentation to the local system. The Javadoc for the APIs is located in the documentation bundle. 
To download the documentation locally on your computer, you must modify your installation by using IBM Installation Manager:


 1. Start Installation Manager and click Modify. 
 2. Select the package for which you want to install the documentation locally. For instance, IBM WebSphere Operational Decision Management for V7.5 or WebSphere Operational Decision management for V8.0. 
 3. After selecting the package, click Next, and then click Next again. 
 4. In the Modify Packages panel which allows you to select the features to install or remove, expand the two modules (Decision Server and Decision Center), and select the option Local Help System for each of these modules. 
 5. Click Next to perform the modification.

Installation Manager downloads the configuration files required to set up a local help system. 
 1. Once the installation is complete, go to the <WODM75>\WODM\doc directory. 
    There are 4 .bat files in the folder. 
 2. Run help_start.bat to start the local help. 
    A browser window with the following URL opens.http://127.0.0.1:3333/help/index.jsp 
 3. Follow the instructions under Procedure to download and install the content of the documentation to the local directory. 
 4. When the installation is completed, refresh the URL. 
    In the left hand frame, the documentation for Decision Center and Decision Server is displayed. 


In Eclipse, you can link the Javadoc with the libraries JAR file via URL or an archive file. 

Option 1: Javadoc URL  * For Rule Designer APIs (e.g. jrules-engine.jar), use:http://127.0.0.1:3333/help/topic/com.ibm.dserver.reference.studio/html/api/html


 * For Rule Execution Server APIs (e.g. jrules-res-execution.jar), use:http://127.0.0.1:3333/help/topic/com.ibm.dserver.reference.res/html/api/html 

 * For Decision Center APIs (e.g. jrules-teamserver.jar), use:http://127.0.0.1:3333/help/topic/com.ibm.dcenter.reference.rts/html/api/html
