APAR STATUS
 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  After you apply JR56456 [http://www-01.ibm.com/support/docview.wss?uid=swg1JR56456], the saved searches that contain fields
   related to process instance (business object) data no longer
   return results correctly in the federated mode in IBM Process
   Portal. The saved searches do not return any results and column
   names are not displayed.
   
   PRODUCTS AFFECTED
   IBM Business Process Manager (BPM) Advanced
   IBM BPM Standard
   IBM BPM Express

PROBLEM SUMMARY
 *  No additional information is available.
   
   
    
   
   

PROBLEM CONCLUSION
 *  A fix is available for IBM BPM V8.5.7.0 cumulative fix 2016.06
   that fixes saved searches in federated mode.
   
   If you don't already have IBM BPM V8.5.7.0 cumulative fix
   2016.06 installed, go to
   http://www.ibm.com/support/docview.wss?uid=swg24042266. [http://www.ibm.com/support/docview.wss?uid=swg24042266.]
   
   After IBM BPM V8.5.7.0 cumulative fix 2016.06 installed, search
   for JR56705 on Fix Central
   (http://www.ibm.com/support/fixcentral:
   
   1. Select IBM Business Process Manager with your edition from
   the product selector, the installed version to the fix pack
   level, and your platform, and then click Continue.
   2. Select APAR or SPR, enter JR56705, and click Continue.
   
   When you download fix packages, ensure that you also download
   the readme file for each fix. Review each readme file for
   additional installation instructions and information about the
   fix.
