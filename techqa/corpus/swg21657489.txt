Title: IBM Attempts to access RQM results in "you are not authorized to access Rational Quality Manager" - United States

Text:
Deferred operation failed; ORA-01536; tablespace; table space; login; authorized; access TECHNOTE (TROUBLESHOOTING)

PROBLEM(ABSTRACT)
 Attempting to access IBM Rational Quality Manager (RQM) results in error "you are not authorized to access Rational Quality Manager". 

SYMPTOM
The error is as follows: 


You have successfully logged into Jazz Team Server but you are not authorized to access Rational Quality Manager.
An error response was received from the Jazz Team server.
Status=400. Message: Deferred operation failed
Return to the Log in screen.



CAUSE
The database tablespace has run out of room to grow.


ENVIRONMENT
Oracle database



DIAGNOSING THE PROBLEM
The jts.log displays the error: 



CRJAZ1635W An error occurred while trying to cleanup expired floating licenses. 
com.ibm.team.repository.common.InternalRepositoryException: Deferred operation failed 
...
Caused by: com.ibm.team.repository.common.InternalRepositoryException: CRJAZ0413I Unable to insert audit trail for delete.Vendor specific exception
SQL: Insert Into {REPOSITORY.DELETED_ITEMS} ({ITEM_UUID}, 
{DELETED_WHEN}, {DELETED_BY}, {ITEM_TYPE_DBID}, {CONTENT_DELETED}, 
{STATES_DELETED}) VALUES (?, ?, ?, ?, ?, ?) 
SQL Exception #1 
SQL Message: ORA-01536: space quota exceeded for tablespace '<Table_Space_Name>' 

SQL State: 72000 
Error Code: 1536 
Class: 
com.ibm.team.repository.service.internal.dataaccess.delete.AuditDelete 



The qm.log displays the error: 


CRJAZ2388E Error encountered when synchronizing user data between servers in the background. The synchronizer will silently continue to attempt to synchronize the data. 




RESOLVING THE PROBLEM
The Oracle error message ORA-01536 means that the tablespace does not have enough space to grow. Contact your Database Administrator for further assistance. 

 

The Setting up an Oracle database topic in the IBM Rational Collaborative Lifecycle Management documentation provides setup information for Oracle databases for the CLM applications. Ensure that the AUTOEXTEND setting is set and there is space for growth.
