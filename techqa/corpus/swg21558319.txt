Title: IBM ORA-01795: maximum number of expressions in a list is 1000 - United States

STERLINGNFX TECHNOTE (TROUBLESHOOTING)

PROBLEM(ABSTRACT)
 ORA-01795: maximum number of expressions in a list is 1000 

SYMPTOM


There is a limitation with Oracle where it does not accept more than 1,000 items in an IN statement. This limitation may cause an issue under the following circumstances.

1. In Multi Channel Selling (MCS) version 7.0.1 only:

 * Updating partner status when hierarchy has more than 1,000 partners
 * PartnerAssignment API's which take an ArrayList of partnerKeys

2. A model with more than 1,000 properties when the "Properties" tab of this model is clicked

 

Error Message ORA-01795: maximum number of expressions in a list is 1000

 


RESOLVING THE PROBLEM


This has been fixed in HF11980 in MCS v7.0.1 and has been reconciled in later versions.
