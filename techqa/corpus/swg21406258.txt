Title: IBM Absence of IIS Management Compatibility impacts Agent functionality. - United States

Text:
5724B98ISTR ; 5724B97ISTR TECHNOTE (TROUBLESHOOTING)

PROBLEM(ABSTRACT)
 Absence of IIS Management Compatibility impacts Agent functionality. 

SYMPTOM
In case of IIS 7.x on Win2k8 and Win2k8 R2 RTM if “IIS 6 Management Compatibility" and “ISAPI Filters” role services are not installed then the Tivoli Enterprise Portal does not display data for the views/workspaces belonging to the following attribute groups: 

• HTTP Custom Error Messages
• Mime Types
• Web Service Extensions Restricted List
• IIS Application Pools Setting
• IIS Web Server Status
• IIS FTP Server Status
• IIS Version Information
• IIS Web Sites Detail
• IIS Web Server Bindings
• IIS Web Server Site Status
• IIS FTP Server Bindings 
• FTP Server Site Status 
• IIS FTP Sites Detail.


RESOLVING THE PROBLEM
While Installing IIS 7 install “IIS 6 Management Compatibility" and “ISAPI Filters” role services. For more information about IIS Installation using the user interface (UI) or command line option refer to the following links: 

http://technet.microsoft.com/en-us/library/cc771209(WS.10).aspx [http://technet.microsoft.com/en-us/library/cc771209(WS.10).aspx]
http://technet.microsoft.com/en-us/library/cc771209.aspx [http://technet.microsoft.com/en-us/library/cc771209.aspx]
