Title: IBM Security Bulletin: An unspecified vulnerability could cause a denial of service for WebSphere eXtreme Scale 8.6 (CVE-2015-4936) - United States

Text:
 SECURITY BULLETIN

SUMMARY
 An unspecified vulnerability in WebSphere eXtreme Scale 8.6 can cause a denial of service. 

VULNERABILITY DETAILS
CVEID: CVE-2015-4936 [http://cve.mitre.org/cgi-bin/cvename.cgi?name=CVE-2015-4936] 

DESCRIPTION: An unspecified vulnerability could cause a denial of service for WebSphere eXtreme Scale 8.6 servers. 

CVSS Base Score: 5 

CVSS Temporal Score: See https://exchange.xforce.ibmcloud.com/vulnerabilities/104410 [https://exchange.xforce.ibmcloud.com/vulnerabilities/104410] for the current score 

CVSS Environmental Score*: Undefined 

CVSS Vector: (AV:N/AC:L/Au:N/C:N/I:N/A:P)

AFFECTED PRODUCTS AND VERSIONS
WebSphere eXtreme Scale 8.6
