Title: IBM Security Bulletin: A security vulnerability has been identified in WebSphere Liberty Profile and WebSphere Application Server shipped with License Metric Tool, IBM Endpoint Manager for Software Use Analysis and IBM Tivoli Asset Discovery for Distributed - United States

Text:
 SECURITY BULLETIN

SUMMARY
 WebSphere Liberty Profile (which is a base component of IBM License Metric Tool v9 & and IBM Endpoint Manager for Software Use Analysis v9) and WebSphere Application Server (which is a base component of IBM License Metric Tool v7.5 and IBM Tivoli Asset Discovery for Distributed v7.5) are vulnerable to privilege escalation vulnerability. 

VULNERABILITY DETAILS
CVEID: CVE-2015-1927 [http://cve.mitre.org/cgi-bin/cvename.cgi?name=CVE-2015-1927] 
DESCRIPTION: IBM WebSphere Application Server could allow a remote attacker to gain elevated privileges on the system, caused by an application not having the correct serveServletsbyClassname setting. By a developer not setting the correct property, an attacker could exploit this vulnerability to gain unauthorized access. 
CVSS Base Score: 6.8 
CVSS Temporal Score: See https://exchange.xforce.ibmcloud.com/vulnerabilities/102872 [https://exchange.xforce.ibmcloud.com/vulnerabilities/102872] for the current score 
CVSS Environmental Score*: Undefined 
CVSS Vector: (AV:N/AC:M/Au:N/C:P/I:P/A:P) 

AFFECTED PRODUCTS AND VERSIONS
 

Principal Product and Versions Affected Supporting Product and Version IBM License Metric Tool v9
IBM Endpoint Manager for Software Use Analysis v9 WebSphere Liberty Profile 8.5 IBM License Metric Tool v7.5
IBM Tivoli Asset Discovery for Distributed v7.5 WebSphere Application Server 7 REMEDIATION/FIXES
- For products in version 9: 

Upgrade to version 9.2.1.0 or later: 

 * In IBM Endpoint Manager console, expand IBM License Reporting or IBM BigFix Inventory node under Sites node in the tree panel. 
 * Click Fixlets and Tasks node. Fixlets and Tasks panel will be displayed on the right. 
 * In the Fixlets and Tasks panel locate Upgrade to the newest version of License Metric Tool 9.x or Upgrade to the newest version of IBM BigFix Inventory 9.x fixlet and run it against the computer that hosts your IBM License Metric Tool or IBM Endpoint Manager for Software Use Analysis server.


Note: In an airgapped environment, you have to run BESAirgapTool and BESDownloadCacher first in order to update your site.  

- For products in version 7.5: 

Apply WebSphere Application Server fix described in the following technote: http://www-01.ibm.com/support/docview.wss?uid=swg21959083 [http://www-01.ibm.com/support/docview.wss?uid=swg21959083]. Please mind, that products in version v7.5 use WebSphere Application Server v7.
