Title: IBM Remote deployment fails with error KDY1024E, agent fails to start - United States

Text:
5724B97SOTR; libsapnwrfc TECHNOTE (TROUBLESHOOTING)

PROBLEM(ABSTRACT)
 Remote deployment fails with KDY1024E :The agent failed to respond to the command..sa did not start or stop agent, 

SYMPTOM
Command : CONFIGURE
Status : RETRYABLE
Retries : 3
TEMS Name : REMOTE_remotehost
Target Hostname:host:KUX
Platform : aix536
Product : sa
Version : 071101000
Error Message : KDY1024E: The agent failed to respond to the command /opt/IBM/ITM/bin/CandleAgent -h /opt/IBM/ITM -o INST start sa did not start or stop agent. The command returned a failure return code.


CAUSE
SAP RFC libraries required by the agent at 7.1.1 FP1 may be missing. 

SAP RFC Classic SDK is not supported from this version onwards. 


DIAGNOSING THE PROBLEM
Check host_sa_*.log in ITMHOME/logs directory, look for messages indicating missing RFC libraries, such as the following:
exec(): 0509-036 Cannot load program /opt/IBM/ITM/aix526/sa/bin/ksaagent because of the following errors:
0509-150 Dependent module libsapnwrfc.so could not be loaded.
0509-022 Cannot load module libsapnwrfc.so.
0509-026 System error: A file or directory in the path name does not exist.




RESOLVING THE PROBLEM
Download SAP NW RFC SDK. 

Access the Compatibility report for the SAP with Related URLs below.


Select Prerequisites tab-> Configuration Management section, and follow instructions in the footnote titled "SAP Agent 7.1.1 Fix Pack 1 or later" to download SAP NW RFC SDK.
