ABSTRACT
 MS12 provides a sample COBOL II program to print EVENT messages from any specified admin queue. 

DOWNLOAD DESCRIPTION
This SupportPac provides a sample COBOL II program to print EVENT messages from any specified admin queue. 

It runs as a z/OS Batch program and prints a report showing all fields of the EVENT messages from a specified queue and queue manager.

There are TWO versions within this SupportPac:

• Version 2.1 is for use with WebSphere MQ version 5.3.1
• Version 3.1 is for use with WebSphere MQ version 6.0

POSSIBLE USES
The sample can be used to print EVENT messages from any specified admin queue. 

The version for WebSphere MQ version 5.3.1 supports Configuration Events.
The version for WebSphere MQ version 6.0 supports Command Events, and 6.0 extensions to Configuration Events.

It shows how the variable length, unspecified order structures which form MQ event messages can be handled in a COBOL program.

NEW IN THIS RELEASE
Added a message to the start of each event to display the date and time the event occurred.
