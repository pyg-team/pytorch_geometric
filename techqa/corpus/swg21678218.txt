Title: IBM Security Bulletin: IBM FileNet System Monitor/IBM Enterprise Content Management System Monitor is potentially affected by security vulnerabilities in IBM® SDK, Java™ Technology Edition - United States

Text:
 SECURITY BULLETIN

SUMMARY
 Multiple security vulnerabilities exist in IBM® SDK, Java™ Technology Edition that is shipped with IBM FileNet System Monitor/IBM Enterprise Content Management System Monitor product. 

VULNERABILITY DETAILS

The IBM FileNet System Monitor/IBM Enterprise Content Management System Monitor product ships with IBM® SDK, Java™ Technology Edition. The IBM® SDK, Java™ Technology Edition is based on the Oracle version of the SDK/JRE. In April 2014, Oracle has released critical patch updates (CPU) that contain security vulnerability fixes. These issues are present in IBM® SDK, Java™ Technology Edition and potentially affect the System Monitor product.

CVEID: CVE-2014-0878 [http://cve.mitre.org/cgi-bin/cvename.cgi?name=CVE-2014-0878]
DESCRIPTION: A vulnerability in the IBMSecureRandom implementation of the IBMJCE and IBMSecureRandom cryptographic providers potentially allows an attacker to predict the output of the random number generator under certain circumstances. Please refer to the Workarounds & Mitigation section below for additional information.
CVSS Base Score: 5.8
CVSS Temporal Score: See https://exchange.xforce.ibmcloud.com/vulnerabilities/91084 [https://exchange.xforce.ibmcloud.com/vulnerabilities/91084] for the current score
CVSS Environmental Score*: Undefined
CVSS Vector: (AV:N/AC:M/Au:N/C:P/I:P/A:N) 

CVEID: CVE-2014-0460 [http://cve.mitre.org/cgi-bin/cvename.cgi?name=CVE-2014-0460]
DESCRIPTION: An unspecified vulnerability related to the JNDI component has partial confidentiality impact, partial integrity impact, and no availability impact.
CVSS Base Score: 5.8
CVSS Temporal Score: See https://exchange.xforce.ibmcloud.com/vulnerabilities/92482 [https://exchange.xforce.ibmcloud.com/vulnerabilities/92482] for the current score
CVSS Environmental Score*: Undefined
CVSS Vector: (AV:N/AC:M/Au:N/C:P/I:P/A:N) 

AFFECTED PRODUCTS AND VERSIONS
The following versions of the System Monitor product are potentially affected:

IBM FileNet System Monitor v4.5.0
IBM Enterprise Content Management System Monitor v5.1.0

REMEDIATION/FIXES
For IBM FileNet System Monitor v4.5.0:
Upgrade to the platform specific version of IBM® SDK, Java™ Technology Edition that is available in IBM FileNet System Monitor v4.5.0.3 Interim Fix 1.

For IBM Enterprise Content Management System Monitor v5.1.0:
Upgrade to the platform specific version of IBM® SDK, Java™ Technology Edition that is available in IBM Enterprise Content Management System Monitor v5.1.0.3 Interim Fix 1.

If you need further assistance, please contact IBM Support.

WORKAROUNDS AND MITIGATIONS
None known
