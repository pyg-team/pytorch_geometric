ERROR DESCRIPTION
 *  After applying APAR PM56986 [http://www-01.ibm.com/support/docview.wss?uid=swg1PM56986], if the following conditions are
   met, the FILSZ= value for index sorts are changed:
     - 'INPUT DFSURWF1' is specified
     - a non-unique key and an OVFLW DD are defined for a secondary
   index.
