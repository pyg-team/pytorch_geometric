websphere partner gateway system requirements requirement solaris V6.2 6.2 PRODUCT DOCUMENTATION

ABSTRACT
 This document lists the hardware and software requirements for WebSphere® Partner Gateway, Version 6.2, on the Solaris operating system.

This document describes the minimum product levels that need to be installed before opening a problem report with the WebSphere Partner Gateway Support team. Because other products frequently ship fixes, updates, and new releases, testing every configuration is not possible. In general, you can install and run with updates to supported products if those updates are forward compatible. 

The following list identifies the products that are supported when WebSphere Partner Gateway runs on the HP-UX operating system. 


Supported Operating Systems [/support/docview.wss?uid=swg27014202&amp;aid=3]  * Sun Solaris 9 with latest recommended Patch Cluster (Sun product website [http://www.sun.com/products/index.jsp]) 
 * Sun Solaris 10 with latest recommended Patch Cluster 

Hardware requirements [/support/docview.wss?uid=swg27014202&amp;aid=3]  * Solaris SPARC workstation at 750 MHz or faster 
 * Minimum 2 GB physical memory 
 * Minimum 1.3 GB (1350 MB) free disk space for installation. The installer also requires approximately 600 MB of space in the /tmp directory during installation. 
 * Additional disk space for document storage, 30 GB recommended 
 * CD-ROM drive 

Base application server requirement [/support/docview.wss?uid=swg27014202&amp;aid=3] One of the following:  * IBM WebSphere® Application Server v6.1 with Fix Pack 19 (WebSphere Partner Gateway simple deployment only) 
 * IBM WebSphere Application Server Network Deployment 6.1 with Fix Pack 19 [http://www-01.ibm.com/support/docview.wss?rs=180&uid=swg24020064] [http://www-1.ibm.com/support/docview.wss?rs=180&uid=swg24017307], Fix Pack 21 [http://www.ibm.com/support/docview.wss?rs=180&uid=swg24020708], Fix Pack 25, [http://www-01.ibm.com/support/docview.wss?rs=180&uid=swg24023274]Fix Pack 27, [http://www.ibm.com/support/docview.wss?rs=180&uid=swg24024195]Fix Pack 29 [http://www-01.ibm.com/support/docview.wss?rs=180&uid=swg24025185], Fix Pack 31 [http://www.ibm.com/support/docview.wss?rs=180&uid=swg24026350]. Note: While using IBM WebSphere Application Server Network Deployment 6.1 with FixPack 25, it is mandatory to install PK86552 [http://www.ibm.com/eserver/support/fixes/fixcentral/swgquickorder?fixes=6.1.0.23-WS-WAS-IFPK86552&productid=WebSphere%20Application%20Server&brandid=5] before installing WebSphere Partner Gateway v6.2Fp1.
