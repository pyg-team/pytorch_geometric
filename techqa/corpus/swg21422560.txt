Title: IBM "The account has expired and is no longer valid., error code: FN_IS_RA_10901" error found in ISRA 3.4 log - United States

Text:
Account expired; No longer valid; FN_IS_RA_10901 TECHNOTE (TROUBLESHOOTING)

PROBLEM(ABSTRACT)
 Error message "The account has expired and is no longer valid., error code: FN_IS_RA_10901" keeps showing in ISRA logs 

SYMPTOM
The error message found in ISRA log is as below

CAUSE
This user was created with a typo and configured in ISRA custom application and is not used for logon. However, after a specific period of time, this user got expired and every logon of ISRA custom application caused a verification of this user and that's why this error message was thrown out repeatedly


ENVIRONMENT
ISRA 3.4 FP4



RESOLVING THE PROBLEM
Reactivate this user in Xapex, IS server side resolved this issue
