OVERVIEW



IBM WebSphere Integration Developer is the "one tool, one set of skills" solution for end-to-end integration in your service-oriented architecture (SOA). Based on Eclipse, WebSphere Integration Developer is a tool for building SOA-based business process management (BPM) and integration solutions across WebSphere Process Server, WebSphere Enterprise Service Bus, and WebSphere Adapters. It simplifies integration with rich features that accelerate the adoption of SOA by rendering existing IT assets as service components, encouraging reuse and efficiency.

To access a wide variety of technical resources for this product, see the WebSphere Integration Developer Version 7.0 information center [http://publib.boulder.ibm.com/infocenter/dmndhelp/v7r0mx/topic/com.ibm.wbit.help.mainwelcome.doc/topics/welcome.html].

DOWNLOADING AND ASSEMBLING THE PRODUCT
 1. Navigate your browser to the Find Downloads and Media [http://www-112.ibm.com/software/howtobuy/passportadvantage/paocustomer/sdma/SDMA?PO=DOWNLOAD_FIND_BY_PART_NUMBER] Web page on the IBM Passport Advantage Online Web site.
    
    
 2. Select the Find by part number search option under Download finder options.
    
    
 3. Search for each downloadable image by part number.
    
    
 4. Download all of the required images.
    
    
 5. Use a file extracting utility that supports large file extractions to unpack each of the images into a single, temporary directory on your system. It is important that you extract the required packages to the same folder on your local system.
    
    
 6. Optionally, open the Quick Start Guide located in the Quick Start image to access documentation to help with product installation.
