ABSTRACT
 Detailed System Requirements (DSR) contain the supported operating system requirements, hardware requirements, software requirements, and other related information for a product. From this document you can select DSR for CICS Transaction Server (CICS TS) for z/OS 5.4 5.3, 5.2, 5.1, 4.2, and 4.1, and CICS TS for z/VSE 2.1 and CICS TS for VSE/ESA 1.1.1.
