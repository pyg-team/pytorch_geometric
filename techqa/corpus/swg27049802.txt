Title: IBM PureApplication Software Release Note: Deployment fails with an error: Unable to create virtual adapters on Virtual I/O Server - United States

Text:
sw2230rlsnotes sw224rlsnotes RELEASE NOTES

ABSTRACT
 You attempt to deploy a pattern, but the deployment fails with error CWZIP8003W. 

CONTENT
Problem: When you attempt to deploy a pattern, the deployment fails with the following errors:


CWZIP2016E The deploy operation failed: Deployment error: CWZIP8003W The virtual machine is in an error state - Unable to create virtual adapters on Virtual I/O Server <ID>. Run the Verify Environment tool. You can either click the Verify Environment button on the GUI Home page or type the powervc-validate command on the command line.

Resolution:
Deploy the pattern again.
