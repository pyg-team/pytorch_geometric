APAR STATUS
 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  A Java Runtime Environment (JRE) is embedded into the IBM MQ
   installation directory structure, and this level of JRE shipped
   is backlevel.
   
   
    
   
   

LOCAL FIX

PROBLEM SUMMARY
 *  ****************************************************************
   USERS AFFECTED:
   The components of IBM MQ v7.5 which utilise the embedded JRE
   include:
   
   - Users of the IBM MQ classes for Java and the IBM MQ classes
   for JMS who use the JRE shipped by IBM MQ to run their
   applications
   
   - IBM MQ Managed File Transfer Edition
   
   - IBM MQ Advanced Message Protocol (AMQP)
   
   - IBM MQ Telemetry Transport (MQTT)
   
   - IBM MQ Explorer
   
   
   Platforms affected:
   AIX, HP-UX Itanium, Linux on x86, Linux on x86-64, Linux on
   Power, Linux on zSeries, Solaris x86-64, Solaris SPARC, Windows
   
   ****************************************************************
   PROBLEM DESCRIPTION:
   The IBM MQ installation includes an embedded Java Runtime
   Environment (JRE).  The level of JRE shipped by IBM MQ was
   backlevel.
   
   
    
   
   

PROBLEM CONCLUSION
 *  The level of JRE shipped with IBM MQ has been updated as
   follows:
   
   IBM MQ v7.5:
   
     6.0.16.65  -  AIX, Linux (x86-64, x86-32, ppc and zLinux),
                 Windows, HP-UX (Itanium), Solaris (x86 and SPARC)
   
   ---------------------------------------------------------------
   The fix is targeted for delivery in the following PTFs:
   
   Version    Maintenance Level
   v7.5       7.5.0.9
   v9.1 CD    9.1.1

   'WebSphere MQ Recommended Fixes'

   If the maintenance level is not yet available information on
