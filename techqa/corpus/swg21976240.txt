Title: IBM Retain zoom level of Map Widget in Mobile Browser upon Login - United States

Text:
DASHL2; DASHL2CONF; map widget; zoom; mobile browser TECHNOTE (TROUBLESHOOTING)

PROBLEM(ABSTRACT)
 A map displayed in the Map widget does not retain its zoom level in Mobile Browser. 

SYMPTOM
Zoom gets lost upon login.


CAUSE
Necessary settings not configured.

ENVIRONMENT
DASH displayed in Mobile browser

RESOLVING THE PROBLEM
Enabling 'Save user settings for Pan and Zoom' will cause the map to appear differently for each user, as it retains the last state.
