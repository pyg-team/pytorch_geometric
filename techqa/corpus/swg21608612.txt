Title: IBM A NullPointerException error occurs when accessing Workplace Site Preferences in FileNet Application Engine - United States

Text:
AE; app_engine.war TECHNOTE (TROUBLESHOOTING)

PROBLEM(ABSTRACT)
 When accessing the Site Preferences in Workplace, a NullPointerException error occurs. Reverting to earlier versions of the Site Preferences for Workplace does not have any effect and the preferences.xml for the Site Preferences is well formed. 

CAUSE
The default SitePreferences.xml located in the<deployed location>\Workplace\WEB-INF\xml directory for Workplace, has been renamed from the default SitePreferences.xml to a different file name.

RESOLVING THE PROBLEM



 1. Check theSitePreferences.xml file in the <deployed location>\Workplace\WEB-INF\xml directory to see if it has the correct file name
    
    
 2. Verify the file is a well formed xml file by opening the file with an xml editor
    
    
 3. If the SitePreferences.xml file name was corrected, restart the FileNet Application Engine JVM
