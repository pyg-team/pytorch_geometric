PROBLEM(ABSTRACT)
 This document covers the product life cycle for IBM HTTP Server, from when the product is generally available as part of IBM WebSphere Application Server solutions, to the time the product is no longer supported. 

CAUSE
The following table displays both the General Availability (GA) and End of Support (EOS) dates for IBM HTTP Server (IHS).
Unless otherwise stated, the End of Support (EOS) date for IBM HTTP Server (IHS), will be the same as the EOS date for WebSphere Application Server (WAS) of the same version.
