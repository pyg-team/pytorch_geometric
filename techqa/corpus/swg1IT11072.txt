Title: IBM IT11072: UPDATE JRE SHIPPED IN MQ 8.0.0.4 TO 7.1.3.10 and 7.0.9.10 - United States

Text:
 SUBSCRIBE
You can track all active APARs for this component.



APAR STATUS
 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  The WebSphere MQ V8 installation contains a version 7
   Java Runtime Environment (JRE), supplied within the JRE
   install component.
   New revisions of this JRE are released periodically to address
   program defects, and products which embed the JRE should
   be updated accordingly.
   
   
    
   
   

LOCAL FIX

PROBLEM SUMMARY
 *  ****************************************************************
   USERS AFFECTED:
   The JRE which is embedded into WebSphere MQ is used
   to run the following Java components of the product:
   
     WebSphere MQ Managed File Transfer
     WebSphere MQ Explorer
     WebSphere MQ Extended Reach (MQXR) service
   
   
   Platforms affected:
   AIX, HP-UX Itanium, Linux on Power, Linux on x86, Linux on
   x86-64, Linux on zSeries, Solaris SPARC, Solaris x86-64,
   Windows, Linux on S390
   
   ****************************************************************
   PROBLEM DESCRIPTION:
   This APAR updates the Java Runtime Environment (JRE) supplied
   with WebSphere MQ 8.0 to the level
   
   7.1.3.10 for Windows, Linux and AIX and 7.0.9.10 for Solaris and
   HP-UX.
   
   
    
   
   

PROBLEM CONCLUSION
 *  This APAR updates the Java Runtime Environment (JRE) supplied
   with WebSphere MQ 8.0 to the level:
   
   7.1.3.10 for Windows, Linux and AIX and 7.0.9.10 for Solaris and
   HP-UX.
