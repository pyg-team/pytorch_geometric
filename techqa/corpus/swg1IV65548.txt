APAR STATUS
 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  Environment:
   ------------
   BS 7.1.1.12
   
   Description:
   ------------
   BMXAA4214E error when selecting a communication template with a
   person group that has on people in it
   
   Steps to replicate:
   -------------------
   Go To  Administration  Communication Templates
   Create a new communication:
   Template - TEST
   Applies To ? WORKORDER
   Accessible From ? ALL
   Send From - maxadmin
   Subject - comm sent from :wonum
   
   Open the Recipients tab
   Expand the Roles section
   Click New Row
   Open the Detail menu next to the Role textbox on
   the new row, Go To Roles
   
   Click New Role
   Role ? testrole
   Type ? Email address
   Value - <email address>
   Click Save, then Return With Value
   
   Check the To checkbox for the new Role
   Expand the Person Groups section
   Click New Row
   Open the Detail menu next to the Person Group
   textbox on the new row, Go To Person Groups
   
   Click New Person Group
   Person Group ? testpg
   Save and return with value
   Check the Broadcast and cc boxes for the
   Person Group
   
   Save and activate the Comm Template
   
   Go to Work Orders  Work Order Tracking
   Open a Work Order
   Select Action Create  Communication
   Template  Select Value
   Select the template TEST
   The following error appears:
   
   BMXAA4214E - An unknown error has occurred.
   Please contact your system administrator for assistance.
   
   
    
   
   

LOCAL FIX
 *  select the template twice - the second time it will be filled
