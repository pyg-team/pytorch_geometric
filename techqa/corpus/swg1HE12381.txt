APAR STATUS
 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  When the application server is upgraded to use Java version
   1.7, timer tasks do not fire automatically according to
   schedule.
   
   
    
   
   

LOCAL FIX
 *  Modify the application server to use Java version 1.6.
   
   Optionallu, you can manually trigger the timer tasks from the
   Admin > Timer Tasks panel.
   
   
    
   
   

PROBLEM SUMMARY
 *  Timer tasks are not triggered automatically when using Java
   version 1.7. They can, however, be manually triggered from the
   Atlas Admin UI.
   
   
    
   
   

PROBLEM CONCLUSION
 *  This fix is available now from FixCentral in 6.0.3.3
