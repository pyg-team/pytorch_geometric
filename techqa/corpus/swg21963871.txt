Title: IBM Content Manager build {Build} failed to start - United States

Text:
 TECHNOTE (TROUBLESHOOTING)

PROBLEM(ABSTRACT)
 Attempting to access the Tivoli Common Reporting portal by clicking the "Common Reporting" link, fails with the error "Content Manager build {Build} failed to start". 

CAUSE
The nofile setting at the operating system is set too low.

DIAGNOSING THE PROBLEM
Review the output of the command 'ulimit -a' and compare the current settings to the settings seen in the "Hardware and software requirements" section of the JazzSM Installation documentation.

RESOLVING THE PROBLEM
Increase the operating system ulimit settings as documented in the JazzSM Installation documentation.
