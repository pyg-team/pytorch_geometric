Title: IBM OutOfMemoryError at performance test run - United States

Text:
RATLC02642688; thread TECHNOTE (TROUBLESHOOTING)

PROBLEM(ABSTRACT)
 Attempts to run an IBM Rational Performance Tester test results in error Failed to create a thread: retVal -1073741830, errno 12.


SYMPTOM
The full error message is as follows.



java.lang.OutOfMemoryError: Failed to create a thread: retVal
-1073741830, errno 12




RESOLVING THE PROBLEM
Install the Agent Controller on the workbench computer.
