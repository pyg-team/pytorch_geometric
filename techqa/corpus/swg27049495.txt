OVERVIEW
IBM Watson Explorer Version 8.2-6 is a fix pack that addresses security issues. See the following Security Bulletins for more information: 

 *  Security Bulletin: Vulnerability in OpenSSL affects Watson Explorer (CVE-2016-7055) [http://www.ibm.com/support/docview.wss?uid=swg21999447] Security Bulletin: Multiple Vulnerabilities Affect Watson Explorer (CVE-2016-8610, CVE-2016-2177, CVE-2016-2183, CVE-2016-5597, CVE-2016-5542) [http://www.ibm.com/support/docview.wss?uid=swg21995749] 

For additional information about IBM Watson Explorer Version 8.2.-6, see the Release Notes [http://www.ibm.com/support/knowledgecenter/SS8NLW_8.2.0/com.ibm.swg.im.infosphere.dataexplorer.product.doc_8.2.2/HTML/ch01.html].
