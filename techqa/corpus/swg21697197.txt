 TECHNOTE (FAQ)

THIS DOCUMENT APPLIES ONLY TO THE FOLLOWING LANGUAGE VERSION(S):
 English 

QUESTION
 How to convert to epochtime from $DateTime varable in rule file 

CAUSE
$_DT = datetotime( $DateTime, "%Y/%m/%d %T")may cause an error in log file below.
2015-02-13T10:42:03: Error: E-ETC-102-036: Fail to parse date string:
using format: %Y/%m/%d %T. Error code: U_PARSE_ERROR
2015-02-13T10:42:03: Warning: W-UNK-000-000: Failed to convert "" to
DATETIME using format "%Y/%m/%d %T" - using current time 



ANSWER
Please try below.. 

$dt_=datetotime($DateTime, "yyyy/MM/dd H:mm:ss zzzz") 

This may not work well in an old probe support environment 7.3.0.* 

I confirmed it works well on my Windows 2012 environment
