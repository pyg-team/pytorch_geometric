Title: IBM IV55650: LIST BOX IS NOT GRAYED OUT. - United States

Text:
 SUBSCRIBE
You can track all active APARs for this component.



APAR STATUS
 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  List box is not grayed out even after clicking Mark For
   Delete icon for the row.
   
   I am able to recreate the problem in L2 MAM 7.5.0.5 test
   environment.
   
   STEPS TO REPRODUCE:
   
   1) From "Go To"-"System Configuration"-"Platform
   Configuration"-"Domains", create a new Numeric
   domain with the following values.
   
   Domain : SEVERITY
   Description : Condition Severity
   Data Type : INTEGER
   Length    : 12
   
   2) Then add the following entries and save the record.
   
   Value : 1     Description : System Down
   Value : 2     Description : Urgent
   Value : 3     Description : Normal
   
   3) From "Go To"-"System Configuration"-"Platform
   Configuration"-"Database Configuration", open "CONDITION".
   
   4) On "Attributes" tab, add a new attribute with the
   following values.
   
   Attribute : SEVERITY
   Description : Condition Severity
   Type      : INTEGER
   Domain    : SEVERITY
   
   5) Turn Admin Mode ON and apply the configuration changes.
   Then turn Admin Mode OFF.
   
   6) From "Go To"-"System Configuration"-"Platform
   Configuration"-"Application Designer", open "CONDEXPMGR".
   
   7) Add a new listbox control in "Details" section and
   specify the following values for its properties.
   
   Attribute : SEVERITY
   
   Then save the record.
   
   8) From "Go To"-"Administration"-"Conditional Expression
   Manager", create a new condition with the following values.
   
   Condition : PMR77381
   Description : Condition for PMR 77381
   Type : Expression
   Expression : 1=1
   
   Then save the record.
   
   9) Click trash (Mark For Delete) icon at the right side of
   the entry for the condition "PMR77381".
   
   10) You can see all the fields has been grayed out now but
   only list box field is not grayed out.
   
   CURRENT ERRONEOUS RESULT:
   Only list box field is not grayed out after clicking trash
   icon.
   
   EXPECTED RESULT:
   List box field should be grayed out like all the other
   fields when clicking trash button.
