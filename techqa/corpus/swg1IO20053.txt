ERROR DESCRIPTION
 *  Testing using APT OOTB application.
   Using the Demo.
   Go to TMWeb and set the following.
   
   1. VScan task to Store by Station ID
   2. Batch Profiler task to Queue by Station
   3. Verify task to Queue by Station
   4. Set both the Verify and Batch Profiler Shortcuts to be Mode =
   Manual
   5. Launch DotScan and logon as Admin to Station 1.
       Select Demo to pull in images to the batch.
       Launch DotScan and logon as Admin to Station 2.
       Select Demo to pull in images to the batch.
       Go to TMWeb and set the batch for Station 1 to be on HOLD
       Launch DotScan and logon as Admin to Station 2
       Select BatchProfiler task shortcut.
       See the HOLD batch from Station 1 showing up in DotScan
   eventhough logon as Station 2.
   6. Same behavior as in DotEdit.
       HOLD batch from Station 2 is showing up when logon to
   DotEdit with Station 1
