Title: IBM Security Bulletin: Denial of Service vulnerability in Apache Commons FileUpload affects IBM Business Process Manager (BPM) - United States

Text:
 SECURITY BULLETIN

SUMMARY
 A security vulnerability exists in the open source library Apache Commons FileUpload that is shipped with, and used by, the IBM Business Process Manager products. 

VULNERABILITY DETAILS
By sending a specially crafted request, an attacker might exploit this vulnerability to cause the application to enter into an infinite loop.
CVE-2014-0050 [http://cve.mitre.org/cgi-bin/cvename.cgi?name=CVE-2014-0050] 
CVSS Base Score: 5.0
CVSS Temporal Score: https://exchange.xforce.ibmcloud.com/vulnerabilities/90987 [https://exchange.xforce.ibmcloud.com/vulnerabilities/90987] 
CVSS Environmental Score*: Undefined 
CVSS Vector: (AV:N/AC:L/Au:N/C:N/I:N/A:P)

AFFECTED PRODUCTS AND VERSIONS
IBM Business Process Manager Advanced 7.5.x, 8.0.x, 8.5.x
IBM Business Process Manager Standard 7.5.x, 8.0.x, 8.5.x
IBM Business Process Manager Express 7.5.x, 8.0.x, 8.5.x
