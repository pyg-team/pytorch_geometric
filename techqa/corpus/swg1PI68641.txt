ERROR DESCRIPTION
 *  You have installed IBM SPSS Statistics 24 on a Macintosh OSX
   operating system some time ago.
   For example on Mac OSX 10.11.x El Capitan or on Mac OSX 10.10.x
   Yosemite.
   You download Fix pack: 24.0-IM-S24STATC-Mac-FP001
   IBM SPSS Statistics Client 24.0 Mac Fix Pack 1 from IBM
   FixCentral
   You copy the fixpack file on the MacOSX desktop
   24.0-IM-S24STATC-Mac-FP001.dmg
   Then, double click on the file unpack the installer and start
   installation
   You entered your sudo password when the installer prompted for
   the password.
   Statistics 24 is installed on default path
   Applications/IBM/SPSS/Statistics/24
   
   
   
   After entering the sudo password the installer proceed fine
   until to the last screen.
   
   You encounter the following behavior:
   
   On last dialog of the installer "Install complete" you see below
   message:
   
   *******************
   The installation of IBM SPSS Statistics 24 Patch is finished,
   but some errors occurred during the install. Please see the
   installation log for details.
   *******************
   
   When you press "Done" the installer is finished and when you
   launch the program SPSS 24 on this Mac it runs fine and it shows
   as installed version 24.0.0.1
   
   On the temporary folder of this Mac there is no install log but
   when you open the Console
   
    - > Applications/Utilities/Console
   
   you can click on "All Messages" below System Log Queries,
   Then on Search section  type "SPSS"
   to filter for SPSS messages
   You may see similar messages on your Console log:

LOCAL FIX
 *  This was reported to IBM SPSS Development. The issue was
   reviewed and the error messages displayed on console and last
   screen of the FixPack installer are not relevant, The FixPack is
   installed successful despite of the message and release
   24.0.0.1 will work normally on Macintosh.
   However, the FixPack 24 FP1 for Macintosh has been refreshed on
   8. September 2016 and if you download it now you will no longer
   see the error on installer dialog.
