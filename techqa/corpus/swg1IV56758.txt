Title: IBM IV56758: UNABLE TO OPEN THE DEPLOYMENT EDITOR WITH JAVA UPDATE 51 OR LATER - United States

Text:

APAR STATUS
 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  If you have updated your desktop java to update 51 or later you
   will be unable to open the deployment editor with default
   security settings. You will see the following error
   message in the java console:
   
   Missing required Permissions manifest attribute in main jar:
   
   http://x.x.x.x/adminconsole/lib/7.x.x.xxxxxx_q1labs_deditor.jar [http://x.x.x.x/adminconsole/lib/7.x.x.xxxxxx_q1labs_deditor.jar]
   
   
    
   
   

LOCAL FIX
 *  Where possible you can add the console url to the Exception Site
   list in java settings.
   
   
    
   
   

PROBLEM SUMMARY
 *  This issue has been resolved in 7.1 MR2 Patch 6 (7.1.0.809045)
