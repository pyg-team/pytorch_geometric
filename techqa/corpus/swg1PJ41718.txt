APAR STATUS
 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  Administration Console for Content Platform Engine (ACCE) will
   encounter the following error:
   
   "Error
   FNRAC1001E
   An unexpected error occurred.
   
   User response: Check the error details for more information and
   contact IBM Software Support if necessary.
   
   Exception details:
   undefined"
   
   when trying to unset the "Property Default Object" whose value
   is a custom object-value property.
   
   To set and unset the property:
   1) Create a Property Template of Object Type myPTObject1
      Click Set Other Attributes checkbox and Select Read-Write on
   Settability drop down list and Modify all properties on Access
   Rights listing
   2) Click on OS->Data Design->Classes->Document and goto Property
   Definitions tab
   3) Click Add button
   4) Select myPTObject1 from the list and click OK
   5) Click on myPTObject1 (hyperlink) which will bring up Property
   Definition panel
       On Required Class drop down menu select Custom Object
       On Target Access Required list - Select Modify all
   properties
   6) Click Save
   7) Goto OS->Browse->Root Folder
   8) Click on Actions->New Custom Object and Create a New Custom
   Object myCustomObject1
   9) Open the myCustomObject1 and Select Actions->Copy Object
   Reference
   10) Goto OS->Data Design->Classes->Document Properties tab on
   Property Definitions value drop down arrow, Select myPTObject1
   from list. myPTObject1 Properties panel will appear and scroll
   down toward the bottom of the list and select Property Default
   Object textbox and select Paste Object.
   11) Goto the Open Document tab(should have a dirty bit * on it)
   and Click Save
   12) Once Save is done(dirty bit gone) go back to myPTObject1
   Properties panel and UNSET the value on Property Default Object
   textbox
   13) Go back to Open Document tab(should have a dirty bit * on
   it) and Click Save
   
   
    
   
   

LOCAL FIX
 *  Use Filenet Enterprise Manager (FEM) to unset the "Property
   Default Object".

   * FNRAC1001E                                                   *
   * An unexpected error occurred.                                *
   *                                                              *
   * User response: Check the error details for more information  *
   * and                                                          *
   * contact IBM Software Support if necessary.                   *
   *                                                              *
   * Exception details:                                           *
   * undefined"                                                   *
   *                                                              *
   * when trying to unset the "Property Default Object" whose     *
   * value                                                        *
   * is a custom object-value property.                           *
   *                                                              *
   * To set and unset the property:                               *
   * 1. Create an object value property and add it to the base    *
   * "Document" class.                                            *
   * 2. Create a custom object.                                   *
   * 3. Copy a reference to the custom object you just created.   *
   * 4. On the "Document" class tab, click on the properties tab  *
   * 5. Click on the dropdown menu in the "Property Value"        *
   * column.                                                      *
   * 6. Select the object value property that you created in step *
   * 1                                                            *
   * 7. A new tab will appear to edit the property. Scroll down   *
   * to                                                           *
   * the bottom to "Property Default Object". Click on the        *
   * dropdown                                                     *
   * in the column to the right of the "Property Value" column    *
   * and                                                          *
   * select "Paste Object"                                        *
   * 8. Switch to the "Document" class tab, click save, then      *
   * switch                                                       *
   * back to the previous tab                                     *
   * 9. Click on the drop down in the column on the right of the  *
   * "Property Value" column and select "Unset Value"             *
   * Note the "undefined" error message and the operation fails.  *
   ****************************************************************
   * RECOMMENDATION:                                              *
   * Customers should update to 5.2.0.3-P8CPE-FP003 and higher.   *
   ****************************************************************
