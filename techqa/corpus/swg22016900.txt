Title: IBM Security Bulletin: Multiple vulnerabilities in IBM Java Runtime affect IBM® SPSS Statistics Server - United States

Text:
 SECURITY BULLETIN

SUMMARY
 There are multiple vulnerabilities in IBM® Runtime Enviornment Java™ Technology Edition, Version 6, 7, 7R1, and 8 used by IBM® SPSS Statistics Server. These issues were disclosed as part of the IBM Java SDK updates in April 2018. 

VULNERABILITY DETAILS
CVEID: CVE-2018-2783 [http://cve.mitre.org/cgi-bin/cvename.cgi?name=CVE-2018-2783]
DESCRIPTION: An unspecified vulnerability related to the Java SE Security component could allow an unauthenticated attacker to cause high confidentiality impact, high integrity impact, and no availability impact.
CVSS Base Score: 7.4
CVSS Temporal Score: See https://exchange.xforce.ibmcloud.com/vulnerabilities/141939 [https://exchange.xforce.ibmcloud.com/vulnerabilities/141939] for the current score
CVSS Environmental Score*: Undefined
CVSS Vector: (CVSS:3.0/AV:N/AC:H/PR:N/UI:N/S:U/C:H/I:H/A:N)

AFFECTED PRODUCTS AND VERSIONS
IBM SPSS Statistics 21.0.0.2
IBM SPSS Statistics 22.0.0.2
IBM SPSS Statistics 23.0.0.3
IBM SPSS Statistics 24.0.0.2
IBM SPSS Statistics 25.0.0.1

* Note: 

SPSS Statistics Server V21 & V22 users can receive a free complementary upgrade to SPSS Statistics Server 25 edition. This will include upgrades of both the version 21/22 Server and all connected version 21/22 clients. The upgrade will be a 1:1 upgrade, meaning equal server PVUs, client users, and module/bundle functionality. The offer is available now until 30 Sep 2018. SPSS Statistics 25 is a major upgrade from versions 21 or 22, with major features, better reliability, and improved security. 

Contact SPSS Statistics Product Support [https://www.ibm.com/mysupport/s/topic/0TO500000001xpQGAQ/1742fe293dcd33daef1b0d51618ce39dspss-statistics?language=en_US&productId=01t50000004XAyOAAW], login with your IBMid, and open a new Support ticket citing this Security Bulletin to take advantage of this offer.
