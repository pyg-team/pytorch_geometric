Title: IBM LO49258: E71 UNABLE TO SUCCESSFULLY INSTALL TRAVLER CLIENT. - United States

Text:
 SUBSCRIBE
You can track all active APARs for this component.



APAR STATUS
 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  Customer reported that they are unable to install the Lotus
   Notes Traveler client on their E71 devices.  The device appears
   to hange during the install/config process.  Other Nokia and
   Windows Mobile devices work fine.
   
   
    
   
   

LOCAL FIX
 *  NA
   
   
    
   
   

PROBLEM SUMMARY
 *  On some Nokia devices, generally E71, the device would hang
   during install of the Lotus Notes Traveler client.
   
   
    
   
   

PROBLEM CONCLUSION
 *  This was found to actually be a defect in the Nokia device
   itself.  We reported the problem to Nokia support, but were able
   to work around it by avoiding the call that would hang the
   device.
   
   
    
   
   

TEMPORARY FIX
 *  This fix will be included in 8.5.1.2 and any future releases.
   See this technote for latest maintenance info.
   
   http://www.ibm.com/support/docview.wss?uid=swg24019529 [http://www.ibm.com/support/docview.wss?uid=swg24019529]
