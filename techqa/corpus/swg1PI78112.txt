APAR STATUS
 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  n IBM Rational Quality Manager (RQM) 6.0.2, when you execute a
   Test Suite within a Test Plan and select the option "Modify
   execution variable values", the existing execute variables are
   not displayed in the next page.
   
   Steps to reproduce:
   1. Create a test plan.
   2. Create a test suite within the test plan.
   3. Open the test suite, add a test case to it so that it can be
   executed. Also add an execution variable.
   4. Try to execute the test suite and select the option "Modify
   execution variable values". Observe that the execution variable
   appears on the second page.
   5. Go back to the test plan, select the test suite, try to
   execute it and select the option "Modify execution variable
   values". Observe that the execution variable does not appear on
   the second page.
   
   Review the following work item on Jazz.net for details:
   https://jazz.net/jazz02/resource/itemName/com.ibm.team.workitem. [https://jazz.net/jazz02/resource/itemName/com.ibm.team.workitem.]
   WorkItem/156595
