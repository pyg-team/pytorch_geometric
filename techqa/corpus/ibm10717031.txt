Title: IBM Security Bulletin: IBM Sterling B2B Integrator Standard Edition is Vulnerable to Cross-site Scripting (CVE-2018-1563, CVE-2018-1513) - United States

Text:
 SECURITY BULLETIN

SUMMARY
IBM Sterling B2B Integrator Standard Edition is vulnerable to cross-site scripting.

VULNERABILITY DETAILS
CVEID:CVE-2018-1563 [http://cve.mitre.org/cgi-bin/cvename.cgi?name=CVE-2018-1563]
DESCRIPTION: IBM Sterling B2B Integrator Standard Edition is vulnerable to cross-site scripting. This vulnerability allows users to embed arbitrary JavaScript code in the Web UI thus altering the intended functionality potentially leading to credentials disclosure within a trusted session.
CVSS Base Score: 5.4
CVSS Temporal Score: Seehttps://exchange.xforce.ibmcloud.com/vulnerabilities/142967 [https://exchange.xforce.ibmcloud.com/vulnerabilities/142967]for the current score
CVSS Environmental Score*: Undefined
CVSS Vector: (CVSS:3.0/AV:N/AC:L/PR:L/UI:R/S:C/C:L/I:L/A:N)

CVEID:CVE-2018-1513 [http://cve.mitre.org/cgi-bin/cvename.cgi?name=CVE-2018-1513]
DESCRIPTION: IBM Sterling B2B Integrator Standard Edition is vulnerable to cross-site scripting. This vulnerability allows users to embed arbitrary JavaScript code in the Web UI thus altering the intended functionality potentially leading to credentials disclosure within a trusted session.
CVSS Base Score: 5.4
CVSS Temporal Score: Seehttps://exchange.xforce.ibmcloud.com/vulnerabilities/141551 [https://exchange.xforce.ibmcloud.com/vulnerabilities/141551]for the current score
CVSS Environmental Score*: Undefined
CVSS Vector: (CVSS:3.0/AV:N/AC:L/PR:L/UI:R/S:C/C:L/I:L/A:N)

AFFECTED PRODUCTS AND VERSIONS
IBM Sterling B2B Integrator 5.2.0.1 - 5.2.6.3
