Title: IBM Unknown status of Host server power supply - United States

Text:
 TECHNOTE (TROUBLESHOOTING)

THIS DOCUMENT APPLIES ONLY TO THE FOLLOWING LANGUAGE VERSION(S):
 English 

PROBLEM(ABSTRACT)
 NPS reports host server power supply as unknown 

CAUSE
I2C bus hung between IMM and power supply

DIAGNOSING THE PROBLEM
check the output of "/usr/bin/ipmitool sdr elist | grep -i power" , if both power supplies show no readying , then it is most likely caused by I2C bus hung 

RESOLVING THE PROBLEM
If IMM IP is accessible, access IMM and reset IMM
