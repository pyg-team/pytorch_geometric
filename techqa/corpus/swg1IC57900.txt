Title: IBM IC57900: Errors occur during pre-filtering if the group name contains an ampersand (&) character - United States

APAR STATUS
 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  OmniFind Enterprise Edition does not handle the ampersand (&)
   character correctly, which needs to be escaped .
   
   The following error is reported in the logs:
   com.ibm.es.api.srv.actions.GetUserInformationAction
   SEVERE: Exception org.xml.sax.SAXParseException: The entity name
   must immediately follow the '&' in the entity reference.
   
   
    
   
   

LOCAL FIX
 *  N/A
   
   
    
   
   

PROBLEM SUMMARY
 *  This problem was caused by calling the entityDecode method
   instead of the entityEncode method in the ESSearchServer?
   application. Code was changed to handle '&' characters
   correctly.
   
   
    
   
   

PROBLEM CONCLUSION
 *  This problem is resolved with OmniFind Enterprise Edition
   Version 8.5 Fix Pack 1.
