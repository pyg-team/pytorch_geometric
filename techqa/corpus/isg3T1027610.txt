Title: IBM Security Bulletin: A security vulnerability has been identified in paramiko shipped with IBM Cloud Manager with OpenStack (CVE-2018-7750) - United States

Text:
 SECURITY BULLETIN

SUMMARY
 Paramiko is shipped as a component of IBM Cloud Manager with Openstack. This vulnerability cannot be exploited as IBM Cloud Manager with OpenStack only uses Paramiko client. Information about a security vulnerability affecting Paramiko has been published in a security bulletin. 

VULNERABILITY DETAILS
Please consult the security bulletin RHSA-2018:0591 - Security Advisory [https://access.redhat.com/errata/RHSA-2018:0591] for vulnerability details and information about fixes.

AFFECTED PRODUCTS AND VERSIONS
 

Principal Product and Version(s) Affected Supporting Product and Version 
IBM Cloud Manager with OpenStack 4.3.0 python-paramiko 1.13.1-3.ibm.el7.noarch 
IBM Cloud Manager with OpenStack 4.2.0 python-paramiko-1.13.1-3.ibm.el6.noarch 
IBM Cloud Manager with OpenStack 4.1.0 python-paramiko-1.11.3-3.ibm.el6.noarch 
As this vulnerability cannot be exploited, you do not have to update the package. 

WORKAROUNDS AND MITIGATIONS
None
