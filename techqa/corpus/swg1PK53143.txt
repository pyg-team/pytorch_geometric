Title: IBM PK53143: UNABLE TO MOVE A DRAFT OF CONTENT AND MOVE OPTION IS NOT GREYED OUT FOR CONTENT WITH A DRAFT. - United States

Text:
 
APAR STATUS
 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  The following exception is thrown if a content item, that has a
   draft is moved:
   
   IWKMU1062X: Message: Move failed because of a repository
   exception.
   Cause: com.ibm.icm.jcr.access.AccessDeniedException: The
   requested
   operation violates one or more lock constraints.:
   [ErrorCode:7591]
   
   
   
   Moving content that has a draft should not be allowed and the
   option will be greyed out
   
   
   Moving drafts should be allowed.
   
   
    
   
   

LOCAL FIX

PROBLEM SUMMARY
 *  Detailed Problem Description:
   
   1) Moving drafts should be allowed.  The move option is not
   available for drafts.
   
   2) The following exception is thrown if a content item, that has
   a draft is moved:
   
   IWKMU1062X: Message: Move failed because of a repository
   exception.
   Cause: com.ibm.icm.jcr.access.AccessDeniedException: The
   requested operation violates one or more lock constraints.:
   [ErrorCode:7591]
   
   
    
   
   

PROBLEM CONCLUSION
 *  Problem Analysis:
   
   Moving of drafts should be allowed
   Moving content that has a draft should not be allowed and the
   option will be greyed out
   
   Failing Module: WCM - Web Content Management (General)
   
   Problem Solution:
   
   Enable moving of drafts
   Grey-out option to move when content is selected that has an
   associated draft (i.e. checked-out)
   
   Affected Users: All Users
