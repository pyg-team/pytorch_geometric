Title: IBM Using Query Tables in Business Process Choreographer Explorer - United States

Text:
 EDUCATION

ABSTRACT
 This document describes the usage of query tables in Business Process Choreographer Explorer to improve performance of predefined and custom queries in customer scenarios. 

CONTENT
This document covers:


 * Software Prerequisites 
 * Using query tables in Business Process Choreographer Explorer predefined views 
 * Using query tables in Business Process Choreographer Explorer custom views 
 * Best practices for defining query tables for Business Process Choreographer Explorer 
 * Samples for developing query tables

Note that the following pdf document refers to interim Fix IZ66120 which is necessary for BPC Explorer 6.2.0.2 to enable query table support. 

In all later releases 6.2.0.x the ServicePack readme file describes how to enable query table support for BPC Explorer.
In all releases 7.x query table support for BPC Explorer is enabled by default.
