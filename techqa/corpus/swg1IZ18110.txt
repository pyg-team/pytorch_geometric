APAR STATUS
 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  When invoking Micro Focus COBOL stored procedure that performs
   I/O operation on AIX, customer will get SQL1131N error.
   
   SQL1131N DARI (Stored Procedure) process has been terminated
   abnormally.
   SQLSTATE=38503
   
   In the db2diag.log file, there will be the following entry.
   .....
   2006-07-12-18.18.21.645088+540 I9383874C401       LEVEL: Warning
   PID     : 3915854              TID  : 1           PROC :
   db2agent (SDB00A) 0
   INSTANCE: db2inst1             NODE : 000         DB   : DDB00A
   APPHDL  : 0-938                APPID: GA21186C.EF07.018C82091904
   MESSAGE : Removing FMP from pool
   DATA #1 : Hexdump, 12 bytes
   0x2FF14EA8 : 3015 00F4 0008 7088 0000 0000
   0.....p.....
   .....
   
   It will be caused by getting the runtime system error 198 when
   dynamically loading the COBOL libraries for I/O operation.
   
   In order to resolve the whole issue,
   a) Apply IZ18110 (integrated into V9.5 FP1)
   b) Compile the stored procedure using cob -y instead of cob -x
   c) Apply the MF Cobol Hotfix
      (AIX 5.3: RPI 1059747. AIX 5.2: RPI 1060932, and AIX
   Concluding Service Pack 5200-09-CSP.)
