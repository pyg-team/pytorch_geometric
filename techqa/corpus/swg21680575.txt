Title: IBM Security Bulletin: Multiple vulnerabilities in IBM InfoSphere Optim Data Growth Solution for JD Edwards EnterpriseOne (CVE-2013-5433) - United States

Text:
 SECURITY BULLETIN

SUMMARY
 Vulnerabilities exist in the IBM® InfoSphere® Optim™ Data Growth Solution for JD Edwards EnterpriseOne product that could be exploited by an attacker because of hard-coded database credentials. 

VULNERABILITY DETAILS
CVE ID: CVE-2013-5433 [http://www.cve.mitre.org/cgi-bin/cvename.cgi?name=2013-5433]
DESCRIPTION: The Optim Data Growth Solution for JD Edwards EnterpriseOne product may contain hard-coded database credentials. To exploit this vulnerability, an attacker does not need any specialized access conditions but does need to authenticate.

CVSS:
CVSS Base Score: 3.5
CVSS Temporal Score: See https://exchange.xforce.ibmcloud.com/vulnerabilities/87639 [https://exchange.xforce.ibmcloud.com/vulnerabilities/87639] for the current score
CVSS Environmental Score*: Undefined
CVSS Vector: (AV:N/AC:M/Au:S/C:P/I:N/A:N)


AFFECTED PRODUCTS:
Versions 3.0 through 9.1 of IBM InfoSphere Optim Data Growth Solution for JD Edwards EnterpriseOne are affected. 


REMEDIATION: : The recommendation is to leave the password value blank in the XML file, and the Optim Data Growth Solution for JD Edwards EnterpriseOne product will prompt the user to enter the password during runtime. 

Please refer to updated installation guides corresponding to your version. 


Fix:
For all versions:
- Apply the iFix (link to iFix [http://www-933.ibm.com/support/fixcentral/swg/downloadFixes?product=ibm/Information+Management/Optim&release=OracleApps.9.1.0&platform=All&function=fixId&fixids=JDE-09.01.00-004&includeRequisites=1&includeSupersedes=0&downloadMethod=http])

Workaround(s) & MITIGATION(s): 
None known, apply fixes
