Title: IBM Windows 10 で Cognos Administration 画面上のリンクテキストに取り消し線が表示されてしまう - United States

Text:
 TECHNOTE(トラブルシューティング)

この文書は以下の言語にのみ有効な内容です
 日本語 

問題(概要)
 Windows 10 Pro を使用して、Internet Explorerで Cognos Administration 画面へアクセスすると、クリック可能なリンクテキストに下線ではなく取り消し線が表示されているように見えることがあります。 

原因
Cognos のデフォルトフォントの設定が、日本語環境に最適化されたフォントファミリーでないことが原因です。

環境
Internet Explorer、Windows 10

解決方法
デフォルトフォントの設定を日本語環境に最適化されたフォントファミリーを使用するよう変更してください。




 1. <インストール先>\webcontent\skins\<style>\fragments\fonts.css をバックアップします。
    <styles>は、Cognos Connection でご使用中のスタイル名が該当します。 
 2. fonts.cssの以下の定義を、下記のように変更します。
    [変更前]
    BODY {
    font-family: tahoma, arial, geneva, helvetica, sans-serif;
    }
    TEXTAREA {
    font-family: tahoma, arial, geneva, helvetica, sans-serif;
    }
    INPUT {
    font-family: tahoma, arial, geneva, helvetica, sans-serif;
    }
    SELECT {
    font-family: tahoma, arial, geneva, helvetica, sans-serif;
    } 
    
    [変更後]
    BODY {
    font-family: 'MS UI Gothic', tahoma, arial, geneva, helvetica, sans-serif;
    }
    TEXTAREA {
    font-family: 'MS UI Gothic', tahoma, arial, geneva, helvetica, sans-serif;
    }
    INPUT {
    font-family: 'MS UI Gothic', tahoma, arial, geneva, helvetica, sans-serif;
    }
    SELECT {
    font-family: 'MS UI Gothic', tahoma, arial, geneva, helvetica, sans-serif;
    } 
    
 3. 設定変更後、Cognos サービスを再起動し、Windows 10 のブラウザ上でキャッシュを削除します。
