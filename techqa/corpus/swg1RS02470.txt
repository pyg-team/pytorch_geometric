LOCAL FIX
 *  move the OUT parameters to IN/OUT parameter in the signature of
   
   the Decision Operation
   
   
    
   
   

PROBLEM SUMMARY
 *  This issue comes from design/architecture choices and also
   limitations with Excel.
   When the feature was implemented with Excel 2003, the list of
   output parameters was stored in a custom property and the size
   of the value of a custom property was limited to 256 characters
   and it has not changed.
   With Excel 2007, Excel 2010, the size of the value of a custom
   property is still limited to 256 characters. So we need to
   handle the case when the OUT parameter lists is larger than 256
   characters
   
   
    
   
   

PROBLEM CONCLUSION
 *  The code is fixed
