ABSTRACT
 This document describes how to download IBM Security Directory Server, Version 6.4 using the Passport Advantage Online website for each supported operating system. 

DOWNLOAD DESCRIPTION


IBM Security Directory Server is the IBM implementation of Lightweight Directory Access Protocol (LDAP) for the following supported operating systems. To obtain download information about specific operating system, click the operating system tab that you want.

You can download IBM Security Directory Server, Version 6.4 from the IBM Passport Advantage Online [http://www-01.ibm.com/software/howtobuy/passportadvantage/pao_customers.htm] website.

 1. Download the parts that you need for your operating system, which are described in the tabbed sections of this document.
 2. Do one of the following:  * If the .iso part is available for your platform, download the .iso part and use the file to create an installation DVD.
     * Download the .zip or .tar part for your platform and unpack the part into a temporary directory on your system.
    
    
 3. Install the product on your system using the information in the links under Installation Instructions of this document.
 4. Install the premium feature activation [http://www.ibm.com/support/knowledgecenter/SSVJJU_6.4.0/com.ibm.IBMDS.doc_6.4/ds_ig_featureentitlement.html] package, which enables the use of the premium features in IBM Security Directory Server, Version 6.4

Note:The Web Administration Tool of IBM Security Directory Server needs to be deployed in WebSphere Application Server. However, WebSphere Application Server is not provided with the IBM Security Directory Server installation package. You must download WebSphere Application Server, Version 8.5.5 from Passport Advantage and install it separately. For more information, see theIBM Knowledge Center for WebSphere Application Server [http://www.ibm.com/support/knowledgecenter/SSEQTP/mapfiles/product_welcome_was.html]. 

To access a wide variety of technical resources, see the IBM Knowledge Center for IBM Security Directory Server, Version 6.4 [http://www.ibm.com/support/knowledgecenter/SSVJJU_6.4.0/com.ibm.IBMDS.doc_6.4/welcome.html].
