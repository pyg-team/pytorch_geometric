APAR STATUS
 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  The document cache might enter a state where it does not cache
   entries, causing all requests to go to the backend.
   
   
    
   
   

LOCAL FIX

PROBLEM SUMMARY
 *  Affected are Datapower configurations using document cache.
   
   In response to some traffic patterns, the document cache may
   start behaving as if it were disabled, failing to return any
   cached document and forwarding all requests to the backend.
   
   
    
   
   

PROBLEM CONCLUSION
 *  Fix is available in 6.0.0.17, 6.0.1.13, 7.0.0.10, 7.1.0.7
   and 7.2.0.1
   
   For a list of the latest fix packs available, please see:
   http://www-01.ibm.com/support/docview.wss?uid=swg21237631 [http://www-01.ibm.com/support/docview.wss?uid=swg21237631]
   
   
    
   
   

TEMPORARY FIX
 *  Restart the application domain affected.
