APAR STATUS
 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  IBM WebSphere DataPower Appliance
   
   Firmware: 3.8.0.x
   
   Description: The MQ Front Side Handler will go to pending state,
   if sum of all MQ FSH concurrent connection is equal to MQ QM
   total connection limit
   
   
    
   
   

LOCAL FIX

PROBLEM SUMMARY
 *  When the connection reach the connection limit boundary, MQ FSH
   might show pending when MQ FSH down and up.
   
   
    
   
   

PROBLEM CONCLUSION
 *  Add a new function to handle connection counting issue when
   connection reach connection limit.
   This fix will be availabe in 3.7.2.13, 3.7.3.12, 3.8.0.6,
   3.8.1.4
