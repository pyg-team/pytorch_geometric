ABSTRACT
 MPLC contains charts showing Performance measurements to present the performance characteristics of IBM MQ V8.0 for Linux on System z to assist in planning for performance. 

DOWNLOAD DESCRIPTION
This report shows the performance highlights of IBM MQ V8.0 for Linux on System z, through a set of performance and scaling test runs on V8.0 (including comparisons with V7.5), and provides advice on planning for performance and tuning MQ. 

Possible Uses
Anyone designing, implementing or sizing solutions using IBM MQ V8.0 who needs to understand the performance characteristics on Linux on System z.
