Title: IBM DB2 Express V10.1 limitation on Ubuntu Server 14.04 Linux on x86 platform in IBM Business Process Manager (BPM) and IBM Business Monitor - United States

Text:
 TECHNOTE (TROUBLESHOOTING)

PROBLEM(ABSTRACT)
 When installing IBM Business Process Manager (BPM) V8.5.6.0 or IBM Business Monitor V8.5.6.0 on Ubuntu 14.04 Linux on x86 platform, if DB2 Express 10.1.0.1 is selected to be installed, the Installation Manager (IM) panel will show warning indicating that it is not a supported OS platform. 

CAUSE
DB2 Express 10.1.0.1 does not yet support Ubuntu Linux.

RESOLVING THE PROBLEM
DB2 Express 10.1.0.1 should not be installed on Ubuntu Linux 14.04. 

For a typical install:
1. Install a database server first, such as a remote database server or local database server that supports Ubuntu 14.04. 
2. Select the 'existing database' option in typical install panel.

For a custom install:
1. Deselect 'DB2 Express 10.1.0.1' in the Installation Manager panel. 
2. Configure IBM Business Process Manager (BPM) or IBM Business Monitor with an existing database server.
