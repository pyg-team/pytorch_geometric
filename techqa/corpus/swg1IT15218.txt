APAR STATUS
 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  When the appliance gets to the state where there are no file
   descriptors available, the appliance could start experiencing
   stability issues.
   
   
    
   
   

LOCAL FIX

PROBLEM SUMMARY
 *  Affected are all customers who manage to drive the
   appliance - most likely with a very high load or by enabling
   certain debugging features - out of available file descriptors
   while connected to the command line interface.
   
   In such cases the appliance can experience various stability and
   connection issues.
   
   
    
   
   

PROBLEM CONCLUSION
 *  Fix is available in 7.0.0.14, 7.1.0.11, 7.2.0.8, 7.5.0.2 and
   7.5.1.0.
   
   For a list of the latest fix packs available, please see:
   http://www-01.ibm.com/support/docview.wss?uid=swg21237631 [http://www-01.ibm.com/support/docview.wss?uid=swg21237631]
