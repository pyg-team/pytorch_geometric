Title: IBM AAA post processing WS-Security Username Token password digest algorithm - United States

Text:
AAA; Post Processing; Username Token; SHA-1. TECHNOTE (FAQ)

QUESTION
 What is the algorithm used by the 'digest' option of the password generated in AAA post processing for insertion of a WS-Security Username Token? 

CAUSE
Algorithm used for UNT in Post Processing of AAA Policy.

ANSWER
The generation of that token follows the WS-Security Username Token Profile: <http://docs.oasis-open.org/wss/v1.1/wss-v1.1-spec-os-UsernameTokenProfile.pdf [http://docs.oasis-open.org/wss/v1.1/wss-v1.1-spec-os-UsernameTokenProfile.pdf]>. This standard states that there is no provision for the use of algorithms other than SHA-1. So, DataPower is using SHA-1.
