Title: IBM Security Bulletin: Multiple Security Vulnerabilities have been identified in WebSphere Application Server shipped with WebSphere Remote Server. - United States

Text:
 SECURITY BULLETIN

SUMMARY
 WebSphere Application Server is shipped as a component of WebSphere Remote Server. Information about security vulnerabilities affecting WebSphere Application Server has been published in a security bulletin. 

VULNERABILITY DETAILS
For vulnerability details, see the security bulletin Multiple Security Vulnerabilities fixed in IBM WebSphere Application Server 8.0.0.11 [http://www.ibm.com/support/docview.wss?uid=swg21963275].
