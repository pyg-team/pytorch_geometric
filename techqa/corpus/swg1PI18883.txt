ERROR DESCRIPTION
 *  Regardless of the setting for ACTLGUSR, the unload fails with
   completion code 8 after having issued messages as follows:
   
   DSNT408I SQLCODE = -551, ERROR:  userid DOES NOT HAVE THE
   PRIVILEGE TO PERFORM
            OPERATION SELECT ON OBJECT SYSIBM.SYSKEYS
   DSNT418I SQLSTATE   = 42501 SQLSTATE RETURN CODE
   DSNT415I SQLERRP    = DSNXOSC SQL PROCEDURE DETECTING ERROR
   DSNT416I SQLERRD    = -100  0  0  -1  0  0 SQL DIAGNOSTIC
   INFORMATION
   DSNT416I SQLERRD    = X'FFFFFF9C'  X'00000000'  X'00000000'
   X'FFFFFFFF'
            X'00000000'  X'00000000' SQL DIAGNOSTIC INFORMATION
   INZU020E ERROR WHILE RETRIEVING TABLE SPACE INFO FROM THE
   CATALOG
   
   where userid is the name of the user that submitted the unload
   job.
   
   This occurs when:
   - The user that submitted the unload job does not have
     the SELECT privilege on the DB2 catalog tables
   - SQLACCES is set to MINIMAL
   - The direct access to the catalog table spaces failed
   
   PB12354 MSGINZU020E MSGINZU020 INZU020E INZU020 SQLCODE551
   DSNT408
   
   
    
   
   

LOCAL FIX
 *  set SQLACCES to YES
