ERROR DESCRIPTION
 *  If you're migrating to BPM 8.5, and the WAS base of BPM 8.5 is
   8.5.5, then the migration will fail at DBUpgrade with
   ClassNotFoundException exception below:
       Caused by: java.lang.ClassNotFoundException:
   com.ibm.ws.javaee.dd.ejb.MethodPermission
   
   It's caused by some changes in WAS 8.5.5, and the ifix will fix
   this issue.

PROBLEM SUMMARY
 *  ****************************************************************
   * USERS AFFECTED:  Business Process Manager Version 8.5        *
   ****************************************************************
   * PROBLEM DESCRIPTION: DBUpgrade fails with                    *
   *                      java.lang.ClassNotFoundException        *
   ****************************************************************
   * RECOMMENDATION:                                              *
   ****************************************************************
   Due to changes in WebSphere Application Server (WAS) 8.5.5,
   the DBUpgrade will fail during migration to Business Process
   Manager Version 8.5 if the base WAS version is 8.5.5
   The DBUpgrade will fail with ClassNotFoundException exception
   below:
   Caused by: java.lang.ClassNotFoundException:
   com.ibm.ws.javaee.dd.ejb.
   
   
    
   
   

PROBLEM CONCLUSION
 *  The iFix corrects the DBUpgrade issue.
   
   Install/Dependency information is available in the readme.txt
   on FixCentral. When the user goes to the download files page
   in FixCentral, the readme will be available with the fix
   package as a separate download.
