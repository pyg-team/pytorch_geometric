Title: IBM JR50983: BPMCONFIG COMMAND FAILS TO CREATE DEPLOYMENT ENVIRONMENT AND RETURNS AN ERROR WHEN INSTALLING BPC CONTAINER - United States

Text:
  A FIX IS AVAILABLE
Version 8.5 Refresh Pack 6 for the IBM Business Process Manager products [http://www-01.ibm.com/support/docview.wss?uid=swg24039185]


SUBSCRIBE

APAR STATUS
 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  If you have the following properties in your BPMConfig input
   properties file, the deployment environment cannot be created:
   
   bpm.de.cluster.#.capability.#.component.#.name=BPC
   
   You can see the following error in the log files:
   
   com.ibm.websphere.management.exception.AdminException: Could
   not find a messaging engine for the destination at the target
   AppCLUSTER
   
   
    
   
   

LOCAL FIX
 *  Remove the properties mentioned in the APAR
