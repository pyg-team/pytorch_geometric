Title: IBM Known limitation: You receive an error when filtering and running the Cognos Custodian Hold Non-compliance report or the Notice Recipient List report for more than 1000 parameters - United States

Text:
 RELEASE NOTES

ABSTRACT
 If you filter and run the reports for more than 1000 parameters, you will receive the following error message: Data source adapter error: java.sql.SQLException: ORA-01795: maximum number of expressions in a list is 1000 - when processing query 

CONTENT
To work around the limitation, filter and run the reports for 1000 or fewer parameters.
