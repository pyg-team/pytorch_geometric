APAR STATUS
 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  When integrating with a product such as IBM Pure Application
   Server or IBM Cloud Orchestrator, if a pattern has a secure
   property with a default value, that default value will not be
   used.
   
   Reproduction steps:
   1. Create a new template, configure the OS root and virtuser
   default password
   2. Import as resource template, create an application, create
   blueprint from it
   3. Create a new environment and do not specify a value for the
   password
   
   
    
   
   

LOCAL FIX
 *  Manually override the password at provisioning time.
