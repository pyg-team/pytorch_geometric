Title: IBM Error 2070 Symptom 5 - SPSS Statistics license issue - United States

Text:
2070 lswhere.exe TECHNOTE (TROUBLESHOOTING)

PROBLEM(ABSTRACT)
 Concurrent user client tries to retrieve a license from the license server. The server's IP has been changed to another subnet. The given server also includes 2 network cards 

SYMPTOM
Error # 2070
The license server is down. Please check to see that the license server is running or ask your local SPSS coordinator to check the license server.
This command is not executed.

Specific symptom number: 5.


CAUSE
The license server IP address was changed to another subnet

ENVIRONMENT
Windows 2012 R2 Server // SPSS 23

DIAGNOSING THE PROBLEM
Modifying the DaemonHost line of the spssprod.inf file on the client to match the server. Created LSHOST and LSFORCEHOST environment variables that point to the IP adress.

RESOLVING THE PROBLEM
Running lswhere.exe on the server shows the actual IP adress of the server. This adress must be added to the Defined Server List on the WLM Admin application on the server
