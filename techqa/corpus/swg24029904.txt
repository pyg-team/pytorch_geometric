INSTALLATION INSTRUCTIONS

1. Confirm to install ITM and Prerequisites.
2. Create a temporary directory <DIR_NAME>.
3. Extract archive file to the directory.
4. Install the language pack.
a) Windows platform
Double-click lpinstaller.bat in the directory to launch the installation program.
b) Unix/Linux platform
- Run the following command to launch the installation program in the temporary directory:
- lpinstaller.sh -c <ITM_Home_Directory> [-i <install_mode>]
Note: <ITM_Home_Directory> is where you installed IBM Tivoli Monitoring; usually it is /opt/IBM/ITM for AIX and Linux. <install_mode> is one of the following: gui, console, silent. On Turkish locale, use GUI instead of gui.
5. Select the language of the installer and click OK.
6. Click Next on the Introduction panel.
7. Click Add/Update and click Next.
8. Select the folder in which the National Language Support package (NLSPackage) files are located.
Note: Usually the NLSPackage files are located in the nlspackage folder where the installer executable is located.
9. Select the language support for the agent of your choice and click Next.
Note: You can hold down the Ctrl key for multiple selections.
10. Examine the installation summary page and click Next to begin installation.
11. Click Finish after installation completes to exit the installer.
12. Restart Tivoli Enterprise Portal, Tivoli Enterprise Portal Server, and/or Eclipse Help Server if any of these components are installed.
