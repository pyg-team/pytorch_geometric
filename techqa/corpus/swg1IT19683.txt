ERROR DESCRIPTION
 *  Several changes and fixes for event and transactional logging
   have been added:
    (1) Added log event to indicate the beginning of new
   transaction policy processing
    (2) Added optional time format for syslog-tcp that includes
   both microsecond precision and time zone
    (3) Added ability to filter WSM transactional data based on
   payload size
    (4) Increased the limit for maximum number of system log
   targets from 500 to 1004
