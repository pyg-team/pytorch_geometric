Title: IBM Setting DASH session timeout at the User or Group level. - United States

Text:
DASH; DASHL2; DASHL2CONF; Session Timeout; Configuration; Controls; User; Group ; level TECHNOTE (FAQ)

QUESTION
 Is it possible to set DASH session timeout parameters on a group and user level? 

CAUSE
For security reasons, customer wants to set DASH session timeout at the group and user level.

ANSWER
Setting DASH session Time-out controls is not supported at the Group and User level.
