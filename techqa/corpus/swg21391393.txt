Title: IBM Cannot install FileNet P8 eForms (eF) 4.0.0 on FileNet Application Engine (AE) 4.0.1 or FileNet Workplace XT (XT)  1.1.x or 1.1.1.x - United States

Text:
eForms; XT Install TECHNOTE (TROUBLESHOOTING)

PROBLEM(ABSTRACT)
 Cannot install FileNet P8 eForms (eF) 4.0.0 on FileNet Application Engine (AE) 4.0.1 or FileNet Workplace XT (XT) 1.1.x or 1.1.1.x 

RESOLVING THE PROBLEM

Software and fix packs must be installed in a very specific order.

On AE 4.0.1: 

 1. Install eF4.0.0-001 
 2. Install eF 4.0.1 
 3. On Workplace XT 1.1.0.0 
 4. Install eF 4.0.0-002 


On Workplace XT 1.1.0 or 1.1.1.0  1. Install eF 4.0.0-002 
 2. Install eF 4.0.1-001 or later
