Title: IBM Domino is not affected by GNU C library (glibc) vulnerability (CVE-2015-0235) - United States

Text:
 TECHNOTE (FAQ)

QUESTION
Is IBM Domino affected by the GNU C library (glibc) vulnerability (CVE-2015-0235) disclosed January 27, 2015? This vulnerability has been referred to as “GHOST”.

ANSWER
No, Domino is not affected by this heap-based overflow vulnerability. For more information on this vulnerability, refer to the following links: 

 * CVE-2015-0235 [https://cve.mitre.org/cgi-bin/cvename.cgi?name=CVE-2015-0235]
