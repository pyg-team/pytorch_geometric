ERROR DESCRIPTION
 *  Problem Description :
   Even though WebSphere 7.0 and Websphere Portal 6.1 are
    supported by ITCAM 6.1, the combination of Portal 6.1 running
    on WebSphere 7.0 is not recognized by TEMA.  It throws an
    exception when DC connects to TEMA similar to the following in
    kyn-tema-trace.log:
   
    com.ibm.tivoli.monitoring.toolkit.workbench.WbException: [(no
    bundle) KWJ_JMXCOLLECTOR_NO_APP_SERVER_REP_VER]('ws.wps
    7.0.0.11')
   
   Users Affected :
     Any version of WebSphere portal running on WebSphere 7.0
   
   Approver : SV
