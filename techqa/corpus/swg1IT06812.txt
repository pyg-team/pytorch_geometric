 DIRECT LINK TO FIX

APAR STATUS
 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  Some panels in the Process Admin Console are vulnerable against
   cross-site scripting attacks.
   
   
    
   
   

LOCAL FIX

PROBLEM SUMMARY
 *  Panels in the Process Admin Console are not protected from
   cross-site scripting attacks.
   
   
    
   
   

PROBLEM CONCLUSION
 *  A fix is available for all supported releases of IBM Business
   Process Manager (V7.5 to and including V8.5.6) and WebSphere
   Lombardi Edition (V7.2) that secures the Process Admin Console
   against cross-site scripting attacks.
   
   On Fix Central (http://www.ibm.com/support/fixcentral), search
   for JR52420 [http://www-01.ibm.com/support/docview.wss?uid=swg1JR52420] (IBM BPM) or IT06812 (WLE):
   
   1. Select IBM Business Process Manager with your edition from
     the product selector, the installed version to the fix pack
     level, and your platform, and then click Continue.
   
   2. Select APAR or SPR, enter JR52420 [http://www-01.ibm.com/support/docview.wss?uid=swg1JR52420] (IBM BPM) or IT06812 (WLE),
     and click Continue.
   
   When you download fix packages, ensure that you also download
   the readme file for each fix. Review each readme file for
   additional installation instructions and information about the
   fix.
   
   
    
   
   

TEMPORARY FIX
 *  Not applicable
