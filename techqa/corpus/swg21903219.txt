Title: IBM Does WebSphere Partner Gateway support TLS 1.2 - United States

Text:
WPG TLS Support TECHNOTE (FAQ)

QUESTION
 Does WebSphere Partner Gateway support TLS 1.2? 

ANSWER
TLS Support in WPG
IBM supportsTLS 1.2 in WPG 6.2.1.3 release. TLS 1.2 is supported from Java 6 SR 10 onwards which is available only in WAS 7. 

WPG support for TLS 1.2
http://www-01.ibm.com/support/docview.wss?uid=swg27039556&aid=1 [http://www-01.ibm.com/support/docview.wss?uid=swg27039556&aid=1]

TLS 1.2 supported from Java 6 SR 10
http://www-01.ibm.com/support/knowledgecenter/SSYKE2_6.0.0/com.ibm.java.security.component.60.doc/security-component/whats_new/security_changes_60/security_whatsnew_sr10.html?lang=en [http://www-01.ibm.com/support/knowledgecenter/SSYKE2_6.0.0/com.ibm.java.security.component.60.doc/security-component/whats_new/security_changes_60/security_whatsnew_sr10.html?lang=en]

Java 7 support
IBM does not support Java 7

WAS 8.5.5 support
IBM only supports JDK 6 in WAS 8.5.5
http://www-01.ibm.com/support/docview.wss?uid=swg24039272 [http://www-01.ibm.com/support/docview.wss?uid=swg24039272]

Java 6 SR 10 is not available in WAS 8.5.5. Therefore TLS 1.2 is not supported
