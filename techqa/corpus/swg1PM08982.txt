 *  Site framework:
   
     SITE (Site access user: anonymous)
       ANONYMOUS      (Site Area access user: anonymous)
       AUTHENTICATED  (Site Area access user: all autenthicated)
   Authoring templates with 2 elements :
     Rich Text - rt.body
     HTML  - ht.bottom
   Presentation template :
     <html>
     <h1><IDCmpnt context="current" type="content"
   field="title"/></h1>
     <Element context="current" type="content" key="rt.body"/>
     </br></br>
     <Element context="current" type="content" key="ht.bottom"/>
     </html>
   
   create 3 content items :
   1. Content Anonymous One (Acces user: anonymous, all
   authenticated)
     value rt.body
       Body Content Anonymous Two
     value ht.bottom
       <empty>
   save under SITE/ANONYMOUS
   2. Content Authenticated One (Acces user: all authenticated)
     value rt.body
       Body Content Authenticated One
     value ht.bottom
       <empty>
   save under SITE/AUTHENTICATED
   3. Content Anonymous Two (Acces user: user anonymous, all
   authenticated)

       Body Content Anonymous Two

       <Element context="selected"
   name="library/site/anonymous/content
   anonymous one" type="content" key="rt.body"/>
       <Element context="selected"
   name="library/site/authenticated/content
   authenticated one" type="content" key="rt.body"/>
   save under SITE/ANONYMOUS
   
   Now open
   http://<server>/wps/wcm/connect/<library>/SITE/ANONYMOUS/Content [http://<server>/wps/wcm/connect/<library>/SITE/ANONYMOUS/Content]
   Anonymous Two

   ANONYMOUS

    Body Content Anonymous Two
