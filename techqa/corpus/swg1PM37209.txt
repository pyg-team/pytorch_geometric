APAR STATUS
 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  Steps to recreate:
   1. Create a new library on the syndicator (also create publish
   workflow and a site & site area).
   2. Syndicate the library to the subscriber using All Items
   syndication.
   3. Create an authoring template and enable the option "Create
   content under new site area".
   4. Create a new content using the above template.
   Note: Content is created under a new site area with the same
   same as the content at the specified location.
   5. Wait until the updates are syndicated.
   
   Result:
   Update fails with NullPointerException on subscriber.
   New site area and content items are never syndicated across.
   
   
    
   
   

LOCAL FIX

PROBLEM SUMMARY
 *  Description:
   Steps to recreate:

PROBLEM CONCLUSION
 *  Solution:
   A check is made to see that the default content id exists before
   a reference to it is created.
