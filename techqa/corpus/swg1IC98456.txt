APAR STATUS
 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  When a log target is configured to upload logs to a remote SFTP
   server, the log upload might fail and error messages similar to
   the following are logged:
    [0x80e00085][network][error] logging target(sftp_log_tar):
   tid(19775): Invalid URL
   'sftp://sftp-hostname:22//datapower/syslog/filename'
    [0x8100043d][system][error] logging target(sftp_log_tar):
   tid(19775): Log upload of target sftp_log_tar failed.
   
   
    
   
   

LOCAL FIX

PROBLEM SUMMARY
 *  Affected is use of log targets that upload logs to a remote SFTP
    server.
   
   The log upload might fail.
   
   
    
   
   

PROBLEM CONCLUSION
 *  Fix is available in 6.0.0.5 and 6.0.1.1.
   
   For a list of the latest fix packs available, please see:
   http://www-01.ibm.com/support/docview.wss?uid=swg21237631 [http://www-01.ibm.com/support/docview.wss?uid=swg21237631]
