APAR STATUS
 * CLOSED AS DOCUMENTATION ERROR.
    
   
   

ERROR DESCRIPTION
 *  The WBIFN PIC Software requirements page describes that
   WebSphere Message Broker 8.0.0.4 is required, when MQ 8 is to be
   used.
   This indicates, that it needs to be exactly this fixpack level.
   The limitation should read:
   8.0.0.4 or later to make clear, that any FP level at or above FP
   4 is allowed for Message Broker 8.

PROBLEM CONCLUSION
 *  In Planning, Installation, and Customization,
   Chapter 2. Hardware and software requirements,
   Table 1. Runtime system requirements, notes text for
   IBM MQ for z/OS V8.0 changed to "Requires WebSphere MB V8
   Fixpack 8.0.0.4 or later and PTF for WebSphere MB V8
   APAR IT03599.
   The changes will be reflected in the next edition of the PIC.
