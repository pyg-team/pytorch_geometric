LOCAL FIX

PROBLEM SUMMARY
 *  The problem is caused by the wrong objectstreamclass being used
   for umarshalling of enum types.
   
   
    
   
   

PROBLEM CONCLUSION
 *  This defect will be fixed in:
   5.0.0 SR12
   6.0.0 SR9
   .
   The enum unmarshalling code has been fixed to use the correct
   objectstreamclass.
   .
   To obtain the fix:
   Install build 20100731 or later
