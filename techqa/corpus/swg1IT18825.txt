 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  DataPower might cause MQ error code 2009 when connections are
   depleted from the pool and mq-qm object fails to create new
   connection.
   
   
    
   
   

LOCAL FIX

PROBLEM SUMMARY
 *  When the appliance creates a new MQ connection, the connection
   attempt might fail intermittently.
   
   
    
   
   

PROBLEM CONCLUSION
 *  Fix is available in 7.2.0.12, 7.5.0.6, 7.5.1.5 and 7.5.2.3
