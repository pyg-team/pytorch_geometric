Title: IBM Security Bulletin: A security vulnerability has been identified in IBM WebSphere Application Server, IBM HTTP Server and IBM DB2 LUW shipped with WebSphere Commerce (CVE-2015-2808) - United States

Text:
 SECURITY BULLETIN

SUMMARY
 IBM WebSphere Application Server, IBM HTTP Server and IBM DB2 LUW are shipped as components of WebSphere Commerce. Information about a security vulnerability affecting IBM WebSphere Application Server, IBM HTTP Server and IBM DB2 LUW have been published in security bulletins. 

VULNERABILITY DETAILS
For vulnerability details and information about fixes, please consult the following security bulletins: 

 * Vulnerability in RC4 stream cipher affects WebSphere Application Server (CVE-2015-2808) [http://www-01.ibm.com/support/docview.wss?uid=swg21701503] 
 *  Vulnerability in RC4 stream cipher affects IBM HTTP Server (CVE-2015-2808) [http://www-01.ibm.com/support/docview.wss?uid=swg21701072] 
 * Vulnerability in RC4 stream cipher affects IBM® DB2® LUW (CVE-2015-2808) [http://www-01.ibm.com/support/docview.wss?uid=swg21717865]
