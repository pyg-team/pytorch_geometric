APAR STATUS
 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  The WebSphere MQ V7.0 installation contains a 1.5 Java Runtime
   Environments (JREs), supplied within the JRE install component.
   New revisions of this JRE are released periodically to address
   program defects, and products which embed the JRE should
   be updated accordingly.
   
   
    
   
   

LOCAL FIX

PROBLEM SUMMARY
 *  ****************************************************************
   USERS AFFECTED:
   The JRE which is embedded into WebSphere MQ is used
   to run the following Java components of the product:
   
     GSKit graphical interface
     WebSphere MQ Explorer
   
   
   Platforms affected:
   AIX, HP-UX Itanium, HP-UX PA-RISC, Linux on Power, Linux on
   S390, Linux on x86, Linux on zSeries
   
   ****************************************************************
   PROBLEM DESCRIPTION:
   This APAR updates the Java Runtime Environment (JRE) supplied
   with WebSphere MQ 7.0 to the level:
   
       Java 5.0 Service Refresh 16 Fix Pack 9 + interim fix
   (SR16-FP9)
   
   
    
   
   

PROBLEM CONCLUSION
 *  This APAR updates the JRE updates for the following platforms:
   
     AIX
     HP-UX Itanium
     HP-UX PA-RISC
     Linux (x86-32)
     Linux (x86-64)
     Linux (PPC)
     zLinux
     Solaris (x86-64)
     Solaris (SPARC)
     Windows
