 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  The Lotus Traveler Nokia client is not showing the middle
   initial on corporate lookup even though it is sent by the
   server.
   
   
    
   
   

LOCAL FIX
 *  NA
   
   
    
   
   

PROBLEM SUMMARY
 *  Middle initial was not displaying on Nokia devices for Traveler
   Corporate Look Up.
   
   
    
   
   

PROBLEM CONCLUSION
 *  We had to change the usage of the Nokia API calls and upgrade
   the toolkit to get functionality that would display the middle
   initial.
   
   
    
   
   

TEMPORARY FIX
 *  This fix will be included in the next fixpack 8522 and any
   future releases.
