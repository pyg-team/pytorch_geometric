APAR STATUS
 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  IBM MQ ships a Java Runtime Environment (JRE) embedded into its
   installation, the level of JRE shipped is backlevel.
   
   
    
   
   

LOCAL FIX

PROBLEM SUMMARY
 *  ****************************************************************
   USERS AFFECTED:
   The components of MQ which utilise the embedded JRE include:
   
   - the IBM MQ classes for Java and the IBM MQ classes for JMS who
   use the JRE shipped by IBM MQ to run their applications
   
   - IBM MQ Managed File Transfer Edition
   
   - IBM MQ Advanced Message Protocol (AMQP)
   
   - IBM MQ Telemetry Transport (MQTT)
   
   - IBM MQ Explorer
   
   
   Platforms affected:
   AIX, HP-UX Itanium, Linux on Power, Linux on S390, Linux on
   x86-64, Linux on zSeries, Windows
   
   ****************************************************************
   PROBLEM DESCRIPTION:
   The IBM MQ installation includes an embedded Java Runtime
   Environment (JRE).  The level of JRE shipped by IBM MQ was
   backlevel.
   
   
    
   
   

PROBLEM CONCLUSION
 *  The level of JRE shipped with MQ has been updated as follows:
   
     8.0.5.11 -  Linux (x86-64, ppcLE, zLinux), Windows, AIX
   
     8.0.5.10 - HP-IX (Itanium)
   
   Solaris (SPARC, x86-64) has not been updated under this APAR,
   and remains at the Java version 8.0.5.10.
