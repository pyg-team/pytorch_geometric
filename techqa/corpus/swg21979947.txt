Title: IBM Can Installation Manager 32bits version use for OMNIbus installation ? - United States

Text:
OMNIbus; Windows; 32; 64; Installation Manager TECHNOTE (FAQ)

QUESTION
 The version of Installation manager packaged is 64bits version. 
Can we use 32bits version of Installation Manager which already exist in the Windows OS where we would install OMNIbus 8.1 ? 


ANSWER
OMNIbus for Windows is 32-bit application. 

The 32 bits version of Installation Manager can be used for the OMNIbus installation and it is supported..
