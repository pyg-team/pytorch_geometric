STERLINGPRI TECHNOTE (FAQ)

QUESTION
 Is there a way to limit the number of documents per outbound interchange? (SCI64771) 

ANSWER


 1.  
 2.  
 3.  Open the Partner Editor 
 4.  
 5.  
 6.  Highlight the partner
 7.  
 8.  
 9.  Click Outbound 
 10. 
 11. 
 12. Select the relationship from the Relationship Description field's list box
 13. 
 14. 
 15. Click Interchanges > Edit > Advanced
 16. 
 17. 
 18. Place a check mark in the box next to Set Max Number of Documents per Interchange
 19. 
 20. 
 21. Fill in the Max Number of Documents per Interchange field
 22. 
 23. 
 24. Click Save > Select > Save > OK > Exit
