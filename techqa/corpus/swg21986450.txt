Title: IBM Security Network Protection firmware update 5.3.3 release notes - United States

Text:
 TECHNOTE (FAQ)

QUESTION
 What are the new features and known issues included in IBM Security Network Protection firmware update 5.3.3? 

ANSWER
IBM Security Network Protection firmware version 5.3.3 is a firmware update for the XGS NGIPS network protection platform. This release provides the following updates to IBM Security Network Protection Firmware Version 5.3.2: 

 * Inbound SSL inspection enhancements:  * Support for SSL configuration of Extended Master Secret TLS extension in the web server.
   
   
 * Outbound SSL inspection enhancements:  * Support for Extended Master Secret TLS extension. 
    * Support for Channel ID TLS extension and ChaCha20-Poly1305 Cipher Suites (for Chrome browser to access Google web sites).
   
   
 * XGS/GX parity features:  * Support for Email Response object content customization. 
    * Added additional information to the IPS events sent to SiteProtector System, including event-type, Issue ID, target-ip-addr-start, and target-ip-addr-end.
   
   
 * Administrator Settings:  * Added OK option in login prompt for both LMI and CLI when enabling login warning banner. 
    * Added a configurable policy control for minimum length of new password.
   
   
 * LMI/CLI enhancements:  * Added system summary information in LMI dashboard and CLI. 
    * Added the capability to configure the unique rule label for the Access events of the Network Access Policy rules. 
    * Added a Clear Events button to the Network Access Events and IPS Events page. 
    * Added filtering function to the network objects when creating a Network Access Policy rule. 
    * Added a Download button to the Manage > Available Updates page to download an update without installing it. 
    * Added Speed/Duplex settings to the Management Interfaces page. 
    * Added a warning message to the pending policy deployment page in LMI for potential link flapping due to policy change. 
    * Support for grep function for log viewer CLI > log > grep 
    * Added a function that allows the system event details list to be downloaded through LMI Help page.
   
   
 * Support for Remote Syslog over TLS. 
 * Serviceability:  * Added the capability to log the enter and exit software bypass in system events as critical events. 
    * Added the capability to report unanalyzed and dropped frames. 
    * Added more details to Outbound SSL Inspection failures in system events and improved existing Inbound SSL Inspection logging. 
    * Added additional system information to support file, including top, date, and uptime. 
    * Added the capability to include appliance policy change history with Subversion in the support file.
   
   

This release includes all of the defect fixes from firmware update 5.3.2.3. For a list of those fixes, see technote #1984078:  * http://www.ibm.com/support/docview.wss?uid=swg21984078 [http://www.ibm.com/support/docview.wss?uid=swg21984078]

Note: Some enhancements are only available when the IBM Security Network Protection XGS appliance is updated with the latest machine code. For details, see technote #1989529:  * http://www.ibm.com/support/docview.wss?uid=swg21986529 [http://www.ibm.com/support/docview.wss?uid=swg21986529]

Compatibility 
The following web browsers are currently supported by the IBM Security Network Protection local management interface:  * Internet Explorer 10 or 11 
 * Firefox 28 and later 
 * Google Chrome 34 and later

To manage Network Protection 5.3.3 appliances using the SiteProtector System, you must apply the following database service packs:  * SiteProtector System 3.0 - Install all DBSPs up to and including SP3.0 DBSP 3.0.0.55 
 * SiteProtector System 3.1.1 - Install all DBSPs up to and including SP3.1.1 DBSP 3.1.1.37

Important: Ensure that the SiteProtector Core is at version 3.1.1.5 before applying this Database Service Pack (DBSP) update to the IBM Security Network Protection appliance. The following schemas will be updated upon installing the DBSP: 

Policy 

 *  Advanced Threat Policy IPS Event Filter Management Interface Access Network Access Open Signatures System Alerts Update Servers Administrator Settings 

Shared Objects  *  Email Response Remote Syslog Intrusion Prevention
