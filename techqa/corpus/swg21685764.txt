Title: IBM 【セキュリティ情報】IBM WebSphere MQ Hypervisor Edition for Red Hat Enterprise Linux ServerのBashの脆弱性 (CVE-2014-6271, CVE-2014-7169, CVE-2014-7186, CVE-2014-7187, CVE-2014-6277, CVE-2014-2014-6278) - United States

Text:
 SECURITY BULLETIN

SUMMARY
 IBM WebSphere MQ Hypervisor Edition for Red Hat Enterprise Linux Server を使用されている場合、Bashの脆弱性に対する対応が必要です。 

VULNERABILITY DETAILS
IBM WebSphere MQ Hypervisor Edition for Red Hat Enterprise Linux Server を使用されている場合、 "Bash Bug" もしくは "Shellshock"と呼ばれるBashの脆弱性と２つのメモリー破壊の脆弱性に対する対応が必要です。 

CVE ID : 

 CVE-2014-6271, CVE-2014-7169, CVE-2014-7186, CVE-2014-7187, CVE-2014-6277 CVE-2014-6278 

AFFECTED PRODUCTS AND VERSIONS
次の製品、バージョンを使用されている場合、対応が必要です。 

・IBM WebSphere MQ Hypervisor Edition for Red Hat Enterprise Linux Server V7.0.1, V7.5

REMEDIATION/FIXES
 

Red Hat Enterprise Linux (RHEL)のBashの脆弱性に対する修正を入手し、導入してください。
