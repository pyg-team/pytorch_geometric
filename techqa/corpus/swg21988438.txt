Title: IBM API Connect V5.0.3.0 is available - United States

Text:
 NEWS

ABSTRACT
 IBM API Connect V5.0.3.0 is available which addresses several APARs, and includes product enhancements. 

CONTENT


IBM API Connect V5.0.3.0 is now available, which provides important development and APAR fixes, and includes product enhancements as well as a more consistent user experience. 

For information on new features that have been included in V5.0.3.0, see What's new for this release [https://www.ibm.com/support/knowledgecenter/SSMNED_5.0.0/com.ibm.apic.overview.doc/overview_whatsnew.html]. 

We advise all users of IBM API Connect V5.0 to install this update to take advantage of the fixes.
