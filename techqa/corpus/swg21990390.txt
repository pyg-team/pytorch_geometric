Title: IBM Security Bulletin: Apache Commons FileUpload vulnerable to denial of service (CPU consumption) are affecting Case Manager Client (CVE-2016-3092) - United States

Text:
 SECURITY BULLETIN

SUMMARY
 Apache Commons FileUpload vulnerable to denial of service (CPU consumption) are affecting Case Manager Client. 

VULNERABILITY DETAILS
Apache Commons FileUpload that are being utilized by the Forms widget in Case Manager Client when you are working with IBM FileNet eForms are vulnerable to denial of service (CPU consumption) attacks.
CVEID: CVE-2016-3092 [http://cve.mitre.org/cgi-bin/cvename.cgi?name=CVE-2016-3092]

DESCRIPTION: The MultipartStream class in Apache Commons Fileupload before 1.3.2 allows remote attackers to cause a denial of service (CPU consumption) via a long boundary string.

CVSS Base Score: 5.3
CVSS Temporal Score: See https://exchange.xforce.ibmcloud.com/vulnerabilities/114336 [https://exchange.xforce.ibmcloud.com/vulnerabilities/114336] for more information.
CVSS Environmental Score*: Undefined
CVSS Vector: (CVSS:3.0/AV:N/AC:L/PR:N/UI:N/S:U/C:N/I:N/A:L)

AFFECTED PRODUCTS AND VERSIONS
IBM Case Manager 5.1.1.0 - 5.1.1.3
IBM Case Manager 5.2.0.0 - 5.2.0.4
IBM Case Manager 5.2.1.0 - 5.2.1.5
