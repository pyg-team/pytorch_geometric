APAR STATUS
 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  You might encounter the following accessibility issues in IBM
   Process Portal, beyond what was previously addressed in APAR
   JR48814 [http://www-01.ibm.com/support/docview.wss?uid=swg1JR48814]. These issues were found by using JAWS V15.0.1.2085,
   which is screen-reading software from Freedom Scientific.
   
   In the Modify Task window
    -The "Escalate or modify the settings" description is not read.
    -When the focus is in the Assignee field, and you press the Esc
   key, the window does not close.
   
   In Work > My Tasks
     -When you tab to the task link, "Task Subject button" is read,
   but the name of the task is not read.
     -The task priority is not read.
   
   PRODUCTS AFFECTED
   IBM Business Process Manager (BPM) Advanced
   IBM BPM Standard
   IBM BPM Express
   
   
    
   
   

LOCAL FIX

PROBLEM SUMMARY
 *  Testing with JAWS V15.0.1.2085 revealed the accessibility issues
   reported in the ERROR DESCRIPTION. Previous versions of JAWS
   did not reveal these issues.
   
   
    
   
   

PROBLEM CONCLUSION
 *  A fix is available for IBM BPM V8.5.0.1 that fixes the reported
   accessibility issues.
   
   On Fix Central (http://www.ibm.com/support/fixcentral), search
   for JR51998:
   
    1. Select IBM Business Process Manager with your edition from
   the product selector, the installed version to the fix pack
   level, and your platform, and then click Continue.
    2. Select APAR or SPR, enter JR51998, and click Continue.
   
   When you download fix packages, ensure that you also download
   the readme file for each fix. Review each readme file for
   additional installation instructions and information about the
   fix.
