APAR STATUS
 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  The issue is that job rerun from impact view fails with
   AWSUI6401E error.
   
   AWSUI6401E An error occurred while submitting the action
   "Rerun..." on object <JOBNAME>. Reason: member jobDefinition
   not found
   
   Environment:
   MDM: IWS 9.3.0 + FP0002 on AIX
   DWC: DWC 9.3.0 + FP0002 on AIX
   Browswer IE 11 -> NG
            FireFox ESR 52.4.0 (32 bit) -> OK
   
   Replication steps:
   Example JS.
   SCHEDULE FAM93MDM#JS001
   :
   FAM93MDM#JOB001
   END
   
   0. Submit JS001 and let JOB001 run.
   1. Login to DWC.
   2. System Status and Health -> Montor Workload and list job
   streams.
   3. Select JS001 -> Graphical View -> Impact View.
   4. Select JBO001 and right click -> Rerun.
   5. Rerun Job window popups, but it is blank popup with "OK" and
   "cancel".
   6. Click "OK" and AWSUI6401E is displayed.
   
   Notes:
   -The same job can be rerun from Monitor "JS001" jobs list.
   -The issue does not occur using FireFox.
    -> Rerun Job popup windows is not blank and expected
   information is displayed.
   
   
    
   
   

LOCAL FIX
 *  Rerun from Monitor jobs list.
