ERROR DESCRIPTION
 *  Multiple tabs display same selection in Admin Console
   
   
   Description: When opening mulitple tabs rapidly in ITIM Admin
   Console
   both tabs display the same action type instead of the screen for
   the
   selected itim.
   
   Steps to Duplicate:
   1. Login to ITIM Admin Console
   2. Click on Manage User
   3. Before this task get completely launched i.e. Before Manage
   User tab
   get displayed on UI, click on Manage Service.
   4. You will be able to see Manage Users tab displaying the UI of
   Manage
   Services.
   
   Desired Behavior: To be able to open multiple tabs in Admin
   Console
   without issue.
   
   Environment: ITIM V5.1 FP6
   *****************************************
   
   
    
   
   

LOCAL FIX
 *  n/a
   
   
    
   
   

PROBLEM SUMMARY
 *  Documentation Update:
   When opening mulitple tabs rapidly in the ITIM Admin Console,
   the tabs may display the same action type instead of the screen
   for the selected ITIM action.
   Note: The following update will be posted in a future revision
   of the ITIM Infocenter in the section of "Known Limitations,
   problems and workarounds".
   This is one of the known problems with the ITIM UI framework.
   No fix is available at this moment. Please use one of the
   following workarounds.
   1. This issue can be prevented by not launching a new task from
   the task pane until the previous task page gets loaded
   completely on the browser.
   2. If multiple tasks get loaded on the same task tab then
   refreshing the page will correct the problem. You can press the
   F5 button on your keyboard or use the browser refresh button to
   refresh the page.
