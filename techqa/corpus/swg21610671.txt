Title: IBM Launching Rational Rhapsody client results in "The desired vendor daemon is down" with Flexlm error -97,121 - United States

Text:
Rhapsody; licensing; desired vendor daemon is down; -97; 121; 97; communication problem TECHNOTE (TROUBLESHOOTING)

PROBLEM(ABSTRACT)
 Attempts to launch IBM Rational Rhapsody client results in the error “The desired vendor daemon is down” with Flexlm error -97,121 

SYMPTOM
IBM Rational Rhapsody throws following error: 


[/support/docview.wss?uid=swg21610671&amp;aid=1]

CAUSE
You are using Rational License Server version which cannot serve latest Rhapsody license keys generated from License Key Center website to the client application.



RESOLVING THE PROBLEM
Download and install IBM Rational License Key Server 8.1.1 or higher to serve latest IBM Rational Rhapsody license keys generated from IBM Rational License Key Center website to the client application.

Note: IBM Rational Rhapsody floating license keys and IBM Rational License Key Server are backward compatible.
