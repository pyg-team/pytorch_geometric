Title: IBM 【セキュリティ情報】WebSphere Commerce 同梱の IBM WebSphere Application Serverの潜在的なセキュリティ脆弱性 (CVE-2016-3485) - United States

Text:
 SECURITY BULLETIN

SUMMARY
 IBM WebSphere Application Serverは、WebSphere Commerceののコンポーネントとして含まれています。IBM WebSphere Application Serverに影響するセキュリティの脆弱性に関する情報が、セキュリティ情報に掲載されました。



VULNERABILITY DETAILS
IBM WebSphere Application Serverのセキュリティ情報が公開されていますので、下記を参照し、対応お願いします。
Security Bulletin: Multiple vulnerabilities in IBM® Java SDK affects WebSphere Application Server July 2016 CPU (CVE-2016-3485)
http://www.ibm.com/support/docview.wss?uid=swg21988339 [http://www.ibm.com/support/docview.wss?uid=swg21988339]
