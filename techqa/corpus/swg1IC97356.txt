APAR STATUS
 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  When a Web Service Proxy service's WSDL file is updated at the
   same time as messages are being processed, a message might not
   be processed properly.  The DataPower appliance might also
   restart.
   
   
    
   
   

LOCAL FIX

PROBLEM SUMMARY
 *  A Web Service Proxy WSDL that is updated while messages are
   being processed does not allow proper processing of the
   messages.
   
   
    
   
   

PROBLEM CONCLUSION
 *  Fix is available in 5.0.0.12, 6.0.0.4 and 6.0.1.0.

   http://www-01.ibm.com/support/docview.wss?uid=swg21237631 [http://www-01.ibm.com/support/docview.wss?uid=swg21237631]
