RESOLVING THE PROBLEM
To remove and re-install the Deployment Engine so that you can successfully in the Webform Server, follow the steps below. 

. 

1. Open command prompt 

2. execute # mv /usr/ibm/common/acsi /usr/ibm/common/acsi_bkp 

3. execute # mv /var/ibm/common/acsi /var/ibm/common/acsi_bkp 

4. Clean up the /tmp directory, by removing the acu_de.log file, if it is exists. 

5. Reinstall the DE using the Webform Server installer. 

Note: Stop the installer once you pass the Deployment Engine installation panel (it is recommended to cancel the install on the Pre-Installation Summary Panel). 

At this point, you have installed an updated version of the Deployment Engine and can proceed to install any of the Forms products.
