Title: IBM Security Bulletin: A security vulnerability has been identified in embedded IBM WebSphere Application Server shipped with Tivoli Netcool/OMNIbus WebGUI (CVE-2015-2017) - United States

Text:
 SECURITY BULLETIN

SUMMARY
 Embedded Websphere Application Server (eWAS) is shipped as a component of Tivoli Netcool/OMNIbus WebGUI. Information about a security vulnerability affecting eWAS has been published in a security bulletin. 

VULNERABILITY DETAILS
Please consult the security bulletin Security Bulletin: HTTP response splitting attack in WebSphere Application Server (CVE-2015-2017) [http://www-01.ibm.com/support/docview.wss?uid=swg21966837] for vulnerability details and information about fixes.
