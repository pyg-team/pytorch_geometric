Title: IBM Security Bulletin: IBM License Metric Tool v9 is vulnerable to two attacks on Ruby on Rails component - CVE-2014-0130, CVE-2014-7829 - United States

Text:
 SECURITY BULLETIN

SUMMARY
 IBM License Metric Tool v9 is vulnerable to two exploits related to Ruby on Rails framework. Ruby on Rails handles, among others, network communications of the IBM License Metric Tool v9 server.
CVE-2014-0130 allows an unauthorized attacker to read any file from the machine that is hosting IBM License Metric Tool v9 server, using a specially prepared HTTP request.
CVE-2014-7829 allows an unauthorizes attacker to determine, whether a given file exists on the machine that is hosting IBM License Metric Tool v9 server, using a specially prepared HTTP request. 

VULNERABILITY DETAILS
CVE-ID: CVE-2014-0130 [https://cve.mitre.org/cgi-bin/cvename.cgi?name=CVE-2014-0130]
Description: Ruby on Rails could allow a remote attacker to traverse directories on the system. An attacker could send a specially-crafted URL request to the implicit render functionality to read arbitrary files on the system.
CVSS Base Score: 5
CVSS Temporal Score: See https://exchange.xforce.ibmcloud.com/vulnerabilities/93202 [https://exchange.xforce.ibmcloud.com/vulnerabilities/93202] for more information
CVSS Environmental Score*: Undefined
CVSS Vector: (AV:N/AC:L/Au:N/C:P/I:N/A:N)

CVE-ID: CVE-2014-7829 [http://www.cve.mitre.org/cgi-bin/cvename.cgi?name=CVE-2014-7829]
Description: Ruby on Rails could allow a remote attacker to obtain sensitive information, caused by an information leak in Action Pack. By sending a specially-crafted request, a remote attacker could exploit this vulnerability to determine if a file exists on the filesystem outside the application's root directory.
CVSS Base Score: 5
CVSS Temporal Score: See https://exchange.xforce.ibmcloud.com/vulnerabilities/98732 [https://exchange.xforce.ibmcloud.com/vulnerabilities/98732] for more information
CVSS Environmental Score*: Undefined
CVSS Vector: (AV:N/AC:L/Au:N/C:P/I:N/A:N)

AFFECTED PRODUCTS AND VERSIONS
IBM License Metric Tool v9

REMEDIATION/FIXES
Upgade IBM License Metric Tool server to version 9.2.0.0 manually or with a fixlet.

WORKAROUNDS AND MITIGATIONS
