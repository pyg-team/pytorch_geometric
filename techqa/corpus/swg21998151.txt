Title: IBM API Connect Developer Portal V5.0.6.1 Public Interim Fix 001 is available - United States

Text:
 NEWS

ABSTRACT
 A public interim fix is available for IBM API Connect Developer Portal V5.0.6.1 which addresses the APAR: LI79500 

CONTENT


IBM API Connect Developer Portal V5.0.6.1 Public Interim Fix 001 is available. This interim fix provides an important APAR fix on top of IBM API Connect Developer Portal V5.0.6.1. We advise all users of IBM API Connect Developer Portal V5.0.6.1 to install this update to take advantage of the fix. 
 

APAR FIXES
THIS IS A CUMULATIVE INTERIM FIX AND INCLUDES THE FOLLOWING APAR FIXES:
APAR Summary LI79500 API CONNECT V5.0.6.1 PORTAL UPGRADE SITES LOST
