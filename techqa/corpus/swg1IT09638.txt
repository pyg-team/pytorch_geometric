ERROR DESCRIPTION
 *  When creating SSL proxy profiles for a DataPower MQ-QM object an
   unexpected disconnect may occur while establishing the
   connection.
   
   
    
   
   

LOCAL FIX

PROBLEM SUMMARY
 *  Customers who use security options will meet unexpected
   disconnection, because the generated security profiles are
   corrupted.

 *  Fix is available in 6.0.0.16, 6.0.1.12, 7.0.0.9, 7.1.0.6 and
   7.2.0.1.
