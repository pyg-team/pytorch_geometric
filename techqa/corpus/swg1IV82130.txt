APAR STATUS
 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  Netcool/OMNIbus WebGUI v8.1.0.5 introduces an API change
   designed to improve GUI performance and scalability in the
   browser for event-based widgets. This API change reduces the
   number of browser connections as well as bandwidth requirements.
   
   Network Health Dashboard v4.2 utilizes this API in its
   event-based widgets (Event Timeline and Unavailable Resources)
   and an update is required in the Network Health Dashboard to
   accommodate the updated WebGUI API. The WebGUI API update is not
   backwards compatible, so you must apply this Network Health
   Dashboard fix when you upgrade to Netcool/OMNIbus WebGUI
   v8.1.0.5. Otherwise, the Network Health Dashboard Event Timeline
   and Unavailable Resources widgets will no longer work.
