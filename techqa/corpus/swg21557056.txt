Title: IBM Microsoft Excel and CSV Export Files containing Unicode Characters - United States

Text:
 TECHNOTE (TROUBLESHOOTING)

PROBLEM(ABSTRACT)
 If you are using Microsoft Excel and the CSV export file you have exported contains unicode characters, to get these characters to appear correctly, you need to follow the steps below when loading the file into Excel. 

SYMPTOM
Unicode characters appear incorrectly when CSV export file is loaded into Microsoft Excel.

CAUSE
Microsoft Excel does not automatically detect that the CSV export file contains UTF-8 characters.

RESOLVING THE PROBLEM
1. Rename the file so that it has a ".txt" extension. 

2. Open Excel.
3. Go to File -> Open and open the .txt file.
4. The "Text Import Wizard" should be displayed.
5. Change the File Type from "Fixed Width" to "Delimited".
6. Change the File Origin from "Windows (ANSI)" to "65001: Unicode (UTF-8)".
7. Click Finish.
