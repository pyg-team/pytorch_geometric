Title: IBM JVMCFRE003 bad major version; class=com/ibm/edms/od/OD - United States

Text:
odwek; java; java 7; java 1.7; 1.7; JVMCFRE003; bad major version TECHNOTE (TROUBLESHOOTING)

PROBLEM(ABSTRACT)
 JVMCFRE003 bad major version; class=com/ibm/edms/od/OD 

SYMPTOM
Attempt to connect to the Content Manager OnDemand server from IBM Content Navigator, fails with an error: "JVMCFRE003 bad major version; class=com/ibm/edms/od/OD"



CAUSE
This error most often occurs when the application server instance (JVM) to which IBM Content Navigator is installed to, is configured to use Java 1.6 and the OnDemand Web Enablement Kit (ODWEK) version installed on the Navigator server is 9.5 or later fixpacks. ODWEK v9.5 requires
that the JVM be configured to use JAva 1.7.

Update the JVM that Navigator is deployed to, to use Java 1.7 to resolve the issue.

As an example, refer to the following two links to update a WAS 8.5.5 JVM to use Java 1.7:
