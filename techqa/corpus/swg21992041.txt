Title: IBM Security Bulletin:Open Source Apache Poi Vulnerability in IBM eDiscovery Manager - United States

Text:
 SECURITY BULLETIN

SUMMARY
 Apache POI could allow a remote attacker to obtain sensitive information, caused by an XML External Entity Injection (XXE) error when XLSX2CSV example uses Java's XML components to parse OpenXML files. An attacker could exploit this vulnerability using an XML document containing an external entity reference to read arbitrary files on the system.
