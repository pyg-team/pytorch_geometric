 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  When ICN was being upgraded, the customer decided to change the
   Context Root.  This caused the Image Viewer in Datacap Navagator
   to stop displaying images (404 error).  All other applications
   using the Image Viewer were able to display images.
