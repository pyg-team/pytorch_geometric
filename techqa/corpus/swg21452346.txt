QUESTION
 Does CICS Transaction Server for z/OS support the use of a DB2 subgroup attach name? You would like to specify the subgroup attach name as the DB2GROUPID on the DB2CONN definition for CICS-DB2 group attach. 

ANSWER
Apply the PTF UK62289 (APAR PM25602) for CICS TS 4.1. No PTFs are required for CICS TS 4.2 and above as the base code supports the use of a DB2 subgroup attach name.

See the following for more information: 

 * Group attachment names and subgroup attachment names [https://www.ibm.com/support/knowledgecenter/SSEPEK_12.0.0/inst/src/tpc/db2z_groupattachnamespecs.html] in the DB2 V10 information center. 
 * DB2GROUPID under DB2CONN Connection attributes [http://publib.boulder.ibm.com/infocenter/cicsts/v4r2/topic/com.ibm.cics.ts.resourcedefinition.doc/resources/db2conn/dfha4_connattr.html] in the CICS TS V4.2 information center.


Note that these APARs add support for new DB2 V10 subgroup attach functionality. For group attach or non group attach, no CICS APARs are required. RELATED INFORMATION
