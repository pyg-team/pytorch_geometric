Title: IBM Security Bulletin: TLS padding vulnerability affects IBM HTTP Server shipped with IBM Presence Zones (CVE-2014-8730) - United States

Text:
tls; CVE-2014-8730 SECURITY BULLETIN

SUMMARY
 IBM HTTP Server is shipped as a component of IBM Presence Zones. Information about a security vulnerability affecting IBM HTTP Server has been published in a security bulletin. 

VULNERABILITY DETAILS
Consult the TLS padding vulnerability affects IBM HTTP Server (CVE-2014-8730) [http://www.ibm.com/support/docview.wss?uid=swg21692502] security bulletin for vulnerability details and available fixes.

AFFECTED PRODUCTS AND VERSIONS
IBM Presence Zones Version 1.0 and later
