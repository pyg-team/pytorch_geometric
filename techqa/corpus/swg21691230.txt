Title: IBM ObjectServer and LDAP for authentication to WebGUI - United States

Text:
login; security; access TECHNOTE (FAQ)

QUESTION
 Can I have ObjectServer and LDAP user repositories to authenticate users logging in to WebGUI? 

CAUSE
Administrators would like to use ObjectServer and LDAP user repositories for authentication at the same time.

ANSWER
Having ObjectServer and LDAP user repositories at the same time is not supported for WebGUI user authentication. You can switch from ObjectServer to LDAP and vice versa. The following WebGUI 8.1 documentation link provides a table on current and planned user repositories and the steps involved to switch repositories:

Configuring user authentication:
