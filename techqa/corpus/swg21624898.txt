Title: IBM DB2 monitoring agent data collection methods - United States

Text:
5724B96DOTR; 5724B96DOIN TECHNOTE (FAQ)

QUESTION
 How does the ITCAM monitoring agent for DB2 database collect monitoring data? 

ANSWER
The DB2 database monitoring agent collects data using following four methods:

1) DB2 Snapshot Monitor - Snapshot API requests and responses. 
2) DB2 SQL Command Processor - When using SQL Commands, the agent uses select query to collect data. 
3) DB2 configuration parameters - DB Get Configuration Parameter API's 
4) DB2 Manager configuration parameters - DBM Get/Set Configuration Parameter API's 


Note: The collection method for each of the DB2 monitoring agent attribute groups is not published. An attribute group may display monitoring data collected using one or more of the above mentioned methods.
