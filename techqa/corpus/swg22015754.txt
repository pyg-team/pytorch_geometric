Title: IBM Security Bulletin: BigFix Platform 9.5.x / 9.2.x affected by multiple 
vulnerabilities (CVE-2018-1600, CVE-2018-1473, CVE-2018-1479, CVE-2018-1475) - United States

Text:
 SECURITY BULLETIN

SUMMARY
 The BigFix Platform versions 9.2 and 9.5 are exhibiting vulnerabilities in the following categories:
Clear text passwords over the internet, Cross site scripting, Cross site request forgery, Improper restriction of excessive authentication attempts. 
These vulnerabilities have been addressed in patch releases 9.2.14 and 9.5.9. 

VULNERABILITY DETAILS

CVEID: CVE-2018-1600 [http://cve.mitre.org/cgi-bin/cvename.cgi?name=CVE-2018-1600] 

DESCRIPTION: IBM BigFix Platform transmits sensitive or security-critical data in clear text in a communication channel that can be sniffed by unauthorized actors. 

CVSS Base Score: 8.6
CVSS Temporal Score: https://exchange.xforce.ibmcloud.com/vulnerabilities/143745 [https://exchange.xforce.ibmcloud.com/vulnerabilities/143745]
CVSS Environmental Score*: Undefined
CVSS Vector: CVSS:3.0/AV:N/AC:L/PR:N/UI:N/S:C/C:H/I:N/A:N 

CVEID: CVE-2018-1473 [http://cve.mitre.org/cgi-bin/cvename.cgi?name=CVE-2018-1473] 

DESCRIPTION: IBM BigFix Console component and IBM BigFix Relay Diagnostic page are vulnerable to cross-site scripting. This vulnerability allows users to embed arbitrary JavaScript code in the Web UI,with the potential to alter the intended functionality. 

CVSS Base Score: 6.1
CVSS Temporal Score: https://exchange.xforce.ibmcloud.com/vulnerabilities/140691 [https://exchange.xforce.ibmcloud.com/vulnerabilities/140691]
CVSS Environmental Score*: Undefined
CVSS Vector: CVSS:3.0/AV:N/AC:L/PR:N/UI:R/S:C/C:L/I:L/A:N 

CVEID: CVE-2018-1479 [http://cve.mitre.org/cgi-bin/cvename.cgi?name=CVE-2018-1479] 

DESCRIPTION: IBM BigFix Relay Diagnostic page is vulnerable to cross-site request forgery which could allow an attacker to execute malicious and unauthorized actions transmitted from a user that the website trusts. 

CVSS Base Score: 4.3
CVSS Temporal Score: https://exchange.xforce.ibmcloud.com/vulnerabilities/140761 [https://exchange.xforce.ibmcloud.com/vulnerabilities/140761]
CVSS Environmental Score*: Undefined
CVSS Vector: CVSS:3.0/AV:N/AC:L/PR:N/UI:R/S:U/C:N/I:L/A:N 

CVEID: CVE-2018-1475 [http://cve.mitre.org/cgi-bin/cvename.cgi?name=CVE-2018-1475] 

DESCRIPTION: IBM BigFix Relay Diagnostic page is vulnerable to brute force attacks on accounts due to improper restriction of excessive authentication attempts. 

CVSS Base Score: 7.5
CVSS Temporal Score: https://exchange.xforce.ibmcloud.com/vulnerabilities/140756 [https://exchange.xforce.ibmcloud.com/vulnerabilities/140756]
CVSS Environmental Score*: Undefined
CVSS Vector: CVSS:3.0/AV:N/AC:L/PR:N/UI:N/S:U/C:H/I:N/A:N

AFFECTED PRODUCTS AND VERSIONS
 

Affected IBM BigFix Platform Affected Versions BigFix Platform 9.5 - 9.5.8 BigFix Platform 9.2 - 9.2.13 REMEDIATION/FIXES
 

Product VRMF Remediation / First Fix BigFix Platform 9.5.9 Apply the upgrade-patch 9.5.9 by looking for the associated upgrade-patch fixlet in the Console, then launch and apply it. BigFix Platform 9.2.14 Apply the upgrade-patch 9.2.14 by looking for the associated upgrade-patch fixlet in the Console, then launch and apply it. WORKAROUNDS AND MITIGATIONS
None
