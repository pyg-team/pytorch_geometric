Title: IBM Is there a maximum number of columns that can be displayed in a view? - United States

Text:
 TECHNOTE (FAQ)

QUESTION
 Is there a maximum number of columns that can be displayed in a view? 

ANSWER


The maximum number of columns that can be displayed in a view is 32. You can have more than 32 columns in a single formal module but these then need to be distributed across at least 2 views.
