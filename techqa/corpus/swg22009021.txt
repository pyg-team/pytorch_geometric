Title: IBM Unable to join meeting with the latest java version - United States

Text:
 TECHNOTE (TROUBLESHOOTING)

PROBLEM
Unable to join meeting with the latest java version

RESOLVING THE PROBLEM
Locate the java.security file. For example,

C:\Program Files (x86)\Java\jre1.8.0_xx\lib\security\java.security

Change the following lines.

BEFORE
jdk.jar.disabledAlgorithms=MD2, MD5, RSA keySize < 1024
AFTER
jdk.jar.disabledAlgorithms=MD2, RSA keySize < 1024 

The above parameter was introduced starting in Java 8 Update 111 (8u111) as per their website: 
https://www.java.com//en/download/faq/release_changes.xml [https://www.java.com//en/download/faq/release_changes.xml]
