Title: IBM Security Bulletin: Cross-site Scripting vulnerabilities affect Rational Engineering Lifecycle Manager - United States

Text:
PSIRT; security SECURITY BULLETIN

SUMMARY
 Rational Engineering Lifecycle Manager is vulnerable to multiple cross-site scripting attacks with potential for credentials disclosure within a trusted session. 

VULNERABILITY DETAILS
CVEID: CVE-2017-1324 [http://cve.mitre.org/cgi-bin/cvename.cgi?name=CVE-2017-1324]
DESCRIPTION: IBM RELM is vulnerable to cross-site scripting. This vulnerability allows users to embed arbitrary JavaScript code in the Web UI thus altering the intended functionality potentially leading to credentials disclosure within a trusted session.
CVSS Base Score: 5.4
CVSS Temporal Score: See https://exchange.xforce.ibmcloud.com/vulnerabilities/125975 [https://exchange.xforce.ibmcloud.com/vulnerabilities/125975] for the current score

REMEDIATION/FIXES
For 6.0 - 6.0.4 releases, upgrade to one of the following versions: 


 * Upgrade to 6.0.2 ifix13 or later:  *  Get the CLM 6.0.2 ifix13 or later from: CLM 6.0.2 iFix13 [https://www.ibm.com/support/fixcentral/swg/selectFixes?parent=ibm~Rational&product=ibm/Rational/Rational+Collaborative+Lifecycle+Management+Solution&release=6.0.2&platform=All&function=all] Start the package installation and select RELM when asked about the products to be updated. 
   
   
 * Or upgrade to version 6.0.3 ifix08 or later:  *  Get the CLM 6.0.3 ifix08 or later from: CLM 6.0.3 iFix08 [https://www.ibm.com/support/fixcentral/swg/selectFixes?parent=ibm~Rational&product=ibm/Rational/Rational+Collaborative+Lifecycle+Management+Solution&release=6.0.3&platform=All&function=all] Start the package installation and select RELM when asked about the products to be updated. 
   
   
 * Or upgrade to version 6.0.4 ifix01 or later:  *  Get the CLM 6.0.4 ifix01 or later from: CLM 6.0.4 iFix01 [https://www.ibm.com/support/fixcentral/swg/selectFixes?parent=ibm~Rational&product=ibm/Rational/Rational+Collaborative+Lifecycle+Management+Solution&release=6.0.4&platform=All&function=all] Start the package installation and select RELM when asked about the products to be updated. 
   
   

For the 5.x releases, upgrade to version 5.0.2 iFix03 or later  *   1. Get the 5.0.2 ifix03 from: Rational Engineering Lifecycle Manager 5.0.2 iFix03 [http://www.ibm.com/support/fixcentral/swg/quickorder?parent=ibm%7ERational&product=ibm/Rational/Rational+Engineering+Lifecycle+Manager&release=5.0.2&platform=All&function=fixId&fixids=5.0.2-Rational-RELM-ifix003&includeRequisites=1&includeSupersedes=0&downloadMethod=http&source=fc] 
    2. Follow the instructions in the Readme file to apply the Server patch.
   
   

For the 4.x releases, and any prior versions of the products listed above, IBM recommends upgrading to a fixed, supported version/release/platform of the product. 

If the iFix is not found in the Fix Portal please contact IBM Support.
