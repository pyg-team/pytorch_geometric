 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  A console user found that a scheduled virtual system pattern
   deployment would not display the instance details.
   
   The recreate steps for this issue are:
   - Select a virtual system pattern, and click "Deploy".
   - Schedule the deployment to start at a later time.
   - Click the "Quick Deploy" button.
   - Navigate to Patterns > Virtual System Instances.
     The scheduled deployment appears with a status of "QUEUEING".
   - Click on that instance to view the instance details.
     The result is the spinning "processing" icon, which never
     completes. The details are not displayed.
   
   
    
   
   

LOCAL FIX

PROBLEM SUMMARY
 *  The console JavaScript code implementing the deployment
   details view was not completing rendering.
   
   
    
   
   

PROBLEM CONCLUSION
 *  A code fix is being provided in PureApplication System version
   2.2.2.0.
