Title: IBM Netcool/OMNIbus Integrations Release Notice - Netcool/OMNIbus Probe for Huawei U2000 CORBA (nco_p_huawei_u2000_corba-3_0) - United States

Text:
release notice; downloads; nco_p_huawei_u2000_corba; Probe for Huawei U2000 CORBA NEWS

ABSTRACT
 A new version of the Netcool/OMNIbus Probe for Huawei U2000 CORBA will be available to download from March 10, 2016. 

CONTENT
 

This release addresses the following Enhancement Requests:

 * RFE 76265: Support for acknowledge and unacknowledge operations added.
 * RFE 71705: Support for using the host name as a lookup in the Interoperable Object Reference added.
 * RFE 70238: ProbeWatch extension for Huawei U2000 Corba Probe. Additional ProbeWatch message is sent after the last re-synchronized alarm received.

This release addresses the following APAR:

 * IV65119: The probe fails to start when the ORBLocalPort is set, but runs correctly when ORBLocalPort is not set.

Netcool/OMNIbus V8.1

If you are running Netcool/OMNIbus V8.1, a single package contains versions of the probe for all supported Operating Systems. To obtain this new package, use the Find by part number field to search for part number: CN9JXEN

 Note: The Netcool/OMNIbus V8.1 version of this probe supports the following Operating Systems: AIX, Linux, Solaris and zLinux. 

Netcool/OMNIbus V7.4.0
