About Fix Pack 9.0.0.4 Fix pack 9.0.0.4 can be applied to WebSphere Application Server traditional Version 9.0 on all supported platforms. The fix pack must be applied on top of an existing Version 9.0 installation. 

To download the fix pack packages, see the Fix Pack 9.0.0.4 download page [http://www.ibm.com/support/docview.wss?uid=swg24043693]. 

To download packages for the initial Version 9.0 installations, see Passport Advantage Online [http://www-01.ibm.com/support/docview.wss?uid=swg27048323] and Shopz [http://www-304.ibm.com/software/shopzseries/ShopzSeries_public.wss] for z/OS.

Prerequisites 
Verify that your system meets the relevant requirements listed on WebSphere Application Server detailed system requirements [http://www.ibm.com/support/docview.wss?rs=180&uid=swg27006921] [Prashanth%20(P.)%20Gunapalasingam:%20%0A10:27:19%20AM:%20http://www-01.ibm.com/support/docview.wss?uid=swg27005002]before proceeding with the installation. 
Fix Pack 9.0.0.4 must be installed on top of existing Version 9.0 offerings. 

Installation Manager 1.8.5 or above is required. Download and install the latest available IBM Installation Manager from the [http://www.ibm.com/support/docview.wss?uid=swg24020212]IBM Installation Manager download website [http://www.ibm.com/support/entry/portal/Recommended_fix/Software/Rational/IBM_Installation_Manager].

Space requirements 
The amount of disk space required varies with the number of features or products installed. If you are installing the product using Installation Manager, the installation summary panel indicates the approximate amount of disk space required based on the features and products you have selected. For a fix pack, you need approximately 2 GB to 4 GB of free disk space.
