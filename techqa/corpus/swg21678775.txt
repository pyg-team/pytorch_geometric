Title: IBM Enable Web GUI 8.1 server side debugging mode - United States

Text:
MustGather; Debug; Server TECHNOTE (TROUBLESHOOTING)

PROBLEM(ABSTRACT)
 Steps to enable Web GUI server in debug mode for troubleshooting. 

RESOLVING THE PROBLEM
Option 1: Using server.init 

a) Set the WebGUI to debug in WEBGUI_HOME/etc/server.init.

trace.level=FINEST
trace.maxsize=100
trace.count=5

log.level=FINEST
log.maxsize=100
log.count=5

profile.maxsize=100
profile.count=5


b) Clear WebGUI and JazzSM/DASH logs, so that we get fresh ones.

JazzSM_HOME/profile/logs/server1/*.log
JazzSM_HOME/profile/logs/ncw/*.*

c) Restart JazzSM, so that changes in server.init will take effect.

Options 2: Using GUI

Using the WebSphere administration console [https://www.ibm.com/support/knowledgecenter/SSSHTQ_8.1.0/com.ibm.netcool_OMNIbus.doc_8.1.0/webtop/wip/task/web_adm_setloglevel.html]
