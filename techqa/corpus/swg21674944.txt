Title: IBM Security Bulletin: Security vulnerabilities have been identified in IBM DB2 shipped with WebSphere Remote Server (CVE-2014-0907) - United States

Text:
 SECURITY BULLETIN

SUMMARY
 IBM DB2 is shipped as a component of WebSphere Remote Server. Information about security vulnerabilities affecting IBM DB2 has been published in a security bulletin. 

VULNERABILITY DETAILS
For vulnerability details, see the Local escalation of privilege vulnerability in IBM DB2 (CVE-2014-0907) [http://www.ibm.com/support/docview.wss?uid=swg21672100] document.

AFFECTED PRODUCTS AND VERSIONS
