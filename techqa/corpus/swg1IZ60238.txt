APAR STATUS
 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  Unexpected or incorrect system message appears when search
   string is not found when using Advanced Search for a
   Classification.
   
   Log in to maxdemo instance as Wilson
   Go to Assets, Work Orders, or any module which uses
   Classifications
   Click on Advanced Search
   In the Classification field enter any invalid string, (such as
   "asdfjkl")
   Error message is presented that does not indicate that the
   string was not found, "BMXAA4214E - An unknown error has
   occurred. Please contact your system administrator for
   assistance. null"
