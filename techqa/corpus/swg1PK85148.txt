ERROR DESCRIPTION

   failing item,but the Syndicator/Subscriber forms will not show
   the exception.
   
   
    
   
   

LOCAL FIX

PROBLEM CONCLUSION
 *  WCM not passing exception up, only logging at trace level.  This
   APAR changes WCM behavior so that it is logged, and the
   exception is passed up.
