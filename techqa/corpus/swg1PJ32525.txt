APAR STATUS
 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  Failed to save the change on modifying Maximum Size of a
   storage area
   Product: Content Engine MP
   Release: 4.0.1
   Steps to Reproduce:
   1. Create a Storage Area with default options
   2. After creation, open the Properties page of the storage
   area, select Statistics tab, notice that the Maximum Size
   radio button "Maximum allowed on device" is selected by
   default, now select radio button "MB", set a value "100"
   in the MB text field, click on Apply button, notice the value
   shows "100", click OK to close the properties page
   3. Reopen the properties page, select Statistics tab again
   o  Actual result: "Maximum allowed on device" radio button is
   selected (as the default), the "MB" radio button is not
   selected
   o  Expected result: the "MB" radio button should be
   selected, and the value should be "100" MB
   
   
    
   
   

LOCAL FIX

PROBLEM SUMMARY
 *  Upgrade CE server to P8CE-4.0.1-001
   
   
    
   
   

PROBLEM CONCLUSION

TEMPORARY FIX
