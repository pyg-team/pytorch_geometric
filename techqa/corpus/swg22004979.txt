Title: IBM Security Bulletin: Cross-Site Scripting Vulnerability in IBM WebSphere Portal (CVE-2017-1303) - United States

Text:
 SECURITY BULLETIN

SUMMARY
 A fix is available for a cross-site scripting vulnerability in IBM WebSphere Portal (CVE-2017-1303). 

VULNERABILITY DETAILS
CVEID: CVE-2017-1303 [http://cve.mitre.org/cgi-bin/cvename.cgi?name=CVE-2017-1303]
DESCRIPTION: IBM WebSphere Portal and Web Content Manager is vulnerable to cross-site scripting. This vulnerability allows users to embed arbitrary JavaScript code in the Web UI thus altering the intended functionality potentially leading to credentials disclosure within a trusted session.
CVSS Base Score: 6.1
CVSS Temporal Score: See https://exchange.xforce.ibmcloud.com/vulnerabilities/125457 [https://exchange.xforce.ibmcloud.com/vulnerabilities/125457] for the current score
CVSS Environmental Score*: Undefined
CVSS Vector: (CVSS:3.0/AV:N/AC:L/PR:N/UI:R/S:C/C:L/I:L/A:N) 

AFFECTED PRODUCTS AND VERSIONS
IBM WebSphere Portal 9.0
IBM WebSphere Portal 8.5
IBM WebSphere Portal 8.0
IBM WebSphere Portal 7.0

REMEDIATION/FIXES
 

Product VRMF APARs Fix IBM WebSphere Portal 9.0 PI80564 Upgrade to Cumulative Fix 14 (CF14).
(Combined Cumulative Fixes for WebSphere Portal 9.0.0 [http://www-01.ibm.com/support/docview.wss?uid=swg24037786] [http://www-01.ibm.com/support/docview.wss?uid=swg24037786]) IBM WebSphere Portal 8.5 PI80564 Upgrade to Cumulative Fix 14 (CF14).
(Combined Cumulative Fixes for WebSphere Portal 8.5.0.0 [http://www-01.ibm.com/support/docview.wss?uid=swg24037786] [http://www-01.ibm.com/support/docview.wss?uid=swg24037786]) IBM WebSphere Portal 8.0 PI80564 Upgrade to Fix Pack 8.0.0.1 with Cumulative Fix 22 (CF22) and then apply the Interim Fix PI80564 [http://www.ibm.com/support/fixcentral/swg/selectFixes?parent=ibm~WebSphere&product=ibm/WebSphere/WebSphere+Portal&release=All&platform=All&function=aparId&apars=PI80564]. IBM WebSphere Portal 7.0 PI80564 Upgrade to Fix Pack 7.0.0.2 with Cumulative Fix 30 (CF30) and then apply the Interim Fix PI80564 [http://www.ibm.com/support/fixcentral/swg/selectFixes?parent=ibm~WebSphere&product=ibm/WebSphere/WebSphere+Portal&release=All&platform=All&function=aparId&apars=PI80564].
