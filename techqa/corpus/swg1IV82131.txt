APAR STATUS
 * CLOSED AS DOCUMENTATION ERROR.
    
   
   

ERROR DESCRIPTION
 *  Prior to ITNM 4.2 , right click menus contained the Telnet tool
   to be able to telnet into the device. The tool was removed from
   4.2 for security reasons but the ITNM 4.2 documentation still
   contains references to the Telnet tool.

PROBLEM CONCLUSION
 *  Documentation will be updated in fix pack | 4.2.0-ITNMIP-FP0001.
   We removed applets from the product. Applets allowed us to open
   the CMD prompt on the host machine.
   We do not have a a technical solution that would allow the
   telnet tool to be readded to the right click framework, it's not
   possible to call O/S level commands directly from Javascript.
   The customer can open the CMD prompt themselves, thats a valid
   work around.
