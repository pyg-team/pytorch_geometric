Title: IBM SDK, Java Technology Edition refreshes: October 2016 - United States

Text:
 NEWS

ABSTRACT
 IBM SDK®, Java™ Technology Edition refreshes are available for all current versions. 

CONTENT
The following refreshes of the IBM SDK, Java Technology Edition are available: 

 * Version 8, Service Refresh 3, Fix Pack 20 (8.0.3.20) 
 * Version 7 Release 1, Service Refresh 3, Fix Pack 60 (7.1.3.60) 
 * Version 7, Service Refresh 9, Fix Pack 60 (7.0.9.60) 
 * Version 6 (VM 2.6), Service Refresh 8, Fix Pack 35 (6.1.8.35) 
 * Version 6, Service Refresh 16, Fix Pack 35 (6.0.16.35)



The installable packages for AIX®, Linux™, and z/OS® platform architectures are available from developerWorks [http://www.ibm.com/developerworks/java/jdk/index.html]. 
Note: Version 6 (VM 2.6) is available to download only for z/OS.  

Highlights

Version 8 and Version 7 Release 1 contain enhancements to the Java Cryptography Extension Common Cryptographic Architecture (IBMJCECCA) security provider, as described in What's new [https://knowledgecenters.hursley.ibm.com/java-internal/SSYKE2_8.0.0/com.ibm.java.security.component.80.doc/security-component/whats_new/security_changes_80/security_whatsnew_sr3fp20.html]. This provider is supported on z/OS systems (version 7.1 and version 8) and Linux on z Systems® (version 8 only). 

All refreshes contain the latest IBM fixes and the most recent Oracle Critical Patch Update (CPU) [http://www.oracle.com/technetwork/topics/security/alerts-086861.html#CriticalPatchUpdates].
