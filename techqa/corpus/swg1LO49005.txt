ERROR DESCRIPTION
 *  Customer reported that some Holidays are not syncing to thier
   device.  The problem seems sporadic in nature, but happens
   pretty frequently.  The customer is using Nokia devices and saw
   the problem often on an E71.
   
   
    
   
   

LOCAL FIX
 *  NA
   
   
    
   
   

PROBLEM SUMMARY
 *  Customer reported some holidays did not sync to their Nokia
   devcie.
   
   
    
   
   

PROBLEM CONCLUSION
 *  The server was updated to handle the calendar entries correctly
   so that the Nokia client would accept them.

   all future releases.  See this technote for latest maint
