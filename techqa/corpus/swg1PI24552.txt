 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  Environment:
   
   IBM UrbanCode Deploy 6.1.0.1
   Windows 7
   
   Steps to reproduce:
   
   Attempt to install the Agent Relay.
   
   Launch the installer from a directory different from the
   directory that contains the installer file (install.cmd)
   
   The following error appears:
   
   Caused by: java.lang.ClassNotFoundException:
   org.mozilla.javascript.tools.shell.Main
        at
   java.net.URLClassLoader.findClass(URLClassLoader.java:434)
        at java.lang.ClassLoader.loadClass(ClassLoader.java:653)
        at
   sun.misc.Launcher$AppClassLoader.loadClass(Launcher.java:358)
        at java.lang.ClassLoader.loadClass(ClassLoader.java:619)
   Could not find the main class:
   org.mozilla.javascript.tools.shell.Main.
   Program will exit.
   
   The referenced Main class is located in:
   
   agent-relay-install\install\js.jar\org\mozilla\javascript\tools\
   shell\Main.class
   
   and this class is accessed by the script using a relative path.
   Launching the installer from a different directory should
   provide a clear error message so the user understand what
   corrective action needs to be taken.
