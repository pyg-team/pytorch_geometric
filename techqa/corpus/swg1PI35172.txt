APAR STATUS
 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  In Workspace Advanced when using a Dimensional data source you
   can create 1 calculation using the right-click, Calculation
   functionality but encounter an error when ceating the second
   calculation.
   
   
    
   
   

LOCAL FIX
 *  Avoid using the right-click menu for creating calculations in
   Workspace Advanced.
   Use Query Calculation in Workspace Advanced toolbox to manually
   defile the calculation.
   
   1. Select Toolbox
   2. Drag Query Calculation to the report
   3. Select Other Expression
   4. In Expression dialog, select Data Items tab
   5. Define the calculation
   6. Repeat steps for additional calculations
