APAR STATUS
 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  In case of  failed imports during processing datasoure files,
   the files are marked as not imported. In specific
   situations, the following imports are
   unable to download such files, until the files are refreshed.
   
   
    
   
   

LOCAL FIX
 *  Force upload of the files on affected endpoints.
   
   
    
   
   

PROBLEM SUMMARY
 *  ****************************************************************
   
   * ERROR DESCRIPTION:
   *
   * In case of huge environments and often scan schedules it happe
   *
   * that
   *
   * some files are changed between import start and file download
   *
   * process.
   *
   *
   *
   * This cause Replication failure and the data will be processed
   *
   * the
   *
   * next import time or the next scan time. This is a big problem
   *
   * especially for VM manager data.
   *
   * **************************************************************
   *
   * EXPECTED RESULTS:
   * The issue does not occure.
   ****************************************************************
   * RECOMMENDATION: Apply appropriate update when available.
