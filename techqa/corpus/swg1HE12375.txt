ERROR DESCRIPTION
 *  Multi-line text fields in Data Sources allow for line breaks,
   but they are not represented when saved. The breaks show up when
   you edit the page, but when saved or approved, the breaks are
   removed.
   
   
    
   
   

LOCAL FIX

PROBLEM SUMMARY
 *  Description/Comments added with line breaks in Data Sources are
   being displayed in single line.
   
   
    
   
   

PROBLEM CONCLUSION
 *  Fix is available in Fix Central in 6033 iFix1.
