ERROR DESCRIPTION
 *  Attempts to Check In a prep product in a prep project results in
   error 'Couldn't set permissions on file ...'
   
   STEPS TO REPRODUCE:
   
   1. As a user in the build_mgr role (in the ccm_root group,
   whether primary
   group or not, it doesn't matter) check out a prep product in a
   prep project with a copy based work area.
   
   2. As a different user in the build_mgr role (also in the
   ccm_root group,
   as above) attempt to check-in this prep product to the released
   state.
   
   ACTUAL RESULTS:
   The check-in fails as follows :
