 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  The CICS Transaction Server Feature Pack for Dynamic Scripting
   V2.0 (5655-Y47) allows users to deploy PHP applications in CICS.
   Support is needed in CICS Explorer to create these applications.
   
   
    
   
   

LOCAL FIX

PROBLEM SUMMARY
 *  This APAR will add PHP support in CICS Explorer SDK V5.1.1,
   which allows users to create the sample PHP applications for
   CICS TS Feature Pack for Dynamic Scripting V2.0.
   
   
    
   
   

PROBLEM CONCLUSION
 *  The fix is to add the PHP support for CICS TS Feature Pack for
   Dynamic Scripting V2.0, which provides a new wizard to deploy
   the example PHP application and a new target platform definition
   to support the developing. This new support added will be
   documented in the CICS Explorer Help and the CICS Information
   Center. The program number for CICS TS Dynamic Scripting V2 is
   5655-Y47.
   This fix will be made available in Version 5.1.1.1 of
   the CICS Explorer.
   For installation instructions please see:
   Ordering maintenance for the IBM CICS Explorer
   http://www.ibm.com/support/docview.wss?uid=swg21380083 [http://www.ibm.com/support/docview.wss?uid=swg21380083]
