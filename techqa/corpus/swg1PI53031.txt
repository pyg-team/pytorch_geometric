APAR STATUS
 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  You are running multiple CROSSTABS commands on Mac OS X.  You
   specify Exact tests and the program crashes.
   If you specify the default Asymptotic method or Monte Carlo, the
   job completes.
   
   
    
   
   

LOCAL FIX
 *  This issue will be addressed in the next Fix Pack. Please
   refer to the Fix Pack schedule at
   http://www.ibm.com/support/docview.wss?uid=swg21599157 for
   information on upcoming Fix Packs.
