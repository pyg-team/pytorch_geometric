Title: IBM Security Bulletin: 10x Vulnerability in IBM Control Center Could Allow Potential XML External Entity (XXE) Injection - United States

Text:
 SECURITY BULLETIN

SUMMARY
 IBM Control Center has addressed the following vulnerability: A potential XML External Entity Injection (XXE) attack when processing XML data. A remote attacker could exploit this vulnerability to expose sensitive information or consume memory resources. 

VULNERABILITY DETAILS
CVEID: CVE-2017-1758 [http://cve.mitre.org/cgi-bin/cvename.cgi?name=CVE-2017-1758]
DESCRIPTION: IBM Control Center is vulnerable to a potential XML External Entity Injection (XXE) attack when processing XML data. A remote attacker could exploit this vulnerability to expose sensitive information or consume memory resources.
CVSS Base Score: 7.1
CVSS Temporal Score: See https://exchange.xforce.ibmcloud.com/vulnerabilities/135859 [https://exchange.xforce.ibmcloud.com/vulnerabilities/135859] for the current score
CVSS Environmental Score*: Undefined
CVSS Vector: (CVSS:3.0/AV:N/AC:L/PR:L/UI:N/S:U/C:H/I:N/A:L) 

AFFECTED PRODUCTS AND VERSIONS
IBM Control Center 6.0.0.0 through 6.0.0.2
IBM Control Center 6.1.0.0 through 6.1.0.2 iFix01 
IBM Control Center 6.1.1.0 base release
