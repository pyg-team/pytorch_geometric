ERROR DESCRIPTION

   integration is Huawei U2000 with SOAP/MTOSI Web Service NBI.

   * Collector discovery using Java MTOSI Collector against       *

   ****************************************************************
   * PROBLEM DESCRIPTION:                                         *
   * Java MTOSI collector failed to discover Huawei EMS via MTOSI *
   * SOAP xml NBI                                                 *

PROBLEM CONCLUSION

   file, missing parameters and logic flow in Java MTOSI collector
   code observed during a Customer's ITNM 4.2 deployment
