Title: IBM PK13031: NEW WARNING MESSAGE WHEN DATASOURCE IS CONFIGURED INCORRECTLY TO RUN ORACLE 10G HELPER. - United States

Text:
 SUBSCRIBE
You can track all active APARs for this component.



APAR STATUS
 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  Unpredictable Oracle behaviour if the daatSource is configure
   with the wrong Oracle dataStoreHelper
   
   
    
   
   

LOCAL FIX

PROBLEM SUMMARY
 *  ****************************************************************
   * USERS AFFECTED: All WebSphere Appplication Server users      *
   *                 using Oracle backend with Oracle 10g         *
   *                 driver.                                      *
   ****************************************************************
   * PROBLEM DESCRIPTION: Application Server could easily be      *
   *                      misconfigured with the wrong            *
   *                      OracledataStoreHelper when running      *
   *                      using Oracle 10g driver.                *
   ****************************************************************
   * RECOMMENDATION:                                              *
   ****************************************************************
   Oracle10gDataStoreHelper should be used when configuring
   Application Server dataSources when customers run with the
   Oracle 10g driver. Failure to do this will now result in a
   warning message.
   
   
    
   
   

PROBLEM CONCLUSION
 *  A new warning messag3e was added to let customers know
   of misconfiguratiom of the dataStoreHelper.
   
   The fix for this APAR is currently targeted for inclusion
   in fixpack 6.0.2.5.
   Please refer to the recommended updates page for delivery
   information:
   http://www.ibm.com/support/docview.wss?rs=180&uid=swg27004980 [http://www.ibm.com/support/docview.wss?rs=180&uid=swg27004980]
   
   
    
   
   

TEMPORARY FIX
