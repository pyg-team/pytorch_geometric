Title: IBM SSL Support for IBM Tivoli Identity Manager (ITIM) Adapters - United States

Text:
 TECHNOTE (FAQ)

QUESTION
 Which version of SSL is supported between ITIM and ITIM Adapters? 

ANSWER
All the IBM Tivoli Identity Manager (ITIM) agents/adapters for ITIM 4.6 will support TLS 1.0 as well as SSLv3 and below. As a result, unless explicitly told to use something else, they will negotiate a TLS session.
On the ITIM 4.6 (and later) server side, it is possible to constrain the negotiation to use a specific protocol. This is done by adding the $ITIM/data/enRole.properties parameter:

com.ibm.daml.jndi.DAMLContext.SSL_PROTOCOL=TLS

The default for this option is "TLS". 

Assuming you are using the default IBM JSSE
com.ibm.daml.jndi.DAMLContext.JSSE_PROVIDER

then other valid values would be:
