ERROR DESCRIPTION
 *  When trying to remove a Impact server on a Windows box
   nci_removeserver.bat  fails repeatedly returning:
   
   WASX7011E: Cannot find file
   "<%NCHOME%>\impact\bin\jython\remove.app.py <servername>"
   
   
    
   
   

LOCAL FIX
 *  %NCHOME%\impact\bin\nci_removeserver.bat is missing a
   double-quote from the end of a line.
   
   Need to have this instead:
   "%JAVA_HOME%\bin\java" %JAVA_OPTS %IMPACT_OPTS% -classpath
   "%C_PATH%" com.micromuse.response.client.cli.WsAdminInvoker %1
   "%WSADMIN_SCRIPT%" "%IMPACT_HOME%\bin\jython\remove-app.py" %1
