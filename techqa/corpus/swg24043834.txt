HOW CRITICAL IS THIS FIX?
There are multiple vulnerabilities in IBM Runtime Environment Java Technology Edition, Version 6, that is used by IBM Content Collector. 


This interim fix addresses the vulnerabilities CVE-2017-3252 (CVSS 5.8), CVE-2016-2183 (CVSS 3.1), CVE-2016-5548 (CVSS 6.5), CVE-2016-5549 (CVSS 6.5), CVE-2016-5547 (CVSS 5.3), and CVE-2017-3241 (CVSS 9). 

 Security Bulletin : IBM Runtime Environment Java Technology Edition, Version 6, CVE-2017-3252, CVE-2016-2183, CVE-2016-5548, CVE-2016-5549, CVE-2016-5547, CVE-2017-3241 [http://www-01.ibm.com/support/docview.wss?uid=swg22001460] 

This interim fix also addresses the following vulnerabilities in IBM WebSphere Application Server (Embedded) CVE-2016-0377, CVE-2016-1181, CVE-2016-1182, CVE-2015-0899, CVE-2016-9736, CVE-2016-8934, CVE-2017-1121, and CVE-2016-8919.
