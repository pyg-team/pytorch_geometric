Title: IBM JR52170: YOU CANNOT SHOW INSTANCE DETAILS FROM A FILTERED COGNOS REPORT IN THE REPORT VIEWER WIDGET - United States

APAR STATUS
 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  You cannot show instance details from a filtered Cognos report
   in the Report Viewer widget. Instead of seeing a filtered list
   of instances in the Instances widget, you see the entire list of
   instances for the monitoring context.
   
   PRODUCTS AFFECTED
   IBM Business Monitor
   
   
    
   
   

LOCAL FIX

PROBLEM SUMMARY
 *  No additional information is available.
   
   
    
   
   

PROBLEM CONCLUSION
 *  A fix is available for IBM Business Monitor V8.5.5.0 that allows
   you to use Cognos filters when populating the data in the
   Instances widget from a Cognos report.
   
   On Fix Central (http://www.ibm.com/support/fixcentral), search
   for JR52170:
   
   1. Select IBM Business Monitor from the product selector, the
   installed version to the fix pack level, and your platform, and
   then click Continue.
   2. Select APAR or SPR, enter JR52170, and click Continue.
   
   When you download fix packages, ensure that you also download
   the readme file for each fix. Review each readme file for
   additional installation instructions and information about the
   fix.
