Text:

CAUSE
The configuration of Solaris OSLC (i.e., Oracle Solaris Legacy Containers, also known as Zones) does not allow "df -k" command to return a valid output. 

ENVIRONMENT
CDD for Oracle 10.2.1 was installed in Solaris OSLC (i.e., Oracle Solaris Legacy Containers, also known as Zones)

DIAGNOSING THE PROBLEM
Execute the df-k (i.e., "df -k <path> | cat" in case the platform is Sun) to see whether or not a valid output is returned.
If the target storage area cannot be detected, it may indicate that there is an issue in Solaris OSLC (Zone) configuration.

Note:
CDC does support Solaris Zones, but its configuration may not allow "df -k" command to return a valid output. 

RESOLVING THE PROBLEM
Find a storage area where "df -k (i.e., df -k <path> | cat)" command returns a valid output.
