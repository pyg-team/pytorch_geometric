Text:
Probe; probes; MustGather; OMNIbus; 8.1 TECHNOTE (FAQ)

QUESTION
 Read first : On OMNIbus 8.1, what information are required for troubleshooting probe problems on UNIX? 

ANSWER
Gather Netcool/OMNIbus version and the installed software on OMNIbus 8.1:
1. Output of following commands: 

[/support/docview.wss?uid=swg21969660&amp;aid=1] [/support/docview.wss?uid=swg21969660&amp;aid=2]

* Non-root install 
/home/<username>/InstallationManager/eclipse/tools/imcl listInstalledPackages -features -long >> InstallManager_non_root_installed_software.txt 


* root install 
./imcl listInstalledPackages -features -long >> InstallManager_root_installed_software.txt 

2. Copy of following files: 

[/support/docview.wss?uid=swg21969660&amp;aid=3] [/support/docview.wss?uid=swg21969660&amp;aid=4]

* Non-root install 
/home/<username>/InstallationManager/eclipse/configuration/config.ini 


* root install 
InstallationManager/eclipse/configuration/config.ini 


Gather Probe Integrations information:


 * MustGather details for Probe Integrations on UNIX [http://www-01.ibm.com/support/docview.wss?uid=swg21642785]



Please note: use listIU.sh command to gather Netcool/OMNIbus version is for OMNIbus prior to 8.1. On 8.1, please use the first section in this technote (Gathering Netcool/OMNIbus version and the installed software on OMNIbus 8.1).
