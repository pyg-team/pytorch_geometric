ERROR DESCRIPTION
 *  When invoking a SQL statement which includes single quotation
   (For example: NULLIF(A.DATE,'0001-01-01') AS DATE) using the
   ADMIN_CMD() procedure, the execution will fail with the
   following error message:

   "0001" was found following "ULLIF(A.DATE,''". Expected tokens
   may include:  "<space>".  SQLSTATE=42601
   Data-Flows  Call ADMIN_CMD : SQL20397W: Errors occur during
   the calling process
   Data-Flows  Error messages dump: Call ADMIN_CMD: SQL20397W:
   Errors occur during the calling process
   Data-Flows  Error messages dump: SQL0104N: An unexpected token
   "0001" was found following "ULLIF(A.DATE,''".  Expected tokens
   may include:  "<space>".  SQLSTATE=42601
   Data-Flows  SQW03563E: Failure to execute Java class
   <class_name>.

   calling process
