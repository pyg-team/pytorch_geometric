APAR STATUS
 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  IBM Content Navigator fails to upload .rar file via
   drag'n'drop method.
   Browser affected: Chrome
   The error shown is:
   CIWEB2053
   The following item cannot be added to the repository:
   failed_item.
   Explanation
   This error can occur for one of the following reasons:
   You are trying to add a document that was moved or deleted.
   You are trying to add a document that was renamed.
   You are trying to add a document that is corrupted.
   You are trying to add a document that is open in another
   application.
   You are trying to add a folder to the repository.
   You do not have the appropriate permissions on the operating
   system to
   access the document.
   
   
    
   
   

LOCAL FIX
 *  Use a different browser for drag 'n' drop.
   
   
    
   
   

PROBLEM SUMMARY
 *  With Chrome browser, using drag and drop to add documents
   sometimes results in an error for some file types.  The error
   message is "CIWEB2053 The following item cannot be added to the
   repository".
   
   
    
   
   

PROBLEM CONCLUSION
 *  This has been fixed in IBM Content Navigator V3.0.1. With this
   fix, drag and drop to add documents using Chrome works for
   these file types.
