THIS DOCUMENT APPLIES ONLY TO THE FOLLOWING LANGUAGE VERSION(S):
 US English 

PROBLEM(ABSTRACT)
 Get a java.lang.NoClassDefFoundError during Jboss 6.x.0 install. 

CAUSE
This is caused due to incorrect java version.

RESOLVING THE PROBLEM
To resolve the problem you must use Java 1.5 or higher.


Do java -version and verify the level of JRE installed. 

If you do not have the latest JRE, download it from http://www.sun.com/java [http://www.sun.com/java] 

Install and verify your Java is at the latest level. 

Do java -version again (make sure your JAVA_HOME is set correctly pointing to the latest java dir.) 

Then re-run the JBoss installer again and this time you will be able to get past this error.
