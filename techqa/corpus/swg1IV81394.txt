ERROR DESCRIPTION
 *  The first time after clearing the browser cache, if a URL is
   entered to a page within the webtop context (e.g.
   https://myhost:16311/ibm/console/webtop/test/test.html) it
   causes an Error 500.  Refreshing the page provides the correct
   result of moving to the DASH login screen.
   
   
    
   
   

LOCAL FIX
 *  Refresh the page.
   Or use the DASH root context (/ibm/console) to login, then
   navigate to the webtop context.
