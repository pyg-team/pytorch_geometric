ERROR DESCRIPTION
 *  Problem Description :
   If the initial attempts to connect to the appliance fail
    subsequent requests can fail with SSL errors.
    The KD4.8....DataPower.operations.log will contain errors like
    KD4DP0004E The data collector could not retrieve a domain list
    for DataPower appliance: 172.21.6.172 using user ID: admin.
    Detailed error message: class javax.net.ssl.SSLException
    java.lang.ArrayIndexOutOfBoundsException: Array index out of
    range: 64

     L2 - IS

PROBLEM SUMMARY
 *  ****************************************************************
   * USERS AFFECTED: Users of the Data Power Data Collector on
   * all platforms.
   ****************************************************************
   * PROBLEM DESCRIPTION: If the initial attempts to connect to
   * the appliance fail subsequent requests can fail with SSL
   * errors. The KD4.8....DataPower.operations.log will contain
   * errors like:
   *
   * KD4DP0004E The data collector could not retrieve a domain
   * list for DataPower appliance: 172.21.6.172 using user ID:
   * admin. Detailed error message: class
   * javax.net.ssl.SSLException
   * java.lang.ArrayIndexOutOfBoundsException: Array index out of
   * range: 64
   *
   * Interim Fix 7 fixes this by initializing a new SSL connection
   * every interval as opposed to reusing the older connection
   * details.
   ****************************************************************
   * RECOMMENDATION: Apply IFix 7 when available.
   ****************************************************************
