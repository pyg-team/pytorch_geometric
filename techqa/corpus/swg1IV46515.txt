ERROR DESCRIPTION
 *  Sybase agent lose connection with Sybase ASE, Server_Status
   shows Unknown until restarting the agent.
   
   If trace_all is enabled, you will find the following information
   from <host>_oy_<sid>_col.out:
   YFR0360T (hhmmss) Rows collected for cursor HART2 for insert = 1
   PLO3095T (hhmmss) Moving '0.000000' from STATUS (8:1) to
   HOLDSTATUS2 (16:1)
   
   The correct log should be:
   YFR0360T (hhmmss) Rows collected for cursor HART2 for insert = 1
   PLO3095T (hhmmss) Moving '1.000000' from STATUS (8:1) to
   HOLDSTATUS2 (16:1)
   
   Detailed Recreation Procedure:
   1) Sybase server works well, but Sybase agent reports
   Sybase_Server_Summary.Server_Status as Unknown and cannot
   recover.
   2) The agent reports correct status after restarting.
