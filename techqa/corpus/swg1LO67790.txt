ERROR DESCRIPTION
 *  The community file upload box contains embedded style elements
   in HTML instead of CSS which prevents theming it. For example:
   
   <a role="treeitem"
   aria-controls="lconn_files_widget_FilePicker_0_content"
   href="javascript:;"
   style="display: block; white-space: nowrap; text-decoration:
   none; font-weight: bold; font-size: 1.1em; margin: 0px 1px;
   clear: both; position: relative; outline-color: rgb(0, 0, 0);
   background-color: rgb(50, 79, 118); color: rgb(255, 255, 255);
   padding-left: 15px;"
   title="Recent Files"
   tabindex="0"
   aria-selected="true">
   <span style="display: inline-block; padding-bottom: 3px;
   padding-top: 3px; padding-right: 5px; background-color: rgb(50,
   79, 118);">Recent Files</span></a>

   line of code by changing the color as you wish in the following

   F",foreground:"#000000",menuText:"#000000",secondaryText:"#84848
   4",selectedBackground:"#324f76",selectedMenuText:"#FFFFFF"}
