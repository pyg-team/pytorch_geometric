PROBLEM SUMMARY
 *  It is possible in Build Forge to start disabled steps.
   Steps to reproduce:
   1)  Create a project
   2)  Disable a step
   3)  Go to the Start tab in the Project Runs
   4)  Use the top check box to deselect the steps, and then
   reselect the steps.  Notice that the step disable are now
   enabled.  It does not appear to always start those steps,
   but occasionally it appears to start the step.
   Build Forge should make it impossible to start disabled
   steps from the Start menu.

 *  Fixed in version 7.0.1
