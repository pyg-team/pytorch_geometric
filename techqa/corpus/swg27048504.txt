ABSTRACT
 The following new feature and fixes are included in IBM Business Process Manager (BPM) V8.5.7 cumulative fix 2016.09.

With this update, you can 


 * Invoke REST services directly from your business processes 
 * Create custom integrations using new low code web-based service editors to discover Java and REST based services 
 * Deploy IBM BPM Advanced assets and manage the health of your processes using new self service facilities for IBM BPM on Cloud 
 * Deactivate users who are no longer in the user registry and clean up their group memberships


IBM BPM V8.5.7 Cumulative Fix 2016.09 also offers new supported environments: 
 * Microsoft Edge (for IBM Process Portal and responsive coaches) 
 * Oracle 12c multi-tenant feature 
 * Microsoft SQL Server 2016 
 * IBM DB2 11.1


For more information about supported configurations, see the IBM BPM support site: 
 * IBM BPM Advanced [http://ibm.biz/BPMAdvSysReqs] 
 * IBM BPM Standard [http://ibm.biz/BPMStdSysReqs] 
 * IBM BPM Express [http://ibm.biz/BPMExpSysReqs]
