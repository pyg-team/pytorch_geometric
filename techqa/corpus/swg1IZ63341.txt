APAR STATUS
 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  ITCAM for SOA 7.1.1 - SDMS config failure on ITM 6.2.2
   
   The error is with both GUI and console utility.
   Here is the error:
   
   SDMS Configuration Utility - Console Mode
   KD4SU1033E The version of Tivoli Enterprise Portal Server is not
   supported. Upgrade your Tivoli Enterprise Portal Server to
   version 6.2 or later and try the configuration again.
   
   
    
   
   

LOCAL FIX
 *  Please refer to the following technote for work-around
   procedure.
   
   http://www-01.ibm.com/support/docview.wss?uid=swg21407894 [http://www-01.ibm.com/support/docview.wss?uid=swg21407894]
   
   
    
   
   

PROBLEM SUMMARY
 *  SDMS config script fails to execute when ITM 6.2.2 is the base
   infrastructure.
   
   Error KD4SU1033E is displayed in both, GUI and console
   configuration utility.
   
   
    
   
   

PROBLEM CONCLUSION
 *  Config SDMS script infrastructure validation needs to
   accommodate for ITM 6.2.2 and above.
   
   Fix to be available in future ITCAM for SOA 7.1.1 Fix Pack 1
   
   
    
   
   

TEMPORARY FIX
 *  https://www-304.ibm.com/support/entdocview.wss?uid=swg1IZ63341 [https://www-304.ibm.com/support/entdocview.wss?uid=swg1IZ63341]
