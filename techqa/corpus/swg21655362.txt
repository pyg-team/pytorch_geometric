Title: IBM WTX node from Message Broker 6.x flow has two Basic properties. - United States

Text:
Toolkit; WTX node; 6; Basic properties TECHNOTE (TROUBLESHOOTING)

PROBLEM(ABSTRACT)
 A WebsphereÂ® Transformation Extender (WTX) node from Message Broker 6.x flow has two Basic properties. 

SYMPTOM
After migrating from Message Broker 6.x to Message Broker 7.x, a WTX node created from Message Broker 6.x flow has two Basic properties.


CAUSE
convertTxMsgFlow82 migration utility was not run against the Message Broker 6.x flow.

ENVIRONMENT
Windows Message Broker Toolkit

DIAGNOSING THE PROBLEM
Confirm that issue only occurs in Message Broker 6.x flow. Flows created from Message Broker 7.x does not show two Basic properties from the WTX node.

RESOLVING THE PROBLEM
Run convertTxMsgFlow82 (flowname.msgflow) as specified in the Integration Servers Release Notes.
