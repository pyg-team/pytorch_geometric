Title: IBM WebSphere Message Broker and IBM Integration Bus Security Bulletin: CVE-2014-0160 Heartbleed vulnerability - United States

Text:
 SECURITY BULLETIN

SUMMARY
 TLS Heartbleed security vulnerability does not apply to WebSphere Message Broker and IBM Integration Bus 

VULNERABILITY DETAILS
CVE ID: CVE-2014-0160 [http://cve.mitre.org/cgi-bin/cvename.cgi?name=CVE-2014-0160]
OpenSSL could allow a remote attacker to obtain sensitive information, caused by an error in the TLS/DTLS heartbeat functionality. An attacker could exploit this vulnerability to expose 64k of private memory and retrieve secret keys.

CVSS:
CVSS Base Score: 7.8
CVSS Temporal Score: See http://xforce.iss.net/xforce/xfdb/92322 [http://xforce.iss.net/xforce/xfdb/92322] for the current score
CVSS Environmental Score*: Undefined
CVSS Vector: (AV:N/AC:L/Au:N/C:C/I:N/A:N) 

AFFECTED PRODUCTS AND VERSIONS
This vulnerability does NOT affect the SSL that is used by products listed below. Note that the IBM Java JSSE does not use OpenSSL. 

IBM WebSphere Message Broker V7.0 

IBM WebSphere Message Broker for z/OS V7.0 

IBM WebSphere Message Broker V8.0 

IBM WebSphere Message Broker for z/OS V8.0 

IBM Integration Bus V9.0 

IBM Integration Bus for z/OS V9.0

REMEDIATION/FIXES
No action required.

WORKAROUNDS AND MITIGATIONS
None
