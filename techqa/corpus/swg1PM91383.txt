APAR STATUS
 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  Fails to upgrade WebSphere Application Server Liberty Profile
   Express edition V8.5.5.0 to Base or Network Deployment edition
   if IBM WebSphere SDK Java Technology Edition Version 6.0 for
   Liberty Profile or IBM WebSphere SDK Java Technology Edition
   Version 7.0 for Liberty Profile is also installed.
   Installation Manager logs an error message, "Package IBM
   WebSphere SDK Java Technology Edition Version x.x  for Liberty
   x.x.x.x can not be installed in the selected package group."
   
   
    
   
   

LOCAL FIX

PROBLEM SUMMARY
 *  ****************************************************************
   * USERS AFFECTED:  All users of IBM WebSphere Application      *
   *                  Server Liberty Profile Express Edition      *
   ****************************************************************
   * PROBLEM DESCRIPTION: Unable to upgrade Liberty Profile       *
   *                      Express Edition to Base or Network      *
   *                      Deployment editions on IBM WebSphere    *
   *                      SDK                                     *
   ****************************************************************
   * RECOMMENDATION:                                              *
   ****************************************************************
   When you try to upgrade a WebSphere Application Server Liberty
   Profile Express edition installation that also has IBM
   WebSphere SDK Java Technology Edition Version 6.0 for Liberty
   Profile or IBM WebSphere SDK Java Technology Edition Version
   7.0 for Liberty Profile installed to a Base or Network
   Deployment edition, Installation Manager fails with an error
   message, "Package IBM WebSphere SDK Java Technology Edition
   Version x.x  for Liberty x.x.x.x can not be installed in the
   selected package group."
   
   
    
   
   

PROBLEM CONCLUSION
 *  The problem will be fixed in 8.5.5.1 and you will be able to
   first update your WebSphere Application Server Liberty Profile
   Express installation to 8.5.5.1 and then upgrade to Base or
   Network Deployment edittion.
   
   The fix for this APAR is currently targeted for inclusion in
   fix pack 8.5.5.1.  Please refer to the Recommended Updates
   page for delivery information:
   http://www.ibm.com/support/docview.wss?rs=180&uid=swg27004980 [http://www.ibm.com/support/docview.wss?rs=180&uid=swg27004980]
