Title: IBM Security Bulletin: AppScan Enterprise can be affected by multiple vulnerabilities (CVE-2014-6136, CVE-2014-8918) - United States

Text:
PSIRT SECURITY BULLETIN

SUMMARY
 Multiple vulnerabilities such as encryption not enforced and incorrect handling of SSL exist in IBM Security AppScan Enterprise Glassbox component. 

VULNERABILITY DETAILS

CVEID: CVE-2014-6136 [http://cve.mitre.org/cgi-bin/cvename.cgi?name=CVE-2014-6136]

DESCRIPTION: 
IBM Security AppScan Enterprise could allow a remote attacker to obtain sensitive information due to encryption not being enforced. An attacker could exploit this vulnerability to obtain cleartext data.

CVSS Base Score: 2.6
CVSS Temporal Score: See https://exchange.xforce.ibmcloud.com/vulnerabilities/96816 [https://exchange.xforce.ibmcloud.com/vulnerabilities/96816] for the current score
CVSS Environmental Score*: Undefined
CVSS Vector: (AV:N/AC:H/Au:N/C:P/I:N/A:N)



CVEID: CVE-2014-8918 [http://cve.mitre.org/cgi-bin/cvename.cgi?name=CVE-2014-8918]

DESCRIPTION: 

IBM Security AppScan Enterprise could allow a remote attacker to bypass security restrictions, caused by the improper validation of SSL certificates. By persuading a victim to visit a Web site containing a specially-crafted certificate, a remote attacker could exploit this vulnerability using man-in-the-middle techniques to cause the victim to impersonate trusted servers

CVSS Base Score: 4.0
CVSS Temporal Score: See https://exchange.xforce.ibmcloud.com/vulnerabilities/99304 [https://exchange.xforce.ibmcloud.com/vulnerabilities/99304] for the current score
CVSS Environmental Score*: Undefined
CVSS Vector: (AV:N/AC:H/Au:N/C:P/I:P/A:N)

AFFECTED PRODUCTS AND VERSIONS
 

 * IBM Security AppScan Enterprise – 9.0.1 
 * IBM Security AppScan Enterprise – 9.0 
 * IBM Security AppScan Enterprise – 8.8 
 * IBM Security AppScan Enterprise – 8.7 
 * IBM Security AppScan Enterprise – 8.6
