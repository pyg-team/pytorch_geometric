APAR STATUS
 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  The workload console Service 60 on the PureSystems Manager
   restarts spontaneously due to an LDAP connectivity issue. As a
   result the user can not logon and gets a "Service unavailable"
   message.
   
   Errors that are seen are:
   CWZIP3345W Workload console health check failed, the workload
     console may not be healthy
   WZIP9604W: Service60 failed. The high availability component
     will attempt to recover the service
   CWZIP4665W The connection to LDAP has failed.
   
   
    
   
   

LOCAL FIX

PROBLEM SUMMARY
 *  The PureSystems Manager code that monitors services was
   attempting to recover LDAP connectivity by repeatedly restarting
   the service60 service.
   
   
    
   
   

PROBLEM CONCLUSION
 *  A code fix is being provided in PureApplication System
   version 2.2.0.0.
