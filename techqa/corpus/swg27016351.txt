Title: IBM System requirements for IBM Optim Performance Manager for DB2 for Linux, UNIX, and Windows Version 4.1 - United States

Text:
IBM Optim Performance Manager for DB2 for Linux; UNIX; and Windows Version 4.1; hardware requirements; software requirements; system requirements PRODUCT DOCUMENTATION

ABSTRACT
 Before you install IBM® Optim™ Performance Manager for DB2® for Linux®, UNIX®, and Windows®, ensure that your system has the following requirements for hardware and software. 

CONTENT
Note: The term UNIX includes AIX®, HP-UX, and Solaris unless otherwise indicated.

 * Requirements for the Web interface
 * 
 * 
 * Display unit requirements
 *  Recommended screen resolution: 1280 x 1024. The Web interface is optimized for this resolution. Supported resolution: 1024 x 768. Although this resolution is not the optimal 1280 x 1024 resolution, the 1024 x 768 resolution will also work so that you can see the contents of the dashboards. Resolutions lower than 1024 x 768 are not supported.
