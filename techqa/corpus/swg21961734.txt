Title: IBM Security Bulletin: Websphere Message Broker and IBM Integration Bus are affected by access control  vulnerability (CVE-2015-2018) - United States

Text:
WMB IIB SECURITY BULLETIN

SUMMARY
 Websphere Message Broker and IBM Integration Bus are affected by access control vulnerability. 

VULNERABILITY DETAILS
CVEID: CVE-2015-2018 [http://cve.mitre.org/cgi-bin/cvename.cgi?name=CVE-2015-2018]
DESCRIPTION: IBM Integration Bus can give incorrect results when using multiple security profiles, which could allow an authenticated user to obtain sensitive information about other users.
CVSS Base Score: 3.5
CVSS Temporal Score: See http://xforce.iss.net/xforce/xfdb/103993 [http://xforce.iss.net/xforce/xfdb/103993] for the current score
CVSS Environmental Score*: Undefined
CVSS Vector: (AV:N/AC:M/Au:S/C:P/I:N/A:N) 

AFFECTED PRODUCTS AND VERSIONS
 IBM Integration Bus V10 

 IBM Integration Bus V9 

 WebSphere Message Broker V8 

 WebSphere Message Broker V7
