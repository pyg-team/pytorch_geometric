Title: IBM Deploy Case Manager solution failed with the error code FNRAM9041 - United States

Text:
 TECHNOTE (TROUBLESHOOTING)

PROBLEM(ABSTRACT)
 When deploy a case manager solution in the production environment but it is not getting deployed. The log file shows
the following error:
The solution deployment failed. View the deployment log to identify the problem. Error code FNRAM9041.



SYMPTOM
From WebSphere system_out.log, we found below error message: 

[5/5/17 7:38:48:828 IST] 0000009a SystemOut O
CIWEB.ICMAdminClientPlugin Error: [p8admin(unknown) @ 0:0:0:0:0:0:0:1]
[REQUEST 57]
com.ibm.ecm.icm.accm.services.DeploySolutionService.execute() service
response =
{"success":false,"failureMessages":[{"adminResponse":null,"ACCMtype":"si
mple-message","explanation":null,"id":"","userResponse":null,"text":"For
input string: \"0:0:0:0:0:0:1:9081\""}],"ACCMtype":"service-response"}


CAUSE
The host IP address was incorrectly set as 0:0:0:0:0:0:1, and the client is trying to connect a non-existing port 0:0:0:0:0:0:1:9081.


ENVIRONMENT
IBM DB2 10.5.4
IBM Content platform Engine 5.2.1.4
IBM content Navigator 2.0.3.7
IBM Case Manager 5.2.1.4



RESOLVING THE PROBLEM
Remove the check box selection for IPv6 in 

Windows -> Network Connection Status -> Properties.
