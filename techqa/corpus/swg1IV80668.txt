APAR STATUS
 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  After an Active Directory password change on a different
   machine, each time the user unlocks the workstation with
   RFID-only, he is prompted to enter his AD password.
   If the user logs-off and logs-on to AccessAgent, the problem
   stops occurring.
   
   
    
   
   

LOCAL FIX
 *  logout/ login
   or wait until periodic sync
   or force sync
   (customer does not accept these workarounds)
   
   
    
   
   

PROBLEM SUMMARY
 *  With RFID-only logon enabled, the user is prompted to enter the
   Active Directory password each time the workstation is
   unlocked.
