PROBLEM(ABSTRACT)
 In order to begin diagnosing any issue on ITDS version 5.2, the first step is to identify the version, including fixpack level of the ITDS environment.

Gathering version information on AIX 
Collect the output of the following commands: 

lslpp -l | grep ldap 
lslpp -l | grep db2 
lslpp -l | grep gsk 
ls -l /usr/ldap/bin 
ibmslapd -V 

If the Web Administration Tool is installed and configured please collect the output of: 
ls -l /usr/ldap/idstools/IDSWebApp.war 

Gathering version information on HP 
Collect the output of the following commands: 

swlist | grep ldap 
swlist | grep -i db2 
swlist | grep -i gsk 
ls -l /usr/IBMldap/bin 
ibmslapd -V 

If the Web Administration Tool is installed and configured please collect the output of: 
ls -l /usr/ldap/idstools/IDSWebApp.war 

Gathering version information on Linux 
Collect the output of the following commands: 

rpm –qa | grep ldap 
rpm –qa | grep db2 
rpm –qa | grep gsk 
ls –l /usr/ldap/bin 
ibmslapd -V 

If the Web Administration Tool is installed and configured please collect the output of: 
ls -l /usr/ldap/idstools/IDSWebApp.war 

Gathering version information on Solaris 
Collect the output of the following commands: 

pkginfo | grep –I ldap 
pkginfo | grep –I db2 
pkginfo | grep –i gsk 
ls –l /opt/IBMldapc/bin 
ibmslapd –V 

If the Web Administration Tool is installed and configured please collect the output of: 
ls -l /opt/IBMldapc/idstools/IDSWebApp.war
