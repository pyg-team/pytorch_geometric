Title: IBM Compatible options between device mode and SiteProtector mode - United States

Text:
Strict; compatible; connect; register; communicate TECHNOTE (FAQ)

QUESTION
 What are the compatible options between the mode the device is running in and the mode SiteProtector is running (Strict versus Compatible)? 

ANSWER
The following table shows the different device and SiteProtector modes and their interoperability with being able to connect.

Notes: 

 * Previous designates a device that is SP 800-131A incapable. 
 * Compatible designates a device running Compatible mode. 
 * Strict designates a device running Strict mode.

Device Mode SiteProtector mode Can it connect? Previous Compatible Yes Previous Strict No. The Previous device does not accept SiteProtector's Strict SHA-2 certificate nor communicate over TLSv1.2. Compatible Compatible Yes Compatible Strict Yes Strict Compatible No. Strict device will not accept SiteProtector's compatible SHA-1 certificate. Strict Strict Yes
