APAR STATUS
 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  When a processing policy receives an HTTP 412 error code, it
   returns no content even if the message contains content. This is
    incorrect. The HTTP 412 error code can have a message body
   according to RFC 2616.
   
   
    
   
   

LOCAL FIX

PROBLEM SUMMARY
 *  Affected is the use of a DataPower service with a backend
   returning the HTTP 412 error code. Content, if it exists, is
   incorrectly removed.
   
   
    
   
   

PROBLEM CONCLUSION
 *  Fix is available in 6.0.0.4 and 6.0.1.1.
