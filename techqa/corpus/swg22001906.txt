Title: IBM Use white color as a default transparent fill color in IBM Daeja ViewONE Virtual viewer - United States

Text:
 TECHNOTE (FAQ)

QUESTION
 How can I use a white color as a default ransparent fill color in IBM Daeja ViewONE Virtual viewer? 

CAUSE
IBM Daeja ViewONE Virtual viewer and IBM Content Manager client viewer use different fill colors for transparent annotations. IBM Daeja ViewONE Virtual viewer uses a yellow color as a fill color whereas IBM Content Manager client viewer uses a white color as a fill color.

ANSWER
Back up and edit the following file to set ANNOTATION_ENGINE2_useWhiteDefaultColor, a new property added in IBM Content Navigator 3.0.0 Limited Availability Fix 003, to true. 

navigator.war/WEB-INF/cmbannotationengine.properties

ANNOTATION_ENGINE2_useWhiteDefaultColor behaviors:

1. If ANNOTATION_ENGINE2_useWhiteDefaultColor is set to false, a default value, transparent annotations with a yellow fill color will be transparent in IBM Daeja ViewONE Virtual viewer but semi-transparent in IBM Content Manager client viewer. Transparent annotations with a white fill color will be semi-transparent in IBM Daeja ViewONE Virtual viewer but transparent in IBM Content Manager client viewer. A yellow color will be used as a default fill color.

2. if ANNOTATION_ENGINE2_useWhiteDefaultColor is set to true, transparent annotations with a yellow fill color will be semi-transparent and transparent annotations with a white fill color will be transparent in IBM Daeja ViewONE Virtual viewer. A white color will be used as a default fill color. Existing transparent annotations will be semi-transparent with a yellow fill color and transparent annotations that were semi-transparent will be transparent with a white fill color.
