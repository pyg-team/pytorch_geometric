Title: IBM Security Bulletin: Vulnerabilities in OpenSSH affect IBM MQ Appliance (CVE-2016-0777) - United States

Text:
 SECURITY BULLETIN

SUMMARY
 An information leak flaw and buffer overflow flaw in the way the OpenSSH client roaming feature was implemented affects IBM MQ Appliance. 

VULNERABILITY DETAILS
CVEID: CVE-2016-0777 [http://cve.mitre.org/cgi-bin/cvename.cgi?name=CVE-2016-0777]
DESCRIPTION: OpenSSH could allow a remote attacker to obtain sensitive information, caused by a client information leak from using the roaming connection feature. By persuading a victim to connect to a malicious server, an attacker could exploit this vulnerability to retrieve private cryptographic keys or other sensitive information.
CVSS Base Score: 6.5
CVSS Temporal Score: See https://exchange.xforce.ibmcloud.com/vulnerabilities/109635 [https://exchange.xforce.ibmcloud.com/vulnerabilities/109635] for the current score
CVSS Environmental Score*: Undefined
CVSS Vector: (CVSS:3.0/AV:N/AC:L/PR:N/UI:R/S:U/C:H/I:N/A:N)

AFFECTED PRODUCTS AND VERSIONS
IBM MQ Appliance M2000

REMEDIATION/FIXES
Apply the ifix for APAR IT13223 [http://www.ibm.com/support/fixcentral/swg/selectFix?product=ibm/WebSphere/IBM%20MQ%20Appliance%20M2000&fixids=8.0.0.4-IBM-MQ-Appliance-M2000-IT13223&function=fixId&parent=ibm/WebSphere].

WORKAROUNDS AND MITIGATIONS
 

In most cases the IBM MQ Appliance is not affected by this issue. The exception 
is where you have a high availability configuration where the two appliances are 
not directly connected (that is, they are remotely situated and connected by means of
