Title: IBM Security Bulletin: Queue manager repository manager integrity/abend caused by multiple transmit queue records (CVE-2015-0189) - United States

Text:
 SECURITY BULLETIN

SUMMARY
 A memory overwrite in the cluster repository manager can be triggered by a malicious user with IBM WebSphere MQ administrator privileges, leading to loss of data integrity and denial of service through abnormal termination of the cluster repository manager. 

VULNERABILITY DETAILS
CVEID: CVE-2015-0189 [http://cve.mitre.org/cgi-bin/cvename.cgi?name=CVE-2015-0189]
DESCRIPTION: A memory overwrite in the cluster repository manager can be triggered by a malicious user with IBM WebSphere MQ administrator privileges, leading to loss of data integrity and denial of service through abnormal termination of the cluster repository manager.
CVSS Base Score: 1.5
CVSS Temporal Score: See http://xforce.iss.net/xforce/xfdb/101003 [http://xforce.iss.net/xforce/xfdb/101003] for the current score
CVSS Environmental Score*: Undefined
CVSS Vector: (AV:L/AC:M/Au:S/C:N/I:N/A:P) 

AFFECTED PRODUCTS AND VERSIONS
IBM WebSphere MQ v8.0.0.1 and earlier 

IBM WebSphere MQ v7.5.0.4 and earlier

REMEDIATION/FIXES
APAR IT05869 fixes the potential for a memory overwrite. 

 

IBM WebSphere MQ v8.0 

 * Apply fix pack 8.0.0.2 [http://www-01.ibm.com/support/docview.wss?uid=swg27043086#8002] or later


IBM WebSphere MQ v7.5  * Apply fix pack 7.5.0.5 [http://www-01.ibm.com/support/docview.wss?uid=swg27038184#7505] or later

WORKAROUNDS AND MITIGATIONS
None.
