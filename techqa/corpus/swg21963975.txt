Title: IBM Rational Business Developer V9 Passport Advantage e-assembly part numbers for Host Access Transformation Services/Webfacing V9.0 - United States

Text:
 TECHNOTE (TROUBLESHOOTING)

PROBLEM(ABSTRACT)
 What eAssembly part numbers should be downloaded from Passport Advantage to install WebFacing as a component of Host Access Transformation Services V9, Rational Business Developer for Websphere V9, Websphere Application Server V8.5. 

CAUSE
Entitled clients can download Rational Business Developer for HATS/Webfacing from Passport Advantage. 

However, when you review media pack search results, it may be difficult to recognize which eAssemblies represent the core product Rational Development Environments for HATS/WebFacing and which additional eAssembly are required or optional.



RESOLVING THE PROBLEM
HATS/WebFacing v9.0 is a plugin to Rational Business Developer or Rational Developer for i and multiple products need to be downloaded/installed in order install HATS/WebFacing: 


Download the version you purchased:
IBM Rational HATS V9.0 for 5250 on Multi platform Multilingual eAssembly (CRNR6ML) 
IBM Rational HATS V9.0 for 5250 on IBM i Multilingual eAssembly (CRNR7ML) 

Download the License file:
IBM Rational HATS V9.0 Runtime Enablement Multiplatform Multilingual (CIPP3ML ) License file. 

If you purchased IBM Rational Business Developer (RBD) download these files:
Quick Start Guide for IBM Rational Business Developer V9.0 Multilingual READ ME FIRST(CIMI1ML) 
IBM Rational Business Developer V9.0 Multiplatform Multilingual Part 1 (CIMI2ML )
IBM Rational Business Developer V9.0 Multiplatform Multilingual Part 2 (CIMI3ML )
IBM Rational Business Developer V9.0 Multiplatform Multilingual Part 3 (CIMI4ML ) 
IBM Rational Business Developer V9.0 Activation Kit Multilingual Multiplatform (CN1TAML ) 

If you purchased IBM Rational Developer for i ( RDi) download these files:
IBM Rational Developer for i RPG & COBOL Tools V9.1 Core Multilingual Multiplatform eAssembly (CRS66ML) 
IBM Rational Developer for i V9.1 Core Product Install 1 Multiplatform Multilingual (CIZD1ML )
IBM Rational Developer for i V9.1 Core Product Install 2 Multiplatform Multilingual (CIZD2ML )
IBM Rational Developer for i V9.1 RPG & COBOL Tools Activation Kit Multiplatform Multilingual (CIZD3ML ) License

You will need to download all three of the Websphere server files:
IBM WebSphere Application Server V8.5.5 (1 of 3) for Multiplatform Multilingual (CIK1QML )
IBM WebSphere Application Server V8.5.5 (2 of 3) for Multiplatform Multilingual (CIK1RML )
IBM WebSphere Application Server V8.5.5 (3 of 3) for Multiplatform Multilingual (CIK1SML )

Passport Advantage will only show you the files you purchased. If you do not see any of the files above you should contact Passport Advantage support 
http://www-01.ibm.com/software/passportadvantage/pa_contacts.html [http://www-01.ibm.com/software/passportadvantage/pa_contacts.html]

To download Rational Application Developer for HATS V9 from Passport Advantage, perform the following steps: 

1. Logon to Passport Advantage

2. Select Software Downloads and Media Access

3. Select the Download Finder

4. Select Find by description or part number 

5. Select the Product Description and All radio buttons and enter Rational Application Developer for WebSphere in the description field, or enter the part numbers from above.

6. Select Continue.

7. Expand the eAssemblies and you will see your entitled software
