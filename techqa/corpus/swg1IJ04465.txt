ERROR DESCRIPTION
 *  WEBGUI version: WebGUI 8.1 +latest fixpack 11.
   DASH 3.1.3.0
   
   Customer is using Event Viewer. When he tries to add a Journal
   entry the text does not get formatted (wrapped around)
   correctlly. Event list field values some time overlapping when
   there is lot of text.
   
   Steps to reproduce the issue:
   
   
   To reproduce the issue:
   0. Login into DASH
   
   1. Open WEBGUI Event viewer.
   
   Click on Incident on the menu bar--> Event Viewer
   
   2. In Event Viewer double click on any event (row).
   
   3. A new window will popup. Click on Journal Tab in that window.
   
   4. In the "New journal entry:" text box at the bottom enter some
   text 3-4 lines. Click on Apply.
   
   5. Enter another long entry (3-4) lines. Click apply.
   
   6. You may have to enter some small entries ( a few words).
   
   7. You will see the text will overlap on the previous entry.
   
   We are able to reproduce this issue on both IE11 and Firefox on
   multiple computers with different browser settings.
   
   It was found the this issue is only reproduciable while using
   "IBM Design" Theme.
