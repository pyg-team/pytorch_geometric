APAR STATUS
 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  Atlas does not allow user to resize the pop-up window when
   adding users to the People in Scope page, Hold Notice
   Recipients page or Release Notice Recipients page.   As a
   result, a low screen resolution forces users to scroll to view
   the entire contents of the pop-up window.
   
   
    
   
   

LOCAL FIX
 *  N/A
   
   
    
   
   

PROBLEM SUMMARY
 *  Atlas does not allow user to resize the pop-up window when
   adding users to the People in Scope page, Hold Notice Recipients
    page or Release Notice Recipients page.   As a result, a low
   screen resolution forces users to scroll to view the entire
   contents of the pop-up window.
   
   
    
   
   

PROBLEM CONCLUSION
 *  This issue is now fixed by adjusting the default dialog size as
   well as making them resizeable. The fix is available from fix
   central in Atlas 6.0.3.2 iFix 2.
