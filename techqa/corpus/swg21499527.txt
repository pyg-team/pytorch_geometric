Title: IBM Differences between using IBM Installation Manager Version 1.4.3, Version 1.4.4, and Version 1.5 to manage product installations - United States

Text:
WASV8RNotes; V8RNotes TECHNOTE (TROUBLESHOOTING)

PROBLEM(ABSTRACT)
 You might want to update your IBM Installation Manager from the version that is packaged with WebSphere Application Server Version 8.0, Installation Manager Version 1.4.3, to Version 1.4.4 or Version 1.5. 

RESOLVING THE PROBLEM
Starting with IBM Installation Manager Version 1.4.4, changes were made to allow the following: 

 * The imcl installAll command is deprecated.
   
   
 * The -silent command option is deprecated.
   
   
 * The -mode wizard command option is deprecated.
   
   
 * You have the option to install all fixes, not install fixes, or install only recommended fixes during silent or command-line installation of an offering by using the -installFixes option.
   
   
 * You can start Installation Manager in another language by specifying a language other than the default English setting.
   
   
 * You can use the listInstallationDirectories command to determine what shared directory is associated with a given instance of Installation Manager.
   
   
 * If you try to remove a feature that is required by an installed product, you are blocked if there are still other features of the product installed.
   
   
 * Various issues with Installation Manager command-line operations are resolved.
   
   
 * You do not receive an unwarranted out-of-memory error message during product installation. 
   


Starting with IBM Installation Manager Version 1.5, changes were made to allow the following: 

 * You can roll back a package from the command line with the imcl rollback command.
   
   
 * You can use the imcl modify command to modify packages from the command line.
   
   
 * You do not receive unwarranted CRIMCCE24D411E errors when you use the imcl updateAll command to update multiple installed WebSphere Application Server Version 8.0 offerings.
   


If you are using Installation Manager 1.4.3 or earlier, upgrade to Version 1.5 or later. You can download the latest version of Installation Manager from the IBM Installation Manager download website [http://www.ibm.com/support/entry/portal/All_download_links/Software/Rational/IBM_Installation_Manager].
