Title: IBM Security Bulletin: IBM FileNet System Monitor/IBM Enterprise Content Management System Monitor is potentially affected by multiple security vulnerabilities in IBM® SDK, Java™ Technology Edition - United States

Text:
 SECURITY BULLETIN

SUMMARY
 Multiple security vulnerabilities exist in IBM® SDK, Java™ Technology Edition that is shipped with IBM FileNet System Monitor/IBM Enterprise Content Management System Monitor product. 

VULNERABILITY DETAILS

The IBM FileNet System Monitor/IBM Enterprise Content Management System Monitor product ships with IBM® SDK, Java™ Technology Edition. The IBM® SDK, Java™ Technology Edition is based on the Oracle version of the SDK/JRE. In January 2014, Oracle has released critical patch updates (CPU) that contain security vulnerability fixes. These issues are present in IBM® SDK, Java™ Technology Edition and potentially affect the System Monitor product.

CVEID: CVE-2014-0416 [http://cve.mitre.org/cgi-bin/cvename.cgi?name=CVE-2014-0416]
DESCRIPTION: Unspecified vulnerability allows remote attackers to affect integrity via vectors related to JAAS.
CVSS Base Score: 5
CVSS Temporal Score: See https://exchange.xforce.ibmcloud.com/vulnerabilities/90349 [https://exchange.xforce.ibmcloud.com/vulnerabilities/90349] for the current score
CVSS Environmental Score*: Undefined
CVSS Vector: (AV:N/AC:L/Au:N/C:N/I:P/A:N) 

CVEID: CVE-2014-0368 [http://cve.mitre.org/cgi-bin/cvename.cgi?name=CVE-2014-0368]
DESCRIPTION: Unspecified vulnerability allows remote attackers to affect confidentiality via unknown vectors related to Networking.
CVSS Base Score: 5
CVSS Temporal Score: See https://exchange.xforce.ibmcloud.com/vulnerabilities/90351 [https://exchange.xforce.ibmcloud.com/vulnerabilities/90351] for the current score
CVSS Environmental Score*: Undefined
CVSS Vector: (AV:N/AC:L/Au:N/C:P/I:N/A:N) 

CVEID: CVE-2014-0411 [http://cve.mitre.org/cgi-bin/cvename.cgi?name=CVE-2014-0411]
DESCRIPTION: Unspecified vulnerability allows remote attackers to affect confidentiality and integrity via vectors related to JSSE.
CVSS Base Score: 4
CVSS Temporal Score: See https://exchange.xforce.ibmcloud.com/vulnerabilities/90357 [https://exchange.xforce.ibmcloud.com/vulnerabilities/90357] for the current score
CVSS Environmental Score*: Undefined
CVSS Vector: (AV:N/AC:H/Au:N/C:P/I:P/A:N)

AFFECTED PRODUCTS AND VERSIONS
The following versions of the System Monitor product are potentially affected: 


IBM FileNet System Monitor v4.5.0
IBM Enterprise Content Management System Monitor v5.1.0

REMEDIATION/FIXES
For IBM FileNet System Monitor v4.5.0:

Upgrade to the platform specific version of IBM® SDK, Java™ Technology Edition that is available in IBM FileNet System Monitor v4.5.0.3 Interim Fix 1.

For IBM Enterprise Content Management System Monitor v5.1.0:

Upgrade to the platform specific version of IBM® SDK, Java™ Technology Edition that is available in IBM Enterprise Content Management System Monitor v5.1.0 Fix Pack 3.

If you need further assistance, please contact IBM Support.

WORKAROUNDS AND MITIGATIONS
None known, apply fixes
