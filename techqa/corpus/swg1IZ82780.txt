APAR STATUS
 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  Deleting a KPI -created from the Person Group Application
   -Generates the following error message
   
   BMXAA4214E - An unknown error has occurred. Please contact your
   system administrator for assistance.
   
   
   Step by Step provided by Support to reproduce this problem on
   maxdemo BS 7117 + 7118
   
   
   Initial Application: "Person Group"
   
   1. From the Person Groups List Tab - Create the KPI
   2. Give the New KPI an ID and Description.
   3. Enter the following sql into the where clause "where
   persongroup. persongroup in (select distinct g.persongroup from
   PERSONGROUPTEAM g where g. resppartygroup = 'BOOTHP_SYS')"
   
   4. Enter values into the fields : Target, "Caution At"  and
   "Alert At"
   
   5. select the Ok button
   
   Go to the KPI Manager and select the newly created KPI
   Select the "Select Action"->"Delete KPI" Option the following
   error message is displayed
   
   BMXAA4214E - An unknown error has occurred. Please contact your
   system administrator for assistance.
   
   Accept the Message  and the KPI details blank as per attachment
   navigating to the KPI  list  tab and look for the KPI  -the KPI
   has been deleted
   
   This error message did not occur on BS 7116 but got introduced
   in 7117.
