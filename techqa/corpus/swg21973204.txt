Title: IBM Security Bulletin: IBM Netezza Host Management is vulnerable to a published glibc vulnerability (CVE-2013-7423) - United States

Text:
 SECURITY BULLETIN

SUMMARY
 GNU glibc could allow a local attacker to obtain sensitive information, caused by the writing of DNS queries to random file descriptors under high load by the getaddrinfo() function. An attacker could exploit this vulnerability to obtain sensitive information. 

VULNERABILITY DETAILS
CVEID: CVE-2013-7423 [http://cve.mitre.org/cgi-bin/cvename.cgi?name=CVE-2013-7423]
DESCRIPTION: GNU glibc could allow a local attacker to obtain sensitive information, caused by the writing of DNS queries to random file descriptors under high load by the getaddrinfo() function. An attacker could exploit this vulnerability to obtain sensitive information.
CVSS Base Score: 1.2
CVSS Temporal Score: See https://exchange.xforce.ibmcloud.com/vulnerabilities/100647 [https://exchange.xforce.ibmcloud.com/vulnerabilities/100647] for the current score
CVSS Environmental Score*: Undefined
CVSS Vector: (AV:L/AC:H/Au:N/C:P/I:N/A:N)

AFFECTED PRODUCTS AND VERSIONS
IBM Netezza Host Management 5.3.5.0 and prior
