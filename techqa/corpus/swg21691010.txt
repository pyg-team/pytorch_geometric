dmmarkexternalunloadstart; external refresh TECHNOTE (FAQ)

QUESTION
 We need to do external refresh for all tables(e.g. 1000+) in a subscription.
We need use dmmarkexternalunloadstart and dmmarkexternalunloadend.
dmmarkexternalunloadstart [-I <INSTANCE_NAME>] [-L <locale>] -s <SUBSCRIPTION_NAME ...> -t <table>
dmmarkexternalunloadend [-I <INSTANCE_NAME>] [-L <locale>] -s <SUBSCRIPTION_NAME ...> -t <table>

Is there any way that we do not need to list all tables names in the dmmarkexternalunloadstart command ? 

ANSWER
If the source is CDC 10.2.0, the -t parameter is the mandatory parameter. We can not run the command without -t option. We have to list every table name in the command.
