Title: IBM Warning messages are displayed in the log when creating an instance on xLinux - United States

Text:
 TECHNOTE (TROUBLESHOOTING)

PROBLEM(ABSTRACT)
 When creating a WebSphere Commerce instance, the following warning messages are displayed in the SystemErr.log:

Mar 22, 2006 2:11:35 PM java.util.prefs.FileSystemPreferences$3 run
WARNING: Could not create system preferences directory. System preferences are unusable.
Mar 22, 2006 2:11:49 PM java.util.prefs.FileSystemPreferences checkLockFile0ErrorCode
WARNING: Could not lock System prefs. Unix error code 273859672.
Mar 22, 2006 2:12:19 PM java.util.prefs.FileSystemPreferences checkLockFile0ErrorCode
WARNING: Could not lock System prefs. Unix error code 268911000.
Mar 22, 2006 2:12:49 PM java.util.prefs.FileSystemPreferences checkLockFile0ErrorCode
WARNING: Could not lock System prefs. Unix error code 273676720. 

CAUSE
The warning messages are caused by SUN JDK Bug# 4666397: 

http://developer.java.sun.com/developer/bugParade/bugs/4666397.html [http://developer.java.sun.com/developer/bugParade/bugs/4666397.html]


RESOLVING THE PROBLEM
You can safely ignore the warning messages. The creation of a WebSphere Commerce instance will not be affected by these warning messages.
