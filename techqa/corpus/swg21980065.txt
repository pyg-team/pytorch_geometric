 How to generate an SPSS authorization code / How to access and use the IBM SPSS License Key Center 

ANSWER
Create your SPSS authorization code through the IBM SPSS License Key Center
The IBM SPSS License Key Center is a self-service Authorization code and License key management service that provides you with easy access to and the ability to custom create your SPSS authorization and license codes.
