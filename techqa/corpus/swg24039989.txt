Title: IBM PI39793;8.5.5: Update WAS 8.5.5.5 License to exclude RAD - United States

Text:
PI39793; PI39793; PI39793 DOWNLOADABLE FILES

ABSTRACT
 Fix license information in WebSphere Application Server V8.5.5.5 to exclude IBM Rational Application Developer for WebSphere Software v9.0 as a supporting program. 

DOWNLOAD DESCRIPTION
PI39793 resolves the following problem:

ERROR DESCRIPTION: 
Update WAS 8.5.5.5 License to exclude RAD 

Remove IBM Rational Application Developer for WebSphere Software v9.0 from the list of Supporting Programs licensed with WebSphere Application Server V8.5.5.5 Program 

LOCAL FIX: 

PROBLEM SUMMARY

USERS AFFECTED:
All users of IBM WebSphere Application Server Base, Express, Developers, Network Deployment and z/OS editions

PROBLEM DESCRIPTION:
Fix license information in WebSphere Application Server V8.5.5.5 to exclude IBM Rational Application Developer for WebSphere Software v9.0 as a supporting program.

RECOMMENDATION:
None

The license in IBM WebSphere Application Server v8.5.5.5 for Base, Network Deployment, Express, z/OS and Developers editions incorrectly included a supporting program for IBM Rational
Application Developer for WebSphere Software v9.0 which should not have been included.

PROBLEM CONCLUSION: 
IBM Rational Application Developer for WebSphere Software v9.0 has been excluded from the license file 

The fix for this APAR is currently targeted for inclusion in fix pack 8.5.5.6. Please refer to the Recommended Updates page for delivery information: 
http://www.ibm.com/support/docview.wss?rs=180&uid=swg27004980 [http://www.ibm.com/support/docview.wss?rs=180&uid=swg27004980]

PREREQUISITES
None
