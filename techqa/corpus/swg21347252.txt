Title: IBM Error 'Exception has been thrown by the target of an invocation' caused by user's Windows account problem - United States

Text:
 TECHNOTE (TROUBLESHOOTING)

PROBLEM(ABSTRACT)
 User launches Excel. User accesses the Controller Link, and attempts to logon to Controller. User receives error. 

SYMPTOM
Exception has been thrown by the target of an invocation

CAUSE
There are many potential causes for this error. 

 * For more examples, see separate IBM Technotes 1347234 and 1506977.


This IBM Technote specifically relates to the scenario where the user's Windows user account has been locked out of the Microsoft Active Directory (domain). For example, this can be caused by an expired Windows password, or several unsuccessful attempts to login. 
ENVIRONMENT
Controller configured to use Windows Authentication as the authentication source.



RESOLVING THE PROBLEM
Ask I.T. department's Windows domain/AD administrator to fix the 'bad' end user's account (for example reset the user's Active Directory logon's password).
