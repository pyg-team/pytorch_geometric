LOCAL FIX

PROBLEM SUMMARY
 *  ****************************************************************
   * USERS AFFECTED:                                              *
   * ITNM 4.2 users only                                          *
   ****************************************************************
   * PROBLEM DESCRIPTION:                                         *
   * The serverStop command returns to early and this lead to     *
   * install failure with error ADMU3027E                         *
   ****************************************************************
   * RECOMMENDATION:                                              *
   * Upgrade to ITNM-IP | fix pack | 4.2.0-ITNMIP-FP0001          *
   ****************************************************************
   
   
    
   
   

PROBLEM CONCLUSION
 *  Added delay.before.start and delay.after.start where the user
   can manually change the values using something like:
   setenv IBM_JAVA_OPTIONS
   -Dom.ibm.tivoli.netcool.itnm.gui.was.delay.before.start=20
   
   The value is set in seconds. default are 15 for before start and
   3 for after start.
