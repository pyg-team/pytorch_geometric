Title: IBM How to enable the ldaploader for Platform Analytics - United States

Text:
 TECHNOTE (FAQ)

QUESTION
 You may need to enable the ldaploader to sample data from active directory. 

ANSWER
In Platform Analytics data collection host, there's a file called ldaploader.properties under $PERF_CONFDIR/dataloader directory. Here you can configure the ldaploader.
Configure how the ldaploader to access the Active Directory with:

ROOTDN
PASSWORD 
HOST 
PORT

Note the Password MUST be AES encrypted which can be produced by using encryptTool.sh to encrypt password.

Here is an example:

ROOTDN:cn=Manager,dc=example,dc=com 
PASSWORD:password 
HOST:127.0.0.1 
PORT:389
