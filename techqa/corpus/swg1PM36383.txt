APAR STATUS
 * CLOSED AS DOCUMENTATION ERROR.
    
   
   

ERROR DESCRIPTION
 *  Collaboration & Deployment Services Installation and
   Configuration guide mentions incorrect location for the Weblogic
   log4j.xml
   
   "The location of the log4j.xml configuration file varies,
   depending on the host application server:
   - JBoss - <JBoss installation directory>\server\default\conf.
   - WebLogic - <repository installation
   directory>\SPSSDomain\lib\. Note that log4j
   components used for logging on to WebLogic are part of the
   repository installation.
   - WebSphere- <repository installation
   directory>\setup\resources\websphere."
   
    The actual location is:
   WebLogic - <repository installation
   directory>\setup\resources\Weblogic
   
   
    
   
   

LOCAL FIX
 *  A request has been filed with R&D to update the documentation.
