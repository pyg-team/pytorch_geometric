APAR STATUS
 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  When using 7.5.2 firmware versions and higher, BluePrint
   console is the default which causes multiple latency issues in
   GUI performances.
   
   
    
   
   

LOCAL FIX
 *  Remove the /dp/index.html from the URL or click on the ? icon
   button at the top right and select WebGUI.
   
   
    
   
   

PROBLEM SUMMARY
 *  Enable old UI / Blueprint dropdown selection for 7.5.2.x
   
   
    
   
   

PROBLEM CONCLUSION
 *  Fix is available in 7.5.2.14
   
   For a list of the latest fix packs available, please see:
   http://www-01.ibm.com/support/docview.wss?uid=swg21237631 [http://www-01.ibm.com/support/docview.wss?uid=swg21237631]
