APAR STATUS
 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  Sap-Agent user IBMMON_AGENT locked in case of 711FP1 agent.
   
   Detailed Recreation Procedure:
   Use username with password having combination of
   uppercase+lowercase letters or complete lowercase letters.
   
   Case A.
   1. Stop SAP central instance.
   2. Start SAP Agent.
   3. Start SAP central instance after 10 minutes.
   3. Check agent log file with error message as
   RFC_LOGON_FAILURE(Password logon no longer possible - too many
   failed attempts) error.
   
   
   Case B.
   1. Start SAP Agent.
   2. Check successful RFC connection with the host.
   3. Stop SAP central instance.
   4. Start SAP central instance after 10 minutes.
   5. Check agent log file with error message as
   RFC_LOGON_FAILURE(Password logon no longer possible - too many
   failed attempts) error.
