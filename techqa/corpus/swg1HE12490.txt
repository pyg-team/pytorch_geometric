ERROR DESCRIPTION
 *  Customer tried the Domino Template Enablement with ICC/Domino
   4.0.0.2 and ran into a problem.
   
   The Domino configuration and template creation failed when
   'Start Configuration" was clicked by customer to enable template
    locally or in the remote Domino server by using "Domino
   Template Enablement" tool.
   
   
    
   
   

LOCAL FIX

PROBLEM SUMMARY
 *  Domino Template Enablement with ICC/Domino may cause problem due
   to unsuccessful paramaters update in notes.ini.
   
   The Domino configuration and template creation fails when 'Start
   Configuration" is clicked  to enable template locally or in the
   remote Domino server by using "Domino Template Enablement" tool.
   
   
    
   
   

PROBLEM CONCLUSION
 *  Starting with 4.0.1.3-IBM-ICC-FP003 release, this issue has been
    fixed.
