ABSTRACT
 This document describes how to download IBM® InfoSphere® Optim™ Data Growth Solution Starter Edition for JD Edwards Applications 11.3 using the Passport Advantage website. 

DOWNLOAD DESCRIPTION
The IBM InfoSphere Optim Data Growth Solution Starter Edition for JD Edwards Applications supports the removal of older, infrequently accessed, data from your JD Edwards database and storing it where business analysts can retrieve it when necessary. If desired, the expunged data can be accessed in real time, using the JD Edwards interface. Using this native application access, JD Edwards users can view current and archived data seamlessly while you reduce the size of your production database, resulting in better performance for your daily processing.

PREREQUISITES
You must install the IBM InfoSphere Optim Solution v11.3 before you install the Optim solution for JD Edwards. 

For information about hardware and software requirements for the Optim solution for JD Edwards, see the Detailed System Requirements at http://www.ibm.com/support/docview.wss?&uid=swg27041747 [http://www.ibm.com/support/docview.wss?&uid=swg27041747].

INSTALLATION INSTRUCTIONS
For general installation instructions, see the Quick Start Guide. For detailed instructions, see the Optim Data Growth Solution for JD Edwards EnterpriseOne Installation and Configuration Guide. Both documents are included in the download package.
