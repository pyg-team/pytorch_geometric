Title: IBM Run tasks failing with LSMR014I (SCI65785) - United States

Text:
STERLINGPRI TECHNOTE (TROUBLESHOOTING)

PROBLEM(ABSTRACT)
 Run tasks failing with LSMR014I (SCI65785) 

SYMPTOM
LSMR014I 
File &EXE not found during RUN JOB/RUN TASK - rc &ERRMSG


CAUSE


A program directory was defined for the local user, and the run task was coded as follows:



SYSOPTS="cmd(c:\incoming\runme.bat)"



The program dir in user authorization was coded as c:\incoming\



cmd.exe (Windows command shell) can not be found as it is located in c:\winnt\system32

RESOLVING THE PROBLEM
Copy cmd.exe to the defined program directory
