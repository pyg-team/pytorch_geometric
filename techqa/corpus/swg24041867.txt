 DOWNLOADABLE FILES

ABSTRACT
 This is the fourth interim fix after ITCAM Agents for WebSphere Messaging Version 7.1.0 fixpack 01 

DOWNLOAD DESCRIPTION
This interim fix addresses a list of APARs in previous version and refreshes the Java certificate of application support files.

PREREQUISITES
1. ITCAM Agents for WebSphere Messaging Version 7.1.0 Fix Pack 1 must be installed. 


 See the link below for details:
http://www-01.ibm.com/support/docview.wss?uid=swg24035244 [http://www-01.ibm.com/support/docview.wss?uid=swg24035244]

2. Make sure that the software installed in your environment is compatible with this interim fix.
