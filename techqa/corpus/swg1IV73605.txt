ERROR DESCRIPTION
 *  The task "Run Capacity Scan and Upload Results" does not retry
   if he encounters a failure by default.
   
   It cause the customer needs to start the task again manually.
   
   This APAR is opened to change the default behavior.
   
   
    
   
   

LOCAL FIX
