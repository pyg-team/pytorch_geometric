Title: IBM Use a power shell script with Sterling Gentran:Server for Microsoft Windows - United States

Text:
 TECHNOTE (FAQ)

QUESTION
 How to run power shell script with Gentran:Server for Microsoft Windows 

CAUSE
Ability to use Gentran's Server run task from process control to execute Power Shell scripts.

ANSWER
Power shell script with Gentran:Server for Microsoft Windows ran successfully from DOS command. 

The command to use in Gentran process control to allow PowerShell to run a script is as below.

powershell -ExecutionPolicy bypass -file "Name of Script.ps1"
