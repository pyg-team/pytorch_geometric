 TECHNOTE (TROUBLESHOOTING)

PROBLEM(ABSTRACT)
 This document provides instructions on how to use the GUI debugger from IBM iSeries® Navigator. 

RESOLVING THE PROBLEM
The following steps show how to use the debugger in IBM iSeries® Navigator: 

Note: IBM iSeries Navigator must be at R530 or later. 

1. Open iSeries Navigator. 2. Make a connection to the IBM System i® products system. 3. Expand the connection. 4. Expand Databases. 5. Right-click on your connection name. 
iSeries Navigator Main Menu with DataBases option open [/support/docview.wss?uid=nas8N1014814&aid=1] [/support/docview.wss?uid=nas8N1014814&aid=1] 6. Select Run SQL Scripts from the drop-down. 
iSeries Nav Run SQL Scripts window [/support/docview.wss?uid=nas8N1014814&aid=2] [/support/docview.wss?uid=nas8N1014814&aid=2] 
7. Click on Run. 8. Select Debugger from the drop-down. 9. A screen will pop up: 
iSeries System Debugger Program screen [/support/docview.wss?uid=nas8N1014814&aid=3] [/support/docview.wss?uid=nas8N1014814&aid=3] 10. You might need to sign on the system. After you sign on the system, the Start Debug screen will pop up. If this screen does not pop up, you must select Debug and Start Debug from the iSeries System Debugger screen. 
iSeries Nav Start Debug [/support/docview.wss?uid=nas8N1014814&aid=4] [/support/docview.wss?uid=nas8N1014814&aid=4]
