Title: IBM CDM connection to HFM via the HFM Connector Web Application - United States

Text:
 TECHNOTE (FAQ)

QUESTION
 How to connect from Cognos Disclosure Management to HFM using the HFM Connector Web Application? 

CAUSE
 From Cognos Disclosure Management 10.2.6, support has been added for the HFM Connector Web Application.

ANSWER
 Through the HFM Connector Web Application, Cognos Disclosure Management can connect to the corresponding CDM Data Source Connector application deployed on the HFM server. The CDM Data source connector is an independent tool, built around the HFM Java API. 

 * The HFM versions supported by the HFM Connector Web Application are 11.1.2.4 or higher. For lower versions the connection is made through native CDM services. 

 * The CDM versions supporting the HFM Connector Web Application are 10.2.6 or higher.


Customers will request the Connector through Support, following the same procedure as for interim fixes.
