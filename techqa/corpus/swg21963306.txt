Title: IBM Security Bulletin: Multiple vulnerabilities have been identified in WebSphere Application Server shipped with IBM Tivoli Service Automation Manager (CVE-2015-0488, CVE-2015-0478, CVE-2015-0204 and CVE-2015-2808) - United States

Text:
CVE-2015-0488; CVE-2015-0478; CVE-2015-0204; CVE-2015-2808 SECURITY BULLETIN

SUMMARY
 WebSphere Application Server is shipped as components of Tivoli Service Automation Manager. Information about a security vulnerability affecting WebSphere Application Server has been published in a security bulletin. 

VULNERABILITY DETAILS
Review security bulletin Multiple vulnerabilities in IBM® Java SDK affect WebSphere Application Server April 2015 CPU [http://www-01.ibm.com/support/docview.wss?uid=swg21902260] for vulnerability details/

AFFECTED PRODUCTS AND VERSIONS
 

Principal Product and Version(s) Affected Supporting Product and Version Tivoli Service Automation Manager V7.2.1 through V7.2.4 WebSphere Application Server V6.1.0.0 through V6.1.0.47. REMEDIATION/FIXES
Review security bulletin Multiple vulnerabilities in IBM® Java SDK affect WebSphere Application Server April 2015 CPU [http://www-01.ibm.com/support/docview.wss?uid=swg21902260] for information about fixes.
