QUESTION
What are the common issues and questions related to IBM Traveler?

ANSWER
Here is a list of the top reported problems and questions related to the IBM Traveler product. There are two sections:
A) IBM Traveler 8.5.3 and 8.5.2
B) General How To for all releases


A. IBM Traveler 8.5.3 and 8.5.2:

Android device specific:
A1. Which Traveler version supports Android devices? 

Android devices (including Android 4.0) are supported with Traveler's latest fix packs for 8.5.2 or 8.5.3. Find the latest fix pack using item B1 below. 
A2. Why can't I download the Traveler client using the Android browser? 
This is a known issue as documented in this technote [http://www.ibm.com/support/docview.wss?uid=swg21457494]. 

A3. Can Traveler be deployed on Kindle Fire or B&N Nook Android devices? 
This is documented here. [http://www.ibm.com/support/docview.wss?uid=swg21573745] [http://www.ibm.com/support/docview.wss?uid=swg21457594] 

A4. Why am I unable to upgrade the Traveler Android client from 8.5.2 to 8.5.3? 
This is documented here. [http://www.ibm.com/support/docview.wss?uid=swg21578314] 




Apple device specific: 
A5: Is Apple iOS 6 and Apple iPhone 5 supported with Traveler? 
Yes, this is documented in this technote. [http://www.ibm.com/support/docview.wss?uid=swg21577369] 

A6. After an Apple OS upgrade, why does Traveler not sync? 
This is an Apple OS issue as discussed here. [http://www.ibm.com/support/docview.wss?uid=swg21495770] 

A7. Newly configured devices added to the server are not able to sync. On the device, you may see the error: Unable to connect to the server.

B. General How To: 

B1. How do I find the latest maintenance for my Traveler release? 
The latest fix packs and fixes can be found on the Traveler Recommended Maintenance page [http://www.ibm.com/support/docview.wss?uid=swg24019529]. It is highly recommended to stay current with the latest fixpacks available for Traveler, which includes fixes for latest Apple OS versions and the Android Client. 

B2. How can I find the list of APARs (defects) included in a particular release? 
The list of APARs for each fix pack can be found in this document. [http://www-10.lotus.com/ldd/dominowiki.nsf/dx/Lotus_Notes_Traveler_APAR_listing] It is highly recommended to stay current with the latest fixpacks available for Traveler, which includes fixes for latest Apple OS versions and the Android Client. Please review the APAR listing regularly to determine if you may be encountering an issue that has already been addressed. 

B3. Where can I find the Traveler documentation (InfoCenter) for my release? 
Traveler Product documentation can be found here. [http://www-01.ibm.com/support/knowledgecenter/SSYRPW/welcome] 

B4. What are the system and device requirements for Traveler? 
The system and device requirements for Traveler 8.5.3 are listed here. [http://www.ibm.com/support/docview.wss?uid=swg27022506] 
The system and device requirements for Traveler 8.5.2 are listed here. [http://www.ibm.com/support/docview.wss?uid=swg27019476] 

B5. How do I find the latest Traveler code on Passport Advantage? 
Follow these steps to find the latest Traveler code on Passport Advantage (There is only one package which will install either 32 bit or 64 bit to match the version of Domino): 

1) Navigate to IBM Passport Advantage Online [http://www.ibm.com/software/howtobuy/passportadvantage/pao_customers.htm] and login 
2) Select: Download finder 
3) Select: Find by description or part number 
4) For Product description enter "traveler" and then Continue 
5) Select the applicable build based on your server operating system: 

For the version of Traveler 8.5.3 Upgrade Pack 2, select this package: 
Package Name: IBM Lotus Notes Traveler 8.5.3 Upgrade Pack 2 
Windows part number: CIB9RML 
Linux part number: CIB9SML 

[/support/docview.wss?uid=swg21450615&aid=1] [/support/docview.wss?uid=swg21450615&aid=1] 



If you are looking for Traveler 8.5.3.3, locate it by searching on the correct part number below: 
Package Name: IBM Lotus Notes Traveler V8.5.3.3 
Windows part number: CIA2XML 
Linux part number: CIA2YML
