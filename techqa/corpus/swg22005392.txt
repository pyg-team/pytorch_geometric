Title: IBM Security Bulletin: IBM MQ and IBM MQ Appliance could allow a local user to crash the queue manager agent thread and expose some sensitive information (CVE-2017-1760) - United States

Text:
 SECURITY BULLETIN

SUMMARY
 Queue manager agent thread crashes with SIGSEGV following execution of valid MQSC command. 

VULNERABILITY DETAILS
CVEID: CVE-2017-1760 [http://cve.mitre.org/cgi-bin/cvename.cgi?name=CVE-2017-1760]
DESCRIPTION: IBM WebSphere MQ could allow a local user to crash the queue manager agent thread and expose some sensitive information.
CVSS Base Score: 4
CVSS Temporal Score: See https://exchange.xforce.ibmcloud.com/vulnerabilities/126454 [https://exchange.xforce.ibmcloud.com/vulnerabilities/126454] for the current score
CVSS Environmental Score*: Undefined
CVSS Vector: (CVSS:3.0/AV:L/AC:H/PR:N/UI:N/S:U/C:L/I:N/A:L)
