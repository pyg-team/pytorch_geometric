Title: IBM How to Print Tape/Media Session Statistics - United States

Text:
 TECHNOTE (TROUBLESHOOTING)

PROBLEM(ABSTRACT)
 This document explains how to print tape/media session statistics. 

RESOLVING THE PROBLEM
To print tape/media session statistics, you should do the following: 

1. Start the System Service Tools by issuing the STRSST command. 2. Sign in to the service tools. 3. Select Option 1 - Start a service tool. 4. Select Option 1 - Product activity log. 5. Select Option 5 - Display or print removable media session statistics. 6. Specify the following on the Select Media Options panel:

Removable media......2
Volume ID.......................*ALL
Print report......................Y

as indicated in Figure 1 below:

This is a screen shot of the SELECT MEDIA OPTIONS panel that is display when you select Option 5 on the Product Activity Log menu. [/support/docview.wss?uid=nas8N1011974&aid=1] [/support/docview.wss?uid=nas8N1011974&aid=1]
Notes:
a. Removable media type 2 is for most of the new LTO type cartridges such as all ULTRIUMn and 3592.
b. If you know the volume ID of your backup, you can isolate results for that specific backup/cartridge/volume. 7. Specify a time period on the Set Time Period panel (Figure 2) that you would like the report to include (it defaults to the last 24 hours):

This is a screen shot of the SET TIME PERIOD option panel that is displayed to choose which session statistics to print. [/support/docview.wss?uid=nas8N1011974&aid=2] [/support/docview.wss?uid=nas8N1011974&aid=2] 8. Wait for Printing or spooling operation is complete message to post to the bottom of the screen 9. Press F3=Exit three times, and then press Enter to exit out of the service tools. 10. Issue the WRKSPLF command to locate the spooled report. The report will list as QPCSMPRT in your spooled output.
