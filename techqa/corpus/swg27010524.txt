ABSTRACT
 This presentation will help you understand the questions you need to ask to determine the qualities of services for CICS Transaction Server for z/OS (CICS TS) Web Services implementation. Knowing the required qualities of service required will enable you to define the appropriate CICS resources and set the correct configuration options. This presentation was written for CICS TS V3.2. 

CONTENT
 

 * How to determine the required Qualities of Service  * Questions to ask
   
   
 * Configuration and resource definition  * Base configuration 
    * Transport definition 
    * Locating the web service 
    * Quality of service 
   
   
 * Documentation 
 * Summary


Note: This presentation was created for CICS TS V3.2 but it is applicable to CICS TS V3.1 and V4.1. Many new options are available in the newer releases of CICS TS.
