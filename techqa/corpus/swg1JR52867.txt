APAR STATUS
 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  When you try to show instance details from a Cognos report that
   is filtered according to a dimension that has multiple levels,
   you do not see any instances although you expected to see a
   filtered list of instances in the Instances widget.
   
   PRODUCTS AFFECTED
   IBM Business Monitor
   
   
    
   
   

LOCAL FIX

PROBLEM SUMMARY
 *  No additional information is available.
   
   
    
   
   

PROBLEM CONCLUSION
 *  A fix is available for IBM Business Monitor V8.5.5.0 that allows
    you to use Cognos filters according to dimensions that have
   multiple levels when you populate the data in the Instances
   widget from a Cognos report.
   
   On Fix Central (http://www.ibm.com/support/fixcentral), search
   for JR52867:
   
   1. Select IBM Business Monitor from the product selector, the
   installed version to the fix pack level, and your platform, and
   then click Continue.
   2. Select APAR or SPR, enter JR52867, and click Continue.
   
   When you download fix packages, ensure that you also download
   the readme file for each fix. Review each readme file for
   additional installation instructions and information about the
   fix.
