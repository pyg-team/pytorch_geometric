Title: IBM Security Bulletin: Multiple vulnerabilities in IBM Java Runtime affect IBM Storwize V7000 Unified - United States

Text:
 SECURITY BULLETIN

SUMMARY
 There are vulnerabilities in IBM® Runtime Environment Java™ Technology Edition. IBM Storwize V7000 Unified has addressed the applicable CVEs. 

VULNERABILITY DETAILS
IBM Storwize V7000 Unified is shipped with Java. Java is required for administration of V7000 Unified, for executing V7000 Unified specific commands on the command line, and providing similar functionality using GUI.

AFFECTED PRODUCTS AND VERSIONS
IBM Storwize V7000 Unified
The product is affected when running code releases 1.5.0.0 to 1.6.2.2

REMEDIATION/FIXES
A fix for this issue is in version 1.6.2.3 of IBM Storwize V7000 Unified. Version 1.5 is end of service. Customers running on this release of IBM Storwize V7000 Unified should upgrade to v1.6.2.3 or a later version, so that the fix gets applied. 

WORKAROUNDS AND MITIGATIONS
Workaround(s): None

Mitigation(s) : Although IBM recommends that you install a level of code with a fix for this vulnerability, you can mitigate, although not eliminate, your risk until you have done so by ensuring that all users who have access to the system are authenticated by another security system such as a firewall.
