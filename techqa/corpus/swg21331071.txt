 What more information exists about IBM Rational RequisitePro supported Microsoft Windows RequisiteWeb Servers 64-bit platform?

- [EM64T] means [Intel x86 EM64T]. 

- RequisitePro is a 32-bit application and the installation files are available from IBM Passport Advantage [http://www.ibm.com/software/howtobuy/passportadvantage/index.html]. 

- The operating system will automatically detect 32-bit applications and automatically install under C:\Program Files(x86)\ as 32-bit application. 

- The RequisitePro application will run in 32-bit emulation mode on 64-bit OS because it is a 32-bit application. 

- Users do not need to configure applications to run in 32-bit/64-bit mode on 64 bit OS.
