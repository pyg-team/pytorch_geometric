APAR STATUS
 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  The toolkit fix packs for V9 are installed via the IBM
   Installation Manager. There are two options available to
   install the fix pack without using the GUI version.
   
   They are web or console mode. Neither are supported by the fix
   pack repositories.
   
   This apar is to allow the fix packs to be installed using
   console mode.
   
   
   Additional Symptom(s) Search Keyword(s):
   
   
    
   
   

LOCAL FIX
 *  LocalFix:
   
   To install a fix pack using console mode:
   1) Navigate to the installation Manager directory and move into
   the eclipse/tools directory.
   2) edit the imcl.ini file,
   3) add the following parameter to the ini file:
   -Dcic.supports.console.mode=true
   4) save the file.
   5) start the installation Manager using the following command:
   imcl.exe -consolMode
   
   To install using the web console (you will need IM ver 1.8.5) :
   1) Navigate to the installation Manager directory and move into
   the eclipse/web directory.
   2) edit the ibmim-web.ini file,
   3) add the following parameter to the ini file:
   -Dcic.suppress.web.ui.checks=true
   4) save the file.
   5) start the installation Manager using the following command:
   ibmim-web
   6) This will display a web address.  Using a browse, go to that
   webpage and you should be able to install the fix pack via that
   webpage.
   
   
    
   
   

PROBLEM SUMMARY
 *  ****************************************************************
   USERS AFFECTED:
   All user of IBM Integration Bus V9.0 Toolkit using the
   Installation Manager in console mode.
   
   
   Platforms affected:
   Windows on x86 platform, Linux on x86 platform
   
   ****************************************************************
   PROBLEM DESCRIPTION:
   The Toolkit can only be installed via the Installation Manager
   GUI.  If your linux box does not permit
   a GUI install then you cannot install a toolkit fix pack.
   
   There are a number of resource name changes between WebSphere
   Message Broker and IBM Integration Bus Version 9.0.  For details
   visit
   http://pic.dhe.ibm.com/infocenter/wmbhelp/v9r0m0/topic/com.ibm.e
   tools.mft.doc/bb23814_.htm [http://pic.dhe.ibm.com/infocenter/wmbhelp/v9r0m0/topic/com.ibm.etools.mft.doc/bb23814_.htm]
   
   
    
   
   

PROBLEM CONCLUSION
 *  The product has been updated to allow the Installation Manger to
   use the console mode to install fix packs.
