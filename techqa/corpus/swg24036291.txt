INSTALLATION INSTRUCTIONS
Complete the following instructions to download the firmware. 

 1. Initialize and configure the appliance.
    The firmware upgrades for the 7199-92x (2426-92X) appliances are available in this document. See Initializing and configuring WebSphere DataPower XC10 Appliance [http://pic.dhe.ibm.com/infocenter/wdpxc/v2r5/topic/com.ibm.websphere.datapower.xc.doc/tinitappl.html] in the product information center for more information about the initialization and firmware upgrade process. 
    
    Note: The type 7199 appliances can have one of two machine types depending on when they were ordered. Type 2426, listed in parentheses, is the machine type model (MTM) if the appliance is ordered from Passport Advantage. 
 2. Install IBM Installation Manager [http://www-304.ibm.com/support/docview.wss?uid=swg27025142].  1. Add the fix pack repository location in IBM Installation Manager: 
     2. Start IBM Installation Manager. 
     3. In Installation Manager, click File > Preferences > Repositories. 
     4. Click Add Repository. 
     5. Browse to or enter the file path to the repository.config file. The repository.config file is located in the directory where you extracted the compressed files.
    
    
 3. Use Installation Manager to install or upgrade the product from the local product repositories that you downloaded in step 2.
    For more information about installing the fix pack, see Installing Fix Packs using Installation Manager [http://pic.dhe.ibm.com/infocenter/wdpxc/v2r5/index.jsp?topic=%2Fcom.ibm.websphere.datapower.xc.doc%2Ftxs_fixpacks.html] in the information center.


Downloading the product with Download Director 


Fix Central (FC) links are provided for you to download the product offerings. However, you can alternatively use Download Director, which is a Java applet that provides enhanced download capability to IBM customers who download our products from the internet. It provides high-performance, reliable downloads with pause and resume capability. It starts in a separate browser window and provides a graphical user interface (GUI) to specify the download location, and view download status. To use Download Director to download the product offerings, complete the following steps:


 1. Click the Fix Central (FC) link for the product offering that you want to download. 
 2. When the Fix Central page is displayed, click the Continue button. 
 3. Enter your IBM ID and password when prompted. 
 4. Click Change download options. 
 5. Select Download using Download Director. 
 6. On the Download files using Download Director page, click Download now to begin downloading the selected product offering.
