Title: IBM Action required for IBM WebSphere MQ Hypervisor Edition for RedHat Enterprise Linux Server for the GNU C library (glibc) vulnerabilities (CVE-2015-1472 & CVE-2013-7423) - United States

Text:
 FLASH (ALERT)

ABSTRACT
 WebSphere MQ Hypervisor Edition requires customer action for the GNU C library (glibc) vulnerabilities (CVE-2015-1472 & CVE-2013-7423) 

CONTENT
WebSphere MQ Hypervisor Edition Versions 7.0.1 and 7.5 ship with Red Hat Enterprise Linux (RHEL) that is vulnerable to GNU C library (glibc) vulnerabilities (CVE-2015-1472 & CVE-2013-7423). 

Remediation/Fixes 

 IBM strongly recommends that you should contact Red Hat to obtain fixes and install fixes for Red Hat Enterprise Linux (RHEL).
