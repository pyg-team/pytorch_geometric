APAR STATUS
 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  For a very small subset of processors (for which brand
   internal-value = Quad-Core Xeon; xS ) the application
   calculates incorrect PVU rating.
   
   This only occurs if the processor configuration is not equal to
   the one defined in PVU table - for example 1 socket instead of 2
   expected by the PVU table (2S).
   
   
    
   
   

LOCAL FIX
 *  No workaround possible (hotfix required).
   
   
    
   
   

PROBLEM SUMMARY
 *  ****************************************************************
   * he PVU value that is assigned to the Intel E6510 and E7520
   * processor is incorrect.
   * EXPECTED RESULTS:
   * The PVU value is correct.
