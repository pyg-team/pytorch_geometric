Title: IBM Security Bulletin: IBM WebSphere MQ Heap storage leak on error path could lead to denial of service (CVE-2016-0260) - United States

Text:
 SECURITY BULLETIN

SUMMARY
 Heap storage allocated on an error path is not deallocated by queue manager agents, a malicious user could use an application to repeatedly cause the queue manager to execute this error path and eventually exhaust storage for the agent, thereby causing a denial of service. 

VULNERABILITY DETAILS
CVEID: CVE-2016-0260 [http://cve.mitre.org/cgi-bin/cvename.cgi?name=CVE-2016-0260]
DESCRIPTION: IBM WebSphere MQ contains an agent memory leak could be exploited by malicious user to cause a denial of service.
CVSS Base Score: 5.3
CVSS Temporal Score: See https://exchange.xforce.ibmcloud.com/vulnerabilities/110602 [https://exchange.xforce.ibmcloud.com/vulnerabilities/110602] for the current score
CVSS Environmental Score*: Undefined
CVSS Vector: (CVSS:3.0/AV:N/AC:L/PR:N/UI:N/S:U/C:N/I:N/A:L)

AFFECTED PRODUCTS AND VERSIONS
IBM WebSphere MQ 8.0.0.0 through 8.0.0.4 maintenance levels are affected.

REMEDIATION/FIXES
Apply IBM WebSphere MQ 8.0.0.5 fixpack [http://www-01.ibm.com/support/docview.wss?uid=swg21982686] or later maintenance.

WORKAROUNDS AND MITIGATIONS
None.
