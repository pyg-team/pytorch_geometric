ABSTRACT
 This document describes how to download IBM® Content Manager OnDemand for i V7.3 using the Passport Advantage® online web site. 

DOWNLOAD DESCRIPTION
The software is available on IBM i systems when ordered but also through Passport Advantage. 

Downloading the product 
1. From a browser, navigate to the Find Downloads and Media [http://www.ibm.com/software/howtobuy/passportadvantage/paocustomer/sdma/SDMA?PO=DOWNLOAD_FIND_BY_PART_NUMBER] page on the IBM Passport Advantage Online Web site. You must log on to continue. 
2. Select the Find by part number search option under Download finder options. 
3. Search for the following part numbers and download the images.
