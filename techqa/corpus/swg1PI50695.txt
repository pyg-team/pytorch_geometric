Title: IBM PI50695: SYNDICATION REPORT SUMMARY FOR EXISTING SYNDICATORS CANNOT BE VI EWED - United States

Text:
  A FIX IS AVAILABLE
Fixes integrated in WebSphere Portal 8.5.0.0 Combined Cumulative Fixes [http://www-01.ibm.com/support/docview.wss?uid=swg24037786]



APAR STATUS
 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  For any syndication pair that exists on the server prior to the
   CF07 upgrade, the syndicator report is corrupt and cannot be
   viewed.
   
   
   Additionally, the following can be seen in the SystemOut log.
   
   ReportTransfo W   REPORT_HEADER_COULD_NOT_BE_READ_FROM_JCR
   
   
    
   
   

LOCAL FIX
 *  Delete the existing syndication pair and create a new syndicator
   pair after the CF07 upgrade is complete.

   Portal & WCM Combined Cumulative Fix 09 (PI50956) for Portal
