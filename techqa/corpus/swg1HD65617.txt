ERROR DESCRIPTION
 *  FILE : ACCELERATOR : CATIA HANGS WHEN
   CLOSING A FILE WITH ACCELERATOR ON AIX
   .
   SCENARIO:
   1. Start CATIA on AIX.
   2. Click on Tools menu and select Customize
   option. Customize dialog box appears.
   3. Go to Commands tab page.
   4. Click on File menu and select Close command.
   5. Click on show properties option.
   6. Define an accelerator ALT+X.
   7. Close the Customize dialog box.
   8. Create a new empty CATPart
   9. Create a Pad with 50mm x 50mm x 50mm
   dimension
   10. Close the part without saving by using the
   accelerator ?Alt + X?.
   .
   OBSERVATION1:
   The window ?Do you want to save changes to
   Part1? appears.
   .
   11. Do not select any option and press the
   accelerator Alt + x again
   .
   OBSERVATION2:
   CATIA hangs and the window cannot be select any
   more. CATIA has to be killed
   .
