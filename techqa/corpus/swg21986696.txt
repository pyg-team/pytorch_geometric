Title: IBM Error 'Argument #1 for function ASFUNCTION is invalid' is encountered. - United States

Text:
WTX ITX Functional maps ASFUNCTION TECHNOTE (TROUBLESHOOTING)

PROBLEM(ABSTRACT)
 Error 'Argument #1 for function ASFUNCTION is invalid' is encountered. 

SYMPTOM
Error 'Argument #1 for function ASFUNCTION is invalid' is encountered.


CAUSE
The output of the functional map is a group. You can only use the AsFunction function with functional maps that produce items.

ENVIRONMENT
Using the ASFUNCTION function in a IBM transformation (ITX) map.

DIAGNOSING THE PROBLEM
The following error is encountering when trying to compile an ITX map that is using the ASFUNCTION function.





M113 ERROR: 
Map: FunctionalMapName Output: Text:Blob 
Argument #1 for function ASFUNCTION is invalid. 


RESOLVING THE PROBLEM
You can only use ASFUNCTION with functional maps that produce items.
