Title: IBM You do not have authorization for the application - United States

Text:
TPAEWORKFLOW ; email ; notification; communication; template; do not have authorization for the application TECHNOTE (TROUBLESHOOTING)

PROBLEM(ABSTRACT)
 User gets a notification from a workflow and they click on the link it gives the error message 

SYMPTOM
The workflow has the communication template assigned to send emails and the link. When a user gets a notification from a workflow and they click on the link it gives the error message. The security settings are granted correctly.

RESOLVING THE PROBLEM
In order to resolve the email link which has sent as 
localhost:9080/maximo/ui/maximo.jsp?event=loadapp&value=WOTRACK&uniqueid=1646

You need to update the hostname of systems property. 
Go to System Configuration, Platform Configuration, System Properties 
Filter mxe.hostname 
And enter the Global Value as actual server name and port number. For an example server.ibm.com:9080 
Save and Live refresh
