 *  ****************************************************************
   * USERS AFFECTED: All users of WebSphere MQ for z/OS Version 7 *
   *                 Release 1 Modification 0.                    *
   ****************************************************************
   * PROBLEM DESCRIPTION: Incompatibility between WebSphere MQ    *
   *                      and WebSphere Message Broker when using *
   *                      the WMB Web Interface.                  *
   ****************************************************************
   * RECOMMENDATION:                                              *
   ****************************************************************
   When using the WebSphere Message Broker
   Web Interface, an incompatibility with
   WebSphere MQ may prevent statistics
   from being correctly delivered and cause
   the Web Interface process to terminate.
   
   
    
   
   

PROBLEM CONCLUSION
 *  WebSphere MQ processing has been amended to ensure that the
   process for delivering statistics remains running.
