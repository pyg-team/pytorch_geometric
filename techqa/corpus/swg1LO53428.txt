ERROR DESCRIPTION
 *  Customer receives exception below when attempting to access ACL
   of database in Java application connecting to Domino via DIIOP.
   
   *  Exception in thread "Thread-0"
      java.util.MissingResourceException: Can't find resource for
      bundle java.util.PropertyResourceBundle, key
      could_not_read_ACL
   
   Steps to reproduce:
   1.  Detached file and expand contents into Domino data
       directory.
   2.  Edit Java code: update server, username and password
       variables.
   3.  Compile and run code, make sure to include NSCO.jar file in
       classpath.
   
   
    
   
   

LOCAL FIX
