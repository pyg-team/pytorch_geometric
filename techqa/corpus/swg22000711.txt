Title: IBM InfoSphere Information Server is not vulnerable to CVE-2017-5638 - United States

Text:
 FLASH (ALERT)

ABSTRACT
 IBM InfoSphere Information Server is not vulnerable to the Apache Struts 2 vulnerability CVE-2017-5638 

CONTENT
IBM InfoSphere Information Server is NOT vulnerable to the Apache Struts 2 vulnerability (CVE-2017-5638). 
Refer to https://cwiki.apache.org/confluence/display/WW/S2-045 [https://cwiki.apache.org/confluence/display/WW/S2-045] for details of the vulnerability.
IBM WebSphere Application Server has published a flash [http://www.ibm.com/support/docview.wss?uid=swg22000122].
