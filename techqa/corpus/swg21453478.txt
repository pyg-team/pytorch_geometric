 TECHNOTE (FAQ)

QUESTION
 How can connectivity to a given endpoint be checked from the command line interface? 

ANSWER
There are some CLI commands which can be used to manage and introspect network settings on the Cast Iron Integration Unit.

‘net’ - Network management CLI commands
For example:
a) net ping -Pings the specified host to determine if it is responding on the network.
b) net traceroute -Traces the network route to the specified node.

‘netspect’ - Network introspection CLI commands
For example:
a) netspect ping - Attempts to ping the specified network host, and displays the result of the ping attempt.
b) netspect portscan - Attempts a limited port scan of the specified network host, and displays the result of the port scan.


Refer to the product documentation [http://www.ibm.com/support/knowledgecenter/SSGR73/welcome] for more information on various other network related commands.
