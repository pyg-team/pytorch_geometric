Title: IBM Warning message 1407 is missing from Messages and Codes of Content Manager OnDemand for Multiplatforms V8.5 - United States

Text:
error messages; cmos; mp; messages and code; v8.5 TECHNOTE (TROUBLESHOOTING)

PROBLEM(ABSTRACT)
 You cannot find message number 1407 in Messages and Codes of Content Manager OnDemand for Multiplatforms, Version 8.5 (SC27-1379-10). 

RESOLVING THE PROBLEM
The following warning message information is not included in the Messages and Codes of Content Manager OnDemand for Multiplatforms, Version 8.5 (SC27-1379-10):
ARS1407W
Warning: A document with >pages< pages and >document_size< bytes was processed. It is recommended that the application be defined to use large object support when dealing with documents of such size. Although data may store successfully, it is possible that the data may not be able to be retrieved successfully.
Explanation: The system determined that a document in the load justifies defining large object support in the application definition.
Response: Update the application definition to define the use of large object support.
