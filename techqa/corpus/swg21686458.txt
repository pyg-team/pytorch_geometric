Title: IBM Security Bulletin: A security vulnerability has been identified in IBM Websphere shipped with IBM Security Directory Integrator (CVE-2014-4263, CVE-2014-4244) - United States

Text:
 SECURITY BULLETIN

SUMMARY
 IBM Websphere is shipped as a component of IBM Security Directory Integrator. Information about two security vulnerabilities affecting IBM Websphere has been published in a security bulletin. 

VULNERABILITY DETAILS
Two security vulnerabilities have been identified in IBM WebSphere shipped with IBM Security Directory Integrator (CVE-2014-4263 , CVE-2014-4244). Please consult the security bulletin "Security Bulletin: Multiple vulnerabilities in current IBM SDK for Java for WebSphere Application Server July 2014 CPU" [http://www-01.ibm.com/support/docview.wss?uid=swg21680418] for vulnerability details and information regarding fixes.

AFFECTED PRODUCTS AND VERSIONS
IBM Security Directory Integrator 6.1, 7.0, 7.1, 7.1.1, 7.2

REMEDIATION/FIXES
 

Product Release Fix information IBM Security Directory Integrator 6.1, 7.0 7.0.0-TIV-TDI-LA0019 [http://www-933.ibm.com/support/fixcentral/swg/selectFixes?parent=Security%2BSystems&product=ibm/Tivoli/Tivoli+Directory+Integrator&release=7.0.0&platform=All&function=all] IBM Security Directory Integrator 7.1, 7.1.1 7.1.1-TIV-TDI-LA0020 [http://www-933.ibm.com/support/fixcentral/swg/selectFixes?parent=Security%2BSystems&product=ibm/Tivoli/Tivoli+Directory+Integrator&release=7.1.1&platform=All&function=all] IBM Security Directory Integrator 7.2 7.2.0-ISS-SDI-LA0002 [http://www-933.ibm.com/support/fixcentral/swg/selectFixes?parent=Security%2BSystems&product=ibm/Tivoli/Security+Directory+Integrator&release=7.2.0&platform=All&function=all] WORKAROUNDS AND MITIGATIONS
None
