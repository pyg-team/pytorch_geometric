Title: IBM Required interim fixes for IBM Business Process Management products - United States

Text:
 DOWNLOADABLE FILES

ABSTRACT
 Interim fixes, including fixes for IBM WebSphere Application Server, are required for several business process management products. 

DOWNLOAD DESCRIPTION
Required Interim Fixes
This document provides you with interim fixes required for V8.0 of the following products,: 

 * IBM Business Monitor V8.0 
 * IBM Business Monitor for z/OS V8.0 
 * IBM Business Process Manager Standard V8.0 
 * IBM Business Process Manager Express V8.0 
 * IBM Business Process Manager Advanced V8.0 
 * IBM Business Process Manager Advanced – Process Server V8.0 
 * IBM Business Process Manager Advanced for z/OS V8.0


Installation Instructions 

 1. Download and install the interim fix package for WebSphere Application Server V8.0.0 Fix Pack 3, which is entitled, "WAS 8.0.0.3 – Required Interim Fixes," from the Download package table in this document. 
    
    These fixes are required for V8.0 of the following products:  * IBM Business Monitor 
     * IBM Business Monitor for z/OS 
     * IBM Business Process Manager Standard 
     * IBM Business Process Manager Express 
     * IBM Business Process Manager Advanced 
     * IBM Business Process Manager Advanced – Process Server 
     * IBM Process Server Advanced for z/OS
       
       
    
    
 2. Download and install the product-specific V8.0 interim fix from the download package table below:  * IBM Business Process Manager Standard V8.0
       Download the following package: 
       Standard – JR42732 
     * IBM Business Process Manager Express V8.0
       Download the following package: 
       Express – JR42732 
     * IBM Business Process Manager Advanced V8.0
       Download the following package: 
       Advanced – JR42732 
     * IBM Business Process Manager Advanced - Process Server V8.0
       Download the following package: 
       Advanced – JR42732 
     * IBM Business Process Manager Advanced for z/OS V8.0 
       Download the package:
       Advanced - PM66793 - Replace the WpsUpdate.py sample script to update messaging endpoint ports correctly.
       
       
       Note: A product-specific V8.0 interim fix is not needed for IBM Business Monitor V8.0 or IBM Business Monitor for z/OS V8.0.
       
    
    
 3. Download and install optional interim fixes for WebSphere Application Server. There are a number of additional interim fixes for WebSphere Application Server V8.0.0 Fix Pack 3 that are not required, but might be needed in certain situations. Download the specific interim fixes from the download package table below.  * PM58769 - This interim fix is to correct a possible performance degradation when users invoke the java.util.logging.Logger.isLoggable method. 
     * PM63369 - This interim fix is for the J2CCommandHelper not finding nested Java™ archive (JAR) files in RAR and Normal polling does not start after recovery for a FlatFile resource. 
     * PM62434 - This interim fix is for a CeaNotificationConsumer.wsdl issue without Internet access. 
     * PM63098 -This interim fix is for a UnsupportedOperationException exception during HTTP servlet filter on the z/OS operating system. 
     * PM63049 – This interim fix is for an out-of-memory error while creating profiles on Solaris non-English operating systems. 
     * PM63963 - This interim fix is to increase the maximum heap size in manageprofiles to prevent an OutOfMemoryError.
       
       
    
    
 4. Download and install the specific interim fixes from the download package table below for the IBM Business Process Manager products. These additional interim fixes are not required, but might be needed in certain situations. Download the specific interim fixes from the download package table below.  * IBM Business Process Manager Express V8.0
       Download the following packages as needed: 
       Express – JR42733 - SQL exception during server startup for Standalone profile, when DB2 version 9.5 is used. 
     * IBM Business Process Manager Standard V8.0
       Download the following packages as needed: 
       Standard – JR42733 - SQL exception during server startup for Standalone profile, when DB2 version 9.5 is used. 
     * IBM Business Process Manager Advanced V8.0
       Download the following packages as needed: 
       Advanced – JR42733 - SQL exception during server startup for Standalone profile, when DB2 version 9.5 is used.
       Advanced – JR42736 - Authentication data aliases for the SCA and CEI Messaging engines are missing for MSSQL Server – corequisite with JR42749
       Advanced – JR42749 - Authentication data aliases for the SCA and CEI Messaging engines are missing for MSSQL Server – corequisite with JR42736 
     * IBM Business Process Manager Advanced - Process Server V8.0
       Download the following packages as needed: 
       Advanced – JR42733 - SQL exception during server startup for Standalone profile, when DB2 version 9.5 is used.
       Advanced – JR42736 - Authentication data aliases for the SCA and CEI Messaging engines are missing for MSSQL Server – corequisite with JR42749
       Advanced – JR42749 - Authentication data aliases for the SCA and CEI Messaging engines are missing for MSSQL Server – corequisite with JR42736
