CAUSE
The jndi Datasource has not be created.



RESOLVING THE PROBLEM
Go back to Configuration Manager, re-save and validate the parameters under Email Services -> Configuration Web Service. If you have the parameters filled in, the save button will be greyed out. Re-enter the values and that should allow you to save again.
