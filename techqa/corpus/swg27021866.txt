1For code page 943 there are two DB2-specific ICU converters: 

 * The default converter is installed when you install your Optim solution. this converter represents the code points reflected in the default DB2 server/client installation.
 * An alternate converter represents the code points used when the alternate CP 943 converter is used within DB2 server/client.

The commands to use the alternate CP 943 ICU converter are:

cd rt
cd bin
copy opmicuda.dat opmicud2.dat
copy opmicuda.als opmicud2.als

The trace entry reflects the use of the db2luw-943-alt converter.

Support for multibyte characters in DB2 for z/OS has the following limitations: 

 * DB2 for Linux, UNIX, and Windows v9.7 FixPack 5 or higher is required on the Optim workstation or server.
 * Processing of CLOB and DBCLOB data is not supported in an ASCII or Unicode tablespace.
