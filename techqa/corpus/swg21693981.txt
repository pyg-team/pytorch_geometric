Title: IBM Security Bulletin: TLS padding vulnerability affects IBM HTTP Server - United States

Text:
padding; vulnerability; CVE-2014-3566 SECURITY BULLETIN

SUMMARY
 TLS padding vulnerability affects IBM HTTP Server. 

VULNERABILITY DETAILS
Review security bulletin Security Bulletin: Vulnerability in SSLv3 affects IBM HTTP Server (CVE-2014-3566) [http://www.ibm.com/support/docview.wss?uid=swg21687172] for vulnerability details and information about fixes.

AFFECTED PRODUCTS AND VERSIONS
 

Principal Product and Version(s) Affected Supporting Product and Version IBM Service Delivery Manager V7.2.4 

IBM HTTP Server 6.1 and 7.0 IBM Service Delivery Manager V7.2.2 IBM Service Delivery Manager V7.2.1 WORKAROUNDS AND MITIGATIONS
None
