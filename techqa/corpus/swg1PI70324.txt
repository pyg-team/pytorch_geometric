ERROR DESCRIPTION
 *  Files can be left in the var/temp folder, particularly if the
   server is restarted while downloading files. The UCD server
   should clean up files left in this directory on startup.
   
   
    
   
   

LOCAL FIX
 *  While the server is offline, remove all files from this
   directory.
