Title: IBM Security Bulletin: A security vulnerability has been identified in IBM WebSphere Application Server shipped with Tivoli Netcool/OMNIbus WebGUI (CVE-2017-1381) - United States

Text:
 SECURITY BULLETIN

SUMMARY
 Websphere Application Server (WAS) is shipped as a component of Tivoli Netcool/OMNIbus WebGUI. Information about a security vulnerability affecting WAS has been published in a security bulletin. 

VULNERABILITY DETAILS
Please consult the security bulletin Information disclosure in WebSphere Application Server (CVE-2017-1381) [http://www.ibm.com/support/docview.wss?uid=swg22004792] for vulnerability details and information about fixes.

AFFECTED PRODUCTS AND VERSIONS
 

Principal Product and Version(s) Affected Supporting Product and Version WebGUI 7.4.0 GA and FP embedded Websphere Application Server 7.0 WebGUI 8.1.0 GA and FP Websphere Application Server 8.5 GET NOTIFIED ABOUT FUTURE SECURITY BULLETINS
