 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  Generation of Atlas Collection Logs are prone to fail when
   custom attributes are defined but are set to null. The message:
   Error: "A system error has occurred
   on the client.  Contact your System Admin for assistance."
   appears in the Atlas UI, but error messages are not
   consistently printed in the logs.
   
   
    
   
   

LOCAL FIX
 *  No known workaround
   
   
    
   
   

PROBLEM SUMMARY
 *  When attempting to view a Collection Log through the custodian
   view, an error is populated.  Error: "A system error has
   occurred on the client.  Contact your System Admin for
   assistance."
   
    When selecting OK, some custom fields are not viewable
   
   
    
   
   

PROBLEM CONCLUSION
 *  The fix for this problem is available now from Fix Central as
   part of IBM Atlas 6.0.3 fixpack 2.
