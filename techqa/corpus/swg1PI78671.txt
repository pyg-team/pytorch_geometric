ERROR DESCRIPTION
 *  The Rhapsody 8.2 Ifix2 update requires a manual installation to
   complete the Eclipse 64bit platform integration plugins upgrade.
   However, once this is completed the plugin no longer loads
   correctly in the Eclipse 64bit client. Looking at the Eclipse
   workspace log file, there is an error:
   
   - Exception in
   com.ibm.rational.rhapsody.platformintegration.ui.PlatformIntegra
   tionPlugin.start()
   
   followed by
   
   - rhapsody.dll: Can't load IA 32-bit .dll on a AMD 64-bit
   platform
   
   STEPS TO REPRODUCE:
   
   1) Install Rhapsody 8.2 64bit #build 9794446
   2) Install Eclipse Mars 4.5.x and install the Rhapsody Eclipse
   plugin from Rhapsody installation
   3) Create a Rhapsody project in Eclipse and visually confirm the
   Rhapsody Modelling perspective loads.
   4) Download Rhapsody82iFix002_x64.zip from fixcentral and
   install following the readme instructions.
   5) Update the Rhapsody plugin in Eclipse 64bit client via the
   Help menu > Install new software.
   6) Rhapsody Modelling perspective no longer loads in Eclipse
   64bit client.
