APAR STATUS
 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  Configure the agent using IBMMON_AGENT user
   Start the agent
   Under PI node, check whether data is reported for XML messages
   logs workspace for attribute "XML Message Monitoring".
   No XML message logs are shown
   
   
    
   
   

LOCAL FIX
 *  Workaround is to add authorization to IBMMON/AUTH profile
   
   
    
   
   

PROBLEM SUMMARY
 *  Additional Information: The issue is
   related to SAP agent 7.1. XML Messages not showing up on Tivoli
   Enterprise Portal workspaces S_XMB_MONI authorization object
   need to be added to see the data of XML messages monitoring.
   This is fixed in this release.
   
   
    
   
   

PROBLEM CONCLUSION
 *  The formal fix for this APAR is contained in the following
   package:
   IBM(R) Tivoli(R) Composite Application Manager Agent for SAP
   Applications,
    Version 7.1.0
   Component ID: 5724B97SO
   
   Interim Fix: 0001, 7.1.0-TIV-ITM_SAP-IF0001
   
   Date: October, 2012
   
   
    
   
   

TEMPORARY FIX
 *  Authorization object "SXMB_MONI" is required for XI Message
   Monitoring.
