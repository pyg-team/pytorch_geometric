ABSTRACT
 This patch provides an update to the for IBM Tivoli Monitoring (ITM) portal server component. 

DOWNLOAD DESCRIPTION
This fix upgrades the embedded WebSphere Application Server (eWAS) which is shipped as part of the IBM Tivoli Monitoring portal server, to version 8.0.0.11 plus additional Interim Fixes referred to as Interim Fix Block 3. The Interim Fixes included as part of Interim Fix Block 3 include:


 * PI61190 - upgrades eWAS IBM SDK, Java Technology Edition, Version 6R1 Service Refresh 8 [http://www-01.ibm.com/support/docview.wss?uid=swg24033359]Fix Pack 25 
 * PI56811 
 * PI56190 
 * PI55780 - included in Interim Fix Block 2 (8.00.11.02) 
 * PI45266 - included in Interim Fix Block 1 (8.00.11.01) 
 * PI52103 - included in Interim Fix Block 1 (8.00.11.01)
