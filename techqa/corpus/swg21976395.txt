Title: IBM Additional MIBs for resource monitoring on the Network IPS (GX) - United States

Text:
MIB; performance; hardware; monitoring; SNMP TECHNOTE (FAQ)

QUESTION
 Are there any MIB files available for resource monitoring on the Network IPS (GX)? 

ANSWER
For monitoring security event information, you need to use the iss.mib file. This can be found in Technote 1639239: ISS.mib file download [http://www.ibm.com/support/docview.wss?uid=swg21639239].

For monitoring other resources on the GX (such as load average, disk usage, memory usage, RAID status, and so on), you can use the UCD-SNMP-MIB file. You can find a copy of this file in /usr/share/snmp/mibs/ on the appliance. 

In the same directory, you see several MIB files available. The most commonly used files are listed below with a brief description of what it can be used monitor:


 * UCD-SNMP-MIB.txt: Load average, disk usage, memory usage, RAID status, and so on 
 * IF-MIB.txt: Management and kill ports 
 * NET-SNMP-MIB.txt: System information 
 * NET-SNMP-MIB-AGENT.txt: System information 
 * HOST-RESOURCES-MIB.txt: Host resources


For information about configuring SNMP on the GX, see Configuring SNMP [http://www.ibm.com/support/knowledgecenter/SSB2MG_4.6.2/com.ibm.ips.doc/tasks/configuring_snmpv3.htm] on the IBM Knowledge Center.
