   Problem Description: some of the SAP ABAP syslog entries are
   truncated in both situation event and TEP workspace view.
   For example a string that should look like:
   "Run-time error "RAISE_EXCEPTION" occurred "
   is instead showed as:
   "Run-time error "" occurred"
   Running se37 and passing function module /IBMMON/ITM_SYS_LOG,
   we can see that the message is already truncated in the output
   of this function module.
   Detailed Recreation Procedure:
   1. Import 711 FP2 transport on SAP system with SAP basis 700
   SP32
   2. Syslog entries of raise exceptions gets truncted in the
   output of  FM '/IBMMON/ITM_SYS_LOG'
