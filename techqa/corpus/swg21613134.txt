Title: IBM Netcool/OMNIbus Integrations Release Notice - Netcool/OMNIbus Gateway for Tivoli EIF (nco-g-tivoli-eif-5_0) - United States

release notice; downloads; Gateway for Tivoli EIF; nco-g-tivoli-eif NEWS

ABSTRACT
 A new version of the Netcool/OMNIbus Gateway for Tivoli EIF will be available to download from August 6, 2015. 

CONTENT
 

The following properties have been added with this release:

 * Gate.TIVOLI_EIF.LastStatechangeFile
 * Gate.TIVOLI_EIF.MaxEventSize
 * Gate.TIVOLI_EIF.MaxIDListSize
 * Gate.TIVOLI_EIF.MinimumSeverity

For details of these new properties, refer to the README supplied in the package and to the following technote:

 www-01.ibm.com/support/docview.wss?uid=swg21692784 [http://www-01.ibm.com/support/docview.wss?uid=swg21692784] .

Netcool/OMNIbus V8.1

If you are running Netcool/OMNIbus V8.1, a single package contains versions of the gateway for all supported Operating Systems. To obtain this new package, use the Find by part number field to search for part number: CN6I2EN

 Note: The Netcool/OMNIbus V8.1 version of this gateway supports the following Operating Systems: AIX, Linux, Solaris, Windows, and zLinux. It also provides 64-bit support.for AIX, Linux, Solaris, and zLinux. 

Netcool/OMNIbus V7.3.0, 7.3.1 or 7.4.0

If you are running Netcool/OMNIbus V7.3.0, 7.3.1 or 7.4.0, a separate package is supplied for each Operating System. To obtain this new package, use the Find by part number field to search for the part number that corresponds to your Operating System as shown in the table below.
