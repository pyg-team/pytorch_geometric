Title: IBM Datacap installation throws error: The Rulerunner XML file is invalid or could not be read - United States

Text:
 TECHNOTE (TROUBLESHOOTING)

PROBLEM(ABSTRACT)
 After a fresh installation of Datacap, when trying to connect Rulerunner to the Datacap server, Datacap displays error; "The Rulerunner XML file is invalid or could not be read". 

Note that the connection still works correctly after getting this error. 

SYMPTOM
When trying to connect Rulerunner to Datacap server, the following exception is seen:

[/support/docview.wss?uid=swg21987149&aid=1] [/support/docview.wss?uid=swg21987149&aid=1]

The windows event logs also show an error (2656):

[/support/docview.wss?uid=swg21987149&aid=2] [/support/docview.wss?uid=swg21987149&aid=2]





CAUSE
The XML file got corrupted at some point in the installation.

RESOLVING THE PROBLEM
Delete the generated "Rulerunner.xml" file and re-run the Rulerunner installation process that generates the file and this should resolve the issue.
