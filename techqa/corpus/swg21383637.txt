 TECHNOTE (FAQ)

QUESTION
 Where is the Search and Indexing API (SIAPI) JavaDoc download location for IBM WebSphere Portal 6.1? 

ANSWER
The download location for the SIAPI JavaDoc for WebSphere Portal 6.1 is the same as for WebSphere Portal 6.0 and can be found in the following document: IBM Search and Indexing API (SIAPI) [http://www.ibm.com/developerworks/websphere/library/specs/0511_portal-siapi/0511_portal-siapi.html].
