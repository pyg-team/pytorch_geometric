ABSTRACT
 The Netcool/OMNIbus Probe for Alcatel-Lucent 5620 SAM v13 JMS will be available to download from July 28, 2016. 

CONTENT
 

This version of the probe has been updated to add the following functionality:

 * RFE 62030: Improved support for Alcatel-Lucent SAM server redundancy. Added support for the ability to configure a single probe instance with the option to failover between the primary server and the backup server.
 * RFE 72415: Provide event enrichment rules to enable ITNM RCA function with OMNIbus.

This probe is written to support Alcatel-Lucent 5620 SAM v13 and v14.

Note: This probe is not backwardly compatible to previous versions of Alcatel-Lucent 5620 SAM. For the Alcatel-Lucent 5620 SAM V9.0, V10.0, V11.0 and V12.0, use the Netcool/OMNIbus Probe for Alcatel-Lucent 5620 SAM v10 JMS. For download details of that probe, see the following Release Notice:
