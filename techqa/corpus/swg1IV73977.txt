ERROR DESCRIPTION
 *  Problem Description:
   In ITIM SSUI Change Password page, since the submit button
   ("OK") is not disabled on time, many requests can get submitted
   if the user clicks on the OK button repeatedly, which is
   causing CTGIMV405E error on UI.
   
   Steps to duplicate :
   Try to Change Password in ITIM SSUI with quick repeated clicks
   on OK button.
   
   Desired Behavior :
   Prevent proceeding the second and further requests, regardless
   of requests being successful or not, by disabling submit button
   ("OK") for subsequent clicks.
