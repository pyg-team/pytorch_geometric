ERROR DESCRIPTION
 *  Experiencing inconsistencies in the data that Impact is
   returning
   from a GetByFilter command versus what is actually in the
   database. This happens when caching is ON.
   
   
    
   
   

LOCAL FIX
 *  One of the Key Selected On the Datatype must be the Primary key
   on the Original Table in the Databse or a CAndidate Key the Idea
   is to make the row Unique.  Also the most used Field in the
   Table,  if primary key not set on db
