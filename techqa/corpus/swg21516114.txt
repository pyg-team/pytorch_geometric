Title: IBM Is there a Limit in DataStage for the total number of columns that can be used for the Basic Transformer Stage? - United States

Text:
 TECHNOTE (FAQ)

QUESTION
 Is there a Limit in DataStage for the total number of columns that can be used for the Basic Transformer Stage? 

ANSWER
Yes, there is a limit on the total number of columns that can be used in a Basic Transformer Stage. The number of columns for each input and output link and the number of stage variables must total less than 8192.
