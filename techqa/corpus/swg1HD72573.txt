ERROR DESCRIPTION
 *  Drafting (IntDraft) / Wrong representation of the
   Font size from Tools > Standards.
   .
   Scenario:
   1. Open CATIA.
   2. Go to Tools > Standards > Drafting > ISO >
   Styles > Length / Distance Dimension > Default
   > Font >  & change these settings (Font size: 14,
   Bold: Yes, Underline: Yes 7 Colour: Green).
   3. Go to part design.
   4. Create a Pad.
   5. Go to Drafting (with the changed standard).
   6. Create Front view.
   7. Create Dimension.
   8. Go to Tools > Options > General > Parameters
   & Measures > Units & change the setting to Inch.
   9. Go to Drafting (with the same changed standard).
   10. Create front view & create dimension.
   .
   Problem: The font size is different.
   .
   Expected result: It should be as per the standard.
   .
   .
