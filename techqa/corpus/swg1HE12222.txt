APAR STATUS
 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  With IBM Content Collector Content P8 Search Services Support
   Version 4.0.0.2, some Microsoft Exchange (MSG) emails cannot be
   indexed
   due to an 'ArrayIndexOutOfBoundsException' exception. The
   following error message is logged:
   
   AFUI0043E  The document could not be processed as a regular
   email.
   
   
    
   
   

LOCAL FIX
 *  n/a
   
   
    
   
   

PROBLEM SUMMARY
 *  When you use IBM Content Collector Content P8 Search Services
   Support Version 4.0.0.2, some Microsoft Exchange(MSG) emails
   cannot be indexed and an error message is logged.
   The error is caused by a parsing error in the email converter.
   
   
    
   
   

PROBLEM CONCLUSION
 *  Starting with IBM Content Collector 4.0 Fix Pack 3, the IBM
   Content Collector P8 Content Search Service Support is updated
   so that the email converter can address the parsing error.
