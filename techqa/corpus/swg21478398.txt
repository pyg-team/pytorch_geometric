Title: IBM Window Installer error opening installation log file. Verify that the specified log file location exists and is writable - United States

Text:
 TECHNOTE (TROUBLESHOOTING)

PROBLEM(ABSTRACT)
 I have gone to Add / Remove program and remove SPSS 16 but the 16.0.2 still list in the removed program and can't be removed It gives me the Window Installer error: " Error opening installation log file. Verify that the specified log file location exists and is writable." When I am trying to reinstall my SPSS16, I don't know where it tries to locate the log file. How can I fix this error so I can reinstall my SPSS again? 

RESOLVING THE PROBLEM
This error could cause by an SPSS registry which has not been removed completely. Verify the registry location below and remove it based on your version. 

HKEY_LOCAL_MACHINE\SOFTWARE\Classes\Installer\Products
HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows\CurrentVersion\Uninstall

Do the reinstall from CD again.

2. If this still give you the same error, Download the Windows installer cleanup to help with uninstall and make sure you have the current update of Windows installer on your computer.

This can be found at the following URL and on the SPSS for Windows installation CDROM
http://support.microsoft.com/kb/290301 [http://support.microsoft.com/kb/290301]

3. Copy the SPSS contents from CD to your local drive (e.g. C:) and create a log.txt inside the folder. Then find the SPSS 16 for Windows Installer and run it
