 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  Using the limited Job Monitor view displayed by an Operations
   shortcut in Taskmaster Web, it is possible to view and (attempt
   to) run batches associated with job-task pairings not defined in
   the shortcut's permissions.
   
   Changing a non-status filter will show both pending and hold
   batches for a shortcut whose mode is Manual for Hold.
