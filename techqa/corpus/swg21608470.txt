Title: IBM Configuring floating license for Rational Business Developer from Installation Manager results in "The server is available, but does not serve license keys" error - United States

Text:
Rbd; license error; There are no licenses on the server; rbd license; business developer license issue; rational business developer license not available; rbd license error; does not serve the license; business developer; Installation Manager; IM; Server cannot be reached; not serving license keys TECHNOTE (TROUBLESHOOTING)

PROBLEM(ABSTRACT)
 Attempts to configure IBM Rational Business Developer with floating license from IBM Installation Manager results in the error "The server is available, but does not serve license keys". 

SYMPTOM
Steps to reproduce: 

 1. Launch IBM Installation Manager from Start > All Programs > IBM Installation Manager > IBM Installation Manager > Manage Licenses. 
    
 2. Select IBM Rational Business Developer from the product list
    
 3. Check the radio button for Configure Floating License Support. 
    
 4. Click Next
    
 5. Accept the terms and conditions
    
 6. Error message would be noticed when clicking on Test Connection while adding the license server information 


CAUSE
IBM Rational Business Developer package installed, is downloaded from the bundle name: IBM Rational Developer for i for SOA Construction Activation and Rational Business Developer V7.5.1 and WAS Windows Multilingual eAssembly which contains the packages below. 


 * IBM Rational Business Developer V7.5.1 
 * IBM Rational Developer for i for SOA Construction V7.5 Activation Kit Windows 
 * IBM Rational Application Developer for WebSphere V7.5 - WebSphere Application Server Test Environment V6.0 
 * IBM Rational Application Developer for WebSphere V7.5 - WebSphere Application Server Test Environment V6.1 
 * IBM Rational Application Developer for WebSphere V7.5 - WebSphere Application Server Test Environment V7.0

RESOLVING THE PROBLEM
Procedure:


 1. Navigate to IBM Rational License Key Center [http://www-01.ibm.com/software/rational/support/licensing/] website. 
    
    
 2. Return the existing license file and regenerate the license [https://www-304.ibm.com/support/docview.wss?uid=swg21308308] file for IBM Rational Test Developer on System i for SOA Floating License Key.
    
    Notes:  * In case the software downloaded is the base IBM Rational Business Developer bundle and the error shows up, please raise a PMR with the IBM Rational licensing support. 
       
     * The IBM Service Request (SR) application can be used to raise a PMR.
