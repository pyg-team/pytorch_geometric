ERROR DESCRIPTION
 *  In the Search Results window, clicking on the Search Results
   only sorts the objects currently visible in the Search
   Results window.
   Please see the attached screenshot.  In this example, there
   are values in the ECO Type column in the returned results,
   just not visible in the current Search Results window.
   Clicking on the ECO Type column produces no change and leads
   the user to the conclusion that this field is empty for all
   the returned objects.
   Clicking on field that does display data results in a screen
   change with an incorrect result as the sort does not reflect
   all objects returned by the search.
