Title: IBM Security Bulletin: A vulnerability in the GSKit component of IBM Mobile Connect (CVE-2016-0201) - United States

Text:
 SECURITY BULLETIN

SUMMARY
 A vulnerability has been addressed in the GSKit component of IBM Mobile Connect. 

VULNERABILITY DETAILS

CVEID: CVE-2016-0201 [http://cve.mitre.org/cgi-bin/cvename.cgi?name=CVE-2016-0201]
[http://cve.mitre.org/cgi-bin/cvename.cgi?name=CVE-2016-0201]
DESCRIPTION: IBM GSKit could allow a remote attacker to obtain sensitive information, caused by a MD5 collision. An attacker could exploit this vulnerability to obtain authentication credentials.

AFFECTED PRODUCTS AND VERSIONS
 
IBM Mobile Connect: 6.1.5.2 

REMEDIATION/FIXES
This vulnerability is tracked as APAR IV80207. To obtain a fix for this issue, refer to technote 7009682 and download build level 20160113 or later. 

 * Recommended Maintenance: IBM Mobile Connect [http://www-01.ibm.com/support/docview.wss?uid=swg27009682] (technote 7009682)

WORKAROUNDS AND MITIGATIONS
Disable MD5 based TLS 1.2 ciphers.
