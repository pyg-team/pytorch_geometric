ERROR DESCRIPTION
 *  Error Description&#65496;
   Domino 853 server
   Mail file mail8.ntf design
   both forms8.nsf and forms85.nsf on server
   Internet Explorer 8.0
   
   Steps to reproduce
   
   1. Using Internet Explorer 8.0 user opens their 8.x design
   iNotes mail file on an 8.5.3 server
   2. User creates a new email
   3. User clicks the 'To' button on the email to launch the
   address picker
   4. User selects a directory and then tries to use the vertical
   scroll bar
   
   Result: The scroll bar flickers but does not move.
   
   Issue is seen in both regular and compatibility modes of
   Internet Explorer 8.0
   
   Issue is not seen if using Internet Explorer 7.0 or 9.0,
   FireFox or Safari browsers
   
   
    
   
   

LOCAL FIX
 *  Local Fix&#65496;
   Arrow keys can be used to scroll through user list
