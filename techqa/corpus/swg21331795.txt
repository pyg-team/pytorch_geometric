Title: IBM How to Restore the Administration NewsBox in Upfront - United States

Text:
 TECHNOTE (TROUBLESHOOTING)

PROBLEM(ABSTRACT)
 The Administration NewsBox has been deleted. Is there a way to restore it? 

RESOLVING THE PROBLEM
When Upfront is initially installed, there are a few unpacks that are executed behind the scenes. These unpacks can be used in this situation to restore this NewsBox.Follow the steps below to restore the NewsBox.

1) Ensure the Upfront Datastore service is started.
2) Stop the Upfront Dispatcher and Administrator services.
3) Go to a command prompt;
4) Navigate to; X:\Program Files\Cognos\cer1\bin
5) Back up Upfront by typing; upfpack
6) You will be prompted for the Administrator's UserID and Password
7) Ensure that a file named UpfPackage.pkg was created -> Check the date and time on the file
8) Run the following:

upfunpack.exe -p StartingThemesLanguages.pkg -r UpfUnpackRulesInstall1.txt -un Administrator -pw <Administrator's password> -ns <namespace> -del

9) Once the above command finishes run:

upfunpack.exe -p StartingNewsIndex.pkg -r UpfUnpackRulesInstall2.txt -un Administrator -pw <Administrator's password> -ns <namespace>

10) Restart all Upfront services
11) The Administration NewsBox should now be accessible from Upfront.
