Title: IBM ITM Agents which run on i5/OS on "IBM System i"  cannot be started after Sep 6th, 2015. - United States

Text:
Agents; i5/OS; IBM System i; Sep 6th 2015; 2015/09/06; MQ FLASH (ALERT)

ABSTRACT
 The ITM i5 OS Agent, the ITM MQ agent and the ITM MC Agent running on "IBM System i" cannot be started after Sep 6th, 2015. 

Please make NOTE of the following: 

All other ITM agents running on other OS platforms are NOT AFFECTED by this problem. 




CONTENT
ITM Agents which run on i5/OS on "IBM System i" cannot be started after Sep 6th, 2015.
Please do not restart the agents until the problem is resolved !
The agents can be started if the date of the Operating System is changed to before Sep 5th, 2015.

Here are the APAR's opened

APAR for i5 agent is IV76801
APAR for MQ agent is IV76825 

FIX Delivery Mechanism: We will use this FLASH to post the necessary files for customers to patch their agents. This allows for customers to pull the files without the need for opening PMRs.
