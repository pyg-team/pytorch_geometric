APAR STATUS
 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  WSRR Studio cannot connect to servers that require TLS V1.1 or
   V1.2 connections.
   
   
    
   
   

LOCAL FIX

PROBLEM SUMMARY
 *  ****************************************************************
   USERS AFFECTED:
   All users of IBM WebSphere Service Registry and Repository
   Studio V8.5
   
   ****************************************************************
   PROBLEM DESCRIPTION:
   WSRR Studio does not support connections using the TLS V1.1 nor
   V1.2 protocols.
   
   
    
   
   

PROBLEM CONCLUSION
 *  WSRR Studio was updated to support TLS V1.1 and V1.2.
   
   This fix is targeted for inclusion in WSRR Studio V8.5.6.1
