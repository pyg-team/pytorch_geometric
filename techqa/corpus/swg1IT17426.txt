APAR STATUS
 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  A JDBC SQL "SELECT" call from the Mapping node fails with a Java
   NullPointer exception when it accesses a table in an iSeries
   (AS400) database where a column with datatype "DATE"  contains a
   default value "0001-01-01"
   
   
    
   
   

LOCAL FIX
 *  Populate the DATE type column with a valid non default value.
   
   
    
   
   

PROBLEM SUMMARY
 *  ****************************************************************
   USERS AFFECTED:
   All users of the IBM Integration Bus Mapping Node making an SQL
   Selection from an AS400 database table which has a column with
   data type "DATE" that contains a default value.
   
   
   Platforms affected:
   z/OS, MultiPlatform
