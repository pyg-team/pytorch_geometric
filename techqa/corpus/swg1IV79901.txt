ERROR DESCRIPTION
 *  Data Power Data Collector Session Threads might hang whilst
   communicating with the Data Power Appliance.
   This is caused by a deadlock in the configuration updating
   functions. Note: This is similar to APAR IV79263 [http://www-01.ibm.com/support/docview.wss?uid=swg1IV79263] - Data Power
   Data Collector may hang whilst updating configuration (90030);
   however it is a different deadlock that requires fixing.
