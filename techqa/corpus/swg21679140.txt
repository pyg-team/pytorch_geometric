Title: IBM An export file creation failure message is seen when attempting to export a snapshot from IBM Business Process Manager (BPM) - United States

Text:
export; failed; file export; process designer; PD; error; export file; cannot export TECHNOTE (TROUBLESHOOTING)

PROBLEM(ABSTRACT)
 When you attempt to export a snapshot from IBM Business Process Manager, the following error message is displayed: "java.lang.Exception: Export file cannot be created" 

RESOLVING THE PROBLEM
To resolve the problem, make sure there is enough free temporary space available in the file system.
