Title: IBM Command line interface in TIP to create new pages - United States

Text:
TIPL2FAQ; Tivoli; integrated; Portal; manual; script; creating; pages; navigatio; bar; TIP TECHNOTE (FAQ)

QUESTION
 Is it possible to create pages and the corresponding navigation bar in Tivoli Integrated Portal thru a command line interface? 

CAUSE
The requirement is to auto generate by means of command line scripts a page and corresponding entry in the navigation bar of Tivoli Integrated Portal. Currently creating the page manually can be done via the TIP UI.

ANSWER
Currently it is not possible to auto generate a page and the corresponding entry in the navigation bar of the portal by means of a command line script.
