Title: IBM Security Bulletin: A security vulnerability has been identified in an IBM Tivoli Monitoring shared component shipped with Agent for NetApp Storage (CVE-2015-4000) - United States

Text:
 SECURITY BULLETIN

SUMMARY
 An IBM Tivoli Monitoring shared component is included as part of Agent for NetApp Storage. Information about a security vulnerability affecting an IBM Tivoli Monitoring shared component has been published in a security bulletin. 

VULNERABILITY DETAILS
Please consult the security bulletin for vulnerability details and information about fixes: 
CVE-2015-4000: http://www.ibm.com/support/docview.wss?uid=swg24040390 [http://www.ibm.com/support/docview.wss?uid=swg24040390]
CVE-2015-4000: http://www.ibm.com/support/docview.wss?uid=swg24040391 [http://www.ibm.com/support/docview.wss?uid=swg24040391]

AFFECTED PRODUCTS AND VERSIONS
 

Principal Product and Version(s) Affected IBM Tivoli Monitoring Version(s) IBM Tivoli Monitoring for Virtual Environments Agent for NetApp Storage Versions 7.1.0 and 7.2.0 IBM Tivoli Monitoring Versions 6.2.0 and 6.2.1 GET NOTIFIED ABOUT FUTURE SECURITY BULLETINS
