PROBLEM SUMMARY
 *  WebSphere Transformation Extender DB2 adapter customers that are
   submitting multiple data rows for table or stored procedure
   insertion will notice extraneous trace file information for each
   successful insert operation.
   
   Problem Description:
   Redundant trace information appears in the WTX database adapter
   trace file.
   
   Platforms Affected:
   All
   
   
    
   
   

PROBLEM CONCLUSION
 *  Redundant column information was removed from the trace file,
   especially during multiple executions of the same stored
   procedure.  Also, completion results for each individual row
   that was successfully inserted has been removed, which is all
   consistent with prior releases of the WTX database adapter trace
   file.
   
   Delivered in:
   8.3.0.3
