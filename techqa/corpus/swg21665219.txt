ABSTRACT
 A new version of the Netcool/OMNIbus Gateway for Message Bus will be available to download from April 12, 2018. The gateway has added support to publish events to the Kafka cluster as a Kafka client. 

CONTENT
 

This version of the Message Bus Gateway contains the following updates:

 * RFE 61812: Added support for the gateway to integrate as a Kafka client 0.10.0 event producer.
 * Upgraded support for Transport Module (common-transportmodule-18_0) integration. 
 * Added enhancement to the SCALA transport.
 * Added enhancement to the HTTP transport.
