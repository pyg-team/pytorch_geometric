Title: IBM Installation Manager version 1.7.1 crashes with JVM Error when running WebSphere Portal rollback in GUI Mode - United States

Text:
 TECHNOTE (TROUBLESHOOTING)

PROBLEM
Install WebSphere Portal version 8.5 and upgrade to the latest cumulative fix. Start IIM version 1.7.1. Select Rollback to rollback the latest cumulative fix. Select Package Group Name: IBM WebSphere Portal Server V85. Click Next. Select the top level check box for IBM WebSphere Portal Server V8.5. Click Next. The window begins to paint but then IIM exits. A JMV terminated window displays. Output is written to the console window where you launched IIM and dump files are written to the IIM/eclipse directory.

SYMPTOM
The IIM version 1.7.1 JVM crashes in GUI mode after you select the packages for rollback.


RESOLVING THE PROBLEM
Upgrade IIM to version 1.7.3. Go to the Related URL section for the links to the instructions. 


After you upgrade IIM, use the GUI to retry the rollback.
