 SECURITY BULLETIN

SUMMARY
 OpenSSL Heartbleed Vulnerability - April 2014 

VULNERABILITY DETAILS
The Heartbleed bug is a worldwide security vulnerability affecting services with SSL/TLS encryption using the OpenSSL library. We have reviewed all the systems hosting IBM Algo Risk Content on Cloud (RCoC) and would like to update you on the following actions that have been taken: 

All RCoC systems have been patched. All IBM-owned SSL Certificates have been revoked and new certificates generated and installed. 
Steps you should take: You should change your Algo Risk Content on Cloud passwords immediately.

Additional information regarding this vulnerability can be found at the following links: http://heartbleed.com/ [http://heartbleed.com/] CVEID: CVE-2014-0160 [http://cve.mitre.org/cgi-bin/cvename.cgi?name=CVE-2014-0160] [http://cve.mitre.org/cgi-bin/cvename.cgi?name=CVE-2013-4353] X-Force: https://exchange.xforce.ibmcloud.com/vulnerabilities/92322 [https://exchange.xforce.ibmcloud.com/vulnerabilities/92322] IBM Product Security Incident Response Blog [https://www.ibm.com/blogs/PSIRT]

IBM appreciates your cooperation regarding this matter. Should you have any further questions, please contact RCoC Support at first_support@vnet.ibm.com [first_support@vnet.ibm.com/].
