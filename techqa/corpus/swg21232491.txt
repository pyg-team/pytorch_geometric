Title: IBM DISPLAY SMS,CICSVR,LOGSTREAMS(ALL) does not display all CICS logstreams - United States

Text:
DISPLAY SMS CICSVR LOGSTREAMS ALL CICSVR LOGSTREAM IEE932I IEE932 periods missing ; not displaying; CICS VR 4.2 ; R420 R42E R42K FMID HCCV420 JCCV42E JCCV42K CICS VR 4.3 ; 5655P3000 R430 R43E R43K 5655-P30 5655P30 FMID HCCV430 JCCV43E JCCV43K CICS VR 5.1; 5655Y2400 R510 R51C R51E R51K 5655-Y24 5655Y24 FMID HCCV510 JCCV51E JCCV51K TECHNOTE (TROUBLESHOOTING)

PROBLEM(ABSTRACT)
 You display a logstream by entering command /D SMS,CICSVR,LOGSTREAMS(CICS.DFHJ01). The information is displayed as periods for the specific logstream name. When you enter command /D SMS,CICSVR,LOGSTREAMS(ALL) to display all logstreams, the CICS online logstream is no longer displayed. 

SYMPTOM
You enter the following command to display logstream CICS.DFHJ01 and periods display to the right of it the IEE932I message output. 

D SMS,CICSVR,LOGSTREAMS(CICS.DFHJ01) 

IEE932I 155
DWW020I DISPLAY SMS,CICSVR
DISPLAY SMS,CICSVR - LOG STREAM CONNECTION STATUS 
SYSNAME 00000000011111111112222222222333
IDENTIFIER 12345678901234567890123456789012
. 
CICS.DFHJ01 ................................
. 
01 SYSNAME: Z3
02 SYSNAME: Z2
03 SYSNAME: Z1 


You enter the following command to display all logstreams but CICS.DFHJ01 is no longer displayed in the IEE932I message output. 

D SMS,CICSVR,LOGSTREAMS(ALL)

IEE932I 164
DWW020I DISPLAY SMS,CICSVR
DISPLAY SMS,CICSVR - LOG STREAM CONNECTION STATUS
SYSNAME 00000000011111111112222222222333
IDENTIFIER 12345678901234567890123456789012
. 
.
01 SYSNAME: Z3
02 SYSNAME: Z2
03 SYSNAME: Z1

CAUSE
Batch logging is not being used. The CICS VSAM Recovery (CICSVR) program, DWW3ODIS, displays logstreams anchored off the RMGS control structure. DWW3GCON Build_RMGS routine adds logstreams to the RMGS structure during logstream connect processing. This activity only occurs for CICSVR batch logging activity.



RESOLVING THE PROBLEM
Use the DISPLAY SMS,CICSVR,LOGSTREAM(logstream) command to display the status of logstreams used by CICSVR batch logging. It is not intended for displaying CICS online logstreams. Reference section Displaying the log stream status [http://pic.dhe.ibm.com/infocenter/cicsts/v5r1/topic/com.ibm.cics.cvr.doc/dwwig/dwwig00075.html] in the CICS information center.

You can use the DISPLAY LOGGER command to display the status of the system logger, individual log streams, or one or all log streams from a sysplex. For more details see topic Displaying the System Logger and its Log Streams [http://publib.boulder.ibm.com/infocenter/zos/v1r13/topic/com.ibm.zos.r13.ieag100/d3slog.htm?path=29_51_6_9_41#d3slog] in the MVS System Commands manual.
