Title: IBM How to enable/disable components of front end load balancer by RESTAPI - United States

Text:
REST appliance LMI TECHNOTE (FAQ)

QUESTION
 I would like to control front load balancer from batch process on remote machine. How to enable/disable each compoent ? 

ANSWER
Following are example of RESTAPI call for each components.
At the end of operation, "5. Apply changes" step is required.

1. Get configuration
# curl -k --user admin:<pass> -H "accept: application/json" https://<appliance>/isam/felb

2. Disable FELB
# echo '{"enabled":false}' >tmp.tmp
# curl -k --user admin:<pass> -H "accept: application/json" -T tmp.tmp https://<appliance>/isam/felb/configuration

3. Disable a virtual server (my-server)
# echo '{"name":"my-server","enabled":false}' >tmp.tmp
# curl -k --user admin:<pass> -H "accept: application/json" -T tmp.tmp https://<appliance>/isam/felb/configuration/services/my-server

4. Disable a real server in the virtual server (192.168.0.1:81)
# echo '{"address":"192.168.0.1","port":"81","active":false}" >tmp.tmp
# curl -k --user admin:<pass> -H "accept: application/json" -T tmp.tmp https://<appliance>/isam/felb/configuration/services/my-server/servers/192.168.0.1:81

5. Apply changes
# curl -k --user admin:<pass> -H "accept: application/json" https://<appliance>/isam/pending_changes -X PUT
