Title: IBM Security Bulletin: A security vulnerability has been identified in IBM WebSphere Application Server and DB2 Enterprise Server Edition shipped with Financial Transaction Manager (CVE-2015-2808) - United States

Text:
 SECURITY BULLETIN

SUMMARY
 IBM WebSphere Application Server and DB2 Enterprise Server Edition are shipped as a component of Financial Transaction Manager. Information about a security vulnerability affecting IBM WebSphere Application Server and DB2 Enterprise Server Edition has been published in a security bulletin.
