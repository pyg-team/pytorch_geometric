Title: IBM WebSphere Business Integration Adapter for Siebel eBusiness Applications 4.6.3 - United States

Text:
Siebel Connector Fix Pack; Siebel Connector; Siebel Connector Patch; Event Status; Child Picklist; Session Timeout; Siebel Fix Pack; Association Update; Simple Link child update; Embedded spaces in the search string; Restart; ConnectErrors; ODA; Siebel ODA DOWNLOADABLE FILES

ABSTRACT
 The WebSphere® Business Integration Adapter for Siebel eBusiness Applications version 4.6.2 has been updated. 

DOWNLOAD DESCRIPTION
The Adapter for Siebel eBusiness Applications 4.6.3 fix pack is now available for the Windows® 2000 and 2003, Solaris, AIX®, and HP-UX operating systems.

The following list is a history of problem tracking numbers and APARs fixed in this fix pack:

Version 4.6.3
83739
The adapter is required to support AIX 5.3.

68880
The adapter omits some exit trace messages. It will now generate and present the necessary exit trace messages.

85860/JR24895
When the error specified in the ConnectErrors property occurs, the adapter does not restart as expected. As a result of the fix, the adapter can now realize that the error specified in the ConnectErrors property has occurred and will restart correctly.

84218/JR23781
The OS variable is not in sync with the CWSharedEnv and the start_siebel files.The start_siebel file has been updated, so that the OS variables are consistent in these two files. A technote has been published that explains how to update the OS variable in the scripts. See the following link: http://www-1.ibm.com/support/docview.wss?rs=0&q1=CWConnEnv.sh&uid=swg21240041&loc=en_US&cs=utf-8&cc=us&lang=en [http://www-1.ibm.com/support/docview.wss?rs=0&q1=CWConnEnv.sh&uid=swg21240041&loc=en_US&cs=utf-8&cc=us&lang=en]

83774/JR23556
The adapter does not release memory until it reaches the maximum memory size.The adapter will clean up all the resources allocated in the event store while exiting pollForEvents.

83287/JR23371
In the Siebel eBusiness Applications application, DTYPE_BOOL is a character with a length of 1. DTYPE_BOOL is mapped to Boolean when generating a business object definition using the Siebel eBusiness Applications ODA. DTYPE_BOOL will be mapped to String with the length of 1 when generating a business object definition using the Siebel eBusiness Applications ODA.

Version 4.6.2
67600
The Siebel eBusiness Applications adapter startup scripts need to be updated for Siebel eBusiness Applications server 7.8.

72358
Siebel eBusiness Applications 7.8 support should be included.

73098/JR22541
The Siebel eBusiness Applications stdout message is more explicit than the one in the flow manager. Both should be same.

71328/JR22554
After successfully invoking a business service, the top level attributes of a response business object cannot be populated correctly.

73549/JR22557
The Siebel eBusiness Applications adapter cannot handle an association and update.

72440/JR22556
The Siebel eBusiness Applications connector generates an error when retrieving more than 10,000 records.


Version 4.6.1
JR20461/60094
An error occurs when more than one child exists with a one to many relationship.

JR20887/60182
If the relationship between parent and child is "association," the adapter cannot set a particular child as a primary record.

JR20330/59209
If a key attribute of type String has empty spaces in the value, the adapter generates an error.

JR20887/59945
When used with DB2- Information Integrator broker, the Siebel eBusiness Applications adapter does not a submit truncated result set warning.

JR20887/59998
The Siebel eBusiness Applications connector does not roll back a child create when an association is involved.


Version 4.5.1
JR20330/37580
The adapter always expects the Siebel eBusiness ApplicationsMessage container attribute to be present in the response business object of the custom business service business object.

JR20330/59209
If a key attribute of type String has empty spaces in the value, the adapter is generates an error.

JR20330/59061
The adapter should not log any messages while the subscription delivery is set at trace level 0.

JR20398/37505
The Siebel eBusiness Applications ODA is unable to generate business object definitions for business services in the Siebel eBusiness Applications application versions 6.x.
