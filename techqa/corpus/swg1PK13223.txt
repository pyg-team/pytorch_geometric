ERROR DESCRIPTION
 *  Creating a document for a work management element, and nesting
   documents underneath this document in which case all document
   names have at least one word in common, then going to the
   document portlet and searching for documents using the word in
   common, returns a recursive document structure when expanding
   the parent document.
   
   
    
   
   

LOCAL FIX
 *  Search from document view rather than the document portlet
