Title: IBM PI24155;8.5.5: WAS 8.5.5 install manager times out on some IBM i hardware - United States

Text:
PI24155; PI24155; PI24155 DOWNLOADABLE FILES

ABSTRACT
 Get time out warnings during installation of the IBM WebSphere Application Server using IBM Installation Manager on i5/OS. 

DOWNLOAD DESCRIPTION
PI24155 resolves the following problem:





ERROR DESCRIPTION: 

WebSphere Application Server version 855 Install manager times out on some IBM i hardware 

 
LOCAL FIX: 
na 

PROBLEM SUMMARY 


USERS AFFECTED:
All users of IBM WebSphere Application Server

PROBLEM DESCRIPTION:
Get time out warnings during installation of the IBM WebSphere Application Server using IBM Installation Manager on i5/OS.


RECOMMENDATION: 

None


When installing IBM WebSphere Application Server on i5/OS using IBM Installation Manager, the configuration operation times out and generates warnings in the IBM Installation Manager log file.

PROBLEM CONCLUSION: 

The time out value for the configuration operation has been increased to avoid the time out issue when installing the IBM WebSphere Application Server on i5/OS using IBM Installation Manager. 


 
The fix for this APAR is currently targeted for inclusion in fix pack 8.5.5.4. Please refer to the Recommended Updates page for delivery information:
