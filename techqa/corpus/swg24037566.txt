ABSTRACT
 This document describes how to download and install IBM Netcool Network Management V9.2.2 using the Passport Advantage website. 

DOWNLOAD DESCRIPTION
IBM Netcool Network Management provides network discovery and event and network management, including identification and resolution of root causes, system, network and service failures, and configuration management capabilities for network devices. Inventory and connectivity data is integrated with IBM Tivoli Netcool/OMNIbus event data. Network management staff can visualize and understand the layout of complex networks, and the impact of network events and failures. They can work with events to locate, diagnose, and solve network problems in real time. 

Downloading and assembling the product

Got to [http://www.ibm.com/software/howtobuy/passportadvantage/paocustomer/sdma/SDMA?PO=DOWNLOAD_FIND_BY_PART_NUMBER]Find Downloads and Media [http://www.ibm.com/software/howtobuy/passportadvantage/paocustomer/sdma/SDMA?PO=DOWNLOAD_FIND_BY_PART_NUMBER] on IBM Passport Advantage Online and log in. 
 2. Search for the part numbers that are associated with the eAssembly. The eAssembly number is CRRR8ML. 
 3. Download all the parts that you need and decompress them into a single temporary location on your system. If you want to install Tivoli Netcool/OMNIbus as part of the Network Manager installation process, download the Tivoli Netcool/OMNIbus installation package to a specific location and point the Network Manager installer to that location. Do not decompress or rename the Tivoli Netcool/OMNIbus installation package. 
 4. See the section "Installation instructions" for links to useful sources of information about installing and upgrading.
