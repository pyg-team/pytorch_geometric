ICR License TECHNOTE (TROUBLESHOOTING)

PROBLEM(ABSTRACT)
 The action RecognizePageFields2CCO_ICR_C () encountered an error, and caused the batch to be aborted during the Rulerunner processing task.

The following error and logged snippets are detected in the rulerunner_rrs.log file:

    [RecognizePageFields2CCO_ICR_C] requested abort [api source:]
    EXCEPTION: code="385875969" msg="Aborting: Action [RecognizePageFields2CCO_ICR_C] requested abort" loc="CIMainAlgorithm::execute4DCO" API=""

ENVIRONMENT
Windows 2008 R2 Standard, 64-bit O/S

DIAGNOSING THE PROBLEM
Check the Taskmaster Client station, Application Event Viewer log, look for a Datacap error entry with the following details:

Activation context generation failed for "C:\Datacap\dcshared\ICRC\ai050001.dll". DependentAssemblyMicrosoft.VC80.OpenMP.processorArchitecture="x86",publicKeyToken="1fc8b3b9a1e18e3b",type="win32",version="8.0.50727.762" could not be found. Please use sxstrace.exe for detailed diagnosis.



RESOLVING THE PROBLEM
Install Microsoft Visual C++ 2005 SP1 Redistributable Package (x86). Once it is installed, the version should show 8.0.50727.762 to be the correct package.
