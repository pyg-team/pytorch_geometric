Title: IBM Loading a Different Level Other Than Latest Firmware From the HMC Using the Web - United States

Text:
 TECHNOTE (TROUBLESHOOTING)

PROBLEM(ABSTRACT)
 When loading firmware from the HMC using the Web, you get only the option to select the latest firmware. These steps will allow you to load a later version when using the HMC to connect to the IBM Web site to load the firmware. 

RESOLVING THE PROBLEM
When loading firmware from the HMC using the Web, you get only the option to select the latest firmware. These steps will allow you to load a later version when using the HMC to connect to the IBM Web site to load the firmware.
