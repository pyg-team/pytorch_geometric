Title: IBM Informix Client Software Development Kit (CSDK) is not affected by the OpenSSL Heartbleed vulnerability (CVE-2014-0160) - United States

Text:
 FLASH (ALERT)

ABSTRACT
 Informix Client Software Development Kit (CSDK) is not vulnerable to the CVE-2014-0160 OpenSSL Heartbleed vulnerability. 

CONTENT
CVE-2014-0160 - Heartbleed Vulnerability 
The components included in Informix Client Software Development Kit (CSDK) in all versions and all platforms are NOT vulnerable to the OpenSSL Heartbleed vulnerability (CVE-2014-0160) because it does not use an affected version of OpenSSL.

Remediation: No action required.

Change History: 11 April 2014, original document published.
