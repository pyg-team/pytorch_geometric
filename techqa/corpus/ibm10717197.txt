Title: IBM Instructions for installing IBM InfoSphere Information Server 11.7.0.1 Service Pack 1 - United States

Text:
 FIX README

ABSTRACT
This readme document contains guidance for the installation of 11.7.0.1. Service Pack 1

CONTENT
The steps below are PRE-REQUISITES and must be installed in the order presented:
Before applying Service Pack 1 customers must first have IBM InfoSphere Information Server 11.7.0.1 installed.
Customers not on version 11.7 must install 11.7.0.0 before applying 11.7.0.1 followed by 11.7.0.1 SP1.

For customers requiring the full suite installation from Passport Advantage, the download document for version 11.7.0.0 can be found here:
AIX http://www.ibm.com/support/docview.wss?uid=swg24044347 [http://www.ibm.com/support/docview.wss?uid=swg24044347]
Linux https://www.ibm.com/support/docview.wss?uid=swg24044366 [https://www.ibm.com/support/docview.wss?uid=swg24044366]
Windows https://www.ibm.com/support/docview.wss?uid=swg24044353 [https://www.ibm.com/support/docview.wss?uid=swg24044353]

The instructions for installing 11.7.0.1 can be found here:
