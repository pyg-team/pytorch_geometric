Title: IBM Security Bulletin: Multiple vulnerabilities have been identified in IBM Java SDK October 2015 CPU shipped with IBM WebSphere Portal ( CVE-2015-4872; CVE-2015-4840; CVE-2015-4734; CVE-2015-5006 ) - United States

Text:
 SECURITY BULLETIN

SUMMARY
 IBM Java SDK October 2015 CPU shipped as component of IBM WebSphere Portal. Information about a security vulnerability affecting IBM Java SDK October 2015 CPU has been published in a security bulletin. 

VULNERABILITY DETAILS
Consult the security bulletin Security Bulletin: Multiple vulnerabilities in IBMÂ® Java SDK affect WebSphere Application Server October 2015 CPU [https://www-304.ibm.com/support/docview.wss?uid=swg21969620] [https://www-304.ibm.com/support/docview.wss?uid=swg21969620] for vulnerability details and information about fixes.
DESCRIPTION:This bulletin covers all applicable Java SE CVE's published by Oracle as part of their October 2015 Critical Patch Update and an additional vulnerability which affects IBM SDK, Java Technology Edition. There are other advisories included in the IBM Java SDK but WebSphere Portal Server is not vulnerable to them. You will need to evaluate your own code to determine if you are vulnerable. Please refer to the Reference section for more information on the advisories not applicable to WebSphere Portal Server.
