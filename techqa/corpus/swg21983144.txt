Title: IBM Security Bulletin: A security vulnerability has been identified in IBM Rational Application Developer shipped with IBM Business Modeler (CVE-2015-7439) - United States

Text:
 SECURITY BULLETIN

SUMMARY
 IBM Rational Application Developer is shipped as a component of IBM Business Modeler. Information about a security vulnerability affecting IBM Rational Application Developer has been published in a security bulletin. 

VULNERABILITY DETAILS
Consult the security bulletin Cross-Site Scripting Vulnerability with the UML Vizualization tools [http://www-01.ibm.com/support/docview.wss?uid=swg21978003] for vulnerability details and information about fixes.

AFFECTED PRODUCTS AND VERSIONS
