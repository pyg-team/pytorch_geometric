Title: IBM Security Bulletin: IBM Security products not affected by the GNU C library (glibc) vulnerability (CVE-2015-0235) - United States

Text:
PSIRT SECURITY BULLETIN

SUMMARY
 The following IBM Security Products are not vulnerable to the GNU C library (glibc) vulnerability that has been referred to as “GHOST”. 

VULNERABILITY DETAILS
The following list of IBM Security Products in all editions and all platforms are NOT vulnerable to the GNU C library (glibc) vulnerability (CVE-2015-0235):


The following products do not package an Operating System. It is up to the customer to update the underlying OS if applicable:
