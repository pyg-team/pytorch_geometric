Title: IBM Corrections to IBM Tivoli Monitoring Version 6.3 Installation guide - United States

Text:
ITM 6.3.0 6.3FP1 6.3FP2 Authorization Policy Server TECHNOTE (TROUBLESHOOTING)

PROBLEM(ABSTRACT)
 IBM Tivoli Monitoring Version 6.3 Installation guide states gtk is required on Linux it but does not state gtk2-devel. 

SYMPTOM
Authorization Policy Server Installation fails


CAUSE
gtk2-devel prerequisite library was not installed on Linux x86-64 system.

ENVIRONMENT
Linux x86-64 

DIAGNOSING THE PROBLEM
During Authorization Policy Server Installation through IBM Installation Manager, the following error message appears:




" 

CRIMA1217E: A problem occurred during the execution of the 
/opt/IBM/JAZZSM/AuthPolicyServer/PolicyServer/bin/com.ibm.tivoli
.granular.deploy.xml file. 

Explanation: A package has an issue that cannot be resolved by 
Installation Manager. 

User Action: Identify the package that has the issue by looking 
at the installation history. [/support/docview.wss?uid=swg21699958&amp;aid=1] 
" 
RESOLVING THE PROBLEM
Installation History refers to a missing gtk2-devel library. 

 

Install gtk2-devel library and repeat Authorization Policy 
Server Installation.
