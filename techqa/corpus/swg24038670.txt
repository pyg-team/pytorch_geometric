ABSTRACT
 This Fix Pack addresses problems reported in IBM Tivoli Netcool/Impact 6.1.1.1 

DOWNLOAD DESCRIPTION
 This IF contains fixes that are only applicable when using 


 Netcool/Impact 6.1.1 Fix Pack 01.


* Fixes that are included in Interim Fix 01.

Advisory 2275, Product Record 43643 - XSS vulnerabilities
reported from TIP 

This APAR is specifically focused to address XSS 
vulnerabilities reported from TIP

---------

- Interim Fix content 6.1.1.1-TIV-NCI-IF0001.zip

Replaced files on the Impact Server: 
[NCHOME]/lib/nciClient.jar

If you have made changes to the above mentioned files you will

INSTALLATION INSTRUCTIONS
Installation, migration, upgrade and configuration information: 

---------------------------------------------------------------

* Hardware and software requirements:

Refer to the Netcool/Impact 6.1.1 Release Notes for a full 
description of all the hardware and software requirements.

To apply this Interim Fix you will need approximately 50 Mb
of free disk space.


* Dependencies:

This Interim Fix must be installed on each server(s) that is
running the Netcool/Impact server. 


This Interim Fix requires Netcool/Impact 6.1.1 FP0001 
to be already installed.


* Special considerations:

1) A Backup of the files will be made and placed in:

[NCHOME]/_uninst/IMPACTInstall6111-IF0001/yyyymmdd_hhmmss


2) The Netcool/Impact server(s) need to be stopped, prior to
installing this Interim Fix. 

* Extracting this Interim Fix:
Windows and Non-Windows

1. Copy the file 6.1.1.1-TIV-NCI-IF0001.zip to a 
temporary location on your Netcool/Impact GUI server.
2. Unzip the file on each server that you need to install on.




* Installing this Interim Fix:

[Interim Fix] Refers to the directory where this Interim Fix
resides (after you have extracted it from the zip file).

for example: /tmp/6.1.1.1-TIV-NCI-IF0001

On the Netcool/Impact server system:

Navigate to the [Interim Fix] directory where the files
were extracted.

Install the Interim Fix with the same user that was used to 
install Netcool/Impact 6.1.1 Base release.

All Netcool/Impact GUI servers
===================================================
1.) Stop the Netcool/Impact server(s)
2.) Open a command prompt and change to the unzip directory 
3.) Set the NCHOME environment variable to locate the Netcool/Impact
6.1.1 installation:
for example: /opt/IBM/tivoli/impact

4.) To install Interim Fix 1,
If you are on a Windows operating system, run the following
command from a prompt: 

impactIF.bat

If you are on a Unix operating system, run the following
command from a prompt:

./impactIF.sh

NOTE: There is no GUI mode for this Interim Fix. 

The above script execution should complete with the
statement "BUILD SUCCESSFUL"

5.) Start the Netcool/Impact GUI server


* Verify the installation

1) cd "[NCHOME]\bin" and run versioninfo.bat|.sh

Confirm that the jar file information reported under
the "Impact JAR Version Information" section shows the affected jar
files as follows: 

nciClient.jar
Build ID: 201410231702
Version: 6.1.1.1 IF0001
Path: C:\IBM\tivoli\impact\lib

nciClient.jar
Build ID: 201410231702
Version: 6.1.1.1 IF0001
Path: C:\IBM\tivoli\impact\uilib

NOTE: On the Impact GUI, the nciClient.jar will be updated on the
uilib directory.

On the Impact server, the nciClient.jar will be updated on
the lib directory.


* Uninstalling this Interim Fix

Please note: This Interim Fix must be uninstalled on each Impact
server(s)


On the Netcool/Impact server system:

Uninstall the Interim Fix with the same user that was used to 
install Impact 6.1.1.1 IF1.


1.) Stop the Impact server(s)

2.) Navigate to the backup directory where the files were backed up
by the Impact 6.1.1.1 IF1 installer:
[NCHOME]/_uninst/IMPACTInstall6111-IF0001/yyyymmdd_hhmmss

For example: /opt/IBM/tivoli/impact/_uninst/IMPACTInstall6111-IF0001/20141020_175039

3.) Issue the following command:
$WAS_HOME/bin/ws_ant.sh (Unix) 
"%WAS_HOME%\bin\ws_ant.bat" (Windows)

For example: /opt/IBM/tivoli/tipv2/bin/ws_ant.sh

During the uninstall the user is asked to confirm they wish to
uninstall. Type y for yes and n for no (followed by the Enter
key!)

The above script execution should complete with the statement
"BUILD SUCCESSFUL"

4.) Start the Dashboard Server

* verify uninstallation

1) cd "[NCHOME]\bin" and run versioninfo.bat|.sh

Confirm that all of the nciClient.jar files have the same Build ID 
and that the Version is no longer IF0001

nciClient.jar
Build ID: 201403061352
Version: 6.1.1.1 FP0001
Path: /opt/IBM/tivoli/impact/lib

nciClient.jar
Build ID: 201403061352
Version: 6.1.1.1 FP0001
Path: /opt/IBM/tivoli/impact/uilib

NOTE: On the Impact GUI, the nciClient.jar will be updated on the
uilib directory.

On the Impact server, the nciClient.jar will be updated on
the lib directory.
