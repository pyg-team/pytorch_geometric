ERROR DESCRIPTION
 *  Arabic characters appears as question marks (?????) in FEM and
   Workplace when federating a document from Image services 4.1.1
   on oracle 9i with (US7ASCII) character set to P8 CE 4.0.1-4 on
   oracle 10g with AR8ISO8859P6 character set.
   
   CE is running on IBM websphere 6.1.0.15 and windows 2003 SP2 .
   IS is running on windows 2003 SP2
   Arabic language is installed on both servers and set it as
   default in both regional options and Advance tabs of the control
   panel
   regional and language options.
   Adding new documents using Workplace containing Arabic
   characters in property fields can be retrieve and display
   properly.
   They can view documents via IDM desktop containing Arabic
   characters in properties without any issue. (please see screen
   shots attached to PMR)
   
   We have changed "Default Character Set" in IS to "Arabic
   ISO-8859-6" and we still have the same issue.
