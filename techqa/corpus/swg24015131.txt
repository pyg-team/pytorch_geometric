ABSTRACT
 This compatibility upgrade contains fixes to allow WebSphere® Transformation Extender (WTX) V8.1.0.1 to run successfully on Microsoft® Windows® Vista®. 

DOWNLOAD DESCRIPTION
WTX does not work properly on Windows Vista unless this compatibility upgrade is installed. The compatibility upgrade can only be installed on top of 8.1.0.1. WTX is a 32-bit application.

How critical is this compatibility upgrade?
Critical. 

Problems solved
The compatibility upgrade provides fixes for WTX that allows the product to function properly on Windows Vista.

Known side-effects
The following is a list of known issues with WTX installation on Vista.

The Program Compatibility Assistant may report an error that states "This program may not have installed correctly". The error only occurs when running the setup.exe file outside the self-extracting zip executable provided in this download and is expected. Check the WTX installation directory for files listed in the Files Affected section below and verify that they have the Date Modified attribute as the current date or 23-Feb-2007. If the dates are correct, the installation completed successfully and you can click the "This program installed correctly" option. If the dates are not correct, click the "Cancel" button and try installing again using the self-extracting zip executable provided in this download. If you get this error in a situation not described previously, contact technical support.

[PMR59440] Java applications, such as the Management Console, running on Windows Vista do not adjust the time correctly for the appropriate time zone. For example, the time returned is equivalent to Greenwich mean time (GMT)+0 instead of GMT-5.00 for eastern standard time (EST). The managementconsole.log and launcher.log files are affected by this issue.

The following product components are not supported in this Windows Vista compatibility upgrade: 

[EC114758] WebSphere Transformation Extender Pack for PeopleSoft®
[EC115549] WebSphere Transformation Extender Pack for SAP® XI

The following is a list of known issues with WTX Pack for SAP R/3

[EC115029,EC115051] ALE Adapter not supported. The ALE Adapter is not supported in the Windows Vista compatibility upgrade for WebSphere Transformation Extender.

[EC115048] When using the JALE Adapter on Windows Vista, there is a known issue with *.tid where the backup file is not created properly. 

The following is a list of known issues with WTX Pack for Siebel®

[EC115550] The Siebel Importer is not supported in the Windows Vista compatibility upgrade for WebSphere Transformation Extender.

PREREQUISITES
IBM WTX for Windows V8.1.0.1 (114) must be installed. If you have received patched libraries for any libraries listed above, this compatibility upgrade supersedes them.

Note: To function properly on the Windows Vista operating system, WTX should be installed outside of the Program Files directory. If you already have WTX installed inside of the Program Files directory, you should uninstall it and reinstall it in a different directory, such as:

C:\WebSphere Transformation Extender 8.1

INSTALLATION INSTRUCTIONS
Note: After installing this compatibility upgrade, there is no uninstall program available.

1. Download the Windows Vista compatibility upgrade using one of the links below.

2. On your Windows Vista operating system, verify that WebSphere Transformation Extender 8.1.0.1 products and components have been installed in a directory that is outside of the Program Files directory. 

3. Run the self-extracting executable file. Instructions are provided to guide you through the installation process.
