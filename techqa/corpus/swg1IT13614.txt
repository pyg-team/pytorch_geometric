APAR STATUS
 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  A large deployment with multiple SOAP nodes (SOAPInput,
   SOAPRequest, SOAPAsyncRequest) in applications,
   and multiple large WSDLs in one or more shared libraries,
   that are referenced by the aforementioned SOAP nodes,
   can cause excessive deployment times.
   
   
    
   
   

LOCAL FIX
 *  N/A
   
   
    
   
   

PROBLEM SUMMARY
 *  ****************************************************************
   USERS AFFECTED:
   All users of IBM Integration Bus V10.0 having multiple SOAP
   nodes in an application that reference multiple large WSDLs in
   shared libraries.
   
   
   Platforms affected:
   MultiPlatform, z/OS
   
   ****************************************************************
   PROBLEM DESCRIPTION:
   A large deployment with multiple SOAP nodes (SOAPInput,
   SOAPRequest, SOAPAsyncRequest) in applications, and multiple
   large WSDLs in one or more shared libraries, that are referenced
   by the aforementioned SOAP nodes, can cause excessive deployment
   times.
   
   
    
   
   

PROBLEM CONCLUSION
 *  The product now reduces the deployment time of applications with
   multiple SOAP nodes which reference large WSDL files.
