Title: IBM How to change the port and the context root of Information Center running on OmniFind Enterprise Edition - United States

Text:
 TECHNOTE (FAQ)

QUESTION
 Currently the Information Center is running on http://<omnifind server>:8889/help on OmniFind server, but I want to execute the Information Center on different port and different context (like http://<omnifind server>:8080/OEEhelp).

Is it possible to change the port and the context root of Information Center running on OmniFind server? 

ANSWER
To change the port and context, follow the instruction as below:
1. Stop the system by esadmin system stopall command.

2. Log on to OmniFind Server and go to ES_INSTALL_ROOT/ibm_help directory.

3. Update IC_start.sh (or IC_start.bat) file for start and IC_stop.sh (or IC_stop.bat) file for shutdown with same way as follows:

To change the context root:


 * From : -Dorg.eclipse.equinox.http.jetty.context.path=/help
 * 
 * To : -Dorg.eclipse.equinox.http.jetty.context.path=<new context>


For example: -Dorg.eclipse.equinox.http.jetty.context.path=/OEEhelp 

To change the port: 
 * From : -port 8889
 * 
 * To : -port <new port Number>


For example: -port 8080 

4. Update ES_INSTALL_ROOT/nodeinfo/es.cfg also to specify the correct port and path in DB2ICDocs4ES property. Also, update DB2ICServerPort property to sync the configuration as well. 
 * From : DB2ICDocs4ES=http\://<oee server>\:8889/help/topic
 * 
 * To : 
 * DB2ICDocs4ES=http\://<oee server>\:<port>/<new context root>/topic

 * From : 
 * DB2ICServerPort=8889
 * 
 * To : 
 * DB2ICServerPort=<new port number>


For example:  * DB2ICDocs4ES=http\://<oee server>\:8080/OEEhelp/topic
 * 
 * 
 * 
 * DB2ICServerPort=8080


5. Start the system again by esadmin system startall command. 

6. Confirm if you can refer to the Information Center on OmniFind server correctly as follows: 
 * Access to the help directly at http://<oeeserver>:<specified port#>/<new context root> 
 * Access to the help from the out of the box search application at http://<oeeserver>:<port#>/search and click "Help" link on top of the page


For example, try to access to http://<oeeserver>:8080/OEEhelp if you specify -port 8080 and -Dorg.eclipse.equinox.http.jetty.context.path=/OEEhelp .
