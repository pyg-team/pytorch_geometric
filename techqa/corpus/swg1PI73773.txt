ERROR DESCRIPTION
 *  Slow performance stopping test suite execution with large number
   of test cases in an RQM project with 3,000 or more members
   
   Steps to reproduce:
   In an RQM project area that has 3,000 or more members:
   
   1. Create 450 test cases and 450 test scripts.  Associate each
   test case with 1 script.
   2. Create a suite
   3. Add the 450 test cases to the suite; set execution to be
   sequential
   4. Generate a suite execution record
   5. Initiate execution of the suite execution record
   6. Execute the first test case in the suite
   7. When execution of the first test case is complete, stop
   execution of
   the suite by clicking the Stop Run button on the Test Suite
   Execution Console
   
   Expected:  Execution is stopped immediately, indicated when the
   Close and Show Results button becomes enabled.
   Actual:  Execution does not stop immediately
   
   Review the following work item on Jazz.net for details:
   https://jazz.net/jazz02/web/projects/Rational%20Quality%20Manage
   r#action=com.ibm.team.workitem.viewWorkItem&id=158748 [https://jazz.net/jazz02/web/projects/Rational%20Quality%20Manager#action=com.ibm.team.workitem.viewWorkItem&id=158748]
