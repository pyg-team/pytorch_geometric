QUESTION
Why do I receive a SQL1822N rc1045 when accessing Mysql? 

ANSWER

From the trace the user is "root" and password is "YES". 

The error information from the MYSQL manual is: 
Error: 1045 SQLSTATE: 28000 (ER_ACCESS_DENIED_ERROR) 
Message: Access denied for user '%s'@'%s' (using password: %s) 

A 'GRANT ALL' for the user may need to be granted on the Mysql database.
