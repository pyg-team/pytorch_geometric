Title: IBM Netcool WEBGUI 8.1 AEL does not show right click menu - United States

Text:
webgui 8.1 AEL no right click menu TECHNOTE (TROUBLESHOOTING)

PROBLEM(ABSTRACT)
 After upgrading WebGUI 8.1 from FP2 to FP4 users were not able to see the right click menu from AEL. The Event Viewer was working fine. 

SYMPTOM
Java console log was showing below exception:

[INFO] [ NCWMenuUtils.getToolMnemonic() ] Tool 
mnemonic not set for tool: ADP_Critical

cache: registerReference:com.sun.deploy.cache.MemoryCache$CachedResourceReference@616b740a: 4

at com.micromuse.ncw.applet.tools.CompoundToolImpl.hasClassCriterionOnly(Co 
mpoundToolImpl.java:318) 

 at com.ibm.tivoli.ncw.applets.ael.ui.actions.ExecuteToolAction.applyAccessC 
ontrol(ExecuteToolAction.java:139)
at com.ibm.tivoli.ncw.applets.ael.ui.actions.ExecuteToolAction.<init>(Execu 
teToolAction.java:109) 


CAUSE
Related to APAR:

http://www-01.ibm.com/support/docview.wss?uid=swg1IV78646 [http://www-01.ibm.com/support/docview.wss?uid=swg1IV78646] 


ENVIRONMENT
Users were able to reproduce the issue using following browsers and Java versions:
IE 11 and FF 45.
Java : 1.7.0_45, Java : 1.7.0_67, Java 1.8 



RESOLVING THE PROBLEM
The APAR fix is included in Web GUI 8.1 Fix pack 5. Installation of FP5 resolved the issue.
