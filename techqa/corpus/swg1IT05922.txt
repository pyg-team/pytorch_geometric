APAR STATUS
 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  SSL/TLS clients and servers that have enabled SSL 3.0 support
   might be exploited by an active man-in-the-middle, to downgrade
   connections to SSL 3.0, even if both sides of connection support
   higher protocols.  SSL 3.0 contains a number of weaknesses
   including POODLE (CVE-2014-3566).
   
   
    
   
   

LOCAL FIX

PROBLEM SUMMARY
 *  Security vulnerabilites exist when using SSLV3 as decribed in
   CVE-2014-3566.
   
   
    
   
   

PROBLEM CONCLUSION
 *  Fix is available in 6.0.0.12, 6.0.1.8, 7.0.0.5 and 7.1.0.0.
   
   For a list of the latest fix packs available, please see:
   http://www-01.ibm.com/support/docview.wss?uid=swg21237631 [http://www-01.ibm.com/support/docview.wss?uid=swg21237631]
