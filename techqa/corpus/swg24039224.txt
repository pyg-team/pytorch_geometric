PREREQUISITES
This fix pack requires that Cognos Express 10.2.1 is already installed. An existing installation may optionally include interim fixes for Cognos Express 10.2.1. Our fix packs are cumulative by design. Hence Cognos Express 10.2.1 Fix Pack 3 includes all updates from earlier Cognos Express 10.2.1 Fix Packs and interim fixes.

INSTALLATION INSTRUCTIONS
 

 * Ensure your system has a dedicated 10GB of disk space to upgrade IBM Cognos Express 10.2.1 FP3, 
 * Ensure the installed IBM Cognos Express is fully up and running, 
 * Ensure you have backed up your IBM Cognos Express content, 
 * Create a folder in your Express 10.2.1 server to extract the required install file, 
 * Using your file compress and decompress utility, decompress the .tar.gz file in the created folder, 
 * Expand the folder and open "winx64h" folder and run "issetup.exe" as administrator to install the fix pack, 
 * Follow the instructions provided in the wizard. The last step may take several minutes to complete, 
 * Completely clear the browser cache before launching the IBM Cognos Express Manager application.


Guidance on updating the IBM Cognos Express 10.2.1 Gateway can be found in this technote (link to http://www-01.ibm.com/support/docview.wss?uid=swg21677697 [http://www-01.ibm.com/support/docview.wss?uid=swg21677697]). DOWNLOAD PACKAGE
It is recommended that you download and apply this fix pack to all of your Cognos Express 10.2.1 installations at your earliest convenience.
