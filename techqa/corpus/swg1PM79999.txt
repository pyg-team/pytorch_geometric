PROBLEM SUMMARY
 *  ****************************************************************
   * USERS AFFECTED:  IBM WebSphere Application Server Version    *
   *                  8.5 Liberty Profile users of JSF MyFaces.   *
   ****************************************************************
   * PROBLEM DESCRIPTION: When using a NavigationHandler to       *
   *                      navigate to an error page after         *
   *                      a ViewExpiredException a                *
   *                      NullPointerException occurs.            *
   ****************************************************************
   * RECOMMENDATION:                                              *
   ****************************************************************
   When trying to handle a
   javax.faces.application.ViewExpiredException in a custom
   ExceptionHandler the following NullPointerException can occur:
   java.lang.NullPointerException
   at
   org.apache.myfaces.application.NavigationHandlerImpl.getNavigati
   onCase(NavigationHandlerImpl.java:203)

   ation(NavigationHandlerImpl.java:77)
