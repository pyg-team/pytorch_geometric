ERROR DESCRIPTION
 *  This APAR includes fixes to various defects identified in CICS
   Explorer V5.4.0.
   
   
    
   
   

LOCAL FIX

PROBLEM SUMMARY
 *  It was not possible to connect to DB2 V11 on z/OS when using
   CICS PA and CICS IA.
   com.ibm.cics.core.comm.
   EnhancedConnectionException: IZE0106E Connect failed with error
   " jcc
    t4  10509  13454  4.21.29  Connection to the data server
   failed. The
   IBM Data Server for JDBC and SQLJ license was invalid
   or was not activated for the DB2 for z/OS subsystem.
   
   
    
   
   

PROBLEM CONCLUSION
 *  An incorrect license file that prevented the correct operation
   of the DB2 driver used by downstream products has been
   corrected.
   .
   This fix will be made available in Version 5.4.0.8 of the CICS
   Explorer.
   .
   For installation instructions please see:
   Ordering maintenance for IBM CICS Explorer and CICS Explorer
   plug-ins
   http://www.ibm.com/support/docview.wss?uid=swg21380083 [http://www.ibm.com/support/docview.wss?uid=swg21380083]
