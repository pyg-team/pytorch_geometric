ABSTRACT
 This describes how to download and assemble Composite Application Manager for Web Resources for WebSphere for AIX components using the Passport Advantage Online Web site. 

DOWNLOAD DESCRIPTION


[http://www.ibm.com/i/c.gif] IBM Tivoli Composite Application Manager (ITCAM) for Web Resources enables IT operations and administrators to assure the availability and performance of critical business applications and services by comprehensively monitoring the health and performance of the underlying J2EE application server and web server resources. ITCAM for Web Resources proactively alerts administrators of health and performance problems, provides real time metrics for problem diagnosis, and collects historical metrics for reporting and capacity trending.

Downloading and assembling the product

 1. Download all of the required parts, which are described by platform in the tabbed sections of this document. 
 2. Unpack each of the parts into a single temporary directory on your system. 
 3. Follow the instructions in the Quick Start Guide and the Installation Guides [http://publib.boulder.ibm.com/infocenter/tivihelp/v3r1/topic/com.ibm.itcamwas_wr.doc_6.2/welcome.htm] to install the product.
