PROBLEM SUMMARY
 *  ****************************************************************
   * USERS AFFECTED:                                              *
   * All Maximo Users on IE 10.                                   *
   ****************************************************************
   * PROBLEM DESCRIPTION:                                         *
   * Pressing Enter key on the list screen when using IE 10       *
   * causes the advanced search screen to appear.                 *
   ****************************************************************
   * RECOMMENDATION:                                              *
   * Upgrade to latest release.                                   *
   ****************************************************************
   STEPS TO REPRODUCE:
   1) Access SCCD 7.5.1 with IE 10.  Log into Maximo - Ensure
   tivoli13 skin is being used.  Go To - Work Orders - Work Order
   Tracking - hit Enter - you will be returned to the Start Center.
   
   2) Access SCCD 7.5.1 with IE 10.  Log into Maximo - Go To -
   System Configuration - Platform Configuration - System
   Properties - filter for the following properties:
   mxe.webclient.systemNavBar - set to 0
   mxe.webclient.homeButtonHeaders - set to 0
   mxe.webclient.gotoButtonHeaders - set to 0
   Sign out - Sign back into Maximo.
   
   Go To - Work Orders - Work Order Tracking - hit enter - the
   Advanced search screen will appear.  Will be unable to 'exit'
   the dialog.
   
   
    
   
   

PROBLEM CONCLUSION
 *  The Enter key will no longer return user to start center or open
   advanced search dialog while using IE 10.
   
   The fix for this APAR is contained in the following maintenance
   package:
   	 | release\fix pack | Interim Fix for Release 7.5.0.4 TPAE
