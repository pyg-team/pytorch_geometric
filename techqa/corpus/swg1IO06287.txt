Title: IBM IO06287: DOCUMENT MANAGER OULTOOK INTEGRATION BUTTONS DUPLICATION. - United States

Text:

APAR STATUS
 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  Problem:
   If user opens an email in Outlook and continue reading emails
   (move between multiple emails using arrow buttons Previous Item
   and Next Item)
   Insert Attachments, Save Attachments and Add Document
   integration buttons will multiply.
   For every new email, opened mail window will add two more
   buttons, so after reading 5 emails in succession you will end up
   with a window with 5 pairs of Insert/Save Attachment buttons.
   
   
    
   
   

LOCAL FIX
 *  No
   
   
    
   
   

PROBLEM SUMMARY
 *  If user opened an email message in Microsoft Outlook and then,
   while that email message was open, the user read other email
   messages by moving between the many messages using the Previous
   and Next buttons, each successive email message that the user
   opened had more Insert Attachments, Save Attachments, and Add
   Document integration buttons than the previous email message.
   For example, after reading five email messages, the fifth
   message had five sets of Insert Attachments and Save Attachments
   integration buttons.
    Important: The fix for this problem has the following
   prerequisite:
   Microsoft Outlook 2003 SP2 + latest security patches
   
   
    
   
   

PROBLEM CONCLUSION
 *  The fix is available in DB2 Document Manager V8.3 Fix Pack 6.
   
   
    
   
   

TEMPORARY FIX
