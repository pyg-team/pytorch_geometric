Title: IBM Graphical output is missing from the Resources Usage Overview panel - United States

Text:
 TECHNOTE (TROUBLESHOOTING)

PROBLEM(ABSTRACT)
 In the IBM Cloud Orchrestrator Administration user interface, the Resources Usage > Stats panel no longer displays any graphical instance metrics. 

SYMPTOM
You expand the ADMIN > System Panel section, click Resource Usage, and then select the Stats tab. Instead of the graphical instance metrics, you see the following error message: 

Error: An error occurred. Please try again later.
Daily reports request end with error:
Something went wrong!
An unexpected error has occurred. Try refreshing the page. If that doesn't help, contact your local administrator.


CAUSE
The graphic does not display due to a known issue with OpenStack Icehouse.


RESOLVING THE PROBLEM
The issue is fixed in IBM Cloud Manager for OpenStack V4.1.0 Fix Pack 3. Install IBM Cloud Orchestrator 2.4.0 Fix Pack 2 [http://www.ibm.com/support/fixcentral/swg/quickorder?parent=ibm%7ETivoli&product=ibm/Tivoli/IBM+SmartCloud+Orchestrator&release=2.4.0.1&platform=All&function=fixId&fixids=2.4.0-CSI-ICO-FP0002&includeRequisites=1&includeSupersedes=0&downloadMethod=ddp&source=fc], which includes the IBM Cloud Manager for OpenStack update. For more information on IBM Cloud Orchestrator V2.4.0 Fix Pack 2, see the Release Notes [http://www.ibm.com/support/docview.wss?uid=swg21883430] document.
