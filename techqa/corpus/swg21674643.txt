Title: IBM Security vulnerability found in IBM WebSphere Application Server shipped with IBM  PureApplication System (CVE-2014-0114) - United States

Text:
security vulnerability; WebSphere Application Server; PureApplication System; classloader; manipulation FLASH (ALERT)

ABSTRACT
 Information about a security vulnerability affecting IBM WebSphere Application Server (shipped as a component of IBM PureApplication System) has been published in a security bulletin. 

CONTENT
IBM WebSphere Application Server is shipped as a component of IBM PureApplication System. Information about a security vulnerability affecting IBM WebSphere Application Server has been published in a security bulletin. 

Vulnerability Details: 

Please consult the security bulletin, Classloader Manipulation Vulnerability in IBM WebSphere Application Server CVE-2014-0114 [http://www-01.ibm.com/support/docview.wss?uid=swg21672316] for vulnerability details. 

 

Affected Products and Versions: 

Principal Product and Version(s) Affecting Supporting Product and Version(s) PureApplication System 1.0 WebSphere Application Server V6.1 PureApplication System 1.1 WebSphere Application Server V7 
Related Information 
IBM Secure Engineering Web Portal [https://www-304.ibm.com/jct03001c/security/secure-engineering/]
