Title: IBM Security Bulletin: Multiple vulnerabilities in IBM Java Runtime affect IBM Security Access Manager for Web (CVE-2014-4263, CVE-2014-4244) - United States

Text:
 SECURITY BULLETIN

SUMMARY
 There are multiple vulnerabilities in IBM® Runtime Java™ Technology Edition, Version 7.0 Service Refresh 7 and earlier that is used by IBM Security Access Manager for Web. These issues were disclosed as part of the IBM Java SDK updates in July 2014. 

VULNERABILITY DETAILS

CVEID: CVE-2014-4263 [http://cve.mitre.org/cgi-bin/cvename.cgi?name=CVE-2014-4263]

DESCRIPTION: An unspecified vulnerability related to the Security component of the IBM® Runtime Environment Java™ Technology Edition has partial confidentiality impact, partial integrity impact, and no availability impact.

CVSS:
CVSS Base Score: 4
CVSS Temporal Score: See https://exchange.xforce.ibmcloud.com/vulnerabilities/94606 [https://exchange.xforce.ibmcloud.com/vulnerabilities/94606] for the current score
CVSS Environmental Score*: Undefined
CVSS Vector: (AV:N/AC:H/Au:N/C:P/I:P/A:N)



CVEID: CVE-2014-4244 [http://cve.mitre.org/cgi-bin/cvename.cgi?name=CVE-2014-4244]

DESCRIPTION: An unspecified vulnerability related to the Security component of the IBM® Runtime Environment Java™ Technology Edition has partial confidentiality impact, partial integrity impact, and no availability impact.

CVSS:
CVSS Base Score: 4

AFFECTED PRODUCTS AND VERSIONS
IBM Security Access Manager for Web 7.0 (Software)
IBM Security Access Manager for Web 7.0 (Web Gateway Appliance)
IBM Security Access Manager for Web 8.0. Firmware versions 8.0.0.2, 8.0.0.3, and 8.0.0.4

REMEDIATION/FIXES
IBM Security Access Manager for Web software version 7.0 

IBM Security Access Manager for Web (ISAM) 7.0 software is not directly affected. However, IBM WebSphere is shipped as a component of ISAM 7.0 software. Information about a security vulnerability affecting IBM WebSphere has been published in a separate security bulletin. Please consult the security bulletin "Security Bulletin: Multiple vulnerabilities in IBM Java SDK affect WebSphere Application Server July 2014 CPU" [http://www-01.ibm.com/support/docview.wss?uid=swg21680418] for vulnerability and patch details.


IBM Security Access Manager for Web versions 7.0 and 8.0 ADKs

All versions of IBM Security Access Manager for Web include a separately downloadable Application Developer Kit (ADK). This ADK includes a version of the IBM Java SDK, which is affected by these vulnerabilities. Customers using this SDK should upgrade to the latest version. Please consult the security bulletin "Security Bulletin: Multiple vulnerabilities in current releases of IBM® SDK, Java™ Technology Edition" [http://www-01.ibm.com/support/docview.wss?uid=swg21680334] for vulnerability and patch details. 


IBM Security Access Manager for Web appliance versions 7.0 and 8.0 

IBM has provided patches for all affected versions. Follow the installation instructions in the README files included with the patch.
