CAUSE
Cached information from old version

RESOLVING THE PROBLEM
This error can be seen if a newer MC version (ie. 10.2) was installed and then it was decided to go back to an older MC version (ie. 6.5.2). MC caches certain information in a folder, C:\Users\<install user>\AppData\Roaming\IBM\Management Console\ (by default), that conflicts with the current installed version. 

In order to fix this you can delete the contents of this folder (configuration, database and workspace) to start fresh. 

You might want to back up the "database" folder (or just not delete it) if you have MC project settings that you want to keep. 

After removing the contents a reboot is recommended before running the new install again.
