CVE-2015-0204 SECURITY BULLETIN

SUMMARY

VULNERABILITY DETAILS
CVEID: CVE-2015-0204 [http://cve.mitre.org/cgi-bin/cvename.cgi?name=CVE-2015-0204]

AFFECTED PRODUCTS AND VERSIONS
-DataQuant Version 1 Release 2, Fix Pack 21 (and lower) 

-DataQuant Version 2 Release 1, Fix Pack 3 (and lower)

REMEDIATION/FIXES
You should verify applying this fix does not cause any compatibility issues. 

Install an updated Java JRE for your version of DataQuant, available for download from IBM Fix Central [http://www-933.ibm.com/support/fixcentral/].
