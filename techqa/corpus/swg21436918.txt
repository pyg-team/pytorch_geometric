Title: IBM Restoring an event that was marked as cleared in SiteProtector - United States

Text:
restore; event; clear; analysis; view; cleared TECHNOTE (FAQ)

QUESTION
 How do you restore an event that was marked as cleared in SiteProtector? 

ANSWER
To restore an event that was previously marked as cleared, go through the instructions below: 

 1. In the SiteProtector Console, open the Analysis view and go to View > Add/Remove Columns from the menu bar.
    
 2. Add the Cleared Count column from Available to the Displayed side.
    
 3. Save the settings and refresh the Analysis view. You should now be able to see the cleared events.
    
 4. Now, right-click the event that you want to unclear and select Restore Events.
