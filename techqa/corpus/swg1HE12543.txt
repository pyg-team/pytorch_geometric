APAR STATUS
 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  A message "System error has occured  on the client" is thrown
   in the browser while creating a collection plan after upgrading
   to Atlas 6.0.3.3 Interim Fix1. There is no error in the Atlas
   log file and you will not be able to create a collection plan.
   This happens only when the "Ongoing" check box is unchecked and
   the due is not populated
   
   
    
   
   

LOCAL FIX
 *  Populate the due date for the Collection Plan
   
   
    
   
   

PROBLEM SUMMARY
 *  A message "System error has occured  on the client" is thrown
   in the browser while creating a collection plan after upgrading
   to Atlas 6.0.3.3 Interim Fix1. There is no error in the Atlas
   log file and you will not be able to create a collection plan.
   This happens only when the "Ongoing" check box is unchecked and
   the due is not populated
   
   
    
   
   

PROBLEM CONCLUSION

   Fix pack 4
