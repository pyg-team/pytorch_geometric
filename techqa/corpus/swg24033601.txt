ABSTRACT
 This cumulative fix pack updates Rules for COBOL V7.1.1 (7.1.1, 7.1.1.1, 7.1.1.3 or 7.1.1.4) to V7.1.1.5. 

DOWNLOAD DESCRIPTION
This cumulative fix pack updates Rules for COBOL V7.1.1 (7.1.1, 7.1.1.1, 7.1.1.3 or 7.1.1.4) to V7.1.1.5. 

The fix pack is not compatible with Rules for COBOL V7.0.X and V7.1.0. Update to V7.1.1 to be able to apply this fix pack.

For details on problems fixed in Fix Pack 4, refer to the Fix list for V7.1 [http://www-01.ibm.com/support/docview.wss?uid=swg27019787#7114] document.

This fix pack is entitled. Only customers with Rules for COBOL V7.1.1 can download this fix pack.
