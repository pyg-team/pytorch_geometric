APAR STATUS
 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  When you have created multiple versions of the same virtual
   system pattern in IPAS, then the "Import Resource Template From
   Cloud" dropdown will create a listing for each version, but you
   will not be able to distinguish between them. The versions will
   appear as duplicate entries in the dropdown.
