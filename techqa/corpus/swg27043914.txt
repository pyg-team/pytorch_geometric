 DETAILED SYSTEM REQUIREMENTS

ABSTRACT
 This document contains system requirements for Sterling Secure Proxy. 

CONTENT
The detailed system requirements information is available through the Software Product Compatibility Reports [http://publib.boulder.ibm.com/infocenter/prodguid/v1r0/clarity/index.jsp] web site. This web site provides you will the ability to dynamically generate operating system, prerequisite, server virtualization environment, translation, end of service, and detailed system requirements reports for your specific product, release, and operating system. 

This document describes the minimum product levels that need to be installed before opening a problem report with Sterling Secure Proxy. Because other products frequently ship fixes, updates, and new releases, testing every configuration is not possible. In general and unless otherwise stated, you can install and run with updates to supported products if those updates are forward compatible.
