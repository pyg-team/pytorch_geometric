Title: IBM DC Configuration fails on Webphere Portal Server 8.0.x - United States

Text:
ITCAM; WPS; Data Collector; installation; installer; CAMADALL; CAMADPORTDC TECHNOTE (FAQ)

QUESTION
 DC Graphical configuration from MTEMS fails and exits at the step "Generating Configuration Template". Is WPS 8.0.x supported? 

CAUSE
WPS 8.0.x is not currently supported with ITCAM for AD v7.1 version.

ANSWER
WPS 8.0.x is not supported yet with ITCAM for AD 7.1 DC.

This can be checked on Clarity site.
The latest supported version for Websphere Portal Server is 7.0.

On the other hand, Websphere Portal Server 8.0.x is supported with ITCAM Agent for Websphere Applications v7.2, as you can see on the same Clarity site.

There are no plans at the moment to introduce WPS 8.x support in v7.1 maintenance stream.
