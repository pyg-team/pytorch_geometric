APAR STATUS
 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  Customer is monitoring the  SAP system and when monitored SAP
   instance is stopped, then sap agent crashed. This issue is
   occurring rarely.
   
   Environment:
   ITM for Applications 6.2 IF0013
   AIX 6.1
   
   Detailed Recreation Procedure:
   1. Install sap agent IF0013 to monitor a SAP Instance.
   2. Start SAP Agent and observe for 15 mins
   3. Stop SAP Central Instance. This will crash sap agent.
   NOTE: If not reproduce then in 2nd step and observe for 1 hr
   then perform 3rd.
   
   Approver: Shen Gang
