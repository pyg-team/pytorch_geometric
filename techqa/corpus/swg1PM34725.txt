APAR STATUS
 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  The WCM Authoring portlet throws an exception if the following
   conditions are true:
   
   1. You are attempting to use relative library access (./ dot
   notation)
   inside an <Element> tag in a Menu Component or Presentation
   Template.
   
   2. The 'defaultLibrary' property in the
   WCMConfigService.properties
   file is pointing to a WCM Library that the currently logged-in
   user
   does not have access to.
   
   The error you will see is:
   
   IWKAU1000E:An error occurred parsing the HTML content. Review
   HTML and
   try again.
   Errors found validating element No result design
   IWKPC1202X: At line 4 : IWKPC1058X: Cannot convert tag: <Element
   context="selected"
   name="./site/siteArea/content"
   
   type="content"
   key="Content_Body"/>
   
   
    
   
   

LOCAL FIX
 *  In the WCMConfigService.properties file, change the
   defaultLibrary parameter to a library the user has access to.
   
   
    
   
   

PROBLEM SUMMARY
 *  Using './' notation in an element tag causes error for non-admin
