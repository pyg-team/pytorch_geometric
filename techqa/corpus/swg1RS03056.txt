 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  Show an error (a warning) if a library contains the same XOM
   multiple times. With the same version it breaks the deployment
   and with different version the execution is going to behave
   unexpectedly.
   
   
    
   
   

LOCAL FIX

PROBLEM SUMMARY
 *  The fix is, during decision service deployment abort deployment
   with an error when trying to deploy multiple XOM resources with
   same name.
   
   
    
   
   

PROBLEM CONCLUSION
 *  The code is fixed
