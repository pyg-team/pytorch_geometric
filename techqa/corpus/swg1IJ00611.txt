APAR STATUS
 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  1. In Automation Script create a new Script with Object Launch
   Point :
   - Lauch Point: ADDEVT
   - Object: TICKET
   - Events: SAVE
   - Save: Add / Before Save
   2. Click next to second step:
   - Script: ADDEVT
   - Click New to add a variable
   - Variable: TICKETID
   - Launch Point Attribute: TICKETID
   3. Click next to third step:
   - In source code, add some script, such as TICKETID="123"
   4. Click create to create the new script
   5. Open the new script again and click Launch point
   6. Click Edit Launch Point Details button
   7. Select Initialize Value in Event section
   Unexpected behaviour:
   The Save section will be grey out, but the Save and Before Save
   will still be selected.
   If click OK - Save the change, then reopen the script and click
   Edit Launch Point Details, the Event would remain Save
   unchanged.
   
   
    
   
   

LOCAL FIX
 *  You may manually remove the tick of Add in Save section and
   change the Event to Initialize, save the Object Launch Point,
   the change will be saved properly.
