Title: IBM Security Bulletin: Vulnerability in OpenSSL affects IBM MQ Light (CVE-2015-3197) - United States

Text:
 SECURITY BULLETIN

SUMMARY
 OpenSSL vulnerabilities were disclosed on January 28, 2016 by the OpenSSL Project. OpenSSL is used by IBM MQ Light. IBM MQ Light has addressed the applicable CVEs. 

VULNERABILITY DETAILS
CVEID: CVE-2015-3197 [http://cve.mitre.org/cgi-bin/cvename.cgi?name=CVE-2015-3197]
DESCRIPTION: OpenSSL could allow a remote attacker to conduct man-in-the-middle attacks, caused by an error related to the negotiation of disabled SSLv2 ciphers by malicious SSL/TLS clients. An attacker could exploit this vulnerability to conduct man-in-the-middle attacks.
CVSS Base Score: 5.4
CVSS Temporal Score: See https://exchange.xforce.ibmcloud.com/vulnerabilities/110235 [https://exchange.xforce.ibmcloud.com/vulnerabilities/110235] for the current score
CVSS Environmental Score*: Undefined
CVSS Vector: (CVSS:3.0/AV:N/AC:L/PR:N/UI:R/S:U/C:L/I:L/A:N)

AFFECTED PRODUCTS AND VERSIONS
The vulnerabilities affect users of the mqlight [https://www.npmjs.com/package/mqlight] IBM MQ Light Client Module for Node.jsÂ® on all platforms at the following versions:
1.0.2014090800
1.0.2014090801
1.0.2014091000-red
1.0.2014091001

It also affects users of the mqlight-dev [https://www.npmjs.com/package/mqlight-dev] IBM MQ Light Client Module for Node.js between versions 1.0.2014090300 and 1.0.2014111002 inclusive.

REMEDIATION/FIXES
Users of the IBM MQ Light Client Module for Node.js at an affected version should update to the latest version of the IBM MQ Light Client Module for Node.js as found on https://www.npmjs.com [https://www.npmjs.com/].

WORKAROUNDS AND MITIGATIONS
