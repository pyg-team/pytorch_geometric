Title: IBM DataPower WebGUI left navigation pane might not display correctly after a firmware rollback - United States

Text:
700limitation xg xi xb ve webgui navigation pane missing empty display TECHNOTE (TROUBLESHOOTING)

PROBLEM(ABSTRACT)
 The DataPower WebGUI might not display the left navigation pane after a firmware rollback from firmware version 7.0.0 to an earlier version. 

CAUSE
The DataPower WebGUI might not display the left navigation pane if you install version 7.0.0 or later of the firmware and then rollback or install an earlier firmware. For example, if you install 7.0.0 and rollback to version 6.0.1, this issue might occur.

RESOLVING THE PROBLEM
To resolve this issue, clear the cache in your browser and refresh the page. The left navigation pane is available again.
