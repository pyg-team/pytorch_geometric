ERROR DESCRIPTION
 *  CLEANER : Ghost link from specific CATDrawing
   can not be cleaned
   .
   Scenario:
   1. Open attached CATDrawing.
   2. This CATDrawing has ghost link. To confirm
   check the results of File Send to and Edit link. File
   send to shows two files, where as in session we
   have only one CATDrawing opened. And Edit link
   shows there is no external link with this
   CATDrawing.
   2. Select File / Desk / CATDUAV5 and clean the
   data.
   Problem: No Error is detected and cannot clean the
   Ghost Link.

   Error should be detected. And the error should be
   cleaned with CATDUAV5.
