ERROR DESCRIPTION
 *  The Send Reminder Notice Task throws an error:
   java.sql.SQLException: ORA-12899: value too large for Column
   "PSSAPL"."HOLD_NOTICE_ESCALATION_RULE"."AS_SENT_SUBJECT""
   This error occurs when a Hold Notice email subject exceeds the
   current limit of 250 characters. This can happen when the
   subject has predefined variables such as $MatterName which gets
   replaced with actual values.
   
   
    
   
   

LOCAL FIX
 *  No workaround
   
   
    
   
   

PROBLEM SUMMARY
 *  Send Reminder Task shows failure messages if subject exceeds 250
    characters when the variables in Subject get expanded.Email is
   sent but the DB is not updated with the email sent details.
