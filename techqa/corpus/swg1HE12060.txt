APAR STATUS
 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  From a legal request's people in scope page, people are not
   searchable using the assistant or manager search criteria.
   
   
    
   
   

LOCAL FIX

PROBLEM SUMMARY
 *  While adding people in scope for a request, if you search for
   people using either the assistant or manager search criteria,
   incorrect results are returned.  When searching by manager
   criteria, no results are returned.  When searching by assistant
   criteria, all people in Atlas are returned.
   
   
    
   
   

PROBLEM CONCLUSION
 *  The fix is available now in Fix Central as part of 6.0.3
   iFix001.
