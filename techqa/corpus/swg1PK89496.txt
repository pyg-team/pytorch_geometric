LOCAL FIX

PROBLEM SUMMARY
 *  1. Help should explain that there are three (not two) ways
   of deploying a Portlet Project: Export, Deploy, and Run On
   Server. Run on Server method was not avaible.
   2. Help should have a page that explains what exactly Run On
   Server does and
   3. Help should clarify that the two approaches should not be
   mixed.
   
   
    
   
   

PROBLEM CONCLUSION
 *  1. Documenated three ways of deploying a Portlet Project:
   Export, Deploy, and Run On Server. Added the missing Run on
   Server method.
   2. Added a new page that explains what exactly Run On Server
   does.
   3. And added a note stating that the Depolyment and Run on
   Server publishing methods are different from each other.
