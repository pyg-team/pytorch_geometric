Title: IBM Security Bulletin: Security vulnerabilities have been identified in IBM Data Science Experience Local - Cryptography.InsecureAlgorithm SHA-1 - United States

Text:
 SECURITY BULLETIN

SUMMARY
 IBM Data Science Experience Local - Information about security vulnerability concerning use of SHA-1 encryption algorithm 

VULNERABILITY DETAILS
DSX Local used the SHA-1 encryption algorithm. 

Please consult this advisory for more information about the security vulnerability. 

CWE-327 [https://cwe.mitre.org/data/definitions/327.html]

AFFECTED PRODUCTS AND VERSIONS
 

Principal Product and Version(s) IBM Data Science Experience Local: 1.1.0, 1.1.1, 1.1.2, 1.1.3 REMEDIATION/FIXES
Upgrade to the 1.2 release of IBM Data Science Experience Local. You can download the latest release from Passport Advantage.
