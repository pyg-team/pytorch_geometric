IBM InfoSphere Information Server Workgroup Edition is designed to offer data integration capabilities in more flexible and innovative ways to meet client requirements. IBM InfoSphere Information Server Workgroup Edition V8.5 enables companies to flexibly purchase key data integration capabilities to meet the needs of small information integration projects. 

InfoSphere Information Server combines the technologies of IBM InfoSphere Business Glossary, IBM InfoSphere DataStage, IBM InfoSphere FastTrack, IBM InfoSphere Information Analyzer, IBM InfoSphere Information Services Director, IBM InfoSphere Metadata Workbench, and IBM InfoSphere QualityStage into a single unified platform that enables companies to understand, cleanse, transform, and deliver trustworthy and context-rich information.

Downloading the product

Your proof of entitlement directs you to your license file. This license file controls which of the product modules you can install and run.

To download the product: 

 1. Go to the Find Downloads and Media [http://www.ibm.com/software/howtobuy/passportadvantage/paocustomer/sdma/SDMA?PO=DOWNLOAD_FIND_BY_PART_NUMBER] Web page and log in.
 2. Under the Download finder options, select the Find by part number search option.
 3. Search for each downloadable image by the part number.
 4. Download all the required images.
 5. Extract each of the images into a single, temporary directory on your system. Use a file extracting utility that supports large file extractions.
 6. Download the Quick Start Guide for IBM InfoSphere Information Server for Workgroup Edition [http://www.ibm.com/support/docview.wss?uid=swg27020975].

After receiving a proof of entitlement, you can download any or all of the installation images listed below.

Installation images Only the Microsoft Windows installation image contains both the server and the client installation software. To install the product on operating systems other than Microsoft Windows, you must download both the Microsoft Windows installation image, for the client installation, and the installation image for the operating system onto which you want to install the server.

Go to the Microsoft Windows download document [http://www.ibm.com/support/docview.wss?uid=swg24029565] to download the installation image for Microsoft Windows.
