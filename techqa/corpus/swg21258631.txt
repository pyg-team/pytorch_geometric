Title: IBM Configuring the Desktop to launch Autodesk AutoCAD or Bentley MicroStation - United States

Text:
 TECHNOTE (FAQ)

QUESTION
 How do I configure the IBM® DB2® Document Manager Desktop to launch Autodesk AutoCAD or Bentley MicroStation when a user views or edits an AutoCAD or MicroStation document? 

ANSWER
To configure the Desktop to launch Autodesk AutoCAD or Bentley MicroStation: 

1. Click Start -> Programs -> IBM DB2 Document Manager -> Designer. 

2. Expand the database, Global - [Administrator], Desktop, and Desktop Templates. 

3. Right-click the Default Template or one of your own Desktop templates and select Modify (non-wizard) from the popup menu. The Modify Desktop Template Configuration window opens. 

4. Click the Desktop Applications tab and provide the following values: 

 * 
 * 
 * Handling of non-specified file types (default viewer)
 * 
 * Advanced
 * 
 * b. From the File type list, select either AutoCAD or MicroStation. 
 * c. From the Application type list, select one of the following values: 
 *  
   Viewing  * 
    * 
    * Field type
    * 
   
   
   Authoring  * 
    * 
    * Field type
    * 
   
   
 * 
 * Launch associated application
 * 
 * Browse
 * 
 * e. Optional: In the Arguments field, type any arguments or command line options. For example, specify the variable %filename%. 
 * f. Click Add to add the file type/application mapping to the table. 
 * g. Click Done to save the configuration and return to the Desktop Applications page.


5. Click OK to save the configuration and close the Modify Desktop Template Configuration window.
