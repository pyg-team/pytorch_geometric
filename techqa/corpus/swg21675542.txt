Title: IBM Active Directory users are unable to log into SPSS Deployment Manger - United States

Text:
log in Active Directory Deployment Manager Portal bind password TECHNOTE (TROUBLESHOOTING)

PROBLEM(ABSTRACT)
 Active Directory users are unable to log into SPSS Deployment Manger 

CAUSE
The Domain User Password set in the Active Directory security provider is no longer valid.

DIAGNOSING THE PROBLEM
A review of the systemout.log shows the following: 

[5/29/14 15:34:10:005 EDT] 00000037 SystemOut O 15:34:10,004 WARN 
[LDAPPagedQuery] javax.naming.NamingException: [LDAP: error code 1 - 
000004DC: LdapErr: DSID-0C0906DD, comment: In order to perform this 
operation a successful bind must be completed on the connection., data 
0, v1772 

This indicates a problem with the bind operation. 

RESOLVING THE PROBLEM
In the Server Administrator, double click on the Active Directory security provider and reenter the Domain User Password. If you do not know the password for the user, you will need to ask the Active 
Directory administrator.
