Title: IBM Security Bulletin: Information regarding security vulnerability in IBM SDK for Java that is shipped with IBM WebSphere Application Server and addressed by Oracle CPU October 2013 - United States

Text:
 SECURITY BULLETIN

SUMMARY
 Multiple security vulnerabilities exist in the IBM SDK for Java that is shipped with IBM WebSphere Application Server and included in the products that are listed in this document. 

VULNERABILITY DETAILS
The products that are listed in the Affected product section are shipped with a version of IBM WebSphere Application Server which is shipped with an IBM SDK for Javaâ„¢. That SDK is based on the Oracle Java Development Kit (JDK). Oracle has released the October 2013 critical patch updates (CPU), which contain security vulnerability fixes and the IBM SDK for Java has been updated to incorporate the critical patch updates.
The following vulnerabilities are applicable to WebSphere Application Server and for the product listed below.

CVEID: CVE-2013-5780 [http://cve.mitre.org/cgi-bin/cvename.cgi?name=CVE-2013-5780]
Description: Potential information disclosure vulnerability in Java Secure Sockets Extension (JSSE). 
CVSS Base Score: 4.3
CVSS Temporal Score: See https://exchange.xforce.ibmcloud.com/vulnerabilities/88001 [https://exchange.xforce.ibmcloud.com/vulnerabilities/88001] for the current score
CVSS Environmental Score*: Undefined
CVSS Vector: (AV:N/AC:M/Au:N/C:P/I:N/A:N) 

CVEID: CVE-2013-5372 [http://cve.mitre.org/cgi-bin/cvename.cgi?name=CVE-2013-5372]
Description: Potential denial of service vulnerability in XML. This issue is specific to the IBM SDK for Java. 
CVSS Base Score: 4.3
CVSS Temporal Score: See https://exchange.xforce.ibmcloud.com/vulnerabilities/86662 [https://exchange.xforce.ibmcloud.com/vulnerabilities/86662] for the current score
CVSS Environmental Score*: Undefined
CVSS Vector: (AV:N/AC:M/Au:N/C:N/I:N/A:P) 

CVEID: CVE-2013-5803 [http://cve.mitre.org/cgi-bin/cvename.cgi?name=CVE-2013-5803]
Description: Potential denial of service vulnerability in JSSE.
CVSS Base Score: 2.6
CVSS Temporal Score: See https://exchange.xforce.ibmcloud.com/vulnerabilities/88008 [https://exchange.xforce.ibmcloud.com/vulnerabilities/88008] for the current score
CVSS Environmental Score*: Undefined
CVSS Vector: (AV:N/AC:H/Au:N/C:N/I:N/A:P)

Vulnerability fixes for other advisories are included in the IBM SDK for Java, but IBM WebSphere Application Server and the products that are listed in the Affected products section are not vulnerable to them. You need to evaluate your own code to determine if you are vulnerable. Refer to the Reference section for more information on the advisories not applicable to WebSphere Application Server.

 * WebSphere Process Server V6.1.2, 6.2.x, 7.0.x 
 * WebSphere Process Server on z/OS V6.2.x, 7.0.x 
 * WebSphere Process Server Hypervisor Edition for Red Hat Enterprise Linux Server for x86 (32-bit) V7.0.0 
 * WebSphere Process Server Hypervisor Edition for Novell SUSE Linux Enterprise Server for x86 (32-bit) V6.2.x, 7.0.x 
 * WebSphere Process Server Hypervisor Edition for Novell SUSE Linux Enterprise Server for System z V6.2.x, 7.0.x

REMEDIATION/FIXES
Install WebSphere Application Server interim fixes as appropriate for your current WebSphere Application Server version as described in Security Bulletin: Multiple vulnerabilities in current IBM SDK for Java for WebSphere Application Server October 2013 CPU. [http://www.ibm.com/support/docview.wss?uid=swg21655990]
