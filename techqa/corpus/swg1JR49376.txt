APAR STATUS
 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  After you upgrade to IBM BPM V8.0.1.2, the /teamworks/login.jsp
   does not render correctly in a BPM deployment environment that
   consists of multiple clusters.
   
   
    
   
   

LOCAL FIX
 *  N/A
   
   
    
   
   

PROBLEM SUMMARY
 *  ****************************************************************
   * USERS AFFECTED:  IBM Business Process Manager (BPM)          *
   *                  V8.0.1.2 Advanced,Standard, and Express     *
   ****************************************************************
   * PROBLEM DESCRIPTION: The /teamworks/login.jsp does not       *
   *                      render correctly after you upgrade to   *
   *                      IBM BPM 8.0.1.2, because the resource   *
   *                      URIs contain the string  'null', the    *
   *                      following example is of an incorrect    *
   *                      URI for the login_logo.png image and    *
   *                      the correct form of the URI for the     *
   *                      same resource:                          *
   *                      The incorrect location is               *
   *                      teamworks/null/contenthandler/mm/dav/fi *
   *                      lestore/login/bspace/images/login_logo. *
   *                      png                                     *
   *                      The correct location is                 *
   *                      /mum/contenthandler/mm/dav/filestore/lo *
   *                      gin/bspace/images/login_logo.png        *
   ****************************************************************
   * RECOMMENDATION:                                              *
   ****************************************************************
   After you upgrade to IBM BPM V8.0.1.2, the
   /teamworks/login.jsp does not render correctly.
   
   
    
   
   

PROBLEM CONCLUSION
 *  A fix for IBM BPM V8.0.1.2 is available so that
   /teamworks/login.jsp works correctly.
   
   On Fix Central (http://www.ibm.com/support/fixcentral), search
   for JR49376:
   
   1. Select the product group, product, installed version, and
   platform, and click Continue.
   2. Select APAR or SPR, enter JR49376, and click Continue.
   
   When downloading fix packages, ensure that you also download
   the readme file for each fix. Review each readme file for
   additional installation instructions and information about the
   fix.
   
   After you apply the fix, restart the Deployment Manager.
