Title: IBM On Statistics 23 for Macintosh, how to License a local Add On Module when getting Base from a Concurrent License - United States

Statistics 23; License; local Add On Module; Base Concurrent License; Mac; Macintosh TECHNOTE (FAQ)

QUESTION
 In Statistics 23 how do I license a local Add-On Module on my local client?

I am getting Statistics 23 base license from a Concurrent (Network) license. 

ANSWER
NOTE: Install location Change: 

 * 
 * 
 * 
 * 
 * <user home directory>/Applications/IBM/SPSS/Statistics/23 
 * 

 * 
 * 
 * 
 * <user home directory>/Applications/IBM/SPSS/Statistics/23 to find SPSS Statistics. 


Part 1: 
Here are the instructions to change the permissions on the license file 'lservrc'. 
1) Locate and launch the "Finder" application 
2) In the Finder menu find the users home directory (Icon is usually a house icon) 
3) Look for the "Applications" folder and click on it, followed by the "IBM", "SPSS", "Statistics", & "23" folders 
4) Double Click on the "23" folder 
5) Ctrl+click on SPSSStatistics.app. A pop-up menu will appear, click on "Show package contents" 
6) Double Click on Contents 
7) Double Click on Bin 
8) Locate the lservrc file and change the permissions for full Read & Write access. If this step cannot be completed successfully see your IT administrator as the next step will fail. 

Part 2: 
Here are the instructions to activate the local add on module authorization code, by using Terminal. 
1) Locate and launch the "Finder" application 
2) Within the Finder application locate and expand "Applications" 
3) Locate and expand "Utilities" 
4) Locate and launch the "Terminal" application 
5) Within Terminal Navigate to the IBM SPSS Statistics 23 installation directory and launch the licin.sh shell script as the 'root' user. Type the following commands: after each line press the "ENTER" key 

NOTE: When using the "sudo" command, you will be prompted for the computers administrator password. 
-- a) cd /Applications/IBM/SPSS/Statistics/23/SPSSStatistics.app/Contents/bin 
-- b) sudo ./licin.sh 

6) Follow the prompts and activate the local add on module. RELATED INFORMATION
