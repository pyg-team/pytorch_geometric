PROBLEM(ABSTRACT)
 A user attempts to run a customized BIRT report that contains a field with special character values

DIAGNOSING THE PROBLEM
The following error will be found in the SystemOut log: 

WSVR0605W: Thread "WebContainer : 1" (00000050) has been active for 756832 milliseconds and may be hung. There is/are 1 thread(s) in total in the server that may be hung.
at java.lang.StringBuffer.indexOf(StringBuffer.java:923)

at com.ibm.tivoli.maximo.report.script.MXReportDataSetImpl.getContent(MXReportDataSetImpl.java:1266)

at com.ibm.tivoli.maximo.report.script.MXReportDataSetImpl.getHtmlString(MXReportDataSetImpl.java:1151)


RESOLVING THE PROBLEM
The field was set to long description type and in the report it is accessed by getHtmlString method. 


Resolve the problem by using getString method rather than getHtmlString method in the report design.
