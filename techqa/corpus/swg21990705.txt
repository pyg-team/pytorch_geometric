Title: IBM Atlas Extensions org loader  unable to import certain orgs - United States

Text:
Atlas Extension Parent org TECHNOTE (FAQ)

QUESTION
 Why does the Atlas Extensions Org loader unable to able import certain organizations? 

CAUSE
The org throws an error message "Parent Org Identifier not found in
ORGUNIT table"

ANSWER
 

 1. Login to Atlas Database 
 2. Lookup the org identifier in the orgunit table 
 3. If not found, lookup the org identifier in the atlas_orgunit and orgunit_staging table 
 4. Identify if there are 2 org units with same org identifier for the same parent org 
 5. Work with org data team to correct the data 
 6. Re-run the Atlas loader to import the parent org unit and the corresponding Child org unit.
