APAR STATUS
 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  The Content Engine server cannot connect to the IBM Content
   Search Services server attcss3 located on host hqpsicss, port
   8199. IBM Content Search Services could not communicate with the
   server that runs on the hqpsicss:8199 host and port. The server
   is not started. Start the server. For information about starting
   the server, see "Starting or stopping IBM Content Search
   Services servers" in the IBM FileNet P8 information center.
   IQQD0040E The client specified the wrong authentication token.
   
   However, the message is not entirely accurate and a bit
   misleading with this statement "The server is not started. Start
   the server. For information about starting the server, see
   "Starting or stopping IBM Content Search Services servers" in
   the IBM FileNet P8 information center. "
   
   
    
   
   

LOCAL FIX

PROBLEM SUMMARY
 *  ****************************************************************
   * USERS AFFECTED:                                              *
   * Customers using Content Engine 5.1.0.4-P8CE-FP003  and prior *
   * connect to the server with the wrong token.                  *
   ****************************************************************
   * PROBLEM DESCRIPTION:                                         *
   * CSS server provided the inaccurate error message to indicate *
   * CSS server is not started when the wrong token is specific.  *
   ****************************************************************
   * RECOMMENDATION:                                              *
   * upgrade to the 5.1.0.4-P8CE-FP004 or later will resolve this *
   * problem.                                                     *
   ****************************************************************
   
   
    
   
   

PROBLEM CONCLUSION
 *  Fix the error message to indicate the wrong token is used in
   5.1.0.4-P8CE-FP004 and higher.
