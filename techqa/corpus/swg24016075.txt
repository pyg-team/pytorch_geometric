This fix pack contains fixes to the version 1.0 release of IBM Installation Manager.

IBM Installation Manager is a program that helps you install, update, modify, and uninstall packages on your workstation.

PREREQUISITES
Before you can install certain products, you must install IBM Installation Manager, Version 1.0; by default, it is automatically installed when you update or install packages by using an earlier version.

For more information, refer to the version 1.0.0.5 readme file at: [http://download.boulder.ibm.com/ibmdl/pub/software/rationalsdp/v7/im/1005/docs/readme/readme.html]http://download.boulder.ibm.com/ibmdl/pub/software/rationalsdp/v7/im/1005/docs/readme/readme.html [http://download.boulder.ibm.com/ibmdl/pub/software/rationalsdp/v7/im/1005/docs/readme/readme.html]

INSTALLATION INSTRUCTIONS
To download and install the fix pack, complete the following steps: 


 1. 
 2. Download the fix pack (im-1.0.0.5.zip) from the download URL below.
 3. 
 4. Extract the compressed file in an appropriate directory. For example, choose to extract to C:\temp.
 5. 
 6. Add the fix pack repository location in IBM Installation Manager:
     1. Start IBM Installation Manager.
     2. 
     3. On the Start page of Installation Manager, click File > Preferences, and then click Repositories. The Repositories page opens.
     4. 
     5. On the Repositories page, click Add Repository.
     6. 
     7. In the Add repository window, browse to or enter the file path to the repository.config file, which is located in the directory where you extracted the compressed files. For example, enter C:\temp\ or C:\temp\repository.config and then click OK.
    
    
    
 7. Installation of the fix pack will begin once you click on Install Packages or Update Packages.
 8.

 4.  Extract the compressed file into an appropriate directory.
 5.  
 6.  Run install.exe (Windows), or install (Linux). 
 7.  
 8.  If IBM Installation Manager is not currently installed, or if the installed version is older than 1.0.0.5, the Installation Manager installer will be launched. Follow the wizard pages to install it.
