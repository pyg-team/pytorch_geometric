Title: IBM SHA-256 signature change starting with FastBack 6.1.12.4 - United States

Text:
 PREVENTIVE SERVICE PLANNING

ABSTRACT
 Starting with v6.1.12.4 Fastback, SHA-256 signature is used. Some hot fixes need to be installed for SHA-256 signature support on Microsoft Windows platforms. 

CONTENT
Windows x86: 
For SHA-256 signature support on Microsoft Windows 7, please install the hot fixes associated with Microsoft Knowledge Base article 3033929 at: http://support.microsoft.com/default.aspx?scid=kb;EN-US;3033929 [http://support.microsoft.com/default.aspx?scid=kb;EN-US;3033929] and Microsoft Knowledge Base article 3035131 at: http://support.microsoft.com/default.aspx?scid=kb;EN-US;3035131 [http://support.microsoft.com/default.aspx?scid=kb;EN-US;3035131] 


Windows x64: 
For SHA-256 signature support on Microsoft Windows 7 or Microsoft Windows 2008 R2, please install the hot fixes associated with Microsoft Knowledge Base article 3033929 at: http://support.microsoft.com/default.aspx?scid=kb;EN-US;3033929 [http://support.microsoft.com/default.aspx?scid=kb;EN-US;3033929] and Microsoft Knowledge Base article 3035131 at: http://support.microsoft.com/default.aspx?scid=kb;EN-US;3035131 [http://support.microsoft.com/default.aspx?scid=kb;EN-US;3035131]
