APAR STATUS
 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  A limitation on debugging validation flow for coach in the
   runtime environment requires a warning message.
   
   AFFECTED PRODUCTS:
   IBM Business Process Manager (BPM) Advanced
   IBM BPM Standard
   IBM BPM Express
   
   
    
   
   

LOCAL FIX
 *  After you debug the validation path and review the debug
   information page, including tw.system.coachValidation error
   information, select one of the following options:
   
     -Run to continue.
     -Run to the next break point that is defined in the next step
   in a regular boundary event flow. This option moves to the next
   step in the regular boundary event flow.
   
   
    
   
   

PROBLEM SUMMARY
 *  A limitation on debugging validation flow for coach in the
   runtime environment (documented in JR47092 [http://www-01.ibm.com/support/docview.wss?uid=swg1JR47092]) requires a warning
   message to help you proactively debug the validation flow.
   
   
    
   
   

PROBLEM CONCLUSION
 *  Fixes for IBM BPM 8.0.1.0 and 8.0.1.2 are available on Fix
   Central (http://www.ibm.com/support/fixcentral), search
   for JR49076:
   
       1. Select the product group, product, installed version, and
   platform, and click Continue.
       2. Select APAR or SPR, enter JR49076, and click Continue.
   
   When downloading fix packages, ensure that you also download the
   readme file for each fix. Review each readme file for additional
   installation instructions and information about the fix.
