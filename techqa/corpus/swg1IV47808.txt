APAR STATUS
 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  The UNIX OS Agent has been enhanced with additional attribute
   groups when running on AIX. Attributes have been migrated from
   the AIX Premium agent, as well as various memory attributes that
   correspond to the nmon, svmon, and vmstat AIX commands.
   
   
    
   
   

LOCAL FIX

PROBLEM SUMMARY
 *  Add additional AIX specific attributes to the UNIX OS Agent.
   
   
   On AIX, the Monitoring Agent for UNIX OS has added multiple new
   attributes and attribute groups.  Attributes have been added for
   memory statistics, volume groups, as well as metrics previously
   available only in the AIX Premium Agent.
   
   Refer to the UNIX OS Agent User's Guide for specific details on
   the attributes added.
