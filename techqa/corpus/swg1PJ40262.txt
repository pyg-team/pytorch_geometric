APAR STATUS
 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  When Single Sign-On (SSO) using Kerberos or SPNEGO is enabled,
   the Export Solution Package wizard in Case Manager
   Administration Client
   generates the following error when exporting a solution from a
   development environment:
   
   The solution package for the solution or template could not be
   created. Use your case management installation and configuration
   checklist to verify the Content Engine server property values
   that you entered in the wizard and ensure that the Content
   Engine server is running.
   
   Click OK to return to the wizard to review and correct your
   settings.
   
   Click Window > Show View > Error Log to review the log for more
   information.
   FNRPA9001E The Business Space server is not available. Verify
   the URL, the administrative user name, and the password.
   
   
    
   
   

LOCAL FIX
 *  Disable SSO when exporting a solution from the development
   environment.
   
   
    
   
   

PROBLEM SUMMARY
 *  ****************************************************************
   * USERS AFFECTED:                                              *
   * System Administrators who want to run Case Manager           *
   * Administration Client's Export Solution Package wizard.      *
   ****************************************************************
   * PROBLEM DESCRIPTION:                                         *
   * See Error Description                                        *
   ****************************************************************
   * RECOMMENDATION:                                              *
   * To fix this problem, apply IBM Case Manager V5.1.0.1 Interim *
   * Fix 1 or later releases.                                     *
   ****************************************************************
   
   
    
   
   

PROBLEM CONCLUSION
 *  The fix for this problem is provided in IBM Case Manager
   V5.1.0.1 Interim Fix 1 and later releases.
   
   A fix was made in the Case Manager Administration Client (CMAC)
   such that when Single Sign-On (SSO) using Kerberos/SPNEGO is
   enabled on a development environment, the Export Solution
   Package wizard in Case Manager Administration Client succeeds.
