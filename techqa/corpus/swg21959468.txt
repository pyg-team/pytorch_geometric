Installation information: 


Important: In order to download and install this fix pack using the IBM Installation Manager, you must ensure that you are using IBM Installation Manager version 1.8.1 or higher. By default, IBM Installation Manager should check for updates for itself, but if it has not, you can click File → Preferences → Updates and select the Search for Installation Manager updates check box and restart the application. 

For installation instructions, see the following IBM InfoSphere Master Data Management version 11.4 Knowledge Center topic: 

http://www-01.ibm.com/support/knowledgecenter/SSWSR9_11.4.0/com.ibm.mdmhs.mdmtoolkit_install.doc/Topics/t_extractingandinstallingmdmapplicationtoolkitforbpmfiles.html [http://www-01.ibm.com/support/knowledgecenter/SSWSR9_11.4.0/com.ibm.mdmhs.mdmtoolkit_install.doc/Topics/t_extractingandinstallingmdmapplicationtoolkitforbpmfiles.html] 
Important Changes: Enhancements : This fix pack adds the new Advance Hierarchy and Relationship visualization widget to the Application Toolkit. This feature is comprised of both the Stewardship Center coaches for graphical tree and network display as well as updates to supporting MDM services. This provides the ability to view hierarchy data either as a tree or graph layout along with the ability to customize the styling of the data using BPM rules.
