Title: IBM Installing HATS 9.0 fails when RAD 9.1 is installed on host - United States

Text:
installation; HATS 9.0; RAD 9.1; HATS 9 TECHNOTE (TROUBLESHOOTING)

PROBLEM(ABSTRACT)
 Attempts to install IBM Rational Host Access Transformation Services (HATS) 9.0 on a system with Rational Application Developer for WebSphere Software (RAD) 9.1 fails. 

CAUSE
HATS 9.0 is not supported with RAD 9.1.

RESOLVING THE PROBLEM
You will need to install latest HATS fixpack at the same time as the initial HATS install of v9.0.0. 

To install both the fixpack and the product you will need to add both repositories to Installation Manager by following the steps below. 


 1. Start Installation Manager
    
    
 2. Click on File->Preferences
    
    
 3. Click the Add Repository.... button
    
    
 4.  Add two repositories, one for HATS 9.0.0, and one for 9.0.0.1 or the HATS latest fixpack
    
    [/support/docview.wss?uid=swg21682626&aid=1] [/support/docview.wss?uid=swg21682626&aid=1]
    
    
 5.  Click Apply and OK 
    
    
 6. Click on the Install button
