ERROR DESCRIPTION
 *  ERROR DESCRIPTION:
   Mouse cursor is sticking to Help > System Information dialog
   box when using Chrome.
   
   PROBLEM:
   When dialog box was dragged, mouse cursor will stick to the box
   even after the mouse button is released. You will have to click
   the left mouse button again to release the dialog box from the
   pointer. This is replicable in 7604, 7605, and 7606 environment
   using Chrome; it works fine with IE and Firefox.
   
   PERFORMANCE ISSUE:  No
   
   STEPS TO REPRODUCE:
   1) Sign into MAM 7.6 with maxadmin using Chrome.
   2) From "Help", select "System Information".
   3) Click the title bar of "System Information" dialog and drag
   it to just a little bit left or right or up or down or anywhere.
   4) Release the mouse button and then move the mouse.
   5) Now you can see the dialog is sticked to the mouse pointer
   and move together everytime you move the mouse.
   
   CURRENT ERRONEOUS RESULT:
   Mouse pointer is sticking to dialog box.
   
   EXPECTED RESULT:
   Cursor should released the dialog box after mouse button is
   released.
   
   ADDITIONAL INFO:
