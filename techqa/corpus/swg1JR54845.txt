PROBLEM SUMMARY
 *  ****************************************************************
   USERS AFFECTED:
   Users of the DataStage DSXImportService commandline.
   ****************************************************************
   PROBLEM DESCRIPTION:
   The DataStage DSXImportService commandline can fail with a
   StringIndexOutOfBoundsException when importing DataStage job
   runtime which contains a column called "JobType"
   ****************************************************************
   RECOMMENDATION:
   Apply patch on Services tier, and on Client and Engine tiers if
   separate and if the DSXImportService is being called from those
   tiers.
   ****************************************************************
   
   
    
   
   

PROBLEM CONCLUSION
 *  Added checking to prevent StringIndexOutOfBoundsException.
   
   
    
   
   

TEMPORARY FIX
 *  Can use Designer import or Windows dscmdimport command to import
    job instead.
