APAR STATUS
 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  When there are a large number of SQL connections configured on
   an appliance, quiescing or disabling all domains may lead to an
   unexpected restart of the appliance which does not produce a
   backtrace.
   
   
    
   
   

LOCAL FIX

PROBLEM SUMMARY
 *  Datapower system that has large number of SQL Data Source
   configured might restart while disabling domains or
   quiesceding the appliance.
   
   
    
   
   

PROBLEM CONCLUSION
 *  Fix is available in 6.0.0.16, 6.0.1.12, 7.0.0.9, 7.1.0.6
   and 7.2.0.0.
