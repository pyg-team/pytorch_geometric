 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  WSRR Eclipse plugin cannot connect to servers that require TLS
   v1.1 or v1.2 connections.
   
   
    
   
   

LOCAL FIX

PROBLEM SUMMARY
 *  ****************************************************************
   USERS AFFECTED:
   All users of the Eclipse plugin for IBM WebSphere Service
   Registry and Repository V8.5
   
   ****************************************************************
   PROBLEM DESCRIPTION:
   The Eclipse plugin does not support connections requiring the
   TLS V1.1 nor V1.2 protocol.
   
   
    
   
   

PROBLEM CONCLUSION
 *  The Eclipse plugin was updated to support TLS V1.1 and V1.2
   connections.
   
   This fix is targeted for inclusion in WSRR V8.5.6.1
