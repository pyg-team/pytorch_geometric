Title: IBM InfoSphere Glossary Anywhere fails to connect to server with error "Client requested protocol TLSv1 not enabled or not supported" - United States

Text:
 TECHNOTE (TROUBLESHOOTING)

PROBLEM(ABSTRACT)
 InfoSphere Glossary Anywhere fails to connect to server error with error "Client requested protocol TLSv1 not enabled or not supported" 

Glossary Anywhere is version BGASetup-11.5.0.1.20160309-1348

The error in IIS SystemOut.log mentions that Glossary Anywhere is requesting
TLSv1 instead of TLSv1.2:

[10/16/17 9:37:07:267 EDT] 000000c6 SSLHandshakeE E SSLC0008E: Unable
to initialize SSL connection. Unauthorized access was denied or
security settings have expired. Exception is javax.net.ssl.
SSLHandshakeException: Client requested protocol TLSv1 not enabled or
not supported
at com.ibm.jsse2.A.B(A.java:32)
at com.ibm.jsse2.an.b(an.java:323)
at com.ibm.jsse2.an.c(an.java:164)
at com.ibm.jsse2.an.wrap(an.java:686)
at javax.net.ssl.SSLEngine.wrap(SSLEngine.java:39)
at com.ibm.ws.ssl.channel.impl.SSLUtils.handleHandshake
(SSLUtils.java:746)
at com.ibm.ws.ssl.channel.impl.SSLConnectionLink.readyInbound
(SSLConnectionLink.java:566)
at com.ibm.ws.ssl.channel.impl.SSLConnectionLink.ready
(SSLConnectionLink.java:295)




CAUSE
Glossary Anywhere is configured for TLS 1.1. The server is configured to only accept TLS 1.2. 

The Glossary Anywhere client config.ini file contains the TLS version in the parameter MinimumTLSProtocolVersion 

Please refer to:
Configuration parameters in the config.ini file 

https://www.ibm.com/support/knowledgecenter/en/SSZJPZ_11.5.0/com.ibm.swg.im.iis.productization.iisinfsv.install.doc/topics/r_bga_Server_Params.html [https://www.ibm.com/support/knowledgecenter/en/SSZJPZ_11.5.0/com.ibm.swg.im.iis.productization.iisinfsv.install.doc/topics/r_bga_Server_Params.html]


DIAGNOSING THE PROBLEM
Look for this message in SystemOut.log. Default location for SystemOut.log is 

/IBM/WebSphere/AppServer/profiles/InfoSphere/logs/server1/ 

[10/16/17 9:37:07:267 EDT] 000000c6 SSLHandshakeE E SSLC0008E: Unable
to initialize SSL connection. Unauthorized access was denied or
security settings have expired. Exception is javax.net.ssl.
SSLHandshakeException: Client requested protocol TLSv1 not enabled or
not supported


RESOLVING THE PROBLEM
Upgrade the Glossary Anywhere client to at least version BGASetup-11.5.0.1.20161212-1904. Also, upgrade to Microsoft .NET 4.6.1
