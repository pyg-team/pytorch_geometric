Title: IBM Unable to launch Visual Editor for Java classes in Rational Software Architect - United States

Text:
RSA; Visual Editor; RSA7; Jav; client; application; editor; JVE TECHNOTE (TROUBLESHOOTING)

PROBLEM(ABSTRACT)
 Unable to launch Visual Editor for Java classes in IBM Rational Software Architect. 

SYMPTOM
When you right-click on a Java class and select Open With from the context menu, the "Visual Editor" option is missing.


CAUSE
The "Java client application editor (JVE)" feature is an optional feature for Rational Software Architect. 

In order to access "Visual Editor" for Java classes, this feature must be selected and installed.


RESOLVING THE PROBLEM
To resolve this problem, follow the instructions below: 

 1. Launch IBM Installation Manager (IM).
    
    
 2. Click Modify Packages
    
    
 3. Check the box titled Java client application editor (JVE) to enable the feature
    
    [/support/docview.wss?uid=swg21305500&amp;aid=1]
    
    
    
 4. Click Modify to install the feature.
