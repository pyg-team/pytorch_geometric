Title: IBM Connections 6.0 CR requirements for IBM FileNet for use with Connections Content Manager (CCM) - United States

Text:
 PREVENTIVE SERVICE PLANNING

ABSTRACT
 This document contains information on the requirements for IBM FileNet for use with IBM Connections 6.0 CR Connections Content Manager (CCM) 

CONTENT
IBM Connections 6.0 environments using Connections Content Management (CCM) are required to upgrade the FileNet installation as part of the upgrade to a 6.0 Cumulative Refresh (CR). After applying IBM Connections 6.0 CR1, upgrades are required for FileNet Content Engine Server.
Table of Contents:


 * Section 1 - Important notes 
 * Section 2 - Compatibility table 
 * Section 3 - Download table 
 * Section 4 - Installation instructions


SECTION 1 - IMPORTANT NOTES 

 * These procedures are required only if you have installed Connections Content Manager (CCM) in IBM Connections 6.0. 

 * Make sure the latest Connections 6.0 CR update is installed before attempting to upgrade the Content Engine (CE) server, CE client, and FileNet Collaboration Services (FNCS). 
 * For information regarding installing CCM on Windows Server 2016 refer to IBM Connections Content Manager 6.0 - Installation fails on Windows Server 2016 [http://www.ibm.com/support/docview.wss?uid=swg22008730] (technote 2008730)

SECTION 4 - INSTALLATION INSTRUCTIONS 

It is important that you understand requirements and dependencies before updating FileNet. The topic Updating IBM Connections 6.0 with the latest Cumulative Refresh (CR) [http://www.ibm.com/support/knowledgecenter/SSYGQH_6.0.0/admin/migrate/c_installing_fix-packs.html] has important information that must be followed when applying IBM Connections 6.0 CRs and the subtopic Post-update tasks [https://www.ibm.com/support/knowledgecenter/SSYGQH_6.0.0/admin/migrate/t_post-update_tasks.html] has information on deploying the latest FileNet FixPacks and Interim Fixes. 

Installation instructions for the FileNet fixpacks with IBM Connections are detailed in "Section 5 - Instructions on applying Cumulative Refresh (CR) updates" in the Upgrade Strategy for IBM Connections 6.0 document [http://www.ibm.com/support/docview.wss?uid=swg21999492].
