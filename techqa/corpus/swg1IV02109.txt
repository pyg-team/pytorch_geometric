ERROR DESCRIPTION
 *  The message "Monitoring Agent for Linux OS Operational Event:
   Agent started successfully" message sometimes is written to
   KLZ:LG0 every two minutes.  The message is written during the
   availability checking for the agent when the agent is running.
   It can occur on the UNIX or Linux OS agent
   
   
    
   
   

LOCAL FIX

PROBLEM SUMMARY
 *  "AGENT STARTED SUCCESSFULLY" message written unnecessarily.
   
   The message "Monitoring Agent for Linux OS Operational Event:
   Agent started successfully" message sometimes is written to
   KLZ:LG0 every two minutes.  The message is written during the
   availability checking for the agent when the agent is running.
   The message can be ignored.  It can occur on the UNIX or Linux
   OS agent.
   
   
    
   
   

PROBLEM CONCLUSION
 *  The code was updated to not write the message during
   availability checking.
