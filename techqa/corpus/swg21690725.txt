Title: IBM Security Bulletin: WebSphere Message Broker and IBM Integration Bus are affected by error handling vulnerability (CVE-2014-6170). - United States

Text:
WMB IIB SECURITY BULLETIN

SUMMARY
 The HTTPInput node of WebSphere Message Broker and IBM Integration Bus can return a soap fault including sensitive information that can be used to conduct an attack on the system. 

VULNERABILITY DETAILS
CVE- ID : CVE-2014-6170 [http://cve.mitre.org/cgi-bin/cvename.cgi?name=CVE-2014-6170]
DESCRIPTION : The HTTPInput node of WebSphere Message Broker and IBM Integration Bus can return a soap fault including sensitive information that can be used to conduct an attack on the system. 
CVSS Base Score: 4.3
CVSS Temporal Score: See https://exchange.xforce.ibmcloud.com/vulnerabilities/98309 [https://exchange.xforce.ibmcloud.com/vulnerabilities/98309] for the current score
CVSS Environmental Score*: Undefined
CVSS Vector: (AV:N/AC:M/Au:N/C:P/I:N/A:N)

AFFECTED PRODUCTS AND VERSIONS
IBM Integration Bus V9.0 

 WebSphere Message Broker V8.0 

 WebSphere Message Broker V7.0
