Title: IBM Using a filter with the IBM WebSphere Application Server V9.0 package does not show the IBM SDK, Java Technology Edition, Version 8 package - United States

Text:
 TECHNOTE (TROUBLESHOOTING)

PROBLEM(ABSTRACT)
 In the Install page of Installation Manager, when you specify a filter for the IBM WebSphere Application Server V9.0 package, the page does not display the IBM SDK, Java Technology Edition, Version 8 package for selection. 

SYMPTOM
You specify a filter for the IBM WebSphere Application Server V9.0 package and select the package. You encounter the warning message, "The IBM WebSphere Application Server package requires the IBM SDK, Java Technology Edition, Version 8 package. Ensure that IBM SDK, Java Technology Edition, Version 8 is installed together with IBM WebSphere Application Server."


CAUSE
Installation Manager incorrectly applies the filter to the IBM SDK, Java Technology Edition, Version 8 package which is required by IBM WebSphere Application Server V9.0.

RESOLVING THE PROBLEM
Click the "Clear Filter Text" icon and select the IBM SDK, Java Technology Edition, Version 8 package beneath the selected edition of IBM WebSphere Application Server.
