Title: IBM SAP Agent - Workload Performance workspace is blank - United States

Text:
5724B97SOTR TECHNOTE (TROUBLESHOOTING)

PROBLEM(ABSTRACT)
 You deployed and configured the ITCAM SAP agent, but there is no data in Workload Performance workspace in Tivoli Enterprise Portal (TEP) 

SYMPTOM
There is no data in the TEP for the Workload Performance workspace for :Ins node. Future time span selected message displays in workspace.


CAUSE
The timezone is not in synch between the ITM servers, SAP agent machine and the SAP server.

RESOLVING THE PROBLEM
First, reconfigure and restart the agent(s). Check to see if there is data in the TEP. 

 * If there is still no data, check for a mismatch of timezone between ITM, the SAP agent machine and the SAP server machine. Correct any discrepancies with time and time zones.
   
   Check the clients for timezone
