Title: IBM IV21371: USERS CANNOT QUERY JOB PLAN RECORDS BY SITE ID. - United States

Text:
 SUBSCRIBE
You can track all active APARs for this component.



APAR STATUS
 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  Users cannot query Job Plan records by Site ID.
   
   To Reproduce:
   
   1. Login as a user that does not have 'Authorize Group for All
   Sites?' enabled for the user security group.
   
   (User will need to have at least one site enabled)
   
   2. Launch the Job Plans application.
   
   3. Select the user's enabled Site from the Site lookup on the
   Advanced Search or List Page.
   
   4. Select the Enter Key to enable search. (no records returned)
   
   
    
   
   

LOCAL FIX
 *  If you remove the equal sign from the site name, the Job Plan
   records are displayed for the selected site.
   
   Example: Change site name from '=BEDFORD' to 'BEDFORD'
