ERROR DESCRIPTION
 *  If a Work Order has a Location, but no Asset, an error message
   appears when trying to create a Defect for the Work Order.
   
   Steps To Reproduce:
   -Log in to maxdemo instance as wilson
   -Go To Work Orders | Work Order Tracking (Oil)
   -Click the New Work Order button on the toolbar
   -In the Asset field enter "br200" and tab out
   -Delete the value from the Asset and Description fields
   -Save the record
   -Click the Change Status button on the toolbar
   -In the New Status field use the drop-down arrow and click
   Approved
   -Click OK
   -Select Action | Create | Defect
   -Error message appears:
   "BMXAA4198E - The GL attribute GLACCOUNT on object {1} requires
    an organization. Specify a value in the Organization field."
