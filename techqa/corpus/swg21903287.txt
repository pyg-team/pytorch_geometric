Title: IBM 【セキュリティ情報】WebSphere Message Broker とIBM Integration Bus におけるDojo Toolkitのクロスサイトスクリプティング(XSS)脆弱性 (CVE-2014-8917) - United States

Text:
 SECURITY BULLETIN

SUMMARY
 WebSphere Message Broker とIBM Integration Busが使用している IBM Dojo Toolkitの下記のファイルにクロスサイトスクリプティング(XSS)の脆弱性があります。
dojox/form/resources/fileuploader.swf
dojox/form/resources/uploader.swf
dojox/av/resources/audio.swf 
dojox/av/resources/video.swf



VULNERABILITY DETAILS
脆弱性の詳細
次の脆弱性の修正が含まれます。

CVEID:CVE-2014-8917 [http://cve.mitre.org/cgi-bin/cvename.cgi?name=CVE-2014-8917]
DESCRIPTION: IBM Dojo Toolkit is vulnerable to cross-site scripting, caused by improper validation of user-supplied input. A remote attacker could exploit this vulnerability using a specially-crafted URL to execute scripts in a victim's Web browser within the security context of the hosting Web site, when the URL is clicked. An attacker could use this vulnerability to steal the victim's cookie-based authentication credentials
CVSS Base Score: 4.3
CVSS Temporal Score: Seehttps://exchange.xforce.ibmcloud.com/vulnerabilities/99303 [https://exchange.xforce.ibmcloud.com/vulnerabilities/99303] for the current score
CVSS Environmental Score*: Undefined
CVSS Vector: AV:N/AC:M/Au:N/C:N/I:P/A:N 

AFFECTED PRODUCTS AND VERSIONS
影響のある製品・バージョン： 

次の製品に影響があります。 

・IBM WebSphere Message Broker V8 

・IBM Integration Bus V9

REMEDIATION/FIXES
修正策： 

Workaround and Mitigations (回避策/軽減策)に記載されている手順に従ってください。 


WebSphere Message Broker V8.0.0.6 または、 IBM Integration Bus V9.0.0.4 に含まれる予定のAPAR IT06833 によって該当のファイルを削除される予定です。

WORKAROUNDS AND MITIGATIONS
回避策/軽減策： 


この脆弱性の影響を受けるファイルは、WebSphere Message Broker もしくは IBM Integration Bus のWebユーザインターフェイス（Web UI）で使用されていないため削除することができます。

この脆弱性に対処するために、以下のファイルを削除してください。

<install>\webadmin\apps\ROOT\dojox\form\resources\fileuploader.swf
<install>\webadmin\apps\ROOT\dojox\form\resources\uploader.swf
<install>\webadmin\apps\ROOT\dojox\av\resources\audio.swf
<install>\webadmin\apps\ROOT\dojox\av\resources\video.swf

注：WebSphere Message BrokerのV8.0.0.0にファイルfileuploader.swfが含まれていません。
