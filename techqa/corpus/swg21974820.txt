Title: IBM Error "No access to data provider" in IE or Chrome - United States

Text:
DASHL2; DASHL2FAQ; no access to data provider; Chrome; Internet Explorer; IE 11; TopologyView Widget TECHNOTE (TROUBLESHOOTING)

PROBLEM(ABSTRACT)
 In DASH 3.1.2 the following error occurs in the TWL TopologyView Widget.

"No access to data provider" . 

It occurs in Internet Explorer and Chrome browser, Firefox works fine. 

SYMPTOM
"No access to data provider" error showing in the TopologyView Widget. 


ENVIRONMENT
DASH 3.1.2 with cumulative patch 2. using Chrome or IE.

RESOLVING THE PROBLEM
This issue is fixed in DASH 3.1.2.1. 

Please download 1.1.2-TIV-JazzSM-multi-FP001_v2 from FixCentral and install DASH 3.1.2.1 
http://www-933.ibm.com/support/fixcentral/ [http://www-933.ibm.com/support/fixcentral/]
