QUESTION
 What are the IBM Passport Advantage part numbers of the IBM WebSphere Application Sever 8 software for IBM Rational ClearQuest Web 8, Rational ClearCase (CCRC) 8, and Rational RequisitePro Web 7.1 installations? 

CAUSE
You might require a download of other components such as WebSphere Application Server in order to install ClearQuest Web, ClearCase Remote Client (CCRC), or RequisiteWeb. When accessing IBM Passport Advantage to download the WebSphere components needed to install ClearQuest Web and/or CCRC, it might not be clear which download part numbers to choose.

ANSWER
After you decide what version of products to install on your server you must acquire the WebSphere Application Server downloads. Here is a list of them: 

For ClearQuest 8.0.0.1 and later fix packs, ClearCase 8.0.0.1 and later fix packs, and RequisitePro 7.1.3.1 and later versions, you download WebSphere Application Server 8.0:



 * WebSphere Application Server 8.0:
 *  
 * 
 *  IBM WebSphere Application Server V8.0 (1 of 4) for Multiplatform Multilingual (CZM9HML ) IBM WebSphere Application Server V8.0 (2 of 4) for Multiplatform Multilingual (CZM9IML ) IBM WebSphere Application Server V8.0 (3 of 4) for Multiplatform Multilingual (CZM9JML ) IBM WebSphere Application Server V8.0 (4 of 4) for Multiplatform Multilingual (CZVG3ML ) 
 * 
 * Supplements 8.0:
 * 
 *  IBM WebSphere Application Server V8.0 Supplements (1 of 4) for Multiplatform Multilingual (CZM91ML) IBM WebSphere Application Server V8.0 Supplements (2 of 4) for Multiplatform Multilingual (CZM94ML) IBM WebSphere Application Server V8.0 Supplements (3 of 4) for Multiplatform Multilingual (CZM95ML) IBM WebSphere Application Server V8.0 Supplements (4 of 4) for Multiplatform Multilingual (CZXR9ML) 

For ClearQuest and ClearCase 8.0.1 and later fix packs, you download WebSphere Application Server 8.5: 

 

 * WebSphere Application Server 8.5:
 * 
 *  IBM WebSphere Application Server V8.5 (1 of 3) for Multiplatform Multilingual (CI6XNML) IBM WebSphere Application Server V8.5 (2 of 3) for Multiplatform Multilingual (CI6XPML) IBM WebSphere Application Server V8.5 (3 of 3) for Multiplatform Multilingual (CI6XQML) 
 * 
 * Supplements 8.5:
 * 
 *  IBM WebSphere Application Server V8.5 Supplements (1 of 3) for Multiplatform Multilingual (CI6X0ML) IBM WebSphere Application Server V8.5 Supplements (2 of 3) for Multiplatform Multilingual (CI6X1ML) IBM WebSphere Application Server V8.5 Supplements (3 of 3) for Multiplatform Multilingual (CI6X2ML) 

Notes: 

 * These downloads comprehend both 32 and 64-bit installable files. You can decide the version to install during installation process. You need both WebSphere Application Server and Supplements downloads. 
 * ClearQuest is a 32-bit application. Some features, such as Full-Text Search, do not work with 64-bit WebSphere installations. Therefore, it is recommend to install WebSphere Application Server in 32-bit mode. Contact IBM Support in case you have other considerations about this option.
