Title: IBM Default certificate (Factory Supplied identity) of https outbound activities is failing in Cast Iron appliance - United States

Text:
 TECHNOTE (TROUBLESHOOTING)

PROBLEM(ABSTRACT)
 Http outbound orchestration with default certificate (Factory Supplied identity) is failing in appliance. 

CAUSE
The default certificate is not available in the trust store. The orchestration is failing with an exception.

DIAGNOSING THE PROBLEM
The request operation failed. Exception: com.ibm.jsse2.util.j: PKIX path building failed: java.security.cert.CertPathBuilderException: unable to find valid certification path to requested target

RESOLVING THE PROBLEM
To resolve this issue, import the default certificate to the trust store and run the orchestration.
