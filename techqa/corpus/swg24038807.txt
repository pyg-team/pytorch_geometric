ABSTRACT
This document describes how to download IBM InfoSphere QualityStage Module for U.S. Address Certification (USAC), Version 8.6, from the IBM Passport Advantage web site.

DOWNLOAD DESCRIPTION
IBM InfoSphere QualityStage Module for U.S. Address Certification provides United States Postal Service-certified Cycle N address verification and enhancement for U.S. addresses. This module meets all of the requirements of the United States Postal Service (USPS) to allow you to qualify for postage discounts. This module ensures the completeness, consistency, and validity of critical U.S. address information in corporate databases.

This version is compatible with IBM InfoSphere Information Server, Versions 11.3.1, 11.5, and 11.7.

PLEASE NOTE: IBM InfoSphere QualityStage Module for U.S. Address Certification 8.6 now only supports the new USPS SHA-256 encrypted data, which also requires a SHA-256 patch [http://www.ibm.com/support/docview.wss?uid=swg24043669]. All users must be on the SHA-256 version as these are the only files supplied by the USPS. 

DOWNLOADING THE PRODUCT
If you already have USAC installed, an upgrade is available to move to SHA-256 support. It is only necessary to install the patch once. Please go to Fix Central and install the appropriate patch for your system. This download document [http://www.ibm.com/support/docview.wss?uid=swg24043669] contains instructions and patch links.

Once your system is patched, you can install the Reference Data parts for SHA-256 listed below.

Steps for a new installation 

 1. Read the Quick Start Guide [http://www-01.ibm.com/support/docview.wss?uid=swg27044378] for the process overview and return here to download the product.
 2. Review the release notes [http://www-01.ibm.com/support/knowledgecenter/SSZJPZ_11.3.0/com.ibm.swg.im.iis.qs.cass.relnote.doc/topics/usac86cycleNrelnt.html] for the latest information to ensure successful installations.
 3. Go to the Find Downloads and Media [http://www.ibm.com/software/howtobuy/passportadvantage/paocustomer/sdma/SDMA?PO=DOWNLOAD_FIND_BY_PART_NUMBER] web page on the IBM Passport Advantage web site.
 4. Under Download finder options , select the Find by part number search option.
 5. Search for each downloadable image by part number.
 6. Download all the required parts.
 7. Use a file extracting utility that supports large file extractions to extract each of the parts into a single, temporary directory on your system.
 8. Extract the files to the location designated for the source data based on the configuration of the U.S. Address Certification Module.
