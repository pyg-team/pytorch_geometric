Title: IBM Fix Pack 8.0.0.10 for WebSphere MQ v8 - United States

Text:
8.0.0.10;csd80010;fix pack;fixpack NEWS

ABSTRACT
Fix Pack 8.0.0.10 for WebSphere MQ v8 is now available.

CONTENT
Fix Pack 8.0.0.10 is the tenth maintenance for the WebSphere MQ v8 product first shipped in June 2018 on the following platforms:

 * AIX
 * HP-UX Itanium
 * IBM i
 * Linux on POWER
 * Linux on Power (Little Endian)
 * Linux on x86-64
 * Linux on zSeries s390x
 * Solaris on SPARC
 * Solaris on x86-64
 * Windows
 * z/OS (JMS feature only)
 * IBM MQ Appliance M2000/M2001
 * IBM MQ Advanced V8.0 Virtual System Pattern Type for IBM PureApplication on AIX
 * IBM MQ Advanced V8.0 Virtual System Pattern Type for IBM PureApplication on Linux

The links in the download section below link to the downloads for MQ v8.0.0.10 on all applicable platforms. You will be asked to register before being presented with a set of downloadable files.

Prerequisites 

Install this fix pack on top of IBM WebSphere MQ v8.0.0.0 , MQ v8.0.0.1, MQ v8.0.0.2, MQ v8.0.0.3, MQ v8.0.0.4, MQ v8.0.0.5, MQ v8.0.0.6, MQ v8.0.0.7, MQ v8.0.0.8 or MQ v8.0.0.9.

The supported environments for WebSphere MQ v8.0 can be found here [http://www.ibm.com/support/docview.wss?uid=swg27041395].
