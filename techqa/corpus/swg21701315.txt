Title: IBM Security Bulletin: A security vulnerability has been identified in both IBM WebSphere Application Server and IBM HTTP Server shipped with WebSphere Commerce (CVE-2015-0138) - United States

Text:
 SECURITY BULLETIN

SUMMARY
 IBM WebSphere Application Server is shipped as a component of WebSphere Commerce. Information about a security vulnerability affecting both IBM WebSphere Application Server and IBM HTTP Server has been published in a security bulletin. 

VULNERABILITY DETAILS
Consult the security bulletins Vulnerability with RSA Export Keys may affect IBM WebSphere Application Server (CVE-2015-0138) [http://www-01.ibm.com/support/docview.wss?uid=swg21698613] and Vulnerability with RSA Export Keys may affect IBM HTTP Server and WebSphere EDGE Caching Proxy (CVE-2015-0138) [http://www-01.ibm.com/support/docview.wss?uid=swg21698959] for vulnerability details and information about fixes.

AFFECTED PRODUCTS AND VERSIONS
 

Principal Product and Version(s) Affected Supporting Product and Version WebSphere Commerce V6.0 and V7.0 IBM WebSphere Application Server  * Version 8.5 and 8.5.5 Full Profile and Liberty Profile 
 * Version 8 
 * Version 7 
 * Version 6.1
