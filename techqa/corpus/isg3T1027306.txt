Title: IBM WEBGUI logged out with Java.io.FileNotFoundException messages - United States

Text:
 TECHNOTE (FAQ)

QUESTION
 Intermittently WEBGUI page will log out automatically and prompt below error messages.
Java.io.FileNotFoundException:SRVE0190E:File not found:/logout.do&error= 

CAUSE
The browser used is not supported.

ANSWER
By checking message_gui_server.log, you can see below messages. Further confirm that the user is using Chrome 64 bit, which is NOT in the supported browser list(https://www.ibm.com/support/knowledgecenter/SSZUMP_7.2.0/product_table/browsers.html [https://www.ibm.com/support/knowledgecenter/SSZUMP_7.2.0/product_table/browsers.html]) 

By using supported browsers there is no such issue.
