Title: IBM IV76038: INCORRECT LIST OF SUPPORTED WINDOWS SERVER 2012 EDITIONS FOR THE WINDOWS OS AGENT. - United States

Text:

APAR STATUS
 * CLOSED AS DOCUMENTATION ERROR.
    
   
   

ERROR DESCRIPTION
 *  The ITM 6.30 Windows OS Agent User's Guide, in "Requirements
   for the monitoring agent" chapter, Table 1, wrongly reports that
   "Windows Server 2012 Essentials Edition on Intel x86-64 (64
   bit)" and "Windows Server 2012 Datacenter Edition on Intel
   x86-64 (64 bit)" are supported.
   
   Actually the ITM 6.30 Windows OS Agent and following fixpacks
   are certified only on "Windows Server 2012 Standard edition on
   Intel x86-64 (64 bit)".
   
   In addition, since 6.30 FP2, the following Windows Server 2012
   R2 editions are certified too:
   "Windows Server 2012 R2 Standard Edition x86-64"
   "Windows Server 2012 R2 Datacenter Edition x86-64"
   "Windows Server 2012 R2 Essentials Edition x86-64"
   
   Affected Platforms / Versions:
   This issue affects the Windows OS Agent 6.30 documentation.
   
   
    
   
   

LOCAL FIX
 *  Refer to Software Product Compatibility Reports site.
   
   
    
   
   

PROBLEM SUMMARY
 *  Problem Description: Incorrect list of Windows Server 2012
    versions supported by the Monitoring Agent for Windows OS
   Problem/Problem Summary: Windows Server 2012 Essentials Edition
    on Intel x86-64 (64 bit) and Windows Server 2012 Datacenter
    Edition on Intel x86-64 (64 bit) are incorrectly indicated as
    supported by the Monitoring Agent for Windows OS in the
    official IBM Tivoli Monitoring version 6.3 documentation.
