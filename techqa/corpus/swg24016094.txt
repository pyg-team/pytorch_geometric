INSTALLATION INSTRUCTIONS
To download and install the fix pack: 


 1. 
 2. Download the fix pack from the download URL below.
 3. 
 4. Extract the compressed file in an appropriate directory. For example, choose to extract to C:\temp.
 5. 
 6. Add the fix pack's repository location in IBM Installation Manager:
     1.  
     2.  Start IBM Installation Manager.
     3.  
     4.  On the Start page of Installation Manager, click File > Preferences, and then click Repositories. The Repositories page opens.
     5.  
     6.  On the Repositories page, click Add Repository.
     7.  
     8.  On the Add repository window, browse to and enter the file path to the C:\temp\pu\updates directory. For example, enter C:\temp\pu\updates\repository.config and then click OK.
     9.  
     10. Click OK to close the Preference page.
    
    
 7.
