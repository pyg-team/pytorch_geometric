Title: IBM Security Bulletin: Multiple vulnerabilities have been identified in IBM Endpoint Manager for Patch Management shipped with IBM Cloud Orchestrator and IBM SmartCloud Orchestrator - United States

Text:
CVE-2015-4000; CVE-2015-1788; CVE-2015-1789; CVE-2015-1790; CVE-2015-1791; CVE-2014-3567; CVE-2014-0224; CVE-2014-3566; CVE-2014-6137; CVE-2014-6113; CVE-2015-2808; CVE-2015-0286; CVE-2015-0289 SECURITY BULLETIN

SUMMARY
 IBM Endpoint Manager for Patch Management is shipped as a component of IBM Cloud Orchestrator, IBM Cloud Orchestrator Enterprise, IBM SmartCloud Orchestrator, and IBM SmartCloud Orchestrator Enterprise. 

VULNERABILITY DETAILS
Review the security bulletins for IBM Endpoint Manager for Patch Management for vulnerability details and for information about fixes.


 * Security Bulletin: Vulnerabilities in OpenSSL affect IBM Endpoint Manager (CVE-2015-0286, CVE-2015-0289) [http://www-01.ibm.com/support/docview.wss?uid=swg21971235] -- NEW
   
   
 * Security Bulletin: Vulnerability in Diffie-Hellman ciphers affects IBM Endpoint Manager (CVE-2015-4000) [http://www.ibm.com/support/docview.wss?uid=swg21971242]
   
   
 * Security Bulletin: Vulnerabilities in OpenSSL including Logjam affect IBM Endpoint Manager (CVE-2015-1788 CVE-2015-1789 CVE-2015-1790 CVE-2015-1791 ) [http://www.ibm.com/support/docview.wss?uid=swg21971238][http://www-01.ibm.com/support/docview.wss?uid=swg21967888] 
 * Security Bulletin: Vulnerability in RC4 stream cipher affects IBM Endpoint Manager (CVE-2015-2808*) [http://www-01.ibm.com/support/docview.wss?uid=swg21967888][http://www.ibm.com/support/docview.wss?uid=swg21692516] 
 * Security Bulletin: IBM Endpoint Manager Platform 9.1 is affected by two OpenSSL vulnerabilities, the "POODLE" vulnerability, and two XSS vulnerabilities [http://www.ibm.com/support/docview.wss?uid=swg21692516]
   
   
 * Security Bulletin: IBM Endpoint Manager Platform 9.0 is affected by two OpenSSL vulnerabilities and the "POODLE" vulnerability [http://www.ibm.com/support/docview.wss?uid=swg21692193]
   
   
 * Security Bulletin: Vulnerability in RC4 stream cipher affects IBM Endpoint Manager (CVE-2015-2808) [http://www.ibm.com/support/docview.wss?uid=swg21883852]
