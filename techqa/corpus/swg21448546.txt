Title: IBM Creating a welcome page with a logo in ClearQuest Web - United States

Text:
1448546; CQ; CQWEB; CQ Web; logo; image; welcome page; custom; login; home page TECHNOTE (FAQ)

QUESTION
 How do you put a logo as the IBM Rational ClearQuest Web welcome page? 

ANSWER
 

1. Create a HTML file with these lines. 

 * 
 * <html>
   <img src="/cqweb/logo.jpg" />
   </html> 
   
   


In this example, logo.jpg is the name of the logo image file.  

2. Place the HTML file in the applicable directory for your ClearQuest Web server, assuming a default installation location: 

 * 
 * Microsoft Windows
 * 
   7.1.0.x: C:\Program Files\IBM\RationalSDLC\common\CM\profiles\cmprofile\installedApps\<hostname>CMProfileNode01Cell\RationalClearQuestWeb.ear\CQWebModule.war
   
   7.1.1.x and 7.1.2.x: C:\Program Files\IBM\RationalSDLC\common\CM\profiles\cmprofile\installedApps\dfltCell\RationalClearQuestWeb.ear\CQWebModule.war 
 *  
 * 8.0.0.x , 8.0.1.x & 9.0.0.x: 
 * <InstallDir>\cqweb\cqwebprofile\installedApps\dfltCell\RationalClearQuestWebReport.ear\CQWebReportModule.war
   
   UNIX or Linux 
 * 
   7.1.0.x: /opt/rational/common/CM/profiles/cmprofile/installedApps/<hostnamsuree>CMProfileNode01Cell/RationalClearQuestWebReport.ear/CQWebReportModule.war
   
   7.1.1.x and 7.1.2.x: /opt/rational/common/CM/profiles/cmprofile/installedApps/dfltcell/RationalClearQuestWebReport.ear/CQWebReportModule.war 
 *  
 * 8.0.0.x, 8.0.1x & 9.0.0.x:
   <InstallDir>/cqweb/cqwebprofile/installedApps/dfltCell/RationalClearQuestWebReport.ear/CQWebReportModule.war 
   


 

3. Login to ClearQuest Web as an administrative user. Access the Site Administration tab > Configuration > Application Options. 

 

4. Enter the name of the HTML file in the Homepage field. For example, use the value TestPage.html. The path to the file location is not necessary. Click Apply or OK. 

 

5. After the next login, the custom logo will appear in the welcome page.
