APAR STATUS
 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  SMARTEAM V5R14SP6 Web Editor Client Login
   failed with the Windows
   Authentication setting.
   Whenever starting IE for the Web Editor Site, the
   Database Selection is
   displayed.
   After clicking the Connect button, the following
   error message is
   displayed.
   Error No : 00904 "The service 'ObjectStore' cannot
   be initialized. No
   user is logged into the session."
   .
   Scenario:
   1. Install SMARTEAM V5R14SP6 Web Editor
   environment.
   2. Perform the Windows Authentication setting as
   described in the
   SMARTEAM Web Editor Installation document.
   3. Start the Login process for the Web Editor on the
   web client machine.
   4. The Database Selection is displayed in IE.
   5. Click the Connect button.
   6. The following error message is displayed.
   Error No : 00904 "The service 'ObjectStore' cannot
   be initialized. No
   user is logged into the session."
   .
   However, the Login process is successful when
   Login on the SMARTEAM Web
   Editor Server.
   .
