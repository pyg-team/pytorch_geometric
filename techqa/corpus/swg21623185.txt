Title: IBM Error message !Could not delete from xdbTrickle remotely! - United States

Text:
 TECHNOTE (TROUBLESHOOTING)

PROBLEM(ABSTRACT)
 When attempting to reduce the numbers of records held in the xdbtrickle
table via the Maintain - System Audit Log Configuration - Delete menu
item the error message '!Could not delete from xdbTrickle remotely!' appears. 

CAUSE
The transaction log was full

RESOLVING THE PROBLEM
The Event log contained the following record: 

‘The transaction log for database 'xxxx' is full. To find out why
space in the log cannot be reused, see the log_reuse_wait_desc column
in sys.databases’
