Title: IBM Display the Sterling Connect:Direct File Agent version - United States

Text:
Connect:Direct File Agent; Version TECHNOTE (FAQ)

QUESTION
 How to see the Sterling Connect:Direct File Agent version? 

ANSWER
To display the version of Sterling Connect:Direct File Agent, issue a 'cdfa -p' command from a command prompt. 



2016-04-19 16:09:56,423 INFO - Connect:Direct File Agent, Version 1.4.0.0_iFix20
(c) Copyright IBM Corp. 2002, 2014,
All rights reserved.
Build 0, Date 2015/02/18



The version shown in this example is 1.4.0.0_iFix20. 

Sterling Connect:Direct File Agent will also write this version information into the log file at every startup.
