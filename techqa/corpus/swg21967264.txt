Title: IBM Install Tivoli Storage Manager Version 7 on Solaris Version 11.2 - United States

Text:
Solaris 11.2; TSM 7.1.1.200 TECHNOTE (FAQ)

QUESTION
 How to install Tivoli Storage Manager Version 7 on Solaris Version 11.2? 

CAUSE
Tivoli Storage Manager Server Version 7.1.1.200 begins to support Solaris Version 11.2. However, Version 7.1.1.200 is a fixpack without license model included. In order to install Version 7.1.1.200, it is required to install Tivoli Storage Manager Version 7.1 GA version.

ANSWER
In order to install Tivoli Storage Manager Version 7.1.1.200 successfully on Solaris Version 11.2, complete the following steps: 

1. Install Tivoli Storage Manager Version 7.1 with bypass option: 

 a. Console mode;
./install.sh -c -vmargs "-DBYPASS_TSM_REQ_CHECKS=true" 
b. GUI Mode;
./install.sh -g -vmargs "-DBYPASS_TSM_REQ_CHECKS=true" 

2. Install Tivoli Storage Manager Version 7.1.1.200 
a. Console mode;
./install.sh -c 
b. GUI Mode;
./install.sh
