APAR STATUS
 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  When copying content from Microsoft Word into an Atlas v6.0.3.x
   Hold Notice, you could see some garbled characters in the
   content especially when using lists or bullets. The issue
   occurs in (a) hold notice content preview (b) email of the
   notice sent out to custodians or "Send Me" when using Outlook
   as the email client. This is visible mainly when using Chrome
   or Firefox to edit/save the hold notice.
   
   
    
   
   

LOCAL FIX
 *  Use Internet Explorer to create the hold notice.
   
   
    
   
   

PROBLEM SUMMARY
 *  When copying content from Microsoft Word into an Atlas v6.0.3.x
   Hold Notice, you could see some garbled characters in the
   content especially when using lists or bullets. The issue
   occurs in (a) hold notice content preview (b) email of the
   notice sent out to custodians or "Send Me" when using Outlook
   as the email client. This is visible mainly when using Chrome
   or Firefox to edit/save the hold notice.
