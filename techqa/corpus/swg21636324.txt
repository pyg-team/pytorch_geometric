Title: IBM Security Bulletin:  WebSphere DataPower XC10 Appliance vulnerability in administrative access (CVE-2013-0600) and multiple TLS vulnerabilities in IBM Java as shipped with the WebSphere DataPower XC10 Appliance (CVE-2013-0440, CVE-2013-0443, CVE-2013-0169 - United States

Text:
 FLASH (ALERT)

ABSTRACT
 A security vulnerability in the WebSphere DataPower XC10 Appliance may allow unathenticated access to adminstrative operations, and multiple security vulnerabilities in Java TLS may allow attackers access to sensitive data. 

CONTENT
VULNERABILITY DETAILS: 
CVE-2013-0600 - A knowledgeable user may submit administrative commands to the WebSphere DataPower XC10 Appliance without authentication.
CVSS Base Score: 9.3
CVSS Temporal Score: See http://xforce.iss.net/xforce/xfdb/83617 [http://xforce.iss.net/xforce/xfdb/83617] for the current score
CVSS Environmental Score*: Undefined
CVSS Vector: (AV:N/AC:M/AU:N/C:C/I:C/A:C)

CVE-2013-0440 - Unspecified vulnerability in Java Runtime Environment allows remote attackers to affect availability via vectors related to JSSE.
CVSS Base Score: 5
CVSS Temporal Score: See http://xforce.iss.net/xforce/xfdb/81799 [http://xforce.iss.net/xforce/xfdb/81799]
CVSS Environmental Score*: Undefined
CVSS Vector: (AV:N/AC:L/Au:N/C:N/I:N/A:P)

CVE-2013-0443 - Unspecified vulnerability in Java Runtime Environment allows remote attackers to affect confidentiality and integrity via vectors related to JSSE.
CVSS Base Score: 4
CVSS Temporal Score: See http://xforce.iss.net/xforce/xfdb/81801 [http://xforce.iss.net/xforce/xfdb/81801]
CVSS Environmental Score*: Undefined
CVSS Vector: (AV:N/AC:H/Au:N/C:P/I:P/A:N)

CVE-2013-0169 - The TLS protocol does not properly consider timing side-channel attacks, which allows remote attackers to conduct distinguishing attacks and plain-text recovery attacks via statistical analysis of timing data for crafted packets, aka the "Lucky Thirteen" issue.
CVSS Base Score: 4.3
CVSS Temporal Score: See http://xforce.iss.net/xforce/xfdb/81902 [http://xforce.iss.net/xforce/xfdb/81902]
CVSS Environmental Score*: Undefined
CVSS Vector: (AV:N/AC:M/Au:N/C:P/I:N/A:N)


AFFECTED PRODUCTS AND VERSIONS: 

WebSphere DataPower XC10 Appliance Version 2.0
WebSphere DataPower XC10 Appliance Version 2.1

REMEDIATION: 

For WebSphere DataPower XC10 Appliance V2.1 on appliance 9235-92X apply the Version 2.1 Fix Pack 3 firmware upgrade [http://www-933.ibm.com/support/fixcentral/swg/selectFixes?parent=ibm~WebSphere&product=ibm/WebSphere/WebSphere+DataPower+XC10+Appliance&release=2.1.0.3&platform=All&function=fixId&fixids=2.1.0.3-WS-DPXC10-IFIC91726-9235&includeSupersedes=0] (APAR IC91726)

For WebSphere DataPower XC10 Appliance V2.1 on appliance 7199-92X apply the Version 2.1 Fix Pack 3 firmware upgrade [http://www-933.ibm.com/support/fixcentral/swg/selectFixes?parent=ibm~WebSphere&product=ibm/WebSphere/WebSphere+DataPower+XC10+Appliance&release=2.1.0.3&platform=All&function=fixId&fixids=2.1.0.3-WS-DPXC10-IFIC91726-7199&includeSupersedes=0] (APAR IC91726)

WebSphere DataPower XC10 Appliance users running 2.0 versions of the firmware can apply one of the Version 2.1 firmware upgrades.
