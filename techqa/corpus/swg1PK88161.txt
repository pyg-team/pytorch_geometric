ERROR DESCRIPTION
 *  Doc defect: Feature removed regarding html code rendering under
   step log is not documented.
   
   Previous Build Forge version step log were able to render html c
   ode.
   
   This feature has been removed from the current version of Build
   Forge, but this has not be documented.
   
   Removed feature will need to be documented.
   
   
    
   
   

LOCAL FIX

PROBLEM SUMMARY
 *  The ability to use HTML tags in step logs was an
   undocumented feature and therefore not warranted. Support
   for it was removed in version 7.1.1.
   
   
    
   
   

PROBLEM CONCLUSION
 *  Starting in version 7.1.1, HTML tags are treated as plain
   text in the step log.
