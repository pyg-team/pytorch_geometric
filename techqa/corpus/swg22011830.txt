Title: IBM 【セキュリティ情報】IBM Integration BusNode.js zlib DOSセキュリティの脆弱性　（CVE-2017-14919） - United States

Text:
 SECURITY BULLETIN

SUMMARY
 IBM Integration Busは以下の脆弱性に対処しました。(CVE-2017-14919) 


VULNERABILITY DETAILS
最新の情報については、下記URLの文書（英語）をご参照ください。
Security Bulletin: IBM Integration Bus is affected by a Node.js zlib DOS security Vulnerability(CVE-2017-14919)
http://www.ibm.com/support/docview.wss?uid=swg22011533 [http://www.ibm.com/support/docview.wss?uid=swg22011533]



CVEID: CVE-2017-14919
DESCRIPTION: Node.jsは、zlibモジュール内のキャッチされていない例外の脆弱性によって引き起こされるサービス妨害に対して脆弱です。 windowBitsパラメータに8を無効な値にすると、リモートの攻撃者がこの脆弱性を悪用してアプリケーションをクラッシュさせる可能性があります。

Node.jsは、zlibモジュール内でキャッチすることのできない例外によって引き起こされるサービス妨害に対して脆弱性があります。ウィンドウサイズに指定できない値である8を設定することによって、リモート攻撃者がこの脆弱性を悪用してアプリケーションをクラッシュさせる可能性があります。
CVSS Base Score: 7.5
CVSS Temporal Score: See https://exchange.xforce.ibmcloud.com/vulnerabilities/134286 [https://exchange.xforce.ibmcloud.com/vulnerabilities/134286] for the current score
CVSS Environmental Score*: Undefined
CVSS Vector: (CVSS:3.0/AV:N/AC:L/PR:N/UI:N/S:U/C:N/I:N/A:H) 

AFFECTED PRODUCTS AND VERSIONS
この脆弱性は、IBM Integration BusおよびWebSphere Message Brokerの下記のバージョン、メンテナンス・レベルにおいて影響を受けます。 

 * IBM Integration Bus V10.0.0.0 - V10.0.0.10
