INSTALLATION INSTRUCTIONS
To download and install the fix pack, complete the following steps:


 1. 
 2. Download the fix pack (im-1.1.2.zip) from the download URL below.
 3. 
 4. Extract the compressed file in an appropriate directory. For example, choose to extract to C:\temp.
 5. 
 6. Add the fix pack repository location in IBM Installation Manager:
     1. Start IBM Installation Manager.
     2. 
     3. On the Start page of Installation Manager, click File > Preferences, and then click Repositories. The Repositories page opens.
     4. 
     5. On the Repositories page, click Add Repository.
     6. 
     7. In the Add repository window, browse to or enter the file path to the repository.config file, which is located in the directory where you extracted the compressed files. For example, enter C:\temp\ or C:\temp\repository.config. Make sure Search service repositories during installation and updates is checked and then click OK.
    
    
    
 7. Installation of the fix pack will begin once you click on Install Packages or Update Packages.
 8. 


If IBM Installation Manager is not currently installed, you can install this version by completing the following steps: 

 1. Download the installer from the download URL in the table below:

 * im.installer.win32-1.1.2.zip (on Windows) 
 * im.installer.linux-1.1.2.zip (on Linux) 
 * im.installer.solaris-1.1.2.zip (on Solaris) 
 * im.installer.aix-1.1.2.zip (on AIX)
   
   1. Extract the compressed file into an appropriate directory.
   2. Run install.exe (Windows), or install (Linux/Solaris/AIX). 
   3. If IBM Installation Manager is not currently installed, or if the installed version is older than 1.1.2, the Installation Manager installer will be launched. Follow the wizard pages to install it.
   4. IBM Installation Manager will start at completion.
