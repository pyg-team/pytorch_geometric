 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  Abstract: TEPS displays alert timestamp as UTC. CR believes
   timestamp should be converted to whatever the time zone is of
   the server.
   Environment: SAP Agent 7.1.1
   Affected SAP Version : MAI configured Solution Manager systems
   Detailed Recreation Procedure:
   1. Import 711 transport on MAI configured Solution Manager SAP
      system
   2. Check the MAI Alerts time stamp in Agent.
   3. Alert time stamp will be UTC format
   
   FIX: In SAP Agent ABAP code. If timezone of SAP Alert Inbox is
   different than convert the timezone of alerts from UTC to local
   timezone.
   
   
    
   
   

LOCAL FIX

PROBLEM SUMMARY
 *  TEPS DISPLAYS ALERT TIMESTAMP AS UTC
   
   Due to BADI implementation  and Day Light Saving  , agent ABAP
   code display   difference in alerts timestamp. ABAP to consider
   this check.
