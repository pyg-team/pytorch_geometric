Title: IBM WPG Destination failes to deliver files > 75MB to WMQ issuing event error BCG250011 - United States

Text:
MT# PMR: 68603; 122; 000 BCG250011 TECHNOTE (TROUBLESHOOTING)

PROBLEM(ABSTRACT)
 WPG Destination failes to deliver files > 75MB to WMQ issuing event error BCG250011 

SYMPTOM
WPG Destination failes to deliver files > 75MB to WMQ issuing event error BCG250011


DIAGNOSING THE PROBLEM
bcg_server.log shows the following error:
A UOW=null source=com.ibm.ejs.j2c.ConnectionEventListener org=IBM prod=WebSphere component=Application Server thread=[Gw_1866_0]
J2CA0056I: The Connection Manager received a fatal connection error from the Resource Adapter for resource JMS$EDI.QCF$JMSManagedConnection@2. The exception which was received is javax.jms.JMSException: MQJMS2007: failed to send message to MQ queue

The failure sequence can also be seen in the systemOut.log:
ConnectionEve A J2CA0056I: The Connection Manager received a fatal connection error from the Resource Adapter for resource JMS$EDI.QCF$JMSManagedConnection@2. The exception which was received is javax.jms.JMSException: MQJMS2007: failed to send message to MQ queue
and nested exception for MQJMS2007: com.ibm.mq.MQException: MQJE001: Completion Code 2, Reason 2102

The MQ utility "mqrc" can be used to get the name of the reason code 2102: mqrc 2102 
2102 0x00000836 MQRC_RESOURCE_PROBLEM.

RESOLVING THE PROBLEM
The following actions suggested by the WMQ team resolved this problem:
1 - Run the mqconfig script to make sure the system settings are at or above the required settings for one queue manager on a system:
http://www-01.ibm.com/support/docview.wss?uid=swg21271236 [http://www-01.ibm.com/support/docview.wss?uid=swg21271236]

2 - To avoid Log full problems, increase the number of log files, updating the qm.ini file:
https://www.ibm.com/support/knowledgecenter/SSFKSJ_7.5.0/com.ibm.mq.con.doc/q018950_.htm [https://www.ibm.com/support/knowledgecenter/SSFKSJ_7.5.0/com.ibm.mq.con.doc/q018950_.htm]
