APAR STATUS
 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  Firmware levels 6.0.0.3/6.0.1.0 or later have a newer version of
   WTX and requires the WTX projects be recompiled before use.
   This is due to a once optional fields now being required.
   
   
    
   
   

LOCAL FIX
 *  Recompile the WTX DPA projects using the newer version of WTX
   Design Studio.
   
   
    
   
   

PROBLEM SUMMARY
 *  Customers using 6.0.0.3/6.0.1.0 or later firmware might
   experience binary transformation failures with their existing
   DPA map files which might have worked well in previous versions
   of the firmware.
   
   
    
   
   

PROBLEM CONCLUSION
 *  Fix is available in 6.0.0.5 and 6.0.1.1.
   
   For a list of the latest fix packs available, please see:
   http://www-01.ibm.com/support/docview.wss?uid=swg21237631 [http://www-01.ibm.com/support/docview.wss?uid=swg21237631]
