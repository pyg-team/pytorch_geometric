 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  If the HttpOnly attribute is not used while setting cookie,
   then the cookie can be accessed through non-HTTP methods such
   as JavaScript. Especially if the application has XSS (cross
   site scripting) flaw, then the cookie can be stolen by
   exploiting XSS flaw.
   
   It has been observed that the application has not set
   "HTTPOnly" attribute for cookie. As the application is
   vulnerable to Cross Site Scripting attack.
