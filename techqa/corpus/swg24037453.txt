ABSTRACT
 This document describes how to download the PeopleSoft with data masking portion of the IBM® InfoSphere® Optim Solution for Oracle Applications v11.3. 

DOWNLOAD DESCRIPTION
You can download the PeopleSoft with data masking portion of the IBM InfoSphere Optim Solution for Oracle Applications using the Passport Advantage® website.


The Optim Test Data Management Solution for PeopleSoft Enterprise with Data Masking Option uses the Optim test data management solution to extract a subset of data from your source database and mask the extracted data when you insert it into your target database. You can use the data masking option to mask employee personal and job information, such as names, addresses, and phone numbers.

PREREQUISITES
You must install IBM InfoSphere Optim Solution v11.3 before you install the Optim Test Data Management Solution for PeopleSoft Enterprise with Data Masking Option. 

For information about hardware and software requirements for the InfoSphere Optim Test Data Management Solution for PeopleSoft Enterprise with Data Masking Option, see the detailed system requirements.
