APAR STATUS
 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  The description of the datasource custom property
   userDefinedErrorMap, contains double quotes and when wsadmin
   code tries to parse the value during an export or import
   operation it fails with java.lang.IllegalArgumentException:
   WASX7122E: Expected "]" not found.
   
   
    
   
   

LOCAL FIX
 *  Manually replace the double quotes in the string with single
   quotes.
   
   
    
   
   

PROBLEM SUMMARY
 *  ****************************************************************
   * USERS AFFECTED:  All users of IBM WebSphere Application      *
   *                  Server                                      *
   ****************************************************************
   * PROBLEM DESCRIPTION: The double quotes in the description    *
   *                      of the datasource custom property       *
   *                      userDefinedErrorMap can cause parsing   *
   *                      errors                                  *
   ****************************************************************
   * RECOMMENDATION:                                              *
   ****************************************************************
   The parsing error was due the the presense of double quotes in
   the description field of the datasource custom property
   userDefinedErrorMap which WebSphere admin config does not
   and hence parsing errors during export or import operation of
   the datasource config using wsadmin.
   This apar will remove the double quotes in the description
   field of datsource custom property userDefinedErrorMap from the
   config templates. Also note that the apar changes will apply
   for newly created profiles that include this apar.
   Note: There is no change to the format of the datasource
   custom property userDefinedErrorMap and its value.
   
   
    
   
   

PROBLEM CONCLUSION
 *  The description field of the custom property has been updated
   so as the remove the text containing double quotes.
   
   APAR PM81646 is currently targeted for inclusion in Fix Packs
   7.0.0.31, 8.0.0.7 and 8.5.5.1 of WebSphere Application Server.
   
   Please refer to URL:
   //www.ibm.com/support/docview.wss?rs=404&uid=swg27006970
   for Fix Pack availability.
