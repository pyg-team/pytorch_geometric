Title: IBM Where to install the DSML (Directory Service Markup Language) service within my network? - United States

Text:
 TECHNOTE (FAQ)

QUESTION
 Where to install the DSML (Directory Service Markup Language) service within my network? 

ANSWER
There are currently two TIPs T0048 and T0047 that communicate with Active Directory. Each require a prerequisite of running the DSML service. The DSML service can be installed on any server running the IIS web server application.
