Title: IBM Security Bulletin: Vulnerability in dojox/form/resources/*.swf and dojox/av/resources/*.swf XSS vulnerability affects  IBM Image Construction and Composition Tool shipped with IBM SmartCloud Orchestrator  (CVE-2014-8917) - United States

Text:
CVE-2014-8917 SECURITY BULLETIN

SUMMARY
 IBM Dojo Toolkit is vulnerable to cross-site scripting and affects IBM Image Construction and Composition Tool (ICCT) shipped with IBM SmartCloud Orchestrator (CVE-2014-8917). 

VULNERABILITY DETAILS
Review Security Bulletin: Vulnerabilities in IBM Dojo Toolkit affect IBM Image Construction and Composition Tool (CVE-2014-8917) [http://www.ibm.com/support/docview.wss?uid=swg21959709] for vulnerability details.

AFFECTED PRODUCTS AND VERSIONS
IBM SmartCloud Orchestrator 2.3
IBM SmartCloud Orchestrator 2.3.0.1

REMEDIATION/FIXES
Resolving the problem: 

 1. Apply fixes referenced in Security Bulletin: Vulnerabilities in IBM Dojo Toolkit affect IBM Image Construction and Composition Tool (CVE-2014-8917) [http://www.ibm.com/support/docview.wss?uid=swg21959709]
    
    
 2. Backup and remove audio.swf, video.swf and uploader.swf from their location.
    
    Example:
    
    mkdir backup
    mv /opt/IBM/icon/icn.app/private/expanded/ibm/icn.dojo-1.2.0.0/public/dojox/av/resources/audio.swf backup/
    mv /opt/IBM/icon/icn.app/private/expanded/ibm/icn.dojo-1.2.0.0/public/dojox/av/resources/video.swf backup/
    mv /opt/IBM/icon/icn.app/private/expanded/ibm/icn.dojo-1.2.0.0/public/dojox/form/resources/uploader.swf backup/
    
    
 3. Restart the ICCT

WORKAROUNDS AND MITIGATIONS
None
