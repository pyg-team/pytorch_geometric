PROBLEM SUMMARY
 *  jextract on winamd64 fails to extract dump generated using
   -Xcompressedref when the core file is placed in a directory
   which is space separated or core file name itslef space
   separated

 *  This defect will be fixed in:
   6.0.0 SR12
   7.0.0 SR3
   .

   space separated arg
