ERROR DESCRIPTION
 *  ERROR DESCRIPTION:
   System will throw SQL exception in Systemout.log after user
   login in LDAP enabled environment and when mxe.ltpatoken.kill=1.
   
   BMXAA6714E - The data for the next record in the mboset could
   not be retrieved for the SQL query select * from ltpatoken
   where token=:1 OPTION (FAST 1000)  . See the log file for more
   details about the error.
   com.microsoft.sqlserver.jdbc.SQLServerException: Incorrect
   syntax near ':'.
   
   PROBLEM:
   When token is null, the system cannot resolve :1 in select
   statement to null which results to SQL exception in log file.
   
   PERFORMANCE ISSUE:  Yes
   
   STEPS TO REPRODUCE:
   n/a
   
   CURRENT ERRONEOUS RESULT:
   Systemout.log produces SQL exception when token is null and
   users login.
   
   EXPECTED RESULT:
   Need to add check for null token to prevent error traces.
