LOCAL FIX

PROBLEM SUMMARY
 *  ERROR DESCRIPTION -
   When you deploy, an application, it fails and you receive an
   StringIndexOutOfBoundsException.
   
   PRODUCTS AFFECTED
   IBM BPM Advanced
   IBM BPM Standard
   IBM BPM Express
   
   LOCAL FIX -
   None
   
   PROBLEM SUMMARY
   When an application is deploying, the installation might be
   interrupted by a StringIndexOutOfBoundsException. You might see
   the following stack trace  in the the SystemOut.log file:
   
   com.lombardisoftware.core.deployment.InstallmentStateMachine
   installPackage CWLLG3535E: The snapshot with
   the ID Snapshot.3ed5e69d-598e-4613-87dc-e8b9b354a68d could not
   be imported.
   
   java.lang.StringIndexOutOfBoundsException
    at java.lang.String.substring(String.java:1150)
    at
   com.ibm.bpm.processcenter.installmessage.InstallMessageHelper.co
   nvertReplacementTextsWithEllipses(InstallMessageHelper.java:834)
    at
   com.ibm.bpm.processcenter.installmessage.InstallMessageHelper.co
   nvertReplacementTexts(InstallMessageHelper.java:802)
   
   This exception occurs due to the truncation of a deployment
   status message which may be masking a lower level exception
   message.
   
   
    
   
   

PROBLEM CONCLUSION
 *  The fix is available for IBM BPM V8.5.6.0. After applying the
   fix, the deployment status  is properly processed and displayed
   to the user.
   
   On Fix Central (http://www.ibm.com/support/fixcentral), search
   for JR53267:
   
   1.Select IBM Business Process Manager with your edition from the
   product selector, the installed version to the fix pack level,
   and your platform, and then click Continue.
   2.Select APAR or SPR, enter JR53267, and click Continue.
   
   When you download fix packages, ensure that you also download
   the readme file for each fix. Review each readme file for
   additional installation instructions and information about the
   fix.
