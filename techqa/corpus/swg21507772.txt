Title: IBM How do I make available new updated software in the FileNet Workplace XT Download directory? - United States

Text:
Download Application Integration in Workplace XT; FIMO; re-create .war; Workplace XT Downloads TECHNOTE (TROUBLESHOOTING)

THIS DOCUMENT APPLIES ONLY TO THE FOLLOWING LANGUAGE VERSION(S):
 English 

PROBLEM(ABSTRACT)
 The menus in FileNet Workplace XT are looking for specific executables in the download directory. 

SYMPTOM
Users need access to newer version of FIMO.


CAUSE
Users need access to a newer version of FIMO or BCS. How can this be made available to FileNet Workplace XT users?

RESOLVING THE PROBLEM



 1. Copy the updated software executable (like FIMO 1.1.5 for example) to the download directory where XT is deployed, for example
    
    ...WebSphere\AppServer\profiles\AppSrv01\installedApps\P8ADNode01Cell\WorkplaceXT.ear\web_client.war\download
    
    
 2. Re-create the Workplace XT .war file 
    
    
 3. Re-deploy Workplace XT
