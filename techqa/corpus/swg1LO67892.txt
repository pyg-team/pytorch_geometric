Title: IBM LO67892: BATTERY LIFE REDUCED USING SECUIRTY POLICY WITH 8.5.3.2 NOKIA CLIENT. - United States

Text:
 SUBSCRIBE
You can track all active APARs for this component.



APAR STATUS
 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  Customer reported that after enabling security policy for their
   Nokia users, the Nokia device would continously send security
   status to the server.  They saw this message in the device logs
   repeatedly:
   
   19/01 09:51:39 Security settings update successfully uploaded to
   the server.
   
   
   The problem was seen using Lotus Traveler 8.5.3.2 client on
   Nokia devices.  The problem would also reduce battery life on
   the Nokia device.
   
   
    
   
   

LOCAL FIX
 *  NA
   
   
    
   
   

PROBLEM SUMMARY
 *  Customer found that when they turned on
   the security policy on the server that the battery life would be
   greatly reduced on some Nokia devices.
   
   
    
   
   

PROBLEM CONCLUSION
 *  It was determined that the device was continually sending its
   security status to the server.  The Lotus Traveler client for
   Nokia devices was updated to resolve this problem.
   
   
    
   
   

TEMPORARY FIX
 *  This fix will be included in Lotus Traveler 8.5.3.3 and all
   later releases.  See this technote for the latest maintenance
   release information.
   
   http://www.ibm.com/support/docview.wss?uid=swg24019529 [http://www.ibm.com/support/docview.wss?uid=swg24019529]
