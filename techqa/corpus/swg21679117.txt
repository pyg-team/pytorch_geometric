Title: IBM An installation failure message is seen when deploying a snaphot to IBM Business Process Manager (BPM) - United States

Text:
install; failed; install failed; snapshot; export; snapshot export; export error; export failed TECHNOTE (TROUBLESHOOTING)

PROBLEM(ABSTRACT)
 When you deploy a snapshot to the Process Server for IBM Business Process Manager, the following error message is displayed: "java.lang.Exception: The install failed. Could not export snapshot" 

RESOLVING THE PROBLEM
To resolve the problem, make sure that there is enough free space available in the file system.
