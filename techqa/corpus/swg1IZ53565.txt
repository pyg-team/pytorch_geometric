Title: IBM IZ53565: CANNOT DELETE A BOOKMARK WHEN THE USERID AND LOGINID VALUES ARE DIFFERENT FOR A USER - United States

Text:

APAR STATUS
 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  Cannot delete a bookmark when the user has a different
   MAXUSER.USERID than the MAXUSER.LOGINID.
   
   Steps:
   Go To Security / Users application
   Enter a new record.
   Enter TEST55 for User value.
   Enter MRSMITH for User Name field value (LOGINID)
   Enter TEST1 for Person value
   Select Set Password button and enter maximo for password.
   Uncheck Email password and Expire on first login options.
   Hit OK
   Select Group tab, select New Row and select MAXADMIN group.
   Save
   Sign Out
   Login as MRSMITH and go to Wotrack
   Select a record.
   Select Action / Add to Bookmarks
   Select List tab and select bookmark link
   You will see the bookmark created in previous steps
   Select the Red X to delete the bookmark
   nothing happens and bookmark is not deleted
   This has to do with the userid being different from the loginid
   If you repeat the steps as wilson, the bookmark can be deleted.
