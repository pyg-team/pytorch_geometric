APAR STATUS
 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  FTP Front Side handler ends the connection when passive port
   equals 65535. But according to RFC 6056:
   "The Dynamic and/or Private Ports, 49152 through 65535".
   65535 should be a valid port for a dynamic port.
   
   
    
   
   

LOCAL FIX

PROBLEM SUMMARY
 *  Affected is the use of FTP with DataPower.
   
   FTP Front Side handler ends the connection when passive port
   equals 65535, although rfc6056 allows that port to be used.
   
   
    
   
   

PROBLEM CONCLUSION
 *  Fix is available in 6.0.0.18, 6.0.1.14, 7.0.0.11, 7.1.0.8 and
   7.2.0.3
