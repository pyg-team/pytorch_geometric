APAR STATUS
 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  Rational Developer for System z 9.0.1.4 LPEX SAVE TOOLBAR ICON
   is unexpectedly disabled.
   
   After editing a source member and making changes the save icon
   (diskette in the tool bar) may be found to be grayed out.  This
   may happen after the editor has lost then gained focus even
   though the changes are still not saved.
   
   This may be a similar problem as RDi APAR SE58364 [http://www-01.ibm.com/support/docview.wss?uid=swg1SE58364]
   
   - edit COBOL program in LPEX
   - right-click on the editor and select menu Software Analyzer
   >Software
   Analyzer Configurations
   - Software Analyzer dialog opens and Save menu becomes disabled
   - close the Software Analyzer dialog
   - Save menu is still disabled
   
   
    
   
   

LOCAL FIX
 *  The CTRL+S key sequence can be used to save the changes.
   
   Upgrade to RDz 9.5 and above
   
   
    
   
   

PROBLEM SUMMARY
 *  After editing a source member and making changes the save icon
   (diskette in the tool bar) may be found to be grayed out.  This
   may happen after the editor has lost then gained focus even
   though the changes are still not saved.
   
   
    
   
   

PROBLEM CONCLUSION
 *  The save icon is no longer incorrectly greyed out after a loss
   of focus.
   
   The problem listed in this APAR has been resolved with
   IBM Rational Developer for System z V9.0.1.6 Fix Pack which
   is available from the Recommended Fixes support download page:
   http://www-1.ibm.com/support/docview.wss?rs=2294&uid=swg27006335 [http://www-1.ibm.com/support/docview.wss?rs=2294&uid=swg27006335]
   
   
    
   
   

TEMPORARY FIX
