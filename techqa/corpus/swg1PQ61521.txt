ERROR DESCRIPTION
 *  Using DB2 Admin Tool command, ALTER COLUMN ( ALC ), to add a new
   column to a table, followed by either:
     LAB line command to add a LABEL to the new column, or
     COM line command to add a COMMENT to the new column
   fails with RC1012 in step CNVRT50 with msgadbc119e :
     ' ADBC119E Column to be deleted has no old name '
   .
   Additional symptom:
   After INSERT line command (I) to add a new column, the new
   column information is added to panel ADB27C , and column titled
   'Operation Type' contains the value ' INSERT '.
   .
   Following either LAB or COM line commands, the value of
   'Operation Type' on panel ADB27C is changed to ' UPDATE '
   for the new column.
   .
   Additional symptoms:  MSGADBC119E  CC1012
