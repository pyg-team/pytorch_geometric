PROBLEM SUMMARY
 *  ****************************************************************
   * USERS AFFECTED:  All users of IBM WebSphere Application      *
   *                  Server                                      *
   ****************************************************************
   * PROBLEM DESCRIPTION: Display IBM SDK Java Technology         *
   *                      Edition version 6 end of service        *
   *                      message during fix pack update.         *
   ****************************************************************
   * RECOMMENDATION:                                              *
   ****************************************************************
   IBM SDK Java Technology Edition version 6 installed with IBM
   WebSphere Application Server in the install_root/java
   directory will be end of service in April 2018. A warning
   message should be displayed during fix pack update to
   recommend customers install and switch the runtime JVM to
   IBM WebSphere Java Technology Edition version 8.
   
   
    
   
   

PROBLEM CONCLUSION
 *  An end of service warning message for IBM SDK Java Technology
   Edition version 6 will be displayed during IBM WebSphere
   Application Server fix pack update. Customers are recommended
   to start using Java 8 by installing and switching the
   WebSphere Application Server runtime JVM to IBM WebSphere
   Java Technology Edition version 8.
   
   For a new installation of WebSphere Application Server
   version 8.5.5.11, customers can choose to install with IBM SDK
   Java Technology Edition version 8 or version 6 in the
   install_root/java directory, where IBM SDK
   Java Technology Edition version 8 is the default choice.
   
   The fix for this APAR is currently targeted for inclusion in
   fix pack 8.5.5.11.  Please refer to the Recommended Updates
   page for delivery information:
   http://www.ibm.com/support/docview.wss?rs=180&uid=swg27004980 [http://www.ibm.com/support/docview.wss?rs=180&uid=swg27004980]
