Title: IBM Customize TEPS LDAP authentication attribute - United States

Text:
5724C04PS TECHNOTE (FAQ)

QUESTION
 How do I use a login attribute besides uid for TEPS LDAP authentication? 

ANSWER
Modify the <ITMHOME>/<arch>/iw/profiles/ITMProfile/config/cells/ITMCell/wim/config/wimconfig.xml file to contain the following two lines: 

1) <config:attributes name="alternate_loginid" propertyName="uid"/> 
beneath the "<config:attributeConfiguration>" tag where alternate_loginid is the attribute your LDAP server uses in place of uid. This will map the alternate_loginid attribute from your LDAP server to the uid field within eWAS. 

2) <config:loginProperties>uid</config:loginProperties> 
beneath the "<config:baseEntries name=" tag specifying the base DN of the LDAP server 

Once the changes have been made and the file is saved, restart the TEPS.
