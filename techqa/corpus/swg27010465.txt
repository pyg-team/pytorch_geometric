ABSTRACT
 This page provides links to the recommended maintenance packages available for IBM WebSphere Service Registry and Repository. 

CONTENT


 Software maintenance updates for WebSphere Service Registry and Repository are supplied in the form of cumulative fix bundled packages, called Fix Packs or Refresh Packs. These are summarized in the tables below, organized by release.

The packages are cumulative, so that each new package has all the fixes contained within the previous package, in addition to the new fixes. Therefore, always install the latest package to bring your installation up to the specified version.

Note that WebSphere Service Registry and Repository Studio Version 8.5 has now replaced all previous Studio versions. If you are using WebSphere Service Registry and Repository Version 8.0 or 8.5 use WebSphere Service Registry and Repository Studio Version 8.5.



 * WebSphere Service Registry and Repository Version 8.5
 * WebSphere Service Registry and Repository Studio Version 8.5
 * WebSphere Service Registry and Repository Version 8.0
