ERROR DESCRIPTION
 *  1) Create a new route (e.g. TEST)
   
   2) Add the following location to the route:
       - OFF101
   
   Make sure to set route configured to generate route stops as
   work order tasks.
   
   
   3) Modify job plan 401 so that each task has an FS predecessor
   to the previous task. So add 10 as a predecessor for 20 and 20
   as step 30 and so on till you get to last one.
   
   4) Create a new PM at site BEDFORD (e.g. 1020):
       - Route: Route-TEST
       - Frequency: 1 DAYS
       - Estimated next due date: Today
       - Job plan: 401
   
   5) Set PM to ACTIVE
   
   6) Disable GL component ACTIVITY: 300
   
   1) Go To > Financial > Chart of Accounts
   2) Select Action -> GL Component Maintenance
   3) Select Component ACTIVITY
   4) Unselect Active for GL Component Value 300.
   
   7) Generate work orders for PM 1020 using frequency criteria
   for 7 days. You will get error of:
   
   BMXAA4214E - An unknown error has occurred. Please contact your
   system administrator for assistance. Gather the logs from the
   <HOME> directory and determine where and why the error is
   thrown.
   
   PRODUCT VERSION:
   Issue was reproduced against Maximo 7609 with
   Scheduler 767 installed.
