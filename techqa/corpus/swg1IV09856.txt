ERROR DESCRIPTION
 *  Installer failed if the PATH environment variable is longer
   than 1024 characters.
   
        Detailed Recreation Procedure:

            attach more directories to make the environment varible
            longer than 1024 charator
        (2) Installing application support for ITM 623 and getting
            error message:
   
            PATH environment variable length exceeds the maximum,
            abort install.
   
            See Microsoft Knowledge Base KB180410, KB906469,
            KB832978.

LOCAL FIX
 *  Workaround:
   Setup a temporary PATH just for the installer, do the
   installation and put the old PATH back.
   (1)Take current PATH and rename to PATHX
   (2)Create a new PATH shorter than 1024
   (3)Install ITM
   (4)Merge the newly added directories in PATH into PATHX.
   (5)Delete PATH and rename PATHX to PATH.
   Don't start anything other than the ITM installer while PATH is
   reset
