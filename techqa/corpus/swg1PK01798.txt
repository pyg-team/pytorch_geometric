ERROR DESCRIPTION
 *  In DB2 Administration Tool ALC line command is used to insert
   a column into the middle of a table that has COMMENT on both the
   table and COMMENT on columns of the table. Operation ends with
   cc8 in step CREAT61 with sqlcode804 (-804) showing an extra
   semi-colon following the terminating semi-colon of the COMMENT
   on columns. Error messages appear as follows:
   .
   IEF142I VNDO327A CREAT61 - STEP WAS EXECUTED - COND CODE 0008
   .
   *** INPUT STATEMENT:
    ;
   SQLERROR ON            COMMAND, PREPARE    FUNCTION
   .
    RESULT OF SQL STATEMENT:
    DSNT408I SQLCODE = -804, ERROR:  AN ERROR WAS FOUND IN THE
   APPLICATION PROGRAM
             REASON 07
   .
   Additional symptoms: msgdsnt408i 804 rsn07 cc08 rc08 rc8
     semicolon comments msgIEF142I
   
   
    
   
   

LOCAL FIX

PROBLEM SUMMARY
 *  ****************************************************************
   * USERS AFFECTED: Users of DB2 Adminstration Tool R510,        *
   *                 who alter a table with COMMENTS, may         *
   *                 be affected.                                 *
   ****************************************************************
   * PROBLEM DESCRIPTION: An extra semi-colon was generated on    *
   *                      the COMMENT ON table DDL, resulting     *
   *                      an ABEND with a -804 SQLCODE.           *
   ****************************************************************
   * RECOMMENDATION:                                              *
   ****************************************************************
   When ALC line command was used to insert a a column into the
   middle of a table that has COMMENT on both the table and
   COMMENT on columns of the table, operation ended with cc8 with
   sqlcode804 (-804) showing an extra semi-colon following the
   terminating semi-colon of the COMMENT.
   
   
    
   
   

PROBLEM CONCLUSION
 *  ALC alter drop/create table process has been modified to remove
   extra semi-colon.
