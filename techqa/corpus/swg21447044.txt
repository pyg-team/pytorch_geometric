QUESTION
 Do you have sample style sheets to get started on monitoring DataPower MPGW services? 

CAUSE
In order for a DataPower Multi-protocol Gateway service to supply monitoring data to ITCAM for SOA, you must create some XSL style sheets and include them in your DataPower processing rules. The necessary style sheets are described in the following link:
http://publib.boulder.ibm.com/infocenter/tivihelp/v3r1/index.jsp?topic=/com.ibm.itcamsoa.doc/kd4inmst148.htm [http://publib.boulder.ibm.com/infocenter/tivihelp/v3r1/index.jsp?topic=/com.ibm.itcamsoa.doc/kd4inmst148.htm]


ANSWER
You can use the style sheets on the following link as samples from which you can create the files you need. https://www.ibm.com/developerworks/community/wikis/home?lang=en#/wiki/Tivoli%20Composite%20Application%20Manager/page/ITCAM%20for%20SOA%20Sample%20Stylesheets%20for%20DataPower%20MPGW%20Services [https://www.ibm.com/developerworks/community/wikis/home?lang=en#/wiki/Tivoli%20Composite%20Application%20Manager/page/ITCAM%20for%20SOA%20Sample%20Stylesheets%20for%20DataPower%20MPGW%20Services]
