Title: IBM BMT-IMP-0002 Failed to execute metadata request. invalidDriverLocation - United States

Text:
 TECHNOTE (FAQ)

QUESTION
 Unable to connect to Oracle datasource through metadata wizard.
Received the following error: 
"BMT-IMP-0002 Failed to execute the metadata request. 
invalidDriverLocationCAF-WRN-2082 An error has occurred."

No problem when testing data source connection with IBM Cognos Administration.

Windows 2012 R2 - Oracle 11G R2 


CAUSE
Oracle JDBC drivers problem on the Cognos BI server !
The JDBC driver was copied to both directories. <..>\drivers> and <..>\webapps\p2pd\WEB-INF\lib


ANSWER
Remove the ojdbc7.jar file from the directory. <..>\ webapps\p2pd\WEB-INF\lib and restart Cognos
