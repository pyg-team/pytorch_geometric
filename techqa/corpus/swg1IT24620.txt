APAR STATUS
 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  When using an SSH client or a SFTP client, an I/O error may
   rarely occur.
   
   
    
   
   

LOCAL FIX

PROBLEM SUMMARY
 *  Affected is using the SSH or SFTP protocols as an outbound
   client from the DataPower device.
   Occasionaly I/O errors or handshake
   failures might occur.
   
   
    
   
   

PROBLEM CONCLUSION
 *  Fix is available in 7.5.0.15, 7.5.1.14, 7.5.2.14 and 7.6.0.7
