ABSTRACT
 This fix updates the IBM Java(R) Runtime Environment for the portal server to address documented security vulnerabilities 

DOWNLOAD DESCRIPTION
This fix updates the IBM Java Runtime Environment (JRE) which the portal server distributes to clients. It addresses security vulnerabilities as documented in the Security bulletin: http://www-01.ibm.com/support/docview.wss?uid=swg21616490 [http://www-01.ibm.com/support/docview.wss?uid=swg21616490]


The vulnerabilities addressed by this fix exist when the affected JRE is installed on systems running the Tivoli Enterprise Portal Browser client or Java WebStart client. 

The affected JRE is installed on a system when logging into the IBM Tivoli Enterprise Portal using the Browser client or WebStart client and a JRE at the required level does not exist. The portal provides an option to download the provided JRE to the system. This fix updates the portal JRE bundles that are provided to the end user by the portal.

The fix provided in 6.2-TIV-ITM_JRE_LA0001 provided updated JRE packages for affected client systems. This fix, 6.2-TIV-ITM_JRE_LA0002, provides updated JRE packages for the portal server which can be downloaded by new client systems.

There are two fixes provided:

It contains the updated IBM Java Runtime Environment (JRE) 5.0 SR15 for portal server distribution. 6.X.X-TIV-ITM_JRE_TEP_6.12.00.00 This is a fix for IBM Tivoli Monitoring, Version 6.23.01.00 and above. .It contains the updated IBM Java Runtime Environment (JRE) 6.0 SR12 for portal server distribution.
