Title: IBM Security Bulletin: Multiple security vulnerabilities exist in the IBM SDK, Java Technology Edition provided with WebSphere eXtreme Scale - United States

Text:
 SECURITY BULLETIN

SUMMARY
 Three security vulnerabilities were identified in the IBM® SDK, Java™ Technology Edition provided with WebSphere eXtreme Scale. 

VULNERABILITY DETAILS
CVEID: CVE-2014-0878 [http://cve.mitre.org/cgi-bin/cvename.cgi?name=CVE-2014-0878]

DESCRIPTION: A vulnerability in the IBMSecureRandom implementation of the IBMJCE and IBMSecureRandom cryptographic providers potentially allows an attacker to predict the output of the random number generator under certain circumstances. Refer to the Workarounds and Mitigation section below for additional information.

CVSS:
CVSS Base Score: 5.8
CVSS Temporal Score: See https://exchange.xforce.ibmcloud.com/vulnerabilities/91084 [https://exchange.xforce.ibmcloud.com/vulnerabilities/91084] for the current score
CVSS Environmental Score*: Undefined
CVSS Vector: (AV:N/AC:M/Au:N/C:P/I:P/A:N) 

CVEID: CVE-2014-0453 [http://cve.mitre.org/cgi-bin/cvename.cgi?name=CVE-2014-0453]

DESCRIPTION: An unspecified vulnerability related to the Security component has partial confidentiality impact, partial integrity impact, and no availability impact.

CVSS:
CVSS Base Score: 4
CVSS Temporal Score: See https://exchange.xforce.ibmcloud.com/vulnerabilities/92490 [https://exchange.xforce.ibmcloud.com/vulnerabilities/92490] for the current score
CVSS Environmental Score*: Undefined
CVSS Vector: (AV:N/AC:H/Au:N/C:P/I:P/A:N)
