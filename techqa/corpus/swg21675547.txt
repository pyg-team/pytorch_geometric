Title: IBM Mozilla Firefox Version 27 and Microsoft Internet Explorer 11 and higher are not compatible with the InfoSphere Information Server installation wizard - United States

Text:
isinst113relnotes TECHNOTE (TROUBLESHOOTING)

PROBLEM(ABSTRACT)
 When attempting to view the IBM InfoSphere Information Server or IBM InfoSphere Data Click installation wizard with Mozilla Firefox Version 27 or higher or Microsoft Internet Explorer Version 11 or higher, the page is empty and you cannot proceed. 

CAUSE
Version 27 of Mozilla Firefox and Version 11 of Microsoft Internet Explorer introduced a JavaScript incompatibility with the InfoSphere Information Server installation wizard.

RESOLVING THE PROBLEM
Use a lower version of Mozilla Firefox or Microsoft Internet Explorer Version 9 or 10. The installation wizard has also reported to work well with Google Chrome.
