Steps to reproduce



 1. Enable resource monitoring in the Performance Schedule Details.
    
    
 2. Click Next ... .
    
    
 3. Select radio button Create new resource monitoring location.
    
    
 4. Select IBM WebSphere PMI monitoring.
    
    
 5. Click Resource tab.
    
    
 6. Observe the error dialog box.


CAUSE

TLS_ECDHE_RSA_WITH_AES_256_CBC_SHA means that the application uses high encryption.

RESOLVING THE PROBLEM
Install the high-encryption libraries. See the following document.


 1245273: Enabling Strong encryption greater than 128 bit key lengths in Rational Performance Tester [http://www.ibm.com/support/docview.wss?uid=swg21245273]
