QUESTION
 How do you enter values for a collection in a Decision Validation Services (DVS) scenario file, whether for setting parameter values or testing expected results? 

ANSWER
As stated in the documentation Excel scenario files [http://publib.boulder.ibm.com/infocenter/brjrules/v7r0m3/topic/com.ibm.websphere.ilog.jrules.doc/Content/Business_Rules/Documentation/_pubskel/JRules/ps_JRules_Global739.html] (WebSphere ILOG Rules 7.0.3): 

"When you test for a list of values, you enter each item in the corresponding cell on a separate row. It is not necessary to duplicate the Scenario ID for each row since the SSP uses the last ID entered."
