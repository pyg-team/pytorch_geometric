Title: IBM IZ19176: NETCOOL/OMNIBUS GATEWAY FOR REMEDY ARS DOCUMENTATION INCORRECTLY STATES SUPPORT FOR REMEDY 6.3 ON AIX - United States

Text:
 SUBSCRIBE
You can track all active APARs for this component.



APAR STATUS
 * CLOSED AS DOCUMENTATION ERROR.
    
   
   

ERROR DESCRIPTION
 *  The Netcool/OMNIbus Gateway for Remedy ARS documentation
   incorrectly states on page 4 that the gateway supports Remedy
   6.0, 6.3 and 7.0 on AIX 5.0, 5.1, 5.2 and 5.3:
   
   
   http://publib.boulder.ibm.com/infocenter/tivihelp/v8r1/topic/com
   .ibm.netcool_OMNIbus.doc/gateways/remgw/remgw-pdf.pdf [http://publib.boulder.ibm.com/infocenter/tivihelp/v8r1/topic/com.ibm.netcool_OMNIbus.doc/gateways/remgw/remgw-pdf.pdf]
   
   
   This information is in conflict with the Gateway patch "Readme"
   file which correctly states that the gateway does not support
   Remedy 6.x or 7.x on AIX.
   
   
    
   
   

LOCAL FIX
 *  None available.
   
   
    
   
   

PROBLEM SUMMARY
 *  ****************************************************************
   * USERS AFFECTED: All users of the Gateway for Remedy ARS.     *
   ****************************************************************
   * PROBLEM DESCRIPTION: The gateway does not support Remedy     *
   *                      versions  6.3  and 7.0 on AIX platform. *
   *                      However, the IBM Tivoli Netcool/OMNIbus *
   *                      Gateway  for Remedy ARS Reference Guide *
   *                      has incorrect information about the     *
   *                      support for Remedy versions  6.3 and    *
   *                      7.0 on AIX platform.                    *
   ****************************************************************
   * RECOMMENDATION:                                              *
   *                                                              *
   ****************************************************************
   Support for Remedy versions  6.x and 7.x on AIX platform will
   be removed  from the Summary section of the guide. Support for
   Remedy version 6.0 will  be retained.
   
   
    
   
   

PROBLEM CONCLUSION
 *  The product documentation will be updated with the 7.2.1
   release of  Netcool/OMNIbus.
   
   
    
   
   

TEMPORARY FIX
