Title: IBM Install BIRT Reports on Tivoli Common Reporting 2.1.1 - United States

Text:
birt reports; tcr; tcr 2.1.1; manually install birt reports; data source for birt reports; trcmd -import; trcmd -modify TECHNOTE (TROUBLESHOOTING)

PROBLEM(ABSTRACT)
 ITCAM for Applications 7.1 includes ITCAM for SOA 7.1.1.3 and Tivoli Common Reporting 2.1.1.
The instructions available in the ITCAM for SOA 7.1.1 User's Guide in Appendix E refer to an older version of Tivoli Common Reporting, therefore, instructions for installing and setting the data source for BIRT reports are incorrect. 

RESOLVING THE PROBLEM
Manually install and configure BIRT Reports as follows: 

1. Copy the required JDBC driver files to the following directory: 
TCR_home\tipv2Components\TCRComponent\lib\birt-runtime-2_2_2\ReportEngine\plugins\org.eclipse.birt.report.data.oda.jdbc_2.2.2.r22x_v20071206\drivers

The required JDBC files are as follows:
-For DB2, copy db2jcc.jar and db2jcc_licence_cu.jar 
-For Oracle, copy ojdbc14.jar and ojdbc14_g.jar
-For Microsoft SQL, copy sqljdbc.jar

2. To import BIRT reports, run the following command from the TCR_home\tipv2Components\TCRComponent\bin directory:

trcmd -import -bulk pkgFile [-reportSetBase rsBase] [-resourceBase resourceBase] [-designBase designBase] [-help]

For example,

trcmd -import -bulk C:\ITCAM\SOA711FP3\KD4\Reports\ITCAMForSOAReports.zip -user tipadmin -password tipadmin

3. To set the data source, run the following command from the TCR_home\tipv2Components\TCRComponent\bin directory:


trcmd -modify -dataSources -reports [-reportName cognosSearchPath] [-displayName displayNameQuery] [-dataSource p1 p2 p3] [-caseSensitive] -setDatasource p1 p2 p3 [-help]

For example, 

trcmd -user administrator -password password -modify -datasources -reports -reportname "/content/package[@name='Tivoli Products']/folder[@name='ITCAM for SOA Reports']/report[@name='Services Inventory']" -setdatasource odaDriverClass=com.ibm.db2.jcc.DB2Driver "odaURL=jdbc:db2:/
/9.162.125.34:50000/WAREHOUS:currentSchema=ITMUSER;" odaUser=db2admin odaPassword=4db2admin 
Note: In all steps, replace TCR_home with the name of the Tivoli Common Reporting installation directory
