APAR STATUS
 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  Many DataPower log messages are parameterized with strings or
   integers that are filled in only at the time when the log
   message is issued by the firmware.  In firmware releases 6.0.0.0
   through 6.0.0.3 there are some non-English log messages where
   the parameters do not properly match those in the English
   version of the message.
   
   When an incorrectly parameterized log message
   fires while in the locale in question unpredictable results may
   occur ranging from garbled log message output to a firmware
   restart.
   
   
    
   
   

LOCAL FIX
 *  Avoid the use of non-English system locales.
   
   
    
   
   

PROBLEM SUMMARY
 *  When the appliance uses a non-English locale some log messages
   with incorrect parameters may cause garbled log message output
   or firmware reloads.  This only applies to firmware releases
   6.0.0.0 - 6.0.0.3.
   
   The message catalog files that contained at least one message
   with incorrect parameters were:
