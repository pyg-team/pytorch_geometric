APAR STATUS
 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  In IBM Content Navigator (ICN), if the currently logged in user
   is added to the 'Allowed' list of the "Limit access to specific
   users and groups" configuration, the user will not be able to
   save any further changes to the desktop.
   
   
    
   
   

LOCAL FIX

PROBLEM SUMMARY
 *  When modifying a desktop with a CM8, CMIS, CMOD, or plugin
   authenticating repository, the Save and Save and Close buttons
   are not enabled and the changes can not be saved.
   
   
    
   
   

PROBLEM CONCLUSION
 *  With this fix, the Save and Save and Close buttons are enabled
   after changing the desktop configuration.
