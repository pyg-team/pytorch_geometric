Title: IBM WebSphere Business Integration Adapter for Siebel eBusiness Applications V4.6.3 supports Siebel V8.1.1 - United States

Text:
Adapter for Siebel eBusiness Applications; artifacts; Siebel 8.1.1 TECHNOTE (TROUBLESHOOTING)

PROBLEM(ABSTRACT)
 WebSphere Business Integration Adapter for Siebel eBusiness Applications Version 4.6.3 now supports Siebel V8.1.1. 

However, the sample .sif file shipped with WebSphere Business Integration Adapter for Siebel eBusiness Applications V4.6.3 cannot be used for creating IBM Events Business Component and IBM Archive Business Component on Siebel V8.1.1. The Siebel Javaâ„¢ Data Bean API V8.1.1 requires Java Runtime Environment (JRE) 5.0 

RESOLVING THE PROBLEM
Use the following step wise procedure to get your WebSphere Business Integration Adapter for Siebel eBusiness Applications to support Siebel V8.1.1: 

1. Install fix pack V4.6.3 

Follow the fix Pack V4.6.3 instructions to install this FixPack. Obtain the fix pack V4.6.3 from the following link: 
http://www.ibm.com/support/docview.wss?rs=0&q1=Siebel+4.6.3&uid=swg24003719&loc=en_US&cs=utf-8&cc=us&lang=en [http://www-1.ibm.com/support/docview.wss?rs=0&q1=Siebel+4.6.3&uid=swg24003719&loc=en_US&cs=utf-8&cc=us&lang=en_] 

2. Upgrade the JRE used by the adapter to JRE V5.0 

Upgrade the JRE used by the adapter to IBM JRE V5.0. The following link provides detailed instructions to upgrade to IBM JRE V5.0 manually.http://www.ibm.com/support/docview.wss?rs=695&context=SSMKUK&dc=D600&uid=swg21265380&loc=en_US&cs=UTF-8&lang=en [http://www-1.ibm.com/support/docview.wss?rs=695&context=SSMKUK&dc=D600&uid=swg21265380&loc=en_US&cs=UTF-8&lang=en_] 

3. Import the ibm_audit.sif file 

The attached ibm_audit.sif file contains the IBM Audit project and can be imported into the Siebel Tool V8.1.1 to create the IBM Events business component and IBM Archive business component 

4. Change the server scripts of the business component that you want the adapter to monitor
Use the following step wise procedure to change the server scripts of the business component that you want to monintor: 

 1. Open Siebel Tool V8.1.1, in Object Explorer windows. Click on the Business Component node and select the business component that you want the adapter to monitor. 
 2. Right- click on the selected business component and select Edit Server Scripts. A script editor is displayed. On the left hand navigation of the script editor window, select the function that you want to edit. In the right hand navigation of the script editor window change the function definition.
