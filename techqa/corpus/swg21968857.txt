Title: IBM Security Bulletin: Multiple vulnerabilities have been identified in WebSphere 
Application Server shipped with WebSphere Enterprise Service Bus Registry Edition (CVE-2015-1283, CVE-2015-3183) - United States

Text:
 SECURITY BULLETIN

SUMMARY
 WebSphere Application Server is shipped as a component of WebSphere Enterprise Service Bus Registry Edition. Information about the security vulnerabilities affecting WebSphere Application Server have been published in a security bulletin. 

VULNERABILITY DETAILS
Please consult the security bulletins Denial of service may affect IBM HTTP Server [http://www-01.ibm.com/support/docview.wss?uid=swg21964428] and HTTP Request smuggling vulnerability may affect IBM HTTP Server [http://www-01.ibm.com/support/docview.wss?uid=swg21963361] for vulnerability details and information about fixes.

AFFECTED PRODUCTS AND VERSIONS
All version of WebSphere Enterprise Service Bus Registry Edition
