Title: IBM ITCAM for J2EE 6.2 does not start on HP with ITM 6.2.3 FP1 - United States

Text:
KYJ; HP; ITCAM; J2EE; TEMA; ITM; KCIIN0206E; KDY1024E TECHNOTE (TROUBLESHOOTING)

PROBLEM(ABSTRACT)
 ITCAM for J2EE Agent 6.2 can not be started successfully where it is installed with ITM 6.2.3 FP1. 
Error message:
KCIIN0206E Could not start/stop agent : com.candle.kin.mcs.plugin.McsPluginException: KWJ_LIBJVM does not exist: /opt/IBM/ITM/JRE/hp11/lib/PA_RISC/server/libjvm.sl

Side effect of the problem - remote deploy of the agent fails with message:
KDY1024E The agent failed to respond to the command /opt/IBM/ITM/bin/CandleAgent -h /opt/IBM/ITM start yj did not start or stop agent. 

ENVIRONMENT
HP-UX 11.3 64bit PA RISC, ITM 6.2.3 FP1

DIAGNOSING THE PROBLEM
Check the folder lib: 

- {ITM_HOME}/JRE/hp11/lib/PA_RISC exists
and
- {ITM_HOME}/JRE/hp11/lib/PA_RISC2.0 does not exist


RESOLVING THE PROBLEM
At the directory {ITM_HOME}/JRE/hp11/lib/ execute the command: 

cp -r PA_RISC2.0 PA_RISC 


* default value of {ITM_HOME} is /opt/IBM/ITM
