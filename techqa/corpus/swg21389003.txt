CAUSE
The Jazz Rest Services database user may not have full administrator permission on the IBM Rational Composer Server.



RESOLVING THE PROBLEM
Adding the Jazz Rest Services database user to the administrator group will resolve the problem. 

To add the the Jazz Rest Services database user to the administrator group 

 1. Open Control Panel 
 2. Click on Administrative tools 
 3. Click on Computer Management 
 4. Expand the local users and group folder 
 5. Click on the Groups folder 
 6. Right Click on the Administrators 
 7. In the field of members of click add to add the Jazz Rest Services user 


This users should be created on the Operating systems once you have ran the server initialization by default this user is called " jrsuser".
