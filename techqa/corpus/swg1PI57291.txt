ERROR DESCRIPTION
 *  STEPS TO REPRODUCE:
   
   - install RIT/RTCP/Jenkins/Ant on a French operating system
   - setup the GreenHat integration plugin on top of Jenkins
   - generate a Ant script from one of your RIT suite
   - setup a job in Jenkins to run the above Ant script and make
   sure to pick "GH Tester Results" as part of the job post-build
   action
   - execute the job in Jenkins
   - click GH Tester Results
   
   EXPECTED RESULTS :
   The execution report should open. This works fine on English OS
   platform.
   
   ACTUAL RESULTS :
   The page remains blank.
   
   
    
   
   

LOCAL FIX

PROBLEM SUMMARY
 *  ****************************************************************
   * USERS AFFECTED:                                              *
   * All users of the IBM Rational Integration Tester Jenkins     *
   * plugin on non-English workstations.                          *
   ****************************************************************
   * PROBLEM DESCRIPTION:                                         *
   * The Jenkins page does not show any test results in the "GH   *
   * Tester Results" section of the browser.                      *
   ****************************************************************
   * RECOMMENDATION:                                              *
   * Upgrade the Jenkins plugin to that supplied with IBM         *
   * Rational Test Control Panel (RTCP) version 9.0.              *
   ****************************************************************
   
   
    
   
   

PROBLEM CONCLUSION
 *  A fix for this issue was provided in the Jenkins plugin supplied
   with IBM Rational Test Control Panel (RTCP)
