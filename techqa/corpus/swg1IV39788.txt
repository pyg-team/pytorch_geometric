APAR STATUS
 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  Installing or upgrading 6.3 TEPS will fail if IIM is installed
   and runnig during ITM installaion process.  Changes are to
   detect this situation and warn the user during TEPS
   installation.
   
   
    
   
   

LOCAL FIX

PROBLEM SUMMARY
 *  Installation of the portal server fails if JazzSM has been
   previously installed. Before performing installation and
   configuration of the portal server, user needs to stop IBM
   Install Manager process(es).
   
   
    
   
   

PROBLEM CONCLUSION
 *  Common installer detects whether any IBM Install Manager(IIM)
   process is running and does not proceed with installation until
   user stops the processes.
   
   The fix for this APAR is included in the following maintenance
   vehicle:
   
      | fix pack | 6.3.0-TIV-ITM-FP0001 |
   
   
    
   
   

TEMPORARY FIX
 *  Whether informed by the installer or not, user needs to manually
    close IIM before proceeding with portal server installation.
