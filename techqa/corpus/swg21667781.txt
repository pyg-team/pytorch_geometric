Title: IBM Web GUI 7.4 upgrade sequence to have TIP 2.2 FP 13 - United States

Text:
tip; fp13; upgrade TECHNOTE (FAQ)

QUESTION
 You currently have Web GUI 7.3.1 with TIP 2.2 and would like to upgrade to Web GUI 7.4 FP2 with TIP 2.2 FP13 

ANSWER
Steps:

1. Inplace upgrade 7.3.1 to 7.4

2. Upgrade Web GUI 7.4 FP 2

3. Upgrade to TIP to 2.2.0.13. 

As of May 15, 2014, Web GUI 7.4 FP2 is now certified with TIP 2.2.0.13.
