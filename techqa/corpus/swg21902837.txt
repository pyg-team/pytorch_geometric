Title: IBM Security Bulletin: WebSphere Message Broker and IBM Integration Bus are affected by cross-site scripting (XSS) Vulnerabilities in Dojo Toolkit - United States

Text:
WMB IIB SECURITY BULLETIN

SUMMARY
 WebSphere Message Broker and IBM Integration Bus using the IBM Dojo Toolkit and providing the dojox/form/resources/fileuploader.swf, dojox/form/resources/uploader.swf, dojox/av/resources/audio.swf or dojox/av/resources/video.swf files are subjected to a cross-site scripting (XSS) vulnerability. 

VULNERABILITY DETAILS
CVEID:CVE-2014-8917 [http://cve.mitre.org/cgi-bin/cvename.cgi?name=CVE-2014-8917]
DESCRIPTION: IBM Dojo Toolkit is vulnerable to cross-site scripting, caused by improper validation of user-supplied input. A remote attacker could exploit this vulnerability using a specially-crafted URL to execute scripts in a victim's Web browser within the security context of the hosting Web site, when the URL is clicked. An attacker could use this vulnerability to steal the victim's cookie-based authentication credentials
CVSS Base Score: 4.3
CVSS Temporal Score: Seehttp://xforce.iss.net/xforce/xfdb/99303 [http://xforce.iss.net/xforce/xfdb/99303] for the current score
CVSS Environmental Score*: Undefined
CVSS Vector: AV:N/AC:M/Au:N/C:N/I:P/A:N 

AFFECTED PRODUCTS AND VERSIONS
IBM WebSphere Message Broker V8 

 IBM Integration Bus V9

REMEDIATION/FIXES
Follow the steps outlined in the workarounds and mitigations section.
APAR IT06833 is targeted for availability in WebSphere Message Broker V8.0.0.6 and IBM Integration Bus V9.0.0.4 which will remove the affected files.

WORKAROUNDS AND MITIGATIONS
The files affected by this vulnerability are not used by the WebSphere Message Broker or IBM Integration Bus web user interface (web UI) and can be deleted.

The product users are requested to delete the following files to address this vulnerability.
