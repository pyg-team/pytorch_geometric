Title: IBM IC70343: SYSLOG CAN LEAK MEMORY IF DNS RESOLUTION FAILS - United States

Text:

APAR STATUS
 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  Syslog-ng log targets that provide a DNS host name for the
   remote target can leak memory if the DNS resolution fails.
   
   
    
   
   

LOCAL FIX
 *  Avoid the use of DNS and instead use an IP Address.  Otherwise
   do not provide a DNS address that would expectedly fail.
   
   
    
   
   

PROBLEM SUMMARY
 *  Do not leak if DNS resolution fails.
   
   
    
   
   

PROBLEM CONCLUSION
 *  The fix will be in 3.7.3.13, 3.8.0.7, 3.8.1.5
