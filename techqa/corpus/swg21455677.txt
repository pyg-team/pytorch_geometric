Title: IBM Rational Team Concert License Keys cannot be installed in the same package group - United States

Text:
1455677; RTC; IM; license keys; trial; RTC server TECHNOTE (TROUBLESHOOTING)

PROBLEM(ABSTRACT)
This technote explains why license keys cannot be installed with the IBM Rational Team Concert Eclipse Client. Error messages indicating this "cannot coexist in the same package group." or "requires components supplied by other packages." indicate that the user is attempting to install the license keys package with the Eclipse Client or some other Rational Team Concert package.

SYMPTOM
When trying to install "Rational Team Concert Required Base License Keys, including Trials". An error messages indicating the feature"cannot coexist in the same package group." or "requires components supplied by other packages." appears. 

User sees error message as shown below:

CAUSE
The Installation Manager package named "Rational Team Concert Required Base License Keys, including Trials" is meant to be installed with the "Jazz Team Server and CCM Application" (the Rational Team Concert server installation package). 

The package containing license keys needs to be installed into a package group that does contain the Rational Team Concert server.



ENVIRONMENT
Installation Manager on any platform (Microsoft Windows, Linux or UNIX).



DIAGNOSING THE PROBLEM
Look for presence of error messages shown above in Symptom Section



RESOLVING THE PROBLEM
Install the Installation Manager package containing the license keys is selected for installation into a package group that also contains the server.
