STERLINGPRI TECHNOTE (TROUBLESHOOTING)

PROBLEM(ABSTRACT)
 Service threw an exception INVALID_CHARACTER_ERR: An invalid or illegal XML character is specified. (SCI71700) 

SYMPTOM
Service threw an exception INVALID_CHARACTER_ERR: An invalid or illegal XML character is specified.


CAUSE
Sender ID contains a comma, which is valid for that field, but causes Gentran 
Integration Suite to error.

RESOLVING THE PROBLEM
Install patch 14 or later.
