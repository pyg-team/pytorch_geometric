ERROR DESCRIPTION
 *  On ISC login page if user enters invalid user id and clicks on
   forgot password then ISC login page displays 'undefined'
   message not a proper message.
   
   In SystemOut.log you can find the bellow exception,
   
    [8/17/15 10:59:30:323 IST] 000000fa webapp        E
   com.ibm.ws.webcontainer.webapp.WebApp logServletError

   .java:113)
    at
   com.ibm.isim.rest.services.PasswordService.getPersonPasswordRese
   tChallenge(PasswordService.java:276)
    at sun.reflect.NativeMethodAccessorImpl.invoke0(Native
   Method)
   
   Environment : ISIM 6.0.
   
   Steps to Duplicate:
   1. Access ISC Login page.
   2. Enter invalid user id.
   3. Click forgot password link.
   4. Login page displays 'undefined' message.
   
   Desired Behavior :
   ISC login page should display proper message on UI instead of
   'undefined' message .
