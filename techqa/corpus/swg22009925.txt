Title: IBM After Logging Into Optim Manager, a Blank Page is Displayed - United States

Text:
 TECHNOTE (TROUBLESHOOTING)

PROBLEM(ABSTRACT)
 A blank page is displayed after logging into Optim Manager when using Microsoft Edge browser. 

SYMPTOM
After clicking the icon to start the Optim Manager, and then logging in, a blank page is displayed.


CAUSE
Microsoft Edge is the default browser for Windows 10. However, it is not supported by Optim Manager.

ENVIRONMENT
Windows 10

RESOLVING THE PROBLEM
Optim Manager is not supported on Microsoft Edge. Use one of the supported browsers which are Chrome, Firefox and Internet Explorer. You should configure your system so that one of these is the default browser so when you click on the Optim Manager icon the default browser is used.
