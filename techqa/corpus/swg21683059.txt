Title: IBM Required offerings for WebSphere Application Server Patterns must be downloaded from Fix Central - United States

Text:
WASPatternsV1RNotes TECHNOTE (TROUBLESHOOTING)

PROBLEM(ABSTRACT)
 The IBM Installation Manager repository does not contain the required offerings to run your WebSphere Application Server Patterns configuration. 

RESOLVING THE PROBLEM
You must download the required offerings for WebSphere Application Server Patterns from Fix Central to the IBM Installation Manager repository. 

IBM WebSphere Application Server Patterns supports IBM WebSphere Application Server Version 8.0.0.9 and 8.5.5.2. 

The following tables contain the supported WebSphere Application Server offerings and additional features that the software components can install.

 * WebSphere Application Server 8.0.0.x software components and fix pack offerings
 * 
 * WebSphere Application Server 8.0.0.x additional features
