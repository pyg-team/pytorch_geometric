 DOWNLOADABLE FILES

ABSTRACT
 This cumulative fix pack updates JRules V7.1.1 (7.1.1, 7.1.1.1, 7.1.1.2, 7.1.1.3 or 7.1.1.4) to V7.1.1.5. 

DOWNLOAD DESCRIPTION
This cumulative fix pack updates JRules V7.1.1 (7.1.1, 7.1.1.1, 7.1.1.2, 7.1.1.3 or V7.1.1.4) to V7.1.1.5. 

This fix pack does not work directly with earlier versions of JRules. Users of V7.0 and V7.1.0 should update to V7.1.1 in order to install V7.1.1.5.

This fix pack is entitled. Only customers who have purchased JRules V7.1.1 can download this fix pack.

PREREQUISITES
WebSphere ILOG JRules V7.1.1
