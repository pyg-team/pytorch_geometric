Title: IBM Unable to import tables/views against SAP HANA - United States

Text:
 TECHNOTE (TROUBLESHOOTING)

PROBLEM(ABSTRACT)
 Unable to import table against Schema _SYS_BIC . Error message:
BMT-IMP-0002 Failed to execute metadata request.
jdbcSQLExceptionCAF-WRN-2082 An error has occurred. Please contact your
administrator 

CAUSE
It is a root cause for these issue. In Cognos 11.0.5 and up SAP hana JDBC provider is 

looking for SCHEMA_NAME in view BIMC_VARIABLE_VIEW. It is only available in SAP HANA 1 version1.0 sps12 and higher 


DIAGNOSING THE PROBLEM
Enable rlmd trace 



RESOLVING THE PROBLEM
SAP hana JDBC provider is looking for SCHEMA_NAME in view BIMC_VARIABLE_VIEW. If the SCHEMA_NAME column is not available then it will generate the above error. 

It is only available in SAP HANA version1.0 sps12 and higher .
