APAR STATUS
 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  An error  with the message "ICWPA1001E An unexpected error
   occurred" is thrown in Atlas when you click edit while
   metrics tab is selected in the Matter Dashboard
   
   
    
   
   

LOCAL FIX
 *  Use the exceptions and scheduled events tabs to edit the matter
   
   
    
   
   

PROBLEM SUMMARY
 *  When a user opens a matter and clicks on the Metrics tab and
   then clicks on the Edit link (to edit the matter), they recieve
   the error below.
   
   ICWPA1001E An unexpected error occurred.
   An unexpected error occurred.
   Contact your system administrator.
   
   This only happens when you are clicking on the Edit link from
   the Metrics tab. Does not happen from the other 2 tabs at the
   Dashboard.
   
   
    
   
   

PROBLEM CONCLUSION
 *  This fix is available now from FixCentral in 6.0.3.3
