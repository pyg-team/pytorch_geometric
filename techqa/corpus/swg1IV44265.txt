APAR STATUS
 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  On TIP console - the agent info are being shown in IPv6 format
   instead of IPv4 format
   
   Example:
   Host XXXXXX08
   
   LMT takes from the CIT output and returns for this
   host IPV6 yyyy:0000:0000:0000:0000:yyyy:yyyy:yyyy instead of
   IPv4 xx.xx.x.xx"
   
   In the results returned by CIT, the hostname values for
   different
   IPAddresses have different letter cases.
   
   <IPAddress version="1" timestamp="1367853829">
     <Index>1</Index>
     <Address IsKey="1">xx.xx.x.xx</Address>
     <Hostname>xxxxxx08</Hostname>
   
    <IPv6Address version="1" timestamp="1367853829">
     <Index>1</Index>
     <Address
   IsKey="1">yyyy:0000:0000:0000:0000:yyyy:yyyy:yyyy</Address>
     <Hostname>XXXXXX08</Hostname>
   
   Since the hostname is defined as:
   
    <UNIXSystemParams version="1" timestamp="1367853836">
    <ComputerAlias>XXXXXX08</ComputerAlias>
   
   The latter one matches.
   
   HostName in group IPAddress and ComputerAlias are received from
   different WinAPI functions. HostName in group IPv6Address we get
   from
   WMI.
   
   The temporary workaround is to change to lowercase the host name
   in the system.
   In this case LMT will match with IPv4.
   
   
    
   
   

LOCAL FIX
 *  The temporary workaround is to change to lowercase the host name
   in the system.
   
   In this case LMT will match with IPv4.
   
   
    
   
   

PROBLEM SUMMARY
 *  ****************************************************************
   * On the Tivoli Integrated Portal console, agent information is
   * shown in IPv6 format instead of IPv4 format. It causes that
   * host name IP addresses are displayed in a non-routable
   hexadecimal
   * format.
   *
   * Behaviour after applying the interim fix:
   * Host names are displayed in the correct format.
   ****************************************************************
   * RECOMMENDATION: Apply appropriate interim fix when available.
   ****************************************************************
