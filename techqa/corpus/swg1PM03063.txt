PROBLEM CONCLUSION
 *  A problem was identified in the code where Execution Results
   were displayed with wrong results when test suite was
   created with "Same Test Case but multiple test scripts" in
   RQM V2.0. The code was fixed to correctly handle the case
   where the only difference in the suite elements was the test
   script.
