 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  In WebSphere Cast Iron Studio, re-opening the project containing
   Invoke RFC activity and browsing the RFC, doesn't refresh the
   schema, when the corresponding RFC in SAP is changed.
   
   
    
   
   

LOCAL FIX
 *  No workaround is available.
   
   
    
   
   

PROBLEM SUMMARY
 *  A product defect meant that reopening the project was not
   updating the RFC details.

   Cloud integration 6.1.0.6 fix pack.
