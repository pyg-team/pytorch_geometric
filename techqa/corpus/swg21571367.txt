Title: IBM Browser tab from RPT displays the message "This program cannot display the webpage" during a script execution - United States

Text:
browser tab; cannot display webpage; RATLC01815532 TECHNOTE (TROUBLESHOOTING)

PROBLEM(ABSTRACT)
 During test execution, the Browser tab of the Protocol Data view in IBM Rational Performance Tester (RPT) displays the error "This program cannot display the webpage". 

SYMPTOM
This message is translated in other languages. It depends on the locale. For example in German the message is as follows. 

The full error message is as follows:

Dieses Programm kann die Webseite nicht anzeigen.


 

You do not see the web page under the transaction execution. Under the Response Content tab, you see expected content. 


CAUSE
RPT versions older than 8.2.1 do not support Internet Explorer 9. Up to version 8.2.1.4, RPT does not display the web page too. For the latter versions, this issue has been identified as a product defect under RATLC01815532. 


RESOLVING THE PROBLEM
FOR VERSIONS 8.2.0X


Downgrade to Internet Explorer 8. [http://windows.microsoft.com/en-us/windows7/how-do-i-install-or-uninstall-internet-explorer-9]




FOR VERSIONS 8.2.1.X UP TO 8.2.1.4


You have two options. 

 

 * Downgrade to Internet Explorer 8.
   
   
 * Update to Rational Performance Tester Fix Pack 5 (8.2.1.5) for 8.2.1 [http://www.ibm.com/support/docview.wss?uid=swg24033156] or newer.
