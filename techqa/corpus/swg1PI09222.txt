ERROR DESCRIPTION
 *  Problem Abstract:
   Unable to configure YN agent from MTEMS GUI after installing
   ITM v6.3 FP02 OS agent
   ---------------------------------
   Users Affected:
   Users configuring TEMA agent via MTEMS user interface
   ---------------------------------
   Problem Description:
   A popup with the title 'Configuration Error' is seen if the
   user attempts to use the GUI to configure the ITCAM for AD v7.1
   FP03 IF7 TEMA agent where an ITM v6.3 FP02 Monitoring agent for
   Unix OS is also present
   .
   To recreate this issue launch the MTEMS using the command
   ./itmcmd manage, select the ITCAM agent and then right click
   and select configure. The popup error is seen and in the
   Messages dialog the following error is visible...
   KCIIN1027E Configuration aborted
   .
   In the resulting CandleManage.log the following error can be
   seen...
   .
   com.ibm.tivoli.monitoring.deployment.config.ui.uiext.ITCAMAgentC
   onfigPanel.<init>(ITCAMAgentConfigPanel.java:183)
   ...
   ..
   .
   Caused by: java.lang.ClassCastException: javax.swing.KeyStroke
   incompatible with java.lang.Comparable
      at java.util.TreeMap.compare(TreeMap.java:1188)
      at java.util.TreeMap.put(TreeMap.java:531)
      at java.util.TreeSet.add(TreeSet.java:255)
   .
   This indicates a potential problem with the TreeSet API
   specification in use not implementing a Comparable interface
   and hence resulting in a specification violation that raises
   the exception seen above at runtime
   
   ---------------------------------
   Keyword/Symptom:
   INCORROUT
   ---------------------------------
   Component ID:
   5724Y9200
   
   ---------------------------------
   Severity: 2
   
   ---------------------------------
   Local Fix/Workaround:
   Use the CLI command
   ./itmcmd config -A yn
   
   ---------------------------------
   Approvals:
   
   L2: Oneil Richardson
   L3: Antonio Moreno
   ---------------------------------
   
   
    
   
   

LOCAL FIX
 *  Use the CLI option to configure the TEMA
   ./itmcmd config -A yn
