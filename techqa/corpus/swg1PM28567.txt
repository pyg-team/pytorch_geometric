ERROR DESCRIPTION
 *  If you try to generate an update site with IBM Rational Software
   Architect for WebSphere v8.0, you will get a 'NoSuchMethod'
   error and it will not be possible to build the update site.
   
   The following steps reproduce the problem:
   
   1. Develop a plug-in project in RSA or RSA for WebSphere v8.0
   (which uses Eclipse 3.6.0)
   2. Develop a feature project that references the plug-in from
   Step 1
   3. Develop an update site project that references the feature in
   Step 2
   4. Open the site.xml in the update site project and click either
   the Build or Build All button
   Expected Result: Update site gets built
   Actual Result: A ' 'Build Site' has encountered a problem' error
   message appears and the following error output is written to the
   .log file:
