APAR STATUS
 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  After upgrading to IBM Content Navigator V2.0.3 Fix Pack 8
   LA002 from V2.0.3 Fix Pack 7, the bookmark URL does not work
   unless the repositoryId parameter is included.
   
   
    
   
   

LOCAL FIX
 *  Specify repositoryId parameter in URL.
   
   
    
   
   

PROBLEM SUMMARY
 *  When the repositoryId parameter is not specified in the
   bookmark URL, the URL is not opened.
   
   
    
   
   

PROBLEM CONCLUSION
 *  This issue has been addressed in IBM Content Navigator V2.0.3
   Fix Pack 8 LA006.
