ERROR DESCRIPTION
 *  The following error can occur while applying the fix PI21973
   which is part of CF14 (8.0.0.1) or CF02 (8.5.0.0):
   
   [wsadmin] WASX7017E: Exception received while running file
   "/opt/IBM/WebSphere/AppServer/profiles/wp_profile/ConfigEngine/c
   
   onfig/work/wp_UpdateWcmLibPortalCommandExtension.jacl";
   exception information: com.ibm.bsf.BSFException: error while
   eval'ing Jacl expression:
   [wsadmin] cannot open file: Web_Content_Templates-PI21973_[YYYYM
   
   MDD-HHMMSS].nochange
   [wsadmin]     while executing
   [wsadmin] "open $reportFile w+"
   ....
   
   where [YYYYMMDD-HHMMSS] is the time stamp of script execution.
   
   This error is likely to happen in case of virtual portals.
