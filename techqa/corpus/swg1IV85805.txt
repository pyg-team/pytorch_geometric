ERROR DESCRIPTION
 *  RFE 83114:
   
   The Linux Sockets Detail attribute group only shows data for
   IPV4 sockets, we need IPV6 data available in same group.
   
   
    
   
   

LOCAL FIX

PROBLEM SUMMARY
 *  Problem Description: Linux IPV6 sockets are not monitored
   
   Problem/Problem Summary: The Linux Sockets Detail and the Linux
    Sockets Status attribute groups in the Monitoring Agent for
    Linux OS only report information about IPV4 sockets.
   
   
    
   
   

PROBLEM CONCLUSION
 *  Problem conclusion: The agent now collects data also for the
    IPV6 sockets.

   maintenance packages:
