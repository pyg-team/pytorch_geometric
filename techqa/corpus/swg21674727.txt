Title: IBM Security Bulletin: A security vulnerability has been identified in Tivoli Integrated Portal (TIP) & embedded Websphere Application Server (eWAS)  shipped with Tivoli Netcool/Impact (CVE-2014-0114) - United States

Text:
 SECURITY BULLETIN

SUMMARY
 Tivoli Integrated Portal (TIP) & embedded Websphere Application Server (eWAS) are shipped as a component of Tivoli Netcool/Impact. Information about a security vulnerability affecting Tivoli Integrated Portal (TIP) & embedded Websphere Application Server (eWAS) has been published in a security bulletin. 

VULNERABILITY DETAILS
Please consult the security bulletin Security Bulletin: ClassLoader manipulation with Apache Struts affecting Tivoli Integrated Portal (CVE-2014-0114) [http://www-01.ibm.com/support/docview.wss?uid=swg21674379] for vulnerability details with regards to Tivoli Integrated Portal.

For Websphere Application Server see Security Bulletin: Classloader Manipulation Vulnerability in IBM WebSphere Application Server CVE-2014-0114 [http://www-01.ibm.com/support/docview.wss?uid=swg21672316].

AFFECTED PRODUCTS AND VERSIONS
 

Principal Product and Version(s) Affected Supporting Product and Version Tivoli Netcool/Impact 6.1.0; 6.1.1 Tivoli Integrated Portal (TIP) 2.x Tivoli Netcool/Impact 5.1.0;5.1.1 Websphere Application Server 6.1 GET NOTIFIED ABOUT FUTURE SECURITY BULLETINS
