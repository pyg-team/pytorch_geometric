COMPUTE AND COMPARE THE CHECKSUM
Use any software to compute the SHA-1 checksum of the downloaded images. If you do not have software to compute the SHA-1 checksum, search the Internet using the following criteria.

+"SHA-1 checksum" +utility

Compare your checksum with the value listed in the text file. If the checksums match, the downloaded image is complete.

 * System i SHA-1 checksums text file [//www-947.ibm.com/systems/support/resources/systemp.txt]
