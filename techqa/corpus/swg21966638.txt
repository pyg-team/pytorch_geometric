Title: IBM Security Bulletin: Vulnerability in IBM WebSphere Application Server Edge Component Caching Proxy (CVE-2015-5004) - United States

Text:
 SECURITY BULLETIN

SUMMARY
 The Edge Component Caching Proxy in IBM WebSphere Application Server could provide weaker than expected security. This is only an issue if you are using the Edge Caching proxy. 

VULNERABILITY DETAILS
CVEID: CVE-2015-5004 [http://cve.mitre.org/cgi-bin/cvename.cgi?name=CVE-2015-5004]
DESCRIPTION: IBM WebSphere Application Server Edge Component Caching Proxy could provide weaker than expected security. This could allow an attacker to decrypt information more quickly and obtain sensitive information.
CVSS Base Score: 5.3
CVSS Temporal Score: See https://exchange.xforce.ibmcloud.com/vulnerabilities/106218 [https://exchange.xforce.ibmcloud.com/vulnerabilities/106218] for the current score
CVSS Environmental Score*: Undefined
CVSS Vector: (CVSS:3.0/AV:N/AC:H/PR:L/UI:N/S:U/C:H/I:N/A:N) 

AFFECTED PRODUCTS AND VERSIONS
The following IBM WebSphere Application Server Versions are affected: 

 * Version 8.5 Full Profile 
 * Version 8.0 

REMEDIATION/FIXES
The recommended solution is to apply the Fix Pack or PTF for each named product as soon as practical. 

Fix: 
Apply an Interim Fix, Fix Pack or PTF [https://www-304.ibm.com/support/docview.wss?rs=180&uid=swg27004980] containing APAR PI41476, as noted below: 

For IBM WebSphere Application Server 
For V8.5.0.0 through 8.5.5.7: 

 * Upgrade to 8.5.5.7 fix pack level then apply Interim Fix 8.5.5-WS-EDGECP-FP00000071 [http://www-933.ibm.com/support/fixcentral/swg/selectFixes?parent=ibm%2FWebSphere&product=ibm/WebSphere/WebSphere+Application+Server&release=All&platform=All&function=fixId&fixids=8.5.5-WS-EDGECP-FP00000071&includeSupersedes=0] 

-- OR  * Upgrade to 8.5.5.6 fix pack level then apply Interim Fix 8.5.5-WS-EDGECP-FP00000062 [http://www-933.ibm.com/support/fixcentral/swg/selectFixes?parent=ibm%2FWebSphere&product=ibm/WebSphere/WebSphere+Application+Server&release=All&platform=All&function=fixId&fixids=8.5.5-WS-EDGECP-FP00000062&includeSupersedes=0]

-- OR  * Apply Fix Pack 8 (8.5.5.8), or later. 
   


For V8.0.0.0 through 8.0.0.11:  * Upgrade to 8.0.0.11 fix pack level then apply Interim Fix 8.0.0-WS-EDGECP-FP000000111 [http://www-933.ibm.com/support/fixcentral/swg/selectFixes?parent=ibm%2FWebSphere&product=ibm/WebSphere/WebSphere+Application+Server&release=All&platform=All&function=fixId&fixids=8.0.0-WS-EDGECP-FP000000111&includeSupersedes=0]
