QUESTION
 When using DB2Â® High Performance Unload, issuing db2hpu results in the following messages:
INZU010I Item starting at (6,1) ending at(7,58)

INZU333E Syntax error in the control file

ANSWER
The character used in the 'FORMAT delimiter delim' parameter must be enclosed in single quotes. 

For example: 

format delimited delim '|'
