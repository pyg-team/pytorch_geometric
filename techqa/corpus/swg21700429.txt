Title: IBM Security Bulletin: Multiple vulnerabilities in IBM Java SDK and IBM Java Runtime affect IBM Tivoli Access Manager for e-business (CVE-2014-6593, CVE-2015-0410) - United States

Text:
PSIRT SECURITY BULLETIN

SUMMARY
 There are multiple vulnerabilities in IBM® SDKs Java™ Technology Edition, Version 5.0, 6, 6R1, 7, 7R1 and IBM® Runtime Environment Java™ Technology Edition that is used by IBM Tivoli Access Manager for e-business. These were disclosed as part of the IBM Java SDK updates in January 2015. 

VULNERABILITY DETAILS

CVEID: CVE-2014-6593 [http://cve.mitre.org/cgi-bin/cvename.cgi?name=CVE-2014-6593]

DESCRIPTION: An unspecified vulnerability in Oracle Java SE and JRockit related to the JSSE component has partial confidentiality impact, partial integrity impact, and no availability impact.

CVSS Base Score: 4
CVSS Temporal Score: See http://xforce.iss.net/xforce/xfdb/100153 [http://xforce.iss.net/xforce/xfdb/100153] for the current score
CVSS Environmental Score*: Undefined
CVSS Vector: (AV:N/AC:H/Au:N/C:P/I:P/A:N)



CVEID: CVE-2015-0410 [http://cve.mitre.org/cgi-bin/cvename.cgi?name=CVE-2015-0410]

DESCRIPTION: An unspecified vulnerability in Oracle Java SE and JRockit related to the Security component could allow a remote attacker to cause a denial of service.

CVSS Base Score: 5
CVSS Temporal Score: See http://xforce.iss.net/xforce/xfdb/100151 [http://xforce.iss.net/xforce/xfdb/100151] for the current score
CVSS Environmental Score*: Undefined
CVSS Vector: (AV:N/AC:L/Au:N/C:N/I:N/A:P)

AFFECTED PRODUCTS AND VERSIONS
IBM Tivoli Access Manager for e-business 6.0, 6.1 and 6.1.1. 

IBM Security Access Manager for Web 7.0 (software installations)

REMEDIATION/FIXES
This vulnerability affects all versions and releases of IBM WebSphere Application Server. The WebSphere Application Server Java Runtime Environment should be updated.

Please determine which fix pack level is appropriate for your environment. 

The fixes for these vulnerabilities are included in IBM SDK, Java 2 Technology Edition, Version 5.0 Service Refresh 16 Fix Pack 9 and subsequent releases
The fixes for these vulnerabilities are included in IBM SDK, Java Technology Edition, Version 6 Service Refresh 16 Fix Pack 3 and subsequent releases
The fixes for these vulnerabilities are included in IBM SDK, Java Technology Edition, Version 6R1 Service Refresh 8 Fix Pack 3 and subsequent releases 
The fixes for these vulnerabilities are included in IBM SDK, Java Technology Edition, Version 7 Service Refresh 8 Fix Pack 10 and subsequent releases
The fixes for these vulnerabilities are included in IBM SDK, Java Technology Edition, Version 7R1 Service Refresh 2 Fix Pack 10 and subsequent releases

Please refer to the Security Bulletin for IBM WebSphere Application Server [http://www-01.ibm.com/support/docview.wss?uid=swg21695362] and the Security Bulletin for IBM Java SDK [http://www-01.ibm.com/support/docview.wss?uid=swg21695474] to remediate your application server.


For Tivoli Access Manager for e-business 5.1, IBM recommends upgrading to a fixed, supported release of the product.
