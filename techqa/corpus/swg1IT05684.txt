APAR STATUS
 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  XC10 Grid Object Documentation states:
   The value can use all alphanumeric characters and the following
   special characters: ! @ # % ^ * & - + = .
   
   However, DataPower XC10 Grid Object for integration does accept
   underscore (_).
   
   
    
   
   

LOCAL FIX
 *  Do not use underscore (_) in XC10 Grid Object Usernames.
   
   
    
   
   

PROBLEM SUMMARY
 *  Affected are configurations using XC10 Grid Objects.
   
   The rules for naming the grid objects in DataPower differ from
   XC10 naming conventions, which might prevent using some
   legitimate XC10 grid object names in DataPower configurations,
   for example those containing underscore character.
   
   
    
   
   

PROBLEM CONCLUSION
 *  Fix is available in 6.0.0.14, 6.0.1.10, 7.0.0.7 and 7.1.0.5.
   
   For a list of the latest fix packs available, please see:
   http://www-01.ibm.com/support/docview.wss?uid=swg21237631 [http://www-01.ibm.com/support/docview.wss?uid=swg21237631]

 *  (none)
