APAR STATUS
 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  Siebel Adapter ignores field value with
   SBL-DAT.
   
   - Problem observed with Siebel Adapter 7.0.0.3
   and WMB 7.0.0.2, AIX
   
   - When a request fails through Siebel Request node,
   then Siebel system can returns an error with tag "Description"
   and the field value <SBL-DAT..>.
   
   - On the Siebel system side, the error message is complete,
   on Siebel Adapter side, the error message does not contain
   any content in the description field, "SBL-DAT" is being
   ignored.
   
   
    
   
   

LOCAL FIX
 *  - no local fix or workaround
   
   
    
   
   

PROBLEM SUMMARY
 *  - When a request fails through Siebel Request node, then Siebel

   Siebel Adapter side, the error message does not contain any
   content in the description field, "SBL-DAT" is being ignored.
   
   
    
   
   

PROBLEM CONCLUSION
 *  This has been fixed in 7.0.0.3_IF04. Please contact IBM Customer
   Support for the fix.
