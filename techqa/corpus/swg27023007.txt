ABSTRACT
 This document provides access to the detailed system requirements information on the supported releases of IBM Business Process Manager Standard. 

CONTENT
The detailed system requirements information is available through the Software Product Compatibility Reports [https://www.ibm.com/software/reports/compatibility/clarity/softwareReqsForProduct.html] web site. This web site provides you with the ability to dynamically generate operating system, prerequisite, server virtualization environment, translation, end of service, and detailed system requirements reports for your specific product, release, and operating system.
