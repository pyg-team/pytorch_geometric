IBM Rational Agent Controller is a daemon process that enables client applications to launch host processes and interact with agents that coexist within host processes. 

Rational Agent Controller, Version 8.2.1.1 supports the following platforms: 

 * IBM z/OS 1.9 64-bit 
 * IBM z/OS 1.10 64-bit 
 * Sun Solaris, version 10 32-bit x86 
 * Sun Solaris, version 10 32-bit x86


For complete details about supported platforms and system prerequisites, see the Installation Instructions file for your operation systems attached below . 
Rational Agent Controller, Version 8.2.1.1 can be used to profile applications running in the following versions of IBM WebSphere Application Server:  * V6.1 and service releases 
 * V7.0 and service releases 


Note: Profiling applications on WebSphere Application Server, V6.0, is not supported for Rational Agent Controller V8.2.1.1 on z/OS or Solaris x86.. 
For support of profiling on with WebSphere Application Server, Version 6.0, on z/OS, use Rational Agent Controller v7.x, which is available at: http://www-01.ibm.com/support/docview.wss?uid=swg27013420 [http://www-01.ibm.com/support/docview.wss?uid=swg27013420]. INSTALLATION INSTRUCTIONS
