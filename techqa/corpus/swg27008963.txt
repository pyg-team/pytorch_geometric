Title: IBM Presentation on Using the IBM Time Zone Update Utility for Java (JTZU) - United States

Text:
DST2007; DST; DST; DST; JTZU; time zone update utility; Daylight Saving Time; Daylight savings time EDUCATION

ABSTRACT
 This presentation (PDF) contains information about using the IBM® Time Zone for Java™ (JTZU) to update your Java SDKs/JREs for Daylight Saving Time (DST) changes taking place in the US, Canada, Bermuda, and Western Australia. 

CONTENT
This presentation provides information about the JTZU, including the following topics: 

 * DST Update requirement for Java 
 * Java Service fix strategy for DST updates 
 * Alternative solution: Time Zone Update Utility 
 * Overview of the IBM Time Zone Update Utility 
 * Graphical Mode 
 * Command Line Mode 
 * How the JTZU works 
 * Known problems with Java and Time Zones 
 * Guidance on using JTZU vs. applying Service Fixes
