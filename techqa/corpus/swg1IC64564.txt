APAR STATUS
 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  Grepping string ( / - Set regexp ) ended with '\' in searchable
   screens, such as Statements, Session, Tablespaces, etc., in
   interactive mode will trap db2top on AIX.
   
   
    
   
   

LOCAL FIX
 *  Don't search for string ended with '\'.
   
   
    
   
   

PROBLEM SUMMARY
 *  GREPPING STRING ENDED WITH '\' IN SEARCHABLE SCREENS IN
   INTERACTIVE MODE WILL TRAP DB2TOP ON AIX.

 *  Grepping string ( / - Set regexp ) ended with '\' in searchable
   screens, such as Statements, Session, Tablespaces, etc., in
   interactive mode will trap db2top on AIX.
