Title: IBM Security Bulletin:Vulnerability in OpenSSL affects IBM PureApplication System. (CVE-2015-1788) - United States

Text:
 SECURITY BULLETIN

SUMMARY
 An OpenSSL denial of service vulnerability disclosed by the OpenSSL Project affects GSKit. IBM PureApplication System uses GSKit in user registry components in the Web application pattern type and GPFS pattern type. IBM PureApplication System addressed the applicable CVE. 

VULNERABILITY DETAILS
CVEID: CVE-2015-1788 [http://cve.mitre.org/cgi-bin/cvename.cgi?name=CVE-2015-1788]

DESCRIPTION: OpenSSL is vulnerable to a denial of service, caused by an error when processing an ECParameters structure over a specially crafted binary polynomial field. A remote attacker could exploit this vulnerability to cause the application to enter into an infinite loop.

CVSS Base Score: 5
CVSS Temporal Score: See https://exchange.xforce.ibmcloud.com/vulnerabilities/103778 [https://exchange.xforce.ibmcloud.com/vulnerabilities/103778] for the current score
CVSS Environmental Score*: Undefined
CVSS Vector: (AV:N/AC:L/Au:N/C:N/I:N/A:P)

AFFECTED PRODUCTS AND VERSIONS
IBM PureApplication System V2.1 
IBM PureApplication System V2.0

REMEDIATION/FIXES
The Tivoli Directory Server user registry components on IBM PureApplication System are affected. The solution is to upgrade the IBM PureApplication System and the Web Application pattern types to the following fix level:

IBM PureApplication System V2.1 
Upgrade to IBM PureApplication System V2.1.2
Upgrade to Web Application pattern type 2.0.5.0 and 1.0.5.0

IBM PureApplication System V2.0
Upgrade to IBM PureApplication System V2.0.0.1 Interim Fix 6
Upgrade to Web Application pattern type 2.0.2.5 and 1.0.2.5

The GPFS Pattern is affected and the solution is to apply one of the following interim fixes:
