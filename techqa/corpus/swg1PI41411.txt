 *  ****************************************************************
   * USERS AFFECTED:                                              *
   * IBM MobileFirst Platform developers using Java 8 to develop  *
   * MobileFirst applications                                     *
   ****************************************************************
   * PROBLEM DESCRIPTION:                                         *
   * Attemping to use Java 8 to develop MobileFirst applications  *
   * would result in a number of issues, including:               *
   *                                                              *
   * * The MobileFirst Platform Foundation test server will not   *
   * start correctly                                              *

   * installer will not run on OSX                                *
   * * MobileFirst Platform Foundation Command Line Interface     *
   * commands such as "build" and "deploy" may fail               *
   ****************************************************************
   * RECOMMENDATION:                                              *
   * -                                                            *
   ****************************************************************
   
   
    
   
   

PROBLEM CONCLUSION
 *  MobileFirst Platform Foundation Studio, Command Line Interface,
   and test server have been updated to support Java 8. ANT has
   also been upgraded to 1.9.4.
