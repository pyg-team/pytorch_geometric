APAR STATUS
 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  The property setting "Delete remote file after successful
   transfer" is missing for FileRead node.
   
   
    
   
   

LOCAL FIX
 *  NA
   
   
    
   
   

PROBLEM SUMMARY
 *  ****************************************************************
   USERS AFFECTED:
   All users of IBM Integration Toolkit v10 who use FileRead node
   in the message flow editor.
   
   
   Platforms affected:
   MultiPlatform
   
   ****************************************************************
   PROBLEM DESCRIPTION:
   The "Delete remote file after successful transfer" check box is
   missing from the properties view of the FileRead node.
   
   
    
   
   

PROBLEM CONCLUSION
 *  The property has been added to FileRead node.
