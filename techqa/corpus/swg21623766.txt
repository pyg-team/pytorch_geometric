Title: IBM Netcool/OMNIbus Integrations Release Notice - Gateway for JDBC (nco-g-jdbc-6_0) - United States

Text:
release notice; downloads; nco-g-jdbc; Netcool/OMNIbus Gateway for JDBC NEWS

ABSTRACT
 A new version of the Netcool/OMNIbus Gateway for JDBC will be available to download from March 12, 2015. 

CONTENT
 

This version of the gateway has been updated to add the following functionality:

 * Added 64-bit support on selected platforms. 

It also incorporates fixes for the following APARs:

 * IV43405: Modify the OwnerKey field in the MembersMap map. 
 * IV45735: JDBC gateway hangs on shutdown when it loses connection to the destination database and to the Object Server.
 * IV50510: When a batch fails it stalls the rest of the batches from processing. 
 * IV61643: JDBC gateway with MySQL JDBC driver, Memory usage grows without bound, resulting in OutOfMemoryException being thrown and gateway exit.

Netcool/OMNIbus V8.1

If you are running Netcool/OMNIbus V8.1, a single package contains versions of the gateway for all supported Operating Systems. To obtain this new package, use the Find by part number field to search for part number: CN4FUEN
