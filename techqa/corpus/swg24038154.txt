ABSTRACT
 Cognos Disclosure Management 10.2.3 FP1 provides important corrections and enhancements for Cognos Disclosure Management 10.2.3 products and should be applied to all installations. 

DOWNLOAD DESCRIPTION
This Fix Pack is an update of IBM Cognos Disclosure Management 10.2.3 for existing IBM Cognos Disclosure Management customers. New features have been added to this Fix Pack including


 *  COREP/FINREP


In addition, this Fix Pack includes fixes addressing defects. PREREQUISITES
This Fix Pack requires an existing installation of Cognos Disclosure Management 10.2.3.

INSTALLATION INSTRUCTIONS
In environments with UAC (both Client and Server), it is recommend that you run the installers with the 'Run as Administrator' option.
