    *  In these presentations, the name of the Image Services CMIS war file is mentioned as Image_Services_connector_for_ICN-1.10.war. The name has now been changed to Image_Services_connector_for_ICN.war. In these presentations, the ImageServicesRPC.jar file is mentioned. However, this jar file is no longer released. The ImageServicesPlugin.jar now includes theImageServicesRPC.jar functionality. The new stamp.bat (windows) and stamp.sh (Unix platforms) scripts are now available in the /fnsw/cmis directory. These scripts may be used to stamp the Image_Services_connector_for_ICN.war and ImageServicesPlugin.jar files.
