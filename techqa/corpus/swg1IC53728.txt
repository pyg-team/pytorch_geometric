ERROR DESCRIPTION
 *  Listener and channels stopped when the TCP/IP process was
   primaried to another CPU.
   Socket Error 4120 (ECONNRESET) is returned for OSS socket
   application
   when takeover occurs on conventional TCP/IP.
   The listener should not exit when receiving error 4120 and the
   channels
   should not end on changing primary CPU of conventional TCP/IP.
   
   
    
   
   

LOCAL FIX

PROBLEM SUMMARY
 *   A ECONNRESET error on the socket caused the
   listener and channels processes to exit when the TCPIP process
   primary was changed.
   
   
    
   
   

PROBLEM CONCLUSION
 *   Handle a ECONNRESET status returned for a
   socket.
   Change released in V5.3.1.1
