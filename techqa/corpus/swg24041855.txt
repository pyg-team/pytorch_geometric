DOWNLOAD DESCRIPTION
WebSphere Transformation Extender with Launcher Hypervisor Edition is a self-contained virtual machine image that contains a guest operating system and WebSphere Transformation Extender. Use it to create, deploy, and manage WebSphere Transformation Extender environments and applications by using a computer, known as a hypervisor, that hosts a virtual system.
