APAR STATUS
 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  With Oracle database, If a product has more than 1000 SKUs, when
   updating the allowed value of classic defining attribute for
   this product, the following exception will be thrown:
    ORA-01795: maximum number of expressions in a list is 1000.
   
   
    
   
   

LOCAL FIX

PROBLEM SUMMARY
 *  USERS AFFECTED:
   WebSphere Commerce Management Center users with V6.0 Feature
   Pack 4 or later or Version 7.0
   
   PROBLEM ABSTRACT:
   Management Center: ORA-01795 error ( maximum number of
   expressions in a list is 1000) occurred when updating a product
   with more than 1000 SKUs.
   
   BUSINESS IMPACT:
   User is not able to view product in Management Center when the
   issue happens.
   
   RECOMMENDATION:
   Applying APAR LI76354 will resolve the issue.
   
   
    
   
   

PROBLEM CONCLUSION
 *  SQL in the Data Service Layer was rewritten to avoid the error
   when updating the classic defining attribute allowed value for a
   product.
