QUESTION
 What commands can I use to verify the software versions installed on my IBM Smart Analytics System for Linux? 

ANSWER
You can use the following commands to see the software versions installed in your IBM Smart Analytics System environment for Linux. If you need the AIX commands, refer to the related links section of this page.
