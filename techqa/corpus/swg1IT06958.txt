APAR STATUS
 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  User was configuring the call home function on their
   PureApplication System. They attempted to create a test ticket
   but it failed with the following error:
   
   CWZIP3509E The Call Home request failed while submitting a
   request to create a new service ticket. ECC Exception error
   received: Invalid object : [Subject]. Reason:[Product
   [IBM/IBM/DP123TEST] component is missing]
   
   
    
   
   

LOCAL FIX
 *  The fix is to run 'psm reset Service9' on the leader
   PureSystems Manager.
   Use 'psm info all' to wait until ipas.trap is online again.
   
   
    
   
   

PROBLEM SUMMARY
 *  The Call Home function is not picking up system configuration
   changes after the user has made updates.
   
   
    
   
   

PROBLEM CONCLUSION
 *  A code fix is being provided in PureApplication System version
   2.1.0.0 so the Call Home configuration file information
   is read when a problem ticket is created.
