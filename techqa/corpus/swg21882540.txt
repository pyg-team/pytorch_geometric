Title: IBM Security Bulletin: Vulnerability in RC4 stream cipher affects IBM Tivoli Provisioning Manager (CVE-2015-2808) - United States

Text:
 SECURITY BULLETIN

SUMMARY
 The RC4 “Bar Mitzvah” Attack for SSL/TLS affects IBM Tivoli Provisioning Manager. 

VULNERABILITY DETAILS
CVEID: CVE-2015-2808 [http://cve.mitre.org/cgi-bin/cvename.cgi?name=CVE-2015-2808]
DESCRIPTION: The RC4 algorithm, as used in the TLS protocol and SSL protocol, could allow a remote attacker to obtain sensitive information. An attacker could exploit this vulnerability to remotely expose account credentials without requiring an active man-in-the-middle session. Successful exploitation could allow an attacker to retrieve credit card data or other sensitive information. This vulnerability is commonly referred to as "Bar Mitzvah Attack".
CVSS Base Score: 5
CVSS Temporal Score: See https://exchange.xforce.ibmcloud.com/vulnerabilities/101851 [https://exchange.xforce.ibmcloud.com/vulnerabilities/101851] for the current score
CVSS Environmental Score*: Undefined
CVSS Vector: (AV:N/AC:L/Au:N/C:P/I:N/A:N)

AFFECTED PRODUCTS AND VERSIONS
IBM Tivoli Provisioning Manager: 7.2 and earlier

REMEDIATION/FIXES
To remediate Tivoli Provisioning Manager server, please refer to How to disable RC4 stream cipher in IBM Tivoli Provisioning Manager server [http://www.ibm.com/support/docview.wss?uid=swg21882537]
To remediate Tivoli Common Agents, please apply the remediation fix indicated below:



Product VRMF APAR Remediation/First Fix Tivoli Provisioning Manager 7.1 7.1.1-TIV-TPM-IF00013 none 7.1.1.0-TIV-TPM-JRE50SR16FP10 [http://www.ibm.com/support/fixcentral/swg/selectFixes?parent=ibm~Tivoli&product=ibm/Tivoli/Tivoli+Provisioning+Manager&release=7.1.1.0&platform=All&function=fixId&fixids=7.1.1.0-TIV-TPM-*JRE50SR16FP10] Tivoli Provisioning Manager 7.2 7.2.2.0-TIV-TPM-IF00003 none 7.2.2.0-TIV-TPM-JRE60SR16FP3 [http://www.ibm.com/support/fixcentral/swg/selectFixes?parent=ibm~Tivoli&product=ibm/Tivoli/Tivoli+Provisioning+Manager&release=7.2.2.0&platform=All&function=fixId&fixids=7.2.2.0-TIV-TPM-*JRE60SR16FP3] 
For out of support versions, contact IBM customer support for assistance. 

To remediate IBM WebSphere Application Server, please consult the security bulletin Security Bulletin: Vulnerability in RC4 stream cipher affects WebSphere Application Server (CVE-2015-2808) [http://www.ibm.com/support/docview.wss?uid=swg21701503] for vulnerability details and information about fixes. 

You should verify applying this configuration change does not cause any compatibility issues. Not disabling the RC4 stream cipher will expose yourself to the attack described above. IBM recommends that you review your entire environment to identify other areas where you have enabled the RC4 stream cipher and take appropriate mitigation and remediation actions.
