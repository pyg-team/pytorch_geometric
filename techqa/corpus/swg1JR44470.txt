ERROR DESCRIPTION
 *  Even though the BPD named BPD with TP - Process Performance
   Errors (Issues Space) does not have any timing intervals
   defined, a Java exception shall not be thrown to the user.
   
   If the is no timing interval,
   BPM portal should not blow up and throw/return a Java exception
   to the
   browser. Instead, a message along the following lines should be
   shown
   to the user: "there are no timing intervals defined; hence
   there is no
   data to show on this page" .
