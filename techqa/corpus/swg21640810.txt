Title: IBM When installing IBM Business Process Manager (BPM) or IBM Integration Designer (IID), do not specify an installation path that includes any Unicode characters - United States

Text:
ibpma85relnotes; ibpms85relnotes; ibpme85relnotes; iid85relnotes; Installation Manager; install; fail; IM; ibpma8501relnotes; ibpms8501relnotes; ibpme8501relnotes; iid8501relnotes TECHNOTE (TROUBLESHOOTING)

PROBLEM(ABSTRACT)
 Installation Manager fails to install if you specify an installation path that contains any Unicode characters. 

SYMPTOM
If the installation path includes any Unicode characters, the following error occurs: 

Eclipse reconciler returned with error status=-1. 


CAUSE
This issue is caused by an Eclipse defect. 


RESOLVING THE PROBLEM
To resolve this issue, remove the Unicode characters from the installation directory path. For more information about this problem, see the Error: Eclipse reconciler returned with error status=-1 [http://www.ibm.com/support/docview.wss?uid=swg21589327] document. [http://www-01.ibm.com/support/docview.wss?uid=swg21589327]
