Title: IBM OMNIbus probe integration with IBM X3750 M4 server - United States

Text:
NcKL; X3750 M4; mttrapd; SNMPTRAP TECHNOTE (FAQ)

THIS DOCUMENT APPLIES ONLY TO THE FOLLOWING LANGUAGE VERSION(S):
 English 

QUESTION
 Is there any probe OMNIbus Probe Integration that would allow an IBM X3750 M4 server to send SNMP traps to an ObjectServer? 

ANSWER
Since the IBM X3750 M4 series server uses the integrated IMM module to monitor HW components, the Netcool Knowledge Library (NcKL) Enhanced Probe Rules V4.2 (CN09KEN), contains a set of IMM rules under directory 

../NcKL/rules/include-snmptrap/ibm/ibm-IMMALERT-MIB.include.snmptrap.rules 

covering the following events from the "ibm-IMMALERT-MIB" agent:

$OS_EventId = "SNMPTRAP-ibm-IMMALERT-MIB-ibmSpTrapTempC"
