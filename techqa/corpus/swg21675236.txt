Title: IBM An unexpected error (0000016e EditionHelper E) can occur when creating a deployment environment in IBM Business Process Manager (BPM) - United States

Text:
deployment environment; application directory; ibpma855relnotes; ibpms855relnotes; ibpme855relnotes TECHNOTE (TROUBLESHOOTING)

PROBLEM(ABSTRACT)
 When creating a deployment environment in IBM Business Process Manager (BPM), you might notice an application directory-related error in the SystemOut.log file. 

SYMPTOM
You see the following error message in the SystemOut.log file: 

[1/15/14 1:27:38:725 CST] 0000016e EditionHelper E Unexpected Error: com
msvc -- The Application's Directory in the Repository is EMPTY.


RESOLVING THE PROBLEM
You can safely ignore this error; it does not affect the creation of the deployment environment.
