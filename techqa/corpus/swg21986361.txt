Title: IBM Commands to manually uninstall BigFix Client on Non-Windows platform - United States

 TECHNOTE (FAQ)

QUESTION
 Commands to manually uninstall BigFix Client on non-Windows platform using the built in package manager. 

ANSWER
Commands to manually uninstall BigFix Client on Linux:
1. Stop the Agent process
/etc/init.d/besclient stop 

2. Run the following rpm command to find the current rpm package.
rpm -qa |grep -i BESA 

3. Uninstall IEM Client listed on item 2 
rpm -e BESAgent-XXX 

4. Remove directories left behind manually 
rm -rf /var/opt/BESClient 
rm -rf /opt/BESClient 
rm -rf /etc/opt/BESClient 
or
rm -rf /etc/opt/BESClient /var/opt/BESClient /opt/BESClient
