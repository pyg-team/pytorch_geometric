ABSTRACT
 This document describes how to download the IBM® InfoSphere® Optim™ Test Data Management Solution Starter Edition for JD Edwards Applications 11.3. 

DOWNLOAD DESCRIPTION
Using the IBM InfoSphere Optim Test Data Management Solution Starter Edition for JD Edwards Applications, you can optimize and automate the processes used to create and manage data in non-production environments such as testing, development, and training. Developers and quality assurance testers can create realistic, right sized JD Edwards test databases and compare before and after test results with speed and accuracy. The solution saves valuable processing time and provides consistency and reduced costs throughout the application life cycle.

PREREQUISITES
You must install the IBM InfoSphere Optim Solution for Test Data Management v11.3 before you install the solution for JD Edwards. 

For information about hardware and software requirements for the solution for JD Edwards, see the Detailed System Requirements at http://www.ibm.com/support/docview.wss?&uid=swg27041911 [http://www.ibm.com/support/docview.wss?&uid=swg27041911].

INSTALLATION INSTRUCTIONS
For general installation instructions, see the Quick Start Guide. For detailed instructions, see the Optim Test Data Management Solution for JD Edwards EnterpriseOne Installation and Configuration Guide. Both documents are included in the download package.
