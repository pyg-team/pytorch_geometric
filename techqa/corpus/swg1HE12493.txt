APAR STATUS
 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  1. Go to Map->Persons in the Policy Atlas application
   2. On this screen search for a user with active legal matters
   3. From here go to Details and then Current Activity.
   4. Select a Matter
   5. This will display the users who are involved with this matter
   6. Select one of these users and Show All
   7. This will show all of the Matters they are involved in. If
   any of those Matter names contain umlaute characters, they will
   be displayed incorrectly.
   Note: Only the above detailed area of the page shows the umlaut
   rendering problem, it does not exist elsewhere on the page.
