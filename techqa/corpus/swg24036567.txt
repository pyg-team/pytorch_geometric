ABSTRACT
 This document describes how to download IBM Integration Bus Retail Pack V1.0 eImages from the Passport Advantage website. 

DOWNLOAD DESCRIPTION
Terminology

IBM Integration Bus Retail Pack V1.0 is supplied in an eImage. The eImage contains the Quick Start Guide and the product code.

Related product documentation 


For information about tasks you must complete before you install the product, and about the installation process itself, see the IBM Integration Bus Retail Pack V1.0 Installation Guide that is supplied with the product, or see the IBM Integration Bus Retail Pack V1.0 Information Center [http://pic.dhe.ibm.com/infocenter/wmbhelp/v9r0m0/index.jsp].

The most up-to-date information can be found in the IBM Integration Bus Retail Pack V1.0 release notes at http://www.ibm.com/support/docview.wss?&uid=swg27006913 [http://www.ibm.com/support/docview.wss?rs=171&uid=swg27006913]. 

Downloading the relevant eImages from the Passport Advantage website 

 1. Find the IBM Integration Bus Retail Pack V1.0 images for your platform in the following eImages table.
 2. Go to the "Download package" section of this document, and follow the link to the Passport Advantage website. 
 3. Sign in to download the images that you need.
 4. Unpack each of the parts onto your system by following the instructions in the Installation Guide.
