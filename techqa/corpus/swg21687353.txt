Title: IBM When clicking on an existing task or trying to add a new task on a newly upgraded Case Manager 5.2.1 system via Case Manager Client I get CIWEB2001 error - United States

Text:
5.2.1; ICM; task; work item TECHNOTE (TROUBLESHOOTING)

PROBLEM(ABSTRACT)
 The Case Manager system has recenetly been upgraded to V5.2.1. When a user clicks on an existing task or tries to add a new task on a newly upgraded system via Case Manager Client they get CIWEB2001 error 

SYMPTOM
 Here is excerpt of the error message : 
"You might need to authenticate to a firewall, or the web application server might not be running or cannot be reached because of network problems. If you need to authenticate to a firewall, authenticate and try to log in again. If the problem persists, report this error to your system administrator. The following HTTP error was returned: 599 Error 599: filenet/vw/api/VWParameter.getAttributeInfo&#40;&#41;Lfilenet/vw/api/VWAttributeInfo&#59; CIWEB2001"


CAUSE
The filenet/vw/api/VWParameter.getAttributeInfo method is missing from the pe.jar file of Content Platform Engine (CPE) CPE 5.2.0.3 Fix Pack. This issue is addressed with CPE 5.2.0.3 IF3, which is required by IBM Case Manager (ICM) V5.2.1. See the reference link below.
http://www-969.ibm.com/software/reports/compatibility/clarity-reports/report/html/prereqsForProduct?deliverableId=1391625400511# [http://www-969.ibm.com/software/reports/compatibility/clarity-reports/report/html/prereqsForProduct?deliverableId=1391625400511#] 

ENVIRONMENT
Windows Server 2012 R2 with ICN 2.0.3, CPE 5.2.0.3, and ICM 5.2.1

RESOLVING THE PROBLEM
Download 5.2.0.3-P8CPE-WIN-IF003 and 5.2.0.3-P8CPE-CLIENT-WIN-IF003 from www.ibm.com/support/fixcentral [www.ibm.com/support/fixcentral] and apply them to CPE, ICM, ICN, WorkplaceXT and customized application servers respectively. 


Refer to IBM FileNet P8 Content Platform Engine 5.2.0.3-P8CPE-IF003 interim fix readme for installing and configuring the server updates as well as installing Content Platform Engine (CPE) client updates. See the reference link below.
http://delivery04.dhe.ibm.com/sar/CMA/IMA/04r2g/2/readme_5203_P8CPE_IF003.htm [http://delivery04.dhe.ibm.com/sar/CMA/IMA/04r2g/2/readme_5203_P8CPE_IF003.htm]
