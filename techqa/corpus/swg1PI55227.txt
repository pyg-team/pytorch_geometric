APAR STATUS
 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  If a conent link is created on syndicator and syndicatd to
   subscriber, there is a JMS message generated for the content
   link on syndicator while there is no message for the content
   link on subscriber.
   
   
   Expected: a JMS message will be generated on subscriber when the
   content link is syndicated
   
   
   Analysis:
   
   Solution: call messaging service to send out JMS message after
   syndication finishes
