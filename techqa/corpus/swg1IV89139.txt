Title: IBM IV89139: ISAM 9.0.1.0 DOES SUPPORT ORACLE 12C (12.1.0.1.X) FOR EXTERNAL RUNTIME AND CONFIGURATION DATABASES - United States

APAR STATUS
 * CLOSED AS DOCUMENTATION ERROR.
    
   
   

ERROR DESCRIPTION
 *  The IBM Clarity Tool for IBM Security Access Manager for
   Advanced Access Control (ISAM AAC) specifies that only Oracle
   Database 11g Enterprise Edition R2 (11.2.0.1.X) is supported in
   the matrix.
   
   ISAM for Mobile 8.0.1.X also lists Oracle Database 12c Release
   1 (12.1.0.1.X) is supported for external Runtime and
   Configuration Databases
   
   Development has confirmed that Oracle Database 12c Release 1
   (12.1.0.1.X) is also supported at 9.0.0.X and 9.0.1.X as an
   available database for the External Runtime and Configuration
   Database
   
   
    
   
   

LOCAL FIX
 *  Refer to this APAR as documentation.

   rectify the issue reported in by this APAR -
