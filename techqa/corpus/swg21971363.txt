Title: IBM HTTP access does not work in DASH 3.1.0.3 (JazzSM 1.1.0.3) - United States

Text:
DASHL2; HTTP access; web.xml; transport; confidential; 3.1.0.3; dashboard application services hub; jazzsm TECHNOTE (TROUBLESHOOTING)

PROBLEM(ABSTRACT)
 The configuration of HTTP access does not work in Jazz for Service Management 1.1.0.3 (DASH 3.1.0.3). After configuring HTTP access, HTTP access attempts are redirected to HTTPS 

CAUSE
An internal defect was raised for DASH 3.1.0.3

RESOLVING THE PROBLEM
This is a known issue for DASH 3.1.0.3 and is addressed in DASH 3.1.1 and newer. To address the issue, upgrade to DASH 3.1.1 or newer.

IBM Support, see internal note.
