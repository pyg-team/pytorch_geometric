APAR STATUS
 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  The following text are not translated correctly in Hebrew locale
   [he] when using Case Builder.
   
   Translation Issue #1:
   The 'solution' text.  This is seen in the confirmation dialog
   when a user clicks on 'deploy' on a solution if there are
   uncommitted changes on the solution.
   
   Translation Issue #2:
   The 'DateLastModified' and 'LastModifier' text.  This is seen
   when navigating to Case Type-> case type name -> views -> case
   summary where it shows the list of system properties.
   
   
    
   
   

LOCAL FIX

PROBLEM SUMMARY
 *  ****************************************************************
   * USERS AFFECTED:                                              *
   * Case Builder users selected to use the Hebrew (he) locale.   *
   ****************************************************************
   * PROBLEM DESCRIPTION:                                         *
   * See Error Description                                        *
   ****************************************************************
   * RECOMMENDATION:                                              *
   * Update to 5.2.1.4-ICM-FP004 or later.                        *
   ****************************************************************
   
   
    
   
   

PROBLEM CONCLUSION
 *  Fixed in 5.2.1.4-ICM-FP004.  With this fix, the text for
   Solution, DateLastModified, and LastModifier in Case Builder are
   translated correctly in the Hebrew locale [he] locale.
