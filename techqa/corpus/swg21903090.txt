Title: IBM Internet Explorer 11 error when opening InfoSphere Information Server 11.3 Web Console - United States

Text:
 TECHNOTE (TROUBLESHOOTING)

PROBLEM(ABSTRACT)
 If you open the InfoSphere Information Server Web Console with Internet Explorer 11, you may get the error message: IBM InfoSphere Information Server supports Mozilla Firefox (ESR 17 and 24) and Microsoft Internet Explorer (version 9.0 and 10.0) browsers. 

CAUSE
Internet Explorer 11 support was added after 11.3.0 was released and the browser check was not updated.

ENVIRONMENT
Internet Explorer 11 not running in compatibility mode.

DIAGNOSING THE PROBLEM
Opening the URL https://services.hostname.com:port/ibm/iis/console results in the error message "IBM InfoSphere Information Server supports Mozilla Firefox (ESR 17 and 24) and Microsoft Internet Explorer (version 9.0 and 10.0) browsers."

RESOLVING THE PROBLEM
You can set Internet Explorer 11 to run in document mode 9 or 10:
