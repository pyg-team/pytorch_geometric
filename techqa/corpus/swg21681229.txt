Title: IBM Security Bulletin: Fix available for security vulnerabilities related to IEHS in IBM Sametime Meetings and Proxy version 9 (multiple CVEs) - United States

Text:
specially-crafter; psirt SECURITY BULLETIN

SUMMARY
 Fix is available for security vulnerabilities related to IBM Eclipse Help System (IEHS) in IBM Sametime Meetings and Proxy version 9 (multiple CVEs) 

VULNERABILITY DETAILS
 

CVE ID: CVE-2014-0918 [http://cve.mitre.org/cgi-bin/cvename.cgi?name=CVE-2014-0918]

DESCRIPTION: 
IBM Eclipse Help System could allow a remote attacker to traverse directories on the system. An attacker could send a specially crafted URL request to view arbitrary files on the system.

CVSS:
CVSS Base Score: 7.1
CVSS Temporal Score: See https://exchange.xforce.ibmcloud.com/vulnerabilities/91980 [https://exchange.xforce.ibmcloud.com/vulnerabilities/91980] for the current score
CVSS Environmental Score*: Undefined
CVSS Vector: (AV:N/AC:M/Au:N/C:C/I:N/A:N)
===================================================== CVE ID: CVE-2014-0917 [http://cve.mitre.org/cgi-bin/cvename.cgi?name=CVE-2014-0917]

DESCRIPTION:
IBM Eclipse Help System is vulnerable to cross-site scripting, caused by improper validation of user-supplied input. A remote attacker could exploit this vulnerability using a specially crafted URL to execute script in a victim's Web browser within the security context of the hosting Web site, once the URL is clicked. An attacker could use this vulnerability to steal the victim's cookie-based authentication credentials.

CVSS:
CVSS Base Score: 4.3
CVSS Temporal Score: See https://exchange.xforce.ibmcloud.com/vulnerabilities/91979 [https://exchange.xforce.ibmcloud.com/vulnerabilities/91979] for the current score
CVSS Environmental Score*: Undefined
CVSS Vector: (AV:N/AC:M/Au:N/C:N/I:P/A:N) ===================================================== 
CVE ID: CVE-2013-5449 [http://cve.mitre.org/cgi-bin/cvename.cgi?name=CVE-2013-5449]

DESCRIPTION:
IBM Eclipse Help System (IEHS), as used in multiple IBM products, is vulnerable to cross-site scripting. A remote attacker could exploit this vulnerability using a specially crafted URL to execute script in a victim's Web browser within the security context of the hosting Web site, once the URL is clicked. An attacker could use this vulnerability to steal the victim's cookie-based authentication credentials.

CVSS:
