Title: IBM Security Bulletin: A security vulnerability has been identified in IBM WebSphere MQ shipped with IBM WebSphere Remote Server (CVE-2014-3566) - United States

Text:
 SECURITY BULLETIN

SUMMARY
 IBM WebSphere MQ is shipped as a component of IBM WebSphere Remote Server. Information about a security vulnerability affecting IBM WebSphere MQ has been published in a security bulletin. 

VULNERABILITY DETAILS
Please consult the security bulletin Vulnerability in SSLv3 affects IBM WebSphere MQ, IBM WebSphere MQ Internet Pass-Thru and IBM Mobile Messaging and M2M Client Pack (CVE-2014-3566) [http://www-01.ibm.com/support/docview.wss?uid=swg21687433] for vulnerability details and information about fixes.

AFFECTED PRODUCTS AND VERSIONS
 

Principal Product and Version(s) Affected Supporting Product and Version All releases of IBM WebSphere Remote Server. All releases of IBM WebSphere MQ GET NOTIFIED ABOUT FUTURE SECURITY BULLETINS
