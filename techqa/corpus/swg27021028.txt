Title: IBM Alcatel-Lucent 5620 SAM Technology Pack v2.8.0.0  User's Guide update for release 1.3A - United States

Text:
 PRODUCT DOCUMENTATION

ABSTRACT
 User Guide for the Technology Pack Alcatel-Lucent 5620 SAM v2.8.0.0 from release 1.3A contains a section describing the supported versions of SAM Server.This section incorrectly states that this version of the Alcatel-Lucent 5620 SAM Technology Pack v2.8.0.0 supports v6, v6.1, and v7 of SAM. This release of the Technology Pack also adds support for SAM v8 

CONTENT
In the Alcatel-Lucent 5620 SAM 2.8.0.0 Technology Pack User's Guide, 

 Chapter 2: Devices and services [http://publib.boulder.ibm.com/infocenter/tivihelp/v8r1/topic/com.ibm.tnpm_tp.doc/13A/HTML/ALU5620SAM_2800/NetcoolProviso_AlcatelLucent5620SAM_2.8.0.0_Tech_Pack_User_Guide.htm#50552880_pgfId-1099817], Under section Alcatel-Lucent 5620 SAM server compatibility [http://publib.boulder.ibm.com/infocenter/tivihelp/v8r1/topic/com.ibm.tnpm_tp.doc/13A/HTML/ALU5620SAM_2800/NetcoolProviso_AlcatelLucent5620SAM_2.8.0.0_Tech_Pack_User_Guide.htm#50552880_pgfId-1114930] on Page 5.

It currently reads like this:

Alcatel-Lucent 5620 SAM server compatibility 
[/support/docview.wss?uid=swg27021028&aid=1] [/support/docview.wss?uid=swg27021028&aid=1]
The Alcatel-Lucent 5620 SAM Technology Pack currently runs on the following Alcatel-Lucent 5620 SAM servers: 

 * Alcatel-Lucent 5620 SAM Server Release 6 
 * Alcatel-Lucent 5620 SAM Server Release 6.1 
 * Alcatel-Lucent 5620 SAM Server Release 7



This section should read: 

Alcatel-Lucent 5620 SAM server compatibility 
[/support/docview.wss?uid=swg27021028&aid=2] [/support/docview.wss?uid=swg27021028&aid=2] 
The Alcatel-Lucent 5620 SAM Technology Pack currently runs on the following Alcatel-Lucent 5620 SAM servers:  * Alcatel-Lucent 5620 SAM Server Release 7 
 * Alcatel-Lucent 5620 SAM Server Release 8


Note: Support for Release 6 and 6.1 was non-functional in v2.8.0.0 but has been restored in release 2.9.1.0 of the Alcatel-Lucent 5620 SAM Technology Pack.
