APAR STATUS
 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  While using the Web Application Bridge in a clustered
   environment, if you modify the VWA Configuration through the
   VWA Manager portlet on one node, then those changes may not be
   reflected on other nodes without restarting those other nodes.
   Opened for release 8001
   
   
    
   
   

LOCAL FIX
 *  Restart all the nodes of the cluster after modifying any VWA
