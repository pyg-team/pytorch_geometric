Title: IBM HE12471: IBM Content Collector cannot mark email with attachments for archiving when the Domino mail file exceeds the mail file quota - United States

IBM Content Collector V4.0.1 Fix Pack 2 [http://www-01.ibm.com/support/docview.wss?uid=swg24041245]

APAR STATUS
 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  If a Domino mail file exceeds the mail file quota, email
   documents with attachments cannot be marked for archivingin the
   Notes client.
   The following error message is shown: "The document cannot be
   marked for this operation at this moment. Ensure that you have
   sufficient rights to edit it".
   
   
    
   
   

LOCAL FIX
 *  N/A
   
   
    
   
   

PROBLEM SUMMARY
 *  If a Domino mail file exceeds the mail file quota, email
   documents with attachments cannot be marked for archivingin the
   Notes client.The following error message is shown: "The document
    cannot be marked for this operation at this moment. Ensure that
    you have sufficient rights to edit it".
   
   
    
   
   

PROBLEM CONCLUSION
 *  Starting with this fix, IBM Content Collector can  mark email
   with attachments for archiving when the Domino mail file exceeds
    the mail file quota
   
   
    
   
   

TEMPORARY FIX
 *  Fixed in 4.0.1.2-IBM-ICC-FP002
