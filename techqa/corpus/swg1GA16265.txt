APAR STATUS
 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  With Japanese language setting, error message "Please contact
   your system administrator." in Japanese when moved to "Support
   Information Gathering Results" page after having must_gather
   system and sniffer.
   
   
    
   
   

LOCAL FIX

PROBLEM SUMMARY
 *  The problem is in permission of snif/tap directory output of
   sniffer must gather and these permissions prevent tomcat from
   reading files under it.
   
   
    
   
   

PROBLEM CONCLUSION
 *  The solution is to give proper permission for the snif directory
    in must gather sniffer_issues and miss_dbuser_prog_issues.
   
   Guardium v10.0 GPU p400 (2017-December-21) and future releases
   (GPU and patches) provide code fix for this issue.
   
   For more details check the v10.1.4 release notes:
   http://www-01.ibm.com/support/docview.wss?uid=swg21997914&aid=1 [http://www-01.ibm.com/support/docview.wss?uid=swg21997914&aid=1]
   
   
    
   
   

TEMPORARY FIX
