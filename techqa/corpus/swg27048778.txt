Title: IBM Removal notices for WebSphere Application Server Liberty 16.0.0.3 - United States

Text:
 RELEASE NOTES

ABSTRACT
 Removal notices for the following items were issued in Fix Pack 16.0.0.3 of WebSphere Application Server Liberty. A removal notice signifies that support for the item can be removed in the future, starting with the indicated fix pack. 

CONTENT
Typically, items are removed no sooner than two years after the removal notice is issued. However, in some instances, an item is removed earlier, if end of support for the item is announced, for example. For more information about Liberty removal notices, see New and changed features for Liberty [http://www.ibm.com/support/knowledgecenter/SSEQTP_liberty/com.ibm.websphere.wlp.nd.multiplatform.doc/ae/rwlp_newchange.html].




If you currently use an item that will be removed, plan to update to a supported product or version as listed in the WebSphere Application Server detailed system requirements [http://www-01.ibm.com/support/docview.wss?uid=swg27006921].
