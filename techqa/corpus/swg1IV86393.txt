ERROR DESCRIPTION
 *  When we configures webseal's inactive timeout longer than WAS's
   one(ISIM Console), when session timeouts SSO logout page does
   not function properly.
   
   Steps to Duplicate:
   1. Configure ISIM with SSO.
   2. Configure inactive-timout of WebSEAL 10 min.
   3. Configure ISIM Console timeout to 2 min.
   4. Access to /isim/console via WebSEAL.
   5. Wait for 2 minutes (over default WAS inactive timeout) and
   Click any link on isim console window.
   6. Try to click on login button on sso logout page.
   
   Desired behavior:
   ISIM should function properly when Login button is clicked on
   SSO logout page.
   
   
    
   
   

LOCAL FIX
 *  1. set enrole.ui.timeoutURL=/itim/console/main in
   ISIM_HOME/data/ui.properties file
   2. Restart application
