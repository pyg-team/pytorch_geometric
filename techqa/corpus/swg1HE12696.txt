APAR STATUS
 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  When creating a Release Notice, the list of Pre-defined
   Variables contains the ability to add confirmation questions to
   the notice.  Confirmation questions are not part of the Release
   Notice functionality, and should be unavailable.
   
   
    
   
   

LOCAL FIX
 *  No ability for customer to add Confirmation Question to the
   Release Notice.  Suggested alternate methodology where multiple
   Hold Notices are created before being released.
   
   
    
   
   

PROBLEM SUMMARY
 *  When creating a Release Notice, the list of Pre-defined
   Variables contains the ability to add confirmation questions to
   the notice.  Confirmation questions are not part of the Release
   Notice functionality, and should be unavailable.
   
   
    
   
   

PROBLEM CONCLUSION
 *  The fix is available now on Fix Central, as part of Atlas 6.0.3
   Fix Pack 6
