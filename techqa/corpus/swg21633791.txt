Title: IBM REST API calls fail intermittently with a SRVE0293E NullPointerException in IBM Business Process Manager (BPM) - United States

Text:
REST; API; SRVE0293E TECHNOTE (TROUBLESHOOTING)

PROBLEM(ABSTRACT)
 You see that some calls through the REST API intermittently fail with a SRVE0068E NullPointerException. 

SYMPTOM
A small percentage of calls through the REST API fail. This issue was introduced by performance APAR JR44404 [http://www.ibm.com/support/docview.wss?uid=swg1JR44404]. With APAR JR44404 installed, the SystemOut.log file contains exceptions that start with the following messages:

RESOLVING THE PROBLEM
To resolve this issue, install APAR JR45955 [http://www.ibm.com/support/docview.wss?uid=swg1JR45955] and APAR JR45811 [http://www.ibm.com/support/docview.wss?uid=swg1JR45811]. The following table contains links to the fixes in Fix Central.
