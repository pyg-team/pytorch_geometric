 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  A custom build JAVA application(on WAS) which contacts PD for
   authentication and also for administration. Its uses a TAM JAVA
   certificate while contacting PD.
   
   Recently the password life of the certificate expired and it
   was not able to automatically refresh after restart of
   application.
   
   Steps to re-create :
   
   1. Change the ssl-pwd-life to 2 days and ssl-cert-life to 4 days
   under /opt/PolicyDirector/etc/ivmgrd.conf
   
   2. Restart PD using pd_start restart
   
   3. Copy the PD.jar provided by IBM to fix certificate refresh
   issue.
   
   4. Run the following command to recreate the certificate.
     java com.tivoli.pd.jcfg.SvrSslCfg -action replcert -admin_id
   sec_master -admin_pwd ****** -cfg_file
   /usr/westpac/properties/wbc_adminsvc.conf
   
   5. Restart WAS including DMG.
