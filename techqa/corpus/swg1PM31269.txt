LOCAL FIX

PROBLEM SUMMARY
 *  When WCM administrators log on to WCM Authoring Portlet using
   Internet Explorer, user see that the WCM Authoring Portlet is
   unavailable and get a Null Pointer Exception in the logs.
   
   
    
   
   

PROBLEM CONCLUSION
 *  For Internet Explorer user, WCL code throws NPE when it checks
   for Netscape v4 browser. Code fixed to catch the NPE and
   continue to allow
   WCM Authoring Portlet to initialize without error.
