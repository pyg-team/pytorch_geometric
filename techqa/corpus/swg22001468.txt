Title: IBM Security Bulletin: IBM MQ and IBM WebSphere MQ Trace enablement could cause denial of service (CVE-2017-1117) - United States

Text:
 SECURITY BULLETIN

SUMMARY
 IBM MQ could allow an authenticated user to cause a denial of service to the MQXR channel when trace is enabled. 

VULNERABILITY DETAILS
CVEID: CVE-2017-1117 [http://cve.mitre.org/cgi-bin/cvename.cgi?name=CVE-2017-1117]
DESCRIPTION: IBM WebSphere MQ could allow an authenticated user to cause a denial of service to the MQXR channel when trace is enabled.
CVSS Base Score: 3.1
CVSS Temporal Score: See https://exchange.xforce.ibmcloud.com/vulnerabilities/121155 [https://exchange.xforce.ibmcloud.com/vulnerabilities/121155] for the current score
CVSS Environmental Score*: Undefined
CVSS Vector: (CVSS:3.0/AV:N/AC:H/PR:L/UI:N/S:U/C:N/I:N/A:L) 

AFFECTED PRODUCTS AND VERSIONS
IBM WebSphere MQ V7.5 

IBM WebSphere MQ 7.5.0.0 - 7.5.0.7 maintenance levels
IBM MQ V8.0 
IBM MQ 8.0.0.0 - 8.0.0.5 maintenance levels
IBM MQ V9 CD
IBM MQ 9.0.1 only 

IBM MQ V9 LTS 

IBM MQ 9.0.0.0 only

REMEDIATION/FIXES
IBM WebSphere MQ V7.5 

Apply 7.5.0.8 [http://www-01.ibm.com/support/docview.wss?uid=swg22005413] maintenance level 

IBM MQ V8.0 
Apply 8.0.0.6 [http://www-01.ibm.com/support/docview.wss?uid=swg21995100] maintenance level 
IBM MQ V9 CD
Upgrade to 9.0.2 [https://www-01.ibm.com/support/docview.wss?uid=swg24043463] 

IBM MQ V9 LTS 

Apply Fix Pack 9.0.0.1 [http://www-01.ibm.com/support/docview.wss?uid=swg27048059]
