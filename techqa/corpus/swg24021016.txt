INSTALLATION INSTRUCTIONS
You can use version 1.2.1 of Installation Manager either as an update to an existing installation or for a first-time installation. The instructions for each method of installation are outlined here.

If Installation Manager is currently installed and you are using this release as a fix pack, complete the following steps:


 1. Download the fix pack (com.ibm.cic.agent.offering.zip) from the download URL below. Note: Do not download a platform specific zip file like agent.installer.win32.win32.x86.zip. 
 2. Extract the compressed file in an appropriate directory. For example, choose to extract to C:\temp. 
 3. Add the fix pack repository location in IBM Installation Manager:  1. Start IBM Installation Manager. 
     2. On the Start page of Installation Manager, click File > Preferences, and then click Repositories. 
     3. The Repositories page opens. 
     4. On the Repositories page, click Add Repository.
     5. In the Add repository window, browse to or enter the file path to the repository.config file, which is located in the directory where you extracted the compressed files. For example, enter C:\temp\ or C:\temp\repository.config. Make sure Search service repositories during installation and updates is checked and then click OK.
    
    
 4. Installation of the fix pack will begin once you click on Install or Update.
    


If IBM Installation Manager is not currently installed, you can install this version by completing the following steps: 

1. Download the installer from the download URL in the table below: 

Filename OS agent.installer.win32.win32.x86.zip Windows® agent.installer.linux.gtk.x86.zip, agent.installer.linux.gtk.s390.zip Linux® agent.installer.solaris.gtk.sparc.zip Solaris agent.installer.aix.motif.ppc.zip AIX 

2. Extract the compressed file into an appropriate directory. 
3. Run install.exe (Windows), or install (Linux/Solaris/AIX). 
4. The Installation Manager installer will be launched. Follow the wizard pages to install it. 
5. IBM Installation Manager will start at completion.
