Title: IBM exec(): 0509-036 Cannot load program rsserver because of the following errors: - United States

Text:
 TECHNOTE (TROUBLESHOOTING)

PROBLEM(ABSTRACT)
 A user has just upgraded their AIX computer to version 5.1 and reinstalled PowerPlay Transformer. When trying to run rsserver, errors occur. 

Error Message - 

exec(): 0509-036 Cannot load program rsserver because of the following errors:
0509-150 Dependent module liblang.so could not be loaded.
0509-022 Cannot load module liblang.so.
0509-026 System error: A file or directory in the path name does not exist 

RESOLVING THE PROBLEM
The user had forgotten to set the COGNOS_HOME variable. Doing so will resolve the issue.
