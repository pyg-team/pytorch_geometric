ERROR DESCRIPTION
 *  When restoring a legacy document using the ICC legacy restore
   component the webapplication will crash and cannot restore the
   document to the target database
   
   
    
   
   

LOCAL FIX
 *  n/a
   
   
    
   
   

PROBLEM SUMMARY
 *  When the IBM Content Collector legacy restore operations were
   used to restore documents that were archived with CommonStore
   for Lotus Domino, the Web Application crashed and did not
   restore the legacy document correctly
   
   
    
   
   

PROBLEM CONCLUSION
 *  Starting with this fix, documents archived with CommonStore for
   Lotus Domino can be restored successfully in IBM Content
   Collector, using the legacy restore operations.
