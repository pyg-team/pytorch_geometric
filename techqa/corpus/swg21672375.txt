Title: IBM Event from posteifmsg to EIF Probe has blank summary - United States

Text:
empty; null; itm; missing; description TECHNOTE (TROUBLESHOOTING)

PROBLEM(ABSTRACT)
 All events sent from posteifmsg to EIF Probe are seen with blank summary. 

CAUSE
The tivoli_eif_tec.rules is not included in the tivoli_eif.rules and tec_db_update.sql has not been run on the target ObjectServer.

RESOLVING THE PROBLEM
In the tivoli_eif.rules, ensure that the following line is not commented out with #: 

include "tivoli_eif_tec.rules" 

On the ObjectServer, run tec_db_update.sql to add the fields for TEC Events.
