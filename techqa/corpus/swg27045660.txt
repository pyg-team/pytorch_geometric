Title: IBM System Requirements for WebSphere Application Server v7.0 - United States

Text:
 PRODUCT DOCUMENTATION

ABSTRACT
 The table below provides the system requirements for all editions of IBM WebSphere Application Server v7.0 in different levels and organization of detail.

This document describes the minimum product levels that need to be installed before opening a problem report with the WebSphere Application Server Support Team. Because other products frequently ship fixes, updates, and new releases, testing every configuration is not possible. In general, you can install and run with updates to supported products if those updates are forward compatible. 

CONTENT
 

To get the most updated software product compatibility report, use the Software Product Compatibility Reports (SPCR) tool [http://pic.dhe.ibm.com/infocenter/prodguid/v1r0/clarity/index.html]. 

For your convenience, the sections offer detailed system requirement reports for different contexts (by Operating System, by component). Note that clicking a link will always generate a new, up-to-date report.
