 TECHNOTE (FAQ)

QUESTION
 What is the recommended License Manager version for the use with IBM SPSS Statistics 24, IBM SPSS Modeler 18, & IBM SPSS Amos 24? 

ANSWER

[Recommended License Manager version]

For the usage with IBM SPSS Statistics 24, IBM SPSS Modeler 18, & IBM SPSS Amos 24, it is recommended to install the License Manager version 8.6 (IBM SPSS Concurrent Licensing Tools 9.6).
With this version you can manage network licenses (concurrent user licenses) for older IBM SPSS software releases as well.

[License Manager download]
On the IBM Software & Download Portal, there are several versions for the License Manager available depending on what operating system (OS) you would like to install the service.

If you look for the License Manager for Microsoft Windows operating system search for part number CNC5JEN and download below package:

IBM SPSS Concurrent Licensing Tools 9.6.1 - License Manager Microsoft Windows English (CNC5JEN ) 

If you would like to download a License Manager on a non-Windows operating system like AIX, Linux or Mac search for the part numbers below:

IBM SPSS Concurrent Licensing Tools 9.6.1 - License Manager 64-bit AIX English (CNC5HEN ) 
IBM SPSS Concurrent Licensing Tools 9.6.1 - License Manager Linux x86-64 English (CNC5IEN ) 
IBM SPSS Concurrent Licensing Tools 9.6.1 - License Manager Mac English (CNC5KEN ) 

[License Manager installation]


Please also see a short video for the installation on:

How do I install the IBM SPSS Concurrent Licensing Tools 9.6? - Video
http://www-01.ibm.com/support/docview.wss?uid=swg21981120 [http://www-01.ibm.com/support/docview.wss?uid=swg21981120]


- Extract the content of the zip file to a local hard drive on the Windows machine where you like to install the License manager. You will get 2 folders after extraction:
manager and tools folder

- Open the manager folder first. It will contain the files below:

Setup.exe
Sentinel RMS License Manager 8.6.1.msi
Data1.cab

- For the interactive installation of Sentinel RMS License Manager Service right click on setup.exe and run as administrator. Follow the instructions on the screen or watch above video first. 

It is recommended to use the default installation path:
C:\Program Files (x86)\Common Files\SafeNet Sentinel\Sentinel RMS License Manager\

- After you have installed the License Manager Service open the tools folder
It contain a setup.exe file. right click on setup.exe and run as administrator.

- After installation of the Sentinel RMS Server Administration tools (WlmAdmin version 8.6.2.3001) you need to license your SPSS product.

See chapter: 'Licensing your product' on Concurrent License Administrator.pdf for your product on the links below for detailed instructions.
