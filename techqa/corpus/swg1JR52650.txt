ERROR DESCRIPTION
 *  You encounter the following exception when trying to use the
   calculation framework to calculate VAT tax for items that are
   linked to more than 1000 products:
   
   java.sql.SQLSyntaxErrorException: ORA-01795: maximum number of
   expressions in a list is 1000
   
   
    
   
   

LOCAL FIX

PROBLEM SUMMARY
 *  USERS AFFECTED:
   WebSphere Commerce V7 users who have items that are linked more
   than 1000 times.
   
   PROBLEM ABSTRACT:
   ORA-01795 exception in calculation framework for items that are
   linked more than 1000 times
   
   BUSINESS IMPACT:
   Calculation framework fails to calculate shipping or tax
   amounts.
   
   RECOMMENDATION:
   
   
    
   
   

PROBLEM CONCLUSION
 *  Split the SQL query into sub queries if the number of
   expressions in IN clause is bigger than 1000.
