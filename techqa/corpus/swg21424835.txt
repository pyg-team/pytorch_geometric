Title: IBM What does the ASN0589I message from the Capture program mean? - United States

Text:
SQLCapture; readIgnTransCriteria TECHNOTE (TROUBLESHOOTING)

PROBLEM(ABSTRACT)
 The following message is returned:

ASN0589I "Capture" : "AS8" : "WorkerThread" The program received an unexpected return code "903" from routine "readIgnTransCriteria". 

SYMPTOM
Capture fails to start after you upgraded to Version 9.7

DIAGNOSING THE PROBLEM
Check DBADM authority for the Capture user

RESOLVING THE PROBLEM
The user ID that starts the Capture program must have SELECT authority on the IBMQREP_IGNTRAN table.
