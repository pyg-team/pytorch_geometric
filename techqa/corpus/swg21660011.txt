Title: IBM Security Bulletin: Fixes available for Security Vulnerabilities in IBM WebSphere Portal - United States

Text:
 SECURITY BULLETIN

SUMMARY
 Fixes are available for multiple security vulnerabilities in IBM WebSphere Portal. 

VULNERABILITY DETAILS
CVEID: CVE-2013-6328 [http://cve.mitre.org/cgi-bin/cvename.cgi?name=CVE-2013-6328]
DESCRIPTION: 
A Cross Site Scripting (XSS) and IFrame Injection vulnerability in the IBM Web Content Manager (WCM) UI has been identified.

CVSS:
CVSS Base Score: 4.3
CVSS Temporal Score: See https://exchange.xforce.ibmcloud.com/vulnerabilities/88909 [https://exchange.xforce.ibmcloud.com/vulnerabilities/88909] for the current score
CVSS Environmental Score*: Undefined
CVSS Vector: (AV:N/AC:M/Au:N/C:N/I:P/A:N)


AFFECTED PRODUCTS AND VERSIONS: 
WebSphere Portal 8
WebSphere Portal 7
WebSphere Portal 6.1.5
WebSphere Portal 6.1.0

REMEDIATION: 
The recommended solution is to apply PM96345 [http://www.ibm.com/eserver/support/fixes/fixcentral/swg/quickorder?productid=WebSphere%20Portal&brandid=5&apar=PM96345] as soon as practical.
Fix: Apply an Interim Fix or a Cumulative Fix containing PM96345 [http://www.ibm.com/eserver/support/fixes/fixcentral/swg/quickorder?productid=WebSphere%20Portal&brandid=5&apar=PM96345] .

For 8.0.0 through 8.0.0.1
▪ Upgrade to Fix Pack 8.0.0.1 with Cumulative Fix 9 (CF09)
(Combined Cumulative Fixes for WebSphere Portal 8.0.0.1: http://www-01.ibm.com/support/docview.wss?uid=swg24034497 [http://www-01.ibm.com/support/docview.wss?uid=swg24034497])

For 7.0.0 through 7.0.0.2
▪ Upgrade to Fix Pack 7.0.0.2 with Cumulative Fix 26 (CF26) and then apply Interim Fix PM96345 [http://www.ibm.com/eserver/support/fixes/fixcentral/swg/quickorder?productid=WebSphere%20Portal&brandid=5&apar=PM96345]
(Combined Cumulative fixes for WebSphere Portal 7.0.0.2: http://www.ibm.com/support/docview.wss?uid=swg24029452 [http://www.ibm.com/support/docview.wss?uid=swg24029452])

For 6.1.5.0 through 6.1.5.3
▪ Upgrade to Fix Pack 6.1.5.3 with Cumulative Fix 27 (CF27) and then apply Interim Fix PM96345 [http://www.ibm.com/eserver/support/fixes/fixcentral/swg/quickorder?productid=WebSphere%20Portal&brandid=5&apar=PM96345]
(Cumulative fixes for WebSphere Portal 6.1.5.3: http://www-01.ibm.com/support/docview.wss?uid=swg24023835 [http://www-01.ibm.com/support/docview.wss?uid=swg24023835])

For 6.1.0.0 through 6.1.0.6

CVEID: CVE-2013-6316 [http://cve.mitre.org/cgi-bin/cvename.cgi?name=CVE-2013-6316]
DESCRIPTION: 
In some specific situations, when using WCM context processors in combination with rendering Taxonomy components, the access control restrictions on individual categories in the taxonomy are not always enforced correctly. This occurs if the context processor changes the effective content selection. Properties of WCM content could be displayed to users, who do not have access rights to see them.

CVSS:
CVSS Base Score: 4.3
CVSS Temporal Score: See https://exchange.xforce.ibmcloud.com/vulnerabilities/88597 [https://exchange.xforce.ibmcloud.com/vulnerabilities/88597] for the current score
CVSS Environmental Score*: Undefined
CVSS Vector: (AV:N/AC:M/Au:N/C:P/I:N/A:N)


AFFECTED PRODUCTS AND VERSIONS: 
WebSphere Portal 8
WebSphere Portal 7


REMEDIATION: 
The recommended solution is to apply PI04897 [http://www.ibm.com/eserver/support/fixes/fixcentral/swg/quickorder?productid=WebSphere%20Portal&brandid=5&apar=PI04897] as soon as practical.
Fix: Apply a Cumulative Fix containing PI04897 [http://www.ibm.com/eserver/support/fixes/fixcentral/swg/quickorder?productid=WebSphere%20Portal&brandid=5&apar=PI04897].

For 8.0.0 through 8.0.0.1
▪ Upgrade to Fix Pack 8.0.0.1 with Cumulative Fix 9 (CF09)
(Combined Cumulative Fixes for WebSphere Portal 8.0.0.1: http://www-01.ibm.com/support/docview.wss?uid=swg24034497 [http://www-01.ibm.com/support/docview.wss?uid=swg24034497])

CVEID: CVE-2013-6723 [http://cve.mitre.org/cgi-bin/cvename.cgi?name=CVE-2013-6723]
DESCRIPTION: 
In some specific situations, when using WCM navigator components that reference other WCM components and in addition specify the compute=”always” option, access control restrictions on individual components are not always enforced correctly. The access control check for the referenced components works not correctly, if the navigator contains more than one entry. Referenced WCM components could be displayed to users, who do not have access to see them.

CVSS:
CVSS Base Score: 4.3
CVSS Temporal Score: See https://exchange.xforce.ibmcloud.com/vulnerabilities/89278 [https://exchange.xforce.ibmcloud.com/vulnerabilities/89278] for the current score
CVSS Environmental Score*: Undefined
CVSS Vector: (AV:N/AC:M/Au:N/C:P/I:N/A:N)


AFFECTED PRODUCTS AND VERSIONS: 
WebSphere Portal 8 (Fix Pack level 8.0.0.1 only)


REMEDIATION: 
The recommended solution is to apply PI05684 [http://www.ibm.com/eserver/support/fixes/fixcentral/swg/quickorder?productid=WebSphere%20Portal&brandid=5&apar=PI05684] as soon as practical.
Fix: Apply a Cumulative Fix containing PI05684 [http://www.ibm.com/eserver/support/fixes/fixcentral/swg/quickorder?productid=WebSphere%20Portal&brandid=5&apar=PI05684].

For 8.0.0.1
▪ Apply Cumulative Fix 9 (CF09)
