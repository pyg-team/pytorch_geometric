APAR STATUS
 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  When trying to open the Process Designer from Case Manager
   Builder using Google Chrome browser, V40 or later, it fails and
   the browser's
   debug console shows following error:
   Uncaught TypeError: undefined is not a function
      dojo.declare.openProcessDesignerCallBack
      dojo.declare.processDesignerSelection
      (anonymous function)
      ...(anonymous function)...
      ...onItemClick
      (anonymous function)
      :
   
   This issue occurs when using Google Chrome browser V40 or later
   for Case
   Manager versions 5.2.0.2-IF001 and prior. Issue is not
   reproducible when using Google Chrome for Case Manager 5.2.1.x
   versions.
   
   
    
   
   

LOCAL FIX
 *  Use one of the following 3 workarounds:
   1. Upgrade Case Manager to the latest Version of 5.2.1.x.
   2. For Case Manager versions 5.2.0.2-IF001 and prior, use
   Mozilla FireFox or Internet Explorer browsers
   3. Open Process Designer using Google Chrome browser from
   WorkplaceXT.
