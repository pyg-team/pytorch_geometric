ABSTRACT
 Further details about WebSphere Cast Iron Cloud integration V6.4.0.0 and the fixes and improvements included in this offering 

CONTENT


 On the 28th June 2013, WebSphere Cast Iron Cloud integration V6.4.0.0 was released. The announcement document can be viewed here [http://www-01.ibm.com/support/docview.wss?uid=swg21641628] 

 WebSphere Cast Iron Cloud integration V6.4.0.0 builds on V6.3.0.0 of the offering and provides the equivalent function along with the following improvements: 

 

 *  WebSphere Cast Iron Hypervisor Edition V6.4 can now operate on IBM Workload Deployer 3.1.0.7 
 * WebSphere Cast Iron Hypervisor Edition V6.4 can now operate on IBM PureApplicationâ„¢ System W1500 1.0.0.4 
 * HTTP queue re-design enhancement which improves performance during high loads with HTTP activities 

We strongly advise that any WebSphere Cast Iron users considering upgrading, to migrate to WebSphere Cast Iron V6.4 instead of V6.3.  In addition to the feature improvements, the following APAR fixes are included in V6.4.0.0 of the offering. These APARs are not currently available in fix packs for any other WebSphere Cast Iron releases. However, the APARs are planned for future releases but the release dates are not confirmed at this time.
