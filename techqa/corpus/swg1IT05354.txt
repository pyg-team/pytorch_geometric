APAR STATUS
 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  When configuring the PureApplication System V2.0 Call Home
   feature using the command line interface described in the steps
   outlined in the Knowledge Center topic titled
   "Call Home - update configuration parameters" located at
   (this URL is all one line with no spaces):
    http://www-01.ibm.com/support/knowledgecenter/SSCR9A_2.0.0/doc [http://www-01.ibm.com/support/knowledgecenter/SSCR9A_2.0.0/doc]
    /clihelp/callhome_update_properties.dita
   
   the user gets the error:
   
   HTTP response: 500 Internal Server Error
   
   
    
   
   

LOCAL FIX
 *  The user can use the System Console to configure the Call Home
   feature. The problem only occurs when using the command line
   interface.
   
   
    
   
   

PROBLEM SUMMARY
 *  The command line interface function to configure the Call Home
   feature was resulting in an HTTP error 500.
   
   
    
   
   

PROBLEM CONCLUSION
 *  A code fix is being provided in PureApplication System V 2.0.0.1
   so that if a value is missing from the system database and
   provided in the request body the error 500 is not generated.
   
   
    
   
   

TEMPORARY FIX
