 PRODUCT LIFECYCLE

ABSTRACT
 This document provides information about operating systems supported by each component of License Metric Tool. The information is sorted by operating system and by component.

REQUIREMENTS
* Discovery on IBM i is based on disconnected scans, a semi-automated procedure for collecting the data about hardware and software inventory.

** Some operating systems are supported only by lower versions of the BigFix agent. It does not mean that the BigFix platform must be downgraded to a lower version to support a specific operating system. The latest version of BigFix supports all lower clients. You can install BigFix 9.5 and yet connect to a client in version 9.0, 9.1 or 9.2.
