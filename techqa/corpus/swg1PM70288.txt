Title: IBM PM70288: PORTAL V8 CF 01 UPGRADE FAILS IN CLUSTER WHEN LOCAL DMGR WAS CREATED MANUALLY - United States

ERROR DESCRIPTION
 *  When a user has created a Portal v8 cluster using a local DMGR,
   and the DMGR was created manually (instead of during Portal
   installation or using IIM -> Modify), the cumulative fix 01
   installation fails during the CONFIG-WP-PTF-CF configuration
   script:

LOCAL FIX
 *  Install the CF silently and add this flag to the response file:
   
   <data
   key='user.wp.dmgr.profilename,com.ibm.websphere.PORTAL.SERVER.v8
   0' value='<dmgr_profile_name>'/>
   
   where <dmgr_profile_name> is your actual DMGR profile name.
