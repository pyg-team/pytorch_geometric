APAR STATUS
 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  During library deletion, an infinite loop occurs when a
   controllable has children which cannot be deleted (eg if there
   are references to the child from another library)
   
   
    
   
   

LOCAL FIX
 *  Stop retrying to delete items which have children or have
   external references
   
   
    
   
   

PROBLEM SUMMARY
