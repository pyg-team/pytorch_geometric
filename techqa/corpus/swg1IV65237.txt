 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  The user can not clear Primary Email field in User application.
   
   STEPS TO REPRODUCE:
   -----------------------------------
   1.  Login to SCCD.
   2.  Open "Security" > "Users" application.
   3.  Open a User (which has "Primary Email" value) and clear its
       "Primary Email" field.
   4.  Move to another field with tab key.
   
   "Primary Email" field is marked with red "X".
   
   5.  Click "X" mark.
   
   The following message is displayed. No value is set in {0}.
   
   > Record already exists for {0}. Ensure that the key value for
   > the given record is unique. (MBXAA4129)
   
   6.  Click "Save" icon.
   
   The following message is displayed.
   
   > BMXAA8123E - Error exist in the application that prevent this
   > action from being performed. Correct the errors and try again.
