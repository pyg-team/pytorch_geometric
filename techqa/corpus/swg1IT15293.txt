APAR STATUS
 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  When sending a message to the IBM MQ Queue Manager via MQPUT,
   there could be a race condition between shared conversations
   when
   they try to write to the same multiplexed physical connection,
   causing high CPU usage.
   
   
    
   
   

LOCAL FIX

PROBLEM SUMMARY
 *  Affected is use of MQ shared conversations.
   
   When sending a message to the IBM MQ Queue Manager via MQPUT,
   there could be a race condition between shared conversations
   when
   they try to write to the same multiplexed physical connection,
   causing high CPU usage.
