Websheet Export Limitations 
The following limitations about exporting slices, snapshots, and PDF files from TM1 Web apply only when Microsoft® Excel is not present on the TM1 Web server. 

This condition applies when you set ExcelWebPublishEnabled=T in the tm1s.cfg file. For more details, see ExcelWebPublishEnabled parameter: 
http://pic.dhe.ibm.com/infocenter/ctm1/v9r5m0/topic/com.ibm.swg.im.cognos.tm1_op.9.5.2.doc/tm1_op_id1889ExcelWebPublishEnabled.html [http://pic.dhe.ibm.com/infocenter/ctm1/v9r5m0/topic/com.ibm.swg.im.cognos.tm1_op.9.5.2.doc/tm1_op_id1889ExcelWebPublishEnabled.html] 

Websheets that contain TM1 Active Forms do not successfully export as either a slice, snapshot, or PDF file when Microsoft® Excel is not present on the TM1® Web server. An "Operation Failed" error displays. 

The following limitations apply to exporting Websheets that do not contain Active Forms:  * Layout and formatting may be inconsistent between the Websheet and the resulting Excel worksheet/workbook 
 * OLE and form controls in the Websheet are either not exported or are converted to images 
 * The selected values and states of form controls are not updated/displayed in the resulting worksheet 
 * Headers and footers in the worksheet are not exported 
 * Images in the Websheet are not exported when exporting to PDF file 
 * Charts in the Websheet are exported to a separate page in the resulting PDF file
