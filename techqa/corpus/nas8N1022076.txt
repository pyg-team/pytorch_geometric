Locate useful installation tools to help you install your new system.





IBM System Planning Tool [//www.ibm.com/systems/support/tools/systemplanningtool/]
The IBM System Planning Tool (SPT) is a browser-based application that helps you design system configurations; it is particularly useful for designing logically partitioned systems. The SPT is integrated with the IBM Systems Workload Estimator (WLE), which enables you to plan a system based on existing performance data or based on new workloads.

Power Systems Capacity on Demand [//www.ibm.com/systems/power/hardware/cod/index.html]
A fast method of activating extra processor and memory capacity without disrupting existing operations.

TCP/IP on IBM i [//www.ibm.com/systems/power/software/i/tcpip/index.html]
IBM i offers robust support for TCP/IP, the protocol of the Internet. With the integrated TCP/IP capabilities of IBM i, users can link to networks throughout the world.

IBM i Virtualization [//www.ibm.com/systems/power/software/i/virtualization.html]
"Virtualization" is the concept of executing software in an environment that minimizes or eliminates the software's dependence on the hardware on which it runs. Simplify management of IT resources with storage virtualization provided by the IBM i operating system.

Platform management on IBM i: Tools [//www.ibm.com/systems/power/software/i/management/platform/tools.html]
IBM tools available to manage your IBM i environment.

Electronic Service Agent [//www.ibm.com/support/esa/index.htm]
IBM Electronic Service Agent is a monitoring tool that proactively reports hardware events as soon as they are detected. A service request is electronically routed directly to the appropriate IBM support engineer, with no intervention needed from you. The result: higher availability and faster fix delivery for your environment.
