ERROR DESCRIPTION
 *  An Error is seen when an object is modified and saved on a
   module opened with a default RCR after changing to Shareable
   eidt , if another module is kept open in shareable edit for
   unmanaged changes.
   
   1. open one module in Exclusive edit mode with RCR
   2. change mode to Shareable edit after its opened, then select
   yes or no when saved changes dialog box appears
   3. open another module in Shareable edit with Unmanaged changes
   4. go back to the first module opened in step 1, unlock then
   modify a requirement, then save it.
   5. Error dialog box appears
