Title: IBM Security Bulletin: Security vulnerability in IBM Business Process Manager shipped with IBM SmartCloud Orchestrator and IBM Cloud Orchestrator (CVE-2014-8730) - United States

Text:
Security; orchestrator; cloud; tls SECURITY BULLETIN

SUMMARY
 IBM Business Process Manager and DB2 Enterprise Server Edition are shipped as components of IBM SmartCloud Orchestrator and IBM Cloud Orchestrator. Information about a security vulnerability (CVE-2014-8730) affecting both IBM Business Process Manager and IBM DB2 has been published in a security bulletin. 

VULNERABILITY DETAILS
For IBM Business Process Manager, consult the security bulletin entitled, TLS padding vulnerability affects IBM HTTP Server shipped with IBM Business Process Manager (CVE-2014-8730) [http://www.ibm.com/support/docview.wss?uid=swg21692582] for vulnerability details and information about fixes.
For IBM DB2 Enterprise Server Edition, consult the security bulletin entitled, TLS padding vulnerability affects IBM DB2 LUW (CVE-2014-8730) [http://www.ibm.com/support/docview.wss?uid=swg21671732] for vulnerability details and information about fixes. 

AFFECTED PRODUCTS AND VERSIONS
 

Principal Product and Version Affected Supporting Product and Version IBM Cloud Orchestrator 2.4 IBM Business Process Manager 8.5.0.1 [https://clearinghouse.raleigh.ibm.com/wps/myportal/Deliverable?ProductID=1344950553051] IBM DB2 Server Enterprise Edition 10.5 IBM HTTP Server 8.5 [https://clearinghouse.raleigh.ibm.com/wps/myportal/Deliverable?ProductID=1344950553051] IBM Cloud Manager with OpenStack 4.1 IBM SmartCloud Orchestrator 2.2, 2.2.0.1, 2.3, and 2.3.0.1 Business Process Manager Standard 8.0.1 DB2 Enterprise Server Edition 10.1 REMEDIATION/FIXES
None

WORKAROUNDS AND MITIGATIONS
As for IBM Business Process Manager, follow the TLS padding vulnerability affects IBM HTTP Server shipped with IBM Business Process Manager (CVE-2014-8730) [http://www.ibm.com/support/docview.wss?uid=swg21692582] security bulletin. 

 

As for IBM DB2 Server Enterprise Edition, follow the TLS padding vulnerability affects IBM DB2 LUW (CVE-2014-8730) [http://www.ibm.com/support/docview.wss?uid=swg21692618] security bulletin. 

 

As for IBM Cloud Orchestrator 2.4, if you install IBM Cloud Orchestrator 2.4.0 Fix Pack 1 after following the TLS padding vulnerability affects IBM HTTP Server shipped with IBM Business Process Manager (CVE-2014-8730) [http://www.ibm.com/support/docview.wss?uid=swg21692582] security bulletin, you need to re-execute the steps under "Workarounds and Mitigations," which are described in the TLS padding vulnerability affects IBM HTTP Server shipped with IBM Business Process Manager (CVE-2014-8730) [http://www.ibm.com/support/docview.wss?uid=swg21692582] security bulletin. These steps are necessary because the httpd.conf configuration file is overwritten when installing IBM Cloud Orchestrator 2.4.0 Fix Pack 1.
