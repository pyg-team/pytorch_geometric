Title: IBM WebSphere DataPower Blade XI50B uses a separate base 3.8.2.0 build than the standard XI50 appliance, but merges for both to use same firmware version starting with the 3.8.2.3 firmware - United States

Text:
 TECHNOTE (FAQ)

QUESTION
 Which firmware should be used for the WebSphere DataPower Blade XI50B on 3.8.2 firmware release? 

CAUSE
Due to product release timing, the Blade XI50B (blade) has a unique 3.8.2.0 firmware build that is comparable to the "non-blade" XI50 3.8.2.2 fixpack release.

NOTE: The 3.8.2.3 firmware release and higher supports both the XI50 and XI50B.

This means that:

1) Prior to 3.8.2.3 firmware, the Blade XI50B requires the "blade-specific" 3.8.2.0 firmware.

2) Blade XI50B should NOT have the "non-blade" XI50 3.8.2.0, 3.8.2.1, or 3.8.2.2 firmware installed. This will cause unexpected results and errors.

2) Starting at firmware 3.8.2.3 and higher, Blade XI50B can use the same firmware builds as "non-blade" XI50.

ANSWER
To confirm the proper firmware is installed, view the firmware details: 

 * Using the CLI: show version 
 * Using the WebGUI: Status -> System -> Version Information
