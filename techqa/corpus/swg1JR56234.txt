APAR STATUS
 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  Interim fix JR56232 [http://www-01.ibm.com/support/docview.wss?uid=swg1JR56232] adds new function in the Deployment editor
   in IBM Integration Designer to allow login configuration to be
   specified for a web service export. However, the server
   validation for installing an application and the serviceDeploy
   tool do not recognize these new properties, preventing the
   application from installing.
   
   PRODUCTS AFFECTED
   IBM Business Process Manager Advanced
   
   
    
   
   

LOCAL FIX

PROBLEM SUMMARY
 *  No additional information is available.
   
   
    
   
   

PROBLEM CONCLUSION
 *  A fix is available for IBM Business Process Manager V8.5.0.1
   that updates the server validation to recognize the login
   configuration deployment descriptor properties generated from
   Integration Designer.
   
   On Fix Central (http://www.ibm.com/support/fixcentral), search
   for JR56234:
   
       1. Select IBM Business Process Manager with your edition
   from the product selector, the installed version to the fix pack
   level, and your platform, and then click Continue.
       2. Select APAR or SPR, enter JR56234, and click Continue.
   
   When you download fix packages, ensure that you also download
   the readme file for each fix. Review each readme file for
   additional installation instructions and information about the
   fix.
