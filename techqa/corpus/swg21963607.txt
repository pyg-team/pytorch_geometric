Title: IBM Security Bulletin: Vulnerability in Apache Cordova affects IBM Worklight and IBM MobileFirst Platform Foundation (CVE-2015-1835) - United States

Text:
CVE-2015-1835 SECURITY BULLETIN

SUMMARY
 An Apache Cordova vulnerability was disclosed May 26, 2015. Apache Cordova is used by IBM Worklight and IBM MobileFirst Platform Foundation. Android applications built with the Cordova framework might allow a remote attacker to cause arbitrary commands to be executed in the application. IBM Worklight and IBM MobileFirst Platform Foundation have addressed the applicable CVE. 

VULNERABILITY DETAILS
CVEID:CVE-2015-1835 [http://cve.mitre.org/cgi-bin/cvename.cgi?name=CVE-2015-1835]
DESCRIPTION:The Apache Cordova might allow a remote attacker to execute arbitrary commands on the system. Applications built with the Cordova framework and lack explicit values set in the Config.xml can have undefined configuration variables set by Intent. By persuading a victim to click a specially crafted URL, an attacker might exploit this vulnerability to inject malicious content in applications and modify application characteristics including force-closing.
CVSS Base Score: 5.800
