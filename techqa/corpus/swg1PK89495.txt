LOCAL FIX

PROBLEM SUMMARY
 *  Problem:
   1. Help should explain that there are three (not two) ways
   of deploying a Portal/Portlet Project: Export, Deploy, and
   Run On Server. Run on Server method was not available.
   2. Help should have a page that explains what exactly Run On
   Server does and
   3. Help should clarify that the two approaches should not be
   mixed.
   
   
    
   
   

PROBLEM CONCLUSION
 *  Conclusion:
   1. Documented three ways of deploying a Portal/Portlet
   Project: Export, Deploy, and Run On Server, added the
   missing Run on Server method.
   2. Added a new page that explains what exactly Run On Server
   does.
   3. And added a note stating that the Depolyment and Run on
   Server publishing methods are different from each other.
   
   Documentation enhanced in V7.5.4 release
