QUESTION
 Does IBM® WebSphere® Portal Server v6.0 come with Rational® Application Developer (RAD) v7.0 and where can you find information on installing WebSphere Portal Server v6.0 with Rational Application Developer v7.0? 

ANSWER
 

WebSphere Portal Server v6.0 comes with Rational Application Developer v7.0. 
It is an optional install and optional multi-part download for Rational Application Developer v7.0 from IBM Passport Advantage [http://www-306.ibm.com/software/howtobuy/passportadvantage/index.html].








In order to install the WebSphere Portal Server v6.0 component of Rational Application Developer v7.0, you need to review the Installing the WebSphere Portal Version 6.0 Test Environment section of the Rational Application Developer v7.0 Installation Guide [http://download.boulder.ibm.com/ibmdl/pub/software/rationalsdp/v7/rad/70/docs/install_instruction/install.html]. 

Note in step 10: 
"....for business process support, and click Next to continue. Important: If you...." 

TIP: Unless you need WPPS, also referred to as Business Process server, it is suggested that you do not install this component for the following reasons:  * WPPS will take up unnecessary disk space 
 * WPPS will increase the startup time of the WebSphere Portal Server v6.0
   
   Note: Review the following artcile for more tips on improving performnace:
   Installing and configuring WebSphere Portal V6.0 Servers for development with Rational Application Developer V7.0 and Rational Software Architect V7.0 [http://www.ibm.com/developerworks/rational/library/07/0327_riordan/index.html]
