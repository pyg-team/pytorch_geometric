Title: IBM Connections 5.5 CR requirements for IBM FileNet for use with Connections Content Manager (CCM) - United States

Text:
 PREVENTIVE SERVICE PLANNING

ABSTRACT
 This document contains information on the requirements for IBM FileNet for use with IBM Connections 5.5 CR Connections Content Manager (CCM) 

CONTENT
IBM Connections 5.5 environments using Connections Content Management (CCM) are required to upgrade the FileNet installation as part of the upgrade to a 5.5 Cumulative Refresh (CR). After applying IBM Connections 5.5 CR2, upgrades are required for FileNet Content Engine Server & Client and FileNet Collaboration Services. There are no required FileNet updates when upgrading from IC 5.5 CR2 to IC 5.5 CR3.

SECTION 1 - IMPORTANT NOTES 

 * These procedures are required only if you have installed Connections Content Manager (CCM) in IBM Connections 5.5. 

 * Make sure the latest Connections 5.5 CR update is installed before attempting to upgrade the Content Engine (CE) server, CE client, and FileNet Collaboration Services (FNCS).

SECTION 4 - INSTALLATION INSTRUCTIONS 

It is important that you understand requirements and dependencies before updating FileNet. The topic Updating IBM Connections 5.5 with the latest Cumulative Refresh (CR) [http://www.ibm.com/support/knowledgecenter/SSYGQH_5.5.0/admin/migrate/c_installing_fix-packs.html] has important information that must be followed when applying IBM Connections 5.5 CRs and the subtopic Post-update tasks has information on deploying the latest FileNet FixPacks and Interim Fixes. 

Installation instructions for the FileNet fixpacks with IBM Connections are detailed in "Section 5 - Instructions on applying Cumulative Refresh (CR) updates" in the Upgrade Strategy for IBM Connections 5.5 document [http://www.ibm.com/support/docview.wss?uid=swg21972646].
