Title: IBM Security Bulletin: IBM WebSphere MQ is affected by a privilege escalation vulnerability (CVE-2017-1612) - United States

Text:
 SECURITY BULLETIN

SUMMARY
 IBM WebSphere MQ has addressed the following vulnerability.

An undocumented environment variable intended for use by IBM support could be used to execute untrusted code using the 'mqm' user. 

VULNERABILITY DETAILS
CVEID: CVE-2017-1612 [http://cve.mitre.org/cgi-bin/cvename.cgi?name=CVE-2017-1612]
DESCRIPTION: IBM WebSphere MQ service trace module could be used to execute untrusted code under 'mqm' user
CVSS Base Score: 7.4
CVSS Temporal Score: See https://exchange.xforce.ibmcloud.com/vulnerabilities/132953 [https://exchange.xforce.ibmcloud.com/vulnerabilities/132953] for the current score
CVSS Environmental Score*: Undefined
CVSS Vector: (CVSS:3.0/AV:L/AC:H/PR:N/UI:N/S:U/C:H/I:H/A:H)
