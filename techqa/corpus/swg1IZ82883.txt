Title: IBM IZ82883: AGENT BUILDER LOCAL INSTALL FAILS ON AIX AND LINUX - United States

Text:
 SUBSCRIBE
You can track all active APARs for this component.



APAR STATUS
 * CLOSED AS PROGRAM ERROR.
    
   
   

ERROR DESCRIPTION
 *  The Agent Builder "Generate Agent" wizard option to "Generate
   the agent files in an ITM installation on this machine", that is
   local install of the agent, fails on AIX and Linux with the
   following message:
   
   "KQZ0161W There is not agent installation script for this
   operating system. The agent is not supported on this operating
   system."
   
   
   RECREATE INSTRUCTIONS:
   
   
   In the Agent Builder, after creating an agent on AIX or Linux:
   - select the "Generate Agent Wizard"
   - select the "Generate the agent files in an ITM installation on
   this machine" option
   - select "Finish"
   
   The install fails with "KQZ0161W There is not agent installation
   script for this operating system.  The agent is not supported on
   this operating system."
   
   
    
   
   

LOCAL FIX
 *  In the Agent Builder, after creating an agent on AIX or Linux:
   
   - select the "Generate Agent Wizard"
   - select the "Create a compressed file so that the agent can be
   installed on another system" option and specify an Output
   directory.
   - Select "Finish".
   
   A K<pc>.tar and K<pc>.zip where <pc> is the agent product code,
   are generated in the output directory.  Use an extract utility
   to expand the archive file and use the installIra.sh script to
   install the agent and application support locally.
