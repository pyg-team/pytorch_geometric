ERROR DESCRIPTION
 *  IndexOutOfBoundsException for BPC Explorer "Manage Views" if
   customer has stored queries in database.
   .
   If a customer has created stored queries without user-defined
   properties in his database he will not be able to use the
   "Manage Views" option in BPC Explorer.
   A java.lang.IndexOutOfBoundsException is thrown.
   
   
    
   
   

LOCAL FIX

PROBLEM SUMMARY
 *  IndexOutOfBoundsException for BPC Explorer "Manage Views" if
   customer has stored queries in database.
   .
   If a customer has created stored queries without user-defined
   properties in his database he will not be able to use the
   "Manage Views" option in BPC Explorer. A
   java.lang.IndexOutOfBoundsException is thrown.
   
   
    
   
   

PROBLEM CONCLUSION
 *  All stored queries are checked if they are valid stored queries
   for BPC Explorer. Invalid stored queries are ignored.
   
   Problem fixed, iFix provided on top of BPM 7.5.0.1
